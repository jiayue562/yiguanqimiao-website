# 龙龟对话 - 知识存储系统

## 📂 存储结构

```
龙龟对话/
├── 日常对话/
│   └── YYYY-MM-DD.md          ← 每日对话记录
├── 方法论/
│   ├── 思维模型/
│   ├── 五色光/
│   └── 五行人格/
├── 人格图谱/
│   └── 一心三界五行九层/
├── 项目记录/
│   └── 观其妙书院/
├── 学习笔记/
│   └── 技能安装/
└── 灵感火花/
    └── 闪念记录/
```

---

## 🏷️ 分类规则

| 对话类型 | 存储位置 | 标签 |
|----------|----------|------|
| 日常聊天、事务处理 | 日常对话/ | #日常 |
| 方法论探讨、思维模型 | 方法论/ | #方法论 |
| 五行、九层、人格分析 | 人格图谱/ | #人格图谱 |
| 书院项目、计划讨论 | 项目记录/ | #项目 |
| 技能学习、技术笔记 | 学习笔记/ | #学习 |
| 灵感、洞见、顿悟 | 灵感火花/ | #灵感 |

---

## 📝 存储原则

1. **自动分类**：对话结束后，龙龟自动判断类型并存储
2. **标签系统**：每个文件添加标签，便于检索
3. **双向链接**：使用 Obsidian 双向链接关联相关知识
4. **定期整理**：每周提炼重要内容到知识库其他位置

---

## 🔄 工作流程

```
对话结束 → 龙龟判断类型 → 创建/追加到对应文件 → 添加标签和链接
```

---

*龙龟会自动将对话分门别类存储，无需手动操作*
