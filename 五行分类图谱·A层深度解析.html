A1 五性：功能的根本定义

核心来源：《尚书·洪范》

五性定义：不是物质，而是五种功能趋势、运动能势

A1.1 木曰曲直

| 维度 | 内容 |

|------|------|

| 字面义 | "曲"指能弯曲，"直"指能伸直 |

| 自然象 | 树木生长时能屈能伸、向上向外舒展的生命力 |

| 抽象核心 | 生长、升发、条达、舒畅的运动能势 |

| 能量特征 | 万物从萌动到舒展的原始动力 |

| 五行定位 | 木行人的核心特质：开创力、引领性 |

引申内涵：

  • 曲直 = 刚柔并济 = 能屈能伸

  • 条达 = 通畅无阻 = 肝胆疏泄

  • 生发 = 从无到有 = 创新创造的源头

A层核心金句

A1 五性核心金句

  1. "木曰曲直"不是树木，而是生长升发的生命力。

  2. "火曰炎上"不是火焰，而是温热升腾的光明势能。

  3. "土爰稼穑"不是土地，而是承载化生的宇宙平台。

  4. "金曰从革"不是金属，而是肃杀收敛的清肃之力。

  5. "水曰润下"不是水流，而是润下闭藏的涵养之道。

A2 五脏神核心金句

  1. "肝藏魂"——魂是创造力的精神基础。

  2. "心藏神"——神是意识清明的心灵核心。

  3. "脾藏意"——意是思虑专注的认知能力。

  4. "肺藏魄"——魄是行动魄力的生理基础。

  5. "肾藏志"——志是志向坚定的深层动力。

A3 五德核心金句

  1. "木主仁"——仁者爱人，如木之生机勃发。

  2. "火主礼"——礼者明理，如火之照破黑暗。

  3. "土主信"——信者诚实，如土之厚德载物。

  4. "金主义"——义者宜也，如金之清肃果断。

  5. "水主智"——智者不惑，如水之深流智慧。

A4 五瑞灵核心金句

  1. 青龙主东方——春天生机的宇宙象征。

  2. 朱雀主南方——光明文明的星空表达。

  3. 麒麟主中央——中和化育的祥瑞代表。

  4. 白虎主西方——肃杀威严的庚金之灵。

  5. 玄武主北方——藏匿智慧的龟蛇合体。

A5 五心核心金句

  1. 慈悲心是仁德的升华——化解嗔恨的甘露。

  2. 恭敬心是礼德的升华——驱散傲慢的阳光。

  3. 感恩心是信德的升华——转化怨尤的妙药。

  4. 惭愧心是义德的升华——清洗贪婪的清泉。

  5. 谦卑心是智德的升华——熄灭愚痴的明灯。

A6 五空核心金句

  1. 地水火风空——五大揭示世界构成的根本。

  2. 土水火木金——五行描绘能量运行的轨迹。

  3. 五大与五行——人类对宇宙本质的共同探索。
