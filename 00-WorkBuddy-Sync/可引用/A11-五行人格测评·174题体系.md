## 三、测评计分与报告

### 174题测评的计分算法是什么？

174题测评采用「五维加权计分算法」：

```
总分 = 心理部分(20%) + 身体部分(30%) + 灵性部分(15%) + 阴阳子维度(30%) + 社会掩饰(5%)
```

**阴阳面判定规则**：
- 阳面总分 > 阴面总分 + 2.0 → 阳面主导·健康区（4-6层）
- 阴面总分 > 阳面总分 + 2.0 → 阴面主导·不健康区（1-3层）
- \|阳面总分 - 阴面总分\| ≤ 2.0 → 阴阳混杂·转化期（3-4层边界）
