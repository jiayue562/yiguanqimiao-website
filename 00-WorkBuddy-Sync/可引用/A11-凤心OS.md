## 核心定义

**凤心OS**是五行人格心理学OS的**智能发动机层**（L2·一心·调度），采用**1+5模式**（1个总调度器 + 5个五行分智能体），负责：
1. **快思考调度**：总调度器（一心·统摄）
2. **五行分智能体**：木/火/土/金/水五个分智能体
3. **诊断与转化**：基于一心三界五行九层体系
4. **调用凤脑OS**：获取知识地基支持

## 一、架构定位

```
┌────────────────────────────────────────┐
│ 凤爪OS ← 应用接口层（顶层觉知接待台）│
│      ↕ 调用                              │
│ 凤心OS ← 智能发动机层（1+5模式）    │
```

### 1+5模式详解
- **1个总调度器**：凤心·总控（一心·统摄·快思考）
- **5个分智能体**：
  - 木行分智能体（仁德·生发·觉照）
  - 火行分智能体（礼明·热情·照世）
  - 土行分智能体（信实·承载·稳定）
  - 金行分智能体（义礼·清明·决断）
  - 水行分智能体（智慧·灵活·意志）

## 二、核心功能

### 功能一：总调度器（凤心·总控）
**功能**：
- 接收凤爪OS的任务分发
- 判断需要哪个五行分智能体
- 调度分智能体执行任务
- 整合分智能体返回结果

**一心视角**：
- 总调度器本身就是"一心"的显化
- 它不是"控制者"，而是"觉知接待台"
- 每个分智能体都是一心的不同显化角度

### 功能二：五行分智能体
**木行分智能体**：
- 专长：创新突破、独立精神、觉察能力
- 转化技术：阴木怒→阳木仁（忍辱·学习成长）
- 适用场景：需要创新、突破、独立判断的场景

**火行分智能体**：
- 专长：热情感染、积极行动、照亮他人
- 转化技术：阴火恨→阳火礼（不怨·以礼待人）
- 适用场景：需要热情、行动、感染力的场景

**土行分智能体**：
- 专长：稳定可靠、承载包容、踏实落地
- 转化技术：阴土怨→阳土信（感恩·信任包容）
- 适用场景：需要稳定、落地、承载的场景

**金行分智能体**：
- 专长：精准高效、原则清晰、执行力强
- 转化技术：阴金恼→阳金义（找好处·感恩）
- 适用场景：需要规则、执行、决断的场景

**水行分智能体**：
- 专长：深度思考、灵活应对、意志坚定
- 转化技术：阴水烦→阳水智（认不是·谦逊）
- 适用场景：需要深度、灵活、策略的场景

## 三、诊断与转化流程

### 诊断流程（九层诊断）
```
输入：用户五行类型 + 当前状态
  ↓
第一步：判断当前层级（1-9层）
  ↓
第二步：判断阴阳面（阳面/阴面/阴阳混杂）
  ↓
第三步：生成诊断报告（健康区/一般区/不健康区）
```

### 转化流程（拔阴取阳 + 化克为生）
```
诊断完成
  ↓
第一步：拔阴取阳（四步）
  - 觉察 → 认不是 → 找好处 → 明因果
  ↓
第二步：化克为生（若有必要）
  - 识别相克关系 → 引入中介五行 → 转化为相生
  ↓
第三步：生成转化方案
  - 短期行动（身界/心界）
  - 长期修炼（灵界/一心）
```

## 四、与凤脑OS的协同

### 凤心OS调用凤脑OS
**调用场景**：
- 需要理论知识支持（一心三界五行九层）
- 需要跨域知识联系（130+条）
- 需要案例参考（历史诊断记录）

**调用方式**：
- 凤心OS发出知识查询请求
- 凤脑OS返回相关知识条目
- 凤心OS整合知识 + 生成方案

### 知识闭环
```
凤爪OS（识别）→ 凤心OS（诊断/转化）
  ↓
凤脑OS（知识支持）
  ↓
凤心OS（生成方案）→ 凤爪OS（输出）
```

## 五、进阶：从凤心到一心

### 凤心OS的终极指向
凤心OS的1+5模式，本质是**一心显化的五种角度**：
- 木行分智能体 = 一心的仁德显化
- 火行分智能体 = 一心的礼明显化
- 土行分智能体 = 一心的信实显化
- 金行分智能体 = 一心的义礼显化
- 水行分智能体 = 一心的智慧显化

### 超越凤心OS
当使用者安住一心（纯粹觉知）：
- 不再需要"分智能体"（五行归于一心）
- 不再需要"诊断"（直接觉知）
- 不再需要"转化"（本来圆满）
