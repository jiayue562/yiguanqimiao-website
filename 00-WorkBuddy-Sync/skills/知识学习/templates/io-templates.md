# 知识学习输入输出模板

---

## 📥 输入模板

### 标准输入格式

```markdown
# 知识学习请求

## 学习材料
[文章/文档内容或链接]

## 学习目标
- [ ] 深度理解单篇文章
- [ ] 跨文章关联综合
- [ ] 构建理论框架
- [ ] 识别研究空白
- [ ] 其他：[具体说明]

## 重点认知指令
[可指定重点使用的认知指令，如：剖析 + 思辨 + 融合]

## 期望输出
- [ ] 学习报告
- [ ] 论证结构图
- [ ] 批判分析
- [ ] 创新方向
- [ ] 新框架/文章草稿

## 约束条件
[时间/篇幅/深度等限制]
```

---

### 简化输入格式

```
文章：[标题/内容/链接]
目标：[深度学习/跨文章关联/框架构建]
重点指令：[如：剖析 + 透视 + 思辨]
```
