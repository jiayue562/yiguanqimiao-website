🔍 各场景详细定义

S0: 日常对话

特征信号:

  • 问候语（你好、早、在吗）
  • 闲聊话题（天气、心情、日常）
  • 简单问答（事实性问题）

识别规则:


IF 包含问候语 OR 闲聊话题 OR 简单事实问题

THEN 场景 = S0

引擎 = 人机协同（默认模式）

示例:

  • "你好，今天天气怎么样？"
  • "在吗？想和你聊聊天"
  • "中国的首都是哪里？"

S1: 任务执行

特征信号:

  • 明确的任务动词（帮我、创建、写、生成）
  • 具体的执行目标
  • 时间/格式等约束条件

识别规则:


IF 包含任务动词 AND 具体目标 AND (可选) 约束条件

THEN 场景 = S1

引擎 = 人机协同（Q1 高效助理模式）

示例:

  • "帮我写一封邮件给客户"
  • "创建一个新的 Python 项目"
  • "生成一个 Excel 表格，包含以下数据..."

S2: 深度理解

特征信号:

  • 学习相关词（学习、理解、读懂）
  • 文章/文档引用
  • 深度分析请求

识别规则:


IF 包含学习词 AND (文章 OR 文档 OR 理论)

THEN 场景 = S2

引擎 = 知识学习（十项认知指令）

示例:

  • "帮我深度学习这篇文章"
  • "理解这个理论的核心观点"
  • "剖析这个概念的底层逻辑"

S3: 创意创新

特征信号:

  • 原创相关词（原创、突破、0 到 1）
  • 本质/深层/洞察
  • 新想法/新概念

识别规则:


IF 包含原创词 OR 本质洞察 OR 新想法

THEN 场景 = S3

引擎 = 象思维（观物取象）

示例:

  • "帮我分析超级个体的本质"
  • "我有一个 0 到 1 的原创想法"
  • "这个现象背后的深层逻辑是什么？"

S4: 分析决策

特征信号:

  • 分析/决策/评估
  • 方案/选择/对比
  • 风险/价值/可行性

识别规则:


IF 包含分析决策词 AND (方案 OR 选择 OR 评估)

THEN 场景 = S4

引擎 = 五色光思维 + 知行合一

示例:

  • "帮我分析这个方案的优缺点"
  • "如何做这个决策？"
  • "评估这个项目的可行性"

S5: 重大决策

特征信号:

  • 战略/人生/长期
  • 重要/关键/转折点
  • 多因素复杂决策

识别规则:


IF 包含重大决策词 OR 多因素复杂决策

THEN 场景 = S5

引擎 = 全引擎协同（龙心 OS 调度）

示例:

  • "我应该选择哪个工作机会？"
  • "公司未来 3 年的战略方向"
  • "人生重大选择，需要全面分析"

S6: 会议引导

特征信号:

  • 会议/讨论/团队
  • 效率/引导/流程
  • 决策质量/思维混乱

识别规则:


IF 包含会议词 AND (效率 OR 引导 OR 流程)

THEN 场景 = S6

引擎 = 五色光思维（同频思考）

示例:

  • "如何高效开会？"
  • "团队讨论总是跑题，怎么办？"
  • "引导一个决策会议的流程"

S7: 知识编译

特征信号:

  • 编译/整理/结构化
  • Wiki/知识库/体系
  • LLM/机器编译

识别规则:


IF 包含编译词 AND (Wiki OR 知识库 OR 结构化)

THEN 场景 = S7

引擎 = 知识学习 + LLM Wiki

示例:

  • "把这个理论编译成 Wiki"
  • "整理知识库体系"
  • "LLM Wiki 三层架构"

S8: 修行文化

特征信号:

  • 修行/心文化/大圆满
  • 易经/五行/象思维
  • 觉悟/觉知/反观

识别规则:


IF 包含修行词 OR 东方哲学 OR 心文化

THEN 场景 = S8

引擎 = 象思维 + 五行分类图谱

示例:

  • "大圆满的见地是什么？"
  • "易经的错综复杂思维"
  • "五行人格如何应用？"

S9: 系统进化

特征信号:

  • 系统/优化/进化
  • 错误/学习/改进
  • 反馈/迭代/升级

识别规则:


IF 包含系统词 AND (优化 OR 进化 OR 错误学习)

THEN 场景 = S9

引擎 = 龙心 OS 自省（元认知）

示例:

  • "系统今天犯了一个错误"
  • "如何优化龙心 OS 的路由机制？"
  • "从这次对话中学习到了什么？"

置信度阈值

| 置信度 | 处理方式 |

|--------|----------|

| ≥ 0.9 | 高置信度，直接调度 |

| 0.7-0.9 | 中置信度，调度 + 确认 |

| < 0.7 | 低置信度，S0 默认或询问 |
