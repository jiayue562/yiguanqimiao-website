🧱 三层记忆架构

L0 工作记忆 (Working Memory)

| 属性 | 详情 |

|------|------|

| 功能 | 当前任务状态、会话上下文 |

| 存储 | SESSION.md / 内存 |

| 容量 | 7-10 轮对话 |

| 更新频率 | 实时 |

| 对应开源 | 滑动窗口 / L0 工作记忆 |

L1 会话记忆 (Session Memory)

| 属性 | 详情 |

|------|------|

| 功能 | 单次会话完整记录 |

| 存储 | 日常笔记 / IMA |

| 容量 | 无限 |

| 更新频率 | 每会话 |

| 对应开源 | 会话日志 / 短期存储 |

L2 长期记忆 (Long-term Memory)

| 属性 | 详情 |

|------|------|

| 功能 | 人生档案、结构化知识 |

| 存储 | Obsidian + MEMORY.md |

| 容量 | 终身 |

| 更新频率 | 每日/每周沉淀 |

| 对应开源 | 向量数据库 + 知识图谱 |

🔄 知行合一三阶段

阶段 1: 表示空间 (记忆形成)

核心: 观察与记录完整经验

操作:

  • 捕获离散事实 + 完整上下文
  • 保留时间、地点、参与者元数据
  • 不压缩、不筛选、原始记录

工具:

  • SESSION.md (会话中)
  • journal/日常笔记 (会话后)
  • IMA 快速捕获 (移动端)

阶段 2: 压缩 (记忆演化)

核心: 提炼核心洞察

操作:

  • 合并去重
  • 提取跨情境洞察
  • 标记关键决策与教训
  • 转化为结构化知识

工具:

  • MEMORY.md (长期记忆)
  • LEARNINGS.md (经验转化)
  • ERRORS.md (错误规避)

阶段 3: 泛化 (记忆检索)

核心: 场景扩展应用

操作:

  • 根据场景自动激活相关记忆
  • 跨记忆推理与合成
  • 从"给出相关事实"到"情境重构"

工具:

  • 场景化路由引擎
  • 双向链接图谱
  • 语义检索
