## 关键五行组合

### 水火既济（组织新陈代谢）
```
熊进(火·运营) × 陈倩彤(水·人事)
  = 激活·点燃·推动 × 供给·滋养·流入
  = 一进一出 = 组织新陈代谢
```

### 火火共振（培训闭环）
```
熊进(火·运营) × 马循德(火·培训)
  = 搭台(熊进) + 唱戏(马循德)
  = 运营之火 → 培训之火 → 前厅之火 → 客户之火
```

### 金金共振（精准执击）
```
俞腾芳(金·店长) × 陈明武(金·堂食总)
  = 精准制击(店长) × 仪式执行(堂食总)
  = 扣分追踪 + 仪式标准化
```
