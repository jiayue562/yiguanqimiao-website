🔄 核心操作流程

Ingest（录入）

新资料 → raw/ → LLM 编译 → wiki/ → 人类审核

步骤:
1. 检测 raw/ 新文件
2. 阅读并提取关键信息
3. 创建 summaries 页
4. 提取实体和概念
5. 更新 index.md 和 log.md
6. 生成录入报告
7. 用户审核确认

Query（查询）

提问 → Wiki 搜索 → 交叉验证 → 综合答案 → 归档复利

步骤:
1. 搜索 index.md 定位
2. 阅读相关页面
3. 交叉验证多个来源
4. 生成综合答案
5. 标注来源引用
6. 高质量问答归档

Lint（检查）

一致性 + 完整性 + 时效性 + 积压 → 报告

检查项:
- 一致性校验（每周）
- 完整性补全（每周）
- 过时检测（每月）
- raw/ 积压（每周）
