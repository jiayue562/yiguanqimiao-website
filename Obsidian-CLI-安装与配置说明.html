使用方法

由于Windows批处理文件的编码限制，请直接使用PowerShell调用：

# 查看仓库信息
powershell -ExecutionPolicy Bypass -File "C:\Users\jia'yue\WorkBuddy\Claw\obsidian-cli.ps1" vault

# 列出所有笔记（前20个）
powershell -ExecutionPolicy Bypass -File "C:\Users\jia'yue\WorkBuddy\Claw\obsidian-cli.ps1" list

# 搜索笔记
powershell -ExecutionPolicy Bypass -File "C:\Users\jia'yue\WorkBuddy\Claw\obsidian-cli.ps1" search "龙心OS"

# 读取特定笔记
powershell -ExecutionPolicy Bypass -File "C:\Users\jia'yue\WorkBuddy\Claw\obsidian-cli.ps1" read "README.md"

手动安装原生Obsidian CLI（可选）

如需安装原生obsidian-cli：

# 使用Scoop（需要网络连接到GitHub）
scoop install obsidian-cli

# 或使用npm
npm install -g @yakitrak/obsidian-cli

原生版本需要配合Obsidian Local REST API插件使用。
