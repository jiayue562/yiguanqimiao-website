3. 学习成果自动归档 - 集成 Hooks

步骤 1: 编辑 self-learning/SKILL.md

在文件末尾添加：


## 🔗 Hooks 集成



### after_complete Hook



学习完成后自动触发归档：

hooks:

after_complete:

  • trigger: auto-archive

mode: auto

script: scripts/auto-archive-learning.ps1

args:

-Topic: "{topic}"

-SourcePath: "{output_dir}"

-AutoSync: true

reason: "学习成果自动归档到三个知识库"

步骤 2: 测试归档脚本


# 创建测试目录

mkdir -p ~/.agents/skills/self-learning/test-output

echo "# 测试学习成果" > ~/.agents/skills/self-learning/test-output/summary.md



# 运行归档脚本

cd ~/.agents/skills/self-learning/scripts

.\auto-archive-learning.ps1 -Topic "测试主题" -SourcePath "../test-output" -AutoSync



# 清理测试

Remove-Item -Recurse -Force ../test-output

⚠️ 常见问题

Q1: sentence-transformers 安装失败

错误: pip install sentence-transformers 失败

解决:


# 使用国内镜像

pip install sentence-transformers -i https://pypi.tuna.tsinghua.edu.cn/simple

Q2: 模型下载慢

问题: 模型文件大（~500MB），下载慢

解决:


# 手动下载模型

# 1. 访问 https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2

# 2. 下载文件到 ~/.cache/torch/sentence_transformers/

# 3. 重新运行脚本

Q3: Linting 报告为空

问题: 知识库质量很好，没问题

解决: 正常，说明知识库健康！

Q4: 归档脚本报错"路径不存在"

问题: OpcClaw 知识库路径不对

解决:


# 检查路径

Test-Path "$env:USERPROFILE\.OpcClaw 知识库"



# 如果不存在，修改脚本中的默认路径
