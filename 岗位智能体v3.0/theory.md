## 理论基础

### 1. 六维框架理论

六维框架（行业×企业×岗位×性格×任务×模型）源于系统论中的多维度分析思想。每个维度代表一个独立的变量空间，六维交叉形成高维度的岗位全景画像。

### 2. 五行人格×岗位适配

基于五行人格心理学OS的L7理论基石，五行能量类型与岗位类型存在天然的匹配关系：

- **木行人**（创新/生发）→ 需要创造力的岗位
- **火行人**（热情/照亮）→ 需要影响力的岗位
- **土行人**（承载/稳定）→ 需要执行力的岗位
- **金行人**（决断/规则）→ 需要判断力的岗位
- **水行人**（智慧/流动）→ 需要策略力的岗位

### 3. 五行阴阳面理论（v3.0新增）

> **核心命题**：每种五行都有阳面（显发优势）和阴面（过刚/过旺/过烈/过滞/过散的风险）。阴面不是缺陷，而是五行能量过度集中的必然结果。

**阴阳面与拔阴取阳**：
- 阳面 = 五行能量的健康显发 → 是优势，要巩固
- 阴面 = 五行能量的过度集中 → 是风险，要转化
- 拔阴取阳 = 从阴面中提取正向价值，通过"借力五行"实现转化
- **五行借力路径**：金过刚借水润泽/借火礼明/借土承载/借木生发

**数据驱动识别**：当用户无法自报五行类型时，从行为/数据特征自动推断：
- 目标导向+执行快+数据驱动 → 金行信号
- 热情高+感染力强+跟进弱 → 火行信号
- 创新力+前瞻性+落地弱 → 木行信号
- 稳健+流程强+变革慢 → 土行信号
- 灵活+策略强+定力弱 → 水行信号

### 4. 7S组织诊断理论（v3.0新增）

麦肯锡7S模型将组织分为7个相互依存的维度：战略(Strategy)、结构(Structure)、制度(Systems)、风格(Style)、人员(Staff)、技能(Skills)、共同价值观(Shared Values)。

**7S评分方法论**：
- 1-10分制，每分对应行为描述
- 评分+诊断依据+核心问题+改善方向四维诊断
- 7维加权平均（默认等权，可自定义）
- 可视化：8-10绿/5-7橙/1-4红

**7S与五行的映射**：
- S1战略 → 木行（生发、方向）
- S2结构 → 土行（承载、稳定）
- S3制度 → 金行（规则、标准）
- S4风格 → 火行（光明、感染）
- S5人员 → 水行（流动、润泽）
- S6技能 → 木行（成长、学习）
- S7共同价值观 → 土行（承载、凝聚）

### 5. 三层指标体系理论（v3.0新增）

**金字塔结构**：
- Layer 1 结果指标（考核用·绿色）：衡量最终经营成果
- Layer 2 过程指标（监控用·蓝色）：衡量执行过程质量
- Layer 3 风险指标（预警用·红色）：需要立即关注的系统性风险

**分层关联规则**：
- 过程指标连续3期低于阈值 → 升级为风险指标
- 风险指标P0 → 当周必须解决，纳入每日行动日历
- 风险指标P1 → 当月必须解决，纳入每周复盘

**P0/P1优先级体系**：
- P0 = 系统级风险，必须当周解决（如：客户链接机制断裂）
- P1 = 门店级风险，必须当月解决（如：某店转化率极低）

### 6. 复盘画布6步框架理论（v3.0新增）

复盘画布是一个6步深度分析框架，将"复盘"从"写工作总结"升级为"发现根因→设计转化→规划进化"：

1. **工作全量盘点**（含7S维度标签）：从7S视角审视工作
2. **五行优劣势剖析**（阳面+阴面+行为证据）：发现暗礁
3. **原因剖析**（根因链条，至少3层）：追问到可执行的根因
4. **拔阴取阳行动方案**（阴面→取阳→行动→借力五行）：五行转化
5. **下周风险预警**（4列：任务→风险→预警信号→预案）：前置预判
6. **进化路径**（3-4级跃迁）：长期成长

### 7. 龙爪断语理论（v3.0新增）

龙爪断语是AI从"工具"到"军师"的关键升级。它不是数据总结，而是高层面的判断和提炼：

- **3大核心判断**：从结果指标、过程/风险指标、7S/文化维度分别提炼
- **金句引用**：从数据中提炼可引用的警句，必须基于数据/行为证据
- **核心命题**：下周/下阶段的核心命题，必须指向行动

**断语与五色光思维的关系**：
- 核心判断1（结果指标）→ 白光思维（数据、事实）
- 核心判断2（过程/风险）→ 蓝光思维（困难、风险）
- 核心判断3（7S/文化）→ 黄光思维（价值、方向）+ 红光思维（直觉）

### 8. 龙爪模式（v2.0新增，v3.0升级）

将岗位智能体的输出对象从"人"扩展到"WorkBuddy AI Agent"。SKILL.md 是 WorkBuddy 认知系统的工作指令文件，包含 YAML frontmatter + 概念澄清 + What/Why/How 结构。

v3.0升级：从单报告到双报告（B1个人复盘画布/B2组织诊断报告），SOP从8步→10步。

### 9. HTML输出视觉设计规范理论（v3.0新增）

**设计哲学**：深色顶栏 + 白色画布 + 浅灰网格 = 专注内容的视觉层次

**颜色编码体系**（五色映射）：
- 绿色 = 达标/完成/优势 → 对应黄光思维（乐观、利益）
- 红色 = 预警/未达标/薄弱 → 对应蓝光思维（困难、风险）
- 橙色 = 关注/偏弱/进行中 → 对应红光思维（直觉、感受）
- 蓝色 = 过程/监控/信息 → 对应白光思维（数据、事实）
- 紫色 = 特殊/战略/文化 → 对应绿光思维（创新、变革）

## 跨域知识联系

### 五行借力路径 × 化克为生

五行借力路径是"化克为生"方法论的工程化实现：
- 金克木 → 金过刚借木生发（加追问、加深度）
- 火克金 → 金过刚借火礼明（加仪式、加情感设计）
- 水克火 → 火过烈借水润泽（加冷静、加反思）
- 木克土 → 土过滞借木生发（加创新、加突破）
- 土克水 → 水过散借土承载（加锚点、加底线）

### 7S × 三层指标体系的联系

7S诊断是"定基"（组织当前状态），三层指标是"定量"（衡量当前状态的标尺）：
- S3制度得分低 → 过程指标往往缺失
- S5人员得分低 → 结果指标往往不稳定
- S7共同价值观得分低 → 风险指标往往集中爆发

### 复盘画布 × 龙爪断语的联系

复盘画布是"结构化"（把散乱信息组织成6步闭环），龙爪断语是"升华"（从结构化信息中提炼高层判断）：
- 复盘画布Step B（优劣势剖析）→ 龙爪断语核心判断1
- 复盘画布Step E（风险预警）→ 龙爪断语核心判断2
- 复盘画布Step F（进化路径）→ 龙爪断语核心命题
