## 操作指南

### 7S评分实操要点

1. 评分必须基于**数据/行为证据**，不可凭印象
2. 诊断依据写法：先写事实，再写判断（"完成率93.9%~148.8%差异巨大→权责不清"）
3. 改善方向必须**具体可执行**（"明确三店店长权责边界"而非"改善管理"）
4. 综合评分=7维加权平均，默认等权，战略/文化维度可加权

### 五行阴阳面实操要点

1. 阳面写法：**特质+行为证据**（"目标导向极强：148.8%完成率"）
2. 阴面写法：**风险+行为证据**（"关系维护弱：0.14%入职转化率"）
3. 阴面警示话术必须**基于五行意象**（金=锋利、木=繁茂、火=炽烈、土=滞重、水=散漫）
4. 五行借力路径必须**指向具体行动**（"借水行润泽→加倾听、加柔性→每日1对1"）

### 复盘画布实操要点

1. 工作盘点每项**必须标注7S维度标签**
2. 根因链条至少追溯**3层**，直到可执行的根因
3. 拔阴取阳的"借力五行"列**不可为空**
4. 风险预警的"预警信号"**必须是可观测现象**（"11:30前等位超5桌"而非"客流可能超预期"）
5. 进化路径3-4级，每级有**核心关键词**

### 龙爪断语实操要点

1. 3大核心判断**分别从不同维度提炼**，不可重复
2. 金句引用**必须基于数据**，不可空泛
3. 核心命题**必须指向行动**，不可停留在认知层面
4. 一周一句话必须包含**五行意象+隐喻+借力+转化方向**

### HTML输出实操要点

1. CSS变量**必须遵循规范**，不可自定义替换
2. 颜色编码**不可混用**（红色=预警≠达标）
3. 进度条动画**必须使用setTimeout渐入**
4. 响应式断点**必须包含900px和600px两个断点**
