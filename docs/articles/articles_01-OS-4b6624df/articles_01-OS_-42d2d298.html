定期健康检查 - 龙心OS知识库维护指南

建议频率: 每周运行一次
目的: 保持知识库结构完整、链接有效、标签一致

📋 检查清单

1. 链接完整性检查

  • [ ] 双向链接有效性（[[文件名]]格式）
  • [ ] 外部链接可访问性
  • [ ] 图片和资源引用完整性
  • [ ] 知识库间交叉引用一致性
    • 检查方法: 
    1. 在Obsidian中打开"图谱视图"
2. 查看是否有孤立的节点（无连接的文件）
3. 检查红色链接（失效链接）
4. 使用搜索功能检查"[["匹配情况

    2. 标签一致性检查

  • [ ] 标签命名规范（统一使用小写）
  • [ ] 标签定义一致性
  • [ ] 无重复标签（如 #龙心OS 和 #龙心os）
  • [ ] 核心标签完整性
    • 核心标签列表: | 标签 | 用途 | |------|------| | #龙心OS | 龙心操作系统相关内容 | | #五行人格 | 五行人格心理学相关内容 | | #象思维 | 原创思维方法论 | | #企业文化 | 企业文化建设相关内容 | | #心文化 | 大圆满与修行体系 | | #知行合一 | 自我进化与经验沉淀 | | #索引 | 导航和索引页面 | | #待整理 | 需要进一步整理的内容 | | #已完成 | 已经完成的工作 |

    3. 文件命名规范检查

  • [ ] 使用数字前缀排序（00-, 01-, 02-）
  • [ ] 避免特殊字符（除-和_外）
  • [ ] 文件名与标题一致
  • [ ] 无重复文件
    • 命名规范: 
    ✅ 正确: 01-龙心OS核心系统.md
✅ 正确: 象思维-理论体系.md
❌ 错误: 龙心OS 核心系统.md（含空格）
❌ 错误: 01--系统.md（双横线）

    4. 元数据完整性检查

  • [ ] YAML frontmatter 格式正确
  • [ ] title 字段存在
  • [ ] created 日期格式正确（YYYY-MM-DD）
  • [ ] tags 字段存在
    • 标准模板: 
    ---
title: "文档标题"
created: "2026-04-11"
version: "1.0"
tags: [标签1, 标签2]

    5. 内存系统同步检查

  • [ ] WorkBuddy Brain 同步状态
  • [ ] Obsidian 知识库更新状态
  • [ ] IMA 笔记同步状态
  • [ ] MEMORY.md 更新状态
    • 三向同步检查: 
    Obsidian（主知识库）
    ↕️
WorkBuddy Brain（工作记忆）
    ↕️
IMA笔记（移动端）

    6. 知识库结构检查

  • [ ] 目录层级符合架构规范
  • [ ] 无空目录
  • [ ] 索引文件存在且最新
  • [ ] 核心文档无缺失
    • 标准架构: 
    AI OS知识库
├── 00-总索引与导航
├── 01-龙心OS核心系统
├── 02-专业领域知识库矩阵
├── 03-知识地基层
├── 04-实践案例库
└── 05-个人成长与修行

    🔧 自动化工具

    Obsidian 插件推荐

    | 插件 | 功能 | 用途 | |------|------|------| | Linter | 自动格式化Markdown | 统一格式规范 | | Tag Wrangler | 标签管理 | 合并重复标签 | | Broken Links | 失效链接检测 | 链接完整性 | | Dataview | 数据查询 | 生成索引和报告 |

    手动检查脚本

    PowerShell 检查脚本（保存为 health_check.ps1）: 
    检查重复标签
    
Get-ChildItem -Recurse -Filter "*.md" | 
    Select-String -Pattern "^tags:" -Context 0,1 |
    Export-Csv "tags_report.csv"


    检查失效链接
    
Get-ChildItem -Recurse -Filter "*.md" |
    Select-String -Pattern "\[\[([^\]]+)\]\]" |
    Export-Csv "links_report.csv"

    🚨 常见问题处理

    问题1: 大量失效链接

    原因: 文件重命名或移动后未更新链接 解决: 1. 使用Obsidian的"重命名并更新链接"功能 2. 或使用全局替换更新链接

    问题2: 标签不一致

    原因: 手动输入时大小写或拼写不一致 解决: 1. 使用 Tag Wrangler 插件合并标签 2. 建立标签使用规范文档

    问题3: 同步失败

    原因: 网络问题或文件冲突 解决: 1. 检查网络连接 2. 手动解决文件冲突 3. 重新触发同步
