    二、核心方法论

    2.1 福格行为模型基础

    核心公式： ``` B = M × A × P 行为 = 动机 × 能力 × 提示 ``` 三要素详解：

    | 要素 | 定义 | 金行人特点 | |------|------|-----------| | 动机（M） | 行为的驱动力引擎 | 对秩序、原则、卓越的追求是核心动机源 | | 能力（A） | 执行行为的容易程度 | 逻辑行为易，情感行为难 | | 提示（P） | 行为的触发开关 | 对规则性提示高度敏感 |

    2.2 金行人五维度特质映射

    | 维度 | 阳面（目标） | 阴面（问题） | 对应B=MAP | |------|-------------|-------------|----------| | 坚固 | 刚毅坚强 | 好胜计较 | M=证明自己强 | | 收敛 | 含蓄持重 | 小气狭隘 | A=分享能力低 | | 锋利 | 公正锋利 | 严苛无情 | A=共情能力低 | | 光洁 | 智慧权威 | 虚荣自负 | M=外在认可 | | 沉降 | 沉稳冷静 | 焦虑多疑 | A=决策能力低 |

    微习惯方案库：

    五、场景应用模板

    场景模板一：决策犹豫（沉降维度）

    ``` 【场景】克服"决策犹豫与过度准备"

    愿望：提升决策效率，减少犹豫时间 黄金行为：设定"信息收集截止点"和"最小可行决策"标准

    微习惯配方：

  • 锚点：重复查找资料超过30分钟
  • 行为：启动"15分钟决策计时器"
  • 庆祝：记录"高效决策完成"

    • M动机：连接"时间成本也是成本" A能力：15分钟限制+1个优势/风险简化框架 P提示：内部提示（意识到重复查找）+计时器提醒 ```

    场景模板二：团队沟通（锋利维度）

    ``` 【场景】改善"团队沟通中的严厉感"

    愿望：反馈精准但更容易被接受 黄金行为：增加"共情启动"步骤

    微习惯配方：

  • 锚点：需要口头指出问题时
  • 行为：先说一句共情启动语
  • 庆祝：观察对方反应

    • M动机：连接"先肯定让批评更有效" A能力：简单开场白公式 P提示：内部提示（组织批评语言第一瞬间） ```

    场景模板三：知识分享（收敛维度）

    ``` 【场景】应对"不愿分享知识"

    愿望：建立个人品牌同时促进团队成长 黄金行为：系统化分享非核心知识

    微习惯配方：

  • 锚点：每周五下午2点
  • 行为：30分钟整理一个Q&A条目
  • 庆祝：记录"资产增加"

    • M动机：连接"一次整理多次复用" A能力：30分钟时间盒，目标明确 P提示：日历重复事件 ```

    六、诊断工具

    6.1 B=MAP快速诊断问题

    当期望行为未发生时，使用以下诊断框架：

    ``` 【问题诊断清单】

    问题描述：[期望行为是什么？]

    □ 动机诊断（M） - 忘了价值？→ 连接阳面动机 - 被情绪覆盖？→ 转化阴面情绪 □ 能力诊断（A） - 场景太复杂？→ 简化行为 - 不知道怎么做？→ 提供技能培训 □ 提示诊断（P） - 锚点被错过？→ 优化锚点选择 - 内部信号没识别？→ 训练自我觉察 ```

    6.2 五维度自测表

    ``` 【金行人五维度行为自测】

    □ 坚固维度 - 我在讨论中是否经常想证明自己是对的？ - 我能否在分歧中寻找共赢点？ □ 收敛维度 - 我是否不愿意分享自己的核心知识？ - 分享是否让我感到"被超越"的焦虑？ □ 锋利维度 - 我的反馈是否经常让人感到"受伤"？ - 我能否在批评前先肯定对方？ □ 光洁维度 - 我是否过度追求外在认可？ - 我能否坦然承认自己的不足？ □ 沉降维度 - 我是否经常陷入过度担忧？ - 我能否在信息不全时做出决策？ ```
