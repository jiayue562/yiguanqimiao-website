6.1 拔阴取阳的技术原理

口诀一：阴水烦转阳水辩（心界转化） 适用于第4-6级，内心充满焦虑、烦躁与多疑时：

  1. 觉察：当情绪升起，立刻觉察"这是阴水的烦乱"
  2. 暂停：停止情绪化的反应，深呼吸，让能量下沉
  3. 转辩：启动理智，问自己"事实是什么？目标是什么？有哪些方案？"
  4. 行动：基于辨析结果，采取明确的行动

口诀二：阴水痴转阳水智（灵界转化） 适用于第5-7级，陷入依赖、妄想或怨恨时：

  1. 反思：反思自己的价值观是否扭曲，是否将自我价值寄托于他人
  2. 明理：学习经典智慧，理解"本自具足"的道理
  3. 发愿：确立"滋养万物、成就智慧"的高层使命

道藏视角下的层级跃迁，核心在于"水火既济"
