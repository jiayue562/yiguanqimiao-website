    │
    └─ 内在成长（安静内向/疏远冷淡）── 自我独处，空间需求

火行人维度关联

炎上（推动提升/急躁争斗）
    │
    ├─ 明亮（明辨守礼/炫耀虚荣）── 社交互动，形象管理
    │
    ├─ 炽烈（热情自信/自卑夸张）── 自我认知，情感特质
    │
    ├─ 发散（主动影响/情绪多变）── 人际影响，支配能力
    │
    └─ 迅疾（气势夺人/无序冲动）── 目标执行，行为风格

土行人维度关联

承载（信实可靠/呆板无趣）
    │
    ├─ 适应（耐力持久/迟钝缓慢）── 环境适应，应对变化
    │
    ├─ 容纳（宽宏厚道/沉闷埋怨）── 情感包容，人际接纳
    │
    ├─ 运化（支持担当/固执不化）── 责任担当，团队支持
    │
    └─ 稳定（镇定稳重/拖拉磨蹭）── 自我调节，情绪稳定

金行人维度关联
