四层学习能力架构详解

第一层：持久记忆系统

核心价值

让AI从"每次从零开始"变成"越用越聪明"

技术实现

1.1 MEMORY.md 工作记忆

  • 路径：c:\Users\jia'yue\WorkBuddy\20260328122151\.workbuddy\memory\MEMORY.md
  • 功能：长期知识存储、用户偏好记录、核心洞察沉淀
  • 格式：Markdown + YAML frontmatter
  • 更新：每次重要对话结束自动更新

1.2 每日日志

  • 路径：c:\Users\jia'yue\WorkBuddy\20260328122151\.workbuddy\memory\YYYY-MM-DD.md
  • 功能：每日工作记录、学习收获、任务追踪
  • 格式：Markdown时间线
  • 归档：每月整合到MEMORY.md

1.3 IMA笔记同步

  • API：https://ima.qq.com/agent-interface
  • 功能：移动端快速记录、跨平台同步
  • 场景：外出时快速记录、灵感捕捉

1.4 Obsidian知识库

  • 路径：D:\以观其妙书院知识库\观其妙书院\
  • 功能：结构化知识沉淀、双向链接、知识图谱
  • 同步：定期备份+手动触发

记忆调用流程

用户请求 → 检查MEMORY.md → 检查当日日志 →
如有相关信息 → 调取并应用 →
如无相关信息 → 从其他来源获取 →
获取后 → 写入MEMORY.md（重要信息）

第二层：实时知识获取

核心价值

不依赖训练数据，实时获取最新知识

技术实现

2.1 网络搜索

  • 技能：web_search_plus / baidu_search
  • 功能：实时信息、新闻事件、最新数据
  • 输出：结论优先，数据来源后置

2.2 公众号检索

  • 技能：wechat-article-search（如已安装）
  • 功能：高质量深度文章、行业洞察
  • 场景：专业领域研究、趋势分析

2.3 内容摘要

  • 技能：summarize / 内容总结
  • 功能：长文章/PDF/网页/YouTube摘要
  • 场景：快速了解、深度阅读前筛选

搜索调用流程

用户请求 → 意图识别 → 选择搜索渠道 →
执行搜索 → 提炼核心 → 格式化输出 →
可选：写入MEMORY.md

第三层：多格式文件读取

核心价值

任何资料都能读懂吸收

技术实现

3.1 文本文件

  • 格式：txt / md / 代码文件
  • 技能：read_file（原生支持）
  • 处理：直接读取+结构化提炼

3.2 PDF文档

  • 技能：markitdown / pdf
  • 功能：PDF → Markdown
  • 处理：提取文本+结构化

3.3 Word文档

  • 技能：markitdown / docx
  • 功能：DOCX → Markdown
  • 处理：提取文本+表格+图片描述

3.4 PPT演示

  • 技能：markitdown / pptx
  • 功能：PPTX → Markdown
  • 处理：提取幻灯片+图表

3.5 图片/音频

  • 技能：read_media_file（图片）/ whisper（音频）
  • 功能：图片内容理解、音频转文字
  • 处理：OCR识别/转录+理解

文件读取流程

用户请求 → 识别文件格式 →
选择对应技能 → 提取内容 →
结构化提炼 → 核心结论+行动建议 →
可选：写入MEMORY.md

第四层：自主进化机制

核心价值

不需要用户操心，系统自动迭代升级

技术实现

4.1 每日知识沉淀

  • 触发：每次重要对话结束
  • 流程：
    1. 回溯对话内容
    2. 提取新知识/新洞察
    3. 识别被验证/修正的偏好
    4. 写入MEMORY.md
  • 输出：标准化沉淀卡

4.2 记忆蒸馏

  • 触发：用户手动或每月一次
  • 流程：
    1. 读取完整MEMORY.md
    2. 识别重复/过时内容
    3. 压缩为精炼版本
    4. 保留核心信息
  • 输出：精炼版MEMORY.md

4.3 定时自动化任务

  • 技能：automation_update
  • 可创建任务：
    • 每周知识蒸馏（周一9:00）
    • 每月记忆归档（每月1日）
    • 每日23:30自动备份
  • 配置：iCalendar RRULE格式

进化流程

日常对话 → 知识沉淀 → MEMORY.md积累 →
定期蒸馏 → 精炼版本 → 知识库归档 →
系统升级 → 更好服务 → 循环往复
