    🔄 二、文件夹使用规范

    2.1 命名规范

    1. 前缀编号: 使用两位数字前缀（00-99）保持顺序 2. 中文名称: 使用简洁明了的中文名称 3. 英文别名: 可添加括号标注英文别名 4. 特殊符号: 必要时使用emoji增强识别度

    2.2 权限规范

    1. 可读权限: 所有文件夹对所有用户可读 2. 写入权限: 按职责分配写入权限 3. 修改权限: 重要文件夹需审核后修改 4. 删除权限: 仅系统管理员有删除权限

    2.3 维护规范

    1. 定期整理: 每月整理一次文件夹结构 2. 内容审核: 每季度审核文件夹内容质量 3. 结构优化: 每半年评估和优化结构 4. 备份验证: 每月验证备份完整性
