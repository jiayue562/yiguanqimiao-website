五、如何安装AI OS？

5.1 系统要求

  • Windows 10/11 或 macOS
  • 安装 [WorkBuddy](https://www.codebuddy.cn/)（VS Code插件）
  • 安装 [Obsidian](https://obsidian.md/)（知识库管理）
  • 可选：IMA笔记（移动端同步）

5.2 安装步骤

Step 1：安装WorkBuddy

  1. 打开VS Code
  2. 扩展商店搜索"WorkBuddy"
  3. 安装并登录

Step 2：配置AI OS

  1. 在WorkBuddy中，进入设置
  2. 配置Skills路径：~/.workbuddy/skills/
  3. 配置Rules路径：~/.workbuddy/rules/

Step 3：安装核心Skills

# 五大引擎
~/.workbuddy/skills/龙心OS/
~/.workbuddy/skills/象思维/
~/.workbuddy/skills/知识学习Skills/
~/.workbuddy/skills/五色光思维/
~/.workbuddy/skills/人机协同五象限/
~/.workbuddy/skills/知行合一/

# 专业系统
~/.workbuddy/skills/五行人格心理学/
~/.workbuddy/skills/企业文化/
~/.workbuddy/skills/思维模型库-思考的框架/

Step 4：配置Obsidian

  1. 创建知识库文件夹
  2. 配置双向链接
  3. 安装核心插件（Graph View、Daily Notes等）

Step 5：启动AI OS 在WorkBuddy中输入：

全系统启动

看到"🐉 龙心OS 全系统启动"，恭喜你，AI OS已就绪！
