二、五饕餮结构

2.1 五饕餮分布

``` 白饕餮（客户·中心） ↓ 蓝饕餮（健康）→ 黄饕餮（财富）→ 红饕餮（关系）→ 绿饕餮（事业） ```

2.2 五饕餮含义

| 饕餮 | 颜色 | 含义 | 内容 | |------|------|------|------| | 白饕餮 | 白 | 客户 | 整个白色底盘，代表客户 | | 蓝饕餮 | 蓝 | 健康 | 第一自己要健康，第二做健康产业 | | 黄饕餮 | 黄 | 财富 | 保证基本物质需求同时要发展 | | 红饕餮 | 红 | 关系 | 有钱不一定关系好，人是情感动物 | | 绿饕餮 | 绿 | 事业 | 把一个平台做无限大，包括有形+无形世界 |

2.3 解读方法

> 解读文化罗盘时回归本质：阿，由中心向宇宙扩散，再收回来。

    四、统一思维五大模式

    4.1 五大模式

    | 模式 | 核心定义 | 应用场景 | |------|---------|---------| | 逆向思维 | 结果导向，以终为始 | 目标设定 | | 同步最大思维 | 固有发展+新增发展+准备发展同步进行 | 多任务管理 | | 坐庄思维 | 谋求第一必坐庄，坐庄等于零风险 | 战略定位 | | 标杆思维 | 有标杆学标杆，没标杆做标杆 | 学习成长 | | 象思维 | 回归本质角度解决全局思维，提高效率 | 决策分析 |

    4.2 思维详解

    逆向思维：
  • 先定目标
  • 倒推路径
  • 以终为始
    • 同步最大思维：
  • 保持固有发展
  • 新增发展同步进行
  • 准备发展同时推进
  • 众多事情、环节同步进行
    • 坐庄思维：
  • 谋求第一
  • 坐庄=零风险
  • 建立壁垒
    • 标杆思维：
  • 外部有标杆 → 学习标杆
  • 外部无标杆 → 自己成为标杆
    • 象思维：
  • 回归本质
  • 解决全局问题
  • 分歧时提高效率

    五、统一工具九大矩阵

    5.1 工具列表

    | 工具 | 功能 | 应用场景 | |------|------|---------| | 5W1H | 为什么做事的动机 | 任务启动 | | PDCA | 闭环管理 | 持续改进 | | SMART | 可衡量的标准 | 目标设定 | | AOP | 年度经营计划 | 战略规划 | | SWOT | 分析可行性（内部+外部） | 项目评估 | | SOP | 产品标准 | 运营规范 | | SOC | 流程标准 | 运营规范 | | MOT | 触达的关键节点 | 客户体验 | | 5S | 环境安全效率 | 现场管理 | | PEST | 外部市场环境分析 | 市场洞察 |

    5.2 工具详解

    5W1H：
  • What（做什么）
  • Why（为什么）
  • Who（谁来做）
  • When（何时做）
  • Where（在哪里做）
  • How（怎么做）
    • PDCA：
  • Plan（计划）
  • Do（执行）
  • Check（检查）
  • Act（行动）
    • SMART：
  • Specific（具体）
  • Measurable（可衡量）
  • Achievable（可达成）
  • Relevant（相关）
  • Time-bound（有时限）
