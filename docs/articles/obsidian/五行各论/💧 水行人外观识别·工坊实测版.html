三、气质特征（心界·意象层）

3.1 整体气质

核心词：柔和、亲近、受柔润

| 状态 | 表现 | |------|------| | 阳水气质 | 温和智慧，包容亲和，流动灵活 | | 阴水气质 | 柔弱涣散，情绪化，爱哭，反应慢 |

3.2 声音特征

| 特征 | 工坊实测数据 | |------|------------| | 声音 | 语速适中，没有鼻音，没有尖的大众声音，不是特别洪亮，柔和 | | 语速 | 较慢，缓和 | | 语调 | 温柔，没有刺激性 | | 常用词汇 | "差不多，可能，好像，（犹豫）还行，还可以" | | 表达方式 | 犹豫，不肯定，留有余地 |

> 深度分析：水行人用词特征——"差不多""可能""好像""还行"——这是水行人智慧的外化：不轻易下定论，保持开放性，也是水行人"不确定性思维"的语言表达。

3.3 动态特征

| 动态维度 | 特征 | |---------|------| | 整体动态 | 多数有捋头发、扶眼镜等小动作 | | 情绪反应 | 爱哭，反应慢 | | 感官 | 不是反差特别大，爱吃 |

> 关键发现："捋头发、扶眼镜等小动作"——水行人的小动作是内心活动的外漏，也是识别水行人的特征之一。
