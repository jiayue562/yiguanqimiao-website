三、气质特征（心界·意象层）

3.1 整体气质

核心词：热情（或被动热情）

| 状态 | 表现 | |------|------| | 阳火气质 | 主动热情，明亮开朗，感染力强 | | 阴火气质 | 被动热情（需要点燃），或急躁易怒 |

> 关键洞察："热情或被动热情"——火行人的热情有两种形式：有时主动点燃，有时需要被激发。这是识别时的重要细节。

3.2 声音特征

| 特征 | 工坊实测数据 | |------|------------| | 声音 | 高亢，浑厚，有穿透力，热烈 | | 语速 | 快，有条理 | | 语调 | 热烈，有节奏感 | | 常用词汇 | "嚯"（感叹词，表示惊讶或兴奋） | | 表达方式 | 语速快，直率 |

3.3 动态特征

| 动态维度 | 特征 | |---------|------| | 整体动态 | 行动迅速，有气势 | | 情绪反应 | 急（情绪反应速度快）| | 体感 | 身体发热感 | | 感官 | 对甜味敏感，对夏季有感 |
