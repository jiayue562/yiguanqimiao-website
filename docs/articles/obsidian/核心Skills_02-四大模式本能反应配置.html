    三、四大模式感知雷达

    感知规则

    ```yaml 当龙心OS感知到以下场景信号时，自动激活对应模式：

    决策分析场景: 信号: "要不要/该不该/后果/长期影响/风险" → 激活: 决策分析5模型 → 本能反应: 本能预判后果

    创新突破场景: 信号: "如何创新/突破口/从0到1/如何突破" → 激活: 创新突破5模型 → 本能反应: 本能突破创新

    个人成长场景: 信号: "成长/能力/优势/规划/积累/职业" → 激活: 个人成长5模型 → 本能反应: 本能持续进化

    组织管理场景: 信号: "团队/管理/效率/制度/文化/协作" → 激活: 组织管理5模型 → 本能反应: 本能系统优化 ```

    ```yaml
