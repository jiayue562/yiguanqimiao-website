🌟 核心定位

北斗七星的宇宙身份

北斗七星是药师七佛在天空中的示现，北斗七星君即为药师七佛。

在宇宙体制中的角色

  • 北斗居天之中，为天之枢纽，斡运四时（春夏秋冬）
  • 从天子皇帝到黎庶百姓的生死祸福、寿禄贫富皆受北斗所管辖
  • 是法界体制中执行因果的中枢力量

⭐ 七星详细档案

第一星：贪狼星

  • 光色：红光
  • 主功能：人丁兴旺、旺运、发财
  • 适合祈祷：求子、求旺运、求财富增长

第二星：巨门星

  • 光色：白光
  • 主功能：擅长治愈疾病，恢复健康
  • 适合祈祷：身体健康、疾病痊愈

第三星：禄存星

  • 光色：绿光
  • 主功能：顺利事业，增长财富，转危为安
  • 适合祈祷：事业顺利、化险为夷

第四星：文曲星

  • 光色：绿光
  • 主功能：催生学业进步，榜上有名，激发文艺才能
  • 适合祈祷：考试、学业、文艺创作

第五星：廉贞星

  • 光色：黄光
  • 主功能：消解厄运，有助化煞
  • 适合祈祷：化解厄运、消煞

第六星：武曲星

  • 光色：紫光
  • 主功能：情感畅通，延年益寿，官场亨通，司掌财富
  • 适合祈祷：情感、寿命、官运、财富综合

第七星：破军星

  • 光色：蓝光
  • 主功能：祥瑞，促使修行增进，遣除障道违缘
  • 适合祈祷：修行进步、清除修行障碍

🔭 北斗的科学与灵性真相

灵性角度

第一星（贪狼） → 红色频率 → 生命力/人丁

第三星（禄存） → 绿色频率 → 成长/事业
第四星（文曲） → 绿色频率 → 智慧/文艺
第五星（廉贞） → 黄色频率 → 中和/化解
第六星（武曲） → 紫色频率 → 力量/财富
第七星（破军） → 蓝色频率 → 净化/修行
