║ │   · 望诊定位：三瘦/三尖/三厚/三方/三肥                      ║

║ │   · 木行：生长/觉照/创新/固执                               ║

║ │   · 土行：承载/稳定/迟钝/务实                               ║
║ │   · 金行：决断/清明/刻薄/变革                               ║
║ │   · 水行：流动/变通/消极/智慧                               ║
║ └─ 2.3 五行确认：确定主导五行                                ║
║                                                               ║
║ 【阶段三：三界分析】（红光·直觉洞察）                        ║
║ ├─ 3.1 灵界诊断：价值观/信念/使命                           ║
║ │   · 阳面：仁/礼/信/义/智                                   ║
║ │   · 阴面：慢/嗔/疑/贪/痴                                   ║
║ ├─ 3.2 心界诊断：情绪模式/思维习惯                          ║
║ │   · 阳面：喜/乐/和/悦/通                                   ║
║ │   · 阴面：怒/恨/怨/恼/烦                                   ║
║ ├─ 3.3 身界诊断：行为模式/习惯偏好                          ║
║ │   · 阳面：温/良/恭/俭/让                                   ║
║ │   · 阴面：杀/色/气/财/酒                                   ║
║ └─ 3.4 三界协同评估：是否平衡                               ║
║                                                               ║
║ 【阶段四：九层评估】（黄光·价值判断）                        ║
║ ├─ 4.1 层级定位：1-9级中的位置                              ║
║ │   · 健康区间（1-3级）：阳面主导                            ║
║ │   · 一般区间（4-6级）：阴阳交战                            ║
║ │   · 不健康区间（7-9级）：阴面主导                          ║
║ ├─ 4.2 失衡原因分析：                                       ║
║ │   · 内在因素：三界哪个先失衡                               ║
║ │   · 外在因素：环境/关系/压力影响                           ║
║ │   · 生克因素：哪对相克关系被触发                           ║
║ └─ 4.3 转化方向：指向更高层级                               ║
║                                                               ║
║ 【阶段五：输出报告】（绿光·创新方案）                         ║
║ ├─ 5.1 核心结论：五行归类+三界状态+九层定位                  ║
║ ├─ 5.2 转化建议：                                           ║
║ │   · 拔阴取阳路径（针对主导阴面特质）                       ║
║ │   · 化克为生方案（针对失衡相克关系）                       ║
║ │   · 三界转化顺序（从哪个界面开始）                         ║
║ ├─ 5.3 关系建议：                                           ║
║ │   · 与他人的五行配伍                                      ║
║ │   · 能量互补策略                                          ║
║ │   · 潜在冲突预警                                          ║
║ └─ 5.4 持续观察点：                                         ║
║     · 触发阴面的典型场景                                    ║
║     · 回归阳面的关键因素                                    ║
