ppt-generator - PPT智能生成 Skills

功能定位

L6层·专业场景工具 - 将主题自动转化为结构化PPT初稿，与龙心OS的教员方法论深度整合。

核心能力

1. 主题结构化 - 分析主题核心内容 - 生成逻辑清晰的章节结构 - 提炼关键要点

2. PPT初稿生成 - 输出Markdown格式的PPT结构 - 包含标题、要点、备注 - 支持多种排版风格

3. 五色光整合 - 白光：客观事实呈现 - 黄光：价值主张提炼 - 绿光：创新观点激发 - 蓝光：风险提示补充

使用场景

  • 培训课程PPT制作
  • 商业提案演示
  • 学术报告结构化
  • 产品介绍梳理

    • 输出格式

    ```markdown

    PPT标题

    第一章：核心主题

  • 要点1
  • 要点2

    • 详细内容

  • 子要点A
  • 子要点B

    • 第二章：应用实践

    ... ```
