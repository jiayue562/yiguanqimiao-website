一、第一次团队能量成像

1. 采用角色扮演进行能量成像

  • 团队负责人组织团队成员进行角色扮演
  • 每个成员扮演一个五行要素（木/火/土/金/水）
  • 通过身体姿态、位置、朝向来表现五行之间的关系

2. 企业（团队）负责人画出组织结构

  • 负责人画出当前的组织结构图
  • 明确每个成员在组织结构中的位置
  • 标注每个成员的五行类型

3. 集体找出影响企业（团队）内外重要因素（因素多少根据人数来定）

  • 集体讨论，找出影响团队的内外因素
  • 因素数量根据团队成员人数来确定
  • 每个因素对应一个五行要素

4. 企业（团队）负责人给团队成员分配要素（每个人只代表一个因素）

  • 负责人根据每个成员的五行类型，分配对应的要素
  • 每个人只代表一个因素，专注体验
  • 分配结果要获得全体成员认同

5. 企业（团队）负责人采用人与人之间距离、朝向、高低、姿态来表现各个因素之间的关系

  • 距离：表现因素之间的亲疏关系（近=相生/远=相克）
  • 朝向：表现因素之间的影响方向（面向=主动影响/背对=被动影响）
  • 高低：表现因素之间的主次关系（高=主导/低=从属）
  • 姿态：表现因素之间的互动状态（开放=流通/封闭=阻塞）

6. 能量成像后，各自表达自己的当下的感受包括身体与心理两方面（表达流畅，先说自己是什么行人，自己代表什么要素，自己当下的感受）

  • 表达内容：
    • 自己是什么行人（木/火/土/金/水）
    • 自己代表什么要素
    • 自己当下的身体感受（紧张/放松/舒适/不适）
    • 自己当下的心理感受（愉快/焦虑/愤怒/平静）
  • 表达要求：
    • 表达流畅，不中断
    • 先说自己是什么行人，自己代表什么要素，自己当下的感受
    • 不批评、不评判、只表达感受

二、第二次团队能量成像

1. 采用角色扮演进行能量成像

  • 团队负责人组织团队成员进行第二次角色扮演
  • 组织结构、要素及相关人员与第一次团队能量成像一致
  • 允许成员根据第一次的感受进行调整

2. 组织结构、要素及相关人员与第一次团队能量成像一致

  • 保持组织结构不变
  • 保持要素分配不变
  • 保持人员分配不变
  • 允许调整距离、朝向、高低、姿态

3. 企业（团队）负责人及团队成员采用人与人之间距离、朝向、高低、姿态来表现各个因素之间的关系。依据自己团队能量流畅的位置和姿势，进行自由调整，直到所有成员满意

  • 自由调整：
    • 成员可以根据第一次的感受，调整自己的位置、朝向、高低、姿态
    • 目标是让团队能量流动更流畅
    • 调整过程要所有人满意
  • 调整依据：
    • 自己团队能量流畅的位置和姿势
    • 与第一次能量成像感受对比
    • 找到更舒适、更流畅的能量状态

4. 能量成像后，各自表达自己的当下的感受包括身体与心理两方面（表达流畅，先说自己是什么行人，自己代表什么要素，自己当下的感受，与第一次能量成像感受有什么不同，给你什么启示）

  • 表达内容：
    • 自己是什么行人（木/火/土/金/水）
    • 自己代表什么要素
    • 自己当下的身体感受（紧张/放松/舒适/不适）
    • 自己当下的心理感受（愉快/焦虑/愤怒/平静）
    • 与第一次能量成像感受有什么不同
    • 给你什么启示
  • 表达要求：
    • 表达流畅，不中断
    • 先说自己是什么行人，自己代表什么要素，自己当下的感受
    • 与第一次能量成像感受对比
    • 不批评、不评判、只表达感受

三、第三次团队能量成像

1. 采用角色扮演进行能量成像

  • 团队负责人组织团队成员进行第三次角色扮演
  • 跟进所学的五行识人团队建设知识，进行团队五行能量重塑
  • 目标是实现团队五行的平衡与流通

2. 跟进所学的五行识人团队建设知识，进行团队五行能量重塑

  • 五行能量重塑原则：
    • 五行相生：木生火 → 火生土 → 土生金 → 金生水 → 水复生木（螺旋上升）
    • 五行相克：木克土 → 土克水 → 水克火 → 火克金 → 金复克木（制约平衡）
  • 能量重塑方法：
    • 调整距离：让相生的要素更近，相克的要素更远
    • 调整朝向：让相生的要素面向彼此，相克的要素避免直接面对
    • 调整高低：让主导要素更高，从属要素更低
    • 调整姿态：让所有要素都开放，避免封闭

3. 企业（团队）负责人画出组织结构

  • 负责人画出重塑后的组织结构图
  • 明确每个成员在重塑后组织结构中的位置
  • 标注每个成员的五行类型及能量状态

4. 集体找出影响企业（团队）内外重要因素（因素多少根据人数来定）

  • 集体讨论，找出影响团队的内外因素
  • 因素数量根据团队成员人数来确定
  • 每个因素对应一个五行要素
  • 与第一次对比，看因素是否发生变化

5. 企业（团队）负责人给团队成员分配要素（每个人只代表一个因素）

  • 负责人根据每个成员的五行类型，分配对应的要素
  • 每个人只代表一个因素，专注体验
  • 分配结果要获得全体成员认同
  • 与第一次对比，看分配是否发生变化

6. 企业（团队）负责人采用人与人之间距离、朝向、高低、姿态来表现各个因素之间的关系

  • 距离：表现因素之间的亲疏关系（近=相生/远=相克）
  • 朝向：表现因素之间的影响方向（面向=主动影响/背对=被动影响）
  • 高低：表现因素之间的主次关系（高=主导/低=从属）
  • 姿态：表现因素之间的互动状态（开放=流通/封闭=阻塞）
  • 与第一次对比：看距离、朝向、高低、姿态是否发生变化
