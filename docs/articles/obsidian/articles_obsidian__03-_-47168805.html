第二章：我们应当倡导的文化和反对的文化

---

一、四大执行文化

1.1 执行文化一：反对依赖文化，树立商业人格

依赖文化表现：出现问题找理由、出事怨别人 商业人格五要素： 1. 独立人格：不依赖上级、同事 2. 原则意识：信守承诺、坚持原则 3. 契约精神：拿人钱财替人消灾 4. 交换意识：价值交换 5. 结果导向：拿结果换报酬

1.2 执行文化二：反对含糊文化，树立结果导向

含糊文化表现：多做少做一样、高低都一样、差不多就行 结果导向四要素： 1. 量化：能量化的必须量化 2. 承诺：承诺的事情必须做到 3. 极致：做就做到最好 4. 检验：用结果说话

1.3 执行文化三：反对面子文化，树立客户价值

面子文化表现：以自我为中心、以本部门为中心、忽视客户 客户价值三层次： 1. 敬畏客户 2. 尊重客户 3. 成就客户

1.4 执行文化四：反对封闭文化，树立开放分享

封闭文化表现：各自为政、横向交流少、背后谈论多、不阳光 开放分享四要素： 1. 成果共享 2. 思想共享 3. 促进协作 4. 阳光坦诚

---

二、三大职业文化

2.1 职业文化一：反对业余文化，树立专业文化

专业文化三要素： 1. 专业技能：吃饭的本领要够硬 2. 持续学习：不断提升专业水平 3. 追求卓越：不满足于现有水平

2.2 职业文化二：反对江湖文化，树立商业文化

商业文化四要素： 1. 契约精神：按规矩办事 2. 诚信经营：说到做到 3. 公平竞争：不搞小动作 4. 规范流程：按制度执行

2.3 职业文化三：反对平庸文化，树立敬业文化

敬业文化四要素： 1. 忘我工作：全身心投入 2. 业绩非凡：追求卓越成果 3. 乐观向上：积极面对困难 4. 持之以恒：长期坚持

四、真正的企业文化四个标准

> 1. 一切与商业原则、市场规律无关的文化，都不是真正的企业文化 > 2. 一切与团队成长无关的文化，都不是真正的企业文化 > 3. 一切与客户价值无关的文化，都不是真正的企业文化 > 4. 一切与社会责任无关的文化，都不是真正的企业文化

  • 敬畏文化：敬畏规律、敬畏客户、敬畏制度
  • 超越文化：超越自我
