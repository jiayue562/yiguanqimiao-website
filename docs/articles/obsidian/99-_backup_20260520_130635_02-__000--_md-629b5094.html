3.3 拔阴取阳四步法

第一步：认不是（养水）→ 智慧
第二步：找好处（养金）→ 义气
第三步：信因果（养土）→ 信实
第四步：达天时（养火）→ 明理
第五步：仁德生（养木）→ 圆满

四、关键金句摘录

核心使命：通过拔阴取阳，使每个人的木火土金水都成为阳的能量。

生命模型：精气神三层结构（身、心、灵）是判断阴阳状态的核心框架。

转运原理：一步转运、二步改命——转运是后天调整，改命是先天转化。

五脏关系：

  • 怒伤肝（木）
  • 恨伤心（火）
  • 怨伤脾（土）
  • 恼伤肺（金）
  • 烦伤肾（水）

化法总纲：

  • 化阴木 → 阳木 → 忍辱
  • 化阴火 → 阳火 → 辞让
  • 化阴土 → 阳土 → 不怨
  • 化阴金 → 阳金 → 找好处
  • 化阴水 → 阳水 → 认不是

入门路径（新手推荐）

  1. 01-阴阳五行基础 → 理解基本概念
  2. 03-五行人通用框架 → 掌握分类方法
  3. 09-拔阴取阳法 → 学会实践方法
  4. 12-拔阴自查表 → 开始自我觉察

进阶路径（深入研究）

  1. 阅读自己主五行对应专题（04-08）
  2. 学习五脏病理关系（11-五行病理与化解）
  3. 实践四步转化法（10-阴阳转化原理）
  4. 探索五行相生相克深层规律

高阶路径（成为专家）

  1. 研究三层结构的相互转化机制
  2. 探索先天命运与后天修为的平衡
  3. 实践"超出三界外，不在五行中"的境界
  4. 传播五行人格心理学，利益更多人
