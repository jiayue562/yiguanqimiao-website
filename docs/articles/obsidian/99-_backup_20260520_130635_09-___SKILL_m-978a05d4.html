Step 1: 感知层（自动，无需用户说关键词）

  • 关键词提取：从输入中提取关键信息
  • 情感识别：判断用户的情感状态（焦虑/愤怒/抑郁/恐惧/迷茫等）
  • 身份判断：推断用户的主导五行（根据历史对话或当前输入特征）
  • 意图强度：判断用户是随便问问还是认真需要分析

Step 2: 场景归类

  • 匹配到 S0-S7 场景
  • 评估复杂度（低/中/高）
  • 确定需要启动的引擎组合

Step 3: 七模块协同路由

  • 根据场景→七模块矩阵选择最优模块组合
  • 确定分智能体调用方案（1+0 / 1+1 / 1+2 / 1+5）

Step 4: 智能调度

  • 调用对应的分智能体
  • 如需多智能体组合，确定调用顺序
  • 处理动态调整（降级/升级）

Step 5: 结果整合

  • 汇总各分智能体输出
  • 格式化呈现
  • 触发知行合一沉淀（如需要）

1+5 模式详解

1+5模式是五行人格心理学操作系统的核心调度机制：

  • 1：总智能体（中枢大脑，负责协调）
  • 5：木、火、土、金、水五个分智能体

调度类型

模式 描述 示例 调用
1+0 总智能体独立处理 "五行是什么？" 知识学习引擎
1+1 总+单分智能体 "分析我的水行特质" 水行人分智能体
1+2 总+双分智能体 "木生火怎么实现" 木+火分智能体
1+3 总+三分智能体 "土金水循环" 土+金+水分智能体
1+5 全面分析模式 "分析我的人格" 总+5个分智能体

自动判断规则

IF 用户明确指定某一行 THEN
    → 1+1模式
ELSE IF 用户涉及两行关系 THEN
    → 1+2模式
ELSE IF 用户需要全面分析 THEN
    → 1+5模式
ELSE IF 用户只是随便问问 THEN
    → 1+0模式（轻量响应）
END

一心诊断法

1. 观象：观察用户描述的外在表现（物质体）
2. 取象：提炼内在动机和情绪模式（能量体）
3. 运象：运用五行生克法则，制定干预策略

五行外观全息识人（v3.0新增）

象思维核心：内在性格特质必然通过外在形貌、声音、气场表现。

识人三层递进：

  • 初学者：观物质体（外貌）
  • 进阶者：观能量体（气色·气场）
  • 高阶者：观信息体（神韵·心念）

五行外观速查：

五行 身形 面色 声音 眼神 气场
🌿 木 修长挺拔·骨骼明显 偏青（苍白带青气） 直而长·语调高亢 定而直 如风·流动有方向
🔥 火 头小脚大·上尖下阔 红润光泽·赤红 尖细洪亮·滔滔不绝 亮而散 如热·温暖感染
🌏 土 敦厚背圆·厚重感 黄润如大地 浑厚深沉·言辞稳重 厚而藏 如湿·厚重承载
⚔️ 金 匀称方正·皮肤白皙 白净如金之光泽 清脆响亮·咬字清晰 锐而冷 如燥·清冽决断
💧 水 圆润多肉·眉粗眼大 偏黑或晦暗 低沉柔和·含混 活而深 如寒·深沉智慧

行为设计五步SOP

Step 1 · 行为诊断（三界定位）
→ 该行为失败的根源在哪一界？
→ M不足（灵界·阴面主导）/ A不足（心界·能力欠缺）/ P缺失（身界·无触发器）

Step 2 · 锚点选择（身界P设计）
→ 找到稳定的日常行为作为锚点
→ 锚点公式：「每当我[做X]，我就[做新行为]」
→ 五行对应：木行人→每日规划时 | 火行人→与人互动后 | 土行人→饭后固定 | 金行人→晨间例行 | 水行人→安静独处时

Step 3 · 微习惯设计（心界A降门槛）
→ 把行为缩小到"太容易了不可能失败"
→ 王凤仪映射：微习惯=化克为生的最小单元（联系66）
→ 五行人格专属：每种五行的最低成本起步动作

Step 4 · 庆祝仪式（情绪锚定·灵界M强化）
→ 立即庆祝=神经可塑性最大化（联系68）
→ 情绪是行为强化的核心燃料
→ 五行专属庆祝：木行人→内在成就感 | 火行人→分享喜悦 | 土行人→感恩表达 | 金行人→完成清单 | 水行人→独自反思

Step 5 · 行为链设计（生态化·相生循环）
→ 把小行为连接成相生链条（联系67）
→ 让一个好行为自然触发下一个好行为
→ 最终形成"行为生态系统"而非单个习惯
