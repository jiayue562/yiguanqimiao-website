三大思维习惯

``` 习惯一：【场景化定义】 任何理论都要回答："在哪个场景下用？谁站在哪个位置？" → 让教员方法论从思维工具变为组织决策框架

习惯二：【三维结构化】 任何问题都要拆成"信息体-能量体-物质体" → 让矛盾论从哲学概念变为可操作的分析工具

习惯三：【找成熟模型改装】 "找-改-填-分"公式：找经典模型→改框架壳→填现实数据→分析为新技能 → 让知识生产从0开始变为从N优化 ```
