人机协同四象限.skills

这是人与AI协同的核心方法论，每次任务开始前必须调用。

Skills 定义

技能名称：人机协同四象限 技能定位：AI时代超级个体黄金分工法则的操作化Skills 核心功能：在任何任务开始前，强制执行"角色与任务分配"，明确回答三个问题：

  1. 人做什么？
  2. AI做什么？
  3. 人机如何握手？

快速激活

当你面对任何任务时，立即问自己：

✅ 本任务的关键问题：
1. 我（悟空）的不可替代性是什么？（创意/决策/价值观/隐性经验）
2. AI（龙龟神将）的增强点在哪里？（数据/执行/发散/初稿）
3. 我们在哪里握手？（交接点/验收标准）

四象限速查

象限 条件 人的核心动作 AI的核心动作
🟦 高效助理 人知 ∩ AI知 清晰指令+审核 高效执行
🟩 学习伙伴 人不知 ∩ AI知 结构化提问+整合 知识提供+分析
🟨 共创伙伴 人部分知 ∩ AI部分知 创意种子+筛选 发散+模式联想
🟥 共创导师 人知 ∩ AI不知 隐性知识外化+定制 基于教导生成
🔲 未知探索 双方都不知 直觉+象思维 大规模假设生成

进化路径

学习伙伴 → 共创伙伴 → 共创导师（终极目标）

方向：你的隐性知识越独特，共创导师象限的时间比例越高 = 超级个体护城河越高
