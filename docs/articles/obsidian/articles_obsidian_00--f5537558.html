    场景 1: 新知识分类

    ``` 1. 输入新知识（文章/PDF） 2. 判断五行属性（取象比类） 3. 归入三层结构（能量/时空/实践） 4. 建立双向链接（同属性/相生/相克） 5. 更新知识图谱 ```

    ``` 1. 查看知识图谱 2. 识别隐性关联（看似无关的节点） 3. 通过五行属性找到共鸣点 4. 建立新的双向链接 5. 涌现创新洞察 ```
