核心定义

立人设是什么

立人设是指在叙事艺术中通过掌握叙事主权来构建具有持久影响力的人物形象的系统方法论。

三位一体结构：

  • 叙事主权：谁讲述故事，谁就定义善恶与是非
  • 情感物理学：通过恐惧与怜悯的转化完成心灵净化
  • 人物矛盾光谱：在性格裂缝中照见真实人性

第一章：叙事主权的底层逻辑

1.1 叙事主权的本质

核心命题：

"谁掌握叙事权，谁就定义善恶；谁赢得最终胜利，谁就拥有永恒正义。"

叙事主权的四重含义：

  1. 话语权定义 - 讲述者拥有对事件 interpretation 的垄断
  2. 价值观构建 - 通过叙事构建善恶标准和是非判断
  3. 情感导向 - 控制读者的情感投入方向
  4. 集体记忆 - 塑造族群和文化的集体无意识

1.2 童话中"后妈"形象的叙事分析

1.2.1 亲妈执笔与后妈背锅

历史机制：

  • 童话的讲述者和传播者大多是亲生母亲
  • 亲妈掌握创作与传播权，将家庭矛盾投射到"后妈"身上
  • 后妈失去为自己辩解的资格，被动接受"邪恶"标签

心理机制（温尼科特理论）：

  • 童话主要由女性口口相传
  • 女性将好的部分放在自己这里，不好的放到"后妈"那里
  • 后妈成为维护亲妈正面形象的工具人

1.2.2 心理防御与负面投射

二元对立简化：

  • 儿童认知世界的需要 → 简化角色复杂度
  • 形成好/坏的二元对立
  • 这是叙事高效运转的基石

心理防御机制：

  • 承认"爱我的妈妈也会伤害我"是难以言喻的痛苦
  • 孩子将负面感受集中投射到"后妈"或"女巫"身上
  • 保护对"完美母爱"的憧憬

1.2.3 自恋母亲的隐喻

《白雪公主》的深度解读：

  • 后妈要猎人拿回白雪公主的肝和肺（排毒/呼吸）
  • 自恋母亲剥夺女儿的辨识能力和表达能力
  • 女儿丧失对美丑、好坏的辨识力
  • 情绪和感受在自恋母亲面前总是受阻

男权社会隐喻：

  • 后妈照镜子 → 寻找男权社会中的自我身份认同
  • 判断自己是否还有对男性的吸引力
  • 魔镜象征男权主义和男性霸权

1.2.4 叙事主权的文化体现

核心逻辑：

  • 每个族群必须掌握自身故事的话语权
  • 这是国家主权在文化层面的深刻体现
  • 新朝建立第一件事：修订史书，将前朝末帝塑造成暴君

东西方文学的"对位书写"：

  • 从《米立：中国童话》到《K》
  • 二元对立关系被刻意书写和有意解构
  • 最终指向权力话语的争夺

第二章：故事科学的情感物理学

2.1 亚里士多德实验与双重情感反应

2.1.1 马车撞人实验

实验设计：

  • 让观众目睹马车撞人的事件
  • 观察观众产生的情感反应

双重情感反应：

  1. 恐惧（Fear） - 来源于代入感："被撞的可能是我"
  2. 怜悯（Pity） - 来源于共情："对他人的不幸感同身受"

2.1.2 情感转化的物理性路径

五步转化模型：

事件发生 → 恐惧生成 → 怜悯触发 → 心灵净化 → 认知升华

故事科学第一秘密：

"人物是可替换的，关键在于事件引发的普遍情感。"

后妈形象的叙事功能：

  • 迫害事件（如毒苹果、毒梳子）= 撞人的马车
  • 无需极其复杂的动机
  • 核心功能：触发观众双重情感的事件内核

2.2 净化机制与认知更新

2.2.1 亚里士多德的"净化"概念

净化（Katharsis）的本质：

  • 从情绪震荡到认知更新的迭代
  • 儿童在故事中完成对恐惧的脱敏
  • 对善恶的认知更新

温尼科特的贡献：

  • 童话把好母亲与孩子分离开来
  • 这种特质使得儿童的自我发展起来并走向整合

2.2.2 现代创作的启示

安徒生的创作动机：

  • 个人经历的家庭破裂
  • 通过描绘后妈的恶劣行为揭示人性黑暗面
  • 启发人们去思考和改变
  • 实现情感物理学的最终闭环：认知的觉醒

有效叙事的三个要素：

  1. 具有冲击力的事件内核
  2. 可引发共情的情感开关
  3. 观众心理状态的迭代

第三章：媒介演变与叙事者身份

3.1 职业定位革命

3.1.1 从"影视技工"到"故事建筑师"

宋方金的革命性观点：

  • 拒绝"影视行业工作者"的局限标签
  • 倡导"故事行业从业者"的元身份认同
  • 叙事艺术是距今2300年的古老行业

媒介演进： | 媒介形态 | 叙事载体 | 核心能力 | |---------|---------|---------| | 口语叙事 | 口耳相传 | 记忆与表达 | | 文字叙事 | 书籍小说 | 结构与文笔 | | 视听叙事 | 影视新媒体 | 视听语言 |

核心观点：

"掌握叙事艺术者永无失业风险，因为叙事能力不受技术迭代影响。"

3.1.2 创作者角色定位

媒介属性 = 时代信息传导者：

  • 创作者是一根"电线"
  • 伟大历史事件是永不枯竭的资源库

成功传导的关键：

  1. 保持信息真实性（"不传谣不信谣"）
  2. 完成时代精神的跨时空传递
  3. 失败的改编 = "被电死"（传导失败）

3.2 叙事主权的跨媒介转移

口传时代：

  • 讲述者（亲妈）拥有绝对主权
  • 叙事是面对面的情感交流

影视时代：

  • 编剧、导演成为新的叙事主权掌握者
  • 技术赋能的规模化传播

永恒不变：

  • 叙事构建价值观的底层逻辑始终如一
  • 真正叙事艺术 = 讲述人们渴望相信却难以做到的故事

第四章：戏剧人物塑造的艺术密码

4.1 哈姆雷特与叙事艺术的进化

4.1.1 复杂人性的持久魅力

哈姆雷特的全球影响力：

  • 每晚都有不同版本在上演
  • 核心矛盾触及生存与死亡的永恒命题

多维性格特征： | 正面特质 | 负面特质 | |---------|---------| | 勇敢 | 鲁莽 | | 深情 | 薄情 | | 细心 | 粗心 |

叙事进化论：

"复杂人性比戏剧冲突更具持久魅力，且人物特质记忆度大于情节细节记忆度。"

4.1.2 麦基方法论与创作黄金法则

罗伯特·麦基的贡献：

  • 1982年创办"故事结构讲习班"
  • 核心理念：重视失败经验的传承

创作黄金法则：

"塑造人物不是堆砌标签，而是展现人性的矛盾光谱——让勇敢者显露怯懦，让深情者暴露薄情，在性格的裂缝中照见真实的人性光芒。"

4.2 失败是成功之母的叙事学意义

4.2.1 失败与情感的深层关联

叙事的双重魅力：

  • 成功的光环
  • 失败的深渊

后妈形象的重构逻辑：

  • 如果后妈仅仅是因为嫉妒而迫害 → 堆砌标签
  • 如果后妈的恶源于：
    • 对衰老的恐惧
    • 对生存的焦虑
    • 对男权体系的无奈屈从

4.2.2 《白雪公主》的现代解读

王后照镜子的深层动机：

  • 确认自己在男权社会中的价值
  • 镜子告知公主比她美一千倍
  • 摧毁的是她的自信和身份认同

反派人物内心矛盾挖掘：

  • 这正是现代叙事区别于传统童话的关键
  • 不再满足于简单的二元对立
  • 在性格裂缝中照见真实的人性光芒

第五章：价值重构与商业变现

5.1 女权主义改写传统

5.1.1 安吉拉·卡特的反叛

《血腥密室》的改写策略：

  • 稍事修改情节
  • 重笔塑造具有女权意识的女性人物
  • 第一人称叙述视角
  • 颠覆佩罗《蓝胡子》中的男权意识
  • 谱写"提高觉悟"之歌

《蓝胡子》改写：

  • 拯救少女的不再是两位哥哥
  • 而是"轮廓如鹰、桀骜不驯"的母亲
  • 一位"愤怒的正义女神"

5.1.2 贝唐·伍尔温的创新

《莴苣姑娘好大胆》：

  • 莴苣不再是等待王子拯救的弱者
  • 剪断长发，让女巫摔死
  • 自己顺着绳梯溜下高塔
  • 成为专杀女巫的"赏金猎人"

改写的本质：

"女性通过自己的诉说，对反驳或挑战父权制意识形态的话语进行编码，夺回属于女性的叙事主权。"

5.2 叙事筛选法则与集体精神图腾

5.2.1 陶渊明案例分析

真相版本：

  • 因妹妹去世回乡奔丧
  • 面临经济困境

流传版本：

  • "不为五斗米折腰"

形成机制：

  1. 文人群体需要精神象征
  2. 大众心理会主动过滤不符合期待的事实
  3. 保留理想化版本

叙事构建的底层逻辑：

"重复讲述会强化集体认知，听众会选择性接受符合预期的版本。"

5.2.2 超人案例

现实：

  • 超人从未实际拯救世界

全球青少年的信念：

  • 坚信其救世主形象

叙事艺术的本质：

"通过理想化表达构建集体精神图腾。"

5.3 短剧赛道：叙事主权争夺的新商业阵地

5.3.1 市场规模与潜力

当前数据：

  • 市场规模：800-1000亿元
  • 预计将诞生估值数百亿的独角兽企业
  • 远超网络大电影巅峰期不足30亿的市值

代表作品分析： | 作品 | 特色 | |------|------| | 《家里家外》 | 表演质感提升 | | 《盛夏芬德拉》 | 视听语言创新 | | 《冒姓琅琊》 | 情节支点设计 |

5.3.2 生态建设原则

必须避免：

  • "副业思维"

企业级布局要求：

  • 建立专业短剧生产线
  • 以长片平台同等规格投入资源

个人创作者面临的挑战：

  • 容易遭遇价值感与收入的天花板

当前态势：

  • 正规军与业余军非对称竞争的关键窗口期
  • 信息不对称是核心挑战

5.3.3 短剧vs网大

核心差异：

  • 网大：市场收缩
  • 短剧：已显现生态构建潜力

叙事主权 = 情感物理学密码：

  • 掌握短剧叙事
  • 意味着掌握下沉市场与碎片化时间的情感密码

5.4 打破刻板印象与现实回归

5.4.1 重组家庭的现实挑战

研究数据：

  • 重组家庭中的孩子抑郁和焦虑发生率是完整家庭的两到三倍
  • 尤其更换姓氏等身份认同问题
  • 孩子感受到身份的被剥夺

5.4.2 王菲对窦靖童的处理

教育智慧：

  • 允许保留与生父的关系
  • 尊重原生情感
  • "不完美接纳"反而给孩子带来安全感

现代叙事主权的最终和解：

  1. 放下塑造"完美家庭"的幻想
  2. 关注孩子的真实情感和需求
  3. 让负面投射在真实的情感接纳中逐渐消解

6.3 人物矛盾光谱构建

四步法：

  1. 设定正面特质 - 主角的核心能力/美德
  2. 植入矛盾面 - 让正面特质有反面阴影
  3. 制造裂缝 - 在压力下暴露矛盾
  4. 照见真实 - 在裂缝中展现人性深度

示例：后妈的矛盾光谱：

正面特质：美丽、权力、地位
    ↓
矛盾面：嫉妒、恐惧、焦虑
    ↓
裂缝显现：对衰老的恐惧、对男权的依赖
    ↓
真实人性：对生存的焦虑，对爱的渴望
