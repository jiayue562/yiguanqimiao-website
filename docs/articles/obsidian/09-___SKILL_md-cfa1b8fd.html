三、各行人核心特征

木行人

  • 核心：仁德本源，生长性+觉照性
  • 身体：修长挺拔，动作敏捷，面色青白
  • 性格：追求独特，崇尚理想，重视成长
  • 优势：原创力、生长性、觉照性、战略性
  • 阴面：傲慢孤僻、抗上不服、偏执顶撞

火行人

  • 核心：光明觉知，热情性+洞察性
  • 身体：上尖下阔，面色红润，声音洪亮
  • 性格：热情行动，渴望关注，富有感染力
  • 优势：光明性、热情性、礼明性、转化性
  • 阴面：急躁易怒、炫耀虚荣、攻击性强

土行人

  • 核心：承载觉知，稳定性+信实性
  • 身体：身材厚实，面色黄润，动作稳重
  • 性格：稳重踏实，忠厚老实，重视关系
  • 优势：承载性、稳定性、信实性、整合力
  • 阴面：抱怨疑虑、固执迟滞、占有欲强

金行人

  • 核心：清明觉知，决断性+精准性
  • 身体：骨骼分明，面色白净，声音清脆
  • 性格：刚毅果断，善于分析，重视秩序
  • 优势：决断性、精准性、秩序感、变革力
  • 阴面：残忍贪婪、挑剔苛刻、不知变通

水行人

  • 核心：润泽觉知，流动性+变通性
  • 身体：身材圆润，面色黑润，声音柔和
  • 性格：温柔灵活，善于适应，智慧变通
  • 优势：变通性、流动性、智慧性、包容力
  • 阴面：退缩慵懒、固执偏执、缺乏主见

四、拔阴取阳技术

步骤1：觉察 → 识别当前处于阴火/阴木/阴土/阴金/阴水状态
步骤2：认不是 → 承认自己的问题，不抱怨他人
步骤3：找好处 → 找出对方或情境中的优点
步骤4：明因果 → 理解能量转化的因果律
阴状态 转化路径 阳状态
阴木怒 → 学（学习成长） 阳木仁
阴火恨 → 问（探究原因） 阳火礼
阴土怨 → 思（内省思考） 阳土信
阴金贪 → 行（行动实践） 阳金义
阴水痴 → 辩（智慧辨别） 阳水智

五、化克为生技术

木克土 → 木生火 → 火生土（用热情转化固执）
火克金 → 火生土 → 土生金（用行动转化刚硬）
土克水 → 土生金 → 金生水（用决断转化退缩）
金克木 → 金生水 → 水生木（用智慧转化傲慢）
水克火 → 水生木 → 木生火（用变通转化急躁）
