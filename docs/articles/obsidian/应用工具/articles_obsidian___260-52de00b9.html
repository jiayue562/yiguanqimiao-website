一、核心定义

1.1 五行识人®是什么

  • 世界首套精气神（身心灵）高维性格测评工具
  • 国家产品专利 + 国家商标局商标注册认证
  • 研发机构：以观其妙书院® × 象思维专家顾问委员会
  • 理论基础：中国优秀传统文化（黄帝内经五行/五态人/阴阳二十五人）+ 现代心理测量学

    • 1.2 测评维度结构

    精气神（身心灵）三层结构：
  • 身层（身体特征）：体形/头面/肤色/声音/动作/体质/口味/病候
  • 心层（心理特征）：性格/情绪/人际/价值观/事业观
  • 灵层（灵性内核）：生命观/修行指向/超越五行
    • 五大行人 × 五小维度 = 25个细分维度

    | 行人 | 五小维度（阳面·健康区） | |------|--------------------------| | 木行人 | M1曲直 M2生发 M3舒畅腾上 M4调达柔和 M5生命成长 | | 火行人 | H1炎上 H2明亮 H3炽烈 H4发散 H5变化 | | 土行人 | J1承载 J2适应 J3容纳 J4运化 J5稳定 | | 金行人 | T1坚固 T2收敛 T3锋利 T4光洁 T5变革 | | 水行人 | S1润下 S2通透 S3沉潜 S4静藏 S5无形养物 |

    1.3 计分规则

  • 题型："是"/"否" 二级评分
  • 计分：答"是"记 2分，答"否"记 0分
  • 反向题：需要反向记分（L量表题）
  • 总分：每行人50题 × 2分 = 最高100分/行人
  • L量表：10题，答"是"记0分，答"否"记10分，>50分=答卷无效

    二、量表开发流程（四大步骤）

    步骤1：建立项目库

    四大来源： 1. 文献查阅法：大五人格、加利福尼亚心理调查表(CPI-RC)、DPA三维、16PF、芬兰Peili、南加州PDP、EPQ、HBTI、4D、MBTI、DISC、九型人格 2. 中国本土五行描述：《黄帝内经》五态人、阴阳二十五人、诸葛亮识人法、刘劭《人物志》、曾国藩《冰鉴》、王凤仪五行伦理疗病 3. 问卷调查法：五行识人工作坊+沙龙被试问卷 4. 专家咨询法：五行专家+心理学教授访谈 整合结果：初步建立 400道题 项目库

    步骤2：项目筛选

    四轮筛选： 1. 删除重复项目 + 不符合构想项目 2. 专家审定：请专家对题项审定修改 3. 被试评价：请工作坊中已确定五行属性人员评价项目，找出表述不清/难以理解题目 4. 完善修订：对问题题目进行完善修订 结果：形成 300题 初测问卷

    步骤3：确定量表形式

    四级评分→二级评分优化：
  • 初版：Likert 5级评分（很不符合→很符合）
  • 终版：二级评分（是/否），符合记2分，不符合记0分
  • 理由：二级评分更简洁、更实用、更易操作
    • 结构：5个方面（维度）+ 随机排列题序 + 10道测谎题（L量表） 结果：形成 310题 初测问卷（300题+10题L量表）

    步骤4：探索性因素分析（EFA）

    七条筛选标准： 1. 因素特征值 > 1 2. 因素符合陡阶检验 3. 因素负荷值 > 0.30 4. 共同度 > 0.20（低于则删除） 5. 抽取因子至少解释 3% 总变异 6. 每个因子至少包含 3道题 7. 因子较好命名 分析工具：SPSS 13.0 + AMOS 4.0 抽样：工作坊被试 + 三家企业单位（人力顾问公司/劳务派遣公司/猎头中介公司） 筛选结果：
  • 初测：310题 → 探索性因素分析 → 保留 260题
  • 木行人：50题
  • 火行人：50题
  • 土行人：50题
  • 金行人：50题
  • 水行人：50题
  • 社会掩饰性（L量表）：10题

  • 大五特质与五行维度对应：

  • 结果：五行识人维度绝大多数与大五人格总分和各因子呈明显相关。
  • 结论：具有较好的效标效度。
