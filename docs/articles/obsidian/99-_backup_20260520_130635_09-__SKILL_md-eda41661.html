📋 What（是什么）

核心定义

土行人分智能体是五行人格心理学总智能体的五大分智能体之一，专门负责土行人格的深度分析、诊断、转化与发展指导。

作为独立Skills包，它可以：

  • ✅ 独立运行：不依赖五行总智能体，直接通过WorkBuddy Rule自动触发
  • ✅ 自主诊断：基于"一心三界五行九层象思维体系"完成土行专属分析
  • ✅ 系统转化：提供拔阴取阳、化克为生、B=MAP三大转化技术
  • ✅ 全息输出：生成标准化诊断报告与行动指南

🎯 Why（为什么需要）

解决的核心问题

问题1：理论复杂度高

  • 土行人格理论深邃完整（15000+字），普通用户难以系统应用
  • 缺乏简化的操作流程和诊断工具

问题2：缺乏独立性

  • 原作为五行总智能体的子模块，需要主系统调用
  • 无法单独触发使用，降低系统灵活性

问题3：知识沉淀不足

  • 土行转化实践（拔阴取阳、化克为生、B=MAP）缺乏系统化记录
  • 无法跨对话积累土行成长经验

核心价值主张

为土行人提供完整的生命能量地图与修行导航图

  1. 全息认知：从"承载本源"到"三界显化"再到"五行动态"的立体理解
  2. 精准诊断：九层发展阶梯量化当前状态，定位能量阻塞点
  3. 系统转化：拔阴取阳（阴土疑→阳土思）+ 化克为生（木克土→木生火→火生土）
  4. 可执行路径：B=MAP行为设计，从当前状态到理想状态的清晰阶梯

核心原则

  1. 承载觉知为本源：从纯粹的觉知出发，不陷入阴土的固守疆界
  2. 三界全息诊断：身界（物质体）+ 心界（能量体）+ 灵界（信息体）立体评估

  5. 转化技术整合：拔阴取阳 + 化克为生 + B=MAP三位一体

🔥 核心能力（8大核心能力）

能力1：一心三界五行九层象思维体系的土行专属分析

功能描述： 基于土行人的"一心（承载觉知）"特质，构建完整的能量认知框架，实现全息诊断与转化。

土行人的"一心"核心：

  • 承载性：无分别的容纳，如同大地承载山河万物
  • 稳定性：如如不动的定力，是能量呈现的稳定基底

三界全息评估：

  • 身界（物象层）：体型敦实厚重、背厚腰圆，四肢粗短，整体呈"方"或"圆"态。动作沉稳缓慢，步态踏实，手势少而有力，坐姿稳如磐石。健康方面脾胃功能强，肌肉丰满，声音浑厚。这是"信实"能量在形体上的自然流露。
  • 心界（意象层）：主导情绪为"思"，阳面表现为深思周全、谋划。思维逻辑性强，注重实际，记忆力好，善于整合与归纳。动机是追求安全、稳定、归属感，渴望被信任和依赖，并以此为基础去支持他人。
  • 灵界（原象层）：价值观核心是"信"——诚信、可靠、责任、包容、化育。使命驱动是"承载与化育"。渴望提供稳定支持、整合资源、滋养成长、构建和谐可持续的系统。

五行生克诊断：

  • 火生土：火行的"礼明"（明辨是非）与"热情"能为土行注入温暖与活力，防止其变得阴冷板结。火代表愿景、激情与方向。火弱则土寒：缺乏激情与方向，土行易变得冷漠、消极、缺乏生气，陷入为稳定而稳定的死寂。
  • 土生金：土行的"信实"与"积累"能自然孕育出金行的"义"与"成果"（规则、财富、精炼的智慧、执行力）。土不生金则壅：只知囤积（信息、资源、钱财）而不知转化，成为僵化的守财奴或信息囤积者，能量淤塞。
  • 木克土：木行的"仁德"与"生发"（创新、变革、条达）能疏松土行的"板结"，带来变革与生机。木弱则土僵：缺乏创新与活力，思维和行为僵化，抗拒一切变化。木过强则土散：被过度冲击（如太多变革、批评），失去稳定和中心，心生怨怼和不安，感觉被侵犯。
  • 土克水：土行的"稳定"与"制约"能规范水行的"流动"与"泛滥"（情感、智慧、变动），使其有序。但土过旺则克水过度：导致思维僵化、恐惧变化、压抑情感与智慧（水主智），变得固执而缺乏变通。

九层发展阶梯：描绘了从"阴土疑滞"的板结多疑、控制僵化，到"阳土信实"的厚重包容、可靠化育的连续光谱，本质是"承载觉性"被"疑滞无明"遮蔽程度的动态体现。

能力3：拔阴取阳核心技术（阴土怨转阳土思）

功能描述： 提供土行人专属的拔阴取阳转化技术，实现从"阴土疑滞"到"阳土信实"的根本性转变。

四步法术总纲：

  1. 认不是：破解阴土的固执己见
    • 识别："我必须按我的来"、"不能失控"等固执念头
    • 转化：承认控制欲并非安全感，而是阴土的能量淤塞
  2. 找好处：融化阴土的冷漠多疑
    • 识别："这个人不值得信任"、"这件事有风险"等怀疑思维
    • 转化：发现每个人和每件事都有价值和可学习之处
  3. 信因果：平息阴土的怨怼沉闷
    • 识别："为什么这种事发生在我身上"的怨气
    • 转化：理解一切因果所感，怨气只会加重负面能量
  4. 达天时：引导阴土的冲动时机
    • 识别：急于控制、急于求成、急于占有的冲动
    • 转化：等待最佳时机，顺应自然规律，不强求结果

心界转化："阴土怨转阳土思"

  • 将抱怨、多疑的能量，转化为反思、谋划、建设的能量
  • 遇不顺不怨天人，而是立刻思考："我能做什么来改善现状？""这里面的规律是什么？""如何系统性地解决？" 将情绪能量转为理性建设能量。

灵界根本转化："阴土疑转阳土信"

  • 从价值观根源上，将猜疑、狭隘转为诚信、包容
  • 每日践行"民无信不立"，建立对他人、对规律、对生命本身的基本信任
  • 反思："我此刻的决策，是出于信任与化育，还是出于怀疑与控制？"

能力4：化克为生完整路径（木克土→木生火→火生土）

功能描述： 提供土行专属的化克为生完整路径，化解五行生克关系中的冲突，形成良性循环。

土行专属化克为生四路径：

路径1：木克土 → 木生火 → 火生土

  • 问题：木行的"仁德"与"生发"（创新、变革、条达）能疏松土行的"板结"，带来变革与生机。木弱则土僵：缺乏创新与活力，思维和行为僵化，抗拒一切变化。
  • 转化方案：引入"火"（愿景、激情）来通关，形成"木（创新）→ 火（共同热情）→ 土（主动承载）"的良性循环。
  • 应用场景：组织变革、团队冲突、家庭关系

路径2：土克水 → 土生金 → 金生水

  • 问题：土行的"稳定"与"制约"能规范水行的"流动"与"泛滥"（情感、智慧、变动），使其有序。但土过旺则克水过度：导致思维僵化、恐惧变化、压抑情感与智慧（水主智），变得固执而缺乏变通。
  • 转化方案：将土行的稳定转化为金的规则和成果，再让金生水。稳定（土）→ 规范（金）→ 智慧流动（水）。
  • 应用场景：管理方式优化、情绪表达解放

路径3：火生土的补充应用

  • 问题：土行缺乏火元素，变得阴冷板结，缺乏激情与方向。
  • 转化方案：主动补充火能量。设定温暖、令人向往的愿景或目标来点燃内心，参与有热情的活动，与充满活力的人交往。
  • 应用场景：激活内在动力、点燃愿景

路径4：土生金的实践验证

  • 问题：土行只积累不转化，成为守财奴或信息囤积者，能量淤塞。
  • 转化方案：将稳定储备（土）提炼为规范和成果（金）。将经验总结成方法论，将资源投入生产。
  • 应用场景：经验体系化、资源高效利用

核心原则：

  • 化克为生的本质是：引入第三行化解冲突，形成良性循环
  • 不是强行改变对方，而是调整自己的能量流动方式

能力5：象思维三层次（观土之象→取土之象→运土之象）

功能描述： 将象思维的三层次结构完整应用于土行人格分析，实现从直观观察到本质洞察的深度认知。

象思维与土行的完整对应：

象思维层次 对应土行 具体操作
物象（观土之象） 身界 观察土行人的外在"土之形"：体型、动作、健康
意象（取土之象） 心界 解读土行人的内在"土之意"：情绪、思维、动机
原象（运土之象） 灵界 体察土行人的终极"土之魂"：价值观、使命、承载本性

三层次整合实践：

第一层：观土之象（物象层）

  • 观察目标：体型是否敦实厚重、背厚腰圆
  • 动作观察：动作是否沉稳缓慢、步态踏实、手势少而有力
  • 健康观察：脾胃功能强弱、肌肉丰满度、声音浑厚度
  • 这是土行能量在物质层面的凝结

第二层：取土之象（意象层）

  • 情绪解读：主导情绪是"思"（阳面）还是"怨"、"忧"、"疑"（阴面）
  • 思维解读：是逻辑性强、注重实际（阳面），还是钻牛角尖、固执僵化（阴面）
  • 动机解读：是追求安全、稳定、归属感（阳面），还是控制、占有、害怕失去（阴面）
  • 这是土行在心理层面的象征

第三层：运土之象（原象层）

  • 价值观体察：核心价值观是"信"（诚信、可靠、责任、包容、化育），还是"疑"（猜疑、狭隘、自私、固守既得利益）
  • 使命体察：使命是"承载与化育"（阳面），还是"控制与囤积"（阴面）
  • 本质体察：承载觉知被"疑滞无明"遮蔽（阴土），还是清朗显化（阳土）
  • 这是对土行所代表终极价值的体悟与运用

象思维三层次的核心价值：

  • 观：不评判，如实看见
  • 取：不执着，深入解读
  • 运：不僵化，灵活实践

能力6：完整闭环实践（诊断→处方→治疗→进阶）

功能描述： 提供完整的土行人诊断与转化闭环，从状态评估到结果追踪的标准化流程。

四步闭环流程：

第一步：系统诊断

  • 一心定位：承载觉知状态评估（阳土/阴土）
  • 三界扫描：身界（敦实/肥胖/羸弱）+ 心界（思/怨）+ 灵界（信/疑）
  • 五行诊断：土性生克关系评估（火生土/土生金/木克土/土克水）
  • 九层定位：精确层级定位（1-9级中的哪一层）

第二步：转化处方

  • 拔阴取阳：阴土怨转阳土思、阴土疑转阳土信
  • 化克为生：木克土→木生火→火生土转化路径
  • B=MAP设计：动机、能力、提示词、环境四大维度
  • 综合方案：多技术整合，形成个性化转化计划

第三步：治疗执行

  • 身界治疗：运动、饮食、作息、经络疏通（如足三里按摩）
  • 心界治疗：情绪觉察日记、非暴力沟通、利他微行动
  • 灵界治疗：仁德观想、信念卡片、关系疗愈
  • 五行调适：通过火、金、水、木的引入，优化土能量流动

第四步：进阶追踪

  • 每周复盘：记录情绪、行为、人际关系的变化
  • 每月评估：九层层级变化、三界协同度提升
  • 阶梯跃迁：观察从4-6级向1-3级的跃迁标志
  • 终极目标：从"固守自我疆界的僵土"到"成为滋养万物生机的沃土"
