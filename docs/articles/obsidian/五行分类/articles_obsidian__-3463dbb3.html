    🔄 同构性分析

    同构 1: 双向互通

    五行生克: ``` 木生火 → 木是火之源头，火是木之展现（双向） 木克土 → 木制约土，土反哺木（双向） ``` Obsidian 双向链接: ``` [[A]] → [[B]] A 的出链 = B 的入链（双向互通） ``` 本质: 关联是双向的，不是单向因果

    ---

    同构 2: 网络涌现

    五行图谱: ``` 单个五行节点 → 无特殊价值 纳入五行体系 → 自动与所有节点形成生克关联 整体涌现 → 系统认知、跨界启发 ``` Obsidian 知识网络: ``` 单个笔记 → 信息孤岛 通过双向链接互联 → 知识神经网络 整体涌现 → 自主关联、系统认知 ``` 本质: 网络价值 > 节点价值之和

    ---

    同构 3: 动态进化

    五行生克制化: ``` 正常：相生相克维持平衡 异常：相乘相侮预警失衡 修复：通过生克调节恢复平衡 ``` Obsidian 知识网络: ``` 正常：链接密度持续增长 异常：孤立节点、断裂链接 修复：健康检查 + 链接修复 ``` 本质: 系统具备自我调节能力

    属性：木

    关联

  • 同属性：[[肝脏]] [[绿色饮食]]
  • 相生：[[养心食谱]] (木生火)
  • 相克：[[健脾食谱]] (木克土)
    • ``` 标签规范:
  • #wood (木) #fire (火) #earth (土) #metal (金) #water (水)
  • #能量层 #时空层 #实践层
