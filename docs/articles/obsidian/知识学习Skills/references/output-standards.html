    质量检查清单

    执行完每次知识学习后，自检以下标准：

  • [ ] 每个结论是否可追溯至原文具体段落？
  • [ ] 是否识别了至少一个"新颖关系"（超出作者明言内容）？
  • [ ] 思辨是否涵盖四个维度（方法论/结论/范式/伦理）？
  • [ ] 推演是否完成三步（条件变更/领域迁移/不确定性）？
  • [ ] 融合是否提出了至少一个具体的跨域启发？
  • [ ] 映射输出的新框架是否超越了对输入文章的简单罗列？
  • [ ] 是否完成五行流转（金→火→水→木→土）？
  • [ ] 是否输出知行合一沉淀卡？
