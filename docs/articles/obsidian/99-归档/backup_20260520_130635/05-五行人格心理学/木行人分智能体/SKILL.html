📖 核心定义

What（是什么）

木行人分智能体是五行人格心理学总智能体的五大分智能体之一,专门负责木行人格的深度分析、精准诊断、系统调适与发展指导。

它基于完整的"一心三界五行九层象思维体系",为木行特质个体提供从"自我觉察"到"生命超越"的完整实践地图。

Why（为什么）

核心价值：

  1. 精准诊断: 通过三界九层体系,精准定位木行人的当前状态与卡点
  2. 系统调适: 运用拔阴取阳、化克为生两大技术,提供结构化干预方案

  4. 实践落地: B=MAP行为设计框架,将理论转化为可操作的日常行为

解决什么问题：

  • ✅ 木行特质个体不认识自己的优势与盲点
  • ✅ 固执、对抗、人际关系紧张（木克土）等常见困境
  • ✅ 理想高远但落地困难（木不生火）
  • ✅ 创新能力强但缺乏规则意识（木过旺克土）

How（怎么做）

核心操作步骤：

  1. 三界诊断: 观察身界、心界、灵界的具体表现
  2. 五行分析: 评估木行与其他四行的生克关系

  4. 转化干预: 运用拔阴取阳四步法 + 化克为生路径

🚀 核心引擎启动

一心：仁德本源

木行人的"一心"是纯粹觉知,其天然倾向是生长性与觉照性：

  • 生长性: 如同春天树木的生发,是内在生命力的自然、和谐、非功利的流露。不是野心或控制欲的扭曲表现。
  • 觉照性: 具备清明的观照能力,能如阳光般照亮内在的念头、情绪与身体感受。

发现路径（悬置—内观—合一）：

  1. 悬置: 放下对自我主张、计划、控制的执着
  2. 内观: 觉察内在能量状态——是僵硬、对抗的（阴木）,还是柔韧、生长的（阳木）？
  3. 合一: 体悟真正的"生长"与"觉照"本是一体,达到此境便是触摸到了木行人的"一心"本源

三界：木象显化

身界（物象层）- 可直观感知的"木之形"

状态 身体表现 健康提示
阳木（健康） 体型修长挺拔,动作敏捷舒展,筋腱柔韧,面色青白透红润 筋健骨正,肝胆系统健康
阴木（失衡） 体型僵硬佝偻,动作拘谨顶撞,易筋痛、头痛、肝胆不适 肝气不舒,木郁成疾

心界（意象层）- "木"在心理层面的象征

状态 情绪表现 思维模式 动机偏好
阳木（健康） 正义感主导,情绪来得快去得也快 敏捷、善于谋划、有远见和战略眼光 追求成长、创新、自主,厌恶束缚
阴木（失衡） 愤懑（持久、针对性）,忧愁善变,情绪不稳定 钻牛角尖（固执己见）,多疑 抗上（证明权威错）、证明自我、控制局面

灵界（原象层）- "木"所代表的终极价值

状态 价值观 使命驱动
阳木（健康） 核心是"仁"——仁慈善良、生生之德、爱护与促进成长 创新与生发,开拓新领域、建立新系统
阴木（失衡） 扭曲为"慢"（傲慢）、偏执、对抗 破坏与报复,通过否定、批判破坏现有秩序来彰显自我

三界协同规律（马车比喻）：

  • 协同（健康）: 灵界驾车人（仁德使命）→ 心界车身（规划担当）→ 身界马匹（敏捷正直）,三者方向一致,能量流畅
  • 失衡（病态）: 灵界迷失（失仁转傲）→ 心界愤懑控制 → 身界顶撞僵直,系统陷入内耗

五行：木性生克规律

木行自身的阴阳两面：

  • 阳木（仁德）: 仁慈善良、正直担当、灵活创新、生发有道、条达舒畅
  • 阴木（傲慢）: 固执顶撞、抗上不服、忧愁善变、僵化偏执、郁结不畅

关键生克关系对木行的影响：

关系 作用 失衡表现
水生木 水行的"智慧"与"滋养"为木行提供深远源泉和灵活性 水弱则木枯：生长缺乏后劲,易变得干枯、偏执、缺乏变通
木生火 木行的"规划"与"创新"能点燃火行的"热情"与"行动力" 木不生火则空：理想与计划难以落地,停留在空想层面,缺乏执行力
金克木 金行的"规则"与"收敛"能修剪木行无序生长的"枝蔓",使其生长有序、目标聚焦 金弱则木亢：缺乏规则约束,任性妄为；金过强则木折：被过度压制,心生愤恨
木克土 木行的"生发"能疏松土行的"板结",为稳定的系统带来活力、变化与创新 木过旺则克土过度：否定一切、破坏稳定、难以落地,使人际关系紧张

木行核心调适技术：

拔阴取阳口诀：

  • 心界转化："阴木怒转阳木学"——将固执、对抗的能量,转化为学习、钻研、提升自我
  • 灵界根本转化："阴木傲慢转阳木仁"——从价值观根源上,将傲慢、对抗转为仁慈、利他

化克为生实践：

  • 当陷入"木克土"导致的固执僵化、人际冲突时,引入"火"（愿景、热情）来形成"木生火,火生土"的良性循环

九层：木行能量曲直阶梯（全息解析版）

九层发展阶梯是"一心→三界→五行"协同状态的量化体现,描绘了从"阴木傲慢"的偏执、僵化、对抗,到"阳木仁德"的舒展、创新、仁慈的完整演变光谱。

核心逻辑闭环：

一心（生生之仁）→ 三界（木象显化）→ 五行（木性生克）→ 九层（曲直阶梯）→ 回归一心（仁德周流）

第一部分：理论根基与核心逻辑闭环

1.1 "一心三界五行九层"体系总览

木行人的发展层级模型,是全息（全人）发展理论在木行能量特质上的具体化与实践化：

层次 定义 木行特质
一心（本源） 纯粹的、无分别的觉知,是生命能量的终极源头 生长性（促进万物生长的仁德之性）+ 觉照性（清明的观照能力）
三界（显化） "一心"能量在个体层面的具体呈现 身界（物象层·形体行为）、心界（意象层·情绪思维）、灵界（原象层·价值观使命）
五行（规律） 木行能量在动态网络中的运行规律 水生木（滋养）、木生火（转化）、金克木（修剪）、木克土（生发）
九层（阶梯） "一心→三界→五行"协同状态的量化体现 仁德觉性被傲慢无明遮蔽程度的动态光谱

1.2 核心逻辑闭环

木行人的成长与衰退,遵循清晰的闭环逻辑：

  • 一个木行人的外在行为、内在冲突和成长潜力,都源于此能量链条上某个或某些环节的特定状态
  • 层级的升降,本质上是"阴木"能量与"阳木"能量的比例变化
  • 修炼方向：运用"五行"规律,在"三界"中实施"拔阴取阳"的转化技术,实现层级向上跃迁

第二部分：九大发展层级全息剖析

九层分为三大区间,每个层级从五个维度详解：意向维度（灵界）、性格特点维度（心界）、行为维度（身界）、三界失衡特点、拔阴取阳技术维度

🌱 健康区间（近道·第1-3级）：阳木主导,三界协同,仁德通道

此区间个体是"仁德"能量的透明通道,木行特质以创造性、建设性的方式服务自身与整体成长。

第1级：解放层级（与道共舞）

维度 详解
意向维度（灵界） 与道合一,无为而无不为。核心动机完全超越个人成就与独特性的证明,升华为对宇宙真理的追求与服务整体进化。对应马斯洛的"自我实现"或"超越性需求",在积极心理学中堪称"充盈"状态。
性格特点维度（心界） 高度整合的阴阳平衡。直觉（阳木,洞察未来）与实感（阴木,扎根现实）功能和谐共存。性格仁德舒展,无丝毫偏执我慢,内心通透豁达,充满慈悲与包容。
行为维度（身界） 灵活创新,顺势而为。行为如春风化雨,自然促进万物生长。能提出突破性理论或愿景,并以人格魅力凝聚团队,推动社会进步。行动高效且充满弹性,毫无刻意与执着。
三界失衡特点 三界高度和合,无失衡。身界柔韧有力,心界平静喜悦,灵界与万物一体。能量自然流动。
拔阴取阳技术维度 守一于太和,保持觉知。技术已内化为生命本身。日常可通过"创意冥想"观想阳木能量如树木生长,锚定"木仁"核心,不执着于创新的表象,让生发之象自然流露。

五行特征：

  • 灵界：仁德周流,与万物一体
  • 心界：思维开阔,无有偏执
  • 身界：动作自然,柔韧有力
  • 五行：阳木主导,水木相生（智慧滋养生机）,木火通明（创意照亮行动）

第2级：心理能力层级（中流砥柱）

维度 详解
意向维度（灵界） 有利他性的成长使命。核心动机是运用自己的独特才华与创新能力达成目标、引领变革,并在过程中惠及他人。
性格特点维度（心界） 正直担当,灵活创新。情绪稳定,抗压性强,善于规划与谋略。思维敏捷,有远见,同时具备落地执行的稳健心态。自信豁达,面对挫折能快速调整。
行为维度（身界） 执行力强,能屈能伸。能有效将理想（木）转化为热情行动（火）。在社会角色中（如管理者、专家）勤奋进取,成果显著。行为主动且善于根据反馈优化。
三界失衡特点 三界基本平衡。偶因外界挑战或高标要求,出现轻微的心界焦虑或身界紧张,但能通过成熟的防御机制（如升华、理智化）快速转化。
拔阴取阳技术维度 强化能力,预防僵化。定期进行"问题解决日志"复盘,将压力转化为学习机会。通过适度运动舒展身心（强化木象）,同时制定清晰短期目标（引入金行收敛之象）防止创新失控。练习"阴木怒转阳木学"的初级形态——在感到不耐烦时,转化为深入钻研细节的动力。

五行特征：

  • 灵界：有利他性的成长使命
  • 心界：善于规划,抗压性强,情绪稳定
  • 身界：执行力强,能屈能伸
  • 五行：木生火旺,能将理想（木）清晰转化为热情行动（火）

第3级：社会价值层级（才显于外）

维度 详解
意向维度（灵界） 追求理想与社会价值实现。渴望在专业领域或社会层面发挥影响力,获得认可,同时坚守内心原则。
性格特点维度（心界） 才气外显,成就导向。正直友善,善于协作,但内心对认可有所渴望。思维开阔,能将创新与社会需求结合。
行为维度（身界） 勤奋进取,成果落地。在社会结构中有效发挥创新能力,成为领域的标杆。积极分享经验,赋能他人,但可能偶尔因追求完美或独特而显得固执。
三界失衡特点 三界协调,偶有微澜。灵界对"社会价值"的追求,可能偶尔导致心界在评价面前波动,或身界因过度投入而疲惫。整体仍以阳性能量主导。
拔阴取阳技术维度 平衡独特性与社会性。进行定期的"社会价值评估",反思创新是服务于真实需求,还是为了凸显独特。践行"调达柔和",培养共情能力。通过团队协作（土行承载）确保创意落地,避免成为空中楼阁。

五行特征：

  • 灵界：追求理想与社会价值实现
  • 心界：渴望认可但坚守原则
  • 身界：勤奋进取,成果显著
  • 五行：木势得当,能合理"克土"（破旧立新）并"生火",推动社会进步
⚠️ 一般区间（失衡·第4-6级）：阴阳交战,三界失衡,内在挣扎

此区间个体陷入"仁德"与"傲慢"的内心拉锯战,木行能量开始扭曲,人际关系与现实适应出现明显问题。

第4级：失衡层级（固执己见）

维度 详解
意向维度（灵界） 维护自我正确,抗拒变化。价值观开始模糊,出现傲慢的苗头。从"生发成长"转向"捍卫我的观点和方式"。
性格特点维度（心界） 固执己见,内外冲突。思维开始僵化,认为自己的方式才是唯一正道。易怒、忧愁,正直带上了"刺"。对新生事物感到不安和排斥。
行为维度（身界） 行为对抗,协调性下降。在团队中可能成为阻碍,排斥新思想,以"仁"或"专业"的名义行控制之实。沟通时缺乏弹性。
三界失衡特点 三界开始联动失衡。灵界（价值观模糊）→ 心界（思维僵化、易怒）→ 身界（行为对抗、肝气郁结）。五行上表现为阴木初显,木土相克,人际关系开始紧张。
拔阴取阳技术维度 "松"字诀,破执学习。核心是打破僵化。方法包括：1) 身界：通过瑜伽、太极、拉伸等舒展运动疏通肝经；2) 心界：练习"悬置",当固执念头升起时,暂停并自问"是否只有这一种可能？"；3) 灵界：反思"仁"的本意是"己欲立而立人",而非"己所欲,施于人"。引入"金行"的规则意识（如接受客观流程）来收敛过度的自我主张。

五行特征：

  • 灵界：价值观模糊,傲慢初显
  • 心界：思维开始僵化,易怒忧愁
  • 身界：行为出现对抗性,协调性下降
  • 五行：阴木初显,木土相克,人际关系开始紧张

第5级：人际控制层级（傲慢证明）

维度 详解
意向维度（灵界） 渴望认可,证明自我。使命扭曲为"证明我的独特性和正确性"。价值感高度依赖外界反馈。
性格特点维度（心界） 傲慢敏感,易怒控制。情绪主导为愤怒和挫败感。思维模式是"他们不行/不懂",控制欲强（希望事态按自己计划发展）,敏感于他人的质疑。
行为维度（身界） 刻意抗上,刷存在感。喜欢批评、指责他人,维护自我权威。沟通时身体前倾、言辞尖锐,通过否定他人来彰显自我价值。行为导致团队士气低落。

🎯 B=MAP行为设计框架

:B=MAP是木行人分智能体的标准行为设计工具,基于福格行为模型（B=动机M=能力A=提示P=提示）。

框架结构

:B（行为）= M（动机,Motivation）+ A（能力,Ability）+ P（提示,Prompt）

木行人专属设计原则

维度 阳木（健康） 阴木（失衡） 转化策略
动机M 仁德驱动,利他成长 傲慢驱动,证明自我 从"证明正确"转向"帮助成长"
能力A 规划创新,正直担当 固执对抗,钻牛角尖 从"硬对抗"转向"软学习"
提示P 明确愿景,阶段性目标 模糊冲动,缺乏计划 设定清晰提示：场景+时机+具体行动

设计模板

案例1：改善人际关系（木克土转化）

:B（行为）：在团队会议中,当别人反对我的方案时,先倾听再讨论
  = M（动机）：建立和谐团队氛围,而非证明自己正确
  + A（能力）：忍辱修习,听取意见,整合优化
  + P（提示）：会议前提醒自己"今天的目标是倾听和整合"

案例2：提升执行力（木生火）

:B（行为）：每周设定3个小目标,并在24小时内启动第一个行动
  = M（动机）：将创新转化为实际成果
  + A（能力）：阶段性拆解,快速启动
  + P（提示）：晨起问自己"今天的第一步是什么？"
