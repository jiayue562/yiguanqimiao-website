核心定义

What：什么是复利？

复利是指利息也计入本金产生新的利息，即"利滚利"。与单利（仅本金产生利息）相比，复利随时间呈现指数增长。

数学公式:

A = P × (1 + r)^t

其中：
- A = 最终金额
- P = 本金
- r = 每期利率
- t = 期数

72法则:

  • 资金翻倍所需时间 ≈ 72 ÷ 年利率(%)
  • 例：年利率8%，约9年翻倍

Why：为什么重要？

  1. 时间的力量：长期复利效应惊人（爱因斯坦称其为"世界第八大奇迹"）
  2. 指数思维：理解非线性增长的关键
  3. 财务规划：退休储蓄、投资规划的基础
  4. 习惯养成：知识、健康、关系的复利效应

How：如何应用？

核心操作流程:

  1. 确定增长率：年化收益率或成长速度
  2. 计算时间跨度：投资或成长的周期
  3. 估算终值：利用公式或72法则快速估算
  4. 反向规划：从目标倒推所需本金或时间

应用场景

场景1：投资理财

情境：每月定投1000元，年化收益8%，30年后有多少？ 计算：

  • 月投1000元，年投12000元
  • 年金终值公式：FV = PMT × [(1+r)^n - 1] / r
  • 结果：约150万元（本金36万，利息114万）
  • 关键洞察：时间是复利最好的朋友，早开始比高回报更重要

场景2：知识积累

情境：每天学习1小时，持续10年 应用：

  • 假设每天进步1%
  • (1.01)^365 ≈ 37.8倍/年
  • 10年后：37.8^10 ≈ 不可想象的成长
  • 关键洞察：知识的复利不仅在于量，更在于连接和深度

场景3：健康习惯

情境：每天运动30分钟 vs 周末突击运动3小时 应用：

  • 每天30分钟：365天×30分钟 = 182.5小时/年
  • 周末突击：52周×3小时 = 156小时/年
  • 但复利效应：每天运动形成习惯，健康收益呈指数增长
  • 关键洞察：持续性比强度更重要

常见误区

误区 正确理解
忽视时间 复利需要时间，短期看不出效果
高估收益率 承诺高复利的往往是骗局
中断损失 复利最怕中断，中断后需重新积累
负复利 亏损也会复利，风险控制至关重要
