🎯 五行系统总览

五行不是五种"物质"，而是五种动态的功能模式和关系类型。

五行 核心功能 运动特征 季节 方位 颜色
木 生发、舒展 向上向外 春 东 青/绿
火 炎上、光明 向上发散 夏 南 赤/红
土 承载、化育 中和稳定 长夏 中 黄
金 收敛、肃降 向下向内 秋 西 白
水 润下、藏纳 向下潜藏 冬 北 黑

🔄 五行生克关系

相生循环（生成关系）

木 → 火 → 土 → 金 → 水 → 木（循环）
│    │    │    │    │
│    │    │    │    └─ 水生木（滋养生长）
│    │    │    └────── 金生水（凝结成液）
│    │    └─────────── 土生金（矿藏于土）
│    └──────────────── 火生土（灰烬成土）
└───────────────────── 木生火（燃烧生火）

相生含义：

  • 水生木：水滋养树木生长
  • 木生火：木材燃烧产生火
  • 火生土：火烧尽后成灰土
  • 土生金：金属矿藏于土中
  • 金生水：金属表面凝结水珠（或金属熔化成液态）

相克循环（制约关系）

木 → 土 → 水 → 火 → 金 → 木（循环）
│    │    │    │    │
│    │    │    │    └─ 金克木（金属砍伐树木）
│    │    │    └────── 火克金（烈火熔化金属）
│    │    └─────────── 水克火（水能灭火）
│    └──────────────── 土克水（土能挡水）
└───────────────────── 木克土（树根穿透土壤）

相克含义：

  • 金克木：金属工具可以砍伐树木
  • 火克金：烈火可以熔化金属
  • 水克火：水可以灭火
  • 土克水：土可以阻挡水流
  • 木克土：树根可以穿透土壤

生克平衡原则

核心智慧：

"生中有克，克中有生，生克制化，动态平衡。"

过度相生：

  • 母旺子衰：生者过强，被生者反而受损
  • 例：水太多，木反而烂根

过度相克：

  • 克者过强：被克者受损
  • 例：金太强，木被过度砍伐

理想状态：

  • 生克有度，动态平衡
  • 生而不溺，克而不伤

个人五行诊断

步骤 1：观象

  • 观察自己的气质、行为模式、偏好
  • 识别主导的五行能量

步骤 2：平衡分析

  • 哪一行过旺？
  • 哪一行过弱？
  • 哪一行被克？

步骤 3：调整策略

  • 旺者泄之（如木旺用火泄）
  • 弱者补之（如木弱用水补）
  • 克者化之（如金克木，用水化解）
