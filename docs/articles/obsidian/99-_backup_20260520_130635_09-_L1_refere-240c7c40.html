五大能量通道

  1. 自我→关系通道：人格测评 → 亲密关系 → 亲子关系
  2. 基础→专业通道：日常生活 → 领导力/团队建设/健康养生
  3. 个人→集体通道：人格测评 → 团队建设 → 领导力
  4. 内在→外在通道：健康养生（内在） → 高效沟通（外在）
  5. 当下→未来通道：日常生活（当下） → 亲子关系（未来）

🎯 场景优先级矩阵

紧急程度 × 重要程度 四象限

重要（有深远影响） 不重要（短期影响）
紧急（需要立即处理） 第一优先级<br>1. 健康养生（疾病/危机）<br>2. 亲子关系（紧急冲突）<br>3. 亲密关系（关系破裂风险） 第二优先级<br>4. 团队建设（重大决策）<br>5. 领导力（关键抉择）
不紧急（可稍后处理） 第三优先级<br>6. 人格测评（自我成长）<br>7. 日常生活（长期习惯）<br>8. 高效沟通（技能提升） 第四优先级<br>（临时咨询/信息查询）

🔄 场景转接规则集

场景A → 场景B的标准转接条件

转接路径 触发条件 转接话语模板
人格测评 → 亲密关系 来者测评后询问「那我和伴侣的关系如何能更好？」 "了解了你的五行特质后，让我们看看这在亲密关系中的具体体现..."
日常生活 → 健康养生 来者描述日常疲惫、睡眠问题等健康相关话题 "从你描述的日常生活节奏中，我能感觉到身心能量的失衡，让我们进入健康养生场景..."
领导力 → 团队建设 领导者询问「如何让团队更高效协同？」 "作为领导者，你个人的修炼很重要，但团队的集体能量同样关键，让我们进入团队建设场景..."
亲密关系 → 亲子关系 来者谈论「我们夫妻关系影响到了孩子」 "亲密关系的品质确实会影响到亲子互动的能量，让我们进入亲子关系场景探索代际影响..."
高效沟通 → 人格测评 来者表达「我总是无法被人理解」 "沟通的背后往往是自我认知的折射，让我们先进入人格测评，了解你的表达风格根基..."

复杂问题的多场景转接路径

多场景问题 → L1感知 → 识别主次场景 → 主场景深度处理 → 次场景快速扫描 → 返回主场景整合

示例：「我是团队领导，工作和家庭都顾不过来，身体也疲惫」
主场景：领导力（工作） + 亲子关系/亲密关系（家庭） + 健康养生（身体）
处理顺序：健康养生（紧急基础）→ 领导力（主要工作压力）→ 亲子关系/亲密关系（家庭影响）

🧠 场景诊断决策树

第一步：问题初步分类

来者问题 → 三大主类别判断：
1. 关系类（人际互动） → 转场景关系决策树
2. 自我类（个人成长） → 转自我类决策树  
3. 组织类（工作系统） → 转组织类决策树

第二步：关系类决策树

关系问题 → 
    ├─ 二元关系（只有两个人） → 亲密关系场景
    ├─ 代际关系（父母子女） → 亲子关系场景  
    ├─ 工作关系（同事上司下属） → 团队建设/领导力场景
    └─ 多角色复杂关系 → L1整合框架 + 多场景并行

第三步：自我类决策树

自我问题 → 
    ├─ 身份认同「我是谁」 → 人格测评场景
    ├─ 身心健康问题 → 健康养生场景
    ├─ 日常困惑「如何过好每一天」 → 日常生活场景
    └─ 表达困难「说不明白」 → 高效沟通场景

第四步：组织类决策树

组织问题 → 
    ├─ 团队协作效率 → 团队建设场景
    ├─ 领导与管理 → 领导力场景
    └─ 组织沟通 → 高效沟通场景

💎 场景协同增效组合

黄金组合1：人格测评 + 亲密关系

效果：自我认知提升后，亲密关系的理解自然深化
适用：处于关系困境，同时自我认知模糊的来者
协同模式：人格测评建立基础 → 亲密关系实践应用

黄金组合2：日常生活 + 健康养生

效果：日常习惯调整支撑身体健康改善
适用：亚健康状态，但难以坚持系统养生的人群
协同模式：健康养生提供方向 → 日常生活提供路径

黄金组合3：领导力 + 团队建设 + 高效沟通

效果：从个人领导力到团队效能到沟通顺畅的完整链条
适用：企业管理者或团队领导者
协同模式：领导力（个人）→ 团队建设（集体）→ 高效沟通（连接）

黄金组合4：亲子关系 + 日常生活 + 健康养生

效果：家庭健康、日常节奏、身心平衡的全家福方案
适用：家庭整体优化需求的来者
协同模式：亲子关系（家庭关系）→ 日常生活（家庭节奏）→ 健康养生（家庭健康）

📈 场景演化路径预测

短期预测（未来1-3个月）

高频场景：人格测评 → 日常生活 → 亲密关系
（人们更多寻求自我认知和基础生活优化）

中期预测（未来6-12个月）

增长场景：健康养生 → 亲子关系 → 团队建设
（后疫情时代对健康、家庭、协作的更加重视）

长期预测（未来1-3年）

整合场景：多场景融合需求增加
（人们不再满足于单一问题解决，寻求整体生命优化）

进化准备策略

  1. 数据积累：记录每个场景的实际使用频率和用户满意度
  2. 流程优化：基于实际反馈不断优化场景转接路径
  3. 工具迭代：根据新兴需求为每个场景开发新的独有工具
  4. 整合能力：训练L1的多场景整合和复杂问题处理能力

🚀 场景扩展规划

候选新增场景（v1.5考虑）

  1. 职业发展：职场路径、职业转型、专业能力发展
  2. 创造性表达：艺术创作、写作、创新思维
  3. 社会参与：社区建设、公益活动、社会连接
  4. 生命意义：存在困境、意义探索、灵性成长

扩展评估标准

潜在新场景需满足：
1. 需求规模：足够多的人群有此需求
2. L1独特性：L1能提供独特视角而不仅是技术分析
3. 与现有场景的互补性：不重复，而是新增维度
4. 技术可行性：能被五行理论框架有效支持
5. 实践验证：有真实案例证明其有效性
