二、维度一：曲直（才气独特--傲慢孤僻）

核心定义：曲直，是枝曲干直的缩语。描述树木生长形态主干挺直向上，树枝曲折向外。引申为"木"有生长、兴发、生机、条达、舒展的特性。冒地而出，旭日东升，阳气动跃之象均属木。简单地理解就是爱独特，木行人兴趣独特、学识独特、想象独特、情感独特、行为独特、感受独特、审美独特......总之，如果做得不够独特，就不是木行人了。

阳面：才气独特

1. 才华横溢

木行人充满新思想。他们乐观、自然、富有创造性和自信，具有独创性的思想和对可能性的强烈感受。对于木行人来说，生活是激动人生的戏剧。因为木行人对可能性很感兴趣，所以他们了解所有事物中的深远意义，喜欢许多可供选择的事物的存在。他们具有洞察力，是热情的观察者，注意常规以外的任何事物。

2. 思路开阔

木行人拥有高度的创造力，思路开阔，观念新，富有想象力，是"点子型的人才"；木行人不受到条条框框约束，不拘小节，难守规则；他们大多性格内向，以奇异的方式工作，与人打交道是他们的弱项。他们喜欢按照自己的方式生活和工作，更喜欢穿牛仔裤、休闲服上班，他们代表着那些个性和其他的成员不太一样的一群新潮人士。

3. 理想主义

木行人是透过自我感受来认知周围世界的。他们看得远。不在乎现实，更倾情理想。他们是自我风格管理的高手，却不是调御他人的高手，因为他们一般不耐繁杂，把自己高明的理论付诸实现，所以不能以事实服人。

4. 足智多谋

木行人具有想象力、适应性和可变性，他们视灵感高于一切，常常是足智多谋的发明人。有时他们不墨守成规，善于发觉做事情的新方法。木行人为思想或行为开辟新道路，并保持它们的开放。在完成新颖的想法的过程中，木行人依赖冲动的能量。他们有大量的主动性。

5. 感受丰富

木行人容易感知自己的情绪和内心世界，他们敏感、重视和接纳对自己内心的感受，能体验到更深的情绪状态，并能把不同的情绪状态区分开来。他们比其他人更强烈地体验到开心和不开心，情绪和感受是木行人生活中的重要组成部分。

阴面：傲慢孤僻

1. 个人主义

有创造力，他们可以不断出新点子，但有时比较个人主义，总是从自己的想法出发，不太考虑周围人的感受，也不太考虑这个点子是否适合企业、适合团队。木行人是如此的理想化，以至于他们习惯于以他们对事物的希望是怎样，便怎样去看待事物。他们易受理想化的人际关系的影响，易于忽视与他们认为相矛盾的事实。木行人没有学会面对自己无法认同的事实，结果他们忽视了自己的问题，而不是去寻找解决的方法。

2. 我行我素

木行人不十分受冷漠与批评的干扰，木行人更喜欢以自己的方式行事。面对相反意见，他们通常持怀疑态度，十分坚定和坚决。权威本身不能强制他们，只有他们认为这些规则对自己有用时才会去遵守。

3. 自卑羞怯

在人群中羞怯。有不自然的姿态，有强烈的自卑感。拙于发言，更不愿和陌生人交谈。凡是采取观望的态度，有时由于过分的自我意识，而忽视了社会环境中的重要事物与活动。

4. 刻板固执

因为木行人如此坚信自己的理想，所以他们常常忽视其他观点的作用，而且有时会很刻板。他们对于物质环境不十分有兴趣，他们经常很忙碌以至于没有注意周围正在发生的变化。

5. 不善分享

木行人对于一种想法的酝酿要比实际中开始一个计划所需要的时间长很多。他们内向，很难被人了解，但是愿意同自己信任的人分享内在的自我。他们往往有一个交往深厚、持久的小规模的朋友圈，在合适的氛围中能产生充分的个人热情和激情。

三、维度二：生发（创新成长--抗上不服）

核心定义：生发描述一个人的认知风格。这类人为了自身的成长对经验的前摄寻求和对知识的理解，以及对陌生情境的容忍和探索让他们充满了木行能量。他们属于好奇的、新颖的、非传统的以及有创造性的个体，偏爱抽象思维，兴趣广泛。

阳面：创新成长

1. 思想前卫

木行人喜欢挑战一切现有的理论，而予以新的评价，企图了解较前卫的思想与行为。尽可能广见多闻。愿意充实自己的生活经验。

2. 好奇心强

木行人好奇心强，而且善于观察，喜欢接触新的事物，比如去外面旅行、体验不同的经验。他们感觉千篇一律令人乏味，愿意去尝试寻求新异和多样性，被视为愿意尝试不同的活动、去新的地方、吃不寻常的食物等。

3. 想象丰富

木行人喜欢冒险和富有想象力的活动。他们对于从经历中，直接了解和感受的东西很感兴趣，常常富有艺术天赋和审美感。他们灵活易变、思维开阔。

4. 足智多谋

木行人是足智多谋、有独立见解的思考者。他们重视才智，对于个人能力有强烈的欲望，有能力也很感兴趣向他人挑战。他们的思考方式极端复杂，而且他们能更好地组织概念和想法而不是人。

5. 精于理论

他们天生精于理论，对于复杂而综合的概念运转灵活。他们是优秀的战略思想家，通常能清楚地看到任何局势的利处和缺陷。对于感兴趣的问题，他们是很好的、具有远见和见解的组织者。

阴面：抗上不服

1. 孤军奋战

因为木行人在工作中常常选择孤独，一心一意地努力，所以他们忽视了在活动中邀请别人参加或协助。征求别人的信息和建议会帮助他们在进程中早些认识到不切实际的想法，或者帮助他们在投人大量时间之前，做出一些改动和提高。

2. 充满怪想

由于不如木行人有条理性，他们有时会对事实判断错误，不能意识到自己的非逻辑性。当他们的梦想脱离现实时，其他人可能认为他们充满怪想、神神秘秘。木行人应该很好地向更现实的人请教，他们的理想在现实世界中是否可行和有用。

3. 对立敏感

木行人很情绪化地陷于自己的工作中，所以对于批评很敏感，更加复杂的是，当他们追求自己不可能有的高标准时，往往对自己的要求太多。即使事实上他们能够完成许多事情，但仍会导致情感上的不满足。当木行人失望时，往往对于他们周围所有的事物都很对立。

4. 不切实际

由于往往注意"思想"，木行人有时不切实际。木行人对于自己的思想如此坚定，以至于发展到眼光狭窄的地步。他们很顽固地对待变化，一旦决定已经做出，他们可能会拒绝改变。有时他们会忽略不支持他们立场的重要事实，或是拒绝与他们的价值观相冲突的看法。

5. 感情抽离

在感情上，木行人会表现得很抽离，像置身外来分析自己境况，从而化解感情上的冲动。木行人很少在读书时期谈恋爱，如果当时木行人的目标就是读书有成，就算情窦已开，也会自己警诫自己，谈恋爱会影响读书。因此，木行人给人逃避的感觉。其实他并非逃避感情，只是不会冲动，事事会以理性角度来处理。

四、维度三：舒畅腾上（正直不屈--偏执顶撞）

核心定义：木行能量腾上而无所止，他们在顺境时，给人的印象是正直不屈，而在逆境（事态不符合他们的观念与理解）时，表现为不顺从的，偏执顶撞。

阳面：正直不屈

1. 专注执着

木行人坚定、富有理想化。他们珍视正直，以至达到倔强的地步。木行人喜欢说服别人，使之相信他们的观点是正确的。通常木行人是深思熟虑的决策者，他们觉得问题使人兴奋，在行动之前他们通常要仔细地考虑。他们喜欢每次全神贯注于一件事情。

2. 讲求实效

他们是优秀的解决问题的人，这是因为他们能够掌握必要的事实情况，然后找到符合逻辑的明智的解决途径，而无需浪费大量的精力。他们能够理解晦涩的原则，在符合逻辑的基础上，而不是基于他们对事物的感受之上做出决定。

3. 有担当力

木行人正直有主意，有担当力，敢做敢当，遇事不盲从，有定见。

4. 心口如一

木行人心口如一，行为端正，能忍辱，常存悲天悯人之心，化俗救世之愿，舍己为人。

5. 超级理想

木行人渴望表达超级的理想。结果是，他们不与其他人有联系，只与哪些他们认为最卓越的，或能与他们分享理想的人有联系。这看起来他们好像在寻找出众的价值、最终的意义。

阴面：偏执顶撞

1. 严肃偏执

木行人会在意其他人的观点。尽管他看上去很平和和几乎没有任何情绪表现，一旦他的意见遭到反对，他可以是非常强硬的，甚至是无情的。木行人在人际关系交往中缺乏随意性，因此会让他人觉得有些冷淡，严肃甚至偏执。他们有时会延误耽搁，因为他太注重细节的分析，而不是行动。如果他可以少一些挑剔，学会放松，多与他人接触，他会变得更富有效率。

2. 过分理性

木行人过分依赖他们的条理性分析，所以他们会忽视与别人有关的事物。如果某一方面不符合逻辑，即使对于他们来说很重要，他们也会冒险抛弃。

3. 迟滞不前

木行人拒绝在整个计划中存在一点不符合逻辑的地方，所以他们在涉及计划某部分的一个细微错误之时迟滞不前，使整个方案难以完成。当他们把自己高度磨练过的理性性思考运用到周围的人身上时，他们的直率会变为一种无意的伤害。他们要学会在情感上与人互动，这个世界并不是只有逻辑。

4. 破坏规则

木行人喜欢挑战权威、常规和传统观念，他们不断检验社会的、政治的和宗教的价值观念。在极端状态下，会表现出对现存规则的敌意，同情那些打破现存法律的人，喜欢混乱、冲突和无序的状态。

5. 缺少耐心

木行人喜爱解决问题，所以往往他们会对常规的细节缺少耐心，而且如果一个计划需要过多的细节或持久贯彻，他们会失去兴趣而完成不了计划。

五、维度四：调达柔和（仁德善良--忧愁善变）

核心定义：木行能量以温柔为体，火可以烧它，土可以埋它，金可以削它，水可以淹它。木是五行中最柔和的，木行人在五行人中是最善良的。你和一位木行人在一起的感觉与与一位金行人、火行人在一起的感觉一定不同，你的内心感受得到他们的温暖善良，不带伤害。

阳面：仁德善良

1. 忠于内心

木行人把内在的和谐视为高于其他一切。他们敏感、理想化、忠诚，对于个人价值具有一种强烈的荣誉感，个人信仰坚定，有为自认为有价值的事业献身的精神。木行人精力集中于他们的梦想和想象。他们思维开阔、有好奇心和洞察力，常常具有出色的长远目光。在日常事务中，他们通常灵活多变、具有忍耐力和适应性。但是他们非常坚定地对待内心的忠诚，为自己设立了事实上几乎是不可能的标准。

2. 坦诚率真

木行人坦诚率真，他们十分坚定地完成自己所选择的事情，他们往往承担得太多，但不管怎样总要完成每件事。除了他们的价值观受到威胁外，他们总是避免冲突，没有兴趣强迫或支配别人。

3. 看透本质

木行人生活在思想的世界。他们是独立的、有独创性的思想家，即使面对怀疑，木行人仍相信自己的看法与决定。通常木行人具有本能的洞察力，能够看到事物更深层的含义。

4. 精神沟通

木行人很注重精神上的沟通，若与对方话不投机，或者没有可以共鸣的话题，他会很寡言，只会敷衍地说几句；若遇上有兴趣的话题，他会滔滔不绝，甚至主动找别人聊天，只是木行人这样做的目的更多是为了"吸收养料"。只要对方在简短几句话中有独特见地，就会吸引木行人对之产生兴趣。然而，当木行人不能在对方身上找到新知识，他就会冷淡下来。

5. 不求物质

木行人是个"清贫书生"型的人，不渴求物质，官能刺激，没有太大欲望，对衣食住行也不太重视，不追求享受，只追求知识，时常给人吝惜节俭之感。木行人的能量属于偏低，请人吃饭时，不会选择高级餐厅，送礼只是象征式，与人见只会打一声招呼，点头响应已代表一切。

阴面：忧愁善变

1. 缺乏控制

对于一个木行人来说，一个计划中有意思的部分是最初问题的解决和引出一些新内容。他们乐在一个问题最重要和富有挑战性的部分施展自己的灵感。这一阶段过后，他们常常会失去兴趣，缺乏完成已经开始的工作所必要的自我约束。他们很可能会开始许多计划，但完成的却寥寥无几。

2. 优柔寡断

木行人始终都注意着新的感官信息，喜欢开放面对所有可进行的选择，所以他们会优柔寡断。对于兴奋事物的需求使得他们很草率而易于厌烦。

3. 不切实际

由于有时对一些不切实际的高标准充满幻想，木行人会对自己和他人的期望过多。事实上，他们往往不在意自己如何符合别人的标准，重要的是自己。他们对于自己的行为如何影响他人缺乏理解，往往在提供改进意见时挑剔而直率。

4. 忽略现实

木行人过多地重视对未来的见解和想法，所以很容易忽略现在的重要事情和现实。他们也无法认识到自己思想中事实上的缺点，这会使他们的想法实施更加困难。收集所有相关的和真实的材料有助于确信他们的想法的可操作性。木行人需要简化自己理论性的、复杂的思想，这样才能把自己的想法传达给别人。

5. 幻想丰富

木行人是个很具幻想力的人，不但对感受敏感，也很想品尝沉浸在每一个感觉中。他们很戏剧化，会将内心的戏份做得很特别夸张。不过，他并非希望别人有同样感觉，只因自己喜欢这样做，木行人可以在白天与朋友聚会时，穿全黑的衣饰，一副赴丧事的打扮，令人侧目，其实他只想那一刻感受黑色，而非引人注意。不少木行人会觉得，感情不会完结，只会转化成一种关系。为了延续感情，木行人可能会将所爱的人衣物收起，放在枕边，晚上孤独一人时拿来嗅一嗅，幻想对方就在身边，其联想力之丰富非同一般。

六、维度五：内在成长（安静内向--疏远冷淡）

核心定义：小草的心声是什么？我长，我长！直至长成参天大树。草木的情怀是什么？只有长成参天大树，我才可以萌阴他人，释放我的仁爱。所以木的能量一直关注成长，他们所有的时间与精力都用来成长！没有心思和时间注意其他的事情。如果你爱木行人就让他（她）自由地成长，如果木行人爱你就希望你一同成长。

阳面：安静内向

1. 敏锐缜密

木行人对于细节有很强的记忆和判断。他们能够引证准确的事实支持自己的观点，把过去的经历运用到现在的决策中。他们重视和利用符合逻辑、客观的分析，以坚持不懈的态度准时地完成工作，并且总是安排有序，很有条理。

2. 外表平静

木行人外表平静、超然，但内心却专心致志于分析问题。他们喜欢有条理和目的的交谈，而且可能会仅仅为了高兴。木行人常常喜欢通过书写而不是口头来表达自己的感情。当木行人劝说别人相信他们的想法的重要性时，可能是最有说服力的。木行人很少显露强烈的感情，常常显得沉默而冷静。然而，一旦他们认识了，就会变得热情友好。木行人很友好，但也避免浮浅的交往。他们珍视那些花费时间去思考目标与价值的人。

3. 独立自主

木行人属于专家型的人才，他们胜任自己的专业技术，习惯在安静的环境里独立工作。木行人的计划里没有严格的时间表，没有重大的外在挑战（金钱与人事这类事情是不会在意的，甚至忽视他的创新是否实用及带来价值收益回报），也不在意成果是否对他人有影响，他完全沉浸在理想的创造乐趣中。

4. 思辨力强

他们喜欢抽象的概念，喜欢讨论理论性问题，喜欢解决复杂的智力问题。大部分表现为有求知欲的、善于分析的、理论定的。他们乐于钻研复杂的理论性问题，力求精通任何他们认为有趣的事物。

5. 逻辑分析

木行人做事是有条理和系统化的。木行人很有条理和分析能力，所以他们通常对要求推理和才智的任何事情都很擅长。他们力求精通整个体系，而不是简单地把它们作为现存的接受而已。木行人乐于完成一些需要解决的复杂问题，他们大胆地力求掌握使他们感兴趣的任何事情。木行人只有通过逻辑的推理才会确信一件事情。

阴面：疏远冷淡

1. 低交际性

木行人在社会情境中感觉不轻松，在人群中畏怯退缩、缺乏自信心，有不自然的姿态，或自卑感。拙于发言，更不愿和陌生人交谈。凡事采取观望的态度。

2. 离群索居

木行人倾向于悠闲和放松，但不一定懒惰。他们在生活工作中是慢节奏，不着急的、缓慢的、从容不迫的。他们宁愿躲在幕后，一般在人群中话很少，习惯让别人处于主导支配地位。

3. 孤寂寡言

木行人从小就有那么点儿不合群，喜欢一个人呆着。他们对人不热情、甚至冷漠，喜欢独自发呆。宁愿独自工作，不轻易放弃自己的主见。经常觉得世道艰辛，人生不如意事常八九，甚至沮丧悲观，时时有患得患失之感。自觉不容于人，也缺乏和人接近的勇气。

4. 热情不高

木行人不容易感受到各种积极的情绪，但并不意味着一定会感受到各种负面情绪。他们只是不那么容易兴奋起来，热情不高、平静的、冷淡的。但通常会被别人理解是对人疏远的、甚至是不友好的，带有敌意的。因为木行人相当冷淡，所以他们错误地推论别人也希望受到同样方式的对待。他们需要学习理解别人貌似"非理性"的感情，认可它们是合理可取的。这有助于防止他们疏远周围的人。

5. 孤独被动

木行人往往是孤独者，他不寻求甚至主动避免社会刺激。他们回避人群，感觉太闹，希望有更多的时间独处，有自己的个人空间。他们几乎对兴奋没有什么需要。木行人的这种特点有时会被人误认为是傲慢或者不友好，其实一旦和他接触你经常会发现他是一个非常和善的人。
