一、生命是时时刻刻值得欢庆的

1.1 核心定义

指导宗旨：缘起性空 现实问题：精神抑郁、烦恼多多、活着为了啥、丰满的肉体空洞的灵魂、无处安放的心 一句话定位：准确定位我们的能量属性（生命意义）

1.2 详细内容

理论根基：佛陀菩提树下悟道

当年佛陀在菩提树下悟道，实证到的生命真谛便是：一切众生皆具如来智慧德相。

佛法演进三乘：

  1. 小乘（声闻乘）：初讲四圣谛、十二因缘法，立位"人生是苦的"，重点修出离心而超越轮回，证阿罗汉
  2. 中乘（缘觉乘）：未证得明心见性、诸法实相
  3. 大乘（菩萨乘）：汉传佛教从明心见性起修，识本心、见本性，才能真正升起菩提心，心不住相勤修六度

佛陀本意：并非宣讲人生是苦的，当我们能够觉察并转化苦因时，就有能力离苦得乐。这种觉性在圣不增、在凡不减、不生不灭，本自圆成。

当今时代密乘心行：

  • 即一切色相、音声、意念的性空缘起
  • "修"即无时无刻不"自成本尊"
  • 即心即佛、即身即佛
  • 可节省许多事相，甚至不需刻意外求

1.3 综合落实行动

  1. 调整心态：坚强乐观面对生命，每一念都欢庆生命是努力目标
  2. 明白道理：明白世界运转的道理和自己人生真正的意义
  3. 学习觉察：在每一天每一刻让自己充满正能量并学会时时调频
  4. 微笑传递：用时刻微笑告诉自己和身边的人生命是值得欢庆的
  5. 提起觉知：无我相无人相无众生相是这个宇宙的真相，烦恼心、分别心自然转化
  6. 立在本质：时刻把自己的能量属性立在本质上，不被世间的妄念流所带着走
  7. 快乐奉献：无论学习、做事都以快乐、奉献的心态进行

1.4 核心金句

"一切众生皆具如来智慧德相，本自具足。"

"这种觉性在圣不增、在凡不减、不生不灭，本自圆成。"

"生命是时时刻刻值得欢庆的，我们愿意成为快乐的源头，让一切生命在觉性中绽放吧！"

二、无条件的陪伴成长

2.1 核心定义

指导宗旨：广发宏愿 现实问题：个人孤独、没有归属感 一句话定位：体现发心愿力，要成为没有判断、评价、分别的心，无条件的陪伴是每个人的需求

2.2 详细内容

三种活法

  1. 受业的一天：业力是未经转化的评判，评判的一天等于未经转化的一天，等于不快乐的一天，就只是活着受业而已
  2. 唤醒的一天：业力模式存在的意义是用来提醒我们，在我们的生命体验中还有一些评判等待我们去转化。业力是用来唤醒我们内在的觉知的
  3. "成为"的一天：无条件的陪伴成长，是从评判的幻象中走出，觉照到我们生命的内在实相，就是无条件的爱

陪伴的本质

"自始至终，没有别人需要你的陪伴，这个人就是你自己。与其说陪伴别人，不如说陪伴自己，安住在脱离二元性的体验之中。"

陪伴修的是禅定：绽放自己，同众生相应。本身就是众生一体。去掉分别心，加重理解心。允许自己一点一点觉察，一点一点改正。

2.3 综合落实行动

  1. 聆听陪伴：不要着急去为对方解决问题，聆听就是最好的陪伴
  2. 常说你是对的：接纳对方的现状
  3. 无分别心：对身边的人没有分别心，不批评，不评判
  4. 无条件付出：付出不讲条件和回报
  5. 主动帮助：对他人的困难主动帮助，帮助影响周围的同修，共同进步
  6. 学习小组：建立学习小组共同学习和进步
  7. 接受本貌：允许一切自然的发生，接受他人如实的本貌
  8. 耐心恒久：凡对人对事都要耐心恒久陪伴

2.4 核心金句

"无条件的陪伴成长，可以看做是我们的内心发出的一个呼唤，让自己从评判的幻象中走出。"

"自始至终，没有别人需要你的陪伴，这个人就是你自己。"

"常说你说的对。"

三、生命提升的功课是勇于补短板

3.1 核心定义

指导宗旨：信解行证 现实问题：心理断奶期、眼高手低、不敢突破、知难而退、勇气不足 一句话定位：纵向提升的必备条件是补短

3.2 详细内容

生命功课的下手处

改变自己的旅途从哪里开始呢？生命功课的下手处应是勇于面对自己能力的不足。

核心原则：

  • 离开心理舒适区，挑战一切不可能
  • 没有条件创造条件
  • 遇到不熟悉、不擅长的事情，第一念是"Try it"（尝试做），而不是"Leave it"（推出去）

"生命的精彩，往往来自于突破自己的那一刻！"

成长的态度

  • 成长就是对于不熟悉、不擅长、不习惯的事情要勇于承担
  • 在商业社会这也许会妨碍你的晋升
  • 可是在一个彼此唤醒的团队之中，你不懂不是羞耻，而是成长点

3.3 综合落实行动

  1. 知难而进：多吃苦
  2. 放下面子：多主动
  3. 提起觉知：哪些是自己抗拒的、排斥的、想逃避的，哪些正是你生命成长的契机
  4. 验证本自具足：对承担的工作是验证自己本自具足，也是对相信所有人本自具足的验证
  5. 勇于请教：对于不擅长的事情要勇于向擅长的人请教
  6. 挑战短处：在组织中勇敢挑战自己的短处用心做事

3.4 核心金句

"生命的精彩，往往来自于突破自己的那一刻！"

"Try it（尝试做），而不是 Leave it（推出去）。"

"你不懂不是羞耻，而是成长点。"

四、性命双修，福慧具足

4.1 核心定义

指导宗旨：性命双修，福慧具足 现实问题：在现实与理想中徘徊（行动力不够）、理想很丰满现实很骨感、智者乏善或力者乏智 一句话定位：在爱心中唤醒内在智慧、行动就是福！

4.2 详细内容

知行合一的三层境界

  1. 见山是山：见一切现象
  2. 见山不是山：悟到本质
  3. 见山还是山：回归现象即本质

两种失衡状态

  • 智者乏善：会想不愿干
  • 力者乏智：会干不愿想

解决方案：知行合一，在行动中体察内心，大家都是自己的一面镜子。

4.3 综合落实行动

  1. 避免假把式：避免光说不练假把式
  2. 带着觉察：带着爱和觉察去作人做事，越干越快乐
  3. 发挥优势：发挥自身优势多奉献
  4. 完成生命：不是在完成一个个目标，而是借由外在的工作来完成生命本身
  5. 觉察初心：行动中经常提醒自己：为什么而做？时刻觉察做这份工作的初心
  6. 执行者收获：承担后发现做一个执行者和旁观者，所得到的收获相差甚远
  7. 发心与技巧：好的发心固然重要，操作中的技巧同样不可忽略
  8. 少说多做：以行动实践，少说多做
  9. 福慧双修：提升自身的综合能力

4.4 核心金句

"见山是山，见山不是山，见山还是山。"

"在爱心中唤醒内在智慧、行动就是福！"

老子智慧

"前识，是道的浮华。"

惯性思维（前识）：又称所知障，是无明最大的帮凶，自以为是的执着使得我们无法认识真相。

国学智慧的核心

国学智慧的核心内涵是"心文化"，并贯穿易、医、儒、道、禅诸家，体认中国传统文化在当代的修心价值。

组织应用：用国学智慧的高维引领，给未来组织赋予"心"的力量。

6.3 综合落实行动

  5. 寻求帮助：自己处理不了负面情绪时，积极主动地向团队其他成员寻求陪伴与帮助

七、一切问题都是自己的问题

7.1 核心定义

指导宗旨：修行是内省内证内悟 现实问题：用做事的心从事工作、打乱组织伦理 一句话定位：团队融合，减少内耗

7.2 详细内容

儒家文化的启示

儒家文化强调内省内证内悟，他们立位于现实世界，却由内而外地发现了这个世界的"内圣外王"创建模式。

对于团队建设的启示：

  • 多元文化的团队融合
  • 减少内耗的思维模式

谦卑的德行

但凡有德行的人，都有一个共同的特征，那就是谦卑。

"用易经的'谦'卦形容，谦谦君子，卑以自牧。"

自牧：向内寻找自己，承认一切问题都是自己的问题。

警示：多疑的人、为自己开脱和辩解的人，损自己的福报。

7.3 综合落实行动

  1. 不评判不指责：不评判，不指责
  2. 每日三省：坚持每日三省吾身及时调整自己的误区
  3. 改变自我：不带有色眼镜看人，知道一切镜像都是自己内心的映射，尽量改变自己
  4. 勇于担当：勇于承认错误，勇于担当责任
  5. 先尽力再求助：自己分内的工作先自己尽努力解决，解决不了再寻求帮助

八、在差异面前自觉转化

8.1 核心定义

指导宗旨：烦恼即菩提 现实问题：一言不合就翻脸、意见不同就清理、没有工作伦理 一句话定位：构建以修行为内核的组织工作伦理

8.2 详细内容

工作场景问题

面对不同意见时咋办？看不惯别人（同一个组织里）又不知道如何转化？

常见错误方法

  • 一言不合就翻脸
  • 意见不同就清理

后果：这些常见的方法不仅不可能转化我们的烦恼，只可能蒙蔽我们的觉察。

"就好像是自己与空气搏击一样，结果发现谁都没有打到，只是自己在那里凭空发力而已。"

转化的智慧

仔细向内觉察，会发现情绪是可以转化的，愤怒的能量也可以转化为慈悲。

核心能力：学会与不同意见和谐相处，坦然面对不同意见的人和事。

8.3 综合落实行动

  1. 不被裹挟：不被烦恼裹挟
  2. 找准位子：在组织中找准位子，适时学习表达，做好应该事宜
  3. 提升生命：以提升自己生命成长为目的，不随境转也不执着于转他（它）
  4. 尊重协调：尊重差异，协调差异，减少差异，彼此融合
  5. 理解宗旨：理解组织的修行宗旨与指导思想，凡是不要看表面功夫
  6. 超越建议：若建议请超越

8.4 核心金句

"烦恼即菩提。"

"情绪是可以转化的，愤怒的能量也可以转化为慈悲。"

"构建以生命成长为内核的组织工作伦理。"

九、看透问题本质后再行动，智慧引领行动

9.1 核心定义

指导宗旨：大道至简 现实问题：一团乱麻、浮躁、慌手忙脚做事情 一句话定位：价值与工具的双轮驱动，对应的是未来的组织形态

9.2 详细内容

大道至简的智慧

"大道至简，直击本源。"

相信自己：

  • 你就是爱，你就是光！
  • 你是爱哪里会有冷漠？
  • 你是光哪里会有黑暗？

用国学智慧引领组织：看透问题本质后再行动。

9.3 综合落实行动
