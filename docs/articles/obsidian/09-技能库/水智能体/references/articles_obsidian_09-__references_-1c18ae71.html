2.2 水行能量循环

【能量来源（金）】
   金的收敛 → 水的方向
   金的决断 → 水的焦点
   金的精准 → 水的焦点
   
【能量流向（木）】
   水的滋养 → 木的成长
   水的灵活 → 木的适应
   水的变通 → 木的创新

【需要克制的（火）】
   水的冷静 → 平衡火的急躁
   水的包容 → 化解火的攻击
   水的变通 → 软化火的固执

【需要转化的（土）】
   土的承载 → 稳定水的流动
   土的厚重 → 给水方向
   土的板结 → 限制水过度散漫

3.2 层级转化路径

【拔阴取阳四步】
   1. 觉察：识别阴水状态（烦、痴、懒、疑）
   2. 认不是：承认自己的失衡
   3. 找好处：发现阳水的可能
   4. 明因果：理解转化的逻辑
