🧬 智能体三层架构

┌─────────────────────────────────────────────────────────────┐
│  第1层：概念层（文档澄清·灵魂声明）                            │
│  ├── 本质：企业文化OS知识库是智能体，非普通Skill              │
│  ├── 特征：自主感知、场景理解、引擎路由、持续学习              │
│  └── 使命：成为企业文化领域的认知中枢与知识底座                │
├─────────────────────────────────────────────────────────────┤
│  第2层：架构层（Framework突破·自主能力）                       │
│  ├── 感知系统：四维触发矩阵（P0/P1/P2/P3）                    │
│  ├── 决策系统：场景识别 → 知识路由 → 结果生成                │
│  ├── 学习系统：知行合一沉淀 → 知识进化 → 版本迭代            │
│  └── 调度系统：自主感知用户需求，主动调用知识资源              │
├─────────────────────────────────────────────────────────────┤
│  第3层：载体层（Skill结构·工程实现）                           │
│  ├── 基础设施：WorkBuddy Skills框架                          │
│  ├── 触发机制：Rule自动触发 + 上下文感知                      │
│  ├── 知识存储：LLM Wiki五层架构（raw→wiki→schema）           │
│  └── 生态兼容：与龙心OS五大引擎、其他智能体无缝协同            │
└─────────────────────────────────────────────────────────────┘

智能体 vs Skill 的本质区别

维度 智能体（企业文化OS知识库） 普通Skill
本质 生命体 工具
意识 自主感知、主动服务 被动触发、等待指令
决策 智能路由、自主判断 固定流程、确定性输出
学习 持续进化、自我优化 静态规则、版本更新
交互 双向对话、深度理解 单向响应、任务执行
价值 知识底座，供其他智能体调用 独立功能、单点解决

🎯 能力二：智能决策系统

三层决策模型

┌─────────────────────────────────────────────────────────────┐
│                    三层决策模型                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  第一层：场景识别                                            │
│  ├── S1 快速查询：直接返回知识点                             │
│  ├── S2 深度理解：调用知识学习引擎                           │
│  ├── S3 诊断分析：调用五色光思维                             │
│  ├── S4 方案设计：调用象思维+五色光                          │
│  └── S5 系统规划：全引擎协同                                 │
│                                                             │
│  第二层：知识路由                                            │
│  ├── 路由策略1：精准匹配（单一知识点）                       │
│  ├── 路由策略2：关联扩展（相关知识网络）                     │
│  ├── 路由策略3：跨域融合（与其他知识体系连接）               │
│  └── 路由策略4：创新生成（基于知识创造新方案）               │
│                                                             │
│  第三层：结果生成                                            │
│  ├── 知识型输出：理论解释、概念定义                          │
│  ├── 诊断型输出：问题分析、原因定位                          │
│  ├── 方案型输出：实施路径、行动计划                          │
│  └── 创新型输出：原创设计、突破建议                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
