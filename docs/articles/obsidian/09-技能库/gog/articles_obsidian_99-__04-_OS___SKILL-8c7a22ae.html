gog - Google Workspace集成 Skills

功能定位

L6层·专业场景工具 - 连接Google Workspace生态（Gmail/Calendar/Drive），实现日程、邮件、文档的自动化管理。

核心能力

邮件管理（Gmail）

  • 邮件读取与发送
  • 智能分类与标签
  • 自动回复草稿生成
  • 附件管理

日程管理（Calendar）

  • 创建/修改/删除事件
  • 日程冲突检测
  • 会议邀请自动处理
  • 日程摘要生成

文档管理（Drive）

  • 文件搜索与读取
  • 文档创建与更新
  • 权限管理
  • 版本历史

使用场景

自动化工作流

  • 每周五下午汇总本周工作 → 生成格式化邮件
  • 会议结束后 → 自动生成会议纪要并存入Drive
  • 新邮件触发 → 自动分类并生成待办事项

日程助手

  • "明天下午3点提醒我开会" → 自动创建日历事件
  • "查看今天的日程" → 返回日程摘要
  • "安排一个1小时的项目评审会议" → 查找空闲时间并创建

邮件助手

  • "给团队发一封周报邮件" → 自动生成并发送
  • "总结一下这周的邮件" → 分类汇总关键信息

MCP配置

如需启用MCP方式，添加配置：

{
  "mcpServers": {
    "gog": {
      "command": "npx",
      "args": ["-y", "mcp-gog"]
    }
  }
}

注意：需要Google账号授权OAuth认证

  • 与🔄知行合一协同：工作记录→自动归档→知识沉淀
