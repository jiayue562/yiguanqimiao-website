📡 调用协议

请求格式

调用请求:
  from: "龙心OS"
  to: "龙脑OS"
  
  必需参数:
    scene_type: "场景类型（S0-S9）"
    user_need: "用户需求描述"
    
  可选参数:
    target_knowledge_base: "目标知识库（思维模型库/五行人格/麦肯锡框架）"
    specific_model: "具体模型（如SWOT、波特五力等）"
    combination_needed: "是否需要组合（布尔值）"
    depth_level: "分析深度（快速/标准/深度）"
    
  上下文信息:
    conversation_history: "对话历史"
    user_preferences: "用户偏好"
    previous_analysis: "之前的分析结果"

响应格式

调用响应:
  from: "龙脑OS"
  to: "龙心OS"
  
  核心结果:
    knowledge_base: "调用的知识库名称"
    model_used: "使用的具体模型"
    analysis_result: "分析结果（结构化数据）"
    
  建议输出:
    recommendations: "行动建议列表"
    next_steps: "后续步骤建议"
    related_models: "相关模型推荐"
    
  元数据:
    execution_time: "执行时间"
    confidence_score: "置信度评分"
    feedback_request: "是否需要反馈"

🎯 场景路由示例

示例1：商业分析场景

场景:
  user_input: "用SWOT分析一下我的副业项目"
  scene_type: "S4分析决策"

龙心OS决策:
  need_knowledge_base: true
  target: "龙脑OS"

龙脑OS路由:
  knowledge_base: "思维模型库"
  model: "SWOT分析"
  
执行结果:
  analysis_result: "SWOT四象限分析"
  recommendations: "基于分析的行动建议"

示例2：人格分析场景

场景:
  user_input: "分析一下我的五行特征"
  scene_type: "S2深度理解"

龙心OS决策:
  need_knowledge_base: true
  target: "龙脑OS"

龙脑OS路由:
  knowledge_base: "五行人格心理学OS"
  model: "五行总智能体"
  
执行结果:

  recommendations: "人格转化建议"

示例3：多知识库组合

场景:
  user_input: "我要做一个重大决策，需要全面分析"
  scene_type: "S5重大决策"

龙心OS决策:
  need_knowledge_base: true
  target: "龙脑OS"
  combination: true

龙脑OS路由:
  knowledge_bases: 
    - "思维模型库"
    - "五行人格心理学OS"
  models:
    - "SWOT分析"
    - "五色光思维"
    - "五行分析"
  
执行结果:
  analysis_result: "多维度综合分析报告"
  recommendations: "综合决策建议"

场景不匹配 SC400 返回推荐场景 "当前场景不匹配，推荐场景：..."
参数缺失 PM400 返回必需参数列表 "缺少必需参数：..."
执行超时 EX504 返回部分结果 "分析超时，返回部分结果：..."
