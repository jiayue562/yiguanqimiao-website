CLI指令解析Skill v1.0

🎯 技能定位

CLI指令解析是AI OS的原生操作入口，负责：

  • 解析CLI指令前缀参数
  • 提取思维模型/价值观/人格特质
  • 让AI的执行结果贴合个人特质

核心定位：CLI是人向AI传递指令、AI操作全系统软硬件的标准化原生入口 共生价值：CLI实现"人少操作、AI多干活"，人从繁琐的操作中解放，专注于创意、决策 个性化价值：将个人的思维模型、价值观封装为CLI指令的"前置参数"，让AI的所有执行都基于人的核心认知

📋 CLI指令分类体系

核心决策指令（人发出）

人向AI发出高层决策指令，AI自动拆解为底层CLI操作指令执行

格式示例

  • "基于[思维模型]分析[主题]"
  • "按[价值观]视角生成[内容]"
  • "从[五行人格]角度解读[文本]"
  • "用[人格特质]风格撰写[文案]"

实际案例

  • "基于我的GEO思维模型分析AI行业趋势"
  • "按长期主义视角生成企业文化方案"
  • "从五行识人角度解读员工行为模式"
  • "用偏理性的逻辑分析问题"

执行操作指令（AI自主调用）

AI根据人的高层决策指令，自主调用底层CLI工具执行

格式示例

  • "检索[知识库标签]相关笔记"
  • "运行[技能名称]处理[任务]"
  • "调用[底层工具]执行[操作]"

实际案例

  • "检索'五行识人'相关笔记"
  • "运行'五行人格分析Skill'处理员工行为模式"
  • "调用Obsidian API检索大圆满见地相关内容"

🔄 CLI指令的人格化适配

思维模型前缀

第一性原理

  • 指令："按第一性原理分析[主题]"
  • 解析：提取"第一性原理"参数，确保分析从最根本的原理出发
  • 应用场景：战略分析、问题诊断、创新突破

长期主义

  • 指令："按长期主义视角生成[内容]"
  • 解析：提取"长期主义"参数，确保内容关注长期价值而非短期利益
  • 应用场景：企业文化、战略规划、投资决策

GEO思维模型

  • 指令："基于我的GEO思维模型分析[主题]"
  • 解析：提取"GEO思维模型"参数，确保分析遵循GEO框架
  • 应用场景：企业战略、组织发展、团队建设

象思维

  • 指令："按象思维0→1突破[问题]"
  • 解析：提取"象思维"参数，确保突破性创新而非渐进式改进
  • 应用场景：创新突破、原创设计、颠覆性想法

人格特质前缀

偏理性逻辑

  • 指令："用偏理性的逻辑分析[问题]"
  • 解析：提取"偏理性"参数，确保分析逻辑清晰、数据支撑
  • 应用场景：数据分析、问题诊断、技术方案

生活化语言

  • 指令："用生活化的语言总结[内容]"
  • 解析：提取"生活化"参数，确保总结通俗易懂、贴近生活
  • 应用场景：知识传播、培训讲解、通俗表达

五行识人视角

  • 指令："从五行识人角度解读[文本]"
  • 解析：提取"五行识人"参数，确保解读基于五行人格理论
  • 应用场景：人格分析、关系调和、团队管理

信仰文化前缀

大圆满见地

  • 指令："按大圆满见地解读[文本]"
  • 解析：提取"大圆满"参数，确保解读基于"本来清净与本自圆满不二"
  • 应用场景：修行体悟、心性解读、文化创作

儒家文化

  • 指令："从儒家文化角度分析[问题]"
  • 解析：提取"儒家文化"参数，确保分析基于儒家思想（仁义礼智信）
  • 应用场景：企业文化、伦理决策、领导力

五行识人

  • 指令："按五行识人理论分析[人格]"
  • 解析：提取"五行识人"参数，确保分析基于五行生克关系
  • 应用场景：人格分析、团队建设、关系调和

🎯 核心功能

功能1：解析CLI指令前缀参数

触发：收到CLI指令（带前缀）

执行流程：

  1. 识别CLI指令中的前缀参数
  2. 提取思维模型/价值观/人格特质/信仰文化参数
  3. 将参数传递给Framework调度模块
  4. 让所有后续执行都携带这些个性化参数

输出格式：

功能2：构建个性化参数包

触发：解析CLI指令后，自动构建个性化参数包

  3. 将个性化参数包传递给MCP协议
  4. 确保全系统执行都携带个人印记
