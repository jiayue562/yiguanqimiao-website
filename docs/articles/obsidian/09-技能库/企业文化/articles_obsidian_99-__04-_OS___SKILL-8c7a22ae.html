📖 核心定义

What（是什么）

企业文化.skills 是一套系统化的企业文化建设工具，帮助企业从四个维度构建完整文化体系：

  • 精神层: 使命、愿景、价值观、经营哲学
  • 制度层: 管理制度、行为规范、组织架构
  • 物质层: 企业标识、办公环境、视觉系统
  • 行为层: 领导行为、员工行为、仪式活动

Why（为什么）

企业文化是企业的灵魂，是组织发展的软实力支撑：

  • ✅ 方向指引: 明确企业存在的意义和发展的方向
  • ✅ 凝聚力量: 统一员工思想，增强团队凝聚力
  • ✅ 行为规范: 明确行为准则，降低管理成本
  • ✅ 品牌塑造: 形成独特的企业形象和市场认知
  • ✅ 传承发展: 确保文化代代相传，企业持续发展

How（怎么做）

一句话工作流:

需求诊断 → 阶段判断 → 模块选择 → 内容生成 → 落地设计 → 持续进化

八大核心模块:

  1. 融入时代 - 8个观的确立（文化/时代/行业/企业/工作/成功/生活/家庭）
  2. 经营心法 - 12个经营要素（使命/愿景/价值观/产品/组织/能力/特征/人员/发展/思维/方法/格言）
  3. 氛围营造 - 8个氛围要素（标识/手势/誓词/歌曲/舞蹈/旗帜/纪念日/着装）
  4. 礼乐仪式 - 6种仪式设计（宣誓/会议/入职转正/干部聘任/拜师/礼乐）
  5. 功勋体系 - 3级荣誉体系（公司级/品牌级/店面级）
  6. 企业商学院 - 系统化培训体系
  7. 企业大事件 - 文化故事挖掘与记录
  8. 附录工具 - 工具模板汇总

🎯 使用流程

标准作业流程（SOP）

Step 1: 需求诊断（5分钟）
  ├─ 收集企业基本信息
  ├─ 识别文化建设痛点
  └─ 确定建设目标

  ├─ 判断企业发展阶段
  ├─ 确定建设优先级
  └─ 制定时间计划

  ├─ 根据需求选择核心模块
  ├─ 确定模块执行顺序
  └─ 分配资源预算

Step 5: 落地设计（15分钟）
  ├─ 设计落地路径
  ├─ 制定实施计划
  └─ 建立评估机制

🧠 核心算法

决策点1: 企业发展阶段判断

算法: 企业发展阶段判断
输入: 成立时间, 员工人数, 营收规模

规则:
  初创期:
    条件: 成立<2年 AND 员工<50
    特征: 生存导向,快速迭代
    文化重点: 使命愿景确立,创始人文化
  
  成长期:
    条件: (2年≤成立<5年) AND (50≤员工<200)
    特征: 扩张导向,规范化需求
    文化重点: 文化体系搭建,价值观传播
  
  成熟期:
    条件: (5年≤成立<10年) AND (200≤员工<1000)
    特征: 稳定导向,效率优化
    文化重点: 文化固化传承,精细化运营
  
  转型期:
    条件: 成立≥10年 OR 业务转型中
    特征: 变革导向,突破创新
    文化重点: 文化变革升级,二次创业

输出: 发展阶段 + 文化重点建议

决策点2: 文化建设优先级判断

算法: 优先级判断
输入: 核心痛点, 发展阶段

规则:
  IF 痛点 == "文化模糊":
    优先模块: [经营心法]
    原因: 需要先明确使命愿景价值观
  
  ELSE IF 痛点 == "执行困难":
    优先模块: [礼乐仪式, 氛围营造]
    原因: 需要通过仪式和氛围固化行为
  
  ELSE IF 痛点 == "传承断裂":
    优先模块: [企业商学院, 功勋体系]
    原因: 需要建立培训和激励体系
  
  ELSE IF 痛点 == "时代脱节":
    优先模块: [融入时代]
    原因: 需要重新与时代同频
  
  ELSE:
    按标准顺序: [融入时代, 经营心法, 氛围营造, 礼乐仪式, 功勋体系, 企业商学院]

输出: 优先模块列表 + 执行顺序

决策点3: 模块执行顺序决策

算法: 执行顺序决策
输入: 发展阶段, 优先模块

标准顺序:
  1. 融入时代（基础认知）
  2. 经营心法（核心内容）
  3. 氛围营造（环境建设）
  4. 礼乐仪式（行为固化）
  5. 功勋体系（激励机制）
  6. 企业商学院（传承体系）
  7. 企业大事件（历史记录）

特殊调整:
  初创期: 可跳过融入时代，直接经营心法
  变革期: 融入时代 + 经营心法同步进行
  传承期: 优先功勋体系 + 企业商学院

输出: 定制化执行顺序

决策点4: 内容生成策略

算法: 内容生成策略
输入: 行业类型, 地域文化, 创始人背景

行业适配:
  餐饮:
    强调: 服务文化, 团队文化, 食品安全文化
    关键词: 顾客至上, 团队协作, 品质第一
  
  科技:
    强调: 创新文化, 开放文化, 快速迭代
    关键词: 拥抱变化, 用户第一, 数据驱动
  
  制造:
    强调: 精益文化, 品质文化, 工匠精神
    关键词: 精益求精, 质量至上, 持续改善
  
  服务:
    强调: 客户文化, 专业文化, 体验至上
    关键词: 客户成功, 专业服务, 超越期望

地域文化融入:
  北方: 豪爽、直接、重情义
  南方: 精细、务实、重契约
  沿海: 开放、创新、国际化
  内陆: 稳健、传统、本土化

创始人影响:
  技术背景: 强调专业、创新、工匠精神
  销售背景: 强调客户、结果、狼性文化
  管理背景: 强调规范、体系、流程文化
  创意背景: 强调创新、自由、人文文化

输出: 定制化内容方向
