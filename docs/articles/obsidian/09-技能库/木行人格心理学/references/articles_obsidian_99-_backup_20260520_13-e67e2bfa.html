核心节点详解

节点一：木行人生克规律（根节点）

属性：核心知识体系 定义：理解木行人生命状态、人际关系与命运起伏的动态法则 连接：

  • ← 五行人格心理学总智能体（归属）
  • ← 象思维·曲直（理论基础）
  • ← 王凤仪化性谈（传统来源）
  • ← 中医五行（身心同调）
  • → 阳木·仁德（健康方向）
  • → 阴木·傲慢（失衡方向）
  • → 水生木（生关系1）
  • → 木生火（生关系2）
  • → 金克木（克关系1）
  • → 木克土（克关系2）

节点二：阳木·仁德之木（健康状态）

属性：A层·能量层（正向） 定义：曲直和谐统一的木行能量状态 核心特质：

  • 仁慈善良（仁）
  • 正直担当（义）
  • 灵活创新（智）
  • 生发有道（信）

连接：

  • ← 一心·仁德本源（本源）
  • ← 水生木畅（滋养条件）
  • ← 金克木成（规范条件）
  • → 木生火旺（转化方向）
  • → 木疏土活（互动方式）
  • → 九层阶梯·1-3级（发展层级）

象征：🌳 根深叶茂，既能独自参天，又能荫庇众生

节点三：阴木·傲慢之木（失衡状态）

属性：A层·能量层（负向） 定义：曲直分离对立的木行能量状态 核心特质：

  • 固执顶撞（曲而不直）
  • 抗上不服（直而不曲）
  • 忧愁善变（曲直失衡）
  • 僵化偏执（失去曲直）

连接：

  • ← 一心·傲慢遮蔽（根源）
  • ← 水弱木枯（滋养不足）
  • ← 金强木折（压制过度）
  • ← 木旺克土（发泄方向）
  • → 拔阴取阳（转化路径）
  • → 九层阶梯·4-9级（发展层级）

象征：🌵 干枯扭曲，刺多而无荫

转化技术体系

            │                                 │
            ├─ 心界转化                       ├─ 遇事不怒
            │   "阴木怒转阳木学"              │   反求诸己
            │                                 │
            └─ 灵界转化                       └─ 己欲立而立人
                "阴木傲慢转阳木仁"                己欲达而达人

核心路径：

  1. 觉察：识别阴木信号（身体/思维）
  2. 暂停：深呼吸，创造转化空间
  3. 转化：运用口诀重构认知
  4. 行动：采取建设性行为

连接：

  • ← 阴木·傲慢（转化对象）
  • → 阳木·仁德（转化目标）
  • → 一心·仁德本源（根本）
