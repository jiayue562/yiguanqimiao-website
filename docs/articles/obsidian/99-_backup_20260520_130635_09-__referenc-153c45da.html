步骤 操作 核心问题
1. 觉察 意识到对变革的本能抗拒 "我在抗拒什么？"

4. 感受 感受那份温暖和期待 "我有什么感受？"

3. 点燃：想象成功后的场景——团队业绩提升...
4. 感受：那份温暖和期待
5. 转化：主动思考"我如何支持这个愿景的实现"

3. 提炼 用"金"的智慧整理和提炼 "这件事的核心是什么？"
4. 放水 允许"水"流动 "我允许什么流动？"
5. 整合 形成新的平衡 "新的平衡点在哪里？"

情境：面对变化感到僵化和恐惧

转化过程：

1. 觉察：意识到自己不想改变，想维持现状
2. 接纳：承认自己害怕未知
3. 提炼：问"这件事最核心的价值是什么？"
4. 放水：允许自己尝试一个小改变
5. 整合：找到"稳定"与"变化"的平衡
