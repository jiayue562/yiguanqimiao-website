📋 核心定义

拔阴取阳

定义：五行人格心理学中的核心转化技术，指将个体从阴性状态（不走运）向阳性状态（走运）转化的系统化方法。

核心四步法：

  1. 认不是：认识到自己的错误和问题
  2. 找好处：发现他人的优点和长处（金行人专属）
  3. 信因果：明白因果规律，不怨天尤人（土行人专属）
  4. 达天时：顺应自然规律，达到天人合一（火行人专属）

本质：不是消除五行，而是超越对五行的执着，实现能量的纯净流通。

五行病诊断

定义：根据五行阴阳属性诊断个人的能量状态（走运/不走运）的诊断系统。

诊断链条：

内在想法 → 外在行为 → 命运结果

五行病诊断表格：

土性病 好怨人 被欺人 受累
金性病 好分辩 伤人 受贫

三界生命模型

定义：从身界、心界、灵界三个维度观察和判断个人能量状态（阴阳属性）的生命架构。

三界对应关系：

  • 身界（物质体）：身体形态、健康状况
  • 心界（能量体）：情绪、心态、性格
  • 灵界（信息体）：直觉、灵感、价值观

诊断方法：

  • 从身界看：观察身体状况和形态（阴木=驼背僵硬，阳木=挺拔健壮）
  • 从心界看：审视自己的情绪和心态（阴木=忧愁沉闷，阳木=积极向上）
  • 从灵界看：观察自己的直觉、灵感和创造力（阴木=魂不守舍，阳木=直觉敏锐）

一、五行阴阳属性分类（1-34行）

1. 五行人的阴阳属性（1-8行）

核心概念：

  • 五行人的阴阳属性，是一个深入且富有智慧的分类系统
  • 每个人可以根据自己的五行属性（木、火、土、金、水）以及阴阳属性（阳或阴）来定位自己

阴阳属性的影响：

  • 阳性（走运）：积极、活跃、外向
  • 阴性（不走运）：消极、沉静、内向

五行阴阳的十种人型：

  1. 阳木人：积极向上、充满活力、善于创新
  2. 阴木人：犹豫不决、缺乏自信、需要支持和鼓励
  3. 阳火人：热情洋溢、充满活力、善于领导
  4. 阴火人：过于急躁、缺乏耐心、需要控制情绪
  5. 阳土人：脚踏实地、沉稳可靠
  6. 阴土人：沉闷多怨、缺乏自信、少言寡语
  7. 阳金人：身体健康、免疫力强
  8. 阴金人：体质较弱、易感冒、需要注重保养
  9. 阳水人：智慧深邃、情感丰富、善于沟通
  10. 阴水人：压抑悲观、固执僵化、缺乏动力

2. 精气神三层结构也分阴阳（9-20行）

三层生命模型：

  • 精身界：身体层面的阴阳状态
  • 气心界：情绪、心态层面的阴阳状态
  • 神灵界：直觉、灵感层面的阴阳状态

判断走运/不走运的方法：

  • 从身界看：观察身体状况和形态
  • 从心界看：审视自己的情绪和心态
  • 从灵界看：观察自己的直觉、灵感和创造力

3. 怎么来看自己走运还是不走运（15-27行）

判断标准：

  • 身界：观察身体状态（阴木=驼背僵硬，阳木=挺拔健壮）
  • 心界：审视情绪状态（阴木=忧愁沉闷，阳木=积极向上）
  • 灵界：观察直觉和灵感（阴木=魂不守舍，阳木=直觉敏锐）

4. 了解阴阳分类的目的是什么（21-27行）

核心目的：实现个人的能量提升和运势转变。

"拔阴取阳"的本质：

  • 不仅是一个理论概念，更是一个实际操作的过程
  • 要求我们深入观察自己，识别出那些阻碍自身成长和幸福的阴性特质
  • 然后积极主动地寻求改变和提升

最终目标：

  • 实现五行能量的全面优化和提升
  • 使每个人都能在生活中展现出最佳的状态，充满正能量

5. 一步改运，两步改命（28-30行）

第一步改运：

  • 将阴性的能量都拔掉，阳的能量才能显现出来
  • 五行才能圆转，才有可能真的改命
  • 短期目标：不管哪种五行，身心灵三层面都先应该是阳的能量

第二步改命：

  • 如果已经是阳木阳火阳土阳金阳水的时候，向你的先天挑战
  • 换一个活法，活出两颗星的质量来，最终活出五颗星的质量来

6. 悟到心的本质（30-33行）

核心观点：

  • 木、火、土、金、水这五颗心，是从高维能量投射出来的
  • 心的本质和心在我们身体上的表现是一回事
  • 我们的生命（身体、健康、养生、情绪、关系）都是心在我们身体上的表现形式（投影象）

修行方法：

  • 慢慢把它倒回去，倒回到我们心的本质（投影源头）
  • 在这个地方进行开心转运

  • 五行具体在每个人的心性中不是单一的，而是五行具有，但各有所偏，主次各异

核心观点：

  • 相由心生，相由心灭，心恶相亦恶，心善相亦善

转化方法：

  • 一转改运（在基本人格范围内，负面能量向正面能量转化）

二、五行人拔阴取阳通关点（82-99行）

1. 木行人拔阴取阳通关点（82-83行）

通关方法：木行人要拔阴取阳，阴木转成阳木，再阳木生出阳火来。

核心操作：

  • 木如何能生火？那就是要忍辱
  • 忍辱：就是要忍住自己的脾气，不要随时想发就发
  • 不可以随时发脾气不控制自己的情绪，像小孩子一样随性而发

五行心能量：

  • 木火土金水的"心"的能量是仁义礼智信
  • 要把我们的能量找出来，仁义礼智信是储藏在我们心肝脾肺肾的先天能量

2. 火行人拔阴取阳通关点（84-85行）

通关方法：火怎么生出土？就是不报委屈，多找多看自己的问题，不是去找别人的问题。

核心操作：

  • 不想干什么干什么，想吃什么吃什么，想说什么说什么
  • 这个就是放任自己，害人又害己

3. 土行人拔阴取阳通关点（86-87行）

通关方法：土先天脾土怎么出阳土？就是不怨人。

核心操作：

  • 怨气堵在那，就把脾胃的运化能力给堵住了
  • 脾胃没有运化能力就看不开想不明白，没有智慧就分辨不出来哪些是好的哪些是坏的
  • 土行人内心有决断了，他知道这个事情该做不该做、该怎么做的时候他就转运了，他就生阳土了

禁忌：

  • 不能老是嘀嘀咕咕，因为这个人容易嘀嘀咕咕就是妄语
  • 妄语有阴土性，损自己的脾胃

4. 金行人拔阴取阳通关点（88-90行）

通关方法：金行人怎么才能生出阳金响亮金？就是感恩心，找别人的好处。

核心操作：

  • 找别人的好处就是聚人缘吸阳光
  • 找别人的不好处就是吸别人身上的阴气
  • 找别人的不好处，最后自己身上的阴气也很旺

人缘与财的关系：

  • 这个人的阳气很旺，大家都喜欢在他（她）身边因为很舒服
  • 找别人的不好处，爱攀比，这个就是盗，盗别人的财就是损别人的义气
  • 义气和财是绑定在一起的，所以有句话叫"缘聚财散"，"财聚人散"

5. 水行人拔阴取阳通关点（90-91行）

通关方法：那怎么生智慧水呢？就是"认不是"。

核心操作：

  • 不要给自己找理由，下次一定改，然后拖着拖着
  • 这样水就不是智慧水，是阴水浑水浊水
  • 浑水浊水就伤肾水，肾水本来应该是清亮清亮的
  • 里面有小错误在里面搅搅，肾水就浑了，容易得很多病

三、五行人详细特征（100-458行）

本部分详细描述木、火、土、金、水五种人的阴阳属性、身心灵特征、拔阴取阳方法等。

1. 木行人特征（100-165行）

阳木人特点：

  • 如同挺拔的松树，永远向着阳光生长
  • 特性是生长与繁荣，不断追求进步和发展
  • 在行为上，乐于助人，总是将他人的利益放在首位
  • 展现出高尚的道德情操，积极向上的态度
  • 在困境中也能坚持不懈，始终坚守自己的理想和原则
  • 富有创新精神，愿意为传播知识和真理而付出努力

阴木人特点：

  • 如同扭曲的树木，过于关注个体的繁荣和外表的光鲜
  • 忽视了内心的真实需求，可能为了个人利益而屈服于权威
  • 可能会失去自我和动力，过于追求被关注和崇拜
  • 忽视了他人的感受，导致人际关系紧张
  • 创新更多是为了个人的展示和炫耀，渴望得到他人的关注和赞美

木行人身心灵特征总结表：

维度 阳木特点 阴木特点
身界 挺拔健壮、姿态优雅 僵硬、佝偻、驼背、容易疲倦
心界 正直有担当、善于思考、乐于助人 自以为是、抗上不服人、不接受意见
灵界 仁慈慈善、担当力强、大公无私 倔强顶撞、性情阴沉、偏见高傲

2. 火行人特征（166-225行）

阳火人特点：

  • 积极主动，追求完美，不仅自我上进，也乐于成就他人
  • 善于赞美和褒扬，展现出积极向上的态度
  • 在团队中往往能发挥领导作用，推动众人之力，引发燎原之势
  • 自信、热情、充满活力，善于决策和决断，敢于挑战风险
  • 有担当能力，能够勇敢地面对挑战和困难

阴火人特点：

  • 野心勃勃，自恃完美，喜欢以自己的喜好为标准来评判他人
  • 容易急躁、好指责，喜欢争斗
  • 过于追求自我的"光辉"，贪图名利、好争斗、批评
  • 表面积极但信心不足，成功时可能骄狂炫耀，失败时则气馁颓废

火行人身心灵特征总结表：

维度 阳火特点 阴火特点
身界 庄重、大气、有气势、身体灵活 拘谨、声音尖锐、容易长痘痘
心界 明理讲道理、守礼有节、谦让恭敬 争强好胜、贪图名利、好夸张
灵界 通情达理、文采节制、光明磊落 外明内暗、急躁虚荣、好夸张

3. 土行人特征（226-289行）

阳土人特点：

  • 如同大地般坚实与宽广，言行一致，诚实守信
  • 对待他人还是事务，都充满忠诚，这种忠诚不仅体现在言语上，更体现在行动上
  • 具有承载和容纳四象的特质，能够与其他四象和谐共处
  • 展现出自信、宽厚的一面，善于合作互助，具有责任感
  • 在较高的精神压力下也能保持自始如一的态度

阴土人特点：

  • 显得有些沉闷和多怨，对事物往往持消极态度
  • 缺乏自信，少言寡语，在做事时往往速度较慢
  • 缺乏条理和头绪，显得保守且偏执，难以接受新鲜事物
  • 固执己见，缺乏灵活性，在面对决断时常常顾虑较多、优柔寡断

土行人身心灵特征总结表：

4. 金行人特征（287-332行）

阳金人特点：

  • 注重正义与公平，行为常受到"义"的引导
  • 具有坚韧与果断，在面对困境时能够迎难而上，不畏艰难
  • 他们善于决策，能够带领团队朝着正确的方向前进
  • 具有感恩的心态，善于发现并学习他人的优点和长处
  • 性格豪爽大方，不拘小节，善于表达自己的看法和想法

阴金人特点：

  • 可能显得有些刻薄，他们的批评性过强，使得在与人交往中容易产生摩擦
  • 多疑与缺乏果决，也使得他们在面对挑战时显得犹豫不决
  • 喜欢计较和比较，对他人的成就和优点心生嫉妒
  • 喜欢议论他人的过错和短处，行事伤人，凶狠成性，刻薄寡恩
  • 特别说明：阴金行人容易感到恼怒，这种恼怒情绪可能表现为对他人的不满和敌意

5. 水行人特征（338-432行）

阳水人特点：

  • 如同广阔的大海或流动的江河，智慧深邃，情感丰富
  • 善于沟通，适应性强，能够灵活应对生活中的各种挑战
  • 善于在低调中发挥自己的作用，不张扬却能够取得卓越的成就
  • 内心清静，不容易受到外界的干扰和影响，能够保持内心的平和与稳定
  • 智慧、性柔和，能够用智慧和温和的方式解决问题和处理人际关系

阴水人特点：

  • 可能像臭河沟里的水或死鱼烂虾的水，情感压抑，消极悲观，固执僵化
  • 缺乏动力，身体界可能表现为松软下坠，心理层面可能忧虑、自卑，不能自立
  • 容易感到烦闷，内心纠结，情绪不稳定，常常感到愚鲁和迟钝
  • 优柔寡断，难以做出决策，常常犹豫不决
  • 缺乏独立性，容易受到他人的影响，难以自我突破

四、拔阴取阳四步法详解（433-451行）

1. 认不是（447-449行）

定义：

  • 所说的认不是，如能达到真处，就是儒家"慎独"的功夫
  • 真正认识到自己心性中的阴面，遇事稍一错念，刚一动烦恼，即认识到他的"不是"
  • 因为他足以污染自性，这乃是最大的不是——"错处"

核心意义：

  • 这是做彻底拨阴的功夫
  • 古人说："认不是是明心丹"

2. 找好处（449-452行）

定义：

  • 所说的找好处，不仅从好人好事中找到好处，即从坏人坏事中，也可找到反面的好处
  • 心中常存他人的好处，才能生出义气和真金

核心意义：

  • 这也是彻底"反阴"的功夫
  • 所以古人说："找好处是暖心丸"

3. 信因果（452-457行）

定义：

  • 所说的信因果，即认识到天地间的一切事物没有无因之果，有因才有果
  • 果是由于一定的因缘而发展变化来的

核心意义：

  • 能明白这个道理，自然就不生怨气，而能生出信实的真土

4. 达天时（457-459行）

定义：

  • 所说的达天时，即包含着老子说的"道法自然"的古代哲学思想

核心意义：

  • 欲生真水，必认不是；欲生真金，必找好处；欲生真土，必信因果；欲生真火，必达天时
  • 这四类都达到了，行真了，才能立起通权达变
  • 阳刚中正仁德的真木来，最后达到"五行攒簇"，浑然一体，不分五行了
