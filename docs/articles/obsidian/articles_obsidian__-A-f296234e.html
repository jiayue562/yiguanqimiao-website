    A2 五脏神：内在的精神主宰

    > 核心来源：中医与道家养生体系 > > 五脏神定义：五脏不仅是有形的器官，更是五种精神能量的居所

    A2.1 木藏魂

    | 维度 | 内容 | |------|------| | 魂的定义 | 随神往来、主司谋虑、想象与梦境的精神活动 | | 肝木关联 | 肝木的条达与否，直接关系到"魂"的安宁与飞扬 | | 失衡表现 | 肝气郁结 → 魂不守舍 → 多梦、失眠、焦虑 | | 调适方法 | 疏肝理气（如舒肝功、柴胡疏肝散） |

    引申内涵：
  • 魂 = 创造力、想象力的精神基础
  • 肝木条达 = 魂有所归 = 思维清晰
  • 谋虑 = 规划未来的能力

    引申内涵：
  • 神 = 意识清明、情感稳定的精神核心
  • 心火清明 = 神有所主 = 情绪平和
  • 明辨 = 洞察本质的智慧

    • A2.3 土藏意

    | 维度 | 内容 | |------|------| | 意的定义 | 思虑、记忆与专注力的精神活动 | | 脾土关联 | 脾土的运化功能，支撑着"意"的持续与稳定 | | 失衡表现 | 脾虚失运 → 意不专注 → 健忘、思维迟钝 | | 调适方法 | 健脾益气（如四君子汤、山药粥） |

    引申内涵：
  • 意 = 思考、记忆、专注的认知能力

  • 感知 = 对外界刺激的敏锐反应

    | 维度 | 内容 | |------|------| | 志的定义 | 志向、意志力与深层记忆（肾主藏精，精生髓通于脑） | | 肾水关联 | 肾水的封藏与涵养，是"志"得以坚定深远的源泉 | | 失衡表现 | 肾精亏虚 → 志不坚定 → 意志薄弱、健忘、腰膝酸软 | | 调适方法 | 补肾填精（如六味地黄丸、黑芝麻丸） |
