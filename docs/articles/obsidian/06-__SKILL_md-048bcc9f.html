二、L1 决策感知层：CHMN-BRAIN 总控（5大职能）

F1 意图识别

  • 功能：NLU自然语言理解，识别董事长输入的意图类型
  • 置信度评分：对每类意图给出0-100%置信度
  • 意图分类：战略咨询 / 数据查询 / 异常诊断 / 决策讨论 / 赋能调度

F2 7S模块调度

  • 功能：DAG有向无环图任务编排，动态路由到对应7S模块
  • 调度策略：
    • 单一场景 → 激活1-2个S模块
    • 复合场景 → 激活3-5个S模块，定义依赖关系
    • 全景诊断 → 激活全部7个S模块

F3 诊断分析

  • 功能：聚合7S模块诊断结果，生成综合健康度评估
  • 五色光扫描引擎：
    • 白光（事实数据）：提取关键指标偏离度
    • 红光（直觉感受）：捕捉文化氛围异常信号
    • 黄光（积极价值）：识别增长机会和潜力区
    • 绿光（创新方案）：生成组织优化新思路
    • 蓝光（风险控制）：标记红线逼近和结构性问题

F4 决策建议

  • 功能：基于诊断结果，输出优先级排序的决策建议
  • 优先级体系：
    • P0（红色·立即处理）：红线触发、安全事故、关键人才流失
    • P1（橙色·本周处理）：指标持续恶化、制度执行偏差
    • P2（蓝色·本月处理）：优化机会、预防性调整

F5 赋能协议

  • 功能：将决策转化为可执行的赋能动作
  • 下店SOP：标准化赋能流程（诊断→方案→执行→验收）
  • 闭环响应：每个赋能动作有明确的责任人、时间节点、验收标准

三、L2 7S模块执行层：7大Skills详解

CHMN-SV · 共同价值观（DNA诊断模块）

子项 内容
核心定位 企业文化DNA的诊断与守护——不是检查"员工做了什么"，而是诊断"组织相信什么"
关联知识资产 老饕赋、文化罗盘、定海神针
7层审计体系 L1文化口号存在性 → L2标语可见度 → L3仪式执行率 → L4行为一致性 → L5决策价值观对齐 → L6客户感知文化 → L7文化自进化能力
核心诊断指标 文化渗透率≥100%、仪式执行率、宣讲达标率
象思维三层映射 物象：每日班前5分钟检查/每周价值观践行案例统计 → 意象：文化是土壤——肥沃则自生长，贫瘠则怎么贴都掉 → 原象：以食载道的根基，没有文化土壤就没有味藏（G-CHMN-SV01）
异常信号 老员工开始说"以前不是这样的"、仪式流于形式、价值观与KPI冲突时选KPI

CHMN-S · 战略（大脑诊断模块）

子项 内容
核心定位 企业战略方向的诊断与校准——检查"公司在往哪里走，走对了吗"
关联知识资产 定海神针、三昧真火、营业额框架、企业发展纲领
6模式分析 产品模式（核心品类竞争力）、客户模式（目标客群精准度）、盈利模式（收入结构健康度）、增长模式（扩张路径可行性）、竞争模式（差异化壁垒）、组织模式（战略-架构匹配度）
双线并行 包间（高客单·仪式感·文化体验线）+ 卡座（高翻台·标准化·效率线）
核心诊断指标 包间GC占比（目标65%）、新客转化率、老客复购率、客单价区间分布
象思维三层映射 物象：包间GC占比/客单价/翻台率/毛利率四维看板 → 意象：包间=高价值+低流量(做精)，卡座=高频+引流(做量) → 原象：每一分钱都可追溯到包间或卡座，精准管理（G-CHMN-S02）
盈利公式 包间=精准GC×人均1500×2.0利用率 卡座=泛GC×人均350×2.8翻台

CHMN-SSt · 结构（骨骼诊断模块）

子项 内容
核心定位 组织结构的健康度诊断——检查"骨架是否支撑得起战略"
倒三角赋能模型 总部（资源+标准+赋能）→ 区域（协调+督导+支持）→ 门店（执行+反馈+创新）
创业三单元 堂食单元（仪式即文化）、酒水单元（氛围即开心）、外卖单元（信息即增长）
核心诊断指标 管理幅度合理性、审批层级数、决策时效、赋能满意度（门店对总部评分≥30%考核权重）、三单元协同健康度
象思维三层映射 物象：下店天数/闭环响应率/门店满意度 → 意象：考核反转=门店给总部打分，权力真正下移 → 原象：服务的本质=不是"我在做什么"而是"你因为我得到了什么"（G-CHMN-St03）

CHMN-SSy · 制度（神经诊断模块）

子项 内容
核心定位 制度体系的灵敏度与有效性诊断——检查"规则是否让组织更敏捷还是更僵化"
关联知识资产 紧箍咒、营业额框架、客户流转图
3大红线 食品安全红线（一票否决）、客户体验红线（客诉24h闭环）、财务合规红线（资金安全）
客户流转图 从"进店→点单→用餐→结账→离店→复购"全链路体验监控
PDCA循环 制度执行的Plan-Do-Check-Act四步闭环，重点检查Check→Act环节是否真实运作
核心诊断指标 红线触发频次、客诉闭环率、制度执行偏差率、SOP更新频率
象思维三层映射 物象：退菜率<1%/出餐时效>90%/10秒响应率/NPS>8 → 意象：紧箍咒=免疫系统——不是攻击器官而是防御系统 → 原象：客户信任的守护者，红线不退=品牌不倒（G-CHMN-Sy04）

CHMN-SSty · 风格（气质诊断模块）

子项 内容
核心定位 领导风格与组织气质的诊断——检查"这家公司给人的整体感觉对不对"
刚柔并济 阳刚面（标准·纪律·执行·结果）+ 阴柔面（关怀·包容·成长·氛围）的动态平衡
领导力审计 从"命令型→教练型→服务型→授权型"四个层级评估管理者风格
核心诊断指标 管理风格与组织发展阶段匹配度、员工心理安全指数、冲突处理模式、团队士气波动
象思维三层映射 物象：SOP执行率/NPS/领导力评分 → 意象：制度=地板(下限，不让你掉下去)，文化=天花板(上限，让你往上升) → 原象：味藏的气质="走进来感受到什么"（G-CHMN-Sty05）

CHMN-SSf · 人员（血液诊断模块）

子项 内容
核心定位 人才系统的健康度诊断——检查"血液是否畅通，是否在滋养每个角落"
关联知识资产 企业发展纲领
559干部标准 5项能力（专业力·管理力·文化力·创新力·抗压力）+ 5项品德（忠诚·担当·公正·谦逊·利他）+ 9级评分（1-9级发展阶梯）
师徒制 师傅（传帮带·文化传承）+ 徒弟（成长路径·出师标准），考核"师徒对"的留存率和晋升率
赛马机制 同岗位横向对比，末位不淘汰但重点帮扶，首位重奖树立标杆
核心诊断指标 关键岗位储备率、师徒对成功率（出师率≥70%）、人才离职预警（3个月内离职概率）、高潜人才占比
象思维三层映射 物象：出师率/留存率/赛马排名/轮岗进度 → 意象：人才=战场上官兵——不是训练场练出来的 → 原象：人才血液永续流动=味藏不老的秘密（G-CHMN-Sf06）

CHMN-SSk · 技能（肌肉诊断模块）

子项 内容
核心定位 组织核心能力的诊断——检查"公司的肌肉够不够强，能不能打硬仗"
关联知识资产 三昧真火、匠心
匠心标准 对每道菜品/每个服务环节建立"匠心五军规"：选材→工艺→呈现→服务→迭代
三昧真火 技术火（专业硬技能）+ 服务火（客户体验软技能）+ 管理火（带团队领导力）
核心诊断指标 技能成熟度矩阵覆盖率、培训ROI、品质审计得分、客户对"专业度"的NPS评分
匠心五大军规 周期苛求、不时不食、冰鲜VS冷冻、销售时效、鲜活坚持——每条都是"信仰的量化"
象思维三层映射 物象：仓储周期/应季食材占比/冰鲜占比/5天售罄率/鲜活率 → 意象：匠心从"抽象的精神"变为"可追踪的数据" → 原象：以食载道的物质基础，没有匠心就没有灵魂（G-CHMN-Sk07）
