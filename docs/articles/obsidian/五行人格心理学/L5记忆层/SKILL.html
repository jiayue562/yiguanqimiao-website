触发规则

直接触发词

  • 用户画像、用户信息、我的五行
  • 记住、记忆、回忆
  • 上下文、上下文管理
  • 长期记忆、短期记忆
  • 状态追踪、历史记录
  • 诊断评估、干预转化

    • 场景触发

  • 用户首次自我介绍五行类型

    核心功能

    1. 用户画像管理（user-context）

    | 功能 | 说明 | |-----|------| | 五行类型记录 | 记录用户的木/火/土/金/水类型 | | 历史状态追踪 | 记录用户在不同场景下的状态变化 | | 偏好设置管理 | 记录用户的沟通偏好、工作习惯 | | 成长轨迹记录 | 记录用户的层级变化和成长轨迹 |

    2. 分层记忆管理（memory-management）

    | 层次 | 存储位置 | 容量 | 生命周期 | |------|---------|------|---------| | 工作记忆 | 内存 | 50条 | 会话内 | | 短期记忆 | 文件 | 100条 | 7天 | | 长期记忆 | 文件 | 无限 | 永久 | | 用户记忆 | 文件 | 无限 | 永久 |

    3. 诊断评估记录

  • 每次五行诊断的详细记录
  • 评估时间线
  • 状态变化追踪

    • 4. 干预转化记录

  • 拔阴取阳的实施记录
  • 化克为生的应用记录
  • 转化效果评估

    • ---

    ``` ┌─────────────────────────────────────────────────────────┐ │ L5 记忆层 │ ├─────────────────────────────────────────────────────────┤ │ 输入：画像查询/记忆检索/评估记录请求 │ ├─────────────────────────────────────────────────────────┤ │ 记忆分流 │ │ ├── user-context → 用户画像管理 │ │ │ ├── 五行类型记录 │ │ │ ├── 历史状态追踪 │ │ │ └── 成长轨迹记录 │ │ ├── memory-management → 分层记忆 │ │ │ ├── 工作记忆（会话内） │ │ │ ├── 短期记忆（7天） │ │ │ └── 长期记忆（永久） │ │ ├── rag-skill → 知识检索 │ │ └── agentdb-vector-search → 向量搜索 │ ├─────────────────────────────────────────────────────────┤ │ 结果输出 │ │ ├── 用户画像 │ │ ├── 历史记忆 │ │ ├── 诊断评估记录 │ │ └── 干预转化进展 │ └─────────────────────────────────────────────────────────┘ ```

    底层Skills

    user-context（用户画像）

  • 路径：`~/.workbuddy/skills/user-context/`
  • 功能：用户画像管理、状态追踪、历史记录

    • memory-management（分层记忆）

  • 路径：`~/.workbuddy/skills/memory-management/`
  • 功能：分层记忆管理、上下文管理、记忆检索

    • rag-skill（知识检索）

  • 路径：`~/.workbuddy/skills/rag-skill/`
  • 功能：知识库检索、语义搜索

    • agentdb-vector-search（向量搜索）

  • 路径：`~/.workbuddy/skills/agentdb-vector-search/`
  • 功能：向量相似度搜索
