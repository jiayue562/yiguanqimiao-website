---

触发规则

直接触发词

  • 任务分解、任务调度
  • 子Agent编排、多智能体协调
  • 结果聚合、整合分析
  • 复杂任务、任务规划

    • 场景触发

  • 用户提出复杂的五行人格分析需求
  • 需要多步骤诊断或评估
  • 需要协调多个分智能体协同工作
  • 需要整合多个维度的分析结果

    • ---

    核心功能

    1. 任务分解

  • 需求识别：分析用户需求，识别任务类型
  • 任务拆分：将复杂任务拆分为可执行子任务
  • 依赖分析：确定子任务执行顺序

    • 2. 子Agent编排

  • agent-orchestrator：多智能体协调
  • create-sub-agent：创建专用子代理
  • agent-swarm：并行任务执行

    • 3. 结果聚合

  • 多维整合：整合各子任务结果
  • 冲突处理：处理分析结果中的矛盾
  • 最终输出：生成完整分析报告

    • ---

    调度架构

    ``` ┌─────────────────────────────────────────────────┐ │ L3 调度层 │ ├─────────────────────────────────────────────────┤ │ 输入：用户复杂需求 │ ├─────────────────────────────────────────────────┤ │ 任务分解 │ │ ├── 需求分析 → 任务类型识别 │ │ ├── 任务拆分 → 子任务清单 │ │ └── 依赖确定 → 执行顺序 │ ├─────────────────────────────────────────────────┤ │ 子Agent编排 │ │ ├── agent-orchestrator → 整体协调 │ │ ├── create-sub-agent → 创建专用代理 │ │ └── agent-swarm → 并行执行 │ ├─────────────────────────────────────────────────┤ │ 结果聚合 │ │ ├── 多维整合 → 统一框架 │ │ ├── 冲突处理 → 一致性验证 │ │ └── 报告生成 → 最终输出 │ └─────────────────────────────────────────────────┘ ```

    工作流程

    场景：复杂人格诊断

    ``` 1. 接收任务：用户需要进行完整的五行人格诊断 ↓ 2. 任务分解： - 子任务1：五行类型初步判断 - 子任务2：一心三界状态评估 - 子任务3：九层发展层级定位 - 子任务4：关系模式分析 ↓ 3. 子Agent编排： - 调用底层知识体系（基础查询） - 调用L6底座层（知识检索） - 调用对应五行分智能体（深度分析） - 调用其他分智能体（交叉验证） ↓ 4. 并行执行：agent-swarm协调执行 ↓ 5. 结果聚合：整合各维度分析结果 ↓ 6. 输出：完整五行人格诊断报告 ```
