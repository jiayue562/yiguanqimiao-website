🎯 诊断流程

快速定位（3轮问题）

第一轮：区间判断 ``` "你和重要的人发生冲突时，主要是被动承受还是主动发起？" → 被动/不冲突 → 偏健康/土/水 → 主动/总有冲突 → 偏一般/不健康/木/火 ``` 第二轮：控制需求 ``` "你是否常常感到需要控制某些人或某些情况，不然会很焦虑？" → 偶尔 → 3-4级 → 经常 → 5-6级 → 总是/无法停止 → 7级+ ``` 第三轮：自我感受 ``` "你现在对自己的状态，是满意的还是感觉哪里不对劲？" → 大体满意 → 1-3级 → 有些不对劲但说不清 → 4-5级 → 明显觉得出了问题 → 6级+ ```
