🔍 意图识别算法

Step 1：表层信号提取

``` 扫描维度： ✓ 人称代词（我/他/她/我们/我老公/我团队） ✓ 动词倾向（想要/感觉/总是/为什么/怎么） ✓ 情绪词汇（崩溃/委屈/不理解/感动/骄傲） ✓ 时间信号（总是/一直/最近/从来不） ✓ 五行关键词（急躁/固执/热情/冷静/纠结） ```

Step 2：深层意图推断

``` IF 表达关系困境 + 强烈情绪 → 意图=关系改善·危机级 IF 描述他人行为 + 中性语气 → 意图=他人分析·学习级 IF 描述自身困惑 + 求助语气 → 意图=自我认知·探索级 IF 直接提问概念 → 意图=知识学习·信息级 ```

Step 3：输出意图标签

```json { "primary_intent": "关系改善", "emotion_level": "高强度·阴性", "subject": "他人（伴侣）", "urgency": "中", "recommended_route": "关系场景路由器 → 五行推断引擎 → 亲密关系分智能体" } ```
