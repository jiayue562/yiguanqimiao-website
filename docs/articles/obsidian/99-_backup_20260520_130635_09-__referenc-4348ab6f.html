🔄 1+5智能体模式详解

总智能体（快思考·中央调度）

核心职责：

  • 意图识别：判断用户是快速判断、深度学习、关系分析还是修行文化
  • 场景归类：S0-S9八大场景精准定位
  • 引擎路由：调度分智能体或组合多个分智能体
  • 执行调度：协调任务执行顺序与并行策略
  • 结果整合：汇总分智能体输出，生成整合报告

关键能力：

  • 多维感知矩阵：情绪信号（8分）+ 行为信号（4分）+ 诉求信号（2分）
  • 动态路由决策树：基于总分自动选择最优分析深度
  • 场景S0-S9全覆盖：简单问答→快速判断→深度学习→关系分析→决策分析→重大决策→任务执行→系统规划→修行文化→系统升级

    • 动机：成长、挑战、掌控
    • 能力：行动力强、规划清晰
    • 提示词：每日提醒"觉察、转念、行动"
    • 环境：创造支持仁德生长的环境

🎯 核心优势

1. 精准调度

  • 基于情绪信号权重自动选择分析深度
  • 避免过度分析或分析不足
  • 每个对话都精准匹配最优智能体组合

2. 模块化设计

  • 分智能体独立封装，可单独使用
  • 总智能体可灵活调度，支持任意组合
  • 生态模式：主智能体+子智能体，架构清晰

3. 七场景全覆盖

  • 从简单问答到系统升级的全覆盖
  • 每个场景都有明确的路由规则
  • 确保用户无论何时提及五行都能得到响应

4. 深度转化能力

  • 每个分智能体都包含完整的转化技术
  • 拔阴取阳四步法
  • 化克为生完整路径
  • B=MAP福格行为模型整合
  • WEASEM八周转化计划（木行人专属）
