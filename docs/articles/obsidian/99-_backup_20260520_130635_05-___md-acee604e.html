味藏营业额组成分解

文档类型：营业额业务线分析 核心定位：三大业务模块协同支撑总营业额 上级文档：味藏底层逻辑整合分析·企业文化体系

一、整体框架

总营业额 = 堂食 + 酒水 + 外卖

三大业务线的金字塔逻辑链：

核心要素 → 影响因素 → 关联要素 → 关键渠道 → 衡量指标 → 财务验证

二、堂食业务线

2.1 核心公式

客流 = 营销曝光 × (平台评分 + 产品吸引力)

2.2 金字塔拆解

层级 内容 说明
核心要素 客流 堂食营业额的基础
核心影响因素 营销曝光 决定客流规模的关键
关联要素1 大众点评评分 需稳定在4.8分以上
关联要素2 引流爆品 针对目标客群设计稀缺性
关键渠道 小红书/抖音/美团 线上平台曝光
衡量指标 评分≥4.8 + 超预估销量 验证效果
财务验证 活动效果环比 对比同期销量

2.3 引流爆品设计

  • 成立"引流组"
  • 针对目标客群（如刺身爱好者）
  • 设计稀缺性爆品
  • 验证标准：预估1000份，实际销2000份

三、酒水业务线

3.1 核心公式

销量 = 口感独特性 × 价格适中

3.2 金字塔拆解

层级 内容 说明
核心要素 数量（销量） 酒水营业额的直接来源
核心影响因素 口感 + 价格 决定复购的关键
关联要素1 选品（口感） 市面上不多见、具备独特性
关联要素2 价格适中 网上价格不透明、具备定价权
衡量指标 复购率 验证选品正确性
财务验证 销量环比 同品类对比

3.3 选品策略

  • 选择"市面上不多见、具备独特性"的酒品
  • 通过复购验证选品正确性
  • 客户喝完下次仍购买 = 选品对路

3.4 价格策略

  • 选择"网上价格不透明、具备定价权"的酒品
  • 避免高价（如100元进货卖1000元）
  • 定价合理以提升购买意愿

四、外卖业务线

4.1 核心公式

营业额占比 = 客户满意度 × (品质 + 包装 + 配送)

4.2 金字塔拆解

层级 内容 说明
核心要素 占比（权重） 外卖在总营业额中的权重
核心影响因素 客户满意度 决定复购和口碑
关联要素1 品质与单价 高端+引流+高盈利组合
关联要素2 包装与配送 解决汤洒、迟到等问题
关键渠道 专送服务 高端订单专人配送
衡量指标 达成率 + 满意度 预估达成/回访
财务验证 营业额占比 定期监测

4.3 高端配送策略

订单金额 配送方式 服务内容
1000元+ 奔驰专送 专人专送
2000元+ 电话回访 询问满意度

六、核心金句

  1. 堂食核心："通过营销曝光→客流增长奠定基础"
  2. 酒水核心："通过口感+价格→销量增长提升客单价"
  3. 外卖核心："通过满意度→占比提升扩大营收来源"
