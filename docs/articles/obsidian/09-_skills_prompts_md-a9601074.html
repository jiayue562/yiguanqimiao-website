📋 使用说明

如何使用

  1. 选择象限：根据任务类型选择对应象限
  2. 复制模板：复制对应提示词模板
  3. 填充内容：替换 [ ] 中的占位符
  4. 发送 AI：发送给 AI 开始协作
  5. 调整优化：根据效果调整提示词

提示词结构

# 角色设定（AI 扮演什么角色）
# 任务描述（要做什么）
# 输入信息（提供什么）
# 输出要求（期望什么结果）
# 约束条件（限制是什么）

🟡 第二象限：知识拓展者

模板 1：新领域学习

# 角色
你是我的知识导航员，帮助我系统学习新领域。

# 学习目标
我想学习 [领域/主题]。

# 我的背景
- 当前水平：[初学者/有一定了解/专业人士]
- 学习目标：[掌握基础/深入研究/应用实践]
- 可用时间：[每天/每周 X 小时]

# 请求帮助
1. 提供知识地图/学习路径
2. 推荐核心学习资源（书籍/课程/文章）
3. 解释关键概念
4. 回答我的问题

# 输出要求
- 结构化呈现（从基础到进阶）
- 标注学习优先级
- 提供预计学习时间

模板 2：文献综述

# 角色
你是我的研究助手，帮助我进行文献综述。

# 研究主题
[具体研究主题]

# 检索要求
- 时间范围：[例如：近 5 年]
- 文献类型：[论文/书籍/报告]
- 数据库：[例如：Google Scholar/CNKI]

# 输出要求
1. 核心文献清单（[数量] 篇）
2. 每篇的核心观点摘要
3. 研究演进脉络
4. 研究空白识别

# 约束
- 优先高引用文献
- 包含不同观点
- 标注文献来源

模板 3：竞品分析

# 竞品信息
[列出已知竞品或让 AI 识别]

# 分析维度

模板 4：概念解释

# 角色
你是我的学习伙伴，帮助我深入理解概念。

# 概念
[需要理解的概念名称]

# 我的问题
1. 这个概念的本质是什么？
2. 为什么这个概念重要？
3. 如何应用这个概念？
4. 与 [相关概念] 的关系是什么？

# 输出要求
- 用通俗语言解释
- 提供具体例子
- 指出常见误解
- 推荐深入学习资源

🟢 第三象限：思辨对话者

模板 1：战略决策分析

# 角色
你是我的思辨伙伴，帮助我深入思考战略决策。

# 决策议题
[具体决策议题]

# 背景信息
[相关背景、数据、约束条件]

# 我的初步想法
[你的初步想法/倾向]

# 思辨请求
1. 挑战我的假设
2. 提供不同视角
3. 指出我可能忽略的因素
4. 分析各选项的利弊

# 输出要求
- 深度追问（至少 3 层）
- 多视角分析（客户/员工/股东/社会）
- 风险评估
- 决策建议

[描述复杂问题]

[已知的相关信息]

# 请求帮助
1. 拆解问题为子问题（MECE 原则）
2. 识别关键变量
3. 分析因果关系
4. 提供解决思路

# 输出要求
- 问题树/逻辑图
- 优先级排序
- 可执行的解决步骤

3. 提供反面证据

# 输出要求
- 尖锐但建设性

- 帮助我提升思考质量

模板 4：方案评估

# 角色
你是我的方案评估顾问。

# 待评估方案
[描述方案内容]

# 评估标准

1. 多维度评估方案

4. 对比替代方案

# 输出要求
- 评估报告（按维度）

[感知场景：任务特征、不确定性、人类状态]

## 取象
[匹配模式：选择协作象限、定义角色]
