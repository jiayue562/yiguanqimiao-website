概述

"三界"是"一心"在个体层面的具体呈现，是火行能量得以被观察和调整的三大场域。

| 三界 | 象思维层次 | 核心功能 | |------|-----------|---------| | 身界 | 物象层 | 可直观感知的"火之形" | | 心界 | 意象层 | "火"在心理层面的象征 | | 灵界 | 原象层 | "火"所代表的终极价值 |

一、身界诊断（物象层）

1.1 体型特征

#### 阳面（健康）

| 特征 | 描述 | |------|------| | 整体 | 上尖下阔（V型），肩宽 | | 头部 | 头顶或鼻尖可能较突出 | | 四肢 | 筋骨有力 | | 比例 | 头小肩宽，呈倒三角形 |

#### 阴面（失衡）

| 特征 | 描述 | 原因分析 | |------|------|---------| | 臃肿 | 体型偏胖 | 火不生土，缺乏成果转化 | | 消瘦 | 体型过瘦 | 火过克金，能量过度消耗 | | 失调 | 比例不协调 | 能量分布不均 |

3.1 价值观

#### 阳面（健康）

| 价值观 | 核心词 | 描述 | |--------|--------|------| | 礼 | 明辨是非 | 能清晰分辨对错 | | | 秩序礼节 | 尊重社会规范 | | | 文明照亮 | 传播正能量 | | | 真诚热情 | 待人真诚热情 |

#### 阴面（失衡）

| 价值观 | 核心词 | 描述 | |--------|--------|------| | 嗔 | 看人不顺 | 总觉得别人不对 | | | 愤世嫉俗 | 对社会不满 | | | 以自我为中心 | 只考虑自己 |

核心总结：嗔——愤世嫉俗、以自我为中心

3.2 使命驱动

#### 阳面（健康）

| 使命 | 描述 | |------|------| | 照亮 | 渴望传播光明 | | 连接 | 促进人与人团结 | | 温暖 | 温暖他人 | | 提升 | 提升环境能量 | | 进步 | 促成积极改变 |

核心总结："照亮与连接"——渴望传播光明、温暖他人、提升环境、促成团结与进步

#### 阴面（失衡）

| 使命 | 描述 | |------|------| | 破坏 | 通过破坏展示力量 | | 征服 | 通过打压彰显自己 | | 刺激 | 盲目追求刺激 | | 毁灭 | 追求破坏性快感 |

核心总结："破坏与征服"——通过打压他人来彰显自己，或盲目追求刺激与破坏性快感

``` ┌─────────────────────────────────────────────────────────────────┐ │ 三界协同马车比喻 │ ├─────────────────────────────────────────────────────────────────┤ │ │ │ 【协同·健康状态】 │ │ ───────────────── │ │ │ │ 灵界"照亮使命"（礼德）—— 方向指引 │ │ ↓ │ │ 心界"热情喜悦"—— 动力系统 │ │ ↓ │ │ 身界"活力感染力"—— 执行表现 │ │ │ │ 三者方向一致，能量光明，马车稳健前行 │ │ │ ├─────────────────────────────────────────────────────────────────┤ │ │ │ 【失衡·病态状态】 │ │ ───────────────── │ │ │ │ 灵界"嗔恨"（失礼）—— 方向迷失 │ │ ↓ │ │ 心界"急躁虚荣"—— 动力扭曲 │ │ ↓ │ │ 身界"躁动攻击"—— 执行失控 │ │ │ │ 三者方向背离，能量黑暗，马车失控狂奔 │ │ │ └─────────────────────────────────────────────────────────────────┘ ```
