📖 黄帝内经·阴阳二十五人

核心定义

《黄帝内经·灵枢·阴阳二十五人》是五行识人"纯象阶段"的集大成之作。

核心贡献：根据五行理论将人分为木、火、土、金、水五形，每形再分五类，共二十五种类型。

五形人分类详解

1. 木形人

特征 描述
形体 体瘦面青，如树木般挺拔修长
性格 温和但意志不坚
能量态势 对应"木曰曲直"的生发舒展能量

识别要点：面青、身形瘦长、挺拔如树、温和内敛

2. 火形人

特征 描述
形体 面尖色赤，如火焰般上尖下阔
性格 性情急躁但富有创造力
能量态势 对应"火曰炎上"的炎上扩散能量

识别要点：面色偏红、头尖下阔、动作敏捷、热情外放

3. 土形人

特征 描述
形体 体厚面黄，如土地般敦厚稳重
性格 性格沉稳敦厚，善于谋划
能量态势 对应"土爰稼穑"的承载化育能量

识别要点：面色偏黄、身形厚实、稳重踏实、善于计划

4. 金形人

特征 描述
形体 面方色白，如金属般方正刚硬
性格 刚毅果决，思维严谨
能量态势 对应"金曰从革"的收敛坚锐能量

识别要点：面色偏白、脸型方正、棱角分明、刚毅果断

5. 水形人

特征 描述
形体 体圆色黑，如水流般圆润灵活
性格 聪明灵活但多疑善变
能量态势 对应"水曰润下"的润下潜藏能量

识别要点：面色偏黑、体态圆润、灵活善变、聪明机敏
