    • 2.3 五行：火性生克规律——能量互动的动态法则

    火行自身的阴阳两面：
  • 阳火（礼德）：明辨守礼、热情真诚、积极引领、照亮他人
  • 阴火（嗔恨）：急躁炫耀、虚荣好胜、愤世嫉俗、攻击性强
    • 关键生克关系对火行的影响： 1. 木生火：木行的“仁德”与“规划”提供持久燃料 - 木弱则火虚：热情缺乏根基，易忽冷忽热

    2. 火生土：火行的“热情”转化为土行的“信实”与“成果” - 火不生土则浮：光说不练，虚火上扬，缺乏实际成果

    3. 水克火：水行的“智慧”与“冷静”调节火行的急躁 - 水弱则火亢：缺乏冷静思考，热情变为躁狂 - 水过强则火灭：热情被过度压抑，变得消沉

    4. 火克金：火行的“感染力”融化金行的“刻板” - 火过旺则克金过度：导致尖刻批评、人际关系破裂

    火行核心调适技术： 拔阴取阳口诀： 1. 心界转化：“阴火恨转阳火问” - 将怨恨、急躁的能量，转化为提问、探索、求真的能量 - 遇冲突问：“发生了什么？”“你的想法是什么？”

    2. 灵界根本转化：“阴火嗔转阳火礼” - 将嗔恨、自我中心转为明礼、利他 - 践行“己所不欲，勿施于人”，创造和谐

    化克为生实践：
  • 当“火克金”导致人际冲突时，引入“土”来生“金”，形成良性循环
  • 停止争论（火），专注完成具体事情（土），重建信誉和规则（金）

    健康区间（1-3级）： ``` 1级：解放 - 明辨守礼，自在照亮。热情源于本然觉知，无求自照。 2级：心理能力 - 热情自信，积极引领。能有效运用热情感染他人。 3级：社会价值 - 活跃豪爽，成就认可。在社会角色中发挥热情。 ``` 一般区间（4-6级）： ``` 4级：失衡 - 急躁炫耀，内外拉扯。内心渴望与外在表现冲突。 5级：人际控制 - 好胜虚荣，操控索取。过度需要关注与赞美。 6级：过度补偿 - 自卑夸张，行为极端。用极端行为补偿内心空虚。 ``` 不健康区间（7-9级）： ``` 7级：侵害 - 愤怒攻击，侵害他人。阴火能量外溢为破坏性言行。 8级：妄想强迫 - 妄想迫害，行为强迫。脱离现实，产生妄想。 9级：病态破坏 - 狂暴毁灭，自我摧毁。能量彻底转向毁灭。 ``` 九层进阶的核心逻辑：
