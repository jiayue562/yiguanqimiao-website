1.4 两种IP命运分野（核心诊断框架）

流量型IP（阴面主导）:
  追求: 注意力/播放量/粉丝数
  内容: 追热点/博眼球/情绪化
  粉丝: 看热闹的围观者
  逻辑: 流量→筛选→成交(漏斗式)
  周期: 热点过气即凉
  本质: 被外部认可驱动的"阴面模式"

信任型IP（阳面主导）:
  追求: 信任值/用户心智占有率
  内容: 立立场/建标准/晒结果
  粉丝: 愿意付费的信任者
  逻辑: 信任→咨询→成交(磁铁式)
  周期: 信任越积累越值钱(复利)
  本质: 内在价值驱动的"阳面模式"

拔阴取阳转化:
  流量型 → 信任型 的转化路径:
    阴面(追逐流量) → 阳面(构建信任)
    木: 随大流 → 明确立场声明
    火: 空洞口号 → 标准可视化
    金: 被动应答 → 顾虑前置
    水: 自说自话 → 案例采集
    土: 爆发式更新 → 节奏化稳定输出
