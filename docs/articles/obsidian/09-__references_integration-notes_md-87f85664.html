与五行总智能体的整合

调度协议

调用者：五行人格心理学总智能体（凤心OS·快思考层） 被调用者：土行人分智能体 调度方式：智能调度系统（场景归类S0-S7 → 引擎路由 → 执行调度 → 结果整合）

场景归类触发条件

S2（深度理解）触发条件：

  • 用户要求深度学习土行人理论
  • 用户询问土行人的核心特质
  • 用户要求分析土行人格

S3（关系分析）触发条件：

  • 用户询问土行人与其他五行的关系
  • 用户要求分析土克土、土生金、水克土等关系
  • 用户要求进行五行关系诊断

S4（拔阴取阳）触发条件：

  • 用户要求转化怨气、猜疑、控制、拖延
  • 用户要求实践拔阴取阳技术
  • 用户要求化克为生

S5（B=MAP）触发条件：

  • 用户要求设计土行人的行为改变
  • 用户要求制定土行人成长计划
  • 用户要求实践相生强化

S6（识人）触发条件：

  • 用户要求识别某人是否是土行人
  • 用户要求外观识人
  • 用户要求全息识人

S7（全系统）触发条件：

  • 用户要求完整的五行人格分析
  • 用户要求土行人的完整诊断与干预方案
  • 用户要求五行总智能体协同输出

执行调度流程

标准七步流程： ① 一心状态定位 → ② 三界扫描 → ③ 五行生克诊断 → ④ 九层定位 → ⑤ 转化方案 → ⑥ B=MAP落地 → ⑦ 输出报告

结果整合：

  • 土行人分析结果 → 总智能体整合输出
  • 如果是多行分析，总智能体整合多个分智能体的结果

与凤心OS的整合

凤心OS快思考层调度

三维感知矩阵打分：

  • 情绪信号：8分（阴性） / 6分（阳性）
  • 行为信号：4分
  • 诉求类型：2分（辅助）

路由决策规则：

  • 总分 ≥ 10 → 直接激活土行人分智能体（精准慢思考）
  • 总分 7-9 → 双行分析（土行+副行并行）
  • 总分 5-6 → 全面诊断（五行总体分析）
  • 总分 ≤ 4 → 先追问1个精准问题，确认后路由

场景分类（S0-S8）：

  • S0（信号不足）→ 追问1个精准问题
  • S1（快速判断）→ 直接输出快速识别
  • S2（深度分析）→ 激活土行人分智能体（慢思考）
  • S3（关系分析）→ 激活土行人+其他分智能体（并行）
  • S4（拔阴取阳）→ 激活土行人分智能体执行转化
  • S5（B=MAP）→ 激活土行人分智能体执行行为设计
  • S6（识人）→ 外观全息识人系统
  • S7（全系统）→ 激活多个分智能体（并行）
  • S8（龙心OS整合）→ 七模块协调输出

与跨域理论的关联

现代心理学理论关联

MBTI关联

土行人与MBTI SJ类型：

  • ISTJ：内倾、感觉、思考、判断——稳重务实、责任导向
  • ISFJ：内倾、感觉、情感、判断——稳重包容、滋养他人
  • ESTJ：外倾、感觉、思考、判断——稳重可靠、执行力强
  • ESFJ：外倾、感觉、情感、判断——稳重包容、和谐导向

认知功能映射：

  • Si（内倾感觉）：土行人的稳定性、经验积累
  • Te（外倾思考）：土行人的执行力、成果导向
  • Fe（外倾情感）：土行人的包容性、和谐导向

Si/Te/Fe功能扭曲与九层对应：

  • Si扭曲（第四级凝滞）：固执己见，内外拉扯
  • Te扭曲（第五级依赖控制）：控制欲强，占有欲重
  • Fe扭曲（第六级怨天尤人）：怨气重，人际关系紧张

九型人格关联

土行人与九型人格类型：

  • 9号和平追求者：追求稳定，避免冲突
  • 6号忠诚者：忠诚可靠，寻求安全感

土行人九层与九型人格层级：

  • 健康层级（1-3）：对应九型人格健康层级
  • 一般层级（4-6）：对应九型人格一般层级
  • 不健康层级（7-9）：对应九型人格不健康层级

大五人格关联

土行人与大五人格特质：

  • 宜人性高：包容接纳，和谐共处
  • 尽责性高：稳重可靠，信守承诺
  • 神经质低：情绪稳定，不轻易波动

霍兰德职业兴趣关联
