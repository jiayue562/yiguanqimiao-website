📊 第一部分：图谱类型详解

1.1 全局图谱

```yaml 定义: 展示整个知识库所有笔记节点及其链接关系

打开方式: 设置 → 外观 → 右侧边栏 → 开启"图形视图" 或快捷键: Ctrl/Cmd + Shift + G

视图范围: - 所有笔记 - 所有双向链接 - 所有标签关系

核心用途: 1. 宏观看清知识全貌 2. 发现整体结构问题 3. 识别知识空白 ```

1.2 本地图谱

```yaml 定义: 聚焦于当前笔记，显示其直接相连的笔记（1-2层关系）

打开方式: 当前笔记 → 右上角"更多选项" → "在本图谱中显示" 或：右键笔记 → "在本图谱中显示"

视图范围: - 当前笔记（中心节点） - 第一层关联（直接链接） - 第二层关联（链接的链接）

核心用途: 1. 微观分析局部网络 2. 理解特定概念的关系网络 3. 发现潜在连接 ```

🎨 第二部分：节点与边的优化

2.1 节点优化

```yaml 【大小设置】

规则: 与连接数（度数）成正比 效果: - 连接数↑ → 节点越大 - 连接数↓ → 节点越小

设置路径: 图形视图 → 三点菜单 → 外观 → 节点大小 → 连接数

【颜色设置】

规则: 按属性映射 映射条件: - 标签（#心理学 → 红色） - 类型（type: 项目 → 蓝色） - 状态（status: 进行中 → 绿色）

设置路径: 图形视图 → 三点菜单 → 颜色 → 标签/类型/状态 ```

2.2 边（连接线）优化

```yaml 【边的类型】

双向链接: ────── (实线) 嵌入关系: ─ ─ ─ (虚线) 共享标签: ┄┄┄┄ (点线)

【边的颜色】

规则: 按链接特征映射 - 同一文件夹 → 同色 - 同标签 → 同色 - 嵌入 → 特殊颜色

【高级样式】

使用Juggl插件可以实现： - 节点渐变色 - 动态边宽度 - 箭头方向显示 - 动画效果 ```

3.2 路径搜索

```yaml 【功能】

查找从笔记A到笔记B的所有可能路径

【使用场景】

当你发现两个笔记看似无关，但想探索潜在联系时

【操作步骤】

1. 右键笔记A → "查找到此处的所有路径" 2. 或：搜索框输入 "path:笔记A 笔记B" 3. 系统展示所有中间节点和路径

【解读】

路径长度 = 关联紧密程度

  • 1步 = 直接链接
  • 2-3步 = 间接关联
  • 更多步 = 弱关联或可建立的新连接
    • ```

    控制图谱显示的关联层数

    【层级说明】

    第0层: 只显示当前笔记 第1层: 显示直接链接（出链/入链） 第2层: 显示链接的链接 第3层+: 依此类推

    【推荐设置】

    日常使用: 1-2层 全局分析: 2-3层 深度探索: 按需展开 ```

    💡 第四部分：图谱思维实践

    4.1 知识复盘三步法

    ```markdown 【第一步：识别孤立笔记】

    打开全局图谱 → 寻找没有连接的节点 → 思考：为什么孤立？

    【第二步：建立连接】

    策略1: 与现有主题建立链接 策略2: 创建新的综述笔记 策略3: 合并到已有主题

    【第三步：持续维护】

    每周一次图谱复盘 保持知识网络的健康度 ```

    4.2 学习规划应用

    ``` ┌─────────────────────────────────────────────────────────────────┐ │ 图谱驱动的学习规划 │ ├─────────────────────────────────────────────────────────────────┤ │ │ │ 【识别知识密度】 │ │ │ │ 密集区域 → 核心节点周围笔记很多 │ │ ↓ │ │ 决策：向深处挖掘还是向广度拓展？ │ │ │ │ 【向深处挖掘】 │ │ → 丰富核心节点的关联 │ │ → 补充细节笔记 │ │ → 深化理解 │ │ │ │ 【向广度拓展】 │ │ → 连接新的孤立节点 │ │ → 建立跨领域关联 │ │ → 发现新主题 │ │ │ └─────────────────────────────────────────────────────────────────┘ ```

    4.3 创作启发

    ```yaml 【密集区域 → 成熟主题】

    信号: 多个笔记高度互联 解读: 这个主题你已经积累充分 行动: 可以开始输出（文章/课程/书籍）

    【稀疏区域 → 研究空白】

    信号: 笔记稀少或无关联 解读: 这个领域你还没深入 行动: - 深入研究（输入） - 或暂时放下（不是你的方向）

    【断链区域 → 待连接】

    信号: 两个区域没有关联 解读: 可能存在有趣的跨域联系 行动: 主动建立连接 → 触发新想法 ```

    🛠️ 第五部分：高级应用

    5.1 Dataview + Graph View

    ```markdown 【组合优势】

    Dataview: 查询特定条件的笔记 Graph View: 可视化这些笔记的关系

    【操作流程】

    1. 用Dataview查询 ```dataview TABLE 评分, 标签 FROM "" WHERE 评分 >= 4 ```

    2. 将结果发送到Graph View

    3. 观察高质量笔记的分布

    4. 发现知识网络中的"明星节点" ```

    5.2 Juggl插件

    ```yaml 【Juggl高级功能】

    1. 交互式图形 - 拖拽节点 - 缩放平移 - 聚焦跟随

    2. 样式控制 - 节点形状（圆形/方形/图标） - 渐变填充 - 阴影效果

    3. 布局算法 - 力导向布局 - 层级布局 - 环形布局

    4. 导出功能 - 导出为PNG/SVG - 嵌入到笔记 ```
