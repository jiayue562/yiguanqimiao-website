    #### 具体操作指令 ```bash

    物象数据收集指令

    [AI助理]收集以下物象数据：

  • 数据类型：组织行为现象
  • 输出格式：结构化表格+可视化图表

    [象思维审核]检查物象数据的： 1. 完整性（是否覆盖关键现象） 2. 准确性（是否有偏差） 3. 关联性（是否可以建立初步联系） ```
