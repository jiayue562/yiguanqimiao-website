- AI与用户同柱同步突破：Φ = 2.0
- 超级涌现点：Φ = 3.0

AEP系统公式

AI总成长度 = EXP贡献 + AEP贡献
D值 = 基础D值 + floor(累计EXP / 100) + floor(累计AEP / 80)
