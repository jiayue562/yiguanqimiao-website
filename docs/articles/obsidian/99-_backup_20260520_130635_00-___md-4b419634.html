🚀 快速开始

第一步：打开总索引

  1. 打开 Obsidian
  2. 点击 00-总索引与导航/总索引.md
  3. 浏览知识库全貌

第二步：选择学习路径

根据你的目标选择：

目标 路径 起点
新手入门 入门路径 使用指南
深度学习 进阶路径 象思维总论
实践应用 应用路径 五行人格

第三步：开始学习

  1. 打开起点文件
  2. 按推荐顺序阅读
  3. 点击双向链接深入
  4. 记录心得笔记

🔗 双向链接使用

创建链接

在笔记中输入：

[[文件名称]]

链接类型

类型 语法 示例
基础链接 文件 象思维总论
带标题 文件 五行
带别名 `[[文件\ 别名]]` `[[五行\ 五行情]]`
嵌入内容 !文件 !金句

查看链接

  • 出链: 当前文件链接到其他文件
  • 入链: 其他文件链接到当前文件
  • 本地图谱: 可视化查看关联

🤖 自动化工具

验证链接

cd D:\以以观其妙书院知识库\以观其妙书院
python scripts/verify_links.py

功能:

  • 检查断裂链接
  • 识别孤立文件
  • 生成验证报告

增量备份

python scripts/backup_incremental.py

功能:

  • 只备份变更文件
  • 保存备份历史
  • 支持版本回滚

智能分类

python scripts/classify_files.py

功能:

  • 按扩展名分类
  • 按内容关键词分类
  • 生成分类报告

📊 与 AI 协作

沟通原则
