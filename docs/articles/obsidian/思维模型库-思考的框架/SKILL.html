一、核心定位

1.1 系统归属

``` 龙心OS（总智能体） ↓ 智能调度 龙脑OS知识库 ├─ 思维模型库（本库·59个模型） │ ├─ 通用思维模型（31个） │ └─ 麦肯锡思考框架（28个） └─ 其他专业知识体系 ```

1.2 核心价值

构建一套"思维工具箱"，帮助在不确定性中：

  • ✅ 提升决策准确度
  • ✅ 规避认知偏差
  • ✅ 看清事物本质
  • ✅ 预见长期后果
  • ✅ 组合多模型应对复杂问题

    五、龙心OS调用机制

    5.1 调用方式

    #### 方式一：单模型调用 ``` 场景：需要特定思维模型分析 调用：直接调用对应模型Skill 示例："用第一性原理分析这个问题" ```

    #### 方式二：多模型组合调用 ``` 场景：复杂问题需要多角度分析 调用：组合2-3个相关模型 示例："用第一性原理+二阶思维+概率思维分析" ```

    #### 方式三：场景套件调用 ``` 场景：特定场景的标准分析流程 调用：预设的模型组合套件 示例："启动决策分析套件" ```

    5.2 调用协议

    ```yaml 调用协议: 请求方: 龙心OS总智能体 响应方: 思维模型库（龙脑OS）

    握手包内容: - 场景描述 - 问题类型 - 期望输出 - 时间约束

    路由决策: - 简单问题 → 单模型 - 复杂问题 → 多模型组合 - 标准场景 → 场景套件 ```
