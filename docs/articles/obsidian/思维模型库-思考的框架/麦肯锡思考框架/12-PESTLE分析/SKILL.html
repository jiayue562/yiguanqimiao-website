一、核心定义

1.1 一句话定义

PESTLE是PEST的扩展版，增加法律(Legal)和环境(Environmental)两个维度，提供更全面的宏观环境分析框架。

1.2 六维度结构

``` PESTLE = P(Political) + E(Economic) + S(Social) + T(Technological) + L(Legal) + E(Environmental) ```
