测试场景一：科技公司战略规划

输入

情境：某AI创业公司计划未来3年拓展海外市场，需要进行宏观环境分析

期望输出

``` 【PEST分析 · 海外市场拓展】

■ 政治维度（P） - 数据保护法规（GDPR等）影响产品合规 - 政府AI产业政策支持力度 - 国际关系对技术人才流动的影响 影响程度：高 | 时间跨度：中期 | 判定：威胁

■ 经济维度（E） - 目标市场经济增长预期 - 汇率波动对收入的影响 - 当地融资环境与成本 影响程度：中 | 时间跨度：短期 | 判定：机会

■ 社会维度（S） - AI接受度与市场需求 - 当地工程师人才供给 - 隐私意识与数据敏感度 影响程度：高 | 时间跨度：长期 | 判定：机会

■ 技术维度（T） - 当地技术基础设施水平 - 开源生态与技术社区活跃度 - AI技术迭代速度 影响程度：高 | 时间跨度：中期 | 判定：机会

【综合评估】 最大机会：AI市场需求旺盛 + 技术人才充裕 最大威胁：数据合规要求 + 国际关系不确定性 ```

验证标准

  • [ ] 四个维度完整覆盖
  • [ ] 每维度识别2-3个关键趋势
  • [ ] 影响程度评估有依据
  • [ ] 机会/威胁判定明确
