一、核心定义

1.1 一句话定义

用户画像是基于调研数据构建的半虚拟理想用户原型，通过人口统计、行为特征、目标动机、痛点需求等维度，刻画目标用户，帮助团队建立共识、指导决策。

1.2 核心公式

``` 用户画像 = 基本属性 + 行为特征 + 目标动机 + 痛点需求 + 关键词 ```

四、实战模板

场景：知识付费用户画像

``` 【用户画像 · 知识付费】

■ 画像一：职场晋升型

基本属性： - 25-35岁，中层管理者 - 本科以上，一线城市 - 月收入2-3万

行为特征： - 购买频次高，客单价中 - 偏好职场管理类内容 - 通勤时间听音频

目标动机： - 提升管理能力 - 升职加薪 - 系统化学习

痛点需求： - 时间碎片化 - 难以坚持 - 需要实践指导

关键词："高效学习"、"实用干货"

■ 画像二：副业变现型

基本属性： - 22-30岁，自由职业/斜杠 - 大专以上，各线城市 - 月收入波动大

行为特征： - 价格敏感度高 - 偏好理财、副业类内容 - 社交电商活跃

目标动机： - 增加收入来源 - 探索第二职业 - 财务自由

痛点需求： - 缺乏靠谱项目 - 执行力不足 - 需要社群陪伴

关键词："副业刚需"、"月入过万" ```
