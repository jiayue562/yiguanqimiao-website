一、核心定义

金字塔结构（Pyramid Principle）是麦肯锡经典的结构化表达方法，核心原则是结论先行、以上统下、归类分组、逻辑递进。

---

二、核心原则

1️⃣ 结论先行 (Top-Down)

  • 先说结论，再说理由
  • 让听众第一时间知道你的观点
  • 符合人类认知习惯

    • 2️⃣ 以上统下 (Vertical Logic)

  • 上层是下层的概括
  • 下层支撑上层的结论
  • 形成金字塔层级

    • 3️⃣ 归类分组 (MECE)

  • 同层要素相互独立、完全穷尽
  • 每组思想属于同一范畴
  • 每组思想按逻辑顺序组织

    • 4️⃣ 逻辑递进 (Horizontal Logic)

  • 时间顺序：步骤先后顺序
  • 结构顺序：空间/组成部分
  • 程度顺序：重要性/程度排序

    四、两种构建方式

    方式一：自上而下 (Top-Down)

    适用场景：已有明确结论，需要组织论证

    ``` 步骤： 1. 确定中心思想（结论） 2. 列出支撑论点（3-5个） 3. 为每个论点寻找事实支撑 4. 检查逻辑完整性 ```

    方式二：自下而上 (Bottom-Up)

    适用场景：有大量信息，需要提炼结论

    ``` 步骤： 1. 收集所有相关信息 2. 对信息进行归类分组 3. 提炼每组核心观点 4. 总结中心思想 ```

    五、SCQA序言结构

    Situation（情境）：背景介绍 Complication（冲突）：存在问题 Question（疑问）：核心问题 Answer（答案）：你的结论

    示例

  • S：公司Q1销售额增长20%
  • C：但客户流失率上升至15%
  • Q：如何在保持增长的同时降低流失？
  • A：建议实施客户分层运营策略

    六、实战案例

    案例：项目汇报金字塔

    中心思想：建议启动新产品线，预计3年内实现盈亏平衡 关键论点： 1. 市场机会巨大（黄光） - 目标市场年增长率25% - 现有竞争者覆盖不足 - 客户需求未被满足

    2. 我们有能力（白光） - 技术储备充足 - 团队经验丰富 - 资金实力支持

    3. 风险可控（蓝光） - 分阶段投入策略 - 快速试错机制 - 退出方案明确
