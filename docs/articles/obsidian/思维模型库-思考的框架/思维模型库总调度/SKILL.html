一、核心定位

思维模型库是龙脑OS知识库的核心组成部分，包含59个思维模型：
  • 通用思维模型：《思考的框架》系列31个跨学科模型
  • 麦肯锡思考框架：28个商业分析专业模型
    • 系统定位：
  • 所属系统：龙脑OS知识库（被调用层）
  • 调度中枢：龙心OS（总智能体）
  • 调用方式：由龙心OS根据场景智能路由
  • 最新升级：2026-04-13 整合麦肯锡思考框架（28个模型）

    三、龙心OS调用协议

    调用方式

    #### 1. 单模型调用 ``` 场景：需要特定思维模型分析 调用：直接调用对应模型Skill 输出：该模型的分析结果 ```

    #### 2. 多模型组合调用 ``` 场景：复杂问题需要多角度分析 调用：组合2-3个相关模型 输出：综合多模型的分析结果 ```

    #### 3. 场景套件调用 ``` 场景：特定场景的标准分析流程 调用：预设的模型组合套件 输出：完整的场景分析报告 ```

    高频场景套件

    #### 决策分析套件 ```yaml 适用场景: 重大决策、战略选择 模型组合: - 03-第一性原理（看本质） - 05-二阶思维（看长期） - 06-概率思维（算风险） - 07-逆向思维（避失败） - 13-能量守恒（权衡利弊） 输出: 综合决策建议 ```

    #### 创新突破套件 ```yaml 适用场景: 创新、颠覆性突破 模型组合: - 03-第一性原理（本质重构） - 04-思想实验（零成本试错） - 14-临界点（突破时机） - 19-催化剂（加速实现） - 31-涌现（系统创新） 输出: 创新方案与实施路径 ```

    #### 个人成长套件 ```yaml 适用场景: 个人发展、能力提升 模型组合: - 02-能力圈（明确边界） - 10-复利效应（长期积累） - 20-活化能（启动改变） - 25-进化（持续迭代） - 26-生态位（差异化定位） 输出: 个人发展策略 ```

    #### 组织管理套件 ```yaml 适用场景: 团队管理、组织优化 模型组合: - 11-惯性（变革阻力） - 12-熵增定律（混乱管理） - 16-反馈回路（系统优化） - 21-化学键（关系构建） - 29-共生（协同发展） 输出: 组织优化方案 ```

    #### 战略规划套件（麦肯锡） ```yaml 适用场景: 企业战略制定、竞争分析 模型组合: - M-01-SWOT分析（内外部评估） - M-02-BCG矩阵（业务组合） - M-14-五力分析（行业竞争） - M-15-3C分析（战略定位） - M-05-安索夫矩阵（增长战略） 输出: 战略规划方案 ```

    #### 问题诊断套件（麦肯锡） ```yaml 适用场景: 复杂问题分析、根因诊断 模型组合: - M-19-MECE原则（结构化分解） - M-06-七步问题解决法（系统流程） - M-07-逻辑树（问题拆解） - M-10-假设思考（验证推理） - M-23-议题树（议题分析） 输出: 问题诊断报告 ```

    > "思维模型不是知识本身，而是处理知识、分析问题、做出决策的工具。"

    > "复杂问题无法用单一模型解决，必须构建思维格栅，组合多个跨学科模型。"

    > "思维模型的价值，不在于知道多少个，而在于如何正确使用。"
