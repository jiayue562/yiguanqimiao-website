AIDMA模型 · 消费者行为阶段分析

🎯 一、核心定义

AIDMA模型是描述消费者从注意到购买决策全过程的经典营销模型，由美国广告学家E.S.刘易斯提出：

| 字母 | 含义 | 中文 | 核心问题 | |------|------|------|----------| | A | Attention | 注意 | 如何让消费者看到？ | | I | Interest | 兴趣 | 如何引发消费者兴趣？ | | D | Desire | 欲望 | 如何激发购买欲望？ | | M | Memory | 记忆 | 如何让消费者记住？ | | A | Action | 行动 | 如何促使购买行动？ |

1.1 AIDMA曲线

``` 消费者数量 ▲ │▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │ │ ★★★★★★★★★★★★★★★★★★★ 注意（A） │ ★★★★★★★★★★★★★★★★★★★ 兴趣（I） │★★★★★★★★★★★★★★★ 欲望（D） │★★★★★★★★★★ 记忆（M） │★★★★★ 行动（A） └──────────────────────────────────→ 时间/触点 ```

1.2 AIDMA的演进

``` AIDMA（传统）→ AISCE（互联网）→ AIPL（电商）→ AFS（A面） ↓ 社交时代：关注（Attention）→ 互动（Interaction）→ 分享（Share）→ 传播（传播） ```

👁️ 二、第一阶段：注意（Attention）

2.1 注意力获取策略

| 策略类型 | 做法 | 示例 | |----------|------|------| | 尺寸放大 | 增大视觉元素 | 大标题、大图片 | | 色彩对比 | 使用高对比色 | 黑底黄字 | | 动态吸引 | 动画、视频、动效 | 视频广告 | | 悬念设计 | 制造好奇 | 悬念文案 | | 名人效应 | KOL、明星代言 | 明星海报 |

2.2 注意力质量评估

| 评估维度 | 说明 | |----------|------| | 触达率 | 多少人看到了 | | 关注率 | 多少人真正注意到 | | 停留时间 | 消费者看了多久 | | 首因效应 | 最初几秒的印象 |

2.3 Attention阶段关键指标

``` Attention KPI：

  • 曝光量（Impressions）
  • 点击率（CTR）
  • 完播率（Video Completion Rate）
  • 首次触达成本（CPM）
    • ```

    💡 九、核心金句

    1. "AIDMA不是线性的，消费者可能在任何阶段进出。" 2. "注意力是入口，行动是出口，中间环节决定转化效率。" 3. "最好的营销是让消费者'不知不觉'走完AIDMA全程。" 4. "每个阶段的触点设计都要对应该阶段的用户心理。" 5. "AIDMA模型需要随媒介和用户行为变化而迭代。"
