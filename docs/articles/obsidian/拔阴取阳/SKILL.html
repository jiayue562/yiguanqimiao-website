📖 核心定义

What（是什么）

拔阴取阳是五行人格心理学的核心能量转化技术，是一套系统性的能量诊断与修炼方法。

它基于精气神三层生命模型（身界/心界/灵界）和五行阴阳分类系统（木火土金水×阴阳=10种人态），提供：

  • 三层诊断：从身、心、灵三界判断自己当前是否走运
  • 五行病诊断：识别五行病链条（内在想法→外在行为→命运结果）
  • 通关点技术：通过5个核心通关点实现五行能量流通
  • 阴性自查：系统化识别自己的阴性特征

    现状问题：
  • 🔴 不知自己当前是否走运（阴阳能量态不明）

  • 🔴 五行能量堵塞（运气不流通）

  • ✅ 三层诊断：明确知道自己的能量状态

    核心公式： ``` 拔阴取阳 = 三层诊断 + 五行病诊断 + 通关点修炼 + 阴性自查 ``` 基本流程： 1. 三层诊断：从身、心、灵三界判断阴阳能量态 2. 五行病诊断：识别五行病链条（内在想法→外在行为→命运结果） 3. 定位主五行：确定自己是木/火/土/金/水哪一行 4. 识别阴性特征：自查对应五行的阴性表现 5. 修炼通关点：针对主五行修炼通关点技术 6. 持续实践：3-6个月形成习惯，能量态转变 五大通关点：

    🎯 核心功能

    功能一：三层诊断（精气神三层生命模型）

    #### 什么是精气神三层？

    ``` 精（身界）= 物质体 - 身体状态 - 外貌形态 - 健康状况 - 行为举止

    气（心界）= 能量体 - 情绪状态 - 心情波动 - 心理能量 - 思维活跃度

    神（灵界）= 信息体 - 价值观 - 使命感 - 灵性觉知 - 直觉洞察 ```

    #### 三层诊断方法

    步骤1：身界诊断 ``` 观察身体状态： - 阴性特征：僵硬、佝偻、不灵活、驼背、面色暗沉 - 阳性特征：挺拔、灵活、舒展、有光泽、充满活力

    判断标准： - 如果身体出现阴性特征 → 当前不走运 - 如果身体出现阳性特征 → 当前走运 ```

    步骤2：心界诊断 ``` 观察情绪状态： - 阴性特征：愤怒、怨恨、焦虑、抑郁、情绪低落 - 阳性特征：愉悦、平和、积极、热情、情绪稳定

    判断标准： - 如果情绪以阴性为主 → 当前不走运 - 如果情绪以阳性为主 → 当前走运 ```

    步骤3：灵界诊断 ``` 观察灵性状态： - 阴性特征：迷茫、无目标、价值观混乱、缺乏使命感 - 阳性特征：清晰、有目标、价值观明确、有使命感

    判断标准： - 如果灵性出现阴性特征 → 当前不走运 - 如果灵性出现阳性特征 → 当前走运 ```

    #### 三层协同判断

    ``` 三层综合判断： - 三层都阳性 → 完全走运，能量充盈 - 两层阳性一层阴性 → 走运但有小问题 - 一层阳性两层阴性 → 走运受阻，需要转化 - 三层都阴性 → 完全不走运，急需转化 ```

    功能三：通关点技术

    #### 五行通关点总览

    ``` 五行能量循环：

    木 → 火 → 土 → 金 → 水 → 木 ↓ ↓ ↓ ↓ ↓ 忍辱 放任 不怨 感恩 认不是

    每个通关点都是五行能量流通的关键节点 ```

    #### 五行通关点详解

    通关点1：木→火 - 忍辱

    ``` 核心方法： - 控制脾气，不随意发火杀人 - 忍住自己的脾气，不随时想发就发 - 不想说什么就说什么（像小孩子一样随性而发） - 关注别人的情绪，不伤害他人

    为什么叫"杀人"？ - 在当下这一秒钟没有关注到别人的情绪 - 一秒杀一秒，最后聚集起来就真的杀了一个人 - 这里的"杀人"不是真杀人，而是伤害别人的心灵

    忍辱的本质： - 不是压抑情绪，而是觉察和控制 - 木行人当老师是最合适的，有主意能忍辱 - 能忍辱的木就是觉悟的木

    转化结果： - 阴木（顶撞）→ 阳木（正直） - 木生火：忍辱→热情→礼明照世 ```

    通关点2：火→土 - 放任

    ``` 核心方法： - 不报委屈，多找多看自己的问题 - 不去找别人的问题 - 不想干什么干什么，想吃什么吃什么，想说什么说什么（这是放任） - 放任自己害人又害己

    正确的做法： - 找自己的问题，不找别人的问题 - 多从自身找原因 - 减少抱怨，增强反思

    转化结果： - 阴火（怨恨）→ 阳火（礼明） - 火生土：放任→承载→信德厚土 ```

    通关点3：土→金 - 不怨人

    ``` 核心方法： - 不怨人，怨气堵在那就把脾胃的运化能力给堵住了 - 停止嘀嘀咕咕 - 内心有决断，知道这个事情该做不该做 - 小埋怨嘀嘀咕咕就是妄语，有阴土性损自己的脾胃

    怨气的危害： - 脾胃没有运化能力就看不开想不明白 - 没有智慧就分辨不出来哪些是好的哪些是坏的 - 一生庸庸碌碌俗俗气气总是跟着外面跑

    不怨人的本质： - 内心决断，知道该做什么 - 停止嘀咕，停止抱怨 - 智慧显现，能够分辨好坏

    转化结果： - 阴土（怨气）→ 阳土（承载） - 土生金：不怨人→感恩→义气如金 ```

    通关点4：金→水 - 感恩

    ``` 核心方法： - 感恩心找别人的好处 - 找别人的好处就是聚人缘吸阳光 - 找别人的不好处就是吸别人身上的阴气 - 最后自己身上的阴气也很旺

    感恩与聚缘： - 这个人的阳气很旺，大家都喜欢在他身边因为很舒服 - 找别人的不好处吸别人的阴气，无论长得多帅多漂亮，别人不喜欢在你边上呆着 - 他们呆你边上觉得难受阴冷阴冷的 - 所以就不能聚缘 - 这个就是找别人的不好处爱比较爱攀比，这个就是盗，盗别人的财就是损别人的义气

    感恩的本质： - 义气和财是绑定在一起的 - 缘聚财散（人缘好了财就散了） - 财聚人散（财聚到自己身上了人就散了）

    转化结果： - 阴金（刻薄）→ 阳金（义气） - 金生水：感恩→智慧→柔和如水 ```

    通关点5：水→木 - 认不是

    ``` 核心方法： - 认不是，不要给自己找理由开溜 - 不要说"下一次改"然后拖拖拖 - 改自己不好的小习惯 - 吃吃喝喝的小习惯就是戒酒，酒水就是养成的小性子小习惯

    认不是的本质： - 承认错误，不找借口 - 改掉小习惯，戒掉不良行为 - 水本来应该是清亮清亮的 - 里面有小错误在里面搅搅搅，肾水就浑了容易得很多病 - 那个就是憋闷出来的
