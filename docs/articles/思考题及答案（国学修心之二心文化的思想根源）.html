思考题及答案（国学修心之二 “心文化”的思想根源）

1、 《易经》 是中国人的第一经典，第一哲学，第一智慧。

2、中国佛教与古印度原始佛教不同的是，在于其 本土化 。

3、中华传统文化的基本内涵有： 儒家 文化的“积善之家必有余庆，积不善之

家必有余殃”。

4、国学 修心 的意义：内心要得到快乐，要调整自己的心态。

5、国学修心的方法与原理： 洗心 → 养心 → 正心 → 静心 → 明心 。

6、 心文化 ，这样的精神文化，体现出中国传统文化的智慧与圆融。

7、以观其妙书院提出的 国学修心，五步到心 系统，指出了传统文化在当代的价值

在于修心。

8、 心文化 是传统文化的大道根基。

9、依照因缘与因果，佛法的基础先从做 人 作起。

10、依照经典的源流，易经是用来 洗心 的；中医是用来 养心 的；儒家是讲 正

心 的；道家是讲 静心 的；禅宗是讲 明心 的，都离不开“心文化”。

11、《易·系辞上》：“圣人以此 洗心 ，退藏于密，吉凶与民同患。”可以说是《易

经》的名句。

12、易道精神，就是通过 乾坤 二卦所体现出来的精神。

13、 《黄帝内经》 奠定了中医养生学的理论基础，历代医家将其视为养生长寿

秘诀。

14、何谓“养心”？祖国医学经典《黄帝内经》认为养生先 养心 ，心平则寿长。

15、儒家文化一贯强调做人应努力自省，并要光明磊落，坦坦荡荡，这就是 正

心 。

16、《道德经》第十六章讲到致虚守静,老子认为心 静 才是归根。

17、禅宗是佛教中国化、本土化最鲜明的代表。最有标志性经典是 《六祖坛经》 。

18、国学修心系统将传统文化的内在视角高度概括为 “心文化” ，并以 “象

思维” 为研究方法，贯穿易、医、儒、道、禅诸家，结合授课对象的不同需要 洗

心，养心，正心，静心，明心 ，使学员掌握一套“修心开智”的学修方法。

19、易医儒道佛人生哲学不同，实践生命的修炼，所成就者分别称为 大人，真

人，圣人，仙人，觉人---佛 。
