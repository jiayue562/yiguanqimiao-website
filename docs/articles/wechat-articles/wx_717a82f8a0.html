✨四大核心更新，硬核又贴心
1️⃣ Scoop 原生安装 obsidian-cli
告别繁琐配置，现在只需一行命令：
scoop install obsidian-cli
版本 v0.2.3 已就绪，支持 list、search-content、daily、create 等全套命令。
价值：AI 直接操作你的本地知识库，读取+写入+管理，一气呵成！
2️⃣ IMA 云端备份集成
配置信息：知识库「以观其妙书院」，目标文件夹「龙龟神将备份资料」，API v1.1.2。
价值：自动备份 WorkBuddy 对话，批量同步 Markdown，智能分类。你的知识不仅在本地，更在云端安全无虞，手机随时查看。
3️⃣ 一键三向同步脚本
完整同步、增量同步、干运行预览，三种模式任你选：

完整同步（WorkBuddy → Obsidian → IMA）

powershell -File"三向同步系统\scripts\tri-sync.ps1"-Mode full

增量同步（只同步变更）

powershell -File"三向同步系统\scripts\tri-sync.ps1"-Mode incremental

干运行（预览不执行）

powershell -File"三向同步系统\scripts\tri-sync.ps1"-Mode full -DryRun
价值：彻底告别手动搬运，一键让知识在三大平台间流淌。
4️⃣ 智能文件过滤
同步规则：✅ 包含 *.md, *.json, *.yaml；❌ 排除 node_modules, .git 等；📦 单文件限 10MB。
价值：只同步干货，不浪费一寸存储空间。
