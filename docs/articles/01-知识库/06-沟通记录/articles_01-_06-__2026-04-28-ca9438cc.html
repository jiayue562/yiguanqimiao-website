核心内容

1. Claude Code 桌面版概览

Claude Code 桌面版是 Anthropic 推出的 AI 编程工具的桌面应用程序，在 CLI 命令行版本基础上增加了图形化操作体验。

平台支持

  • ✅ macOS
  • ✅ Windows
  • ❌ Linux（暂不支持）

    • 桌面版独有功能

    | 功能 | 说明 | |------|------| | 可视化差异审查 | 通过图形界面查看代码变更，比终端 diff 更直观 | | 实时应用预览 | 直接在桌面版中预览应用运行效果 | | GitHub PR 监控 | 监控 Pull Request 状态变化 | | 定期任务调度 | 支持定时自动执行编程任务 | | 文件拖放附件 | 支持拖拽文件作为上下文输入 |

    桌面版不支持的功能

    | 功能 | 说明 | |------|------| | 第三方 LLM 提供商 | 不支持 Bedrock/Vertex/Foundry 等第三方接入 | | Linux 平台 | 暂无 Linux 版本 | | 内联代码建议 | 暂不支持 IDE 内联补全 | | 脚本自动化 | 不支持 --print 模式和 Agent SDK 编程调用 |

    关键决策/结论

    1. Claude Code 桌面版适合的场景：需要图形化操作界面的开发者，尤其是需要可视化差异审查和实时预览的场景 2. CLI 版本仍是主力：对于高级用户和企业级部署，CLI 版本功能更完整，支持更多集成方式 3. 与龙心OS的关联：Claude Code 源码与龙心OS已有融合框架（v1.0），桌面版的可视化差异审查等能力可进一步丰富龙心OS的反馈层
