德育学位认证体系的架构

摘要：新中国建立以来，特别是改革开放以来，中国奉行拿来主义，教育的弊端已经显现，结合儒家体系的特点，对未来教育的学位架构体系进行初步的设计，期望恢复中国的德育本位主义教育。

历史背景分析：

  1. 二千五百年前：孔子开办了第一家私学，走出以文化人的第一步。
  2. 汉代董仲舒提出罢黜百家，独尊儒术，从此儒家方化成为民族之魂。
  3. 清代戊戌变法时提出课程改革，中学为体、西学为用。
  4. 民国时教育提出理念：通识为本，专识为末。
  5. 新中国建立时期，第一次全国教育工作会议在北京召开,会议指出“建设新教育要以老解放区新教育经验为基础。吸收旧教育某些有用的经验,特别要借助苏联教育建设的先进经验。” 当时的高等教育可以选择三种模式：即延安模式、苏联模式和民国政府模式。
  6. 2015年《高等职业教育创新发展行动计划》提出以立德为根本的指导思想，促进文化传承创新与传播。

目前教育弊端分析：

  1. 重知识轻德行的培养模式已经显露其弊端，食品安全问题从冰山一角暴露了道德沦陷之后的社会危机。
  2. 专识教育下人与人之间的勾通障碍开始显现，现代人都执着于各自专识教育的知识，各执一词，在思想上的融通变得越来越困难。
  3. 对外国教育方式不加鉴别的盲目引用，邯郸学步、防碍了本土文化的发展，特别是将自然科学的评价方法应用了人文社科领域，唯“知”唯“量”的评价标准，破坏了人文社会科学发展的生态。让我国的教育失去了创新发展的根基。
  4. 中国教育没有自已特有标准，在世界文化领域失去了话语权，论为国外教育的消费终端。千万学子回国适应困难，或留在国外为人所用，花出一大笔钱，学回一大堆无用的知识。

德育认证框架构建

儒家德育的核心为：仁、义、礼、知、信。对应于我们传统文化的五行，也对应着五行人格中不同的类别。仁、义、礼、智、信，可以成为网上人大德育人证的载体。由于德育认证项目的包容性，我们可以把目前的课程体系，划分为五个模块，（比如我们可以把所有技能科学划分到“智”这一类别）每个模块修满多少个学分，网上人大为其颁发德育认证书。这将为解决学分互转问题提供一个方便的解决途径。

德育认证层级的构建

《中庸》中提出修道要博学、审问、慎思、明辨、笃行。可以概括为三个层次。即文、思、修。加上这三个层级，则德育认证项目可拓展为五类三级。这要求在课程设计上也要“文”、“思”、“修”三者并重。当然也可以划分为五级对应“黄帝内经”中提到的25人。

德育认证三步走

由于历史的继承影响，德育认证项目还处于空白阶段，在政策层上很得到教育部技持，或在发展过程中会遇到一些困难。在推行上必定要经历一个过程。现设想德育认证项目大致可以分为三个阶段：

  1. 课程分类阶段，探索“仁”、“义”、“礼”、“智”、“信”提深层含义。从本土文化中发掘评价途径。探索德育课程混合式培养方式。整合培养机构。最终完成课程体系的建构。
  2. 非学历认证阶段。通过半年的混合式线上线下培养。培养部分高端人才。为德育认证进行口碑宣传。
  3. 将非学历认证的成果向学历教育延伸，实现德育认证课程的全面覆盖。

德育认证的教育形式

德育认证如何认证：这要分析在建国之初的三种教育模式：即延安模式、苏联模式和民国政府模式。其中延安模式与民国下府模式为本土模式，苏联模式为国外模式，无疑民国模式为官学模式，延安模式为民间模式，当时国共两党的对立分歧，采取本土模式中的任何一种，都容易在政治上产生派别之争，而消除争论的最好方式就是两边各打一棒，为此，国外的苏联模式取代本土模式成为当时政治选择之必然。但苏联模式并不是本土模式，时间一长必然出现水土不服现象。向本土模式回归也必然是历史之必然。在向本土模式回归中，官学的民国模式的通识化教育将成为学历教育的肥沃土壤，私学的延安模式的批判精神将由非学历教育引发为教育改革提供源源不断的动力。

当下，在本土化回归的道路中，延安模式将成为改革的先导。私学书院为改革浪潮中的先锋官，学院混合式培养模式线上线下的配合将成为这次教育变革的核心环节。

延安教育模式下：毛泽东在上个世纪年代曾赞赏书院允许不同知识领域之间进行争论,有密切联系基层和向人民群众开放的价值。'他在《湖南自修大学创立宣言》中评价书院道“一来是师生感情甚笃。二来,没有教授管理,但为精神往来,自由研究。三来,课程简而研讨

周,可以优游暇豫,玩学有得。”他认为“自修大学之所以为一种新制,就是取古代书院的形式,纳入现代学校的内容,而为适合人性便利研究的一种特组织。”`自修大学的理念被应用到延安的高校。

互联网是高度自由化的，与自修大学的理念比较容易融合在一起，同时延安模式也是革命性的，由延安模式主导直接引发了文化大革命，所以采取延安模式要把握一个度。这个度以不推翻现在的教育制度的权威为前提，与现在的教育制度并行融合。
