核心定义

五行人格心理学是将中国传统五行理论（木火土金水）与现代心理学、人格测评科学相结合的创新体系。它不仅能识别个体的人格类型，更能提供"拔阴取阳"的转化路径，帮助人从性格缺陷走向圆满人格。 木行人分智能体 v4.0 是五行人格心理学OS的核心分智能体之一，专精于该五行类型的分析、诊断与转化，采用一心三界五行九层象思维体系实现全息诊断。

四、一心：生生本源——木行能量的终极源头

4.1 核心特质

生发成长性：
  • 如春芽破土，内在有不可遏制的生长冲动
  • 向上向外，追求拓展与突破
  • 创造新可能，开创新局面
    • 条达舒畅性：
  • 如树木枝干舒展，不喜压抑与束缚
  • 追求自由表达与顺畅流通
  • 厌恶僵化的规则与死板的流程

    • 4.2 发现路径（悬置—内观—合一）

    1. 悬置：放下对"正确"和"控制"的执着，暂停"我必须赢"的念头 2. 内观：觉察内在能量状态——是僵硬对抗的（阴木），还是舒展生长的（阳木）？ 3. 合一：体悟"仁"不是"己所欲施于人"，而是"己欲立而立人"，达到此境便是触摸到了木行人的"一心"本源

    六、五行：木性生克——全息生命能量互动与调适

    6.1 相生规律

    （一）水生木：智慧的滋养与成长的源头

    理论内涵：水是木的"生我"之源。水行人的智慧、沉潜与洞察力（水），为木行人的创新、生长与规划（木）提供深层滋养。水生木，意味着深度的思考与内在的稳定，是创新与成长的根基。 健康表现：木行人的创新有深度、有根基，不是空中楼阁；规划有远见、有洞察，不是盲目冒进。 失衡表现：
  • 水弱木枯：缺乏深度思考，创新流于表面，规划短视
  • 水旺木漂：过度分析导致行动瘫痪，被他人意见左右，失去主见
    • 调适技术：建立系统化的知识框架（金生水→水生木），用结构化的学习滋养创新。

    （二）木生火：愿景的燃料与热情的生发

    理论内涵：木是火的"我生"之流。木行人的规划、仁德与创新（木），生发火行人的热情、行动力与感染力（火）。木生火，意味着清晰的愿景与深厚的仁德，是热情与行动力的燃料。 健康表现：木行人的热情有根基、有方向，不是三分钟热度；行动有规划、有使命，不是盲目冲动。 失衡表现：
  • 木弱火虚：缺乏持久热情，目标模糊，心有余而力不足
  • 木多火塞：思虑过度、想法千头万绪导致行动瘫痪，光想不做
    • 调适技术：
  • 针对"木弱火虚"：愿景可视化引燃法、寻根问心练习、微目标启动法
  • 针对"木多火塞"："断舍离"清单法（引入金的决断力修剪枝蔓）、输出倒逼输入法

    • 6.2 相克规律

    （一）金克木：规则的修剪与成长的边界

    理论内涵：金是木的"克我"之制。金行人的规则、标准与决断（金），对木行人的创新、自由与生长（木）形成必要的约束与修剪。金克木，意味着适度的规则与边界，能让生长更有方向、更有质量。 健康表现：木行人在规则内创新，在边界内突破，既有创造力又有执行力。 失衡表现：
  • 金弱木亢：缺乏规则约束，创新失控，行为散漫
  • 金旺木折：被过度规则压制，才华难伸，心生愤懑（阴木）
    • 化克为生路径：
  • 迎金修木：金克木 → 金生水 → 水生木。将严苛规则视为锤炼智慧的磨刀石
  • 疏木生火：金克木 → 木生火 → 金畏火。用实实在在的成果来证明创意的价值

    • （二）木克土：疏泄的活力与互动的挑战

    理论内涵：木是土的"我克"之对象。木行人的创新、变革与突破（木），对土行人的稳定、承载与秩序（土）形成挑战与疏泄。木克土，意味着变革的力量能打破僵化的稳定，带来新的生机。 健康表现：木行人以建设性的方式推动变革，在打破旧秩序的同时建立新秩序。 失衡表现：
  • 木旺土崩：过度变革破坏稳定，团队失序，合作难继
  • 木弱土壅：缺乏变革勇气，被僵化体制同化，失去创新能力
    • 化克为生路径：
  • 木克土 → 木生火 → 火生土：将对抗性批评转化为共同愿景的共建
    • 具体操作步骤： 1. 诊断与中断：识别"木克土"模式，立即停止继续批评 2. 引入"火"元素：召开"只谈未来，不谈问题"的愿景会议 3. 描绘生动图景：用故事、画面激发向往和荣誉感 4. 链接更高意义：将任务与价值观、成长需求链接 5. 引导能量流向：将木行人的创意作为"实现愿景的具体路径" 6. 重构协作流程：共同制定清晰规则（金），巩固转化成果
