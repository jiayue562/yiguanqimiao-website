🌟 性识无定与转识成智

性识无定

含义： 未解脱者心性飘摇不定、心猿意马，没有一刻停止，难以控制念头，造善造恶都不自主，带着很大的偶然性。 根本原因：
  • 第七识（潜意识）里蕴藏着"我"的种子
  • 我喜欢、我不喜欢、我爱、我不愿
  • 这些都是催生恶因的种子

    • 转识成智

    不是把第七识扔掉，而是将恶的一面转化成好的一面：
  • "我"要修道
  • "我"要证悟
  • "我"要解脱度众生
    • 性质变了，叫转识成智

    🔥 五无间地狱

    五种造业

    1. 不孝敬父母，甚至杀害

    2. 出佛身血（谤佛谤法谤僧）

    3. 侵损常住（毁坏道场财物）

    4. 心非沙门（穿僧衣不守戒律或假装僧人）

    5. 偷窃（尤其是偷窃常住财物）

    五种无间特点

    | 无间 | 含义 |

    |------|------|

    | 时间无间 | 受苦永无间断 |

    | 形无间 | 一人受罪满床，千万人受罪也满床 |

    | 苦无间 | 痛苦永无中止 |

    | 感无间 | 任何人都会感召 |

    | 命无间 | 死死生生永不停止 |

    🌊 修行三种道

    | 道 | 适合人群 | 方法 |

    |----|---------|------|

    | 自解脱道 | 第一类：本性即佛的学生 | 直接了悟心性，稍加训练即可自解脱 |

    | 转化道 | 第二类：金刚智慧的学生 | 转化烦恼为菩提，有转化的智慧 |

    | 出离道 | 第三类：修行者 | 遵从戒律，发展内心善的力量克制恶行，从无常痛苦中出离 |

    核心洞见

    一切外境都是自心能量的投射，是你自己制造了它。

  • 夫妻在一起：因为福报相等
  • 国籍相同：有共同的根器和智慧
  • 所投生的国家、星球：都是自己的果报

    • 世界的差异

  • 有些世界有地狱，有些没有
  • 有些有女人，有些没有
  • 有些有佛法，有些没有
  • 这都是不同世界的众生共业所感
