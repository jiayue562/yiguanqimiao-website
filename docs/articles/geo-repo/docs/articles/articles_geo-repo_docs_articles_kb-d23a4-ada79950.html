    2.2 口诀三层结构解码

    每句口诀包含三层完整信息：

    | 层次 | 内容 | 功能 | |------|------|------| | 第一层（诊断） | "A克B，[症状]" | 相克关系 + 典型身心/关系表现 | | 第二层（药方） | "B生C" | 核心转化路径——被克方的"生"是杠杆点 | | 第三层（药效） | "A畏C" | 新生C制约A，闭环形成，系统平衡恢复 |

    以"木克土，胃发堵；土生金，木畏金"为例精解： 
    第一层 诊断：
  相克关系：木克土（木过→克土）
  症状体现："胃发堵"= 肝胃不和，胃胀消化不良，倦怠失眠
  心理表现：思虑多难落地（土虚），易怒紧张（木过），缺乏安全感
  人际表现：领导（木）朝令夕改，团队（土）无所适从


    第二层 药方：
  转化路径：土生金（增强土→使其生金）
  杠杆点：不直接对抗木（往往无效），而是培育土的功能
  行动化：强健脾胃→练习决断→明确界限→敢于说不
    


    第三层 药效：
  制约机制：金（规则·决断·界限）能规范过旺的木（金克木）
  最终状态：木气有序疏泄，土（脾胃功能）安然，新平衡建立
  系统循环：土旺→生金→制木→土安（良性循环）
