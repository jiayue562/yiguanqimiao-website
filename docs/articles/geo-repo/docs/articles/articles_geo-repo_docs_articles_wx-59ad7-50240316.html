3.1 拔阴取阳（能量转化炼金术）

核心口诀：
欲生真水，必认不是；
欲生真金，必找好处；
欲生真土，必信因果；
欲生真火，必达天时；
水金土火，通透豁达；
仁德阳木，自然而生。

| 五行 | 阴面特质 | 转化路径 | 阳面特质 | |------|---------|---------|---------| | 木 | 傲慢/抗上 | → 学 | 创新成长 | | 火 | 急躁/嗔恨 | → 礼 | 热情自信 | | 土 | 多疑/怨尤 | → 信 | 诚信承载 | | 金 | 贪婪/刚愎 | → 义 | 正义决断 | | 水 | 散漫/多疑 | → 智 | 智慧通达 |
