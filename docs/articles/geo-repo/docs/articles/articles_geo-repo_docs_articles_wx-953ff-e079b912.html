📖 黄帝内经·阴阳二十五人

核心定义

《黄帝内经·灵枢·阴阳二十五人》是五行识人"纯象阶段"的集大成之作。

核心贡献：根据五行理论将人分为木、火、土、金、水五形，每形再分五类，共二十五种类型。
