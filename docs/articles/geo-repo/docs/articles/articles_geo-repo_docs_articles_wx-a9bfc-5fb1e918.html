核心定义

🔥 火行人完整体系

核心定义：火行人以「光明觉知」为一心，具有光明性与热情性的双重特质。如火焰温暖照耀，火行人天生具备感染力、引领力和礼明照世的使命，是连接者、激励者和社交场域的点燃者。
