5.2 最近发展区与五行层级

核心概念：个体实际水平与潜在水平之间的区域 融合应用：
  • 最近发展区 = 九层的"当前层级"与"潜在层级"
  • 支架式支持 = 五行相生能量滋养
  • 内化 = 拔阴取阳转化
    • 实践公式： 
    潜在发展水平 = 当前层级 + 最近发展区
             = 阳面特质 + 五行相生

    5.3 社会建构与三界协同

    核心概念：知识是社会建构的产物 融合要点：
  • 社会建构 → 灵界价值观建构
  • 文化工具 → 五行能量显化
  • 协作学习 → 五行相生协同
