核心定义

五行人格心理学是将中国传统五行理论（木火土金水）与现代心理学相结合的人格分析体系。

九层发展层级详解 · 五行人各层级表现

一、九层概述

1.1 核心定义

九层发展阶梯是"一心→三界→五行"协同状态的量化体现，本质是五行阴阳能量在三界中"原创性显化—概念化阻滞—原创性断裂"的动态光谱。

1.2 三区间特征

| 区间 | 层级 | 原创性程度 | 三界状态 | 阴阳特征 | |------|------|-----------|---------|---------| | 健康区间 | 1-3级 | 顺畅 | 三界共振 | 阳面主导 | | 一般区间 | 4-6级 | 阻滞 | 三界拉扯 | 阴阳对抗 | | 不健康区间 | 7-9级 | 断裂 | 三界失衡 | 阴面主导 |

三、一般区间（4-6级）

3.1 第四层级：失衡

通用特征：
  • 灵界：意向维度被概念固化，原创性受阻
  • 心界：偏执敏感、爱较真，五行意象被阴面概念扭曲
  • 身界：固执己见、难变通，五行物象被阴面概念锁定
    • 三界状态：轻度失衡，心界主导 拔阴取阳技术核心：用所胜五行原创性，去概念化，反思自身局限

    3.2 第五层级：人际控制

    通用特征：
  • 灵界：意向维度被"证明自我"概念扭曲，原创性断裂
  • 心界：傲慢/急躁/固执等，阴五行意象被概念控制
  • 身界：刻意对立、刷存在感，五行物象被阴面概念固化
    • 三界状态：中度失衡 三界失衡链条："灵界概念化→心界概念化→身界概念化" 拔阴取阳技术核心：心界转化，去对抗概念，回归学习/反思/行动原创性

    3.3 第六级：过度补偿

    通用特征：
  • 灵界：意向维度被"掩饰自卑"概念驱动，原创性扭曲
  • 心界：冲动激进、爱冒险/焦虑，五行意象被概念干扰
  • 身界：不计后果、求关注，五行物象被概念显化
    • 三界状态：中度失衡，身界主导 拔阴取阳技术核心：用相生五行原创性，去概念化，沉下心落地实践
