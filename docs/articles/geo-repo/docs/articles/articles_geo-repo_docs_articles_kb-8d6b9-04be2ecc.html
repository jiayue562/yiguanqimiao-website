🌲 五大维度性格特征

维度一：曲直（才气独特 ⇌ 傲慢孤僻）

定位: 最表层的性格，聚焦"独特性"的外在表现

顺境·才气独特

| 特征 | 描述 | 心理学解释 | |------|------|------------| | 足智多谋 | 具有想象力、适应性和可变性，视灵感高于一切 | 不墨守成规，善于发觉新方法 | | 感受丰富 | 容易感知自己的情绪和内心世界，敏感、重视内心感受 | 能体验到更深的情绪状态 | | 才华横溢 | 充满新思想，乐观、自然、富有创造性和自信 | 具有独创性思想和对可能性的强烈感受 | | 思路开阔 | 高度创造力，思路开阔，观念新，富有想象力 | "点子型的人才"，不受条条框框约束 | | 理想主义 | 透过自我感受来认知周围世界，长得高看得远 | 不在乎现实，更倾情理想 |

逆境·傲慢孤僻

| 特征 | 描述 | |------|------| | 个人主义 | 有创造力但比较个人主义，从自己想法出发，不太考虑周围人感受 | | 我行我素 | 能够看到新计划实行后的结果，更喜欢以自己的方式行事 | | 自卑羞怯 | 在人群中羞怯，有不自然的姿态，有强烈的自卑感 | | 刻板固执 | 常常忽视其他观点的作用，而且有时会很刻板 | | 不善分享 | 内向，很难被人了解，但愿意同自己信任的人分享内在自我 |

维度三：舒畅腾上（正直不屈 ⇌ 偏执顶撞）

定位: 人际互动中的价值观与原则坚守

顺境·正直不屈

| 特征 | 描述 | |------|------| | 专注执着 | 坚定、富有理想化，珍视正直，十分坚定以至达到倔强的地步 | | 讲求实效 | 是优秀的解决问题的人，能够掌握必要的事实情况 | | 有担当力 | 正直有主意，敢做敢当，心口如一，常存悲天悯人之心 | | 超级理想 | 渴望表达超级的理想，只和能分享理想的人有联系 |

逆境·偏执顶撞

| 特征 | 描述 | |------|------| | 严肃偏执 | 一旦意见遭到反对，可以是非常强硬的，甚至是无情的 | | 过分理性 | 过分依赖条理性分析，会忽视与别人有关的事物 | | 迟滞不前 | 拒绝在整个计划中存在一点不符合逻辑的地方 | | 破坏规则 | 喜欢挑战权威、常规和传统观念，检验社会的政治的宗教的价值观念 | | 缺少耐心 | 对常规的细节缺少耐心，如果一个计划需要过多细节会失去兴趣 |

维度五：内在成长（安静内向 ⇌ 疏远冷淡）

定位: 自我独处时的状态，体现对自我空间的需求

顺境·安静内向

| 特征 | 描述 | |------|------| | 独立自主 | 属于专家型人才，习惯在安静的环境里独立工作 | | 思辨力强 | 能轻易地看出事物具有的可能性，是头脑灵活的思想家和长远规划者 | | 外表平静 | 外表平静、缄默、超然，但内心却专心致志于分析问题 | | 逻辑分析 | 很有条理和分析能力，对要求推理和才智的任何事情都很擅长 |

逆境·疏远冷淡

| 特征 | 描述 | |------|------| | 低交际性 | 在社会情境中感觉不轻松，在人群中畏怯退缩、缺乏自信心 | | 离群索居 | 倾向于悠闲和放松，宁愿躲在幕后，让其他人谈论 | | 孤寂寡言 | 从小就有那么点儿不合群，喜欢一个人呆着 | | 疏远冷淡 | 不容易感受到各种积极的情绪，热情不高、平静的、冷淡的 | | 孤独被动 | 往往是孤独者，不寻求甚至主动避免社会刺激 |

三界

| 界 | 阳木状态 | 阴木状态 | 修炼方向 | |----|----------|----------|----------| | 心界 | 仁爱、善良、有担当 | 忧愁、善变、缺乏控制 | 修炼仁德恒定心 | | 性界 | 核心是"仁"，正直不屈 | 偏执、顶撞、抗上不服 | 向内看，学会柔和 | | 灵界 | 化俗救世的理想 | 孤傲、疏远、冷淡 | 链接高维能量 |

九层

从种子到参天大树的成长旅程：

1. 种子层: 潜能蕴藏，待时而发 2. 萌芽层: 初现生机，探索世界 3. 幼苗层: 茁壮成长，吸收养分 4. 小树层: 独立自立，迎风而立 5. 大树层: 枝繁叶茂，庇护他人 6. 参天层: 高耸入云，视野开阔 7. 森林层: 成林成片，生态共生 8. 古木层: 历经沧桑，智慧沉淀 9. 道木层: 与道合一，生生不息

修炼方向

木行人的修炼关键在于"柔和"，学会调达。

| 修炼方向 | 具体方法 | |----------|----------| | 修炼柔和 | 练习倾听、接纳不同意见，将向外生长的能量适当收回 | | 提升耐受力 | 增强对批评、挫折和常规细节的耐受 | | 聚焦与深化 | 克服虎头蛇尾和缺乏控制，练习聚焦目标 | | 关注他人感受 | 将注意力从自我表现转移到体察他人需求 |
