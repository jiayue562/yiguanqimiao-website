    六、与木行人相处指南

    理解木行人

  • 需要空间和时间成长
  • 重视自主性和创造性
  • 对限制敏感
  • 需要目标和愿景驱动

    • 沟通建议

    1. 给予选择权 2. 强调愿景和意义 3. 避免过度控制 4. 鼓励创新尝试

    激发木的正向特质

  • 提供挑战机会
  • 给予成长资源
  • 认可创新贡献
