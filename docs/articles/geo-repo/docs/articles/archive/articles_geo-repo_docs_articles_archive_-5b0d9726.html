    化克为生路径（木行人专属）

    化克为生的核心定义：不直接对抗或消除"相克"的力量,而是洞察相克关系内部蕴藏的转化密码,通过引导能量流向"相生"的方向,将原本对抗性、消耗性的"克制"局面,创造性转化为滋养性、建设性的"相生"循环。 核心公式： 
    A克B导致B失衡 → 强化B生C的功能 → C能量壮大后,对A形成制衡（C克A）或转化（C生A）→ 系统形成新的A、B、C共生循环

    🎯 B=MAP行为设计框架

    :B=MAP是木行人分智能体的标准行为设计工具,基于福格行为模型（B=动机M=能力A=提示P=提示）。

    框架结构

    :B（行为）= M（动机,Motivation）+ A（能力,Ability）+ P（提示,Prompt）

    木行人专属设计原则

    | 维度 | 阳木（健康） | 阴木（失衡） | 转化策略 | |------|-------------|-------------|---------| | 动机M | 仁德驱动,利他成长 | 傲慢驱动,证明自我 | 从"证明正确"转向"帮助成长" | | 能力A | 规划创新,正直担当 | 固执对抗,钻牛角尖 | 从"硬对抗"转向"软学习" | | 提示P | 明确愿景,阶段性目标 | 模糊冲动,缺乏计划 | 设定清晰提示：场景+时机+具体行动 |

    设计模板

    案例1：改善人际关系（木克土转化） 
    :B（行为）：在团队会议中,当别人反对我的方案时,先倾听再讨论
  = M（动机）：建立和谐团队氛围,而非证明自己正确
  + A（能力）：忍辱修习,听取意见,整合优化
  + P（提示）：会议前提醒自己"今天的目标是倾听和整合"
    案例2：提升执行力（木生火） 
    :B（行为）：每周设定3个小目标,并在24小时内启动第一个行动
  = M（动机）：将创新转化为实际成果
  + A（能力）：阶段性拆解,快速启动
  + P（提示）：晨起问自己"今天的第一步是什么？"
