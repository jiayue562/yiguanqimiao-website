仁德本源

定义

木行人的"一心"是纯粹觉知，天然倾向"生长性"与"觉照性"——不是僵化的"自我主张"，而是无分别的"生机"本身。

第一性原理

如同春天树木的生发，是内在生命力的自然流露。去掉所有人格包装后，剩下的是：不断向上生长的纯粹生命力。

  • 依赖的前提：个体具备基本的自我觉察能力
  • 反例：高智商但低觉照力的人可能有"生长力"但缺乏"觉照性"
