💎 核心见地

噶达陇竹尼美（核心公式）

本来清净（空性）× 本自圆满（光明）= 不二

| 概念 | 含义 |

|------|------|

| 本来清净 | 空性，一切法没有任何本质 |

| 本自圆满 | 光明潜能，具足一切的光明 |

| 不二 | 空性与光明的不可分割性 |

一心的定义

纯粹的觉知：无分别、无对境、纯然知晓，是赤裸、坦然、明明了了的觉性。

第二要：确信无疑

核心确信： 你是觉性而非身心 三重确认：

1. 身体是无常的，觉性不生不灭

2. 语言是无常的，觉性超越语言

3. 念头是无常的，觉性本自清净

确认方式： "此即我之本性，确认无误"

✨ 三本初智慧

| 智慧 | 内容 |

|------|------|

| 本体空性 | 心性的本体是空的，无实质 |

| 自性光明 | 空性中自然具足光明智慧 |

| 能量持续显现 | 大悲周遍，智慧能量不断显现 |

🕯️ 阿底上师瑜伽

修持步骤

1. 观想：明点白"阿"（声、光、光线）

2. 完全放松：不造作，不紧张

3. 不追随念头：念头升起，不跟随

4. 认证状态：不断认出觉性本面

观想要点

  • 明点：一个纯白色光点
  • 白"阿"：藏文阿字，象征本初状态
  • 声、光、光线：三位一体，超越形相

    • 💡 日常修持要点

    悟空境界

  • 悟空不是消极虚无，而是积极智慧显现
  • 对缘起性空真理的现量体证
  • 超越概念思维，安住于无造作的自然状态
