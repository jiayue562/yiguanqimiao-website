一、什么是逻辑

1.1 从语义说起

  • 词源："逻辑"音译自希腊语"逻各斯(λογος)"，原义包括言辞、理性、秩序、规律
  • 译名演变：名学→辩学→名辩学→理则学→逻辑（严复最早译"逻辑"，20世纪30-40年代后通用）
  • 创始人：古希腊亚里士多德，《工具论》奠定古典形式逻辑基础
  • 日常四义：①客观规律（"事物发展的逻辑"）②理论观点（"强盗逻辑"）③思维规律④逻辑学本身

    • 1.2 思维和语言

  • 思维：人脑对客观事物的概括和间接反映
  • 语言：思维的物质外壳，二者密不可分
  • 抽象思维三要素：
    • - 概念：思维的最小单位，反映事物本质属性 - 判断：对事物情况有所断定（肯定/否定，全称/特称/单称） - 推理：由已知判断推出新判断，是思维最高层级

    1.3 思维的形式与内容

  • 思维内容：反映的对象及其属性
  • 思维形式：表达思维内容的联结方式（逻辑形式）
  • 逻辑常项：决定逻辑形式的性质（如"所有""都是"）
  • 逻辑变项：可代入不同概念（如S、P）

    • 三、现代逻辑的基本思想与方法

  • 形式化思想：通过人工语言（符号系统）精确表达逻辑结构
  • 三大方法：①符号化 ②公理化 ③形式证明
  • 分支拓展：模态逻辑、道义逻辑、语言逻辑等

    • 五、预备知识

  • 内涵与外延：内涵是本质属性，外延是事物范围
  • 命题与判断：判断是对命题的真值赋值
  • 推理有效性：只与形式相关，与内容无关

    • 金句

    "所有股票都是有价格的；所有生物都是进化的；所有法律都是具有强制性的。三者内容完全不同，但逻辑形式一致，均为'所有S都是P'。"
    "谣言止于智者的本质就是逻辑判断能力的体现。"
