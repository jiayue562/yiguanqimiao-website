    核心问题

    1. 缺乏仁爱之心（木行核心是仁） 2. 人际关系困难（孤僻、冷漠、缺乏社交技能） 3. 情绪管理差（易怒、消极、情绪化） 4. 自我认知偏差（自恋、自我否定、自我欺骗） 5. 缺乏主动性（被动、随波逐流、怕麻烦）

    转化方向（拔阴取阳）

    1. 培养仁爱之心 → 阳木的仁爱、孝顺、尊敬 2. 提升社交能力 → 阳木的亲和力、社交技能 3. 改善情绪管理 → 阳木的情绪稳定、积极思维 4. 客观自我认知 → 阳木的自知之明、自信 5. 培养主动性 → 阳木的主动、进取、领导力
