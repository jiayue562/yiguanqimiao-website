二、三层结构的阴阳判断

2.1 身界的阴阳判断

身界观察的是我们的身体状况和形态特征： 
身界阴阳判断 ════════════════════════════════════════▶
          │
          ├── 【阳性身界】
          │     ├─ 挺拔有力、灵活敏捷
          │     ├─ 体态端正、不佝偻
          │     ├─ 面色红润、眼神明亮
          │     └─ 动作协调、反应迅速
          │
          └── 【阴性身界】
                ├─ 佝偻僵硬、懒散无力
                ├─ 体态松垮、缺乏活力
                ├─ 面色暗淡、眼神涣散
                └─ 动作迟缓、反应迟钝

五行人身界特征对照

| 五行 | 阳性身界 | 阴性身界 | |------|----------|----------| | 木 | 挺拔如松、玉树临风 | 佝偻僵硬、枯藤老树 | | 火 | 端庄大气、守礼有节 | 拘谨紧缩、声音破音 | | 土 | 稳健朴素、表里如一 | 笨拙拖沓、反应迟钝 | | 金 | 活泼有力、洁净整齐 | 单薄瘦弱、懒散迟钝 | | 水 | 圆润亲和、眼神明亮 | 松软下坠、眼神涣散 |

2.2 心界的阴阳判断

心界观察的是我们的情绪和心态： 
心界阴阳判断 ════════════════════════════════════════▶
          │
          ├── 【阳性心界】
          │     ├─ 心情愉悦、积极向上
          │     ├─ 情绪平和、稳定
          │     ├─ 待人友善、易于相处
          │     └─ 有主见而不固执
          │
          └── 【阴性心界】
                ├─ 焦虑低沉、情绪低落
                ├─ 烦躁不安、容易发怒
                ├─ 待人冷漠、难以亲近
                └─ 优柔寡断、缺乏决断

五行人阴性能量表现

| 五行 | 阴性能量 | 情绪表现 | |------|----------|----------| | 木 | 怒气 | 抗上不服人、顶撞、偏执 | | 火 | 恨气 | 急躁、好争理、爱面子 | | 土 | 怨气 | 多怨、呆板、疑心重 | | 金 | 恼气 | 爱计较、好比较、多恼 | | 水 | 烦气 | 多烦、多虑、自卑 |

2.3 灵界的阴阳判断

灵界观察的是我们的价值观、直觉和灵性层面： 
灵界阴阳判断 ════════════════════════════════════════▶
          │
          ├── 【阳性灵界】
          │     ├─ 直觉敏锐、灵感丰富
          │     ├─ 价值观清晰、在道上
          │     ├─ 创造力强、善于创新
          │     └─ 魂神安定、两眼有光
          │
          └── 【阴性灵界】
                ├─ 直觉迟钝、灵感枯竭
                ├─ 价值观混乱、偏离正道
                ├─ 思维混乱、创造力丧失
                └─ 魂不守舍、两眼无光
⚠️ 特别说明：灵界的判断最难，需要相处了解。除非开天眼，否则需要与对方接触后才能判断其是否"在道上"。

五行山警示：
