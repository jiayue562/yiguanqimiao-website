同构 2: 网络涌现

五行图谱: 
单个五行节点 → 无特殊价值
纳入五行体系 → 自动与所有节点形成生克关联
整体涌现 → 系统认知、跨界启发
Obsidian 知识网络: 
单个笔记 → 信息孤岛
通过双向链接互联 → 知识神经网络
整体涌现 → 自主关联、系统认知
本质: 网络价值 > 节点价值之和
