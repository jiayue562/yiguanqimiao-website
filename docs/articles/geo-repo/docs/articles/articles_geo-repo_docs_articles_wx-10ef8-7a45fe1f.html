核心定义

五行人格心理学是将中国传统五行理论（木火土金水）与现代心理学相结合的人格分析体系。

💧 水行人完整体系

核心定义：水行人以「润泽觉知」为一心，具有润泽映现性与沉潜含藏性的双重特质。如冬水深沉蕴藏，水行人天生具备深度洞察力、意志力和智慧沉潜的使命，是深度思考者、策略家和能量的守藏者。

| 三界 | 阳水显化（健康） | 阴水失衡（不健康） | |------|----------------|-------------------| | 身界 | 体型深沉，动作缓慢而有力，面色深润，眼神深邃 | 体型虚胖或消瘦，动作迟缓，易发肾虚、腰痛、水肿 | | 心界 | 情绪以深沉、智慧、意志为主；思维善于洞察本质 | 情绪以烦躁、恐惧、封闭为主；思维疑神疑鬼 | | 灵界 | 价值观核心是"智"——深度智慧、意志坚定；使命"洞察与守护" | 价值观扭曲为"痴"（执迷恐惧）；使命扭曲为"封闭与逃避" |
