二、象思维

1、学习传统文化的入门处是体认"象思维"

《易经》是中国人的第一经典，第一哲学，第一智慧.来自不同时代的三位圣人的同一缕思维曙光为悠悠八千年华夏民族的文化点燃了薪火。这缕曙光就是易道的独特思维--“象思维”。它，开启了中华民族生生不息的灿烂文明。

易道之思不仅仅是中华民族最古老、最原创的心智模式，其显现的“象思维”智慧构筑了中华文明最基本的特征和最本原的方法!

我们的古圣先贤，伏羲氏、周文王和孔夫子从不同时空维度创立和丰富了易道的精神， 对易道之象作直觉体悟之观，妙悟易道观象心法，留给后人以“象思维”视野下的中国智慧。“易道广大，无所不包”是《四库全书》对它的评价，那么“易道”到底是什么呢？ “象思维”对我们的事业和生活又能带来多大的帮助？ 在今天，对我们的社会又将起到怎样的作用呢？

要想了解“易道”从而掌握并运用它成就我们的事业与生活，和谐人与人、人与社会、人与自然的关系，就需要回归到东方的“象思维”中来，开启潜藏在每一个中国人意识深处的智慧密码，而非凭借西方的逻辑思维方式。

“象思维”作为动态整体直观的悟性思维，最富于原创性并且比概念思维更为本原。用概念思维可以透彻解读西方形上学经典,却未必能透彻解读中国诸家经典. 中西在思维方式上不同，主要表现为最高思想范畴，其性质根本不同。西方是理性的逻辑概念思维方式；而中国表现为悟性的隐喻的“象思维”方式。

重新打开被西方中心论长期遮蔽的"象思维",是把握中国传统经典本真的需要."象思维"是人类最本原的思维,是前语言、前逻辑的思维,又是富于原创性的思维.语言逻辑都经过"象思维"孕育而产生. “轴心时期”（公元前800年至公元前200年），中国语言文字与西方语言文字在符号化中走着不同的道路。西方拼音文字表现为左脑优势。中国的汉字仍然保有象形性根基,表现为左右脑均势。所以“象思维”是人类所共有的本原性思维，人类后来文明中的一切创造都萌芽于其中。对现代国人而言，“象思维”不仅可以激发灵感，还是创新的源泉.

2、“观象”是回归国学之源的修炼方法

中国文化的特殊性在于语言文字和逻辑产生之后,由于其语言文字在符号化中仍然保留象形性根基,以及在国学源头《易经》中,把中国成熟的思维方式显示为"观物取象"和"象以尽意",这就使得中国易、医、武、儒、道、禅经典都主要是"象思维"产物.“国学”始于“象”，源于“象”，没有对中国传统文化的核心思维------观象(右脑潜意识训练)的体悟，就无法升起对传统文化本质内涵和基本特征的认识。换而言之，如果我们不能了解古圣先贤实证的境界，无论是儒家的，道家的，医家的,武家的,还是禅宗的经典，在那个时空状态下，我们的祖先是在高度右脑“象思维”状态下因感悟而创造的，仍依靠我们有限的左脑思维方式（非中国人之特色，实西化教育后天而为）去诠释我们的原典，不仅不可能回归和真正把握经典的本真，反而会与古人之心相隔千里，而且这种对西方主流思维方式的依赖会分裂我们的“天人和谐”之美.我个人的比喻是，这就好比是让中国人用刀叉吃饺子一样别扭，那个饺子，从馅到皮，它是和合的，无需用刀叉去分割.而基于这样的"内容"，最完美的形式是用筷子，一阴一阳去合住它.保和，太和的思维是我们培育悟性和推进原创的动力。

观象---用象的方法来思考（象以尽意）是研究中国传统文化的本原方法.国学乃内省、内证、内修、内观之学,直觉、动态、整体、感悟,用象的思维观察世界,用象的思维解决我们身心灵的问题:只要循着洗心->养心->正心->静心->明心的次第去修炼,就不会入国学之宝山却空手而归.

3、“观象”具有融贯古今,会通中西的品质

“易道”是“易贯儒道禅医武，道统天地人心灵”的简缩。这个“道”是“易”之“道”，落实在“修心开智”上。伏羲氏、周文王和孔夫子三圣穿越时空，以观象思维汇通易道，是中华民族修心开智的点灯人。

“观象”的修炼是要我们开智慧，即修心开智，修证中华八千年历代古圣先贤流传下来对世界，对社会，对家庭的生命哲学。《易经》本具的智慧是引导所有有缘接触它的人亲身实践“自强不息，厚德载物”的易道精神，真正提升自己生命的品质！实践生命的修炼：健康——快乐——智慧，要达到这个境界，知易行难，需要我们效法古圣，以身心性命的修证，在自己内心的最深处点燃光明，照亮我们本具的潜能！

如果说儒道禅医武互补的结合点在“象思维”,那么易贯儒道禅医武的贯通点就在于“观象”的修炼方法!

“观象”指开发中国人主流文化基因，回归原创的、动态的、整体的、直观的悟性思维的方法.在这里，现象和本质是完美的结合，时空一体，心物一元，是超越了思维的“象”。我们在具备了中国传统文化修养的同时，可通过自身努力打破中西的思维界限而求得会通。

西方形上学理念从亚里士多德开始,显示为实体性、对象性、现成性.而中国的太极、道、心性、禅则显示为动态整体的非实体性、非对象性、非现成性.把握实体,需要把握主客二元，用理性的逻辑的概念思维,如定义、判断、推理等；而把握动态整体或非实体的太极、道等,则需要天人合一，用悟性的隐喻的诗意联想的"象思维"。

历史的经验教训是,在我们打开逻辑概念思维这扇窗户时,却错误地关闭了"象思维"这扇窗户.今天,在我们重新打开"象思维"之窗时,决不是要把逻辑概念思维之窗关上.我们应记住明代哲人徐光启的箴言:"会通以超胜".

"象思维"的研究不仅对于中国思想文化振兴具有重要意义，而且具有融贯古今与会通中西以推进思想文化创新的广阔前景。以观其妙书院以 “国学管理，修心开智”为己任，专注于开拓、发扬东方传统文化价值理念，融会贯通中华传统智慧在现代管理中的应用，以深具实效性和持续性的产品服务于快速成长的个人、企业、政府部门及非盈利性组织。

中西思想文化核心问题是“象思维”与逻辑概念思维会通的问题。“会通以超胜”的文化观，不仅适合中国，而且具有普遍意义。会通不是一方取代另一方，而是促使两种思维方式、两种精神境界有机地连接起来。我们所为正是帮助人们开放身心，体悟真理，实践道法。一旦我们通过调伏自心而止息概念性的思维，才有可能回归原创性的象思维，我们才会本真地体认到心性。惟如此，所有修持和创造才可能趋于正道。

4、“观象”思维的修炼，是构建通向和谐世界的思维高速通路

"象思维"的渊源上溯到古代三圣，伏羲氏、周文王和孔夫子都是在历史上真实存在过的人物，在现实中我们似乎在疑惑——我们现在已不存在圣人们在过去经历的那样一种高度（右脑）象思维发达的状态，但这丝毫不能影响到我们现在体认国学的精神与作用！通过体认国学的精神，古圣显现的是一个完满境界，如以两字统之或可谓“阴阳”，如以一字统之或可谓之“和”。这种信息无时无处不存在。所以我们读经研经的时候那样的一种虔诚，对古圣先贤的那样的一种尊敬，逐渐的我们的身心会远离浮躁与粗猛。对国学修炼的认识和感受，随着内心的清静，我们愈接近“洁净精微”（孔夫子语），愈可以接收到这种微妙的信息，直至超越现象存在的感受，开启本质境界的感悟，这也许就是“易无思也，无为也，寂然不动，感而遂通天下之故”的境界吧。

在诱发回归生命潜能的过程中需要我们坚持作“观象”思维的训练，我理解依据象思维的生命潜意识良能与人相处，来指导自己的身心，把我们身心灵上面那些负面的信息全部去掉，即是用功。“观象思维” 训练的收益首先表现在自己的身心是十分和谐快乐的，也就是身体健康，心情愉悦，直至开启生命的智慧，能够产生实实在在的力量和作用。易经有卦象洗心（圣人以此洗心）之说，直至修炼至“洁净精微”的高能生命体，就需要借助象思维的独特表达系统，也就是要依据国学的思维去表达，而不是用以往左脑的逻辑思维判断推理。

第三个功课就是每一行人他跟他相隔的那两个人都有相克关系，这种相克的关系不一定表现出来见面就吵架，他也可能是一种强烈的吸引。这就好像有人穿衣服，我们以前说红配绿赛狗屁，但现在大家说红配绿穿起来最有个性。所以你如果是红的，碰上绿的，可能就是强烈的吸引。强烈吸引的背后是一个比较难做的功课，因为彼此不同，有时候就是因为彼此不同才强烈吸引。这个功课他本来可能是逆缘的，但是你怎么把他变成善缘？如果我们不去努力，那人生真是艰苦。所以佛陀讲人生是苦的，有些人一看人生是苦的，他的心往下一沉。佛陀接着往下讲了，只有人能转这个境，因为只有人心是活的，人是可以转这个境的，我们的心不随着外境所转，我们的心转这个境，所以转恶为善，转凡为圣，所以怎么选择、处理我们的关系，是智慧。五行识人是镜子，照出的是关系。

第五个关键词，高维能量。高维能量就是一定不要把五行识人当作三维世界的工具。怎么理解为三维世界？就是把五行物质化，所以有些人把木理解为木头,火理解为自然界的火，土理解为大地土，金理解为金属，水理解为喝的水，如果这样理解，都是五行在三维世界成型了的物质，是一种很低的能量态里。以金行人为例，金行人怎样就能变成水行人，除非你象孙悟空一样，你内心中知道，作为金行人内心密码是目标坚定。但是最为水行人的内心密码是善解人意？怎么从目标坚定转变为善解人意，在什么地方改，只能在高维能量上改，高维能量是在什么地方，是在你的心理。你越是在心上改，转的越快。你越是在五行本身上改，改的越慢。

高维能量讲的是五行是好几个层次的，最低的层次就是把他实体化的层次。已经显像了，而且是三维世界的那个。比他高一点的，就说到象思维的三个层次，第一个象是实体的象，实体的象我们是用眼睛来观的。会懂五行的第一个层次就是他的实体象，长得不一样，所以一眼定乾坤。这是五行的第一个标准，通过看五行人外在的不同的五官、相貌、身材、手、脚、动作、动静的，了解他第一个实体象。第二个像是他的意象，举个例子，水的意象，上善若水，水是流动性的，所以水的意象是柔和的、滋润的、向下的。拿眼睛看了以后，跟他一接触，他的音调，他的声音，他的表情，他的原始的肝心脾肺肾。第一个是联系到外型上，第二个是联系到肝心脾肺肾上，甚至生理和病理都能帮你做辅助的，在身体那些投影的地方会有病，比如说肝胆病，肝胆病不会找别人，找上的是木行人，呼吸病就比较喜欢找金行人，这是第二个层次，这个识人就比他高一点，连接到内在心肝脾肺肾上，但这是中间投影，也不是最高的、N维的，这是三维和N维中间的，但也是有据可循的。那么最高的就是识人，无论是学紫微斗数、梅花易数，学任何的工具都承认识人的时候最后拿出一个工具来，摆那个盘，然后推那个盘，都不是最厉害的。因为最厉害的就是心，无论是从邵康杰一直到曾国藩，最厉害的就是拿心来感应，他的最高的能量是他投影的源头，他那个时候，完全不把对方当成人。这话说起来有点那什么，不是说你不是实体的人。所以那个时候第一个解读的就是第一层的木头、石头、火那个实体。第二个解读的是你的肝心脾肺肾，第三个层次解读的是你的能量态，这是宇宙最早的五种能量态，他是通过宇宙化生、生成的时候，跟生发的、向上的、向下的、收敛的、扩散的，跟这五种能量相类似的，这个时候你对面的就是五个能量态，让你去感知他是向上的、收敛的，还是向下的，还是外放的，这种能量态，你可以把它看成整个的一团气或者能量，或者一团光怎么样的，去感应他的能量态。识人有三个层次，第一个层次是实体态，那眼睛能观。第二个层次是意象，拿心去感应。最后，第三个层次是最高的能量态，是无心的，完全是直觉。

第一层次观象谁都能学会，都有眼睛，都会学会，看得高的、矮的、胖的、瘦的。第二层次应该训练一下也都学会了。第三层次不借助任何的外在的工具，也是最准确的。第三个层次关键是自己要修炼，修炼到高维能量态，你才能看出来别人是哪一种能量态。所以五行是一种通路，我们不是为了学一个技术而学习。

前言

人分五行，金木水火土，会识人，方有大福田。

企业发展靠人才，那么究竟什么才是人才？企业如何找到适合自身发展的优秀人才，实现人岗匹配，打造高效团队呢？

企业和员工都有自己的性格，自己的能量和属性，势必二者之间就存在着生克的转化关系，我们希望企业和员工是一种五行相生的顺转关系，而不是一种五行相克的逆转关系。

人分五行，识人要先读他的气质格局。

什么是气质格局呢？气质格局就是木，火，土，金，水。

长的锐隆强壮，就一定是火行人吗？

长的敦厚老实，就一定是土行人吗？

长的端正有行，就一定是金行人吗？

长的圆润和蔼，就一定是水行人吗？

这些只是从他的外观跟形体上识人，我们更强调的是瞬间---学习快速识破人心的观相术，掌握洞察心性的智慧心。所以，我们的五行识人更着重于读懂他的心性五行。

什么是一个人在职场中的心性五行以及他的心性五行在职场中的修炼。

具体而言，木象生发，如何才能不断创新而又不被指责为“木秀于林风必摧之”呢？木行人在职场中的修炼就一个字，“仁”，“仁义礼智信”的“仁”字。仁德的能量是绿色的，绿色代表关怀他人，由己推人，所以绿色的能量会注入到我们的肝胆系统，一旦肝胆系统接收到仁德的力量，就不会出现我们看到的，在职场中好顶撞，不服人，无意中出风头，抢了团队其他人的位置，争得青筋暴露。而会转顶撞不服人成不断创新，正直的一种动力。所以你会发现，身边的木行人是非常好的有慈善心的创新行的专家。

第二个，就是火行人。火象是炎上，火行人热情，主动，有礼，火行人如何不被人指责为爱虚荣炫耀呢？这需要在职场中修炼，火行人修炼要一个字“礼”。礼貌的礼，也是“仁义礼智信”的“礼”。礼德的系统是红色的，红色代表高度的外向，热情，大方而“知度”，传统文化中讲的“守礼而知节度”。当红色的能量注入心脏系统的时候，就会发现，自己不会出现职场中经常的抗上，争理，双方争执到面红耳赤的情况。而是转虚荣炫耀为对企业价值的推动和提升。由此可以看到，火行人是我们身边大方热情的销售人员和公关人员。

第三个就是土行人，土象叫做稼穑。稼穑的意思就是承载一切，孕育一切。职场中的土行人要修炼“仁义礼智信”的“信”字，诚信，讲究信誉。信德的系统是黄色的能量，当黄色能量注入到脾胃系统里，就不会出现职场中的工作效率低下，磨磨蹭蹭，拖拖拉拉的现象。一个人如果动作迟缓，是因为他的内心有怀疑，不信任，所以工作效率被人为的控制在一个很低的水平，自然也就提不到“管理是为了最大的提高工作效率，最大的释放人天生的全部潜力”。怎么办呢？通过转内心的怀疑不信任，变成合作互助。由此会发现，土行人是团队中不可缺少的合作伙伴，是天生的项目经理，叫聚缘。

第四个需要识别的是金行人。金象叫做从革，从革就是锋利的意思。如何能够有胆识，果断的做决断，并且不伤到团队成员的心呢？这就需要在目标结果和关系情感上做中和，这就是金行人需要在职场中的修炼，简而言之，就是修炼“义”，义气的义。古语言：‘德不配位必遭灾遣’、‘积善之家必有余庆，积不善之家必有余殃’，所以金行人在职场上要特别注意‘财聚人散，财散人聚’。 义德是白色的能量，白色就代表决断力，果断，开创，当白色能量注入到肺脏系统的时候，它就转好计较、爱比较，嫌人懒、恨人笨为和气聚缘，大家朝着共同的愿景而奉献。由此会发现，金行人是职场中有出色的执行能力和卓越工作效率的CEO。

第五种要识别的就是水行人，水象是润下。水行人亲善有智慧，但是如何避免成为受气包，被指责为工作懒散，任性随意呢？水行人需要在职场中修炼“仁义礼智信”的“智”。因为水向低处走，智德的系统是黑色的，黑色代表善解人意、婉转亲和，道家有言：“上善若水”“善利万物而不争”。当黑色的能量注入肾脏系统的时候，就会发现水行人高度内向，非常的慈祥通透，通过职场的修炼会转放任散漫为高度的和谐，水行人内心最希望达到一个和谐的团队。由此可以发现，水行人是职场中不可或缺的促进彼此连接跟凝聚力的合作伙伴。

这个和谐世界从和谐社会开始，和谐社会从和谐企业开始，和谐企业从和谐团队开始，和谐团队从哪里开始呢？从这里（心里）开始，从人人的主动的，积极的，不是来自外在，不是来自你的老板，不是来自你的客户，不是来自你的团队，不是来自你的家庭，不是外在的任何一种力量，而是完完全全的由我们自身自我成长的需要，和企业发展的需要开始！希望读者可以通过木，火，土，金，水的圆转修炼，来打造一个企业与个人和谐进步，共同成长的完美团队！！
