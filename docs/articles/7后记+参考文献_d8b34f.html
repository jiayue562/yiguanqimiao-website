后记

已经记不清这是在哪一个晚上发生的事了，我在QQ上与一位从未见过面的网友聊天，我说起我正在潜心研究五行人格，并说五行人格理论的最大特点和优点是“身心合一”，即从身体外形可以推知一个人的性格类型，反过来，从一个人的性格类型也可以推知一个人的长相特征，他觉得很惊奇，虽然他在理论上也相信这是可能的，但还是不太相信真有什么方法能做到这一点。

我说，那我们就来试试吧，正好我也没有见过你的面。

他同意了，于是我给他发了一套我研发的《五行人格心理测试题》。

很快，十分钟之后，他就把题目做完了，并把测试结果统计出来，从QQ上传给了我。

我一看他的测试结果，非常明显，“E”项最高分，属于“水行人”，于是我对他的长相下了如下推断：

“你身材较胖，不高，面圆，脸色偏黑，双下巴，说话和动作较慢，声音低沉，性格谨慎……”

我把这行字打过去之后，他回复说：“是的，非常准确。我很胖，170公分左右，最重时超过170斤，而且我的双下巴很明显。你帮我测测我夫人的性格和长相看看。”

我让他把题目给他夫人做，很快，她夫人测试的结果也发给我了。

这时，他说要照顾孩子入睡了，明天再谈。

我看他夫人的测试结果，各项得分比较平均，不过“D”项还是稍有突出。于是我对他夫人的身材长相做了如下推断，并给他的QQ留言：

“你夫人不高，苗条，偏瘦，脸色白，眉清目秀，动作利索，反应敏捷，口齿伶俐，有点好计较……”

对方QQ已下线了，没有回复。

我给他留言说不知道推断准确与否，希望他给予回复。

一夜过去了，我多少还是不太有把握，因为我这也是第一次做这样的测试。

第二天上午，我打开电脑QQ，立即看到昨天那位网友的留言：

“你判断的基本准确，除了脸色不太白之外，其它的都很准确。”

他还发来了他们二人的合影，我一看，确实与我心中推想的形象很接近。由于个人隐私的原因，就不公布他们的照片了。

那么，他夫人的脸色不太白，这是什么原因呢？我反思了自己的推断过程，原来是因为她测试的“D”项得分并不占决定性优势，所以她不是个纯粹的“金行人”，所以脸色并不太白。

不管怎么说，这一次的五行人格测试，让我感到振奋，从那以后，我的五行人格测试一发不可收拾，直到现在。

所以，我一直非常感激那位让我做第一次测试的朋友，他促使我更加专注于研究五行人格理论。

随着对五行人格研究和应用的深入，我越来越觉得，五行人格理论是迄今为止最好的人格理论。

五行人格理论来源于博大精深的中国传统文化，具有其它人格理论无法比拟的二大特点：

一、五行人格理论是“身心合一”的统一科学理论

物质和精神是一性的，身与心也是一性的，身体的五行与性格的五行完全是对应的，五行人格理论有助于消除千百年来人们在哲学上关于物质与精神关系的论争，有助于使各种知识走向真正的统一，有助于形成真正的统一科学。

五行人格理论可以做到——

1、认识了身体的五行属性也就能推知性格的五行属性；

2、认识了性格的五行属性也同样能推知身体的五行属性。

运用五行人格理论，可以从身体外形推知内在性格，也可以从内在性格推知身体健康状况、精力、耐力等，相互印证，从而达到对人的全面认识。

二、五行人格理论是完备而简明的系统论

五行学说是最完备的系统论，金、木、水、火、土这五种元素或五种能量、物质，相互都发生着“生”（助长）或“克”（抑制）的相互作用，也就是说，任何一种能量、物质都与其它四种能量、物质之间存在着“生”或“克”的作用，任何一个“我”，都有“我生”的和“生我”的，也有“我克”的和“克我”的，这就构成了一个完备而均衡的宇宙系统。五行学说这一系统论，最完备，包含宇宙万事万物；又最简明，局部、个体的特点和相互关系一目了然。所以说，五行理论是最好的系统论。

五行人格理论正是基于五行学说的人格理论，所以五行人格理论也是最完备的系统论。

五行人格理论有五大优点：

一、科学性：

五行人格理论实现了身与心、生理与心理、物质与精神的统一，使心理、意识、精神完全建立在物质基础之上；它是可以验证的，知道了一个人身体的五行属性就可以推断其性格的五行属性和心理特征，反过来，知道了一个人性格的五行属性和心理特征，也能推知其身体的五行属性、身体特征和健康状况。所以，五行人格理论超越了其它所有的心理学说，它不只是一种理论，不只是一种假说，而是真正的科学理论。

三、系统性：

五行人格理论具有完整而严密的系统性，它对事物的分类非常系统，对事物之间的关系的分类也非常系统，而且，这个系统是客观的、科学的，不是哪个人主观臆想出来的。

五行人格理论至简至易，具有非常良好的简约性，可以说已经简约到不能再简约的程度了；人们很容易掌握和运用，稍微用一点功夫就能理解，并且能够进行广泛的演绎类推，真正可以做到闻一而知十，闻十而知百，闻百而知千万……

一、人是一“太极”：整体看一个人

认识“人”，研究人格，从哪里开始？

看人，首先要从整体上看。用道家的方法看人，每个人都是一个“太极”。

太极图

每个人都是一个统一的生命体：有精神有身体，有心理有生理，有内脏有身躯，有左边有右边，有前胸有后背……各部分，通过内在的生命机制统一在一起，互相协调运作，构成完整的生命整体。这就是“太极”。每个人都是一个“太极”。

引申开去，任何一物都是一“太极”，天地也是一“太极”，宇宙也是一“太极”，从宏观到微观，任何一个物体或粒子，都是一个“太极”。

道家把“太极”称为“道”，整个宇宙就是一个最大的“太极”，这就是最高的“道”。

人在宇宙中，所以人这个小“太极”要顺应宇宙这个最大的“太极”，也就是要顺应宇宙之“道”。

宇宙之“道”，古代也称为“天”或“天地”（古人有时把“道”与“天”、“天地”分开，有时又不分），在河图洛书中则称为“数”，名虽各异，其实为一。

宇宙之“道”作为宇宙的本体，是永远不变的，即周易所称的“不易”，本体之“道”，可“动”，但不“变”，动与变是不同的。

老子说：“道”“独立而不改，周行而不殆”。“有物混成，先天地生”，“吾不知谁之子，象帝之先”。也就是说，“道”是早于天地万物而生的，是无法追问其产生时间的。至于说谁产生了“道”，那就更不知道了，也不能追问了。因为“道”是最本源、最根本，是一切知识的起点。

老子说：“道”——“渊兮似万物之宗”，“可以为天下母”。还说：“道生一，一生二，二生三，三生万物。”“故道大、天大、地大、人亦大。域中有四大，而人居其一焉。人法地，地法天，天法道，道法自然。”也就是说，本体的道，是天地万物的“造物主”，是一切的起源和原因，是制约一切的最根本力量，是天地万物包括人类在内必须顺从的最高法则。

所以，认识人和人格，与认识宇宙和天地，是相通的，都是要认识“太极”，都是要认识“道”。这是认识人的起点，也是五行人格研究的起点。只有理解了宇宙之“道”，才能理解人的本质。

二、“太极”生“两仪”：把人分成二类——人的“阴阳”与西方个性理论

“太极”是“一”，我们从整体上看人，发现每个人都是一个“太极”，人与人之间没有差别，人与人之间也没有办法做出分类。

但是人和人又毕竟是不同的。那么，怎么识别人与人之间的不同呢？

“太极”可分为“阴阳”，因此每个人自身都呈现出两面性：身体与精神，情感与理智，外向与内向，勤奋与懒惰，忍让与宽容……几乎每个人身上都具有人性的优点与弱点。

人与人的不同，是因为每个人身上所具有的各种性格特点和健康状况的强弱程度不同。

“太极”的阳面强大的人就是刚强的人，是“君子”，是成功者；太极”的阴面强大的人就是软弱的人，是“小人”，是失败者。

所以人就分出了二大类：男和女，老和少，强和弱，善与恶，智与愚，外向与内向……

西方个性理论把人的个性分为“外向”与“内向”二种，外向为阳性，内向即为阴性。东西方文化是相通的。我们会看到，在五行人格中，木行与火行是阳性，金行与水行是阴性，而土行则是“中性”——土行人实际上可具体再分出“阴性”和“阳性”，即可区分出外向还是内向。

三、“两仪”生“四象”：把人分成四类——四象、四大与气质、血型理论

（一）中国的“四象”理论：

把人一分为二，进而可把人二分为四。

中国道家认为，无极生太极，太极生两仪（阴阳），两仪生四象，四象生八卦……

四象即把万物分成四大类，如中季春夏秋冬、四方东南西北，也可把人分成四类，如少阳（半外向）、太阳（外向）、老阴（内向）、少阴（半内向）。这种理论同样已经运用到人格理论上来，“四象”加上“中”就是五行，所以，四象学说与五行学说是相通的。关于这一点，我们在论述“五行”与“四象”关系的时候再详细介绍。

西方有“四大”学说（古印度）和“四元”学说（古希腊），与中国“四象”理论和五行学说完全是相通的。

（二）西方的气质理论：

体液说最早是由古希腊医生希波克拉底 ( 又译作希波克拉特 ,Hippocrates, 公元前 460-371) 提出并系统解释的。他认为人体内有血液、粘液、黄胆汁和黑胆汁四种体液，以四种体液在人体内的不同比例关系为依据，将人的气质类型分为四种，即多血质、粘液质、胆汁质和抑郁质。

多血质类型的人活泼好动、反应迅速、行动敏捷、热衷交际、兴趣及注意力容易转移；胆汁质类型的人精力旺盛、热情直率、易于冲动发怒、情绪波动大；粘液质类型的人沉默寡言、安静稳重、善于忍耐、反应迟缓、注意稳定；抑郁质类型的人行动及反应迟钝、不善交际、多愁善感、谨小慎微。

罗马医生盖伦 (Galen, 公元二世纪 ) 在体液说的基础上，首次使用了气质这个概念。
