规律二：信息反馈律

核心定义：系统通过信息反馈机制实现稳定与发展演化的普遍规律。

详细内容：
- 维纳控制论核心：反馈是系统保持稳定和实现目的性的关键
- 负反馈：消除偏差，维持稳态（体温调节、恒温器）
- 正反馈：放大偏差，导致突变（链式裂变、马太效应）
- 单纯负反馈 = 停滞；单纯正反馈 = 崩溃
- 系统发展 = 负反馈（维稳）+ 正反馈（促变）的辩证统一

五行人格应用：
- 拔阴取阳的负反馈：觉察阴面 → 纠正 → 回归阳面
- 化克为生的正反馈：一次转化成功 → 强化转化能力 → 更多转化
- 知行合一闭环 = 信息反馈律的行为版本

龙心OS应用：
- 自主进化闭环：GAPS.md（偏差检测）→ EVOLUTION.md（纠正行动）→ 反馈验证
- 三库同步：Obsidian → WorkBuddy → IMA → Obsidian（闭环反馈）

B=MAP转化：
- M：成长反馈带来的成就感
- A：超级小——每天记录1条"今天我学到了什么"
- P：睡前5分钟回顾提醒

规律四：涨落有序律

核心定义：系统通过涨落（随机起伏）达到有序状态，偶然性与必然性在分叉点辩证统一的规律（普利高津耗散结构理论核心）。

详细内容：
- 涨落 = 系统状态在宏观平均值附近的随机起伏
- 近平衡态：涨落是干扰，系统惯性衰减它（微涨落）
- 远离平衡临界区：涨落被非线性正反馈放大为巨涨落，决定系统走向
- 通过涨落达到有序 = 普利高津核心命题
- 分叉点：偶然的微涨落 → 必然的宏观有序

五行人格应用：
- 拔阴取阳的"悟" = 微涨落（日常积累）→ 巨涨落（突然开悟）
- 修行中的"瓶颈期" = 涨落积累期，不是失败是准备
- 接受涨落 = 接受不确定性 = 系统有序化的前提

觉悟路径：
- L1~L3：微涨落积累（学习方法、修正认知）
- L4~L6：涨落加速（开始有意识转化）
- L7~L9：巨涨落（觉悟时刻、突破性转化）

B=MAP转化：
- M：对"顿悟时刻"的渴望
- A：超级小——每天允许自己"不知道"，留出混沌空间
- P：冥想/散步时问"今天有什么新浮现的念头？"
