步骤 3: 安全审查

自动审查 (skill-vetter)

openclaw skills vet <skill-name>

检查项:
- [ ] 权限申请是否合理
- [ ] 网络请求目标是否可信
- [ ] 代码是否有混淆
- [ ] 依赖来源是否可靠
- [ ] 是否有恶意代码迹象

手动审查清单

检查项 通过标准 状态
来源可信 官方市场/知名作者 ⬜
权限合理 最小权限原则 ⬜
代码清晰 无混淆/加密 ⬜
依赖安全 无未知依赖 ⬜
文档完整 有使用说明 ⬜

风险等级

等级 标准 处理
🟢 安全 全部通过 直接安装
🟡 低风险 1-2 项存疑 告知用户后决定
🔴 高风险 3 项以上存疑 拒绝安装
