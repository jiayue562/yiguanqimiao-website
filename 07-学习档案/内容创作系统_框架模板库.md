# 🧱 框架模板库

> **定位**：反复验证有效的内容结构框架
> **使用方式**：创作时选择合适的框架，填入具体内容即可
> **维护原则**：数据验证有效的框架标记⭐，无效的标记❌

---

## ⭐ 高效框架（数据验证）

### 框架一：五行识人科普框架
> **适用**：五行人格基础科普类内容
> **平均表现**：待数据积累
```
1. 【钩子】用一个日常场景引入（你是哪种人？）
2. 【定义】一句话定义该五行核心特质
3. 【三界】身界表现 → 心界表现 → 灵界表现
4. 【阴阳】阳面（优势）vs 阴面（陷阱）对比
5. 【识别】3个快速识别该五行的方法
6. 【转化】拔阴取阳的核心方法
7. 【升华】五行不是标签，是成长的起点
```

### 框架二：关系冲突分析框架
> **适用**：五行关系、亲密关系、团队冲突类内容
> **平均表现**：待数据积累
```
1. 【钩子】描述一个典型的关系冲突场景
2. 【诊断】用五行生克分析冲突根源
3. 【双视角】分别从两方的五行视角解读
4. 【案例】真实案例佐证（脱敏处理）
5. 【方法】化克为生的具体操作步骤
6. 【升华】理解差异是和谐的前提
```

### 框架三：成长转化指导框架
> **适用**：自我成长、人格转化类内容
> **平均表现**：待数据积累
```
1. 【钩子】直击读者的成长痛点
2. 【定位】用九层阶梯定位当前层级
3. 【觉察】引导读者自我觉察（三界扫描）
4. 【方法】拔阴取阳四步法（认不是→找好处→信因果→达天时）
5. 【案例】真实转化案例
6. 【工具】B=MAP行为设计（动机×能力×提示）
7. 【升华】成长不是消灭阴面，是让阳面主导
```

### 框架四：心文化科普框架
> **适用**：心文化、大圆满、修行类内容
> **平均表现**：待数据积累
```
1. 【钩子】现代人的普遍焦虑/困惑
2. 【概念】一句话解释核心概念（如大圆满）
3. 【对比】与西方心理学的异同
4. 【实践】可操作的日常修法（抄经/静坐/瑜伽）
5. 【故事】修行人的真实体验
6. 【升华】修行不在深山，在每一个当下
```

## 📋 通用写作结构

### AIDA结构（经典）
```
Attention（注意）：钩住读者
Interest（兴趣）：展开核心内容
Desire（欲望）：让读者想要这个改变
Action（行动）：给出具体步骤
```

### SCQA结构（麦肯锡）
```
Situation（情境）：描述背景
Complication（冲突）：引入矛盾
Question（问题）：抛出核心问题
Answer（回答）：给出解决方案
```

### 教员方法论结构
> **适用**：系统性培训/方法论输出
```
1. 矛盾定位（根本矛盾→当前主矛盾）
2. 三维诊断（物质体×能量体×信息体）
3. 化克为生（转化路径）
4. 知行合一（表示→压缩→泛化）
```
