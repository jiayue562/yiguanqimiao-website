## 一、紧箍咒

### 1.1 核心定义

> **紧箍咒是管理者的日常行为准则和警报系统，是基于数据分析得出的、直接影响客户复购的三大核心问题。**

### 1.2 来源

> **紧箍咒比喻束缚人的东西，具有收束心神的含义。**

**味藏的紧箍咒定义**：
- 不符合味藏300人均调性的东西
- 不符合味藏产品要求的行为
- 出现以上几点，及时调整改正，不然就会影响复购

## 三、服务效率公式

### 3.1 核心公式

> **服务效率 = 产品稳定 + 上菜快 + 服务及时（适时）**

### 3.2 "适时"的关键

| 维度 | 说明 |
|------|------|
| **不是** | 一味求快 |
| **而是** | 根据客户属性调控上菜速度 |
| **核心** | 智能调控 |

### 4.1 核心定义

> **匠心是味藏对过程的坚守，是内在的驱动力。本质是：匠心 = 质价比 = 有序经营。**

## 五，食材层面的匠心

### 5.1 五大维度

| 维度 | 要求 | 示例 |
|------|------|------|
| **周期** | 最新鲜、仓储周期最短 | 应季食材 |
| **不时不食** | 遵循时令 | 应季食材 |
| **冰鲜坚持** | 不用冷冻品 | 冰鲜牛肉 |
| **销售时效** | 严格定义 | 松叶蟹5天最佳赏味期 |
| **鲜活坚持** | 不卖冻蟹死蟹 | 活蟹供应 |

### 5.2 详细解读

**周期苛求**：
- 坚持选择最新鲜的食材
- 仓储周期最短

**不时不食**：
- 遵循时令
- 使用应季食材

**冰鲜 VS 冷冻**：
- 坚持使用冰鲜牛肉等高品质食材
- 而非冷冻品

**销售时效**：
- 对金枪鱼最佳赏味期定义为5天
- 这种在细节上追求极致，不留模糊空间，正是匠心精神的体现

**鲜活坚持**：
- 每只松叶蟹保证其鲜活
- 不卖冻蟹死蟹

## 七、核心金句

1. **紧箍咒定义**："不符合味藏300人均调性的东西，不符合味藏产品要求的行为"

2. **服务公式**："服务效率 = 产品稳定 + 上菜快 + 服务及时（适时）"

3. **适时关键**："根据客户属性和用餐节奏，智能调控上菜速度，而非一味求快"

4. **匠心公式**："匠心 = 质价比 = 有序经营"

5. **食材匠心**："在细节上追求极致，不留模糊空间"

6. **管理匠心**："口味稳定是餐饮的生命线，自然微笑是服务的底线"
