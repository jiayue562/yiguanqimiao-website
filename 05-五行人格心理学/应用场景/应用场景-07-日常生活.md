# 日常生活 · 五行人格应用场景

> 场景映射编号：SC-07 | 凤爪OS.skills 标签：scene-mapping/日常生活

## 场景定位
- 核心问题：如何将五行智慧融入日常
- 目标用户：所有人
- 核心理念：知行合一

## 日常应用指南
| 领域 | 应用方式 |
|------|----------|
| 作息 | 按五行时辰调整作息 |
| 饮食 | 五味调和、按体质选择 |
| 着装 | 五行色搭配调节能量 |
| 居住 | 方位/色彩环境调整 |
| 社交 | 选择合适的社交圈 |

## 每日五行实践
1. 晨起静坐（水→滋养）
2. 制定当日目标（木→生长）
3. 热情投入（火→燃烧）
4. 踏实执行（土→承载）
5. 总结复盘（金→收敛）
