> **形：瘦长挺拔（修长）**
> **气：清雅舒展**
> **神：亮且有神**
> **声：低沉平缓**
> **行：能快能慢**

## 二、体型特征（身界·物象层）

### 2.1 整体体型

| 特征维度 | 描述 | 补充说明 |
|---------|------|---------|
| **体行（整体）** | 瘦长（修长）、瘦 | 核心识别点：修长是木行最显著特征 |
| **头行** | 中等偏圆 | 非特别大或特别小 |
| **脸行** | 偏长脸长 | 长脸是木行重要特征 |
| **面色** | 青色（偏黄）、中等偏白 | 青色对应肝木；有时偏黄或偏白 |
| **肤色** | 偏黄 | 与土行偏黄区别：木行体型修长 |

### 2.2 五官细节

| 部位 | 特征 | 工坊补充数据 |
|------|------|------------|
| **鼻子** | 直、正直 | 鼻梁正直是木行特征 |
| **眼睛** | 亮且有神、不小 | 眼神清明是识别关键 |
| **眉毛** | 粗长、稀 | 注意：有时粗长，有时稀疏 |
| **耳朵** | 小且无耳垂 | 耳垂小或无耳垂（区别于水/土行） |
| **手足** | 手不大，脚不小 | 均匀修长 |

### 2.3 体型综合数据

```
体形：瘦
肤色：中等偏白
肩背：一般（不特别宽厚）
腹股：小（区别于土/水行的敦实圆润）
手足：手不大，脚不小
头型：中等偏圆
脸型：偏长
```

### 2.4 识别要诀

> **木行三字经：长·直·清**
> - 长：身形修长，脸型偏长
> - 直：鼻子正直，站姿挺拔
> - 清：气质清雅，眼神清明

## 三、气质特征（心界·意象层）

### 3.1 整体气质

**核心词**：**清雅**

| 状态 | 表现 |
|------|------|
| **阳木气质** | 清新脱俗，自然飘逸，充满生机 |
| **阴木气质** | 清高孤僻，不苟言笑，距离感强 |

### 3.2 声音特征

| 特征 | 工坊实测数据 |
|------|------------|
| **声音** | 低沉、厚重 |
| **语速** | 中速、平缓 |
| **语调** | 稳定，不急不躁 |
| **常用词汇** | "就是就是" |
| **表达方式** | 语言干脆，表达准确 |

> **核心发现**：木行人语速"能快能慢"，这与其"动态特征：能快能慢"相呼应——木行人有内在节奏，可根据情境调整。

### 3.3 动态特征

| 动态维度 | 特征 |
|---------|------|
| **整体动态** | 能快能慢（适应性强） |
| **行动特点** | 行动果断，执行力强 |
| **情绪感知** | 敏感（对外界敏感，尤其是不公平） |

| 维度 | 工坊实测数据 |
|------|------------|
| **整体风格** | 休闲、舒适 |
| **颜色偏好** | 喜欢冷色调 |
| **细节特征** | 关注细节，追求完美 |
| **配饰** | 配饰搭配得当 |

> **识别提示**：木行人的着装是"低调的精致"——看似简单，但细节讲究，整体协调。

## 七、与其他五行的外观区分

| 对比 | 木行 | 区分要点 |
|------|------|---------|
| **木vs火** | 木：瘦长清雅；火：壮实热情 | 体型差异最明显 |
| **木vs土** | 木：修长直立；土：敦实圆厚 | 整体轮廓截然不同 |
| **木vs金** | 木：瘦长柔和；金：方正有棱角 | 脸型差异：偏长vs方正 |
| **木vs水** | 木：修长干练；水：圆润柔软 | 体感差异：紧实vs松软 |
