> **火者，炎上也。**
> **形：锐隆壮实（中等偏壮）**
> **气：热情明亮**
> **神：目光热烈**
> **声：高亢有力**
> **行：急而有势**

## 二、体型特征（身界·物象层）

### 2.1 整体体型

| 特征维度 | 描述 | 补充说明 |
|---------|------|---------|
| **体行（整体）** | 锐隆，身体壮而结实，中等 | 面相上宽下尖或长宽一样 |
| **头行** | 尖圆 | 头顶相对尖，区别于土行的宽圆 |
| **脸行** | 圆 | 圆脸是火行特征之一 |
| **面色** | 肤色偏白，热情或被动热情 | 白里透红；精神好时红润发光 |
| **肤色** | 黑（工坊补充）；亮，平，无凹凸 | 汗毛重；皮肤细腻无明显凹凸 |

### 2.2 五官细节

| 部位 | 特征 | 工坊补充数据 |
|------|------|------------|
| **鼻子** | 两翼有肉，出油，有肉 | 鼻翼饱满，是火行识别特征 |
| **眼睛** | 近视，目光柔和，细有血丝，中等 | 注意：目光"柔和"但行动"急"，是火行特有反差 |
| **眉毛** | 粗重，中间连上了，浓 | 眉毛粗浓，甚至相连（连眉） |
| **耳朵** | 内耳高于外耳，耳朵相对较硬，内耳轮廓明显 | 内耳轮廓明显是火行特征 |
| **肩** | 宽 | 肩宽体壮，有力量感 |
| **腹股** | 有肉 | 腹部有肉，区别于木行的腹股小 |
| **手足** | 小 | 火行人手足偏小（与体型形成对比） |

### 2.3 体型综合数据

```
体形：中等
肤色：黑（受日晒影响）；亮，平，无凹凸；汗毛重
肩：宽
腹股：有肉
手足：小
头型：尖圆
脸型：圆
```

### 2.4 识别要诀

> **火行三字经：圆·宽·热**
> - 圆：脸型偏圆，头型尖圆
> - 宽：肩宽，眉粗，耳内廓明显
> - 热：肤色发亮，气质热情，声音热烈

## 三、气质特征（心界·意象层）

### 3.1 整体气质

**核心词**：**热情（或被动热情）**

| 状态 | 表现 |
|------|------|
| **阳火气质** | 主动热情，明亮开朗，感染力强 |
| **阴火气质** | 被动热情（需要点燃），或急躁易怒 |

> **关键洞察**："热情或被动热情"——火行人的热情有两种形式：有时主动点燃，有时需要被激发。这是识别时的重要细节。

### 3.2 声音特征

| 特征 | 工坊实测数据 |
|------|------------|
| **声音** | 高亢，浑厚，有穿透力，热烈 |
| **语速** | 快，有条理 |
| **语调** | 热烈，有节奏感 |
| **常用词汇** | "嚯"（感叹词，表示惊讶或兴奋） |
| **表达方式** | 语速快，直率 |

### 3.3 动态特征

| 动态维度 | 特征 |
|---------|------|
| **整体动态** | 行动迅速，有气势 |
| **情绪反应** | 急（情绪反应速度快）|
| **体感** | 身体发热感 |
| **感官** | 对甜味敏感，对夏季有感 |

## 四、着装风格（身界·物象层）

| 维度 | 工坊实测数据 |
|------|------------|
| **整体风格** | 随意、休闲 |
| **特征** | 衣着光鲜靓丽，明星范 |
| **配饰** | 带金属类配饰 |
| **特点** | 注重外表，有"明星感" |

> **识别提示**：火行人的穿着有两面性——日常可能"随意休闲"，但在社交场合会主动展示，有"明星范"。配金属配饰是火行特征（金属与光泽对应火行的明亮性）。
