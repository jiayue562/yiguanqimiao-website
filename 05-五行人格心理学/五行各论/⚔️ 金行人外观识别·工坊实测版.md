## 二、体型特征（身界·物象层）

### 2.1 整体体型

| 特征维度 | 描述 | 补充说明 |
|---------|------|---------|
| **体行（整体）** | 匀称 | 既不特别高也不特别矮，比例均衡 |
| **头行** | 平整 | 头顶平整，区别于火行的尖圆 |

| **面色** | — | 需进一步验证 |

| 部位 | 特征 | 工坊补充数据 |
|------|------|------------|

## 三、气质特征（心界·意象层）

### 3.1 整体气质

**核心词**：**镇定（表面）**

| 状态 | 表现 |
|------|------|
| **阳金气质** | 沉稳自信，清明决断，表里如一 |
| **阴金气质** | 表面镇定，内心压抑，先忍后爆 |

> **关键洞察**："镇定（表面）"——金行人擅长情绪管理，外表镇定，但内心可能在积压。这是识别金行人的重要特点：看起来冷静，但一旦爆发，力量惊人。

### 3.2 声音特征

| 特征 | 工坊实测数据 |
|------|------------|
| **声音** | 洪亮，磁性 |
| **语速** | 快 |
| **语调** | 有力，清晰 |
| **口味** | 甜辣 |
| **常用词汇** | "我是这么想的""我认为""我觉得" |
| **表达方式** | 有主见，强调自己的观点 |

> **语言特征分析**："我是这么想的""我认为""我觉得"——金行人习惯用第一人称表达主观判断，体现了金行的"清明决断"。同时，金行人"模仿能力强，容易被认可的人影响"——这是金行人的隐性特征。

### 3.3 动态特征

| 动态维度 | 特征 |
|---------|------|
| **整体动态** | 行动轻快 |
| **情绪反应** | 镇定（表面）；先容忍，试探，再爆发 |
| **感官** | 经络不敏感；口味甜辣 |
