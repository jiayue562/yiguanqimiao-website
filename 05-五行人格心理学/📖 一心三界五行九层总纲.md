# 📖 一心三界五行九层总纲

> **核心定义**：「一心三界五行九层」是悟空独创的五行人格心理学完整理论框架，以「一心」（纯粹觉知/大圆满见地）为根本，通过「三界」（法身/报身/化身，信息体/能量体/物质体）的结构分层，展现「五行」（木火土金水）的动态流转，并以「九层」（健康1-3级/一般4-6级/不健康7-9级）量化人格发展状态，构成完整的生命理解与转化系统。


## 💎 二、一心——纯粹觉知

### 核心定义
一心是超越三界五行的纯粹觉知本身。不是任何具体的感受、思想或行为，而是能够觉知这一切的「觉知本身」。

### 大圆满见地
- **本来清净**（空性）：一切法无任何本质，超越轮回、因果、缘起
- **本自圆满**（光明）：具足一切光明潜能，不增不减
- **不二**（噶达陇竹尼美）：空性与光明本是一体

### 一心在五行中的显化

| 五行 | 一心的显化特质 | 阳面（近一心） | 阴面（远一心） |
|------|--------------|--------------|--------------|
| 木 | 仁德本源·生长性·觉照性 | 生发创新，仁慈包容 | 傲慢顶撞，固执不服 |
| 火 | 光明觉知·热情性·礼明性 | 礼明照世，温暖感染 | 急躁嗔恨，炫耀虚荣 |
| 土 | 承载觉知·承载性·稳定性 | 信实包容，稳重落地 | 怨恨拖延，疑虑过重 |
| 金 | 清明觉知·清明性·决断性 | 义礼清明，精准执行 | 刚硬偏执，批评过度 |
| 水 | 润泽觉知·润泽映现性·沉潜含藏性 | 智慧深沉，灵活应对 | 恐惧封闭，疑神疑鬼 |

### 发现一心的三步法
1. **悬置**：放下对五行惯性模式的执着，暂停自动化反应
2. **内观**：觉察当下内在能量——是阳面还是阴面在主导？
3. **合一**：体悟阴阳本是一体，在觉知中自然转化

## 🌍 三、三界——能量的三层结构

### 三界对应体系

| 三界 | 传统名称 | 现代对应 | 功能 |
|------|---------|---------|------|
| **灵界** | 法身 | 信息体 | 价值观·使命·信仰·根本动力 |
| **心界** | 报身 | 能量体 | 情绪·思维·心理模式·感受 |
| **身界** | 化身 | 物质体 | 行为·表情·体态·物理表现 |

### 三界协同机制

**健康状态（马车比喻）**：
```
灵界（仁德使命）→ 驱动 → 心界（积极情绪）→ 驱动 → 身界（健康行为）
三界方向一致，能量流畅，走运
```

**失衡状态**：
```
灵界迷失（阴面价值观）→ 心界紊乱（负面情绪）→ 身界失常（问题行为）
三界方向不一致，能量阻塞，不走运
```

### 走运与否的判断标准
- **走运**：三界能量阳面主导，协同一致，方向清晰
- **不走运**：三界能量阴面主导，相互矛盾，方向混乱

## 🔄 四、五行——动态能量系统

### 五行基本属性

| 五行 | 方位 | 季节 | 脏腑 | 情志 | 五色光 | 五方佛 | 五大智慧 |
|------|------|------|------|------|--------|--------|---------|
| 木 | 东 | 春 | 肝胆 | 怒 | 绿 | 不空成就佛 | 成所作智 |
| 火 | 南 | 夏 | 心小肠 | 喜/恨 | 红 | 阿弥陀佛 | 妙观察智 |
| 土 | 中 | 长夏 | 脾胃 | 思/怨 | 黄 | 宝生佛 | 平等性智 |
| 金 | 西 | 秋 | 肺大肠 | 悲/恼 | 白 | 阿閦佛 | 大圆镜智 |
| 水 | 北 | 冬 | 肾膀胱 | 恐/烦 | 蓝 | 毗卢遮那佛 | 法界体性智 |

### 五行生克基本规律

**相生（滋养）**：
```
木→火→土→金→水→木（循环滋养）
```

**相克（制约）**：
```
木克土·土克水·水克火·火克金·金克木
```

### 五行阴阳两面总表

| 五行 | 阳面（近道） | 阴面（远道） | 核心情志 |
|------|-----------|-----------|---------|
| **木** | 生发创新·正直担当·仁德包容 | 傲慢顶撞·固执偏执·忧愁善变 | 怒（阴）→ 学·仁（阳）|
| **火** | 礼明照世·热情感染·积极引领 | 急躁嗔恨·炫耀虚荣·愤世嫉俗 | 恨（阴）→ 礼·明（阳）|
| **土** | 信实包容·稳重落地·承载滋养 | 怨恨拖延·疑虑过重·固执守旧 | 怨（阴）→ 信·思（阳）|
| **金** | 义礼清明·精准执行·自律高效 | 刚硬偏执·批评苛刻·恼怒强控 | 恼（阴）→ 义·行（阳）|
| **水** | 智慧深沉·灵活应对·意志坚定 | 恐惧封闭·疑神疑鬼·烦躁退缩 | 烦（阴）→ 智·辩（阳）|

## 📊 五、九层——发展层级光谱

### 九层总览

| 区间 | 层级 | 能量状态 | 核心特征 |
|------|------|---------|---------|
| **健康区间** | 1级：解放 | 觉性主导 | 超越五行，近于一心，自然流露 |
| | 2级：心理能力 | 阳面主导 | 充分发挥正面特质，利他有效 |
| | 3级：社会价值 | 阳面活跃 | 在社会角色中发挥优势 |
| **一般区间** | 4级：失衡 | 阴阳拉锯 | 内外开始冲突，能量消耗加大 |
| | 5级：人际控制 | 阴面初显 | 开始用阴面模式控制他人 |
| | 6级：过度补偿 | 阴面主导 | 行为极端化，自我保护过度 |
| **不健康区间** | 7级：侵害 | 阴面炽盛 | 主动伤害他人，能量失控 |
| | 8级：妄想强迫 | 三界断裂 | 脱离现实，强迫性行为 |
| | 9级：病态破坏 | 系统崩解 | 能量彻底黑暗，走向毁灭 |

### 层级判断三维度
1. **灵界**：价值观是利他还是利己？使命是清晰还是混乱？
2. **心界**：情绪是阳面还是阴面？思维是开阔还是僵化？
3. **身界**：行为是建设性还是破坏性？体态是舒展还是紧张？

### 发展方向
- **向上**（升维）：通过拔阴取阳、化克为生，从下层升至上层
- **向下**（降维）：压力过大、失去觉察，从上层滑落下层
- **终极**（超越）：超出三界外，不在五行中——纯粹觉知（一心）

## 🎯 七、核心应用原则

### 识人三步法
1. **观象**：观察身界（体型、动作、面色、声音）
2. **察心**：了解心界（情绪模式、思维习惯、关系方式）
3. **知灵**：理解灵界（价值观、使命感、人生哲学）

### 转化六字诀
- **觉察**：发现自己当前在哪个层级
- **认是**：承认阴面状态，不逃避不压制
- **找好**：主动寻找对方（或自己）的好处（阳面特质）
- **不怨**：放弃怨恨，断掉阴面能量的供给
- **转化**：用拔阴取阳口诀，主动切换到阳面模式
- **践行**：在身界实际行动，完成三界转化闭环
