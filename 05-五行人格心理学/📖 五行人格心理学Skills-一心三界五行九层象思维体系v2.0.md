# 五行人格心理学Skills

> **核心定位**：植根于中华传统文化本原性认知范式（象思维）的生命能量认知与发展全息理论模型

> **体系本质**：动态、整体、直观的生命发展模型

> **核心目标**：实现个人与组织的"全息（全人）发展"，导向生命觉醒与超越

---

## 体系架构总览

### 核心逻辑闭环
```
一心（源头） → 三界（显相） → 五行（规律） → 九层（进阶） → 回归一心
```

### 五大核心要素
1. **一心**：终极本源，纯粹觉知
2. **三界**：显化载体，信息体/能量体/物质体
3. **五行**：运行规律，木火土金水五种能量模式
4. **九层**：发展阶梯，从健康到病态的九级量化
5. **象思维**：认知灵魂，方法论根基

## 核心概念

### 一心：体系原点与终极目标

**定义**：纯粹的觉知，是超越一切二元对立、无有任何内容的"知性"本身。

**核心特质**：
- 无对境：不是"知道某个东西"，而是"知道"本身
- 无分别：没有"我"与"非我"的分别念
- 本自具足：每个生命内在固有
- 如如不动：不随念头、情绪、外境生灭而变化

**体认方法**：
- 直指法：在念头生起的刹那或熄灭的间隙，直接反观那个"能知"的背景
- 默照禅：静静地坐着，不追逐念头，也不压制念头，单纯地"知道"一切的发生

### 三界：显化载体与分析维度

| 三界 | 对象思维层次 | 定义 | 功能 |
|------|------------|------|------|
| 信息体 | 原象层 | 意识、念头、价值观、信念系统 | 主导方向与意义 |
| 能量体 | 意象层 | 情绪、情感、气场、生命力 | 驱动行为与感受 |
| 物质体 | 物象层 | 身体、行为、语言、外貌 | 执行与显现 |

### 五行：运行规律与动力学工具

| 五行 | 功能 | 阳面特质 | 阴面特质 |
|------|------|---------|---------|
| 木 | 生发 | 仁德、正直、有主见 | 傲慢、顶撞、固执 |
| 火 | 炎上 | 明理、热情、守礼 | 急躁、虚荣、控制欲 |
| 土 | 承载 | 信实、包容、稳重 | 多疑、迟钝、抱怨 |
| 金 | 收敛 | 义气、果断、清爽 | 刻薄、计较、冷酷 |
| 水 | 润下 | 智慧、柔和、谦卑 | 恐惧、懒散、优柔 |

### 九层：发展阶梯

| 区间 | 层级 | 核心状态 | 干预重点 |
|------|------|---------|---------|
| 健康 | 1-3层 | 解放/心理能力/社会价值 | 提升与奉献 |
| 一般 | 4-6层 | 失衡/人际控制/过度补偿 | 觉察与转化 |
| 不健康 | 7-9层 | 侵害/妄想强迫/病态破坏 | 治疗与稳定 |

## 核心干预技术

### 拔阴取阳

| 阴面 | 转化 |
|------|------|
| 阴木（怒） | 转向"有主见但尊重他人" |
| 阴火（恨） | 转向"明理、温暖" |
| 阴土（怨） | 转向"包容、信任" |
| 阴金（恼） | 转向"果断、仗义" |
| 阴水（烦） | 转向"深思、灵活" |

### 化克为生

**金克木案例**：
- 上级转阳金（清晰公正制定规则帮助员工成长）
- 员工转阳木（理解规则价值主动规划工作）
- 结果："金克木"转化为"金规范木，木成就于规则"

## 象思维方法

### 三层次结构
1. **物象**：感官可直接观察的外在形象
2. **意象**：物象与主观情志结合的象征性形象
3. **原象**：超越具体物象的终极本质

### 实践流程
```
观象 → 取象 → 运象
```
