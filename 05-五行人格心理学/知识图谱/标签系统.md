# 五行人格心理学标签系统

## 一、标签系统设计原则

### 1.1 三层标签架构
```
核心层（一级标签）
    ↓
系统层（二级标签）
    ↓
应用层（三级标签）
```

### 1.2 标签命名规范
- 使用中文标签名，便于中文用户检索
- 使用 # 作为标签前缀
- 使用下划线连接多字词组
- 同一文档多个标签用逗号分隔

## 八、标签使用指南

### 8.1 文档标签添加方法
在Obsidian文档头部YAML frontmatter中添加tags字段：
```yaml
---
title: "文档标题"
created: "2026-03-23"
tags: [#纯粹觉知, #三界, #五行, #九层, #大圆满]
---
```

### 8.2 标签查询方法
使用Obsidian的#标签搜索功能，可以：
- 按标签筛选文档
- 查看标签统计
- 发现标签关联

### 8.3 标签维护原则
- 每月检查一次标签使用情况
- 合并重复标签
- 删除不使用标签
- 更新标签定义
