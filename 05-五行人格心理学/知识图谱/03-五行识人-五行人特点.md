# 第3章：五行人特点

## 🌲 木行人——才华横溢的艺术家
**木曰曲直**: 生发·舒畅·调达

| 维度 | 内容 |
|------|------|
| **形体** | 修长、面长、色青、三长（脸·四肢·手指） |
| **心性** | 有才华、劳心、少力、多忧劳 |
| **优势** | 才气独特、创新成长、正直不屈、仁德善良 |
| **劣势** | 傲慢孤僻、抗上不服、偏执顶撞、忧愁善变 |
| **方位** | 东方（吉位），北方亦宜（水生木） |
| **声音** | 高唱嘹亮，齿音 |
| **病理** | 怒伤肝，头昏眼花、两臂麻木 |
| **职场** | 专家型、策划人、文秘/出版/设计/教育 |

## 🔥 火行人——热情主动的领导者
**火曰炎上**: 明亮·炽烈·发散

| 维度 | 内容 |
|------|------|
| **形体** | 锐隆、头尖面红、三尖（头顶·鼻头·下颚） |
| **心性** | 有气、轻财、少信多虑、见事明、急心 |
| **优势** | 推动提升、明辨守礼、热情自信、主动影响 |
| **劣势** | 急躁争斗、炫耀虚荣、自卑夸张、情绪多变 |
| **方位** | 南方（吉位），东方亦宜（木生火） |
| **声音** | 焦烈燥怒，舌音 |
| **病理** | 恨人心，心热心跳、失眠癫狂 |
| **职场** | 领袖型、宣传公关、演艺/餐饮/文化 |

## 🪨 土行人——诚实稳重的协作者
**土曰稼穑**: 承载·容纳·运化

| 维度 | 内容 |
|------|------|
| **形体** | 敦实、圆面、色黄、三厚（身材·胸背·手背） |
| **心性** | 安心、好利人、不喜权势、善附人 |
| **优势** | 信实可靠、耐力持久、宽宏厚道、支持担当 |
| **劣势** | 呆板无趣、迟钝缓慢、沉闷埋怨、固执不化 |
| **方位** | 正南/西南/东北（火生土·土本位） |
| **声音** | 深厚沉稳，鼻音 |
| **病理** | 怨伤脾，膨闷胀饱、腹痛吐泻 |
| **职场** | 协调型、财务经理、房地产/建筑/营销 |

## ⚔️ 金行人——仗义果断的管理者
**金曰从革**: 坚固·收敛·洁净

| 维度 | 内容 |
|------|------|
| **形体** | 方正、面白、三薄（嘴唇·眼皮·手背） |
| **心性** | 身清廉、急心、静悍、善为吏 |
| **优势** | 刚毅坚强、含蓄持重、仗义果断、权威出众 |
| **劣势** | 好胜计较、小气狭隘、尖酸刻薄、自负嫉妒 |
| **方位** | 西方/西北（吉位），西南/东北亦宜（土生金） |
| **声音** | 和润清脆，唇音 |
| **病理** | 恼伤肺，气喘咳嗽、肺痨咳血 |
| **职场** | 支配型、业务经理、金融/法律/科技/管理 |

## 💧 水行人——左右逢源的协调者
**水曰润下**: 滋润·沉潜·静藏

| 维度 | 内容 |
|------|------|
| **形体** | 圆满、色黑、三肥（眼皮·下巴·手背） |
| **心性** | 不敬畏、善欺绐人、情感丰富 |
| **优势** | 滋润祥和、智慧亲和、外柔内刚、善解人意 |
| **劣势** | 消极放任、圆滑世故、忧虑多疑、散漫无序 |
| **方位** | 北方（吉位），西方/西北亦宜（金生水） |
| **声音** | 圆清缓急，喉音 |
| **病理** | 烦伤肾，腰腿病痛、遗精阳萎 |
| **职场** | 适应型、人事经理、水产/流通/旅游/媒体 |
