### 联系十一：三界结构 ↔ 马车模型 ↔ 五行四种失衡形态（2026-03-25新增）

```
【马车模型 × 五行失衡诊断】
驾车人（灵界·信息体）= 价值观·使命·方向
  → 失衡表现：扭曲（错误的方向）、模糊（迷失方向）

  → 失衡表现：冲突（跑错了方向）

  → 失衡表现：遮蔽（动力被堵塞）

   → 如：傲慢取代仁德，贪婪取代义礼
2. 模糊（灵界）：信念系统薄弱，找不到生命意义
   → 如：不知自己是何行人，缺乏内在罗盘

   → 如：想热情（阳火），却情绪化失控（阴火）

诊断步骤：
① 先定位失衡在哪一界（身·心·灵）
② 再识别是哪种失衡形态（扭曲·模糊·冲突·遮蔽）
③ 最后对应五行选择转化口诀
```

**深度洞察**：马车模型是三界关系的最佳教学工具——当"马乱跑"（身界失控）时，要先检查"驾车人方向"（灵界价值观），再调整"车的性能"（心界情绪），而非只是"强行拉缰绳"（意志压制）。这是三界诊断比行为矫正更深层的原因。
