## 一、文化类型诊断

### 1.1 老板文化境界测评

| 测评维度 | 问题 | 得分(1-5) |
|----------|------|-----------|
| **境界高度** | 老板是否有超越利润的使命追求？ | |
| **格局宽度** | 老板是否考虑员工、客户、社会的利益？ | |
| **修为深度** | 老板是否以身作则践行文化？ | |
| **分享气度** | 老板是否愿意与员工分享成果？ | |

**诊断结论**：  
□ 求法型老板（4项≥4分）  
□ 用术型老板（境界≤3，其他较高）  
□ 愚人型老板（3项≤2分）  
□ 骗子型老板（境界≥4，其他≤2）

### 1.2 团队文化状态测评

| 测评维度 | 问题 | 得分(1-5) |
|----------|------|-----------|
| **求法意愿** | 团队是否有学习和成长的追求？ | |
| **实用程度** | 团队是否只看眼前利益？ | |
| **凝聚程度** | 团队是否有共同的价值观？ | |
| **执行程度** | 团队是否相信并践行文化？ | |

**诊断结论**：  
□ 高境界团队（4项≥4分）  
□ 实用主义团队（只看利益）  
□ 迷茫团队（没有追求）  
□ 不信团队（说一套做一套）
