# 五行特质完整解析 · 五行人详解

## 一、木行人详解

### 1.1 自然象根基

**取象**：曲直之象（幼苗破土、旭日东升）

**核心体性**：
- 以"温柔"为体：生发能量的原创性本然，不刚硬、不强迫
- 以"曲直"为性：顺应环境而不迷失本质

**原创性本质**："生发"——突破束缚、自然成长、舒展向上的原创性能量

### 1.2 阴阳特质

#### 阳性特质（顺境滋养）

| 特质 | 表现 | 本质 |
|------|------|------|
| **生发** | 创新突破、主动成长 | 生发原象的原创性流动 |
| **调达** | 仁德善良、处事灵活 | 曲直之象的原创性适配 |
| **舒畅** | 正直豁达、内心通透 | 生发意象的原创性共振 |
| **内蕴** | 安静积淀、厚积薄发 | 生发物象的原创性沉淀 |

#### 阴性特质（逆境阻滞）

| 特质 | 表现 | 本质 |
|------|------|------|
| **曲迂** | 傲慢孤僻、才气偏执 | 生发象的概念化固化 |
| **抗逆** | 抗上不服、固执顶撞 | 生发象的对抗化 |
| **郁结** | 忧愁善变、疏远冷淡 | 生发象的阻滞化 |

### 1.3 外观识别

- **核心特征**：三瘦（长）——身直/肢长/形瘦
- **观象要点**：不是"必须瘦长"，而是"生发"能量的物象流动

### 1.4 发展层级表现

| 层级 | 核心表现 |
|------|---------|
| 1级（解放） | 仁德舒展，无偏执 |
| 2级（心理能力） | 积极进取，有韧性 |
| 3级（社会价值） | 正直豁达，善分享 |
| 4级（失衡） | 偏执敏感，爱较真 |
| 5级（人际控制） | 傲慢易怒，爱抗上 |
| 6级（过度补偿） | 冲动激进，爱冒险 |
| 7级（侵害） | 孤僻冷漠，爱指责 |
| 8级（妄想强迫） | 极端偏执，爱钻牛角 |
| 9级（病态破坏） | 仇恨满溢，无底线 |

### 1.5 转化技术

**拔阴取阳**：
- 阴木傲慢 → 阳木学（创新成长）
- 阴木抗上 → 阳木仁（仁德善良）

**化克为生**：
- 木克土（胃发堵）→ 土生金→金克木
