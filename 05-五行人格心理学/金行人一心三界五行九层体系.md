# 金行人"一心三界五行九层"象思维体系

> **核心要义**: 从清明决断到义理圆融的完整动态路径  
> **象思维特质**: 清明决断性、收敛定义性  
> **终极目标**: 从"切割万物的利刃"到"衡量天道的尺度"  

## 核心逻辑闭环
一心（清明觉知） → 三界（金象显化） → 五行（金性生克） → 九层（义贪阶梯） → 回归一心（义理圆融）

## 三界显化特征

### 身界（物象层）- 金之形
**阳面**: 轮廓方正、骨骼清朗、动作干脆利落、声音清脆  
**阴面**: 过度清瘦干枯、僵硬拘谨、易患呼吸系统疾病

### 心界（意象层）- 金之心理
**阳面**: 情绪深沉肃穆、逻辑严密、追求秩序公正  
**阴面**: 烦躁挑剔、思维僵化、计较控制

### 灵界（原象层）- 金之价值
**阳面**: 价值观为"义"——公正、原则、责任、革新  
**阴面**: 扭曲为"贪"——狭隘、自私、刻薄、固守

## 九层能量阶梯
- **健康区间（1-3级）**: 义理圆融 → 冷静果决 → 公正严明
- **一般区间（4-6级）**: 固执好辩 → 计较尖刻 → 自卑孤傲  
- **不健康区间（7-9级）**: 刻薄报复 → 偏执强迫 → 病态破坏

## 核心调适技术
- **拔阴取阳口诀**: "阴金恼转阳金行"、"阴金贪转阳金义"
- **化克为生**: 火生土→土生金（稳定原则）、金生水（原则转智慧）
