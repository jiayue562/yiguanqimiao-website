## 🔗 五行维度关联网络

### 木行人维度关联
```
曲直（才气独特/傲慢孤僻）
    │
    ├─ 生发（创新成长/抗上不服）── 认知风格，成长动力
    │
    ├─ 舒畅腾上（正直不屈/偏执顶撞）── 人际互动，原则坚守
    │
    ├─ 调达柔和（仁德善良/忧愁善变）── 情感道德，柔和关怀
    │
    └─ 内在成长（安静内向/疏远冷淡）── 自我独处，空间需求
```

### 火行人维度关联
```
炎上（推动提升/急躁争斗）
    │
    ├─ 明亮（明辨守礼/炫耀虚荣）── 社交互动，形象管理
    │
    ├─ 炽烈（热情自信/自卑夸张）── 自我认知，情感特质
    │
    ├─ 发散（主动影响/情绪多变）── 人际影响，支配能力
    │
    └─ 迅疾（气势夺人/无序冲动）── 目标执行，行为风格
```

### 土行人维度关联
```
承载（信实可靠/呆板无趣）
    │
    ├─ 适应（耐力持久/迟钝缓慢）── 环境适应，应对变化
    │
    ├─ 容纳（宽宏厚道/沉闷埋怨）── 情感包容，人际接纳
    │
    ├─ 运化（支持担当/固执不化）── 责任担当，团队支持
    │
    └─ 稳定（镇定稳重/拖拉磨蹭）── 自我调节，情绪稳定
```

### 金行人维度关联
```
坚固（刚毅坚强/好胜计较）
    │
    ├─ 收敛（含蓄持重/小气狭隘）── 行为收敛，情感克制
    │
    ├─ 锋利（仗义果断/尖酸刻薄）── 人际立场，原则行动
    │
    ├─ 光洁（权威出众/自负嫉妒）── 权威认知，价值实现
    │
    └─ 变革（目标计划/焦虑紧张）── 目标执行，变革推动
```

### 水行人维度关联
```
润下（滋润祥和/消极放任）
    │
    ├─ 通透（智慧亲和/圆滑世故）── 智慧社交，外在表现
    │
    ├─ 沉潜（外柔内刚/忧虑多疑）── 内在韧性，原则坚守
    │
    ├─ 静藏（善解人意/散漫无序）── 情感共情，感知能力
    │
    └─ 养物（多才多艺/意志薄弱）── 能力展现，意志状态
```
