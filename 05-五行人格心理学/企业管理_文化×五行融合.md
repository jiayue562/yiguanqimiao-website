## 五行闭环与文化落地

### 文化五行闭环

```
木（规划）→ 火（执行）→ 土（沉淀）→ 金（规则）→ 水（调整）
    ↑                                              ↓
    ←←←←←←←←←← 持续迭代 ←←←←←←←←←←←←←←←
```

### 各行详解

#### 木行：Context传递与目标规划

**木行特质**：
- 生发性：目标清晰才能自主生发
- 规划性：OKR明确O与KR
- 觉照性：员工能清晰看见Context

**文化落地**：
```
木生火：清晰的目标（Context）点燃执行的热情
```

#### 火行：绝对坦诚与光明照世

**火行特质**：
- 光明性：信息透明，照见问题
- 洞察性：直接反馈，洞察本质
- 礼明性：直接≠粗暴，遵循礼仪

**文化落地**：
```
火生土：坦诚的反馈沉淀为组织知识
```

#### 土行：人才密度与信实承载

**土行特质**：
- 承载性：高密度人才承载文化
- 信实性：稳定的文化根基
- 稳定性：文化代代传递

**文化落地**：
```
土生金：稳定的文化形成明确的规则
```

#### 金行：规则制度与决断精准

**金行特质**：
- 决断性：明确边界，果断决策
- 精准性：规则清晰，执行精准
- 清白性：红线分明，不可逾越

**文化落地**：
```
金生水：规则的执行需要灵活的调整
```

#### 水行：持续迭代与变通智慧

**水行特质**：
- 变通性：适时调整，灵活适应
- 流动性：持续迭代，动态发展
- 智慧性：审时度势，顺势而为

**文化落地**：
```
水生木：适时的调整孕育新的规划
```

## 文化病的五行诊断

### 文化病征与五行归属

| 文化病征 | 五行归属 | 病根 | 症状 | 后果 |
|----------|---------|------|------|------|
| 控制过度 | 金过旺克木 | 缺乏信任 | 不敢自主决策 | 创新抑制 |
| 信息封闭 | 水过弱 | 恐惧透明 | 隐瞒问题 | 信任缺失 |
| 责任推诿 | 土失衡 | 不担责任 | 互相推诿 | 效率低下 |
| 竞争恶化 | 火过旺 | 缺乏和谐 | 恶意竞争 | 团队分裂 |
| 僵化守旧 | 木受阻 | 恐惧变化 | 拒绝创新 | 发展停滞 |

### 文化转化：拔阴取阳

| 阴面文化 | 五行 | 阳面文化 | 转化口诀 | 实践路径 |
|---------|------|---------|---------|---------|
| 控制型 | 金过旺 | 赋能型 | 木生火 | 信任→授权→赋能 |
| 封闭型 | 水过弱 | 透明型 | 火照明 | 恐惧→坦诚→信任 |
| 推诿型 | 土失衡 | 担当型 | 土生金 | 自保→担当→沉淀 |
| 破坏型 | 木受阻 | 创新型 | 木生火 | 压制→引导→生发 |
| 僵化型 | 水不足 | 适应型 | 水生木 | 保守→调整→灵活 |
