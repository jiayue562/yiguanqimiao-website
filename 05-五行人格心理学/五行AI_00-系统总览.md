## 二、各层详解

### L1 应用层
多场景入口，用户交互界面

| 场景 | 主导五行 | 协作五行 |
|------|----------|----------|
| 亲密关系 | 火 | 金、水 |
| 亲子关系 | 土 | 火、木 |
| 领导力 | 火 | 金、木 |
| 团队建设 | 木 | 火、土 |
| 高效沟通 | 火 | 金、水 |
| 健康养生 | 土 | 水、木 |

### L2 路由层
意图识别 → 场景分类 → 五行分流 → 技能匹配

**核心文件**：`06-Skills/wuxing-router/SKILL.md`

### L3 调度层
任务分解 → 子Agent调度 → 结果聚合 → 执行监控

**核心文件**：`06-Skills/wuxing-orchestrator/SKILL.md`

### L4 核心层
五个五行分智能体独立处理

| 智能体 | 核心特质 | 职责 |
|--------|----------|------|
| 🌿木 | 生发、条达 | 成长、规划、创新 |
| 🔥火 | 热情、连接 | 关系、沟通、领导 |
| 🏔️土 | 稳定、承载 | 养育、整合、稳定 |
| ⚔️金 | 决断、收敛 | 评估、决策、秩序 |
| 💧水 | 沉潜、智慧 | 洞察、分析、适应 |

**核心文件**：`02-核心智能体/` 目录

### L5 记忆层
用户画像 → 会话记忆 → 诊断评估 → 干预转化

**核心文件**：`06-Skills/wuxing-user-profile/SKILL.md`

### L6 底座层
一心三界五行九层理论 + Obsidian知识库

**核心文件**：`02-五行人心理学/` 目录（12个文档）

## 六、协作流程示例

### 示例：亲密关系咨询

```
用户输入：「我和老公总是吵架，不知道怎么办？」

L2 路由层：
├── 识别场景：亲密关系
├── 五行分流：🔥火（主）
└── 协作：金⚔️ + 水💧

L3 调度层：
├── 火Agent处理：情感分析
├── 金Agent协作：冲突评估
└── 水Agent支持：深层洞察

L4 核心层：
├── 🔥火Agent：分析情感需求和沟通模式
├── ⚔️金Agent：评估冲突根源
└── 💧水Agent：提供深度洞察

L5 记忆层：
├── 更新用户画像
├── 记录本次咨询
└── 追踪关系变化

L1 应用层：
└── 输出：综合建议报告
```
