# 📖 诸葛亮·七步识人法

## 核心定义

诸葛亮的"**七步识人法**"是五行识人"**象辞结合阶段**"的实践典范，展示了从静态观象转为**动态考象**的方法。

## 七步识人法

| 步骤 | 方法 | 观察重点 |
|------|------|---------|
| 1 | 问志 | 询问志向抱负 |
| 2 | 观变 | 观察应变能力 |
| 3 | 咨识 | 咨询见识学识 |
| 4 | 告勇 | 告知勇武之事 |
| 5 | 醉性 | 以酒观性 |
| 6 | 临廉 | 临之以利观廉 |
| 7 | 期信 | 期以信义 |

## 行为情境设计

诸葛亮识人法的创新点在于通过**行为情境**激发内在能量态势的外显：

| 测试情境 | 对应五行 | 识人原理 |
|---------|---------|---------|
| 醉之以酒而观其性 | 水形人智慧潜藏 | 酒属水，激发本性 |
| 临之以利而观其廉 | 金形人义利抉择 | 金属财利，考验义德 |

## 动态考象方法

```
静态观象（形体） ──→ 动态考象（情境）
       │                      │
       ▼                      ▼
   纯象阶段            象辞结合阶段
   黄帝内经            诸葛亮
       │                      │
       └──────────┬───────────┘
                  │
                  ▼
           能量态势外显
```

## 核心金句

1. **"通过行为情境激发内在能量态势外显"**
2. **"从静态观象转为动态考象"**
3. **"酒属水，激发本性；金属财利，考验义德"**
