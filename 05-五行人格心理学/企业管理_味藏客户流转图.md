## 二，流转路径

### 2.1 完整路径图

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│    入口（精准线上引流）                                       │
│          ↓                                                   │
│    客户池（精准人群：刺身爱好者）                               │
│          ↓                                                   │
│    ┌──────────────────────────────────────────────────┐     │
│    │                                                      │     │
│    │  核心转化 → 储值会员（最强粘性）                       │     │
│    │                                                      │     │
│    │  次要转化 → 普通微会员（日后升级）                       │     │
│    │                                                      │     │
│    └──────────────────────────────────────────────────┘     │
│          ↓                                                   │
│    价值提升 → 堂食+酒水（高客单）/ 外卖（增加频次）            │
│          ↓                                                   │
│    闭环裂变 → 转介绍（高质量新客户）                           │
│          ↓                                                   │
│    回到客户池（循环）                                         │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 五大节点

| 节点 | 目标 | 策略 |
|------|------|------|
| **入口** | 精准引流 | 刺身爱好者 |
| **转化** | 储值会员 | 最强粘性 |
| **次要转化** | 普通微会员 | 退而求其次 |
| **价值提升** | 高客单+多频次 | 产品组合 |
| **裂变闭环** | 转介绍 | 高满意度 |

## 三、入口：精准线上引流

### 3.1 引流目标

> **引流的目标必须精准，是"喜欢刺身的人"，以确保后续转化率。**

### 3.2 精准定位

| 维度 | 内容 |
|------|------|
| **目标人群** | 刺身爱好者 |
| **精准原因** | 确保后续转化率 |
| **平台** | 线上渠道 |

## 四，客户池：精准人群聚集

### 4.1 核心定位

> **客户池的中心枢纽是"储值会员"。**

### 4.2 精准人群

| 人群 | 说明 |
|------|------|
| **刺身爱好者** | 精准定位 |
| **转化潜力** | 高 |

## 五、转化：会员体系

### 5.1 核心转化

| 类型 | 目标 | 粘性强度 |
|------|------|---------|
| **储值会员** | 核心目标 | 最强 |

**储值会员价值**：
- 把钱放这里 = 最强信任
- 粘性最强

### 5.2 次要转化

| 类型 | 目标 | 说明 |
|------|------|------|
| **普通微会员** | 次要目标 | 退而求其次 |

**升级路径**：
- 先转化为普通微会员
- 日后再思考如何转化为储值会员

---

## 六、价值提升

### 6.1 提升策略

| 维度 | 策略 | 目标 |
|------|------|------|
| **堂食+酒水** | 组合推荐 | 高客单 |
| **外卖** | 增加消费频次 | 多场景 |

### 6.2 价值最大化

- **高客单**：通过堂食+酒水组合
- **多频次**：通过外卖增加消费场景
- **多场景**：覆盖不同用餐需求

---

## 七、闭环裂变

### 7.1 裂变机制

> **引导满意的高价值客户进行"转介绍"，为增长引擎注入源源不断的高质量新客户。**

### 7.2 裂变路径

```
高满意度客户
    ↓ 转介绍
高质量新客户
    ↓ 导入
客户池
    ↓ 转化
储值会员
    ↓ 循环
价值提升
    ↓ 裂变
更多高质量客户
```
