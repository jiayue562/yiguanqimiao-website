## 🎯 应用指南

### 快速五行识人流程

1. **外观观察**（1分钟）
   - 体型：瘦高挺拔、中等匀称、结实魁梧、瘦削匀称
   - 面部：棱角分明、圆润饱满、方正端实、圆润柔和
   - 神态：锐利有神、热情外露、稳重内敛、清明冷静、柔和内敛

2. **能量感知**（30秒）
   - 气息：急促短浅、充满活力、稳重深沉、锐利断续、柔和绵长
   - 眼神：坚定有光、热情闪烁、温润包容、锐利聚焦、柔和流转
   - 动作：敏捷有力、热情大方、稳重缓慢、精准干练、流畅灵活

3. **五行匹配**（30秒）
   - 木：木秀于林、才华出众
   - 火：火样能量、热情大方
   - 土：土般宽容、耐受力强
   - 金：金峰利刃、锐意改革
   - 水：水般圆润、善意聪明

4. **阴阳判断**（30秒）
   - 阳木：正直担当、创新破局
   - 阴木：孤傲抗上、固执己见
   - 阳火：热情自信、积极引领
   - 阴火：急躁炫耀、虚荣好胜
   - 依此类推...

5. **职业定位**（1分钟）
   - 根据五行职业匹配表定位
   - 结合阴阳状态调整建议

### 团队配伍设计流程

1. **识别核心需求**
   - 团队目标是什么？
   - 项目特点是什么？
   - 市场环境如何？
   - 资源约束有哪些？

2. **五行诊断**
   - 现有团队的五行分布
   - 每个成员的阴阳状态
   - 团队能量流动状态
   - 存在的冲突点

3. **设计配伍方案**
   - 五行全备（木火土金水）
   - 相生循环（木→火→土→金→水→木）
   - 主导+支持（某一行主导，其他行支持）
   - 阴阳平衡（阴阳比例优化）

4. **能量配伍**
   - 确定五行主次
   - 设计能量流动路径
   - 建立相生循环
   - 配置化克为生机制

5. **动态调整**
   - 定期评估效果
   - 根据阶段调整
   - 优化配伍方案
   - 持续进化提升

### 化克为生实践法

1. **识别克制关系**
   - 木克土：木克土
   - 土克水：土克水
   - 水克火：水克火
   - 火克金：火克金
   - 金克木：金克木

2. **引入相生中介**
   - 木克土 → 木生火 → 火生土
   - 土克水 → 土生金 → 金生水
   - 水克火 → 水生木 → 木生火
   - 火克金 → 火生土 → 土生金
   - 金克木 → 金生水 → 水生木

3. **能量转化实践**
   - 觉察对抗能量
   - 理解克制的根源
   - 引入中介五行能量
   - 建立相生流动
   - 持续观察效果
