## 一、五行学说核心原理与关系法则

### 1.1 五行的哲学本质

> **关键洞察**：五行的核心不在"金木水火土"五种物质，而在"行"字——运行、运动与相互作用。这是一套**动态系统模型**，不是元素分类表。

五行是中国古代先贤"近取诸身、远取诸物"后提炼的**五种基本能量状态与功能属性**：

| 行 | 核心属性 | 运行趋势 | 象征特征 |
|----|---------|---------|---------|
| **木曰曲直** | 生发·条达·舒展 | 向上向外突破 | 春·东·青·肝·仁 |
| **火曰炎上** | 温热·明亮·升腾 | 向上扩散 | 夏·南·红·心·礼 |
| **土爰稼穑** | 承载·化育·稳定 | 中和运化 | 长夏·中·黄·脾·信 |
| **金曰从革** | 肃降·收敛·清洁 | 收缩变革 | 秋·西·白·肺·义 |
| **水曰润下** | 寒凉·滋润·闭藏 | 下行潜藏 | 冬·北·黑·肾·智 |

**思维方式**：**取象比类**——不拘泥于实体，重在功能与关系的类比。用同一套逻辑贯穿天地人万事万物，形成"全息生命观"。

### 1.2 五行关系的核心法则

#### 相生关系（母子关系·促进资生）
```
木生火 → 火生土 → 土生金 → 金生水 → 水生木
（无限循环·五行能量持续流转）
```
| 生克对 | 自然比喻 | 人体对应 |
|--------|---------|---------|
| 木生火 | 木柴燃烧生火 | 肝藏血以济心 |
| 火生土 | 火烬化为灰土 | 心阳温煦脾土 |
| 土生金 | 土矿提炼金属 | 脾土化精微以充肺 |
| 金生水 | 金属凝结露水 | 肺气清肃助肾纳气 |
| 水生木 | 水润泽树木 | 肾水滋养肝木 |

#### 相克关系（制约关系·防止过亢）
```
木克土 → 土克水 → 水克火 → 火克金 → 金克木
（循环制约·维持动态平衡）
```

#### 制化与平衡（生中有克·克中有生）
> 正是"生中有克，克中有生"，系统才能实现**动态平衡**，即"制则生化"。

#### 失衡状态：相乘与相侮
- **相乘**：过度的克（量的失衡）→ 如木气过亢→木乘土→肝气乘脾
- **相侮**：反向的克（质的失衡）→ 如木气极盛→木侮金→肝火犯肺
