# 💧 水行人关系模式
## 关系功课 · 情感业力

## 一、情感业力模式

### 1.1 核心渴望

```
【核心渴望】
   源于成长环境（可能父母严格）
   → 内心深深渴望"被需要"
   → 认为只有被别人需要时，才是安全和幸福的
```

### 1.2 关系倾向

| 关系类型 | 互动模式 | 潜在挑战 |
|---------|---------|---------|
| **木行人** | 付出-依赖循环 | 依赖循环 |
| **火行人** | 初时"水火不容"，后可能"水火既济" | 冲突与和解 |
| **土行人** | 容易"和稀泥" | 相互憋闷 |
| **金行人** | 顺从-边界 | 边界问题 |
| **水行人** | 温和盟友 | 决策困难 |

## 二、关系诊断矩阵

```
                          对方状态
                    阳（成长）    │    阴（失衡）
              ┌─────────────────┼─────────────────┐
      阳      │   ☀️ 滋养共生  │   🌊 过度付出  │
      (自我)  │   双向成长      │   单向消耗      │
              ├─────────────────┼─────────────────┤
      阴      │   🌙 被迫成长  │   🌑 互相消耗  │
              │   被动改变      │   双输局面      │
              └─────────────────┴─────────────────┘

【四象限解读】
☀️ 滋养共生：最佳状态，相互滋养，共同成长
🌊 过度付出：需要设立边界，减少单方面付出
🌙 被迫成长：虽不舒适，但有成长机会
🌑 互相消耗：需要退出或转化
```

## 三、关系成长关键

### 3.1 关键课题

> **区分"阳水"（无条件的爱，本自具足）与"阴水"（付出是为了填补缺失的爱）**

### 3.2 核心功课

| 功课 | 说明 | 实践方法 |
|------|------|---------|
| **学会说不** | 区分真正愿意 vs 讨好 | 边界练习 |
| **内在确认** | 不依赖外在认可 | 自我肯定 |
| **选择滋养** | 从"必须付出"到"选择滋养" | 觉察动机 |
| **互惠关系** | 建立双向滋养关系 | 沟通需求 |
