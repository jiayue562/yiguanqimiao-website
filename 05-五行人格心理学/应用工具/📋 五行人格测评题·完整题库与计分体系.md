# 📋 五行人格测评题·完整题库与计分体系

> **核心定义**：基于象思维观象取象运象三法和一心三界五行九层体系构建的全息化人格测评工具，涵盖心理、身体、灵性三个维度，包含外观全息识人法，实现从"外在形貌→内在心理→灵性内核"的五行人格完整诊断。

---

## 🎯 一、测评体系总览

### 1.1 三维全息测评矩阵

```
                          ┌──────────────┐
                          │  灵性部分    │ ← 生命观/价值观/使命感
                          │  (4题·内核)  │
                          ├──────────────┤
┌──────────────┐          │  心理部分    │
│  外观识别    │  ──────→ │  (13题·心智) │ ← 行为模式/情绪反应/社交风格
│  (五行全息)  │  相互验证 │              │
└──────────────┘          ├──────────────┤
                          │  身体部分    │
                          │  (23题·载体) │ ← 体形/五官/声音/体质/病候
                          └──────────────┘
```

### 1.2 三大部分权重分配

| 维度 | 题数 | 权重 | 覆盖领域 | 核心原理 |
|------|------|------|---------|---------|
| **心理部分** | 13题 | 35% | 人际关系/自然偏好/自我认知/情绪模式/职场角色/领导风格/养生心性 | 象思维·意象取象 |
| **身体部分** | 23题 | 40% | 体形/头面/肤色/声音/动作/体质/口味/病候/五官/手足/外气耐受 | 象思维·物象观象·全息对应 |
| **灵性部分** | 4题 | 25% | 生命打击/领袖品质/性格缺陷 | 一心·觉性层·超越五行 |
| **外观识别** | 五行各6条 | 验证用 | 体形/脸型/五官/骨相/动态/气质 | 全息识人·不依赖答题的独立验证 |

> **核心原则**：三维交叉验证。身体是物质载体（物象），心理是行为模式（意象），灵性是内核驱动（原象）。三维数据相互印证，减少单一维度的误判。

## 📊 五、计分规则与算法

### 5.1 三维加权计分

```
五行综合得分 = 心理部分得分 × 0.35 + 身体部分得分 × 0.40 + 灵性部分得分 × 0.25
```

### 5.2 各部分计分

| 维度 | 题数 | 每题分数 | 满分 | 权重 |
|------|------|---------|------|------|
| 心理部分 | 13题 | 每选1分 | 13分 | 35% |
| 身体部分 | 23题 | 每选1分 | 23分 | 40% |
| 灵性部分 | 3题* | 每选1分 | 3分 | 25% |

> *注：灵性部分实际4题，但第3题（理想领袖品质）为探测题，不直接计入五行得分，作为灵性倾向的参考维度。

### 5.3 五行得分矩阵（示例）

```
被测评者：张三

         心理(×0.35)  身体(×0.40)  灵性(×0.25)  加权总分
木：       5→1.75      7→2.80       1→0.25      4.80
火：       3→1.05      4→1.60       0→0.00      2.65
土：       2→0.70      5→2.00       0→0.00      2.70
金：       2→0.70      3→1.20       2→0.50      2.40
水：       1→0.35      4→1.60       0→0.00      1.95

判定结果：主导五行 = 木（4.80），次主导 = 土（2.70）
```

### 5.4 判定规则

| 条件 | 判定结论 |
|------|---------|
| 最高分 - 次高分 ≥ 2.0 | 单行确认·主导五行明确 |
| 最高分 - 次高分 在 1.0~2.0 之间 | 主行+副行·双行并行 |
| 最高分 - 次高分 < 1.0 | 多行混合·建议深度追问 |
| 三维得分方向不一致（如心理木高、身体火高） | 五行相生转化中·需追问最近一年的生活变化 |

## 🧬 九、隐秘知识联系（深度学习发现）

### 9.1 心理-身体-灵性的全息对应

| 发现 | 说明 |
|------|------|
| **第11题（怒恨怨恼烦）↔ 第9题（五臟病候）** | 情绪的五行归属与臟腑病变完全对应——怒伤肝（木）、恨伤心（火）、怨伤脾（土）、恼伤肺（金）、烦伤肾（水）。这是《黄帝内经》七情致病理论的精确映射。 |
| **第3题（季节偏好）↔ 第6题（体质适应）↔ 第14题（外气不耐受）** | 三题形成"偏好-适应-不耐受"的体质三连问——喜欢的季节反映五行偏盛，适应的季节反映五行调和，不耐受的外气反映五行偏虚。 |
| **第7题声音↔ 第15题声音感觉↔ 第16题讲话特点** | 声音三连问构成完整的"五音诊断系统"——音质（齿/舌/鼻/唇/喉）→ 音感（高/嘹/闷/脆/低）→ 语式（直/夸/寡/强/慢），三维锁定五行声音人格。 |
| **第18-21题四重身材确认** | 第1题（体形）+ 第18题（直觉身材）+ 第19题（脸型）+ 第21题（身材再确认）构成四重交叉验证，大幅降低单一题目的误判率。 |

### 9.2 测评题间的隐性相生线索

| 相生链 | 题号 | 解读 |
|--------|------|------|
| **木（第1题·人际关系）→ 火（第4题·气质）** | 一个人在人际关系中的模式（木），决定了他展现出来的气质（火）。抢风头的人自然显得有气势。 |
| **火（第2题·聚会场景）→ 土（第8题·助人角色）** | 喜欢热闹聚会的人（火），在团队中往往成为整合者和助人者（土）。火的能量辐射出去后，自然吸引他人靠近形成土的信赖。 |
| **土（第5题·居中转化）→ 金（第6题·老虎威慑）** | 能量居中能化虚实的人（土），内在往往有金的决断和威慑力（金）。土生金——承载的力量最终转化为决断的力量。 |
| **金（第10题·集权领导）→ 水（第11题·自卑退缩）** | 高支配型领导风格的背面（金），是"别人嫌他慢，他反嫌别人急"的水型自我怀疑。金生水·但不是所有金都生阳水。 |
| **水（第13题·冷月寒潭）→ 木（第2题·早晨太阳）** | 水的沉静（冬藏）必然导向木的生发（春生）。水生木——测评题中隐藏着五行相生的心理链条。 |

## 💎 十、核心金句

> "身体是五行的庙宇，心理是五行的香火，灵性是五行的神明。三维不全，何以识人？"

> "五行人格测评不是给你贴一个'木行人'或'火行人'的标签——它是在告诉你，你的身体、心理和灵魂正在往哪个方向流动。知道方向，才能选择转向。"

> "外观识人不是相面术，而是《黄帝内经》'有诸内必形诸外'的全息科学——你的面部长相、身体形态、声音质地、走路姿态，都是内在五行之气的签名。"

> "40道测评题，每一道都是一面镜子。镜子不会说话，但你站在它面前，一定会看到自己。"

> "测评的目的是开始，不是结束。知道自己是哪个五行，只是有了地图——怎么走，往哪里走，能走多远，那是拔阴取阳、化克为生的功夫。"
