## 一、《老饕赋》文化内核

### 1.1 来源

> **《老饕赋》是宋代苏轼所作的一篇赋文，把美食理解得很精妙、很精髓。**

### 1.2 "老饕"定义

> **"老饕"指贪吃之人，特指对美食有极高热爱和追求、懂得品尝美食的人。**

**《老饕赋》可理解为《食客赋》或《美食家赋》**

### 1.3 三层内涵

| 层次 | 内容 | 味藏映射 |
|------|------|---------|
| **烹饪之道** | 庖丁操刀、易牙烹调、水新釜洁、柴火恰到好处 | 匠心标准 |
| **食材之精** | 小猪颈后肉、霜冻螃蟹大螯、樱桃煎蜜、杏仁浆蒸羔羊 | 食材苛求 |
| **享受之境** | 音乐歌舞、香茗美酒、仙境回味 | 体验价值 |

### 1.4 文化内核

- 厨师的心念注入食物，影响能量和口感
- **匠心 = 专注 + 严谨 + 追求极致**
- 完整的身心愉悦的文化体验

### 1.5 客户画像

> **"服务一群懂吃会吃、有要求的客户"**

**老饕画像**：

| 特征 | 说明 |
|------|------|
| **资深** | 美食经验丰富 |
| **专业** | 品鉴能力强 |
| **有要求** | 对品质有高追求 |
| **高价值** | 客单价高 |

## 二、饕餮图腾八大特征

### 2.1 图腾选择理由

> 选择饕餮，因为它既具备**尊贵感**（与定位高端相符），又与"食"相关。

### 2.2 八大核心特征

| 序号 | 特征 | 核心含义 | 商业映射 |
|------|------|---------|---------|
| **1** | 绝非善恶，实为护法 | 守护中国文化，是寺庙龙柱、貔貅、狮子石像一样的护法神兽 | 企业护佑 |
| **2** | 忽略排行，出身尊贵 | 龙和凤的儿子，排行老大，血脉传承尊贵 | 品牌血统 |
| **3** | 放弃萌宠，霸气第一 | 不怒自威，自带气场，不是主动出击 | 创始人底色 |
| **4** | 饕餮盛宴，财富封印 | 国宝级财神，人民币上的饕餮纹，代表国家级财富 | 高端路线 |
| **5** | 爱好夸张，饮食大王 | 饕餮盛宴，吸引高端、爱吃的客户 | 客户筛选 |
| **6** | 不是怪兽，驱妖除魔 | 具有超度生命的能力，让食客感受与众不同 | 餐饮杀业 |
| **7** | 眼睛超大，光明守护 | 艺术化纹路，离卦（九紫离火），太阳神之眼 | 辨识度高 |
| **8** | 告别兽面，纹路封鼎 | 权威高贵，以前皇室贵族用来煮食物和盛食物 | 能量地位 |

### 2.3 饕餮与财富

- 饕餮纹在古代与"鼎"和财富紧密相连
- 代表国家级财富
- 人民币上印的饕餮和饕餮龙纹
- 决定了味藏必须走高端路线

### 2.4 饕餮与饮食

- 饕餮最喜欢的就是食物
- 不仅是物质层面的富足
- 还有能量层面的霸气
- 能量上是地位的象征

## 四、与文化罗盘的关系

### 4.1 价值链接

```
《老饕赋》（客户定位）
    ↓ 吸引
理想客户（老饕）
    ↓ 服务
《文化罗盘》（自身能力）
    ↓ 凭什么
五饕餮 + 文化底层
```

### 4.2 核心逻辑

- **老饕赋** = 吸引谁来（客户画像）
- **文化罗盘** = 用什么吸引（能力展示）
- **价值闭环** = 用深厚实力吸引同频老饕
