## 📚 核心定义

### 五行人格心理学
基于中国传统五行哲学（木火土金水）的人格分析体系，将人的性格特质分为五种基本类型，每种类型具有独特的能量特征、行为模式和发展路径。

### 一心三界五行九层体系
- **一心**：纯粹觉知（五行能量的终极源头）
- **三界**：身界（物象层）、心界（意象层）、灵界（原象层）
- **五行**：木火土金水的相生相克关系
- **九层**：从解放（1层）到病态破坏（9层）的发展阶梯

### 九层发展层级

| 状态 | 层级 | 核心特征 |
|------|------|---------|
| **健康** | 1层·解放 | 超越基本恐惧，从个性转向本体 |
| | 2层·心理能力 | 自我及防御机制发展，应对焦虑 |
| | 3层·社会价值 | 自我积极主动，展现典型个性 |
| **一般** | 4层·失衡 | 屈从于相反特质，自我膨胀 |
| | 5层·人际控制 | 需要他人认可，强化自我形象 |
| | 6层·过度补偿 | 过度补偿冲突焦虑，可能伤害他人 |
| **不健康** | 7层·侵害 | 防御失效，严重反应 |
| | 8层·妄想强迫 | 与现实断绝联系 |
| | 9层·病态破坏 | 公开破坏行为，需立即干预 |

## 🌿 第六章 五行识人的人格分析描述

### 6.1 木行人的五个子维度

**核心能量特征**：生长与条达

**维度逻辑关系**：
```
曲直（才气独特）→ 生发（创新成长）→ 舒畅腾上（正直不屈）→ 调达柔和（仁德善良）→ 内在成长（安静内向）
     ↓                    ↓                    ↓                    ↓                    ↓
  独特性表现        成长动力来源        原则立场维护        柔和关怀展现        自我空间需求
```

#### 维度一：曲直（才气独特 ↔ 傲慢孤僻）

**核心定义**：枝曲干直的缩语，描述树木生长形态主干挺直向上，树枝曲折向外。引申为"木"有生长、兴发、生机、条达、舒展的特性。

**顺境·才气独特**：

| 子维度 | 特质描述 |
|--------|---------|
| 才华横溢 | 充满新思想，乐观、自然、富有创造性，具有独创性思想和对可能性的强烈感受 |
| 思路开阔 | 高度创造力，思路开阔，打破规则的思维方式，点子型人才 |
| 理想主义 | 透过自我感受认知世界，看得远，不在乎现实，更倾情理想 |
| 足智多谋 | 具有想象力、适应性和可变性，视灵感高于一切，足智多谋的发明人 |
| 感受丰富 | 容易感知自己的情绪和内心世界，敏感、重视内心感受，能体验更深的情绪状态 |

**逆境·傲慢孤僻**：

| 子维度 | 特质描述 |
|--------|---------|
| 个人主义 | 从自己的想法出发，不太考虑周围人的感受，忽视与自己希望相矛盾的事实 |
| 我行我素 | 不十分受冷漠与批评干扰，面对相反意见持怀疑态度，十分坚定和坚决 |
| 自卑羞怯 | 在人群中羞怯，有不自然姿态，有强烈自卑感，拙于发言 |
| 刻板固执 | 坚信自己的理想，忽视其他观点作用，对物质环境不十分有兴趣 |
| 不善分享 | 对想法的分享意愿低，内向，很难被人了解，只有小规模朋友圈 |

#### 维度二：生发（创新成长 ↔ 抗上不服）

**核心定义**：描述一个人的认知风格，对经验的前摄寻求和对知识的理解，以及对陌生情境的容忍和探索。

**顺境·创新成长**：

| 子维度 | 特质描述 |
|--------|---------|
| 思想前卫 | 喜欢挑战一切现有理论，企图了解较前卫的思想与行为 |
| 好奇心强 | 善于观察，喜欢接触新的事物，感觉千篇一律令人乏味 |
| 想象丰富 | 富有艺术天赋和审美感，对从经历中直接了解和感受的东西很感兴趣 |
| 足智多谋 | 有独立见解的思考者，重视才智，对于个人能力有强烈欲望 |
| 精于理论 | 天生精于理论，对于复杂而综合的概念运转灵活，优秀的战略思想家 |

**逆境·抗上不服**：

| 子维度 | 特质描述 |
|--------|---------|
| 孤军奋战 | 选择孤独，一心一意努力，忽视邀请别人参加或协助 |
| 充满怪想 | 对事实判断错误，不能意识到自己的非逻辑性，梦想脱离现实 |
| 对立敏感 | 情绪化陷于自己工作，对批评很敏感，追求不可能有的高标准 |
| 不切实际 | 注意"思想"，眼光狭窄，顽固对待变化，忽略不支持立场的重要事实 |
| 感情抽离 | 像置身外来分析自己境况，化解感情冲动，事事以理性角度处理 |

#### 维度三：舒畅腾上（正直不屈 ↔ 偏执顶撞）

**核心定义**：木行能量腾上而无所止，在顺境时正直不屈，在逆境时不顺从、偏执顶撞。

**顺境·正直不屈**：

| 子维度 | 特质描述 |
|--------|---------|
| 专注执着 | 坚定、富有理想化，珍视正直，喜欢说服别人相信观点正确 |
| 讲求实效 | 优秀的解决问题的人，掌握必要事实，找到符合逻辑的明智解决途径 |
| 有担当力 | 正直有主意，有担当力，敢做敢当，遇事不盲从，有定见 |
| 心口如一 | 心口如一，行为端正，能忍辱，常存悲天悯人之心 |
| 超级理想 | 渴望表达超级理想，只与最卓越或能分享理想的人有联系 |

**逆境·偏执顶撞**：

| 子维度 | 特质描述 |
|--------|---------|
| 严肃偏执 | 意见遭到反对时可以非常强硬甚至无情，缺乏随意性，冷淡严肃 |
| 过分理性 | 过分依赖条理性分析，忽视与别人有关的事物 |
| 迟滞不前 | 拒绝计划中存在一点不符合逻辑的地方，因细微错误迟滞不前 |
| 破坏规则 | 喜欢挑战权威、常规和传统观念，对现存规则表现敌意 |
| 缺少耐心 | 对常规细节缺少耐心，需要过多细节或持久贯彻会失去兴趣 |

#### 维度四：调达柔和（仁德善良 ↔ 忧愁善变）

**核心定义**：木行能量以温柔为体，是五行中最柔和的，木行人在五行人中是最善良的。

**顺境·仁德善良**：

| 子维度 | 特质描述 |
|--------|---------|
| 忠于内心 | 把内在和谐视为高于其他一切，敏感、理想化、忠诚，有强烈荣誉感 |
| 坦诚率真 | 坚定地完成自己所选择的事情，承担太多但总要完成每件事 |
| 看透本质 | 独立的、有独创性的思想家，具有本能的洞察力，看到事物深层含义 |
| 精神沟通 | 注重精神上的沟通，遇上有兴趣的话题会滔滔不绝 |
| 不求物质 | "清贫书生"型，不渴求物质，只追求知识，给人吝惜节俭之感 |

**逆境·忧愁善变**：

| 子维度 | 特质描述 |
|--------|---------|
| 缺乏控制 | 开始许多计划但完成寥寥无几，失去兴趣，缺乏自我约束 |
| 优柔寡断 | 始终注意新的感官信息，喜欢开放面对选择，草率而易于厌烦 |
| 不切实际 | 对自己和他人期望过多，不在意如何符合别人标准 |
| 忽略现实 | 过多重视对未来的见解，忽略现在的重要事情和现实 |
| 幻想丰富 | 对感受敏感，沉浸在每一个感觉中，内心戏份特别夸张 |

#### 维度五：内在成长（安静内向 ↔ 疏远冷淡）

**核心定义**：小草的心声是"我长，我长"，木的能量一直关注成长，所有时间精力都用来成长。

**顺境·安静内向**：

| 子维度 | 特质描述 |
|--------|---------|
| 敏锐缜密 | 对细节有很强记忆和判断，重视逻辑客观分析，坚持不懈准时完成工作 |
| 外表平静 | 外表平静超然，内心专心致志于分析问题，喜欢有条理和目的的交谈 |
| 独立自主 | 专家型人才，胜任专业技术，习惯在安静环境独立工作 |
| 思辨力强 | 喜欢抽象概念，讨论理论性问题，解决复杂智力问题 |
| 逻辑分析 | 有条理和系统化，力求精通整个体系，大胆力求掌握感兴趣的事物 |

**逆境·疏远冷淡**：

| 子维度 | 特质描述 |
|--------|---------|
| 低交际性 | 在社会情境中感觉不轻松，畏怯退缩、缺乏自信心，对社交主动回避 |
| 离群索居 | 倾向于悠闲放松，慢节奏不着急，宁愿躲在幕后 |
| 孤寂寡言 | 从小不合群，喜欢一个人呆着，对人不热情甚至冷漠 |
| 热情不高 | 不容易感受到积极情绪，热情不高、平静、冷淡 |
| 孤独被动 | 孤独者，不寻求甚至主动避免社会刺激，回避人群 |

### 6.2 火行人的五个子维度

**核心能量特征**：温热与发散

**维度逻辑关系**：
```
炎上（推动提升）→ 明亮（明辨守礼）→ 炽烈（热情自信）→ 发散（主动影响）→ 迅疾（气势夺人）
     ↓                    ↓                    ↓                    ↓                    ↓
  主动释放能量        社交形象管理        热情自信来源        人际支配能力        目标执行风格
```

#### 维度一：炎上（推动提升 ↔ 急躁争斗）

**核心定义**：火具有发光放热、光芒四散、蒸腾上升之象。火行人属于社会性、主动的个体，人越多越刺激越开心。

**顺境·推动提升**：

| 子维度 | 特质描述 |
|--------|---------|
| 积极乐观 | 容易感受到各种积极情绪，对生活与环境有积极态度，喜欢与人接触 |
| 社交达人 | 热情对待他人，与人相处合作适应能力强，萍水相逢一见如故 |
| 活力四射 | 行动发起者，团队中最有效成员，敢于面对困难义无反顾加快速度 |
| 满腔热情 | 以热情态度对待生命，精力充沛、满腔热情，对尊重和敬佩的事业非常忠诚 |
| 鼓动煽情 | 天然领导者，受人欢迎有魅力，具有煽动能力，向别人传播观点 |

**逆境·急躁争斗**：

| 子维度 | 特质描述 |
|--------|---------|
| 制造冲突 | 天生有放大事实的本领，可以制造冲突和挑起是非 |
| 愤怒敌意 | 容易发火，感到不公正待遇后充满愤恨，报复心真实 |
| 喜欢挑战 | 喜欢挑战别人，没有结果誓不罢休，可能做出错误臆断 |
| 自视甚高 | 自以为是，可能非常武断，容易被批评伤害 |
| 喜欢争辩 | 喜欢争辩，在争论中不赢不罢休，以自我为中心 |

#### 维度二：明亮（明辨守礼 ↔ 炫耀虚荣）

**核心定义**：火以明热为体，顺境时明辨守礼，逆境时炫耀虚荣。

**顺境·明辨守礼**：

| 子维度 | 特质描述 |
|--------|---------|
| 积极互动 | 热情对待他人，与人相处合作适应能力强，参加或组织社团活动 |
| 喜爱陪伴 | 偏爱有他人陪伴，喜欢人多热闹的场合，开朗有许多朋友 |
| 敢做敢为 | 冒险敢为、少有顾忌，不掩饰不畏缩，保持刚毅性格 |
| 富有魅力 | 社交场合中富有魅力、轻松自如受人欢迎，善于化解紧张局势 |
| 开朗坦率 | 乐意与人相处，有真正的生活热情，顽皮活泼，适应性强 |

**逆境·炫耀虚荣**：

| 子维度 | 特质描述 |
|--------|---------|
| 主观武断 | 认为自己高人一等，太关心别人如何看待自己，害怕被嘲笑 |
| 粗心大意 | 对细节不感兴趣，厌烦搜集材料，觉得处理细节特别麻烦 |
| 缺少计划 | 偏好得过且过，对危机采取应急态度，承担许多任务后发现负担过重 |
| 乐于挑战 | 兴趣在于迅速有效解决问题，对缺少兴奋部分不能坚持完成 |
| 肤浅浮夸 | 知识都很表面，给人浮夸感觉 |

#### 维度三：炽烈（热情自信 ↔ 自卑夸张）

**核心定义**：火焰炽烈燃烧，顺境时热情自信，逆境时自卑夸张。

**顺境·热情自信**：

| 子维度 | 特质描述 |

| 活跃豪爽 | 喜欢周围的人，表达积极友好情绪，喜欢引起别人注意 |
| 多动少静 | 快节奏，显得充满精力，喜欢运动，喜欢刺激冒险 |
| 英雄主义 | 无法忍受乏味生活，恐惧被限制，目标过多，爱出风头 |

**逆境·自卑夸张**：

| 子维度 | 特质描述 |
|--------|---------|
| 情感激动 | 对能力表示疑问时容易感情激动，对无足轻重情况有粗暴反应 |
| 粗犷大胆 | 显露粗犷大胆行为表现勇敢不害怕，"最好的防守就是攻击" |
| 变幻无常 | 恐惧承诺，缺乏识别能力，没法平静下来 |
| 追逐刺激 | 不停止追逐激烈和刺激，参与危险不实际事情 |
| 纵容自己 | 典型消费者，享受逛街、旅游、纵容自己 |

#### 维度四：发散（主动影响 ↔ 情绪多变）

**核心定义**：火的能量一波一波不断向四周主动扩散，顺境时主动影响，逆境时情绪多变。

**顺境·主动影响**：

| 子维度 | 特质描述 |

| 反应敏捷 | 对外界信息非常敏感，最早知道外界变化，高度热忱，反应敏捷 |
| 永不言倦 | 达成目标后常感失落，寻找新目标继续追赶，有追求就有动力 |
| 活力十足 | 喜欢接触新事物，见闻比别人多，害怕没有选择 |

**逆境·情绪多变**：

| 子维度 | 特质描述 |
|--------|---------|
| 独断专行 | 自视甚高，力图改变他人，非常武断，力排众议 |
| 紧张激动 | 情绪不稳定，缺乏耐心，心神不安，态度兴奋 |
| 过多要求 | 有心理压力，过多要求和冲动，对情绪调节应对能力差 |
| 喜新厌旧 | 容易对工作失去兴趣，注意力转移到新事物上 |
| 爱发脾气 | 爱发脾气，情绪激昂，口不择言，以最严厉言语责骂对方 |

#### 维度五：迅疾（气势夺人 ↔ 无序冲动）

**核心定义**：火的能量在于燃烧，火体瞬间飘忽不定，顺境时气势夺人，逆境时无序冲动。

**顺境·气势夺人**：

| 子维度 | 特质描述 |
|--------|---------|
| 气场强大 | 清楚自己需要，眼里没有重复性任务，期待震撼气氛 |

| 强权霸气 | 充满霸气豪气压迫感，有情有义，重视感觉，保护"自己人" |

**逆境·无序冲动**：

### 6.3 土行人的五个子维度

**核心能量特征**：承载与稳定

**维度逻辑关系**：
```
承载（信实可靠）→ 适应（耐力持久）→ 容纳（宽宏厚道）→ 运化（支持担当）→ 稳定（镇定稳重）
     ↓                    ↓                    ↓                    ↓                    ↓
  人际信任度          环境适应能力        情感包容广度        责任协作担当        自我调节能力
```

#### 维度一：承载（信实可靠 ↔ 呆板无趣）

**核心定义**：土的能量与大地有关，孕育生化、吐含万物。中国人推崇"厚德载物"，土行人尚信德。

**顺境·信实可靠**：

| 子维度 | 特质描述 |
|--------|---------|
| 注重他人 | 很少在公共场合批评人，敏锐意识到合适行为，彬彬有礼富有魅力 |
| 重视承诺 | 有责任心和通情达理，值得信赖，言语是庄严宣誓 |
| 善于补位 | 坦率真诚，可靠老实，易与人相处，顺应合作，善于补位 |
| 言行一致 | 言行一致，为人处世赢得尊重，对公司忠诚度高 |
| 信任他人 | 按人和事物表象认识，过于信任他人，容易上当受骗 |

**逆境·呆板无趣**：

| 子维度 | 特质描述 |
|--------|---------|
| 不善言辞 | 通过行为而非言辞表达深沉情感，谦虚缄默，容易被误解 |
| 患得患失 | 决定前深思熟虑仔细分析利弊得失，总是患得患失 |
| 漠不关心 | 性格内向不怎么与人交流，对不喜欢事物漠不关心 |
| 思想简单 | 思想简单，与人无争，反应慢，理解东西不灵通 |
| 单调乏味 | 爱好不多，生活单调，没有幽默感 |

#### 维度二：适应（耐力持久 ↔ 迟钝缓慢）

**核心定义**：大地土的适应性很高，天要下雨打雷都得接着，土行人适应力很强。

**顺境·耐力持久**：

| 子维度 | 特质描述 |
|--------|---------|
| 坚韧不拔 | 斟酌现实条件后决定取舍，紧要关头保持镇静，信念坚定意志顽强 |
| 重视和谐 | 需要最基本信任理解，对冲突和分歧很敏感，重视和谐 |
| 淡泊寡欲 | 有耐心易相处，很少支配控制别人，客观实事求是接受他人行为 |
| 沉着稳定 | 不断为安全感及稳定努力，应对环境变化沉着冷静 |
| 平缓松弛 | 随和，对生活不提出太多要求，喜欢简单爱好 |

**逆境·迟钝缓慢**：

| 子维度 | 特质描述 |
|--------|---------|
| 迟缓低效 | 很难对自己发展成就做出有效行动，用拖延方法被动对付 |
| 漫条斯理 | 不慌不忙，为小事焦虑，做最坏打算，寻找朋友速度慢 |
| 举棋不定 | 做事情有很多顾忌，犹豫不决 |
| 固执己见 | 顽固坚持自己意见，不肯改变 |
| 慢慢吞吞 | 很难说"不"，虽然点头答应但不一定会去做 |

#### 维度三：容纳（宽宏厚道 ↔ 沉闷埋怨）

**核心定义**：土以含散持实为体，兼容并收，土行人结识三教九流，亲近大众。

**顺境·宽宏厚道**：

| 子维度 | 特质描述 |
|--------|---------|
| 与世无争 | 对自我和他人都能容忍接受，不会强加愿望于他人 |
| 宅心仁厚 | 时时做出奉献，心地善良，宅心仁厚，慷慨热情 |
| 豁达大度 | 胸襟开阔，大度宽宏开通，能容人 |
| 凝聚力强 | 非权利性影响力突出，引导不同技能个性的人向共同目标努力 |
| 有求必应 | 处理焦虑方式是寻找朋友或同盟，表现得友善有求必应 |

**逆境·沉闷埋怨**：

| 子维度 | 特质描述 |
|--------|---------|
| 怨天尤人 | 遇到挫折或出问题，一味报怨天责怪别人 |
| 牢骚满腹 | 总有一肚子委屈，心中充满不满情绪 |
| 缺乏情感 | 缺乏情感，很难获取自保所需东西，用吃喝压抑愤怒 |
| 不闻不问 | 对人家说的不听，也不主动去问，表现出不关心 |
| 死气沉沉 | 对外界事物反应迟钝，精神消沉不振作 |

#### 维度四：运化（支持担当 ↔ 固执不化）

**核心定义**：土的能量在于运化，顺境时支持担当，逆境时固执不化。

**顺境·支持担当**：

| 子维度 | 特质描述 |
|--------|---------|
| 自我控制 | 能抵挡冲动和渴望，拒绝诱惑，对挫折有容忍力 |
| 善于协调 | 座右铭"有控制的协商"，只要在控制范围内就好商量 |
| 兼顾整合 | 能够整合各种人，同时兼顾人和目标两个方面，待人相对公平 |
| 当仁不让 | 碰到应该做的好事积极主动去做，不推托不谦让 |
| 任劳任怨 | 做事不辞劳苦，虽劳苦但没有怨言，能忍受别人怨言 |

**逆境·固执不化**：

| 子维度 | 特质描述 |
|--------|---------|
| 墨守成规 | 无条件接受社会权威性见解，不愿探求新境界，激烈反对新思想 |
| 传统保守 | 具有传统价值观，十分保守 |
| 顽固不化 | 坚持自己观点，拒不改变 |
| 自行其是 | 按照自己想法去做，不接受别人意见 |
| 推三阻四 | 以各种借口推托，不去办理交给的事情 |

#### 维度五：稳定（镇定稳重 ↔ 拖拉磨蹭）

**核心定义**：土地给人最大感受是稳定，反映土行人情感调节过程和稳定的意志倾向。

**顺境·镇定稳重**：

| 子维度 | 特质描述 |
|--------|---------|
| 心态平静 | 心态平静放松，不容易感到害怕 |
| 稳定成熟 | 以沉着态度应付现实问题，行为稳定而成熟 |
| 平衡决策 | 判断时倾向于注意事实以事物实用意义，主观与客观取平衡 |
| 处事老练 | 处事老练，行为得体，能冷静分析一切 |
| 情绪稳定 | 冷静，不会高度情绪化，有很好的自控力 |

**逆境·拖拉磨蹭**：

| 子维度 | 特质描述 |
|--------|---------|
| 忽略自我 | 清楚了解别人需求，过度工作满足需求而忽略自我 |
| 易受影响 | 应付阻扰挫折容易受环境支配，心神动摇不定 |
| 战战兢兢 | 时常感觉疲乏无法彻底摆脱，每日生活战战兢兢 |
| 漫无目的 | 很难理解复杂环境事务，做事情经常漫无目的 |
| 往坏处想 | 总喜欢往负面悲观严重方面幻想揣测，时常处于不安焦虑 |

### 6.4 金行人的五个子维度

**核心能量特征**：坚固与变革

**维度逻辑关系**：
```
坚固（刚毅坚强）→ 收敛（含蓄持重）→ 锋利（仗义果断）→ 光洁（权威出众）→ 变革（目标计划）
     ↓                    ↓                    ↓                    ↓                    ↓
  意志品质           行为收敛性          人际立场行动        权威认知追求        目标执行层面
```

#### 维度一：坚固（刚毅坚强 ↔ 好胜计较）

**核心定义**：金行能量在五行中最坚固，金属具有顺从人意、改变外形、制成器皿的特性。

**顺境·刚毅坚强**：

| 子维度 | 特质描述 |
|--------|---------|
| 非常务实 | 组织能力强，非常务实，把主意转化成实际行动并具体实施 |
| 持之以恒 | 持之以恒的毅力，注重细节力求完美，能长期专注同一目标 |
| 尽职尽责 | 善于控制调节自身冲动，有责任感可信赖有礼貌有组织 |
| 坚持不懈 | 依照固定规则生活，坚持不懈值得依赖，维护现存制度 |
| 不容犯错 | 不容忍自己或别人犯错，内心深处有衡量是非对错的标尺 |

**逆境·好胜计较**：

| 子维度 | 特质描述 |
|--------|---------|
| 保守坚固 | 尊重传统观念与行为标准，不愿体验新观念开放性 |
| 冷酷无情 | 有攻击性，为达到目的伤害别人，不能容忍悠闲爱空想的人 |
| 谨慎挑刺 | 冷静，每做一件事都要谨慎思考判断，喜欢挑刺找毛病 |
| 情感迟钝 | 无法理解别人需求和情感理由，显得迟钝缺乏耐心麻木 |
| 强加于人 | 希望别人也有条理，把自己的规则强加于人，被认为独裁 |

#### 维度二：收敛（含蓄持重 ↔ 小气狭隘）

**核心定义**：金行人象秋天盼望丰收，对胜利果实的渴望化作对自己别人的束缚。

**顺境·含蓄持重**：

| 子维度 | 特质描述 |
|--------|---------|
| 深思熟虑 | 实干，具有坚定不移深思熟虑的思想，着手相信最好的方法 |
| 高度自律 | 情感内敛，能克制自己，言行一致，合理支配感情行动 |
| 工作缜密 | 工作缜密讲求实际，注重工作过程细致程度，有条理可靠 |
| 道德规范 | 对自己和别人采取严格道德规范 |
| 自我反省 | 自省心很强，害怕被良知与别人批评，常反省有无做错事 |

**逆境·小气狭隘**：

| 子维度 | 特质描述 |
|--------|---------|
| 严肃刻板 | 谨慎严肃有逻辑性，行动前仔细考虑后果 |
| 默守陈规 | 习惯于格式化，凡事都有规矩，很少更换工作 |
| 固定思维 | 习惯事务性现成工作，不喜欢思想挑战，感觉抽象思考是浪费时间 |
| 一毛不拔 | 对金钱比较在意，很吝啬 |
| 现实有余 | 现实，对所有事情都能找到最优方法实现，不为没必要事情浪费精力 |

#### 维度三：锋利（仗义果断 ↔ 尖酸刻薄）

**核心定义**：金属有棱角，金行人有立场，骨子里带着善良的杀伤力。

**顺境·仗义果断**：

| 子维度 | 特质描述 |
|--------|---------|
| 精明干练 | 有节奏出效率，常被认为是天生管理者，果断行为带来暂时满足 |
| 效率可靠 | 可靠性高效率及处理具体工作能力，根据组织需要完成工作 |
| 责任感强 | 对待工作非常认真，交给他的任务保质保量完成 |
| 黑白分明 | 世界观黑白分明，是非优劣分得很清楚，只有100分与0分 |
| 干脆利落 | 做事不拖泥带水，按照既定规划与程序办事 |

**逆境·尖酸刻薄**：

| 子维度 | 特质描述 |
|--------|---------|
| 人情淡薄 | 行动拘禁，对人对事缺乏感情，例行公事，认真可靠 |
| 媚俗势利 | 与别人交往有企图，掩饰自己，看机会场合，媚俗势利 |
| 自我中心 | 多少有点自我中心，不愿意卷入别人麻烦，帮助别人是负担 |
| 吹毛求疵 | 对人冷漠，落落寡合，喜欢吹毛求疵，不轻易放弃主见 |
| 冷酷无情 | 很难理解其他人情感需求，对情感反应隐蔽，冷酷无情 |

#### 维度四：光洁（权威出众 ↔ 自负嫉妒）

**核心定义**：顺境时权威出众，逆境时自负嫉妒。

**顺境·权威出众**：

| 子维度 | 特质描述 |
|--------|---------|
| 出人头地 | 崇拜权威树立权威模仿权威，渴望成为权威希望出人头地 |
| 严于律己 | 尽力完成工作任务，克服困难专家，廉洁奉公公私分明 |
| 控制约束 | 目的性强，高度自律，希望所有人善于管理自己 |
| 崇尚规则 | 细心周到有始有终，崇尚遵从社会化标准和外在强制性规则 |
| 事必躬亲 | 对工作要求很高，担心授权下属做不到期望结果 |

**逆境·自负嫉妒**：

| 子维度 | 特质描述 |
|--------|---------|
| 不择手段 | 缺乏较高目标和理想，没有绝对责任感，有时不择手段 |
| 妒功忌能 | 对水平相同的人产生很强嫉妒心，对水平低的人看不起 |
| 妒富愧贫 | 嫉妒比自己富裕的人，对自己不如别人富有感到惭愧 |
| 盛气凌人 | 表现出骄横气焰，以骄横气势压人 |
| 尊己卑人 | 对伴侣家庭亲近朋友有很高要求，身边人总达不到要求 |

#### 维度五：变革（目标计划 ↔ 焦虑紧张）

**核心定义**：金曰从革，有顺从和变革两方面含义，顺境时有目标有计划，逆境时焦虑紧张。

**顺境·目标计划**：

| 子维度 | 特质描述 |
|--------|---------|
| 目标明确 | 追求成功和卓越，被当作工作狂，不能不允许自己停歇 |
| 现实主义 | 更在意外在因素，斟酌现实条件后决定取舍，脚踏实地 |
| 计划性强 | 定下目标并能按时完成，有计划有节奏，目标责任清晰 |
| 执行到位 | 以结果为中心，天生行动派，把任务分解成可测量结果 |
| 深入思考 | 决策前进行非常仔细思考，做事细心周到有始有终 |

**逆境·焦虑紧张**：

| 子维度 | 特质描述 |
|--------|---------|
| 容易担忧 | 内在充满竞争，停下来就不知所措，情况不在控制中莫名担忧 |
| 缺乏灵活 | 缺乏灵活性，对未被证实的想法不感兴趣，容易阻碍变革 |
| 思前想后 | 批判性强，凡事找出问题，思前想后综合考虑各因素 |
| 压抑渴望 | 有很强动力满足自己，但自我压抑满足渴望，造成持续压力 |
| 教条主义 | 活在条条框框里面，不仅要求自己，也要求别人遵守规则 |

### 6.5 水行人的五个子维度

**核心能量特征**：滋润与沉潜

**维度逻辑关系**：
```
润下（滋润祥和）→ 通透（智慧亲和）→ 沉潜（外柔内刚）→ 静藏（善解人意）→ 养物（多才多艺）
     ↓                    ↓                    ↓                    ↓                    ↓
  人际关怀           智慧社交表现        内在韧性支撑        情感共情能力        能力意志层面
```

#### 维度一：润下（滋润祥和 ↔ 消极放任）

**核心定义**：水曰润下，指水的滋润寒凉、流动趋下性质。水的能量在五行中最无形、最柔软、最灵活。

**顺境·滋润祥和**：

| 子维度 | 特质描述 |
|--------|---------|
| 无我利他 | 主动关心别人幸福，被伤害九十九次第一百次还敢去爱 |
| 温顺从容 | 尊重服从他人，避免冲突场景，克制攻击性，言语温和 |
| 和平大使 | 寻求和平，在矛盾中让步，害怕面对困难和冲突 |
| 温文尔雅 | 温文尔雅性情温和，把自己同化到群体中，团队润滑剂 |
| 以和为贵 | 信守"以和为贵"，在意别人建议，在意自己行为对别人影响 |

**逆境·消极放任**：

| 子维度 | 特质描述 |
|--------|---------|
| 不受约束 | 水无形不受约束，感觉规矩是束缚，懒散缺乏远大抱负 |
| 沮丧悲观 | 觉得世道艰辛，人生不如意事常八九，自觉不容于人 |
| 推卸责任 | 把做决定看作冒风险，有时不愿意承担工作压力 |
| 回避冲突 | 高度重视和睦关系，往往回避冲突 |
| 自暴自弃 | 自己瞧不起自己，甘于落后或堕落，做一日和尚撞一天钟 |

#### 维度二：通透（智慧亲和 ↔ 圆滑世故）

**核心定义**：顺境时智慧亲和，逆境时圆滑世故。

**顺境·智慧亲和**：

| 子维度 | 特质描述 |
|--------|---------|
| 智慧淡泊 | 象冬天老人难得糊涂，心知肚明故不与人争 |
| 顾及他人 | 富有同情心，容易感受到别人悲伤，总在顾及别人感受 |
| 自我满足 | 处事老练，对一切事物看法能容忍，与人无争心满意足 |
| 温和圆通 | 温和圆通，支持朋友和同伴 |
| 和蔼可亲 | 和善态度温和，容易接近 |

**逆境·圆滑世故**：

| 子维度 | 特质描述 |
|--------|---------|
| 感情用事 | 对事物判断容易受情感和价值观影响，心肠软敏感易受感动 |
| 八面玲珑 | 处世圆滑，待人接物面面俱到 |
| 避免冲突 | 忽略自己需要，别人想要怎样就怎样，环境和谐更重要 |
| 摸棱两可 | 容易收藏意愿迎合别人，没有明确态度，没有自己主张 |
| 诱惑他人 | 拥有诱惑另一个人的需求，喜欢赢得吸引他们的人 |

#### 维度三：沉潜（外柔内刚 ↔ 忧虑多疑）

**核心定义**：水是深沉含蓄的，象冬天一样神秘，象老人一样深藏不露。

**顺境·外柔内刚**：

| 子维度 | 特质描述 |
|--------|---------|
| 不摆架子 | 谦逊不摆架子，不爱出风头，讲话和没讲过一样 |
| 人际认同 | 拥有良好人际关系，基本无摩擦，不善于公开交流 |
| 需要呵护 | 传达需要受到保护的需求，设法使自己表现更可爱 |
| 婉约细腻 | 喜爱生活中精美事物，温柔体贴情感细腻 |
| 外圆内方 | 表面看比较随和，内心严正 |

**逆境·忧虑多疑**：

| 子维度 | 特质描述 |
|--------|---------|
| 充满矛盾 | 既不能克制自己，又不能尊重礼俗，充满矛盾无法解决 |
| 优柔寡断 | 做事犹豫缺乏决断，做决定时果断性不够 |
| 缺乏规划 | 需要跟从架构和目标，没有架构觉得很混乱 |
| 掠夺对方 | 对渴望东西感到焦虑时会追赶对方，变得更激进过分要求 |
| 自我牺牲 | 期望别人照顾需求但不会直接开口，倾向于感觉被牺牲 |

#### 维度四：静藏（善解人意 ↔ 散漫无序）

**核心定义**：水是深静潜藏的，象月夜下寒潭一样。

**顺境·善解人意**：

| 子维度 | 特质描述 |
|--------|---------|
| 乐于助人 | 非常乐意满足他人要求，乐于接收别人意见 |
| 察颜观色 | 有强烈直觉，不需别人开口已知道需要什么，懂得如何令人喜欢 |
| 情感支持 | 帮助别人后喜欢获得赞美嘉赏回报，希望对方喜欢他倚赖他 |
| 成就他人 | 抱着"成就别人"心态助人，积极介绍新朋友予对方 |
| 无私忘我 | 真心帮助别人，把别人需求看得比自己重，往往忽视自己需求 |

**逆境·散漫无序**：

| 子维度 | 特质描述 |
|--------|---------|
| 爱心泛滥 | 以人为导向，缺爱所以泛爱，时刻关注对方情绪 |
| 依赖权威 | 依赖权威随群附和，放弃个人主见附和取得好感 |
| 半途而废 | 做事拖延经常半途而废，遇到困难容易退缩 |
| 杂乱无序 | 做事没有条理没有头绪 |
| 因爱成恨 | 最痛恨"烂泥扶不上壁"的人，一旦决绝后不会回头 |

#### 维度五：养物（多才多艺 ↔ 意志薄弱）

**核心定义**：顺境时多才多艺，逆境时意志薄弱。

**顺境·多才多艺**：

| 子维度 | 特质描述 |
|--------|---------|
| 扩大交际 | 非常渴望受到社交圈内人喜爱认可，享受介绍别人扩大交际网 |
| 适机善变 | 通过知名度认识名人寻找认可，操纵成为成功人士拥护者 |
| 兴趣多样 | 不断寻找新鲜超越普遍东西，排斥乏味，要求有激烈感受 |
| 聪颖灵活 | 能按团队单位需求自我调节，有"变色龙"本领 |
| 能说会道 | 善于用言辞表达，很会说话 |

**逆境·意志薄弱**：

| 子维度 | 特质描述 |
|--------|---------|
| 逆来顺受 | 以逆来顺受态度应付阻扰挫折，容易受环境支配情绪动摇不定 |
| 不思进取 | 压力下容易感到惊慌混乱无助，不思进取 |
| 迷失自我 | 每天陷入劳累和无休止劳作中，常常会迷失自己 |
| 迎合别人 | 温顺迎合别人，即使处在十全十美境地也有"事事不如人"之感 |
| 感情用事 | 对事物判断容易受情感因素影响 |
