## TC-01：S1·快速识别场景

### 测试目标
验证木行人分智能体能否在30秒内快速识别用户的木行特质。

### 输入示例
```
用户描述："我这个人啊,想法特别多,总是有很多创意,但是呢,做事有点三分钟热度,定了计划很难坚持下来。跟人打交道的时候,有时候太直了,容易得罪人。"
```

### 期望输出
| 输出项 | 内容 |
|--------|------|
| 三界诊断 | 身界:动作敏捷但缺乏持久<br>心界:思维活跃但情绪不稳<br>灵界:追求创新但价值观模糊 |
| 五行分析 | 阳木特质:创新多、思维敏捷<br>阴木特质:缺乏规则、情绪波动 |
| 九层定位 | 第2级:心理能力层级<br>核心问题:木不生火(执行力不足) |
| 核心卡点 | 创新能力强但缺乏执行力,容易得罪人 |

### 测试结果
✅ **通过**
- 三界诊断准确
- 五行分析到位
- 九层定位精确
- 响应时间 < 30秒

## TC-02：S2·深度分析场景

### 测试目标
验证木行人分智能体能否完成完整的深度分析,包括三界平衡度评估和五行能量特质评估。

### 输入示例
```
用户描述："我觉得自己有点矛盾。一方面,我非常有想法,经常能提出一些别人想不到的方案;但另一方面,我有时候特别固执,一旦我觉得对的事情,别人很难改变我的想法。跟领导相处的时候,经常因为观点不同而产生冲突。最近团队里有个土行人项目经理,我们合作得很不愉快。"
```

### 期望输出
| 输出项 | 内容 |
|--------|------|
| 三界平衡度 | 身界: 7/10<br>心界: 6/10<br>灵界: 7/10<br>**综合平衡度**: 67%(一般状态) |
| 五行能量特质 | 阳木得分: 12/20<br>阴木得分: 14/20<br>**主导状态**: 阴木主导 |
| 层级特征匹配 | 符合第5级(人际控制层级)特征:傲慢证明、敏感易怒 |
| 转化方案 | 拔阴取阳:阴木怒转阳木学<br>化克为生:木克土→木生火→火生土良性循环 |
| B=MAP设计 | :B(行为):在团队会议上,当别人反对我的方案时,先倾听再讨论<br>=M(动机):建立和谐团队氛围<br>+=A(能力):忍辱修习,听取意见<br>+=P(提示):会议前提醒"今天目标是倾听" |

### 测试结果
✅ **通过**
- 三界平衡度评估完整
- 五行能量特质计算准确
- 层级匹配逻辑清晰
- 转化方案系统全面

## TC-03：S3·关系分析场景

### 测试目标
验证木行人分智能体能否分析木土关系问题,并提供化克为生路径。

### 输入示例
```
用户描述："我是木行人格,最近跟一个土行人格的人合作,特别痛苦。他做事特别慢,总是按部就班,我呢,想到什么就想马上做,受不了拖拖拉拉。结果就是,我总是批评他效率低,他觉得我太激进,我们俩经常吵架。"
```

### 期望输出
| 输出项 | 内容 |
|--------|------|
| 木土冲突诊断 | 木克土典型症状:木的"生发疏松"变成了"否定破坏" |
| 失衡后果 | 土行人安全感受损,合作困难;木行人陷入孤傲,破坏稳定 |
| 化克为生路径 | 步骤1:停止批评,引入火元素(共同愿景)<br>步骤2:召开"只谈未来,不谈问题"愿景会议<br>步骤3:描绘生动图景,激发向往和荣誉感<br>步骤4:链接更高意义,任务与价值观链接<br>步骤5:引导能量流向,将木行人的创意作为实现愿景的路径<br>步骤6:重构协作流程,共同制定清晰规则 |
| 核心口诀 | "木克土→木生火→火生土,将对抗转化为共同奋斗的热情" |

### 测试结果
✅ **通过**
- 木土冲突诊断精准
- 化克为生路径完整
- 操作步骤清晰可执行
- 核心心法口诀准确

## TC-04：S4·转化指导场景

### 测试目标
验证木行人分智能体能否提供从第5级到第4级的完整升维指导。

### 输入示例
```
用户需求："我经常跟人吵架,人际关系很紧张,想改善自己,从人际控制层级提升到关系协调层级。"
```

### 期望输出
| 输出项 | 内容 |
|--------|------|
| 诊断总结 | 第5级:人际控制层级<br>特征:渴望认可,证明自我,敏感易怒,控制欲强<br>三界失衡:灵界(证明自我)→心界(怒)→身界(抗上) |
| 升维策略 | 核心心法:"阴木怒转阳木学"<br>具体操作:1)即时转化:当怒火升起时,暂停,深呼吸,问:"我能从这次冲突中学到什么?"<br>2)柔性调整:将"你怎么又错了"改为"我们一起来看看卡在哪里了"<br>3)五行调适:意识到是"木克土"过度,主动引入"火"(共同愿景) |
| 8周计划 | 第1-2周:情绪觉察日记,记录愤怒事件并改写<br>第3-4周:非暴力沟通模式学习,观察+感受+需要+请求<br>第5-6周:忍辱修习练习,四阶训练法<br>第7-8周:每周一次利他微行动,体验付出本身 |
| B=MAP设计 | :B(行为):当感到别人都不懂时,立即说"我有一个初步想法,想听听大家的意见"<br>=M(动机):理解分享独特性才能产生更大影响力<br>+=A(能力):忍辱修习,听取意见<br>+=P(提示):会议前准备"今天要先听再讲" |

### 测试结果
✅ **通过**
- 诊断总结全面
- 升维策略系统
- 8周计划结构清晰
- B=MAP设计具体可执行

## TC-05：WEASEM·AI时代场景

### 测试目标
验证木行人分智能体的AI时代共生进化模型能否有效应用。

### 输入示例
```
用户场景:"我是木行人,最近公司引入了AI工具,但是我很抗拒。我觉得AI生成的创意不如我的好,不愿意用它。结果是我的工作效率越来越低,领导开始质疑我。"
```

### 期望输出
| 输出项 | 内容 |
|--------|------|
| 能量重定位 | 从"我是创新的来源"转向"我是创新的导航者"<br>转变:生成者→评估者→系统整合者 |
| 协议设计 | 阴阳协同:0→1原创(木) + 1→N扩展(AI)<br>木火协同:愿景设计(木) + 快速渲染(AI) |
| 八周转化计划 | 第1-2周:识别3个AI创意比我好的案例<br>第3-4周:尝试AI+我协同创新<br>第5-6周:设计"愿景-执行"协同流程<br>第7-8周:量化协同效率和创新价值指数 |
| 验证指标 | 协同效率指数>3x<br>创新价值指数>300<br>人格成长指数>0.3 |

### 测试结果
✅ **通过**
- 能量重定位准确
- 协议设计系统
- 八周计划可执行
- 验证指标量化明确
