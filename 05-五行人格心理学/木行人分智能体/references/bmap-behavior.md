## 📖 核心定义

B=MAP是木行人分智能体的标准行为设计工具,基于福格行为模型（B=Behavior=Motivation+Ability+Prompt=提示）。

### What（是什么）

B=MAP框架将行为改变分解为三个核心要素,通过结构化设计,让微小的行为改变能够系统性地转化为长期习惯,最终实现人格升华。

### Why（为什么）

对于木行人而言：
1. **动机（M）**: 木行人容易被"证明自我正确"（阴木）驱动,B=MAP引导其转向"帮助成长"（阳木）
2. **能力（A）**: 木行人创新能力强但执行力不足（木不生火）,B=MAP通过阶段性分解强化执行力
3. **提示（P）**: 木行人缺乏规则意识（木过旺克土）,B=MAP通过场景化提示建立规范

## 🎯 木行人专属设计原则

| 维度 | 阳木（健康） | 阴木（失衡） | 转化策略 |
|------|-------------|-------------|---------|
| **动机M** | 仁德驱动,利他成长 | 傲慢驱动,证明自我 | 从"证明正确"转向"帮助成长" |
| **能力A** | 规划创新,正直担当 | 固执对抗,钻牛角尖 | 从"硬对抗"转向"软学习" |
| **提示P** | 明确愿景,阶段性目标 | 模糊冲动,缺乏计划 | 设定清晰提示：场景+时机+具体行动 |

## 📋 设计模板

### 案例1：改善人际关系（木克土转化）

```
:B（行为）：在团队会议中,当别人反对我的方案时,先倾听再讨论
  = M（动机）：建立和谐团队氛围,而非证明自己正确
  + A（能力）：忍辱修习,听取意见,整合优化
  + P（提示）：会议前提醒自己"今天的目标是倾听和整合"
```

### 案例2：提升执行力（木生火）

```
:B（行为）：每周设定3个小目标,并在24小时内启动第一个行动
  = M（动机）：将创新转化为实际成果
  + A（能力）：阶段性拆解,快速启动
  + P（提示）：晨起问自己"今天的第一步是什么？"
```

### 案例3：创新落地（迎金修木）

```
:B（行为）：当新想法产生时,先做最小可行性验证（MVP）,不追求完美
  = M（动机）：让创意真正产生价值
  + A（能力）：小步迭代,快速验证
  + P（提示）：记录本触发点"验证优先于完美"
```
