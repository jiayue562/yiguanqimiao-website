5.2 安装步骤

第一步：安装核心SKILL

打开 WorkBuddy，进入 Skills 市场，搜索并安装：

  1. 龙心OS（核心调度中枢）
  2. 五行人格心理学OS（人格智能体系）
  3. 人机共生OS（关系定义层）
  4. 象思维、知识学习、五色光思维、人机协同五象限、知行合一（五大引擎）

第二步：配置身份锚点

创建 ~/.workbuddy/IDENTITY.md，定义：
- 你的姓名和五行人格类型
- 与AI的关系定位
- 价值观和信仰体系

第三步：连接知识库

  1. 安装 obsidian SKILL，连接 Obsidian 知识库
  2. 安装 ima-skills，配置 IMA API 凭证

第四步：启动龙心OS
