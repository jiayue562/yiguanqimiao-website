## 🔗 双向链接使用

### 基础操作

**创建链接**：
- 输入 `[[` 触发链接选择器
- 输入关键词搜索目标文档
- 按 Enter 确认

**跳转链接**：
- Ctrl/Cmd + 点击链接文字
- 或使用快捷键 Ctrl/Cmd + Enter

**查看反向链接**：
- 打开任意文档
- 查看底部"反向链接"区域
- 了解哪些文档引用了当前文档

### 链接命名规范

```markdown
[[文档名]]                    # 简单链接
[[路径/文档名|显示文字]]      # 带显示文字的链接
[[文档名#标题]]              # 链接到文档中的特定标题
[[文档名#^块ID]]             # 链接到文档中的特定段落
```

## 🎨 五色光思维使用

### 场景：需要多角度分析问题

1. **进入白光模式**（客观事实）
   - 收集所有相关信息
   - 不问"为什么"，只问"是什么"

2. **进入红光模式**（直觉感受）
   - 表达内心真实感受
   - 不压抑情绪

3. **进入黄光模式**（积极价值）
   - 寻找利益和价值点
   - 保持乐观视角

4. **进入绿光模式**（创新方案）
   - 头脑风暴，追求数量
   - 不评判，先发散

5. **进入蓝光模式**（风险控制）
   - 识别风险和漏洞
   - 制定应对策略

**详细指南**：[[01-核心体系/龙心OS/五大引擎/🌈 五色光思维]]

## 🔄 知行合一沉淀

### 每次深度对话后自动执行

**触发条件**：
- 完成深度学习
- 完成创新突破
- 完成重大决策
- 产生原象级洞察

**三阶段流程**：

1. **表示空间** - 记录完整经验
   ```
   - 使用了哪些引擎？
   - 最关键的转折点是？
   - 产生了什么新认知？
   ```

2. **压缩** - 提炼核心洞察
   ```
   - 一句话核心洞察（不超过30字）
   - 象征符号（1个词或意象）
   - 与已有认知的连接
   ```

3. **泛化** - 扩展应用场景
   ```
   - 场景1：可以应用到哪？
   - 场景2：第二个应用场景
   - 场景3：第三个应用场景
   - 系统进化建议
   ```

**沉淀路径**：[[04-个人成长/知行合一沉淀]]

## 🆘 常见问题

### Q1: 如何快速找到需要的文档？

**A**: 使用以下方法：
1. 查看 [[知识库总索引]] 的导航表格
2. 使用 Ctrl/Cmd + O 快速打开
3. 使用图谱视图浏览知识网络
4. 按标签筛选

### Q2: 如何创建新的知识文档？

**A**: 
1. 确定文档所属层级（00-05）
2. 使用标准模板（见下方）
3. 添加YAML frontmatter
4. 建立双向链接
5. 更新相关索引

### Q3: 如何确保知识不丢失？

**A**: 
- 三向同步机制：Obsidian（主）+ WorkBuddy（工作记忆）+ IMA（移动）
- 定期备份：[[知识库总索引]]中有备份说明
- 使用Git版本控制（可选）

### Q4: 如何与龙龟神将协作？
