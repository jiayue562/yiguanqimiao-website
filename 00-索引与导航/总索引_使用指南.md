## 🔗 双向链接使用

### 创建链接

在笔记中输入：

```markdown
[[文件名称]]
```

### 链接类型

| 类型 | 语法 | 示例 |
|------|------|------|
| **基础链接** | `[[文件]]` | `[[象思维总论]]` |
| **带标题** | `[[文件#标题]]` | `[[五行#生克关系]]` |
| **带别名** | `[[文件\|别名]]` | `[[五行\|五行情]]` |
| **嵌入内容** | `![[文件]]` | `![[金句]]` |

### 查看链接

- **出链**: 当前文件链接到其他文件
- **入链**: 其他文件链接到当前文件
- **本地图谱**: 可视化查看关联

## 📝 创建笔记

### 使用模板

1. 复制 `00-总索引与导航/模板/标准文档模板.md`
2. 粘贴到目标目录
3. 填写内容

### 文档结构

```markdown
# 标题

> 💡 核心定义
> [一句话定义]

## 🔍 核心定义
[详细阐述]

## 📖 详细内容
[分点详述]

## 🔗 关联文件
- [[上游文件]]
- [[下游文件]]
- [[平行文件]]

## 💎 核心金句
> [金句]

## 🏷️ 标签
#标签 1 #标签 2
```

## 🤖 自动化工具

### 验证链接

```bash
cd D:\以以观其妙书院知识库\以观其妙书院
python scripts/verify_links.py
```

**功能**:
- 检查断裂链接
- 识别孤立文件
- 生成验证报告

### 增量备份

```bash
python scripts/backup_incremental.py
```

**功能**:
- 只备份变更文件
- 保存备份历史
- 支持版本回滚

### 智能分类

```bash
python scripts/classify_files.py
```

**功能**:
- 按扩展名分类
- 按内容关键词分类
- 生成分类报告

## 📊 与 AI 协作

### 沟通原则

1. **先调取知识库**: AI 会先查看 Obsidian 相关内容
2. **基于已有知识**: 在现有基础上深化
3. **持续沉淀**: 对话内容存入知识库

### 对话存储

重要对话会自动存储到：
- `Notes/Conversation/`
- 按日期和主题分类
- 添加双向链接到相关知识
