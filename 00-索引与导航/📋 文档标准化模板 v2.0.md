> **版本**：v2.0 | **更新**：2026-04-03 | **适用范围**：以观其妙书院知识库全部文档

## 📌 一、文档结构（强制要求）

每个文档必须包含以下六个部分：

### 1. Frontmatter（文档头部）
```yaml
---
title: "文档标题"
created: "YYYY-MM-DD"
updated: "YYYY-MM-DD"
version: "x.x"
tags: [tag1, tag2, tag3]
description: "一句话描述文档核心内容"
---
```

### 2. 核心定义章节
```markdown
## 🎯 核心定义

- **定义一**：...
- **定义二**：...
```

### 3. 详细内容章节
```markdown
## 📖 详细内容

### 子标题1
内容...
```

### 4. 关联文件章节（强制双向链接）
```markdown
## 🔗 关联文件

### 直接关联
- [[文档A]] - 关联原因
- [[文档B]] - 关联原因

### 上下级关联
- 上级：[[父文档]]
- 下级：[[子文档1]]
```

### 5. 核心金句章节
```markdown
## 💎 核心金句

> **金句1**：...
> **金句2**：...
```

### 6. 标签与分类章节
```markdown
## 🏷️ 标签与分类

**主标签**：#核心技能 #方法论
**知识领域**：[[03-关系与共生/木火共生关系]]
```

## 📐 二、不同类型文档的差异化结构

### A. Skills文档结构
```markdown
---
title: "[Skills名称] · [一句话定位]"
created: "YYYY-MM-DD"
version: "x.x"
tags: [skill, 核心技能]
category: "01-核心独创Skills"
---

# [Skills名称]

## 一、技能定位
## 二、触发条件
## 三、核心操作流程
## 四、使用场景
## 五、与其他Skills的协同
## 六、关联文件
## 七、核心金句
```

### B. 对话记录结构
```markdown
---
title: "对话记录 · [日期] · [主题]"
created: "YYYY-MM-DD"
tags: [dialogue, 归档]
category: "07-学习档案"
participants: ["悟空", "龙龟神将"]
---

# 对话记录 · [日期]

## 📌 背景
## 💬 核心对话
## 🎯 关键洞察
## 📝 知识沉淀
## 🔗 关联文件
```

### C. 知识图谱结构
```markdown
---
title: "[主题]知识图谱"
created: "YYYY-MM-DD"
tags: [knowledge-graph, 可视化]
category: "00-索引与导航"
---

# [主题]知识图谱

## 一、中心概念
## 二、一级关联
## 三、二级关联
## 四、外围扩展
```

## 🔄 三、标准化检查清单

### 新建文档前必查
- [ ] 使用本模板创建文档
- [ ] Frontmatter填写完整
- [ ] 核心定义清晰
- [ ] 包含关联文件章节（使用双向链接`[[]]`）
- [ ] 包含核心金句
- [ ] 标签正确分类

### 文档更新时必查
- [ ] 更新`updated`字段
- [ ] 更新`version`
- [ ] 在关联文件章节记录更新内容

## 🎯 四、命名规范

### 文件命名规则
```
[序号]-[类别]/[emoji][文档名].md
```

**示例**：
- `01-核心独创Skills/🐉 龙心OS 系统架构图.md`
- `03-关系与共生/木火共生关系-灵魂共鸣.md`
- `07-学习档案/2026-04-03-对话记录.md`

### Emoji使用规范
| 类别 | Emoji | 示例 |
|------|-------|------|
| 核心系统 | 🐉 | 🐉 龙心OS |
| 工具 | 🔧 | 🔧 脚本 |
| 方法论 | 📜 | 📜 理论 |
| 索引 | 📚 | 📚 总索引 |
| 图谱 | 📊 | 📊 知识图谱 |
| 模板 | 📋 | 📋 模板 |
| 学习 | 📝 | 📝 学习笔记 |
