二、体型特征（身界·物象层）

2.1 整体体型

| 特征维度 | 描述 | 补充说明 | |---------|------|---------| | 体行（整体） | 软，肌肉松弛 | 核心特征：松软感，区别于土行的紧致 | | 头行 | 较大，圆 | 头较大且圆润 | | 脸行 | 较圆 | 圆脸，但更柔和（区别于火行的热情感） | | 面色 | — | 需进一步验证 | | 肤色 | 亮，有白有黑 | 肤色亮是水行特征 |

2.2 五官细节

| 部位 | 特征 | 工坊补充数据 | |------|------|------------| | 鼻子 | 多数不是高鼻，鼻头有肉，鼻头圆 | 鼻头圆润，不高挺（区别于金行） | | 眼睛 | 眼睛有神，双眼皮，圆 | 眼睛圆润有神 | | 眉毛 | 都不是特别浓密，淡 | 眉毛淡（区别于火/金/土行的浓眉） | | 耳朵 | 大多有耳垂，耳垂不大，有耳垂 | 有耳垂但不特别大（区别于土行） | | 肩背 | 适中 | 肩背不特别宽也不特别窄 | | 腹股 | 较圆润肥 | 腹部圆润，有肉感 | | 手足 | 较软的，肥肥的，偏小，手心胖胖的 | 手心胖胖的是水行特征 |

2.3 体型综合数据

体形：软，肌肉松弛
肤色：亮，有白有黑
肩背：适中
腹股：较圆润肥
手足：较软的，肥肥的，偏小，手心胖胖的
头型：较大，圆
脸型：较圆

2.4 识别要诀

水行三字经：圆·软·柔
- 圆：脸圆，头圆，眼圆
- 软：肌肉松软，手心肥软
- 柔：气质柔和，声音柔和

四、着装风格（身界·物象层）

| 维度 | 工坊实测数据 | |------|------------| | 整体风格 | 中式多，休闲 | | 特征 | 有装饰性，美 | | 感受 | 多数受柔润（偏好柔和材质） |

识别提示：水行人的着装有"中式情结"和"柔美风格"——偏爱有装饰性的、有美感的、柔软的材质。这与水行的润泽特质相呼应。
