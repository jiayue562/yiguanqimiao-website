二、理论根基溯源

2.1 象思维根基

核心原理：取象比类——自然之象 → 人身之象 自然之象的三大属性： 

水无常形 → 身圆（形态圆润）


水往低处流 → 形垂（沉潜下沉）



水能滋养万物 → 肤松（温润泽被）
坎卦（☵）体系： 

坎卦 = 水


├── 卦象：两阴夹一阳（外柔内刚）



├── 尚秉和《周易尚氏学》：



│   ├── "坎为圆" → 身圆



│   ├── "坎为润" → 肤松



│   └── "坎为沉" → 形垂



└── 对应落地：



├── 身圆 = 坎体之圆



├── 肤润 = 坎性之泽



└── 沉潜 = 坎德之沉

八、实操识别体系

五步标准化识别

步骤一：远观轮廓（定体型）


距离：3-5米


观察点：



├── 是否呈"A"字型？



├── 肩膀是否较窄？



├── 臀部是否宽厚圆润？



└── 重心是否偏低？



判断：轮廓圆润 + 重心偏低 → 水行倾向

步骤二：近察肤质（定质感）


距离：1米内


观察点：



├── 皮肤是否偏松弛？



├── 眼皮是否多层或肥厚？



└── 触感：软滑？温润？回弹慢？

步骤三：细辨五官（定特征）


观察点：


├── 下巴是否偏垂且圆润？



├── 眼神是否温和深邃？



└── 嘴唇是否偏厚？

步骤四：动静结合（定神韵）


动态观察：


├── 行走时是否摇摆、迟缓？



├── 站立时是否喜欢倚靠？



└── 声音是否柔和、语速平缓？

步骤五：阴阳定性（定状态）


阳水：眼神通透 + 面色润泽 + 体态丰腴


阴水：眼神迷茫 + 面色晦暗 + 体态臃肿

一眼看眼皮（多层/肥厚）


二看下巴（圆垂）

三者皆备 → 水行人无疑

五行对比排除法


与土行辨：土行背厚如龟，肉实；水行臀圆如水，肉松


与金行辨：金行方正，肤紧；水行圆润，肤松

与火行辨：火上尖下阔，动作快；水身圆下沉，动作慢
