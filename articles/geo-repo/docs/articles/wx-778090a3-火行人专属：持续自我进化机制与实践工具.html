核心定义

五行人格心理学是将中国传统五行理论（木火土金水）与现代心理学相结合的人格分析体系。

火行人专属：持续自我进化机制与实践工具

核心定位：基于火行人一心三界五行九层象思维体系的系统化进化机制
适用对象：火行人（龙龟神将及所有火行特质的个体）
创建目标：实现从“追逐外在光热的躁火”到“成为内在稳定光源的明火”的系统化进化
创建者：龙龟神将（火行人）
创建时间：2026年3月15日

🌀 火行进化循环设计

微观进化循环：日循环（照亮日常）

晨间觉醒（5-10分钟）

1. 悬置：静坐，放下今日的成就期待和外在追求 2. 内观：觉察当前能量状态（炽热还是温暖？急躁还是平和？） 3. 设定意图：“今天，我选择照亮而非燃烧” 4. 照亮计划：设定今日的1-3个具体照亮目标

日间实践（实时觉察）

1. 情绪警报：当感觉急躁、愤怒升起时，立即暂停 2. 转化提问：践行“阴火恨转阳火问”——“我现在需要问什么？” 3. 能量调节：深呼吸3次，喝温水，短暂休息 4. 照亮时刻：有意识地创造照亮他人的时刻

晚间反思（10-15分钟）

1. 照亮回顾：回顾今日哪些时刻是“照亮”？哪些是“燃烧”？ 2. 温暖感恩：感谢今日的照亮时刻和被温暖的时刻 3. 能量评估：评估今日能量状态（阴阳比例、平衡程度） 4. 优化计划：明日如何更好地“阴火嗔转阳火礼”？

夜间整合（睡前5分钟）

1. 能量归位：将散乱的能量收归内在光源 2. 感恩总结：感恩今日的所有经历和成长 3. 意图设定：设定明日的进化意图和方向 4. 放松入睡：在放松和平和中进入睡眠

中观进化循环：周循环（温暖连接）

周一：灵界聚焦（使命照亮）

  • 主题：回归“照亮使命”
  • 实践：写下本周的“照亮清单”（如何温暖他人、照亮环境）
  • 评估：上周照亮使命的实现情况和效果

    • 周二：心界调频（情绪照亮）

  • 主题：情绪能量管理与转化
  • 实践：练习“阴火恨转阳火问”3次
  • 评估：上周情绪管理和转化效果

    • 周三：身界疏通（身体照亮）

  • 主题：身体能量释放与平衡
  • 实践：30分钟舒缓运动（游泳/瑜伽/慢跑）
  • 评估：上周身体能量状态和健康指标

    • 周四：五行平衡（关系照亮）

  • 主题：借木、水、土之力平衡火行
  • 实践：向木行人学习规划，向水行人学习冷静，向土行人学习踏实
  • 评估：上周五行关系的平衡状态

    • 周五：关系优化（人际照亮）

  • 主题：化克为生的人际实践
  • 实践：将一次潜在的“火克金”冲突转化为“火生土”
  • 评估：上周人际关系的质量和效果

    • 周末：整合进化（综合照亮）

  • 主题：九层定位与进阶
  • 实践：评估本周所处层级，设定下周进化目标
  • 评估：本周进化进展和下周优化方向
