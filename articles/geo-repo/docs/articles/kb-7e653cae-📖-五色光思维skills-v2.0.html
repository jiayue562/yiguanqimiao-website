🎯 核心定义

五色光思维是悟空独创的一套结构化、同频化的集体思考与决策系统。 核心痛点解决： 

思维混乱 × 情绪对抗 × 智慧耗散 → 五色分治，同频共振
核心问题转变：
  • ❌ 传统思维："这个方案怎么样？"（所有维度同时涌现，必然混乱）
  • ✅ 五色光思维："现在是白光时间——关于这个方案，我们有哪些确定的事实和数据？"

    • 🔄 传统思维 vs 五色光思维

    传统冲突思维特征

  • 判断和争论：坚持"我对，你错"，从自己立场看对方问题
  • 混乱：试图在同一时间解决很多问题，思考时思维纠结混乱
  • 情绪化对抗：观点与人身攻击混淆，情绪干扰理性判断

    • 五色光思维特征

  • 同频思考：让每个人在同一时间使用相同的思维模式
  • 转化思维：通过转换思维角度来重新构建新概念
  • 智慧凝聚：使每个人的经验和智慧在同一方向得到充分运用

    • 🔗 三体一心整合框架（悟空独创）

    五色光思维与象思维的关系： 
    
信息体层面（原象·道）  ←→  象思维（整体直观·0→1·悟性）


    能量体层面（意象·场）  ←→  五色光场（五色能量态势·动态流动）
    


    物质体层面（物象·用）  ←→  五色光操作工具（白红黄绿蓝具体思维方法）
    


    三体是一心（纯粹的觉知）的三个显现层面

    🟥 红光思维——直觉与感受

    核心定义：代表我们的直觉和预感。具有形成观点的功能。 红光三问：
  • 我现在的感受是什么？
  • 我的直觉告诉我什么？
  • 我的本能反应是什么？
    • 红光使用原则：

    1. 正确认识和运用直觉与情绪

    2. 不要证明或解释自己的感觉

    3. 认可预感，但非凭预感作决定

    4. 避免争辩

    5. 须在30秒以内作出回答，避免过度使用红光

    何时使用红光：
  • 气忿沉闷
  • 新的创意或计划
  • 不好的消息
  • 会议前后可用红光调动情绪
  • 当没有渠道表达时

    • 🟨 黄光思维——正面与价值

    核心定义：代表正面的思想，具有识别事物积极因素的功能。 黄光思维五问：

    1. 有什么利益？

    2. 有哪些积极因素？

    3. 这样做会带来哪些正面的影响？这样可行吗？

    4. 存在哪些有价值的地方？

    如何使用黄光思维：

    1. 积极的思考不是在看到建议中有利的一面以后才开始的，而是在还没有看到之前就采取积极的态度

    2. 瞥见未来可能出现的利益和价值

    3. 想象到最好的前景和最大的利益

    4. 即使遥不可及，我们也要把他考虑在内（"如果"）

    5. 黄光也可以是正面的判断，但需要做解释

    概念提炼四步法：

    1. 提出焦点问题

    2. 提出想法

    3. 提取概念

    4. 想出其它方法

    🎯 主持人——思维过程的控制与组织

    核心定义：代表着思维过程的控制与组织，它管理其他思考光的使用。 主持人责任：
  • 控制、组织、指挥和协调整个思考过程
  • 使思考过程程序化、清晰化、条理化
  • 技术要点：问正确的问题和定义问题
  • 在思考过程中不断做摘要、概括、总结
  • 维持思考的秩序，使思考集中到一个方向
    • 主持人三阶段工作：

    1. 初始阶段：提出主题、确定焦点、设定思维序列和时间

    2. 中间阶段：总结目前的内容、核对步骤、纠偏深度沟通

    3. 结尾阶段：总结/结束和决定下一步

    常见应用场景序列

    解决问题的思维序列

    1. 主持人/自己：定义问题

    2. 白光：考虑已有的信息

    3. 绿光：提出可能的解决方案

    4. 黄光：检查每种方案的可行性

    5. 蓝光：评估每种方案的缺点

    6. 白光：检查解决方案和信息是否匹配

    7. 主持人/自己：选择最终解决方案

    是否问题决策（如是否买房）

    1. 自己：想要买100平米的房子

    2. 白光：看房，比较价格、位置、配套等信息

    3. 黄光：PK看过的楼盘优势

    4. 蓝光：PK看过的楼盘劣势

    5. 绿光：楼盘的劣势有无其他弥补的办法

    6. 红光：做最优的选择

    7. 自己：交定金，筹备首付

    绩效面谈序列

    1. 自己：确定面谈的目标和框架

    2. 红光：简短描述对整体表现的感受

    3. 白光：描述设定目标的完成状况

    4. 黄光：肯定哪些工作做得不错

    5. 蓝光：存在哪些不够好的问题、缺陷

    6. 绿光：针对这些问题的改善方向、方法

    7. 红光：强化改进的信心，给予鼓励

    8. 自己：双方达成共识，拟定下一步工作计划

    向上汇报工作序列

    1. 自己：陈述焦点问题

    2. 白光：介绍目前的情况、数据、事实

    3. 蓝光：存在的问题、不足、风险

    4. 绿光：分析问题的原因并提出初步的解决方法

    5. 黄光：解决方案带来的价值及利益

    6. 自己：制定行动计划
