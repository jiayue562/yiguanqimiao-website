一、核心对应

三、心理应激 × 五行

应激源偏好：木行=创新受阻、火行=认同缺乏、土行=人际冲突、金行=规则破坏、水行=安全威胁 应对策略（Lazarus & Folkman）：

| 五行 | 应对策略 | |-----|---------| | 木行 | 问题聚焦(主动创新解决) | | 火行 | 情绪聚焦(发泄+寻求支持) | | 土行 | 回避应对(压抑+忍耐) | | 金行 | 问题聚焦(严格按规则) | | 水行 | 情绪聚焦(沉思+回避) |

五、三级预防体系

一级预防(能量平衡)：月度五行自评+依五行特性调节作息/饮食/运动+匹配环境 二级预防(早期识别)：季度阴面行为筛查+心身症状监测+阴面初显即启动干预 三级预防(康复回归)：五行配伍康复方案+三界协同评估+社会功能回归训练
