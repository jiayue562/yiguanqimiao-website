C层核心金句

C1-C9 物质世界金句

1. "木长、火热、土化、金敛、水润——五材是宇宙的基本物质形态。"

2. "十二属相的五行归属，决定了生肖间的生克关系与命运互动。"

3. "木主文教创意，火主传媒餐饮，土主农业建筑，金主金融法律，水主贸易物流。"

4. "五行形煞的化解之道：木煞用金、金煞用火、火煞用水、水煞用土、土煞用木。"

5. "木宜青绿、火宜赤红、土宜黄棕、金宜白银、水宜蓝黑——服饰颜色的五行智慧。"

C10-C16 职业社会金句

1. "木行人创新引领，火行人愿景激励，土行人稳定执行，金行人制度保障，水行人智慧决策。"

2. "理想主义（木）、愿景驱动（火）、稳健务实（土）、制度规范（金）、智慧灵活（水）——五行企业文化。"

3. "木好独处重精神，火善社交爱热闹，土重稳定慢热型，金重界限社交广，水善变通重深度。"

4. "木需空间自由，火需关注认可，土需稳定承诺，金需界限清晰，水需深度连接。"

5. "木天赋战略眼光，火天赋表达能力，土天赋细致耐心，金天赋分析执行，水天赋洞察变通。"

C17-C25 中医养生金句

1. "肝木疏泄、心火温煦、脾土运化、肺金肃降、肾水封藏——五脏是生命的五大核心。"

2. "肝开窍于目、心开窍于舌、脾开窍于口、肺开窍于鼻、肾开窍于耳——五脏的体外窗口。"

3. "泪为肝液、汗为心液、涎为脾液、涕为肺液、唾为肾液——五液的健康信号。"

4. "筋属木、脉属火、肉属土、皮属金、骨属水——五体支撑生命结构。"

5. "风伤肝、暑伤心、湿伤脾、燥伤肺、寒伤肾——五邪侵袭的路径。"

6. "怒伤肝、恨伤心、怨伤脾、恼伤肺、烦伤肾——五毒内伤的根源。"

7. "肝经疏泄心经明，脾经运化肺经清，肾经封藏五脏安——经络是生命的河流。"

8. "舒肝功疏肝、养心功清心、健脾功和胃、润肺功润燥、补肾功固精——传统功法的五行智慧。"

C26-C32 生活实践金句

1. "木行舒肝导引、火行清心拍打、土行健脾摩腹、金行润肺吐纳、水行补肾踮脚——五行动功。"

2. "绿养肝、红养心、黄健脾、白润肺、黑补肾——五色饮食是五行的直接应用。"

3. "木宜瑜伽太极、火宜跑步游泳、土宜力量散步、金宜健身拳击、水宜游泳站桩——五行运动处方。"

4. "油属木（木行人易长痘）、红属火（火行人易敏感）、黄属土（土行人易暗黄）、干属金（金行人易粗糙）、暗属水（水行人易长斑）——五行美容诊断。"

5. "木从创意来、火从名望来、土从积累来、金从专业来、水从智慧来——五行财富密码。"
