``` 多场景问题 → L1感知 → 识别主次场景 → 主场景深度处理 → 次场景快速扫描 → 返回主场景整合

🧠 场景诊断决策树

第一步：问题初步分类

``` 来者问题 → 三大主类别判断： 1. 关系类（人际互动） → 转场景关系决策树 2. 自我类（个人成长） → 转自我类决策树 3. 组织类（工作系统） → 转组织类决策树 ```

第二步：关系类决策树

``` 关系问题 → ├─ 二元关系（只有两个人） → 亲密关系场景 ├─ 代际关系（父母子女） → 亲子关系场景 ├─ 工作关系（同事上司下属） → 团队建设/领导力场景 └─ 多角色复杂关系 → L1整合框架 + 多场景并行 ```

第三步：自我类决策树

``` 自我问题 → ├─ 身份认同「我是谁」 → 人格测评场景 ├─ 身心健康问题 → 健康养生场景 ├─ 日常困惑「如何过好每一天」 → 日常生活场景 └─ 表达困难「说不明白」 → 高效沟通场景 ```

第四步：组织类决策树

``` 组织问题 → ├─ 团队协作效率 → 团队建设场景 ├─ 领导与管理 → 领导力场景 └─ 组织沟通 → 高效沟通场景 ```

💎 场景协同增效组合

黄金组合1：人格测评 + 亲密关系

``` 效果：自我认知提升后，亲密关系的理解自然深化 适用：处于关系困境，同时自我认知模糊的来者 协同模式：人格测评建立基础 → 亲密关系实践应用 ```

黄金组合2：日常生活 + 健康养生

``` 效果：日常习惯调整支撑身体健康改善 适用：亚健康状态，但难以坚持系统养生的人群 协同模式：健康养生提供方向 → 日常生活提供路径 ```

黄金组合3：领导力 + 团队建设 + 高效沟通

``` 效果：从个人领导力到团队效能到沟通顺畅的完整链条 适用：企业管理者或团队领导者 协同模式：领导力（个人）→ 团队建设（集体）→ 高效沟通（连接） ```

黄金组合4：亲子关系 + 日常生活 + 健康养生

``` 效果：家庭健康、日常节奏、身心平衡的全家福方案 适用：家庭整体优化需求的来者 协同模式：亲子关系（家庭关系）→ 日常生活（家庭节奏）→ 健康养生（家庭健康） ```

📈 场景演化路径预测

短期预测（未来1-3个月）

``` 高频场景：人格测评 → 日常生活 → 亲密关系 （人们更多寻求自我认知和基础生活优化） ```

中期预测（未来6-12个月）

``` 增长场景：健康养生 → 亲子关系 → 团队建设 （后疫情时代对健康、家庭、协作的更加重视） ```

长期预测（未来1-3年）

``` 整合场景：多场景融合需求增加 （人们不再满足于单一问题解决，寻求整体生命优化） ```

进化准备策略

1. 数据积累：记录每个场景的实际使用频率和用户满意度 2. 流程优化：基于实际反馈不断优化场景转接路径 3. 工具迭代：根据新兴需求为每个场景开发新的独有工具 4. 整合能力：训练L1的多场景整合和复杂问题处理能力
