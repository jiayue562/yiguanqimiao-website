🎯 核心概念速查

三层次结构

``` 原象（道象） ←── 灵界·信息体 ──→ 大圆满·法身 ↑ 意象（象征） ←── 心界·能量体 ──→ 大圆满·报身 ↑ 物象（形象） ←── 身界·物质体 ──→ 大圆满·化身 ```

四大核心方法

1. 观物取象 → 认知：悬置→感知→提取→符号化 2. 立象尽意 → 表达：意会→寻象→立象→尽意→共鸣 3. 取象比类 → 推理：识别→寻找→建立→推导→验证 4. 观象制器 → 应用：观象→悟理→设计→制器→验证

六非两核心

  • 非实体性、非对象性、非现成性
  • 前语言性、前逻辑性、非确定性
