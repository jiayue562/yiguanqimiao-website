一、核心定义

1.1 一句话定义

客户旅程图以客户视角展示从认知→考虑→购买→使用→推荐的全过程，跟踪每个触点的客户行为、情感和痛点，识别优化机会。

1.2 核心公式

``` 客户旅程 = 阶段(Stage) × 行为(Action) × 触点(Touchpoint) × 情感(Emotion) ```

---

二、旅程阶段

2.1 典型五阶段

| 阶段 | 定义 | 客户问题 | |------|------|---------| | 认知 | 发现品牌/产品 | 这是什么？ | | 考虑 | 评估对比选择 | 值得买吗？ | | 购买 | 完成交易行为 | 怎么买？ | | 使用 | 体验产品服务 | 好用吗？ | | 推荐 | 推荐给他人 | 要分享吗？ |

2.2 各阶段关键指标

| 阶段 | 核心指标 | |------|---------| | 认知 | 触达率、品牌认知度 | | 考虑 | 转化率、对比率 | | 购买 | 成交率、客单价 | | 使用 | 活跃度、留存率 | | 推荐 | NPS、转介绍率 |

三、旅程图要素

3.1 核心组件

| 要素 | 内容 | |------|------| | 阶段 | 旅程的主要步骤 | | 触点 | 客户接触品牌的渠道/方式 | | 行为 | 客户在各阶段的具体行为 | | 情感 | 客户的情绪波动（高/中/低） | | 痛点 | 客户的不满和障碍 | | 机会 | 可优化的突破点 |

3.2 情感曲线

``` 情感 高 ┃ ╭─╮ ┃ ╭──╮ ╭╯ ╰╮ ┃╭──╯ ╰─╯ ╰─── ┃│ 低 ┗━━━━━━━━━━━━━━━━━━→ 阶段 认知 考虑 购买 使用 推荐 ```

四、实战模板

场景：电商购物旅程

``` 【客户旅程图 · 电商购物】

阶段① 认知 - 触点：小红书/抖音种草 - 行为：刷到产品内容 - 情感：好奇、兴趣 - 痛点：信息太多难选择 - 机会：精准推荐

阶段② 考虑 - 触点：搜索、详情页、评价 - 行为：对比价格/功能/口碑 - 情感：犹豫、权衡 - 痛点：评价真实性存疑 - 机会：真实评价展示

阶段③ 购买 - 触点：商品页、购物车、支付 - 行为：领券、下单、付款 - 情感：期待、些许焦虑 - 痛点：支付流程复杂 - 机会：一键下单

阶段④ 使用 - 触点：收到商品、开箱、使用 - 行为：体验产品 - 情感：满意或失望 - 痛点：实物与预期不符 - 机会：超预期体验

阶段⑤ 推荐 - 触点：社交媒体、评价、口碑 - 行为：分享、评价 - 情感：满足或不满 - 痛点：分享动力不足 - 机会：激励机制 ```
