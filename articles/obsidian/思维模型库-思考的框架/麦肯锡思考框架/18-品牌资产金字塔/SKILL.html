| 要素 | 内容 | |------|------| | 品牌认知 | 知道这个品牌 | | 品牌记忆 | 能在众多品牌中想起 | | 品牌想起 | 购买时第一时间想起 |

2.2 第二层：品牌意义

定义：品牌在消费者心中的内涵

| 要素 | 内容 | |------|------| | 功能利益 | 产品/服务能做什么 | | 情感利益 | 使用带来什么感受 | | 自我表达 | 品牌代表什么身份 |

2.3 第三层：品牌反应

定义：消费者对品牌的评价

| 要素 | 内容 | |------|------| | 品质感知 | 看起来/用起来怎么样 | | 品牌认可 | 这个品牌好不好 | | 品牌差异化 | 和竞品有什么不同 |

2.4 第四层：品牌共鸣

定义：消费者与品牌的情感连接

| 要素 | 内容 | |------|------| | 品牌依恋 | 离不开这个品牌 | | 品牌社区 | 参与品牌社群 | | 主动推荐 | 愿意向他人推荐 |

| 要素 | 内容 | |------|------| | 品牌忠诚 | 反复购买不换 | | 品牌拥护 | 主动捍卫品牌 | | 终身价值 | 最高客户价值 |
