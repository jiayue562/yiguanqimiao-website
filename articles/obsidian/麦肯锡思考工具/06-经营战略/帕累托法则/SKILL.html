帕累托法则 · 80/20效率法则

🎯 一、核心定义

帕累托法则（Pareto Principle）源于意大利经济学家维尔弗雷多·帕累托的观察：80%的结果来自20%的原因。这一法则广泛应用于资源优化、问题诊断和时间管理。

1.1 80/20法则的本质

``` 80/20法则核心： ┌─────────────────────────────────────┐ │ 80%的产出 ←── 20%的投入 │ │ 80%的问题 ←── 20%的原因 │ │ 80%的价值 ←── 20%的客户 │ │ 80%的效果 ←── 20%的努力 │ └─────────────────────────────────────┘ ```

1.2 帕累托分布

``` 帕累托分布图： 结果 │ │ ████ ← 20%的投入 │ ████ │███ ← 80%的结果 │███ │███ └──────────────────────→ 投入 ```

1.3 帕累托法则的价值

``` 帕累托法则价值： ┌─────────────────────────────────────┐ │ 聚焦关键 → 识别最重要的20% │ │ 效率提升 → 用20%努力获得80%结果 │ │ 资源优化 → 资源配置到关键少数 │ │ 持续改进 → 找到杠杆解 │ └─────────────────────────────────────┘ ```
