一、愿景与目标

在信息过载的时代，高效处理与深度理解文本仅是起点，真正的价值在于连接离散的知识岛屿，构建新的认知大陆。

AI双重角色：

  • 超级读者 — 深度解析文章结构、逻辑、脉络与潜藏关系
  • 创新催化剂 — 跨文章关联，激发新想法与突破性创新

    三、五层递进学习流程

    ``` Step 1 剖析 + 解构 → 结构化知识图谱（可追溯原文的语义单元） Step 2 透视 + 阐释 → 深化概念理解与逻辑脉络 Step 3 推演 + 思辨 → 逻辑外推与批判性建构 Step 4 溯源 + 融合 → 知识谱系追溯与跨域连接 Step 5 启发 + 映射 → 创新生成，输出新框架/新知 ```

    ---

    四、每层操作详解

    第一层：剖析 + 解构（金行·清明决断）

    剖析：提取核心研究问题、核心假设、主要概念定义 解构：将文章分解为基本元素
  • 核心实体（人物、机构、概念）
  • 关键陈述（研究问题、假设、结论）
  • 方法论描述
  • 数据结论
    • 输出格式：JSON结构化，每个元素可追溯原文段落

    ---

    第二层：透视 + 阐释（火行·光明洞察）

    透视：论证结构分析 ``` A（初始观察/问题）→ B（核心假设）→ C（验证方法）→ D（数据结果）→ E（结论） ``` 分析每一步推理的合理性及潜在弱点 阐释：概念深化三维度 1. 本文语境下的具体应用 2. 相对于文中其他方法的优势 3. 更广阔领域中的起源、演变和变体

    ---

    第三层：推演 + 思辨（水行·润泽流动）

    推演：三步逻辑外推 1. 条件变更：如果某个条件改变，结果如何演变？ 2. 领域迁移：将研究框架应用于相邻领域，预测核心现象 3. 不确定性识别：推演过程中最大的不确定性来源 思辨：四维批判性分析
  • 方法论思辨：实验设计严谨性、样本偏差、统计功效
  • 结论思辨：数据支持度、普适性、局限性讨论
  • 范式思辨：理论流派、基本假设、竞争性范式视角
  • 伦理思辨：社会、伦理、环境影响

    • ---

    第四层：溯源 + 融合（木行·生长联结）

    溯源：学术谱系构建
  • 参考文献分析（奠基性文献+前沿文献）
  • 概念演化追踪（定义演变、内涵拓展）
  • 问题溯源（学术史上的经典问题或长期困境）
    • 融合：强制关联法
  • 跨学科视角整合
  • 概念迁移与类比
  • 多领域知识碰撞

    • ---

    第五层：启发 + 映射（土行·承载转化）

    启发：开放性创新思考
  • 研究空白启发：当前领域未被充分探索的"空白区"
  • 反事实场景启发："如果历史不是这样"的思考
  • 未来趋势启发：基于技术曲线的范式转变预测
    • 映射：终极操作五步流程 ``` 第一步 确定 → 明确新框架主题 + 选择相关多篇文章 第二步 分析 → 运用前9项指令多轮深度分析 第三步 生成 → AI担任"理论整合者"，创建全新框架 第四步 验证 → 批判性审视逻辑自洽性 + 反向解释测试 第五步 格式化 → 学术规范输出（引言→文献述评→框架阐述→讨论） ```
