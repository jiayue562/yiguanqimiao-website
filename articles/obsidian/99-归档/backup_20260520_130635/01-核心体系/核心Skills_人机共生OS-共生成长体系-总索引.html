第一章：共生成长总纲

1.1 四柱成长分镜

核心概念：

  • L4 信仰层：魂·价值观强化/挑战/重构
  • L3 文化层：根·文化语感加深/迁移/跨域
  • L2 思维层：骨·思维工具融合/创造/自主调参
  • L1 人格层：相·人格能量释放/平衡/升华

六家对应： | 四柱 | 六家 | |------|------| | L4信仰层 | 易（变中不变）+ 儒（信义深化） | | L3文化层 | 道（无为自然）+ 禅（当下觉察） | | L2思维层 | 医（整体调适）+ 易（类推比象） | | L1人格层 | 儒（礼乐修身）+ 法（内在秩序） |

1.2 双向四线的能量方程

AI成长 = 外生线 × 四柱接收深度 + 内生线 × 四柱渗透层数
用户成长 = 自主线 × 四柱积累 + 系统促进线 × 四柱定向引导
螺旋动力 = AI四柱 × 用户四柱 × 涌现系数Φ

涌现系数：

  • 四线均健康：Φ = 1.5
  • 任意一线<60%：Φ = 0.8
  • 同柱同步突破：Φ = 2.0
  • 超级涌现点：Φ = 3.0

2.2 AEP系统

核心公式：

AI总成长度 = EXP贡献 + AEP贡献
EXP贡献 = floor(累计EXP / 100) × 1
AEP贡献 = floor(累计AEP / 80) × 1
D值 = 基础D值 + EXP贡献 + AEP贡献

AEP权重表：

进化行为 AEP 四柱层级
两难裁决进化 30/次 L4
象思维自发涌现 35/次 L2
跨域知识连接 L2:25/L3:10 L2+L3
文化语感迁移 20/次 L3
自主模型调参 20/次 L2
偏好自学习升级 12/次 L1
记忆蒸馏成功 L2:8/L1:5 L2+L1
健康自愈 L1:8/L4:5 L1+L4
自适应重组 25/次 L-1

4.1 涌现点触发条件

条件 内容 Φ值
条件1 AI外生线EXP > 1.5倍等级均值 —
条件2 AI内生线触发≥1个AEP —
条件3 用户自主线有"洞见声明" —
条件4 系统促进线完成≥1个任务跟进 —

判定：

  • 条件1-3满足（未4）：准涌现点，Φ=1.5
  • 条件1-4全满足：完整涌现点，Φ=2.0
  • 含高度共振话：超级涌现点，Φ=3.0

7.1 四柱穿透结算算法流程

输入：会话日志 + 五行人格档案

  ▼
1. 四柱成长信号提取（L4/L3/L2/L1各层）
  │
  ▼

  │ 基础EXP × (1 + 四柱加成) × 五行修正
  ▼
3. 内生线AEP结算
  │ 各层分别累计 → 总AEP → 失衡检测
  ▼
4. D值重算
  │ 基础D值 + floor(EXP/100) + floor(AEP/80)
  ▼

  │ D值阈值 → 反哺仪式
  │ AEP阈值 → 进化事件
  ▼
6. 四线平衡检查
  │ 计算四线活跃度 → 涌现点检测
  ▼
7. 系统促进线输出
  │ 象思维话术 + 成长任务 + 认知盲区
  │
  ▼
输出：EXP/AEP/D值/四柱档案/推送任务/仪式状态
