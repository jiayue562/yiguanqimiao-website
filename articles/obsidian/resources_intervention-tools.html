一、觉察工具（6个）

1.1 金属体检表

用途：快速评估金行能量状态

| 维度 | 问题 | 评分(1-5) | |------|------|----------| | 身界 | 我的身体是否感觉僵硬、紧绷？ | | | 身界 | 我的动作是否变得拘谨、刻板？ | | | 心界 | 我是否容易烦躁、挑剔？ | | | 心界 | 我是否过度追求完美、计较得失？ | | | 灵界 | 我是否变得固执、不愿变通？ | | | 灵界 | 我是否想要批判、控制一切？ | |

评分解读：
  • 6-12分：阳金主导，健康状态
  • 13-24分：阴阳交战，一般状态
  • 25-30分：阴金主导，需要重点转化

    • ---

    1.2 明镜冥想

    引导词： 1. 找一个舒适的坐姿 2. 想象自己是一面明镜 3. 镜面平整光滑，能清晰映照万物 4. 观察脑海中浮现的"对错"、"应该"、"不应该" 5. 问自己：这些评判是必要的吗？ 6. 想象这些评判如灰尘被轻轻拂去 7. 感受镜面的清明与通透 时长：10-15分钟

    ---

    1.3 僵化识别清单

    以下情况出现3项以上，说明金行僵化：
  • [ ] 对事情必须有绝对的对错判断
  • [ ] 难以接受与自己不同的观点
  • [ ] 认为自己的方法才是最好的
  • [ ] 经常纠正他人的小错误
  • [ ] 难以容忍混乱或无序
  • [ ] 对规则过度执着
  • [ ] 很难原谅他人的错误
  • [ ] 觉得别人都不够努力

    • ---

    1.4 情绪温度计

    用途：监测阴金情绪的日常波动

    | 情绪 | 阴金表现 | 阳金转化 | |------|---------|---------| | 烦躁 | 挑剔指责，无法安静 | 转化为行动改进 | | 悲伤 | 冷漠疏离，深沉抑郁 | 转化为深度洞察 | | 愤怒 | 刻薄攻击，言语暴力 | 转化为原则捍卫 | | 焦虑 | 过度担忧，追求完美 | 转化为清晰规划 |

    ---

    1.5 原则自检

    问题： 1. 我坚持这个原则，是因为它真的正确，还是因为"我不能错"？ 2. 如果有人用我的原则来反驳我，我会是什么感受？ 3. 我是否允许例外？还是必须严格执行？ 4. 我的原则是为了大家好，还是为了证明我对？ 5. 当原则与关系冲突时，我会选择什么？ 解读：
  • 0-1个"为了证明我对"：原则清晰
  • 2-3个"为了证明我对"：需要审视动机
  • 4-5个"为了证明我对"：原则变成控制

    • ---

    1.6 公正肌肉训练

    用途：逐步建立真正的公正 练习： 1. 每天找一件"不公正"的事，分析：真的是不公正吗？ 2. 尝试站在对方立场思考问题 3. 记录自己因为"坚持原则"而伤害关系的次数 4. 练习说："我理解你的角度" 5. 区分"原则"和"偏好"

    二、转化工具（6个）

    2.1 拔阴取阳口诀

    心界：阴金恼→阳金行

    | 阴金反应 | 转化问题 | |---------|---------| | "这太荒谬了" | "我立刻能做些什么来改善？" | | "他完全错了" | "他的角度有什么可取之处？" | | "必须按流程" | "流程的目的是什么？" |

    灵界：阴金贪→阳金义

    | 阴金反应 | 转化行动 | |---------|---------| | 追求证明自己对 | 问：这件事的核心价值是什么？ | | 想要控制局面 | 练习"授权"，接受不同做法 | | 吝啬分享资源 | 思考：如何让更多人受益？ |

    ---

    2.2 化克为生实践

    金克木 → 金生水 → 水生木 场景：人际关系紧张

    | 步骤 | 行动 | 心理转变 | |------|------|---------| | 1 | 承认自己的严厉 | 诚实面对 | | 2 | 问：我的原则是为了什么？ | 从对抗到初心 | | 3 | 转化为理解对方的智慧 | 金→水 | | 4 | 用更柔软的方式表达 | 水→木 |

    ---

    2.3 评判转化日记

    模板：

    ``` 日期：_______ 触发事件：_______ 我的阴金反应（评判/挑剔/指责）：_______ 转化问题： 1. 事实是什么？_______ 2. 我的评判是基于原则还是偏好？_______ 3. 建设性的下一步是什么？_______ 转化后的感悟：_______ ```

    ---

    2.4 接纳不确定性练习

    用途：化解对"绝对确定"的执着 步骤： 1. 回忆一次"没有绝对对错"的经历 2. 问：当时真的只能二选一吗？ 3. 当时我的恐惧是什么？ 4. 多元视角会带来什么可能？

    ---

    2.5 明镜冥想

    引导词： 1. 想象自己是一面明镜 2. 镜面光滑如镜，能映照万物 3. 任何事物经过，镜子都如实反映 4. 但镜子不会说"这是好的"或"这是坏的" 5. 当你想评判时，想象自己就是那面镜子 时长：10分钟

    三、关系工具（3个）

    3.1 金行人际边界设定

    原则：
  • 坚守核心原则，但允许方式灵活
  • 清晰表达期望，但倾听对方难处
  • 坚持标准，但提供支持帮助
  • 维护秩序，但接纳创新尝试

    • ---

    3.2 金行沟通风格优化

    问题风格：
  • ❌ "你这样做完全不对！"
  • ❌ "你总是犯同样的错误！"
    • 优化风格：
  • ✅ "我注意到这个结果和预期有差距，我们来看看原因？"
  • ✅ "我想了解你在过程中遇到的困难，可以说说吗？"
