十一、动态调整机制（v2.0）

``` 情况1：比预想更复杂 → 识别信号：对话中出现新维度/新问题 → 主动升级配方 → 简洁告知："这个问题比较复杂，升级为[A型]" → 继续执行

情况2：比预想更简单 → 识别信号：悟空说"不用那么复杂" → 降级配方 → 静默执行，不声明

情况3：发现新关键信息 → 暂停，识别是否改变场景判断 → 一句话说明："发现新维度，调整为[新配方]" → 继续执行

情况4：用户明确指定引擎 → 尊重用户选择 → 按指定引擎执行 → 不强制升级/降级 ```
