🎯 8 条核心原则

| 原则 | 说明 | |------|------| | 1. 一个入口，多个终点 | 所有输入走 Notes/，统一流程分发 | | 2. 结构够用就行 | 目录分到能找到东西就够了 | | 3. AI 建议，你拍板 | 分类反映你的思维结构 | | 4. 链接宜少不宜多 | 两三条强关联比十条弱关联有用 | | 5. 对话也是知识 | 跟 AI 的深度讨论本身就在生产知识 | | 6. 存着不用等于没有 | 知识库的终极检验标准是产出 | | 7. 索引到目录，不到文件 | 目录结构稳定后几乎零维护 | | 8. 不绑定工具 | AGENTS.md 兼容多种 Agent |

---

🔧 三大核心操作

Ingest（入库）

``` 新资料 → 列方案 → 确认 → 分发 → 加工 → 清理 ```

Query（查询）

``` 提问 → 搜索 → 综合回答 → 发现关联 → 建议补链接 ```

Lint（健康检查）

``` 孤立页面 / 过时内容 / 缺失引用 / Notes 积压 → 生成报告 ```
