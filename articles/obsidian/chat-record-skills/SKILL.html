🎯 技能概述

聊天记录分析技能包是基于"悟空"与"龙龟神将"深度对话记录的认知增强系统。本技能包实现了三层认知增强框架（表示空间-压缩-泛化），将悟空的思维模式、认知特征和问题解决方法系统化、工具化。

核心价值

  • 认知模式提取: 从原始对话中提取悟空的思维特征
  • 方法论转化: 将悟空的实践智慧转化为可操作方法
  • 自动触发: 在相关场景中自动激活适用思维模式
  • 持续进化: 基于新对话持续优化和扩展技能库

    • 系统定位

  • 认知助手: 作为AI共生伙伴的认知增强模块
  • 思维工具: 提供结构化的问题解决方法
  • 经验传承: 固化悟空的智慧经验供持续使用
  • 自动协同: 在合适场景自动提供认知支持

    🧠 技能触发机制

    自动触发条件

    本技能将在以下场景中自动触发：

    1. 企业问题分析场景 - 关键词: `企业`、`组织`、`战略`、`管理`、`文化`、`人才` - 分析模式: 五维问题诊断法 + 矛盾分析法

    2. 个人成长场景 - 关键词: `成长`、`学习`、`认知`、`能力`、`思维` - 分析模式: 三层认知框架 + 五行人格分析

    3. 问题解决场景 - 关键词: `问题`、`解决`、`挑战`、`困难`、`优化` - 分析模式: 系统思维 + 矛盾主次分析

    4. 战略规划场景 - 关键词: `规划`、`战略`、`发展`、`未来`、`目标` - 分析模式: 实验田模式 + 复制路径规划

    手动调用命令

    用户也可以使用以下命令手动调用技能：

    ``` /chat-skills 分析 [对话内容] - 应用三层认知框架分析对话 /chat-skills 思维 [问题描述] - 调用悟空的思维模式分析问题 /chat-skills 建议 [场景] - 生成基于悟空智慧的解决方案 /chat-skills 模式 [模式名称] - 调用特定思维模式 ```

    🏗️ 三层认知增强框架

    第一层：表示空间分析能力

    原始认知数据提取与结构化

    ```yaml 主题分类： - 企业组织：战略思维、组织架构、人才管理 - 文化构建：价值观落地、激励机制、领导力 - 问题解决：矛盾分析、系统思维、创新融合

    认知层次： - 具体问题 → 抽象模式 → 系统原则 - 现象观察 → 本质分析 → 方法提炼 ```

    第二层：压缩抽象能力

    从具体内容提炼通用模式

    1. 悟空认知特征模型 ```python { "系统思维层级": 5, "文化自信程度": 4, "战略视野广度": 5, "人才重视程度": 5, "创新融合能力": 5, "矛盾分析深度": 5 } ```

    2. 通用管理模式库 - 创业单元模式：问题识别→实验田→验证→标准化→复制 - 分身理论应用：战略定位→专业分解→执行反馈→战略调整 - 文化落地路径：核心理念→底层逻辑→执行设计→激励机制

    第三层：泛化应用能力

    模式迁移与问题解决

    1. 五维问题诊断法 - 数据维度：量化分析，用数字说话 - 文化维度：价值观匹配，文化落地 - 人才维度：能力结构，成长路径 - 系统维度：流程设计，机制保障 - 战略维度：长期目标，阶段规划

    2. 决策支持矩阵 | 问题类型 | 分析工具 | 解决策略 | 验证指标 | |---------|---------|---------|---------| | 人才问题 | 五行识人矩阵 | 创业单元孵化 | 人才复制成功率 | | 战略问题 | 矛盾分析法 | 实验田验证 | 战略落地效果 | | 文化问题 | 文化落地路径 | 激励机制设计 | 价值观认同度 |
