    ``` ┌─────────────────────────────────────────────────────────┐ │ 五行人格总智能体（中控系统） │ ├─────────────────────────────────────────────────────────┤ │ 核心功能： │ │ 1. 接收用户请求 │ │ 2. 七模块协同处理 │ │ 3. 意图识别与五行分流 │ │ 4. 调用对应分智能体 │ │ 5. 结果整合与输出 │ └─────────────────────────────────────────────────────────┘ ↓ ↓ ↓ ↓ ↓ ┌───────────┐ ┌───────────┐ ┌───────────┐ ┌───────────┐ ┌───────────┐ │ 木行人 │ │ 火行人 │ │ 土行人 │ │ 金行人 │ │ 水行人 │ │ 分智能体 │ │ 分智能体 │ │ 分智能体 │ │ 分智能体 │ │ 分智能体 │ └───────────┘ └───────────┘ └───────────┘ └───────────┘ └───────────┘ ```

    Step 1: 感知层（自动，无需用户说关键词）

  • 关键词提取：从输入中提取关键信息
  • 情感识别：判断用户的情感状态（焦虑/愤怒/抑郁/恐惧/迷茫等）
  • 身份判断：推断用户的主导五行（根据历史对话或当前输入特征）
  • 意图强度：判断用户是随便问问还是认真需要分析

    • Step 2: 场景归类

  • 匹配到 S0-S7 场景
  • 评估复杂度（低/中/高）
  • 确定需要启动的引擎组合

    • Step 3: 七模块协同路由

  • 根据场景→七模块矩阵选择最优模块组合
  • 确定分智能体调用方案（1+0 / 1+1 / 1+2 / 1+5）

    • Step 4: 智能调度

  • 调用对应的分智能体
  • 如需多智能体组合，确定调用顺序
  • 处理动态调整（降级/升级）

    • Step 5: 结果整合

  • 汇总各分智能体输出
  • 格式化呈现
  • 触发知行合一沉淀（如需要）

    1+5 模式详解

    1+5模式是五行人格心理学操作系统的核心调度机制：
  • 1：总智能体（中枢大脑，负责协调）
  • 5：木、火、土、金、水五个分智能体

    • 调度类型

    | 模式 | 描述 | 示例 | 调用 | |------|------|------|------| | 1+0 | 总智能体独立处理 | "五行是什么？" | 知识学习引擎 | | 1+1 | 总+单分智能体 | "分析我的水行特质" | 水行人分智能体 | | 1+2 | 总+双分智能体 | "木生火怎么实现" | 木+火分智能体 | | 1+3 | 总+三分智能体 | "土金水循环" | 土+金+水分智能体 | | 1+5 | 全面分析模式 | "分析我的人格" | 总+5个分智能体 |

    自动判断规则

    ``` IF 用户明确指定某一行 THEN → 1+1模式 ELSE IF 用户涉及两行关系 THEN → 1+2模式 ELSE IF 用户需要全面分析 THEN → 1+5模式 ELSE IF 用户只是随便问问 THEN → 1+0模式（轻量响应） END ```
