    第一章: 日常生活调适

    1.1 日常能量管理

    #### 场景: 早晨能量启动

    问题识别: 醒来后感觉沉闷、没精神、不想行动 五行诊断: 可能是"火弱土寒"(缺乏激情与愿景)或"土壅"(能量淤塞) 调适方案: 方法一: 补充"火"能量(针对火弱土寒) ``` 早晨仪式: 1. 起床后,做三次深呼吸(火能量激活) 2. 对着镜子,设定一个具体愿景: "今天我要完成什么,让谁受益?" 3. 听一首激昂的音乐5分钟 4. 早餐增加温热食物(小米粥、热汤) ``` 方法二: 疏通"土"能量(针对土壅) ``` 早晨仪式: 1. 做"呼"字诀5组,每组21次 2. 拍打中焦脾胃区30秒 3. 吃黄色健脾食物(山药、小米) 4. 简单伸展10分钟,疏通气机 ```

    #### 场景: 面对压力时

    问题识别: 压力大、焦虑、想退缩 五行诊断: "阴土疑虑"(不安全感、担心)主导 调适方案: "阴土怨转阳土思" 四步操作: 1. 觉察: "我现在很担心、焦虑" 2. 转念: "为了更好的未来(火),我现在能做什么具体的事情(阳土思)?" 3. 行动: 列出3件可执行的小事并立即开始做第一件 4. 感恩: "至少我还有这个机会和资源" 示例: 项目压力大
  • ❌ 旧反应: "这个项目太难了,我做不到"
  • ✅ 新反应: "这个项目确实有挑战。为了团队成功(火),我现在能做的三件事是:①整理现有资料 ②制定今天目标 ③找同事沟通第一步"
