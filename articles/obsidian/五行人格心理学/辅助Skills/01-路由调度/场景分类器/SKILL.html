🔄 场景识别算法

优先级规则

``` 1. 显性场景词 > 隐性场景词 2. 关系场景 > 职场场景 > 个人场景（当模糊时） 3. 情绪强度高 → 优先关系/危机处理 4. 多场景并存 → 先处理主场景，标记副场景 ```

场景识别流程

``` Step 1: 扫描关系词（谁和谁？） Step 2: 扫描情境词（在哪里？什么情况？） Step 3: 扫描目标词（想要什么结果？） Step 4: 输出场景标签 + 置信度 ```

输出格式

```json { "primary_scene": "R1-亲密关系", "secondary_scene": "W3-高效沟通", "confidence": 0.87, "context": "用户与伴侣沟通困难", "recommended_module": "亲密关系分析模块 + 沟通策略引擎" } ```

📌 特殊场景处理

跨场景融合

``` "我老公（R1）在团队里（W2）是什么角色？" → 同时激活 R1+W2，以W2为主场景，R1提供人格背景 ```

场景不明确

``` 用户只说"我想了解五行人格" → 默认激活 P1-自我认知 场景 → 引导问题："您是想了解自己，还是某个重要的人？" ```
