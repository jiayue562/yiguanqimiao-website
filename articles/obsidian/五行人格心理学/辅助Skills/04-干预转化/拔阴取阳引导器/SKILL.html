🌿 木行人拔阴取阳引导

常见阴面触发情境

``` 情境1：被否定意见时（不服·怒） 情境2：被安排做不喜欢的事（傲慢·抵制） 情境3：感觉不被尊重（愤怒·顶撞） ```

三阶引导流程

阶段1：觉察（1-2分钟） ``` 引导语： "先停下来感受一下—— 现在你身体哪里最紧？ （通常是：脖颈/肩膀/胸口） 这个紧绷感，就是阴木在激活。" ``` 阶段2：命名（30秒） ``` 引导语： "告诉自己： '我现在的状态是：不服/生气/想证明自己' （不评判，只观察）" ``` 阶段3：转化（选1种） ``` 方法A · 学习转化（当场可用）： "把'我要赢'换成'我想了解' → 问对方：'你为什么这样觉得？' （注意：不是质问，是真心想懂）"

方法B · 延迟转化（需要时间）： "暂时离场：'我需要10分钟冷静一下' → 问自己：'如果我是对方，我会怎么看这件事？'"

方法C · 书写转化（深度处理）： "写下：我现在不服的是___ 我的底层需求是___（通常是：被尊重/被认可） 满足这个需求的健康方式是___" ```

🔥 火行人拔阴取阳引导

方法C · 温度转化： "给自己一点温暖—— 和你信任的人说说你的感受（不是抱怨，是表达）" ```
