阴阳状态检测 · Yin-Yang State Detector

> 核心定位：五行能量的实时晴雨表，快速读取用户当前是在"阳面运行"还是"阴面运行"，是所有干预的前置诊断。

---

🌗 阴阳状态基础模型

阳面状态（健康运行）

``` 能量特征： - 情绪稳定，能量流动顺畅 - 与五行本性同频（木·仁德 / 火·礼明 / 土·信实 / 金·清明 / 水·智慧） - 关注点向外（他人/世界/贡献） - 能灵活应对挑战

外在表现： - 语言中有"我可以" / "怎么办" / "试试" - 情绪表达有边界 - 愿意倾听不同观点 ```

阴面状态（失衡运行）

``` 能量特征： - 情绪主导行为，能量内耗 - 阴性五行主导（木·傲慢 / 火·嗔恨 / 土·忧郁 / 金·贪恼 / 水·恐惧） - 关注点向内（自我保护/防御/索取） - 模式化反应，无法灵活

外在表现： - 语言中有"总是" / "又是" / "为什么总..." - 情绪超出边界（爆发或完全压制） - 防御性强，难以接受不同意见 ```

🔍 实时检测算法

检测维度（4个快速指标）

指标1：情绪色彩 ```
  • 正向情绪词（感激/开心/平静/希望）→ 阳面信号
  • 负向情绪词（愤怒/委屈/绝望/恐惧）→ 阴面信号
  • 情绪词缺失（过于理性/麻木）→ 压制状态
    • ``` 指标2：主体归因 ```
  • 归因于自己（"我可能需要...""我想改变"）→ 阳面
  • 归因于外部（"都是他的问题""总是这样"）→ 阴面
  • 混合归因 → 过渡状态
    • ``` 指标3：能量方向 ```
  • 扩展型语言（"想要""期待""计划"）→ 阳面
  • 收缩型语言（"不想""避免""算了"）→ 阴面
  • 防御型语言（"只要不...就好"）→ 阴面初期
    • ``` 指标4：时间视角 ```
  • 面向未来 → 阳面（行动导向）
  • 困于过去 → 阴面（反刍模式）
  • 只有当下且无力感 → 危机前兆
    • ```

    检测公式

    ``` 阴阳指数 = (阳面指标数 - 阴面指标数) / 总指标数

    > 0.5 → 明显阳面，正常继续 0.2~0.5 → 轻微偏阳，关注维护 -0.2~0.2 → 临界状态，需要关注 -0.5~-0.2 → 轻微偏阴，启动干预 < -0.5 → 明显阴面，优先调节 ```

    主导五行（当前）：[元素·阴/阳] 信号依据： - [具体信号1] - [具体信号2]

    建议策略： → [阳面稳定：继续深入分析] → [临界状态：先做状态调节再分析] → [明显阴面：优先激活干预模块] ```

    阴面激活时的优先处理

    ``` Step 1: 先共情，不急于分析 "我听到你现在[情绪]，这种感觉很[正常/难受/...]"

    Step 2: 确认安全 高强度阴面（绝望/崩溃信号）→ 先确认基本安全

    Step 3: 能量稳定后 轻度激活→直接进入分析 中度激活→先给出当下调节建议 重度激活→只做支持，不做分析 ```
