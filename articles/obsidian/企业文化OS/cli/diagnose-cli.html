🎯 核心定位

企业文化诊断CLI是企业文化OS的原生操作入口，提供极简、语义清晰、人格化适配的命令行接口，让企业管理者能够通过纯文本指令一键启动企业文化诊断流程。

设计哲学：
  • 极简设计：指令长度控制在 5-10 个词
  • 语义清晰：指令意图明确，无需二次确认
  • 人格化适配：所有指令都携带企业文化OS的个性化参数
  • 全系统CLI化：AI 可通过纯文本指令一键调用诊断、设计、落地全流程

    通用参数

    | 参数 | 类型 | 说明 | 示例 | |------|------|------|------| | `--企业` / `-c` | string | 企业名称 | `--企业="XX科技"` | | `--行业` / `-i` | string | 行业类型 | `--行业="互联网"` | | `--规模` / `-s` | string | 企业规模 | `--规模="100-500人"` | | `--部门` / `-d` | string | 目标部门 | `--部门="销售部"` | | `--维度` / `-v` | string | 诊断维度 | `--维度="精神层"` | | `--输出` / `-o` | string | 输出路径 | `--输出="./report.md"` | | `--详细` / `--verbose` | bool | 详细输出 | `--详细` |

    💡 使用示例

    示例1：完整诊断流程

    ```bash

    启动全面企业文化诊断

    企业文化诊断 \ --企业="XX科技有限公司" \ --行业="互联网/软件开发" \ --规模="200-500人" \ --诊断类型="全面" \ --调研方法="问卷,访谈,观察" \ --样本量=150 \ --输出="./diagnose-report.md" \ --详细 ```

    系统响应： ``` 🔍 企业文化诊断启动 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 企业：XX科技有限公司 行业：互联网/软件开发 规模：200-500人 诊断类型：全面诊断 调研方法：问卷 + 访谈 + 观察 样本量：150人 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    📋 诊断流程： 1. 资料收集与初步分析 2. 问卷调研设计 3. 深度访谈实施 4. 现场观察记录 5. 数据分析与诊断 6. 生成诊断报告

    ⏱️ 预计耗时：5-7个工作日 🎯 开始执行...（调用企业文化诊断子智能体） ```

    示例2：快速现状扫描

    ```bash

    快速扫描销售部文化现状

    文化现状扫描 \ --企业="XX科技" \ --部门="销售部" \ --维度="行为层,制度层" \ --快速 ```

    示例3：基于诊断的设计

    ```bash

    基于诊断报告启动文化设计

    企业文化设计 \ --诊断报告="./diagnose-report.md" \ --重点模块="经营心法,礼乐仪式,功勋体系" \ --风格偏好="融合" \ --输出="./culture-design.md" ```

    示例4：分阶段落地

    ```bash

    启动第一阶段落地

    企业文化落地 \ --设计方案="./culture-design.md" \ --阶段="第一阶段：理念宣贯" \ --试点部门="人力资源部,销售部" \ --周期="3个月" ```
