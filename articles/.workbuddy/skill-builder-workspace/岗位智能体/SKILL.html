🤖 岗位智能体 · SKILL.md

核心定位：基于五行人格心理学OS的岗位AI工作流设计系统
一句话功能：通过"行业×企业×岗位×性格×任务×模型"六维框架，生成精准适配的AI提示词与智能体使用方案
版本：v1.0 | 创建日期：2026-04-20 | 维护者：龙龟神将

📖 核心定义

What（是什么）

岗位智能体是龙龟神将独创的业务场景×人格特质×思维模型深度融合的AI提示词生成与岗位工作流设计工具。

它解决了传统提示词"一刀切"的问题，通过六维框架精准识别岗位特性，结合五行人格心理学，为每个岗位生成专属的AI协作方案。

核心组成：
  • 六维输入框架：行业 + 企业简介 + 岗位 + 性格（五行人格）+ 核心任务 + 适用模型
  • 关键动作设计：目标量化（结果指标）+ 资源分配 + 数据验证（过程指标）+ 风险控制
  • AI增强关键词：洞察、剖析、透视、阐释、推演、解构、思辨、溯源、融合、启发、映射
  • 提示词工程体系：黄金公式 + 基础结构 + 高级技巧 + 专家级资产

    • Why（为什么）

    现状问题：
  • 🔴 通用提示词无法满足特定岗位的深度需求
  • 🔴 忽视人格特质导致AI协作效果不佳
  • 🔴 缺乏系统化工作流设计方法
  • 🔴 结果指标与过程指标混淆
  • 🔴 风险控制缺乏预警机制
    • 解决方案：
  • ✅ 五行人格×业务场景深度融合（木生火共生视角）
  • ✅ 工作流化思维（信息交汇点精准识别）
  • ✅ 六维框架全面覆盖（行业→企业→岗位→性格→任务→模型）
  • ✅ 结果指标与过程指标双轨制
  • ✅ 风险指标前置预警

    • How（怎么做）

    核心工作流： 
    六维输入（行业/企业/岗位/性格/任务/模型）
    ↓
五行人格识别与适配（金木水火土）
    ↓
关键动作拆解（目标量化/资源分配/数据验证/风险控制）
    ↓
AI增强关键词选择（十项认知指令）
    ↓
提示词框架生成（黄金公式/基础结构/高级技巧）
    ↓
完整智能体方案输出

    ⚡ 触发条件

    P0·直接触发词（权重5·绝对触发）

    出现任意1个即触发（5分）：

  • "岗位智能体"
  • "生成提示词"
  • "岗位AI"
  • "工作流设计"
  • "智能体方案"
  • "构建岗位助手"
  • "AI岗位赋能"

    • P1·场景触发词（权重4·强触发）

    出现1个累积4分即触发：

  • 岗位描述：市场总监、总经理、运营经理、产品经理、销售总监
  • 行业描述：餐饮行业、科技行业、教育行业、金融行业
  • 性格描述：水行人、火行人、木行人、金行人、土行人
  • 任务描述：渠道投放、用户增长、品牌策划、产品运营
  • 指标描述：KPI、OKR、结果指标、过程指标、量化目标
  • 风险控制：风险预警、过程控制、切换机制、ROI

    • P2·信号触发（权重3·弱触发）

    出现≥2个累积6分即触发：

  • 同时提到行业+岗位+任务（六维框架信号）
  • 描述性格特质（五行人格信号）
  • 询问"信息交汇点"（工作流信号）
  • 询问"如何量化目标"（指标设计信号）
  • 询问"用什么模型分析"（模型选择信号）

    🎯 使用流程

    Step 1: 六维信息采集

    输入模板： 
    六维框架:
  行业: [餐饮/科技/教育/金融等]
  企业简介: [规模/阶段/文化特色]
  岗位: [市场总监/总经理/运营经理等]
  性格: [水行人-灵活/火行人-热情/木行人-创新/金行人-目标/土行人-稳定]
  核心任务: [渠道投放/用户增长/品牌策划等]
  适用模型: [GROW/OKR/第一性原理/五色光方法论等]
    示例： 
    六维框架示例:
  行业: 餐饮行业，日料，金枪鱼专门店
  企业简介: 味藏餐饮，专注东方健康烹饪，蓝鳍金枪鱼核心产品
  岗位: 总经理
  性格: 金行人-目标计划
  核心任务: 执行高盈利产品战略目标
  适用模型: 企业经营管理模型

    Step 2: 五行人格识别与适配

    五行-岗位适配表：

    | 五行 | 核心特质 | 适配岗位类型 | 适配任务类型 | |------|----------|--------------|--------------| | 木行人 | 创新、生发、仁德 | 战略规划、产品研发、创意设计 | 0→1突破、长期规划、创新项目 | | 火行人 | 光明、热情、礼明 | 市场推广、团队管理、公关传播 | 品牌传播、团队激励、危机公关 | | 土行人 | 承载、稳定、信实 | 运营管理、客户服务、供应链管理 | 流程优化、服务提升、稳定执行 | | 金行人 | 决断、精准、义理 | 财务管理、数据分析、质量管理 | 决策分析、风险控制、标准制定 | | 水行人 | 智慧、流动、润泽 | 战略规划、人力资源、商务拓展 | 策略制定、人才发展、资源整合 |

    五行适配策略： 
    木行人适配:
  提示词风格: 强调创新、突破、前瞻性
  AI关键词: 洞察、启发、融合
  协作模式: 共创伙伴（第四象限）
  
火行人适配:
  提示词风格: 强调热情、感染力、行动力
  AI关键词: 透视、阐释、推演
  协作模式: 高效助理（第一象限）
  
土行人适配:
  提示词风格: 强调稳定、执行、落地性
  AI关键词: 剖析、解构、溯源
  协作模式: 学习伙伴（第二象限）
  
金行人适配:
  提示词风格: 强调精准、数据、决策性
  AI关键词: 思辨、推演、解构
  协作模式: 共创导师（第三象限）
  
水行人适配:
  提示词风格: 强调灵活、策略、应变性
  AI关键词: 洞察、溯源、映射
  协作模式: 未知探索（第五象限）

    关键动作示例:
  
  目标量化:
    年度目标: 实现高盈利产品战略
    结果指标:
      - 市场份额: 占据30%
      - 品牌影响力: 知名度提升80%
      - 年收入增长: 10%
  
  资源分配:
    预算分配:
      - 产品创新: 50% - 研发新品
      - 供应链优化: 30% - 降低成本
      - 文化体验: 20% - 提升服务
  
  数据验证:
    监控指标:
      - 年销售额增长: 实时监控
      - 净利润率: 月度分析（目标15%）
    风险控制:
      - 产品要符合味藏文化属性
      - 净利润不低于13%
      - 关键岗位连续3月未达标更换

    Step 4: AI增强关键词选择

    十项认知指令：

    | 关键词 | 功能 | 适用场景 | 五行适配 | |--------|------|----------|----------| | 洞察 | 深入理解本质 | 需求分析、问题诊断 | 木行人、水行人 | | 剖析 | 拆解概念要素 | 复杂问题分解 | 金行人、土行人 | | 透视 | 透过现象看本质 | 根因分析、趋势判断 | 火行人、水行人 | | 阐释 | 全面解释展开 | 概念传达、方案说明 | 火行人、土行人 | | 推演 | 逻辑推理预测 | 方案评估、风险预测 | 金行人、水行人 | | 解构 | 拆分为基本元素 | 流程优化、系统重构 | 金行人、木行人 | | 思辨 | 批判性多角度分析 | 决策分析、方案对比 | 金行人、水行人 | | 溯源 | 追根溯源 | 历史分析、根因挖掘 | 水行人、木行人 | | 融合 | 整合不同领域 | 跨界创新、综合方案 | 木行人、火行人 | | 启发 | 开拓新思路 | 创意激发、突破瓶颈 | 木行人、火行人 | | 映射 | 框架结构转化 | 知识迁移、方案套用 | 水行人、土行人 |

    关键词选择策略： 
    初级复杂度（1-2个关键词）:
  适用: 简单任务、明确需求
  示例: [洞察] + [阐释]
  
中级复杂度（3-4个关键词）:
  适用: 中等复杂度、需要分析
  示例: [洞察] + [剖析] + [推演] + [思辨]
  
高级复杂度（5个以上关键词）:
  适用: 复杂任务、系统方案
  示例: [洞察] + [剖析] + [透视] + [推演] + [融合] + [启发]

    Step 5: 提示词框架生成

    黄金公式

    提示词 = 角色 + 任务 + 上下文 + 要求
    示例： 
    角色: 你是一位经验丰富的餐饮行业运营总经理，拥有15年高端日料管理经验
任务: 设计一份高盈利产品的战略方案
上下文: 味藏餐饮，专注东方健康烹饪，蓝鳍金枪鱼核心产品，客单价80元
要求: 
  1. 符合SMART原则
  2. 具有可操作性、易复制性
  3. 包含具体行动项和负责人
  4. 输出格式为Markdown表格

    基础结构技巧

    1. 角色扮演： 
    你是一位[具体角色]，拥有[具体背景]。请以[语气/风格]完成[任务]。

    3. 格式化约束：
