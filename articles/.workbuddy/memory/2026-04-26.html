    核心发现

    1. 开发者模式开启路径： - 菜单栏 → "帮助" → "故障排除" → 启用开发者模式 - 重启后新增"开发者"选项菜单

    2. 第三方模型配置： - 路径："第三方推理配置"界面 - 需填写 Gateway base URL + API Key - 推荐使用 OpenRouter 作为网关 - 配置后自动重启生效

    3. 支持的国产模型： - 腾讯、小米、百度、Kimi、智谱等 - 可自由切换，免费开源模型也可接入

    4. 双模式设计： - Cowork 模式：办公创作场景 - Code 模式：开发场景
