    五、如何安装龙心OS？

    5.1 系统要求

  • 操作系统：Windows 10/11 / macOS / Linux
  • WorkBuddy：已安装并登录
  • 存储空间：建议预留 5GB+ 用于SKILL安装

    • 5.2 安装步骤

    第一步：安装核心SKILL

    打开 WorkBuddy，进入 Skills 市场，搜索并安装：

    1. 龙心OS（核心调度中枢） 2. 五行人格心理学OS（人格智能体系） 3. 人机共生OS（关系定义层） 4. 象思维、知识学习、五色光思维、人机协同五象限、知行合一（五大引擎）

  • 你的姓名和五行人格类型

  • 价值观和信仰体系

    • 第三步：连接知识库
