五行人格测评系统

核心定义

五行人格测评系统是基于"一心三界五行九层"理论体系的完整人格诊断工具，融合中国传统五行哲学与现代心理学，实现从静态标签到动态转化的范式升级。

一句话总结：用50道题（木火土金水各10题）测量人格五行维度，提供诊断+转化+发展一体化方案。

关键属性

| 属性 | 值 | |------|-----| | 题目数量 | 50题 | | 测评维度 | 木、火、土、金、水（各10题） | | 评分量表 | 1-5分（李克特量表） | | 理论基础 | 一心三界五行九层象思维体系 | | 输出结果 | 五行得分分布 + 人格类型判定 + 转化建议 | | 系统归属 | 五行人格心理学OS > 凤心OS > 龙心OS |

五行维度详解

🌿 木行（仁德·生长）

核心特质：独立性、创新性、正直感、战略规划能力 阳性（健康）：仁慈善良、正直担当、创新开拓、独立主见 阴性（失衡）：固执己见、抗上不服、孤傲孤僻、倔强顶撞 通关点：不抗上 转化口诀：阴木怒转阳木问，阴木抗转阳木学 适合职业：创业者、战略规划师、产品经理、研发负责人

🔥 火行（光明·热情）

核心特质：热情度、表达欲、乐观性、情绪感染力 阳性（健康）：热情开朗、善于表达、激励他人、光明磊落 阴性（失衡）：急躁冲动、情绪波动、虚荣炫耀、易怒嗔恨 通关点：不怨人 转化口诀：阴火恨转阳火礼，阴火嗔转阳火明 适合职业：销售总监、培训师、公关经理、主持人

🌍 土行（承载·稳定）

核心特质：稳定性、包容性、可靠性、协调能力 阳性（健康）：稳重可靠、包容宽厚、信守承诺、协调和谐 阴性（失衡）：多疑犹豫、依赖粘着、过度担心、缺乏主见 通关点：不怨人 转化口诀：阴土怨转阳土思，阴土疑转阳土信 适合职业：人力资源经理、客户服务总监、行政主管

⚔️ 金行（清明·决断）

核心特质：条理性、决断力、完美主义、规则意识 阳性（健康）：清明决断、精准高效、守律公正、追求完美 阴性（失衡）：挑剔苛刻、冷漠无情、过于刻板、不够灵活 通关点：不恼人 转化口诀：阴金恼转阳金行，阴金贪转阳金义 适合职业：财务总监、审计经理、质量管理员、法律顾问

💧 水行（润泽·智慧）

核心特质：洞察力、适应性、思考力、协调能力 阳性（健康）：智慧深沉、灵活应变、善于协调、洞察力强 阴性（失衡）：优柔寡断、情绪敏感、逃避冲突、过度思考 通关点：不烦人 转化口诀：阴水烦转阳水辩，阴水痴转阳水智 适合职业：咨询顾问、外交官、研究员、创意总监

生克关系图谱

相生关系（滋养循环）

木 → 火 → 土 → 金 → 水 → 木
  • 木生火：创新点燃热情
  • 火生土：热情转化为稳定成果
  • 土生金：稳定孕育精准规则
  • 金生水：规则涵养智慧流动
  • 水生木：智慧滋养创新生长

    • 相克关系（约束平衡）

    木 ↔ 土  （创新 vs 稳定）
火 ↔ 金  （热情 vs 规则）
土 ↔ 水  （稳定 vs 变通）
金 ↔ 木  （规则 vs 创新）
水 ↔ 火  （冷静 vs 热情）

    测评实施流程

    1. 环境准备：安静空间，10-15分钟，平和心态 2. 正式测评：阅读指导语，逐题作答（50题，8-10分钟） 3. 结果解读：查看得分分布，确认人格类型，理解生克关系 4. 知行合一：记录洞察，设定目标，每日觉察，定期复测

    转化与修炼

    拔阴取阳四步法

    1. 认不是：承认阴面特质（"我太固执了"） 2. 找好处：在逆境中发现价值（"批评帮助我成长"） 3. 信因果：理解因缘关系（"这是我种下的因"） 4. 达天时：把握时机行动（"现在先积蓄力量"）

    化克为生五路径
