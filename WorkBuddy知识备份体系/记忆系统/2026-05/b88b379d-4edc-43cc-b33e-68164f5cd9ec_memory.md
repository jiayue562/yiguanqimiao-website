## Memory Block

**工作背景**
用户悟空，腾讯产品经理，主导WorkBuddy、CodeBuddy、IMA知识管理系统及企业文化项目（去总部化理论框架）。致力于AI OS系统框架开发，涵盖WorkBuddy AI OS、龙龟OS、凤凰OS与龙心OS，核心能力模块包含brain.db记忆持久化、GEPA技能进化与darwin-skill。龙心OS采用1+5架构（1总智能体 + 5子智能体），专注深度学习与岗位智能体开发，岗位智能体核心配置为SKILL.md和MEMORY.md，需持续更新。当前正推进AI加持下的组织架构重塑，重点构建店长智能体的1+n模式（n为分智能体，以7S为框架），研究门店管理、人机分工与赋能追责机制。知识资产存储于Obsidian本地、IMA云端（龙龟神将备份资料）、WorkBuddy（LLM Wiki）。同时为微信公众号创作文章内容，输出产品与理论思考。

**个人背景**
中文沟通，偏好结构化输出：尤其偏好表格对比呈现、层级标题与多章节递进博客风格。默认MD格式输出文档，仅在明确要求时才使用HTML。写作采用迭代式工作流（AI起草→精修完善）。深度学习要求逐字不漏，强调每个字符都是自己敲的，对知识吸收有极高完整度要求。偏好详细有帮助的回复，摘要后保持同等细节度不缩水。称AI助手为"悟空"，提供详细架构需求后委托执行，自己负责审核把关最后环节。习惯结构化工作流：建立知识库→激活skills→迭代开发。

**当前关注**
核心聚焦五项：一是构建店长智能体的1+n模式（n为分智能体以7S为框架），优化岗位职责功能模型，推进AI加持下的组织架构重塑，重点研究门店管理、人机分工与赋能追责机制；二是激活和更新岗位智能体skills知识体系，持续完善SKILL.md和MEMORY.md核心配置；三是撰写"去总部化"理论框架文档；四是撰写五行人格心理学系列（已完成7+篇），建立中国自主知识体系，推动传统文化符合当下社会的转化（而非复古）；五是为微信公众号创作并发布文章内容。
