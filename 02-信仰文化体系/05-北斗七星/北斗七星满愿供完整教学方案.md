## 二、信息体篇

### 第一章：阿底上师瑜伽（纯粹的"知"）

**核心目标**：直接体认并安住于"信息体"——纯粹的觉知。

#### 1.1 信息体的本质

```
┌─────────────────────────────────────────────────────────────┐
│                     信息体：万法之源                           │
├─────────────────────────────────────────────────────────────┤
│  • 空性、本源、纯粹觉知                                       │
│  • 无形无相，但具足一切显现的潜能                             │
│  • 被称为：佛性、道、空性、本觉                               │
└─────────────────────────────────────────────────────────────┘
```

#### 1.2 三步直指明觉

| 步骤 | 操作 | 核心心要 |
|------|------|---------|
| **第一步** | 观想与念诵 | 观想白色"阿"字在五色明点中，长声念诵"阿"，有造作的起点 |
| **第二步** | 彻底放松 | 放下对一切景象和念头的追随，平息心意识造作，转向细微觉知 |
| **第三步** | 反观与安住 | 反观"纯粹的知道本身"，安住于无对境、赤裸的觉性 |

#### 1.3 为何它是首要环节

> **确立正确的"基位"**——只有在信息体的纯粹觉知中，后续仪轨的能量沟通才能真正"接通"并生效。
