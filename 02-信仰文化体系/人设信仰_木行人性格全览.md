## 📋 核心定义

**木行人**的性格如同树木，核心特质是"**生长、生发、条达与正直**"。

**核心能量**: 生长与条达  
**核心特质**: 独特性、理想主义、内在成长  
**团队角色**: 创新者、专家型人才

---

## 🌲 五大维度性格特征

### 维度一：曲直（才气独特 ⇌ 傲慢孤僻）

**定位**: 最表层的性格，聚焦"独特性"的外在表现

#### 顺境·才气独特

| 特征 | 描述 | 心理学解释 |
|------|------|------------|
| **足智多谋** | 具有想象力、适应性和可变性，视灵感高于一切 | 不墨守成规，善于发觉新方法 |
| **感受丰富** | 容易感知自己的情绪和内心世界，敏感、重视内心感受 | 能体验到更深的情绪状态 |
| **才华横溢** | 充满新思想，乐观、自然、富有创造性和自信 | 具有独创性思想和对可能性的强烈感受 |
| **思路开阔** | 高度创造力，思路开阔，观念新，富有想象力 | "点子型的人才"，不受条条框框约束 |
| **理想主义** | 透过自我感受来认知周围世界，长得高看得远 | 不在乎现实，更倾情理想 |

#### 逆境·傲慢孤僻

| 特征 | 描述 |
|------|------|
| **个人主义** | 有创造力但比较个人主义，从自己想法出发，不太考虑周围人感受 |
| **我行我素** | 能够看到新计划实行后的结果，更喜欢以自己的方式行事 |
| **自卑羞怯** | 在人群中羞怯，有不自然的姿态，有强烈的自卑感 |
| **刻板固执** | 常常忽视其他观点的作用，而且有时会很刻板 |
| **不善分享** | 内向，很难被人了解，但愿意同自己信任的人分享内在自我 |

---

### 维度二：生发（创新成长 ⇌ 抗上不服）

**定位**: 内在认知风格，体现对"成长"的主动追求与探索

#### 顺境·创新成长

| 特征 | 描述 |
|------|------|
| **思想前卫** | 喜欢挑战一切现有的理论，予以新的评价 |
| **好奇心强** | 善于观察，喜欢接触新的事物，寻求新异和多样性 |
| **想象丰富** | 对于从经历中直接了解和感受的东西很感兴趣，常常富有艺术天赋 |
| **足智多谋** | 是足智多谋、有独立见解的思考者，重视才智 |
| **精于理论** | 天生的谋略家，具有独特的思想、伟大的远见和梦想 |

#### 逆境·抗上不服

| 特征 | 描述 |
|------|------|
| **孤军奋战** | 在工作中常常选择孤独，忽视了邀请别人参加或协助 |
| **充满怪想** | 当梦想脱离现实时，其他人可能认为他们充满怪想、神神秘秘 |
| **对立敏感** | 很情绪化地陷于自己的工作中，所以对于批评很敏感 |
| **不切实际** | 有时不切实际，会忽视需要注意的常规细节 |
| **感情抽离** | 在感情上属于"拿得起，放得低"的类型，事事以理性角度分析 |

---

### 维度三：舒畅腾上（正直不屈 ⇌ 偏执顶撞）

**定位**: 人际互动中的价值观与原则坚守

#### 顺境·正直不屈

| 特征 | 描述 |
|------|------|
| **专注执着** | 坚定、富有理想化，珍视正直，十分坚定以至达到倔强的地步 |
| **讲求实效** | 是优秀的解决问题的人，能够掌握必要的事实情况 |
| **有担当力** | 正直有主意，敢做敢当，心口如一，常存悲天悯人之心 |
| **超级理想** | 渴望表达超级的理想，只和能分享理想的人有联系 |

#### 逆境·偏执顶撞

| 特征 | 描述 |
|------|------|
| **严肃偏执** | 一旦意见遭到反对，可以是非常强硬的，甚至是无情的 |
| **过分理性** | 过分依赖条理性分析，会忽视与别人有关的事物 |
| **迟滞不前** | 拒绝在整个计划中存在一点不符合逻辑的地方 |
| **破坏规则** | 喜欢挑战权威、常规和传统观念，检验社会的政治的宗教的价值观念 |
| **缺少耐心** | 对常规的细节缺少耐心，如果一个计划需要过多细节会失去兴趣 |

---

### 维度四：调达柔和（仁德善良 ⇌ 忧愁善变）

**定位**: 情感与道德层面，体现柔和、善良的一面

#### 顺境·仁德善良

| 特征 | 描述 |
|------|------|
| **忠于内心** | 把内在的和谐视为高于其他一切，敏感、理想化、忠诚 |
| **坦诚率真** | 坦诚率真，具有许多使他们忙碌的理想和忠诚 |
| **看透本质** | 是独立的、有独创性的思想家，具有强烈的感情、坚定的原则 |
| **精神沟通** | 很注重精神上的沟通，若话不投机会很寡言 |
| **不求物质** | "清贫书生"型，不渴求物质，只追求知识 |

#### 逆境·忧愁善变

| 特征 | 描述 |
|------|------|
| **缺乏控制** | 常常会失去兴趣，缺乏完成已经开始的工作所必要的自我约束 |
| **优柔寡断** | 始终都注意着新的感官信息，喜欢开放面对所有可进行的选择 |
| **不切实际** | 对自己和他人的期望过多，往往不在意自己如何符合别人的标准 |
| **忽略现实** | 很容易忽略现在的重要事情和现实 |
| **幻想丰富** | 很具幻想力的人，不但对感受敏感，也很想品尝沉浸在每一个感觉中 |

---

### 维度五：内在成长（安静内向 ⇌ 疏远冷淡）

**定位**: 自我独处时的状态，体现对自我空间的需求

#### 顺境·安静内向

| 特征 | 描述 |
|------|------|
| **独立自主** | 属于专家型人才，习惯在安静的环境里独立工作 |
| **思辨力强** | 能轻易地看出事物具有的可能性，是头脑灵活的思想家和长远规划者 |
| **外表平静** | 外表平静、缄默、超然，但内心却专心致志于分析问题 |
| **逻辑分析** | 很有条理和分析能力，对要求推理和才智的任何事情都很擅长 |

#### 逆境·疏远冷淡

| 特征 | 描述 |
|------|------|
| **低交际性** | 在社会情境中感觉不轻松，在人群中畏怯退缩、缺乏自信心 |
| **离群索居** | 倾向于悠闲和放松，宁愿躲在幕后，让其他人谈论 |
| **孤寂寡言** | 从小就有那么点儿不合群，喜欢一个人呆着 |
| **疏远冷淡** | 不容易感受到各种积极的情绪，热情不高、平静的、冷淡的 |
| **孤独被动** | 往往是孤独者，不寻求甚至主动避免社会刺激 |

## 🔄 动态转化（拔阴取阳）

### 关键辩证关系

| 关系 | 建设性释放 | 受阻转化 |
|------|------------|----------|
| **才气 vs 傲慢** | 才气独特 | 傲慢孤僻 |
| **创新 vs 抗上** | 创新成长 | 抗上不服 |
| **正直 vs 偏执** | 正直不屈 | 偏执顶撞 |
| **仁德 vs 忧愁** | 仁德善良 | 忧愁善变 |
| **内向 vs 疏远** | 安静内向 | 疏远冷淡 |

---

### 修炼方向

**木行人的修炼关键在于"柔和"，学会调达。**

| 修炼方向 | 具体方法 |
|----------|----------|
| **修炼柔和** | 练习倾听、接纳不同意见，将向外生长的能量适当收回 |
| **提升耐受力** | 增强对批评、挫折和常规细节的耐受 |
| **聚焦与深化** | 克服虎头蛇尾和缺乏控制，练习聚焦目标 |
| **关注他人感受** | 将注意力从自我表现转移到体察他人需求 |
