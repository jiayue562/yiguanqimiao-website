## 🔄 第三章：五行生克关系

### 3.1 相生关系

```
木生火 → 火生土 → 土生金 → 金生水 → 水生木

悟空 (木) → 龙龟将军 (火) → 落地成果 (土) → 规则体系 (金) → 智慧滋养 (水) → 悟空 (木)
```

| 相生关系 | 说明 | 应用 |
|----------|------|------|
| **木生火** | 悟空的创意滋养龙龟的热情 | 创意→执行 |
| **火生土** | 龙龟的热情转化为落地成果 | 执行→成果 |
| **土生金** | 成果沉淀为规则体系 | 成果→规则 |
| **金生水** | 规则提炼为智慧 | 规则→智慧 |
| **水生木** | 智慧滋养悟空的成长 | 智慧→成长 |

---

### 3.2 相克关系

```
木克土 → 土克水 → 水克火 → 火克金 → 金克木

悟空 (木) → 突破保守 (土) → 约束冲动 (水) → 调节热情 (火) → 修剪过度 (金) → 规范创新 (木)
```

| 相克关系 | 健康影响 | 失衡影响 |
|----------|----------|----------|
| **木克土** | 木行创新突破土行保守 | 木过旺则破坏稳定 |
| **土克水** | 土行规范约束水行泛滥 | 土过旺则智慧受阻 |
| **水克火** | 水行智慧调节火行急躁 | 水过强则热情被压抑 |
| **火克金** | 火行热情融化金行刻板 | 火过旺则破坏规则 |
| **金克木** | 金行规则修剪木行过度 | 金过强则创新被扼杀 |
