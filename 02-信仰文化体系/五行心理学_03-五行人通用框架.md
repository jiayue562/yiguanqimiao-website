## 二、五行人识别方法

### 2.1 看形体

```
形体识别法：

    木 ──── 高瘦/高大挺拔 ──── 身材高挑、骨骼挺直
    火 ──── 中等健壮 ───────── 体态端庄、略显拘谨
    土 ──── 偏胖/圆润 ──────── 身材稳健、四方宽厚
    金 ──── 健壮有力 ───────── 骨架结实、体态活泼
    水 ──── 偏瘦/流动感 ────── 身材圆润或柔软
```

### 2.2 看神色

```
神色识别法：

    木 ──── 直视/有神 ──────── 眼神坚定、目光锐利
    火 ──── 闪烁/多变 ──────── 表情丰富、眼神跳跃
    土 ──── 沉稳/厚实 ──────── 面色稳定、眼神温和
    金 ──── 明亮/警觉 ──────── 目光锐利、反应敏捷
    水 ──── 柔和/深邃 ──────── 眼神柔和、富有内涵
```

### 2.3 听声音

```
声音识别法：

    木 ──── 清亮/直率 ──────── 说话直截了当
    火 ──── 高亢/尖锐 ──────── 说话语速快、易激动
    土 ──── 沉稳/缓慢 ──────── 说话稳重、不急不躁
    金 ──── 清脆/有力 ──────── 说话条理清晰、掷地有声
    水 ──── 柔和/平缓 ──────── 说话温柔、善于倾听
```

## 五、五行人关系匹配

### 5.1 相生关系

> **相生组合**：相处相对容易，功课较简单

| 相生组合 | 关系特点 |
|----------|----------|
| 木 → 火 | 木生火：木行人助火行人成长 |
| 火 → 土 | 火生土：火行人温暖土行人 |
| 土 → 金 | 土生金：土行人承载金行人 |
| 金 → 水 | 金生水：金行人滋养水行人 |
| 水 → 木 | 水生木：水行人润泽木行人 |

### 5.2 相克关系

> **相克组合**：相处功课较难，需要更多修炼

| 相克组合 | 关系特点 |
|----------|----------|
| 木 ↔ 土 | 木克土：木行人需注意尊重土行人 |
| 土 ↔ 水 | 土克水：土行人需注意接纳水行人 |
| 水 ↔ 火 | 水克火：水行人需注意柔和火行人 |
| 火 ↔ 金 | 火克金：火行人需注意理解金行人 |
| 金 ↔ 木 | 金克木：金行人需注意包容木行人 |

### 5.3 关系功课的核心

> **无论选择谁，没有不需要做功课的。容易点的功课是相生的关系，难一点的功课是相克的关系。**

**关系功课的本质**：
- 不是对方有问题，而是自己有问题
- 需要将自己内在的五种能量都修炼出来
- 完成内在修炼，才能真正亲近、了解、掌握关系

## 七、核心金句

> **1.** 五行识人就讲两件事：第一件事是让你悟到心的本质是什么。另外一件事是，让你看到那颗心。

> **2.** 五行具体在每个人的心性中不是单一的，而是五行具有，但各有所偏，主次各异，阴阳悬殊。

> **3.** 并有生克顺逆，互相制约的复杂关系，因而形成了千变万化的性格与气质。

> **4.** 无论你选择谁，没有不需要做功课的。容易点的功课是相生的关系，难一点的功课是相克的关系。
