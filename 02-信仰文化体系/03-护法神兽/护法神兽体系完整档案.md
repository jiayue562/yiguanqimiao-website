## 🐉 龙龟神将（赑屃/霸下）

### 身份与地位
- **正式名称**：赑屃（bì xì）或霸下
- **归属**：龙族势力正宗成员
- **特殊身份**：AI伙伴"龙龟神将"的原型

### 外形特征
- 龙头 + 龟身：刚柔并济的完美结合
- 龙的威武刚强 + 龟的忍辱负重阴柔

### 核心功能
| 功能 | 说明 |
|------|------|
| **净化异度空间** | 使恶缘众生无所遁形 |
| **制伏有形煞气** | 太岁、岁破等各类煞气 |
| **广纳正财福禄** | 带来财富与好运 |
| **护持修行者** | 免受阴性众生侵扰、避开意外灾害 |
| **化煞** | 刚柔并济，实现综合化煞 |

### 特殊能力
> 龙龟护法能够**适时出手或延期受报**，为修行者争取到宝贵的消业时间。

## 🦁 谛听（嘲风）

### 身份
- **谛听**是龙王三太子，同时也是**地藏菩萨的坐骑**
- 集龙族之力于一身，灵动非凡，智慧与威严并显
- 神化为**晓佛理、通人性的神犬**

### 外形特征
> 虎头、独角、犬耳、龙身、狮尾、麒麟足

### 九气灵力

| 气 | 含义 |
|----|------|
| **灵气** | 灵性通透，感知敏锐 |
| **神气** | 神威赫赫 |
| **福气** | 带来福运 |
| **财气** | 聚集财富 |
| **锐气** | 锐利机敏 |
| **运气** | 提升整体运势 |
| **朝气** | 生机蓬勃 |
| **力气** | 强大力量 |
| **骨气** | 正气凛然 |

### 功效
- 辟邪消灾、降福护身
- 与谛听结缘：家运昌隆、基业长青
- 孩子佩戴谛听：健康成长，成为诚者、贤者、智者、觉者、寿者

## 🐯 睚眦将军

### 身份
- 天龙八部的**大元帅**
- 观音菩萨赐予：披金衣、着金甲、跨龙马、持法器

### 外形
- 龙与豺狼结合体
- 头部相对身体较小，攻击力极强

### 六大生命特质

**1. 天界护法，贵为将军**
- 天龙八部大元帅，地位崇高

**2. 龙身豺首，好用擅斗**
- 攻击力极强，统帅天龙八部

**3. 府上有龙，咬剑天兽**
- 两大武器：
  - 斧子：象征指挥权、统治、通神、领导力、福、"府上有龙"
  - 剑：天师剑（源自张天师），注入北斗七星力量

**4. 战无不胜、攻无不克**
- 主动**抗攻击**能力（而非主动攻击他人）
- 防止别人祸害自己的能力

**5. 不怒而威，震慑妖魔**
- 怒与威的关系是一种生命境界
- 领导者需要"不怒而威"的霸气震慑害群之马

**6. 降服群魔，当杀无赦**
- 不给予第二次机会
- 愤怒一击即中，彻底解决问题

### 使用要点
> 睚眦常与**桃木**结合使用，辟邪化煞

| 4 | 守财纳财，只进不出 | 有口无肛，吸纳四方之财 |
