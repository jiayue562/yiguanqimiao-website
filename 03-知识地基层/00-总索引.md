## 🎯 使用场景

### 场景 1: 新知识分类
```
1. 输入新知识（文章/PDF）
2. 判断五行属性（取象比类）
3. 归入三层结构（能量/时空/实践）
4. 建立双向链接（同属性/相生/相克）
5. 更新知识图谱
```

### 场景 2: 隐秘知识发现
```
1. 查看知识图谱
2. 识别隐性关联（看似无关的节点）
3. 通过五行属性找到共鸣点
4. 建立新的双向链接
5. 涌现创新洞察
```

### 场景 3: 知识自主成长
```
1. 定期运行健康检查
2. 识别孤立节点
3. 通过五行生克找到关联
4. 修复网络平衡
5. 知识网络持续进化
```
