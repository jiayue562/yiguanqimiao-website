# 📜 系统论——系统科学哲学 · 总索引

> **S（情境）**：在当代知识体系中，系统科学已成为横跨自然科学、社会科学和工程技术的基础范式，但缺少一个统摄性的哲学框架。
> **C（冲突）**：老三论（系统论、信息论、控制论）、新三论（耗散结构论、协同学、超循环理论）各自为政，概念混乱，缺乏"一论统摄"。
> **Q（疑问）**：如何建立系统科学的哲学体系，实现"三论归一"？
> **A（回答）**：钱学森指出"根本只有一论——系统论"，魏宏森、曾国屏在钱老指导下，建立了八条原理+五条规律+五大系统观的完整系统论哲学体系。

## 四、核心概念速查

| 概念 | 定义 | 出处 | 文档 |
|------|------|------|------|
| **系统** | 由相互联系、相互作用的要素组成的具有一定结构和功能的有机整体 | 贝塔朗菲 | C |
| **整体涌现性** | 整体具有而部分或部分之和所没有的性质 | 整体性原理 | C |
| **开放系统** | 与环境持续交换物质、能量、信息的系统 | 贝塔朗菲 | C |
| **自组织** | 系统在无外部指令下自发从无序走向有序 | 普利高津/哈肯 | C |
| **耗散结构** | 远离平衡态的开放系统中通过能量耗散维持的有序结构 | 普利高津 | C |
| **序参量** | 协同学中支配系统宏观模式的变量 | 哈肯 | D |
| **等终局性** | 不同初始条件可通过不同途径达到相同终态 | 贝塔朗菲 | C |
| **负熵** | 系统从环境引入的秩序，抵消内部熵增 | 薛定谔 | C |
| **三论归一** | 信息论、控制论归入系统论，唯有一论 | 钱学森 | A |
| **分叉** | 系统在临界点面临多种演化路径的节点 | 突变论 | C |
