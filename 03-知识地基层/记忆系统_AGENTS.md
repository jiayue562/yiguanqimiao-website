## 🔄 智能体调度规则

### 任务分配原则
1. **简单任务**: 直接执行，不调用子智能体
2. **复杂任务**: 分解为子任务，调度相应子智能体
3. **探索任务**: 使用 code-explorer 进行大规模搜索
4. **开发任务**: 使用 bmad-dev + bmad-architect 组合

### 协同工作流
```
用户需求 → 龙心OS 分析 → 引擎调度 → 子智能体执行 → 结果整合 → 知行沉淀
```
