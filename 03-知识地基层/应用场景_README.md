## 场景详解

### 亲密关系

**应用场景**：伴侣关系、婚姻关系、恋爱咨询

**核心应用**：
- 五行相生相克在亲密关系中的体现
- 不同五行类型的伴侣相处模式
- 冲突化解与关系修复
- 亲密关系中的拔阴取阳

**入口**：[[亲密关系/亲密关系]]

### 亲子关系

**应用场景**：亲子教育、孩子成长、家庭和谐

**核心应用**：
- 识别孩子的五行类型
- 因材施教的教育方法
- 亲子沟通的五色光应用
- 家庭五行能量平衡

**入口**：[[亲子关系/亲子关系]]
