# 五行分类图谱·A层深度解析
## 能量核心与精神原象——五行的本体论维度

> **A层定义**：A层是五行体系的"源代码"，探讨五行作为纯粹能量形态和意识原型的内在特性。它是最抽象、也是最根本的层面，决定了后续所有推演与分类的内在逻辑。
>
> **A层核心**：五性（A1）→ 五脏神（A2）→ 五德（A3）→ 五瑞灵（A4）→ 五心（A5）→ 五空（A6）

---

## A1 五性：功能的根本定义

> **核心来源**：《尚书·洪范》
>
> **五性定义**：不是物质，而是五种功能趋势、运动能势

### A1.1 木曰曲直

| 维度 | 内容 |
|------|------|
| **字面义** | "曲"指能弯曲，"直"指能伸直 |
| **自然象** | 树木生长时能屈能伸、向上向外舒展的生命力 |
| **抽象核心** | 生长、升发、条达、舒畅的运动能势 |
| **能量特征** | 万物从萌动到舒展的原始动力 |
| **五行定位** | 木行人的核心特质：开创力、引领性 |

**引申内涵**：
- 曲直 = 刚柔并济 = 能屈能伸
- 条达 = 通畅无阻 = 肝胆疏泄
- 生发 = 从无到有 = 创新创造的源头

**双向链接**：
- → A2.1 木藏魂
- → A3.1 木主仁德
- → A4.1 木-青龙
- → B3.1 木-东方
- → B4.1 春-木-生

### A1.2 火曰炎上

| 维度 | 内容 |
|------|------|
| **字面义** | "炎"指燃烧的光热，"上"指向上腾升 |
| **自然象** | 火焰升腾、光热四射 |
| **抽象核心** | 温热、升腾、明亮、繁盛的能势 |
| **能量特征** | 能量释放、显现与达到顶峰的阶段 |
| **五行定位** | 火行人的核心特质：热情、光明、照亮 |

**引申内涵**：
- 炎上 = 光热发散 = 照亮他人
- 升腾 = 积极向上 = 追求卓越
- 明亮 = 洞察本质 = 智慧显现

**双向链接**：
- → A2.2 火藏神
- → A3.2 火主礼德
- → A4.2 火-朱雀
- → B3.2 火-南方
- → B4.2 夏-火-长

|------|------|
| **字面义** | "稼"为播种，"穑"为收获 |

| **能量特征** | 转化与运化的平台，能量升降的枢纽 |
| **五行定位** | 土行人的核心特质：承载、稳定、化育 |

**引申内涵**：
- 稼穑 = 播种收获 = 生生不息
- 承载 = 包容万物 = 厚德载物
- 化生 = 转化孕育 = 成就他人

**双向链接**：
- → A2.3 土藏意
- → A3.3 土主信德
- → A4.3 土-麒麟
- → B3.3 土-中央
- → B4.3 长夏-土-化

### A1.4 金曰从革

| 维度 | 内容 |
|------|------|
| **字面义** | "从"为顺从，"革"为变革、革新 |

- → A4.4 金-白虎
- → B3.4 金-西方
- → B4.4 秋-金-收

- → A2.5 水藏志
- → A3.5 水主智德

## A2 五脏神：内在的精神主宰

> **核心来源**：中医与道家养生体系
>
> **五脏神定义**：五脏不仅是有形的器官，更是五种精神能量的居所

### A2.1 木藏魂

| 维度 | 内容 |
|------|------|
| **魂的定义** | 随神往来、主司谋虑、想象与梦境的精神活动 |
| **肝木关联** | 肝木的条达与否，直接关系到"魂"的安宁与飞扬 |
| **失衡表现** | 肝气郁结 → 魂不守舍 → 多梦、失眠、焦虑 |
| **调适方法** | 疏肝理气（如舒肝功、柴胡疏肝散） |

**引申内涵**：
- 魂 = 创造力、想象力的精神基础
- 肝木条达 = 魂有所归 = 思维清晰
- 谋虑 = 规划未来的能力

### A2.2 火藏神

| 维度 | 内容 |
|------|------|
| **神的定义** | 狭义的心神，主宰意识、情感、思维明辨 |
| **心火关联** | 心火的温煦与清明，是"神"得以明澈的基础 |
| **失衡表现** | 心火亢盛 → 神不守舍 → 心悸、失眠、多梦 |
| **调适方法** | 清心泻火（如莲子心茶、黄连解毒汤） |

**引申内涵**：
- 神 = 意识清明、情感稳定的精神核心
- 心火清明 = 神有所主 = 情绪平和
- 明辨 = 洞察本质的智慧

### A2.3 土藏意

| 维度 | 内容 |
|------|------|
| **意的定义** | 思虑、记忆与专注力的精神活动 |
| **脾土关联** | 脾土的运化功能，支撑着"意"的持续与稳定 |
| **失衡表现** | 脾虚失运 → 意不专注 → 健忘、思维迟钝 |
| **调适方法** | 健脾益气（如四君子汤、山药粥） |

**引申内涵**：
- 意 = 思考、记忆、专注的认知能力
- 脾土健运 = 意有所养 = 思维专注
- 思虑 = 分析决策的智力基础

### A2.4 金藏魄

| 维度 | 内容 |
|------|------|
| **魄的定义** | 与形俱来、主司本能动作、感知和体魄的精神 |
| **肺金关联** | 肺金的收敛与肃降，赋予"魄"以坚韧与边界 |
| **失衡表现** | 肺气虚损 → 魄力不足 → 体弱多病、意志薄弱 |
| **调适方法** | 补肺益气（如黄芪、百合银耳羹） |

**引申内涵**：
- 魄 = 本能反应、行动魄力的生理基础
- 肺金肃降 = 魄有所养 = 体魄强健
- 感知 = 对外界刺激的敏锐反应

### A2.5 水藏志

| 维度 | 内容 |
|------|------|
| **志的定义** | 志向、意志力与深层记忆（肾主藏精，精生髓通于脑） |
| **肾水关联** | 肾水的封藏与涵养，是"志"得以坚定深远的源泉 |
| **失衡表现** | 肾精亏虚 → 志不坚定 → 意志薄弱、健忘、腰膝酸软 |
| **调适方法** | 补肾填精（如六味地黄丸、黑芝麻丸） |

**引申内涵**：
- 志 = 远大志向、坚定意志的深层动力
- 肾水充盈 = 志有所养 = 志向远大
- 志向 = 人生方向与使命感的来源

### A2 五脏神·总结

| 五行 | 脏腑 | 所藏 | 核心功能 | 失衡表现 | 调适原则 |
|------|------|------|---------|---------|---------|
| 木 | 肝 | 魂 | 谋虑、想象、梦境 | 多梦失眠焦虑 | 疏肝理气 |
| 火 | 心 | 神 | 意识、情感、思维 | 心悸失眠多梦 | 清心泻火 |
| 土 | 脾 | 意 | 思虑、记忆、专注 | 健忘思维迟钝 | 健脾益气 |
| 金 | 肺 | 魄 | 本能、感知、体魄 | 体弱意志薄弱 | 补肺益气 |
| 水 | 肾 | 志 | 志向、意志、深记 | 意志薄弱健忘 | 补肾填精 |

> **记忆锚点**：肝藏魂主谋虑，心藏神主明辨，脾藏意主思记，肺藏魄主体魄，肾藏志主意志

## A5 五心：修心的能量指向

> **核心来源**：儒释道修心理论
>
> **五心定义**：将内在德性转化为具体心性修持的指导

### A5.1 慈悲心（属木）

| 维度 | 内容 |
|------|------|
| **心性内涵** | 对众生苦难的感同身受，发愿救度 |

## A层核心金句

### A1 五性核心金句

1. **"木曰曲直"不是树木，而是生长升发的生命力。**
2. **"火曰炎上"不是火焰，而是温热升腾的光明势能。**
3. **"土爰稼穑"不是土地，而是承载化生的宇宙平台。**
4. **"金曰从革"不是金属，而是肃杀收敛的清肃之力。**
5. **"水曰润下"不是水流，而是润下闭藏的涵养之道。**

### A2 五脏神核心金句

1. **"肝藏魂"——魂是创造力的精神基础。**
2. **"心藏神"——神是意识清明的心灵核心。**
3. **"脾藏意"——意是思虑专注的认知能力。**
4. **"肺藏魄"——魄是行动魄力的生理基础。**
5. **"肾藏志"——志是志向坚定的深层动力。**

### A3 五德核心金句

1. **"木主仁"——仁者爱人，如木之生机勃发。**
2. **"火主礼"——礼者明理，如火之照破黑暗。**
3. **"土主信"——信者诚实，如土之厚德载物。**
4. **"金主义"——义者宜也，如金之清肃果断。**
5. **"水主智"——智者不惑，如水之深流智慧。**

### A4 五瑞灵核心金句

1. **青龙主东方——春天生机的宇宙象征。**
2. **朱雀主南方——光明文明的星空表达。**
3. **麒麟主中央——中和化育的祥瑞代表。**
4. **白虎主西方——肃杀威严的庚金之灵。**
5. **玄武主北方——藏匿智慧的龟蛇合体。**

### A5 五心核心金句

1. **慈悲心是仁德的升华——化解嗔恨的甘露。**
2. **恭敬心是礼德的升华——驱散傲慢的阳光。**
3. **感恩心是信德的升华——转化怨尤的妙药。**
4. **惭愧心是义德的升华——清洗贪婪的清泉。**
5. **谦卑心是智德的升华——熄灭愚痴的明灯。**

### A6 五空核心金句

1. **地水火风空——五大揭示世界构成的根本。**
2. **土水火木金——五行描绘能量运行的轨迹。**
3. **五大与五行——人类对宇宙本质的共同探索。**
