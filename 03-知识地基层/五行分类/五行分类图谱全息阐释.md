## 🔄 五行生克制化关系

### 相生关系（正向链接）

**循环链**: 木→火→土→金→水→木（如环无端）

| 相生 | 自然象义 | 知识关联 |
|------|----------|----------|
| 木生火 | 木之生长为火提供燃料 | 仁德衍生慈悲心 |
| 火生土 | 火之能量释放化为灰土 | 礼德支撑信德 |
| 土生金 | 土之凝聚沉淀生成金属 | 信德孕育义德 |
| 金生水 | 金之沉降凝聚水汽 | 义德滋养智德 |
| 水生木 | 水之滋润涵养助力生长 | 智德支持仁德 |

**双向特性**: 每一组相生关系都是双向的
```
木是火之源头，火是木之展现
火是土之成因，土是火之归宿
```

---

### 相克关系（反向链接）

**循环链**: 木→土→水→火→金→木（闭环制约）

| 相克 | 自然象义 | 知识关联 |
|------|----------|----------|
| 木克土 | 木之疏泄克制土之壅滞 | 创新制约保守 |
| 土克水 | 土之承载克制水之泛滥 | 规范制约散漫 |
| 水克火 | 水之寒凉克制火之炎上 | 冷静制约冲动 |
| 火克金 | 火之温热克制金之肃杀 | 热情制约冷漠 |
| 金克木 | 金之收敛克制木之升发 | 规则制约发散 |

**双向特性**: 每一组相克关系都是相互制衡的
```
木制约土，土反哺木
土制约水，水滋养土
```

## 🧠 象思维认知基础

### 象思维 vs 逻辑思维

| 维度 | 逻辑思维（西方） | 象思维（东方） |
|------|------------------|----------------|
| 认知核心 | 定义、判断、推理 | 取象、比类、推演 |
| 知识观 | 独立节点、线性因果 | 功能态势、关联共振 |
| 分类方式 | 精准界定、层级分类 | 取象比类、网状关联 |
| 关联规则 | 单向因果 | 双向生克 |
| 认知目标 | 碎片化精准 | 系统化整体 |

### 取象比类方法

**核心步骤**:
1. **观象**: 观察事物的功能态势、运动趋向
2. **取象**: 提取核心特征（生长/温热/承载/沉降/寒凉）
3. **比类**: 将相似特征的事物归入同一五行
4. **推演**: 通过五行生克推演关联

**示例**:
```
肝脏 → 主疏泄、主升发 → 木之象
眼睛 → 肝开窍于目 → 木之象
绿色 → 生机盎然 → 木之象
春天 → 万物生长 → 木之象
东方 → 日出之地 → 木之象
青龙 → 东方神兽 → 木之象
仁德 → 慈爱生长 → 木之象

所有属木节点天然关联，形成木属性知识子网络
```

## 📊 知识自主成长机制

### 机制 1: 天然关联
```
任何事物纳入五行分类 → 自动与体系内所有事物形成生克关联
无需额外建立链接 → 知识网络自主生长
```

### 机制 2: 动态平衡
```
相生提供正向支撑 → 知识衍生与扩展
相克提供反向约束 → 知识平衡与规范
相乘相侮预警 → 知识网络失衡修复
```

### 机制 3: 触类旁通
```
认知一个节点 → 通过五行关联推演至所有同属性节点
例如：认知"春天"→ 推演至"东方""角音""酸味""怒气""肝"
```
