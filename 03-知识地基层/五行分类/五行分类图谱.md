## 🎯 五行系统总览

**五行**不是五种"物质"，而是五种**动态的功能模式**和**关系类型**。

| 五行 | 核心功能 | 运动特征 | 季节 | 方位 | 颜色 |
|------|----------|----------|------|------|------|
| **木** | 生发、舒展 | 向上向外 | 春 | 东 | 青/绿 |
| **火** | 炎上、光明 | 向上发散 | 夏 | 南 | 赤/红 |
| **土** | 承载、化育 | 中和稳定 | 长夏 | 中 | 黄 |
| **金** | 收敛、肃降 | 向下向内 | 秋 | 西 | 白 |
| **水** | 润下、藏纳 | 向下潜藏 | 冬 | 北 | 黑 |

## 🔄 五行生克关系

### 相生循环（生成关系）

```
木 → 火 → 土 → 金 → 水 → 木（循环）
│    │    │    │    │
│    │    │    │    └─ 水生木（滋养生长）
│    │    │    └────── 金生水（凝结成液）
│    │    └─────────── 土生金（矿藏于土）
│    └──────────────── 火生土（灰烬成土）
└───────────────────── 木生火（燃烧生火）
```

**相生含义**：
- **水生木**：水滋养树木生长
- **木生火**：木材燃烧产生火
- **火生土**：火烧尽后成灰土
- **土生金**：金属矿藏于土中
- **金生水**：金属表面凝结水珠（或金属熔化成液态）

### 相克循环（制约关系）

```
木 → 土 → 水 → 火 → 金 → 木（循环）
│    │    │    │    │
│    │    │    │    └─ 金克木（金属砍伐树木）
│    │    │    └────── 火克金（烈火熔化金属）
│    │    └─────────── 水克火（水能灭火）
│    └──────────────── 土克水（土能挡水）
└───────────────────── 木克土（树根穿透土壤）
```

**相克含义**：
- **金克木**：金属工具可以砍伐树木
- **火克金**：烈火可以熔化金属
- **水克火**：水可以灭火
- **土克水**：土可以阻挡水流
- **木克土**：树根可以穿透土壤

### 生克平衡原则

**核心智慧**：
> "生中有克，克中有生，生克制化，动态平衡。"

**过度相生**：
- 母旺子衰：生者过强，被生者反而受损
- 例：水太多，木反而烂根

**过度相克**：
- 克者过强：被克者受损
- 例：金太强，木被过度砍伐

**理想状态**：
- 生克有度，动态平衡
- 生而不溺，克而不伤

## 🌳 木行系统

### 核心特征

| 维度 | 特征 |
|------|------|
| **功能** | 生发、舒展、向上生长 |
| **气质** | 柔和中带坚韧，能屈能伸 |
| **方向** | 向上、向外扩展 |
| **时间** | 春季、清晨、少年期 |
| **状态** | 萌芽、生长、发展 |

### 自然象

| 类别 | 象 |
|------|-----|
| 自然 | 树木、森林、花草、藤蔓 |
| 天象 | 风、云气升腾 |
| 地理 | 东方、山林、植被茂盛之地 |
| 动物 | 龙、蛇、鸟类（飞翔） |

### 人体象

| 类别 | 象 |
|------|-----|
| 脏腑 | 肝、胆 |
| 五官 | 目 |
| 情志 | 怒（过度则伤肝） |
| 功能 | 疏泄、藏血、主筋 |

### 人事象

| 领域 | 象 |
|------|-----|
| 性格 | 仁慈、温和、有创造力、易怒 |
| 职业 | 教育、艺术、设计、创业 |
| 行为 | 规划、启动、拓展、创新 |
| 关系 | 父子、师徒（传承生长） |

### 木行失衡表现

| 类型 | 表现 |
|------|------|
| **木过旺** | 易怒、冲动、过度扩张、肝火旺 |
| **木过弱** | 缺乏主见、优柔寡断、生长乏力 |
| **木被克** | 压抑、抑郁、创造力受阻 |

### 木行应用

**个人成长**：
- 🌱 春季适合启动新项目
- 🌱 清晨适合规划一天
- 🌱 少年期注重兴趣培养

**组织管理**：
- 🌲 创新部门需要木行能量
- 🌲 创业团队需要生发之力
- 🌲 文化建设需要柔和滋养

**决策智慧**：
- 🌳 生发时机：春季、清晨、项目初期
- 🌳 生发方向：向东、向上、向外
- 🌳 生发方法：柔和坚持，能屈能伸

## 🎯 五行应用框架

### 个人五行诊断

**步骤 1：观象**
- 观察自己的气质、行为模式、偏好
- 识别主导的五行能量

**步骤 2：平衡分析**
- 哪一行过旺？
- 哪一行过弱？
- 哪一行被克？

**步骤 3：调整策略**
- 旺者泄之（如木旺用火泄）
