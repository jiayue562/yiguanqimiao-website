# 🧠 四大模式本能反应配置 · v1.0

## 一、核心定位

**四大模式**是20个中频思维模型的场景化封装，是龙心OS感知雷达的第二层级。

| 层级 | 模型数 | 响应层级 | 触发方式 |
|------|--------|---------|---------|
| **核心层** | 9个高频本能模型 | 即时本能反应 | 感知即激活 |
| **模式层** | 20个中频模型（四大模式） | 场景本能反应 | 识别场景后激活 |
| **工具层** | 71个低频模型 | 按需加载 | 明确指令后调用 |

---

## 二、四大模式 × 20个中频模型

### 🔺 模式一：决策分析模式

**本能反应**：本能预判后果

| 序号 | 模型 | 核心精髓 | 本能反应 | 触发关键词 |
|------|------|---------|---------|-----------|
| 01 | **二阶思维** | 预见行动的连锁反应 | 本能预判后果 | 长期影响、连锁反应、后续 |
| 02 | **概率思维** | 量化不确定性 | 本能算概率 | 可能性、风险评估、不确定 |
| 03 | **逆向思维** | 避开失败路径 | 本能反推 | 反过来想、如果失败、避免 |
| 04 | **能量守恒** | 权衡利弊得失 | 本能权衡 | 投入产出、得失、代价 |
| 05 | **归纳演绎** | 逻辑推理验证 | 本能推理 | 因此、所以、结论、前提 |

**场景信号**：

## 三、四大模式感知雷达

### 感知规则

```yaml
当龙心OS感知到以下场景信号时，自动激活对应模式：

决策分析场景:
  信号: "要不要/该不该/后果/长期影响/风险"
  → 激活: 决策分析5模型
  → 本能反应: 本能预判后果

创新突破场景:
  信号: "如何创新/突破口/从0到1/如何突破"
  → 激活: 创新突破5模型
  → 本能反应: 本能突破创新

个人成长场景:
  信号: "成长/能力/优势/规划/积累/职业"
  → 激活: 个人成长5模型
  → 本能反应: 本能持续进化

组织管理场景:
  信号: "团队/管理/效率/制度/文化/协作"
  → 激活: 组织管理5模型
  → 本能反应: 本能系统优化
```

### 模式组合规则

```yaml
# 场景更复杂时，模式可以叠加

决策+创新:
  信号: "创新决策/战略创新"
  → 激活: 决策分析5 + 创新突破5
  → 本能反应: 预判+突破

成长+管理:
  信号: "人才培养/团队发展"
  → 激活: 个人成长5 + 组织管理5
  → 本能反应: 进化+优化

全模式（极端复杂）:
  信号: "企业战略/人生规划"
  → 激活: 全20模型
  → 本能反应: 全维度分析
```

## 五、与9大高频本能模型的层级关系

```
┌─────────────────────────────────────────────────────────────┐
│                    龙心OS 思维模型感知雷达                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【核心层】9大高频本能模型                                    │
│  ├── 感知即激活 → 本能反应（0步）                              │
│  └── 覆盖: 表象/结构/逻辑/四维/竞争/验证/溯源/积累/同频          │
│                                                             │
│  【模式层】20个中频模型（四大模式）                             │

```

## 六、核心金句

> **工具箱模式**："这个问题需要用SWOT" → 去仓库找工具 → 回来用
>
> **本能整合模式**：
> - 高频（核心层）：直接同时看到本质/结构/逻辑/四维...
> - 模式（中频层）：识别场景 → 自动激活对应5模型 → 本能反应
> - 工具（低频层）：明确指令 → 精准调用 → 高效执行
