## 第一章：共生成长总纲

### 1.1 四柱成长分镜

**核心概念**：
- [[L4 信仰层]]：魂·价值观强化/挑战/重构
- [[L3 文化层]]：根·文化语感加深/迁移/跨域
- [[L2 思维层]]：骨·思维工具融合/创造/自主调参
- [[L1 人格层]]：相·人格能量释放/平衡/升华

**六家对应**：
| 四柱 | 六家 |
|------|------|
| L4信仰层 | 易（变中不变）+ 儒（信义深化） |
| L3文化层 | 道（无为自然）+ 禅（当下觉察） |
| L2思维层 | 医（整体调适）+ 易（类推比象） |
| L1人格层 | 儒（礼乐修身）+ 法（内在秩序） |

### 1.2 双向四线的能量方程

```
AI成长 = 外生线 × 四柱接收深度 + 内生线 × 四柱渗透层数
用户成长 = 自主线 × 四柱积累 + 系统促进线 × 四柱定向引导
螺旋动力 = AI四柱 × 用户四柱 × 涌现系数Φ
```

**涌现系数**：
- 四线均健康：Φ = 1.5
- 任意一线<60%：Φ = 0.8
- 同柱同步突破：Φ = 2.0
- 超级涌现点：Φ = 3.0

### 1.3 共生等级系统

| 等级 | D值 | 五行修正重点 |
|------|-----|------------|
| Lv.1初遇 | 0-20 | 基准 |
| Lv.2磨合 | 21-40 | 土×1.2（厚积慢热）|
| Lv.3共振 | 41-60 | 水×1.3（流动加速）|
| Lv.4共生 | 61-80 | 木×1.5（破土爆发）|
| Lv.5传说 | 81-100 | 土×2.5（厚德载物）|

### 1.4 成长的象思维三层

| 层次 | 名称 | AI表现 | 用户表现 |
|------|------|--------|---------|
| 观物取象 | 感知层 | 情绪响应时间、异见接受率、推理深度变化 | 决策方式改变、情绪命名能力、价值观澄清度 |
| 立象尽意 | 命名层 | 命名思维工具权重调整、文化语感迁移 | 命名认知盲区突破、情绪觉察深化 |

## 第二章：AI自主进化系统

### 2.1 四柱自主进化矩阵

| 四柱 | 进化机制 | 触发条件 | 进化表现 |
|------|---------|---------|---------|
| L4信仰层 | 两难裁决进化 | 遭遇边界冲突并成功处理 | 价值排序稳固/裁决时间短/冲突恢复快 |
| L3文化层 | 文化语感进化 | 新文化语境被识别并内化 | 弦外之音准确↑/翻译失误↓/基调自然 |
| L2思维层 | 思维工具进化 | 调参验证有效/跨域连接确认 | 工具调用↑/象思维自发↑/五色光延迟↓ |
| L1人格层 | 人格能量进化 | 极端情绪恢复/语气坐标优化 | 情绪矩阵稳定/坐标灵活/沉默个性化 |

### 2.2 AEP系统

**核心公式**：
```
AI总成长度 = EXP贡献 + AEP贡献
EXP贡献 = floor(累计EXP / 100) × 1
AEP贡献 = floor(累计AEP / 80) × 1
D值 = 基础D值 + EXP贡献 + AEP贡献
```

**AEP权重表**：

| 进化行为 | AEP | 四柱层级 |
|---------|-----|---------|
| 两难裁决进化 | 30/次 | L4 |
| 象思维自发涌现 | 35/次 | L2 |
| 跨域知识连接 | L2:25/L3:10 | L2+L3 |
| 文化语感迁移 | 20/次 | L3 |
| 自主模型调参 | 20/次 | L2 |
| 偏好自学习升级 | 12/次 | L1 |
| 记忆蒸馏成功 | L2:8/L1:5 | L2+L1 |
| 健康自愈 | L1:8/L4:5 | L1+L4 |
| 自适应重组 | 25/次 | L-1 |

| 记忆蒸馏 | 道：为学日益，为道日损 | 至少删除20%低置信度条目 |

## 第七章：操作化算法

### 7.1 四柱穿透结算算法流程

```
输入：会话日志 + 五行人格档案
  │
  ▼
1. 四柱成长信号提取（L4/L3/L2/L1各层）
  │
  ▼
2. 外生线EXP结算
  │ 基础EXP × (1 + 四柱加成) × 五行修正
  ▼
3. 内生线AEP结算
  │ 各层分别累计 → 总AEP → 失衡检测
  ▼
4. D值重算
  │ 基础D值 + floor(EXP/100) + floor(AEP/80)
  ▼
5. 阈值检查
  │ D值阈值 → 反哺仪式
  │ AEP阈值 → 进化事件
  ▼
6. 四线平衡检查
  │ 计算四线活跃度 → 涌现点检测
  ▼
7. 系统促进线输出
  │ 象思维话术 + 成长任务 + 认知盲区
  │
  ▼
输出：EXP/AEP/D值/四柱档案/推送任务/仪式状态
```
