## 一，核心定义

### 1.1 一句话定义

> 框架思维模型库是一套「从工具使用者到认知架构师」的进化体系，包含框架（地图）、思维框架（肌肉记忆）、框架思维（向导视角）三层结构，以及系统思维、逻辑思维、结构化思维、批判性思维、创造性思维五大思维模式。

### 1.2 核心公式

```
问题解决进化 = （思维模式 + 方法论）× 真实实践 × 持续反思
```

### 1.3 三轴定位

| 轴线 | 定义 | 回答问题 |
|------|------|----------|
| **对象轴（结构线）** | 认知的内容之维 | 我们认识什么、如何认识、如何检验 |
| **工具轴（框架线）** | 认知的方式之维 | 我们用什么来认识、内化成什么、如何驾驭工具 |
| **进化轴（闭环）** | 认知的螺旋上升 | 内化→觉醒→创造→新框架→螺旋上升 |

## 二，框架线三层结构

### 2.1 第一层：框架（地图）

**定义**：解决问题时可以直接套用的标准化步骤

**典型案例**：四层次原因分析法
```
第一层：拆解问题
第二层：直接原因
第三层：流程/系统原因
第四层：人的原因
```

### 2.2 第二层：思维框架（肌肉记忆）

**定义**：反复使用框架后形成的下意识思考方式

**形成路径**：反复使用某个框架 → 大脑神经通路被强化 → 成为默认思考模式 → 形成思维框架

### 2.3 第三层：框架思维（向导视角）

**定义**：主动选择、调整、甚至创造最合适框架的元能力

**三问决策**：
1. 「这个问题属于什么类型？该用哪种横向拆解框架？」
2. 「这个框架适合当前场景吗？需要裁剪或组合吗？」
3. 「如果没有现成框架，我能不能快速创造一个？」

## 三，五大思维模式

| 思维模式 | 比喻 | 核心功能 | 解决的问题 |
|----------|------|----------|------------|
| **系统思维** | 导航仪 | 看整体、看关联、看动态 | 横向展开、纵向深挖 |
| **逻辑思维** | 方向盘与刹车 | 概念清晰、判断准确、推理有效 | 拆不乱、推不错 |
| **结构化思维** | 驾驶操作手册 | 化繁为简、变乱为序 | 搭骨架、钻下去 |
| **批判性思维** | 质检员 | 质疑假设、检验证据 | 防止偏见、验证假设 |
| **创造性思维** | 突破者 | 打破框架、发现新可能 | 突破路径依赖、创新解决方案 |

## 五，核心金句

> **框架是「凝结的实践」**——它是前人经验的标准化模板，让复杂问题有章可循，但真正的好菜，还得靠做菜的人。

> **思维框架是「长」在脑子里的思考路径**——它让你反应快，也可能让你看不全。

> **框架思维是俯瞰迷宫的向导视角**——让你永远手里有牌、心里有数，找到解决问题的最优路径。
