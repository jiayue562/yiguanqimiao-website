# 逻辑基本知识·第四章 演绎推理

> **核心结论**：演绎推理是从一般性前提推出个别性结论的推理，只要前提为真、推理形式有效，结论**必然为真**；核心形式包括三段论、复合命题推理、选言推理、假言推理和二难推理。

---

## 第一节 三段论

### 核心定义

**三段论**：传统形式逻辑的核心推理形式，由三个部分组成：

| 组成部分 | 定义 | 示例 |
|---------|------|------|
| **大前提** | 一般性原理 | 所有M都是P |
| **小前提** | 个别性情况 | 所有S都是M |
| **结论** | 根据两个前提推出的个别性结论 | 所以所有S都是P |

**典型形式**：
```
所有M都是P（大前提）
所有S都是M（小前提）
∴ 所有S都是P（结论）
```

### 课文案例：《拿来主义》

> "没有拿来的，人不能自成为新人；没有拿来的，文艺不能自成为新文艺。所以，我们要拿来。"

**逻辑分析**：
- 大前提："没有拿来的，人/文艺不能创新"
- 小前提："我们要创新人/文艺"
- 结论："我们要拿来"

### 三段论规则（核心）

| 规则 | 内容 | 违反谬误 |
|------|------|---------|
| 规则1 | 中项（M）至少周延一次 | 中项不周延 |
| 规则2 | 前提中不周延的项，结论中也不能周延 | 不当周延 |
| 规则3 | 两个否定前提不能推出结论 | — |
| 规则4 | 前提有一个否定，结论必否定 | — |

**隐秘联系**：
- → [[逻辑基本知识·第一章 绪论#1.3 思维的形式与内容]]：三段论的核心是逻辑形式"所有M都是P，所有S都是M，所以所有S都是P"
- → [[逻辑基本知识·第五章 思维的基本规律#同一律]]：三段论要求中项保持同一，不能偷换概念

## 第二节 复合命题

**核心定义**：复合命题是由**简单命题**和**逻辑联结词**构成的命题。

### 四类基本复合命题

#### 1. 联言命题

| 维度 | 内容 |
|------|------|
| **形式** | p **且** q |
| **逻辑规则** | 全真才真，一假则假 |
| **示例** | "鲁迅是文学家，并且是思想家" |

#### 2. 选言命题

| 类型 | 形式 | 逻辑规则 |
|------|------|---------|
| **相容选言** | p **或** q | 可以同时存在；否定一部分→肯定另一部分；**肯定一部分不能否定另一部分** |
| **不相容选言** | **要么** p **要么** q | 只能存在一个；否定一个→肯定另一个；肯定一个→否定另一个 |

#### 3. 假言命题

| 类型 | 形式 | 说明 |
|------|------|------|
| **充分条件** | **如果** p，**那么** q | p是q的充分条件 |
| **必要条件** | **只有** p，**才** q | p是q的必要条件 |
| **充要条件** | **当且仅当** p，**才** q | p是q的充要条件 |

#### 4. 负命题

| 维度 | 内容 |
|------|------|
| **形式** | **并非** p |
| **逻辑规则** | 负命题的真值与原命题**相反** |
| **示例** | "并非所有鸟都会飞" |

**隐秘联系**：
- → [[逻辑基本知识·第一章 绪论#2.3 命题形式、真值和逻辑]]：复合命题的形式化表达是命题逻辑的核心
- → [[逻辑基本知识·第四章 演绎推理#第四节 假言命题与假言推理]]：假言命题是假言推理的基础
- → [[逻辑基本知识·第六章 谬误#语义谬误]]：混淆"相容"与"不相容"选言命题是常见语义谬误

## 第三节 选言命题与选言推理

### 相容选言推理

**规则**：
1. ✅ **否定一部分选言支，就要肯定另一部分选言支**
2. ❌ **肯定一部分选言支，不能否定另一部分**（两种情况可以同时存在）

**有效式**：
```
（p 或 q）
非p
∴ q
```

**示例**：
> "他要么是生病了，要么是家里有事迟到了；他没生病，所以他是家里有事迟到了。"
> **注意**：若改为"他生病了，所以他不是家里有事迟到的"——**无效**，因为两种情况可以同时存在。

### 不相容选言推理

**规则**：
1. ✅ 否定一个，就要肯定另一个
2. ✅ 肯定一个，就要否定另一个

**有效式**：
```
（要么p，要么q）
非p
∴ q

（要么p，要么q）
p
∴ 非q
```

**示例**：
> "要么生，要么死；他没死，所以他活着。"

**隐秘联系**：
- → [[逻辑基本知识·第五章 思维的基本规律#排中律]]：不相容选言推理的逻辑基础是排中律（两个矛盾命题不能同假）
- → [[逻辑基本知识·第五章 思维的基本规律#不矛盾律]]：相容选言推理"肯定不能否定"的逻辑基础是不矛盾律（两个情况可以同时为真）

## 第四节 假言命题与假言推理

### 充分条件假言推理

**规则**：
1. ✅ **肯定前件就要肯定后件**（肯定前件式）
2. ✅ **否定后件就要否定前件**（否定后件式）
3. ❌ 否定前件不能否定后件
4. ❌ 肯定后件不能肯定前件

**有效式**：
```
如果p，那么q（充分条件）
p（肯定前件）
∴ q

如果p，那么q（充分条件）
非q（否定后件）
∴ 非p
```

**示例**：
> "如果下雨，地就湿；下雨了，所以地湿。"（肯定前件式）
> "如果下雨，地就湿；地没湿，所以没下雨。"（否定后件式）

### 必要条件假言推理

**规则**（与充分条件**相反**）：
1. ✅ **否定前件就要否定后件**（否定前件式）
2. ✅ **肯定后件就要肯定前件**（肯定后件式）
3. ❌ 肯定前件不能肯定后件
4. ❌ 否定后件不能否定前件

**有效式**：
```
只有p，才q（必要条件）
非p（否定前件）
∴ 非q

只有p，才q（必要条件）
q（肯定后件）
∴ p
```

**示例**：
> "只有努力，才能成功；不努力，所以不能成功。"（否定前件式）
> "只有努力，才能成功；成功了，所以努力了。"（肯定后件式）

**隐秘联系**：
- → [[逻辑基本知识·第一章 绪论#2.2 演绎逻辑与归纳逻辑]]：假言推理是演绎逻辑的核心内容，前提真则结论必然真
- → [[逻辑基本知识·第六章 谬误#语形谬误]]：混淆充分条件和必要条件是常见推理谬误

## 第五节 二难推理

### 核心定义

**二难推理**：由两个假言命题和一个选言命题作为前提构成的推理，常让对方陷入"进退两难"的困境而得名。

### 经典案例：半费之讼

> 普罗泰戈拉教欧几里得诉讼，约定：
> - 欧几里得先付一半学费
> - 另一半等欧几里得**打赢第一场官司**再付
> 
> 后来欧几里得一直不打官司，普罗泰戈拉就起诉他：
> - "如果你打赢这场官司，按照**合同**要付我另一半学费；"
> - "如果你打输这场官司，按照**判决**要付我另一半学费；"
> - "你要么打赢要么打输，所以你都要付我学费。"

**逻辑结构（二难推理构造）**：
```
如果打赢 → 按合同付学费
如果打输 → 按判决付学费
要么打赢，要么打输
∴ 总要付学费
```

### 二难推理的正确形式要求

| 要求 | 内容 |
|------|------|
| **假言前提** | 前件必须是后件的充分条件 |
| **选言前提** | 必须**穷尽所有可能** |

### 假二难推理谬误

**定义**：违反上述要求（假言前提不充分，或选言前提未穷尽）而产生的谬误。

**示例**：
> "要么支持我们，要么就是反对我们。"（选言前提未穷尽——还有"中立"的可能）

**隐秘联系**：
- → [[逻辑基本知识·第五章 思维的基本规律#排中律]]：二难推理的选言前提必须穷尽所有矛盾可能，这正是排中律的要求
- → [[逻辑基本知识·第三章 归纳推理#穆勒五法]]：二难推理与归纳推理形成对照——演绎必然vs归纳或然

## 核心金句

1. **"演绎推理的核心：前提真且形式有效，结论必然真。"**
2. **"三段论的规则：中项至少周延一次，前提中不周延的项结论中也不能周延。"**
3. **"充分条件假言推理：肯定前件→肯定后件；否定后件→否定前件。"**
4. **"必要条件假言推理：否定前件→否定后件；肯定后件→肯定前件。"**
5. **"二难推理让对方进退两难——但假二难推理的选言前提往往没有穷尽所有可能。"**
