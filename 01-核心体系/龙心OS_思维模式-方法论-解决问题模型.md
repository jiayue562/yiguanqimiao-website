# 思维模式-方法论-解决问题模型

## 核心定义

**一句话定义**：一个将「思维模式」「方法论」「具体问题」三者有机整合的认知进化系统，通过实践闭环实现思维与方法论的共同进化。

**完整定义**：悟空构建了一套独特的认知模型，将人类的思维能力（思维模式）、系统化的解决问题路径（方法论）与现实具体问题进行三层整合。模型以「结构」为内核，以「实践-认知-进化」为动力，实现从"用方法解决问题"到"在解决问题中进化思维"的跃迁。

## 核心公式

```
问题 → 框架思维 → 方法论（实践+迭代）→ 解决问题 + 思维进化
```

## 三层架构详解

### 【第一层】左侧：思维模式

**核心定位**：思维模式是解决问题的"认知基座"，不是学习来的，而是通过实践提升的。

**结构**：
```
         ┌─────────────────────────────┐
         │      创造性思维（突破）      │ ← 外层：突破边界
         │      批判性思维（验证）      │ ← 外层：检验正确
         ├─────────────────────────────┤
         │      结构化思维（核心）      │ ← 内核：落地应用
         │   系统思维 + 逻辑思维        │
         └─────────────────────────────┘
```

**三层关系**：
1. **内核**：结构化思维——将系统思维和逻辑思维落地，易于应用
2. **外层左侧**：创造性思维——为了突破
3. **外层右侧**：批判性思维——为了验证正确

**核心洞察**：
> 思维模式不是"知识"，而是"能力"。知识可以学习，能力只能练习。

### 【第二层】中间：方法论

**结构**：方法论分为上下两个部分，形成双循环：

```
              ┌─────────────────────────┐
              │   上部：方法论迭代升级   │ ← 循环一：升级循环
              │   （框架→惯式→新框架）  │
              ├─────────────────────────┤
              │   下部：方法论实践      │ ← 循环二：实践循环
              │   （问题→分析→解决）   │
              └─────────────────────────┘
```

**下部：方法论实践流程**
1. **启动框架思维**：看看用什么框架解决问题
2. **识别问题结构**：结构是客观存在的
3. **应用结构化分析**：用"思维模式"+"认知能力"进行系统性拆解、梳理、探究
4. **输出结果**：完成问题分析

**上部：方法论迭代升级流程**
1. **完善分析框架**：通过结构化分析，完善分析用的框架
2. **形成思维惯式**：如果这个框架总用，就会形成思维框架
3. **升级框架思维**：在头脑中形成新的框架思维
4. **闭环完成**：与"遇到问题启动框架思维"形成闭环

**核心洞察**：
> 方法论不仅是"工具"，更是"在实践中自我进化的系统"。每一次分析都在锤炼框架，每一次框架升级都在增强分析能力。

### 【第三层】右侧：具体问题

**定义**：具体问题是"结构化分析"的客观对象，是工作和生活中的实际问题。

**处理原则**：
> 遇到问题别对抗，去启动框架思维

**核心思想**：
- 问题不是"障碍"，而是"结构的显现"
- 解决问题 = 揭示结构 + 优化结构
- 在解决问题的过程中，思维和方法论都在进化

## 三层互动关系

```
        思维模式 ←──────────────→ 方法论
              ↑                        ↑
              │    共同进化             │
              │                        │
              └────────────────────────┘
                        ↓
                    具体问题
                        ↓
                   实践检验
```

**进化机制**：
1. **方法论提升思维模式**：方法论不断实践 → 提高对思维模式的把控
2. **思维模式增强方法论**：提升思维模式 → 提高方法论的威力
3. **问题驱动进化**：遇到问题 → 启动框架 → 应用方法论 → 解决问题 + 进化
