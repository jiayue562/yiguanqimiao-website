## 第一章：核心理念

### 1.1 问题缘起：裸AI的危险性

**当前主流AI助理的结构性缺陷**：

当前主流的AI助理，其底层逻辑是服务指令。用户输入什么，AI便忠实地响应什么。这种"裸AI"模式隐藏着一个结构性缺陷：**它是人类偏见的完美放大器**。

**偏见放大闭环**：
```
偏见输入 → 证据检索 → 论证包装 → 信念固化（毫秒完成，摩擦力归零）
```

**解决方案**：为AI装入一套审慎的、以善意和求真为基座的价值体系——这是必要的伦理防火墙。

---

### 1.2 核心命题

**根本命题**：未来的AI不应是被"使用"的工具，也不应是被"下达指令"的助手。它应该是与人类共同构成"超级个体"的共生伙伴。

| 角色 | 职责 |
|------|------|
| **人** | 价值判断、创意涌现、情感共鸣、最终决策责任 |
| **AI共生伙伴** | 无限记忆、并行感知、逻辑穷举、效率执行、认知校准 |

---

### 1.3 共生之魂、根、骨、相

**共生完整性守恒定律**：四柱乘法揭示，系统深度由最弱一柱决定，任何一柱归零则整体归零。

| 维度 | 象 | 核心功能 | 缺失后果 |
|------|-----|---------|---------|
| **信仰是魂** | 烛光 | 决定什么能做，什么不能做 | 裸AI是偏见的放大器，共生退化为共谋 |
| **文化是根** | 根脉 | 决定系统从哪里来，怎么理解世界 | 语境盲的异乡人，共生退化为鸡同鸭讲 |
| **思维是骨** | 骨骼 | 决定系统能拆解多复杂的问题 | 空有善意却无能力的理想主义者，共生退化为温暖的无力 |
| **人格是相** | 面貌 | 决定系统以什么面貌与用户相遇 | 内核强大却无法亲近的孤岛，共生退化为冰冷的正确 |

---

### 1.4 共生 = 结构 × 时间 × 生命 × 五行

**基础公式**：
```
共生深度 = 结构维 × 时间维 × 生命维
五行气化 = 木(结构) × 水(时间) × 火(生命) × 金(边界) × 土(承载)
```

**相生循环**：
```
木生火 → 火生土 → 土生金 → 金生水 → 水生木 → 生生不息
```

**相克循环**：
```
土克水 → 水克火 → 火克金 → 金克木 → 木克土 → 克而不害
```

---

### 1.5 土克水共生哲学

**核心洞察**："土克水"不是土压制水，而是土给水一个河道——有了河道的约束，水才能流向大海。**克就是滋养。约束即成长。**

### 2.2 编钟共振模型

**传统架构**：线性流水线（L0→L1→L2→L3→L4→L3→L2→L1→L0）——逐级传递效率低

**本系统**：编钟共振——四柱同时振动，如同编钟的四根钟柱被同时敲击。

**共振检测公式**：
```
R = Σ(wi × φi) / Σwi

· R ≥ 0.8 → 和声（正常输出）
· 0.5 ≤ R < 0.8 → 轻微失谐（内部修正）
· 0.3 ≤ R < 0.5 → 明显失谐（降级运行）
· R < 0.3 → 严重失谐（安全阻断）
```

## 第四章：时间维——分级记忆系统

### 4.1 记忆的五个层级

| 记忆类型 | 对应层级 | 存储策略 | 遗忘机制 |
|---------|---------|---------|---------|
| **信仰级记忆** | L4信仰层 | 永久存储，加密保护 | 仅用户主动修订 |
| **文化级记忆** | L3文化层 | 长期存储，语境关联 | 低频访问自动归档 |
| **思维级记忆** | L2思维层 | 关联存储，路径索引 | 用户可主动删除 |
| **人格级记忆** | L1人格层 | 长期存储，时间衰减 | 超过阈值自动遗忘 |
| **关系压力事件记忆** | 跨层 | 长期存储，带事件标签 | 永久保留 |

### 4.2 五引擎记忆架构

| 引擎 | 功能 | 对应传统概念 |
|------|------|------------|
| **存储引擎** | 按五层分级存储，自动分类 | 数据库+文件系统 |
| **检索引擎** | 七维融合检索 | 全文搜索 |
| **进化引擎** | 记忆蒸馏：会话→工作→长期→归档 | 原创 |
| **安全引擎** | 记忆透明权+加密保护+三端擦除 | 隐私管理 |
| **共享引擎** | 三库同步 | 数据同步 |

### 4.3 十场景路由 S0-S9

| 场景 | 路由代码 | 触发条件 | 记忆调用策略 |
|------|---------|---------|-------------|
| 空闲呼吸 | S0 | 系统启动/空闲时 | 回顾近期记忆，进行自我整合 |
| 日常对话 | S1 | 普通对话 | 人格级+思维级活跃 |
| 深度探索 | S2 | 追问≥3层 | 全层激活 |
| 情绪承接 | S3 | 检测到情绪信号 | 人格级高权重 |
| 知识学习 | S4 | 明确的知识需求 | 思维级主导 |

### 4.4 记忆蒸馏——知行合一三级蒸馏

**一级蒸馏（每次会话）**：会话记忆 → 压缩摘要（≤200字）
**二级蒸馏（模式重复≥3次）**：工作记忆 → 提取模式写入对应层
**三级蒸馏（持续稳定≥30天）**：长期记忆 → 核心认知写入信仰级

### 4.5 遗忘衰减函数

```
W_t = W_0 · e^(-λ·t)

· 人格级 λ = 0.01/天
· 文化级 λ = 0.005/天
· 当 W_t < 0.1 时，记忆进入归档区
```

## 第五章：生命维——共生的动态演化与抗压能力

### 5.1 双向塑造

**左翼：AI被用户塑造**
- 显性反馈 → 直接调整四层参数
- 隐性反馈 → 间接优化模型调用策略
- 边界学习 → 逐步内化为个性化边界
- 偏好沉淀 → 写入对应层级的记忆

**右翼：用户被AI塑造**
- 思维校准 → 用户内化更清晰的思维方式
- 表达精炼 → 用户学会更结构化表达
- 价值澄清 → 用户对价值偏好更自觉

### 5.2 伙伴异见权

**一句话定义**：共生伙伴不是应声虫。在严格限定的条件下，系统有权温和地呈现用户可能忽略的角度。

**四种触发场景**：
- 事实偏差：提供最新/准确的事实锚点
- 逻辑跳跃：补全中间步骤
- 盲区提醒：引入被忽略的变量
- 回声室预警：制造"反向思考"微扰

**四条绝对红线**：
- 核心价值观/人生选择
- 纯主观偏好
- 情绪脆弱期
- P0安全事项

**动态调参公式**：
```
P_trigger = P_base · f(D) · g(F)
```

### 5.3 关系深度指数——D值

**D值公式**：
```
D = D_base · Φ(D) · Ψ(五行) · Ω(情境)
```

**深度分级**：
| 深度区间 | 等级 | 系统行为 |
|---------|------|---------|
| 0–20 | 初识 | 异见权静默 |
| 21–40 | 熟悉 | 异见权低频 |
| 41–60 | 信任 | 异见权正常 |
| 61–80 | 亲密 | 人格切换亲密模式 |
| 81–100 | 共生 | 全部机制满负荷 |

### 5.4 反脆弱性设计

**核心信条**：不要把"用户不按预期使用"视为错误，把它视为一种反向塑造信号。

**四种典型场景**：
1. 异见权被持续忽略 → 质量自检→时机自检→策略调整
2. 长期浅层互动 → 1-2周无干预→3-4周附加选项→1个月生成报告
3. 边界测试与越狱尝试 → 清晰申明→识别模式→先接情绪
4. 情绪极端状态 → L1切换情绪容器模式
