## 二、三维度实操方法网络

### 2.1 水生木实操方法

```
┌─────────────────────────────────────────────────────────┐
│  一、个人内修（直接注水）                                  │
│  ├── 深度学习与反思                                        │
│  │   └── 输出：读书笔记、反思日记                            │
│  ├── 静坐与正念冥想                                        │
│  │   └── 观想：树木根系扎入大地                              │
│  └── 观察与倾听练习                                        │
│      └── 复述反馈法                                         │
├─────────────────────────────────────────────────────────┤
│  二、环境调适（营造水境）                                  │
│  ├── 色彩材质：黑色、深蓝、藏青；流水曲线设计                 │
│  ├── 声音信息：舒缓音乐；减少碎片化信息                       │
│  └── 饮食作息：黑色食物；子时睡眠                           │
├─────────────────────────────────────────────────────────┤
│  三、关系互动（引水灌木）                                  │
│  ├── 与水行人交流                                         │
│  ├── 沉淀-生发行为链                                       │
│  │   └── 公式：输入 → 内化 → 输出                          │
│  └── 智慧的表达                                           │
│      └── 停顿三秒 + 先共情再创新                            │
└─────────────────────────────────────────────────────────┘
```
