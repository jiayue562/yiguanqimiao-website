二、维度一：炎上（推动提升--急躁争斗）

核心定义：炎上指火具有发光放热、光芒四散、蒸腾上升之象。引申为"火"有温热、光明、变化、活动、升腾的特性。南方炎热、阳气用事、万物布施之象均属火。火行人属于社会性的、主动的个体，人越多越刺激越开心，就是他们。快速卷入人际互动，越有节奏越有活力。

阳面：推动提升

1. 积极乐观

火行人容易感受到各种积极的情绪，如快乐、乐观、兴奋等。他们有积极体验快乐情绪（如喜悦、爱和兴奋）的倾向。火行人喜欢与人接触，充满活力，经常是情绪高涨的。火行人口才好，典型的外貌协会成员，这都为他们成为超级明星创造了条件。有火行人在场，必炫必闪，必煽动群心。他们天生具有夸大其词的本事。加之直觉敏感，似乎骨子里就潜藏着激发他人梦想的能力。

2. 社交达人

火行人通常是鼓舞人心的，活跃的，独立的。他喜欢刺激有创造性的活动。社交中他是开朗的，友好的，并试图成为有趣的乐意与人相处的人。火行人通常热情对待他人，与人相处，合作与适应的能力特强。萍水相逢也可以一见如故。他们把更多的精力倾注于维持个人关系而不是客观事物，喜欢保持一种广泛的人际关系。

3. 活力四射

火行人常常是行动的发起者，在团队中活力四射，尤其在压力下工作精力旺盛；敢于面对困难并且义无反顾地加快速度，敢于独自做决定而不介意别人反对不反对。火行人他是团队快速行动的最有效成员。

4. 满腔热情

他们以热情的态度对待生命，感受与个人相关的所有事物。由于他们很理想化，按照自己的价值观生活，因此火行人对他们所尊重和敬佩的人、事业和机构非常忠诚。他们精力充沛、满腔热情。

5. 鼓动煽情

火行人是在感受的基础上做决定，而不是基于事实情况。他们是天然的领导者，受人欢迎而有魅力，他们愿意成为出色的传播工作者。火行人具有煽动能力，并且向别人传播他们的观点。他们一旦做出决定就立刻行动，而不是停下来重新检查细节和形势的现实情况。

阴面：急躁争斗

1. 制造冲突

火苗遇风就蹿高，火行人天生有放大事实的本领。在工作中，你要顺着他们的意思来，否则的话，火行人是可以制造冲突和挑起是非的团队成员。一分不满意没有及时安稳住，一会儿就增长到十分。有时候好打不平，和他们没有关系的事情，他们也愿意替人出头。

2. 愤怒敌意

火行人容易发火，在感到自己受到不公正的待遇后会充满愤恨，暴躁的、威胁的、报复的。比一般人更容易体验到急躁以及痛苦挫折的状态。他们的痛苦因为自己的主观想象而更加痛苦，他们的愤怒必须找个出口发泄出去，哪怕只是扬言说说狠话，让别人感受到来自他们的威胁也行。所以他们的报复心是真实的，因为火行人无力自行消解愤怒，那样只会让他们更加痛苦。

3. 喜欢挑战

火行人喜欢挑战别人，没有结果誓不罢休，对新观点接受更快，工作中总可以看到他们风风火火的劲头。因为他们满怀热情，急切地开始下一次挑战，所以火行人有时会做出错误的臆断或过于急促地做决定，而没有收集到所有重要的事实。他们需要放慢速度，更仔细地注意计划的细节。

4. 自视甚高

火行人通常表现为自视甚高，自以为是，可能非常地武断。火行人对于赞扬有着很好的反应，但却很容易被批评所伤害，这些批评使他们很易怒。他们当面受到甚至是最无恶意或动机良好的批评时，反应是激动、感到受伤害或生气。

5. 喜欢争辩

火行人大多性格比较外向，干劲十足，在人际关系方面比较喜欢挑战别人，喜欢争辩，而且在争论中不赢不罢休。火行人往往以自我为中心，缺乏相互理解。

四、维度三：炽烈（热情自信--自卑夸张）

核心定义：火焰炽烈燃烧，太阳就是一个超级大火球。他们在顺境时，给人的印象是热情自信，而在逆境（严重不自信）时，表现为自卑夸张。

阳面：热情自信

1. 受人欢迎

火行人把他们同别人的关系放在十分重要的位置，所以他们往往健谈、受人欢迎、有礼貌、渴望取悦他人。

2. 活跃豪爽

火行人喜欢周围的人，经常会向他们表达积极友好的情绪。在一个社交群体当中，喜欢引起别人的注意。火行人的兴趣广泛，喜欢同时张罗许多事，实则是他们害怕被遗忘被忽略，所以希望涉足很多事项。

3. 多动少静

火行人在生活工作中快节奏，显得充满精力，他们喜欢运动，喜欢刺激冒险。

4. 英雄主义

他们无法忍受乏味的生活，又恐惧被限制，所以目标过多，爱出风头。火行人考虑到个人能力时不够现实，往往自视甚高，个人英雄主义。

5. 寻求刺激

火行人在缺乏刺激的情况下容易感到厌烦，喜欢喧嚣吵闹，喜欢冒险，寻求刺激。越刺激越积极。他们喜欢浮华的、华丽的、奢华的场景。有时候在感受到强烈的诱惑时，不容易控制，容易追求短时的满足而不考虑长期的后果。

阴面：自卑夸张

1. 情感激动

如果他们发现某个人，尤其是他们尊敬的人，对他们的能力表示疑问时，更容易感情激动。他们对貌似无足轻重的情况会有粗暴的反应，这种爆发会伤害接近他们的人。火行人与其任情感左右自己，不如给感情一个积极的发泄途径，这样才会真正地充分控制感情。

2. 粗犷大胆

火行人经常显露出粗犷大胆的行为，来表现他们是勇敢和不害怕的。他们有一种很强的信念就是"最好的防守就是攻击"。危险真的出现时，他们会直接面对危险，利用语言或肢体的行动来对付危险。这是一种拒绝和应付内在恐惧的方式。

3. 变幻无常

火行人变幻无常，他们恐惧承诺，但喜爱初期醉心的感觉，因为没法平静下来，造成他们缺乏识别能力，不知什么是好，什么是不好的。

4. 追逐刺激

火行人不停止的追逐激烈和刺激。他们可能参与到危险的、不实际的事情上。他们变成了恐怖惊心的追求者，寻找常人没做过的、没见过、没听过的事。

5. 纵容自己

他们是典型的消费者。他们享受逛街、旅游、纵容自己等。他们会花力气去寻找好完的地方（餐馆、电影等）他们更注意哪里有大减价，也很喜欢谈论这些话题，去那里获得心爱的东西等。

六、维度五：迅疾（气势夺人--无序冲动）

核心定义：火的能量在于燃烧，火体瞬间飘忽不定，他们在顺境时，给人的印象是气势夺人，而在逆境（没有达到他们期望的震撼效果）时，表现为无序冲动。

阳面：气势夺人

1. 气场强大

火行人很清楚自己的需要，他们眼里没有重复性的任务，没有细节关注。当项目开始时，立刻进入自由讨论环节，他在期待众人的表扬。这种公开的以个人为中心的情绪，必须以象征荣誉的头衔、证书等来认可他的地位与成就，才撑得起面子。火行人要的不是意料中的结果，他们期待的永远是震撼的气氛，就像是聚光灯下他夺冠的感觉。

2. 风风火火

火行人在工作中总是风风火火，渴望满足周围人的要求，他们想象力丰富，对新的想法采取开放的态度，并希望通过他个人的贡献来评估他。经常虎头蛇尾，风风火火开场，草草率率收尾。有句话很适合他们，坚持就是胜利。

3. 欣赏强者

火行人喜欢挑战，不过他只喜欢挑战的过程，他不会像金行人般对下属诸多要求。虽然有时看起来很专横，独断，但其实他们也喜欢吸收其它人的看法。火行人看不起没有真材实料的人，轻视弱者，要得到他的尊重，本身需要一定实力。总的来说，与火行人相处只有二种方式，一是做强者，让他尊重你，一是受他控制，让他保护你。

4. 强权霸气

用多数中国人熟悉的概念来说，火行人的性格就如充满霸气，豪气，压迫感，一言不发已令人害怕，一出场已有"大地在我脚下"的感觉。他们有情有义，重视感觉，因为火行人心中会画一个圈，分辨你是不是他们的人，如果你是他们的人的人，他们会尽一切力量去保护你。

5. 能量爆发

火行人办事可以很拼命，玩起来可以很投入。不过若懒惰起来，也可以懒惰十分，就像季节变化一样，若火行人处于创业时期，绝不能去玩，否则可能一发不可收拾，一旦没有了斗志，便要等有很大危机出现时，才能推动他重拾事业心，再去冲刺。

阴面：无序冲动

1. 藐视规则

火行人属于大闹天宫的孙悟空，他们精力充沛，我行我素，挑战权威，藐视规则。对他们领导能力的挑战在于要建立共同的目标和游戏规则，并能坚守它们。否则一有机会，火行人就会跳出管制。

2. 虎头蛇尾

火行人做事情的时候开始时声势很大，到后来劲头很小，有始无终。

3. 精力分散

火行人每次把精力集中在一件事上很困难，在做决定时很麻烦。他们了解许多可能性，以至他们很难挑选最好的活动或兴趣去追求。有时他们会做出拙劣的决定，立刻卷入过多的事物之中。

4. 喜欢挑衅

火行人喜欢挑衅，容易发火，耐心不够；有什么说什么，不管会不会得罪别人。明知自己犯了错误，自己的做法不对，也不会用道歉的方式来缓和局势。

5. 易受诱惑

火行人很容易受诱惑，很难自我约束，火行人优先考虑工作而不是娱乐，在工作与娱乐之间寻找—种平衡。
