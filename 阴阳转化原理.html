二、四步详解

2.1 认不是

所说的认不是，如能达到真处，就是儒家"慎独"的功夫，即真正认识到自己心性中的阴面，遇事稍一错念，刚一动烦恼，即认识到他的"不是"，因为他足以污染自性，这乃是最大的不是——"错处"。这是做彻底拨阴的功夫，所以古人说"认不是是明心丹"。

认不是的核心

认不是的本质：

    【什么是"不是"】
    ├─ 认识到自己心性中的阴面
    ├─ 遇事稍一错念
    ├─ 刚一动烦恼
    ├─ 认识到他的"不是"
    └─ 这乃是最大的不是——"错处"

    【认不是的作用】
    ├─ 明心丹 → 看清内心的镜子
    ├─ 慎独功夫 → 儒家修身的功夫
    └─ 彻底拨阴 → 清除阴性能量

    【认不是的方法】
    ├─ 不给自己找理由开溜
    ├─ 承认自己的错误
    └─ 不拖延，及时改正

2.2 找好处

所说的找好处，不仅从好人好事中找到好处，即从坏人坏事中，也可找到反面的好处。心中常存他人的好处，才能生出义气和真金，这也是彻底"反阴的功夫"。所以古人说："找好处是暖心丸"。

找好处的核心

找好处的本质：

    【找什么好处】
    ├─ 从好人好事中找到好处
    ├─ 从坏人坏事中找到反面的好处
    └─ 任何情况下都能找到好处

    【找好处的作用】
    ├─ 暖心丸 → 温暖内心的良药
    ├─ 生义气 → 培养正义的能量
    ├─ 生真金 → 转化阴性为阳性
    └─ 反阴功夫 → 彻底转化阴性

    【找好处的方法】
    ├─ 常存他人的好处
    ├─ 不看短处看长处
    └─ 吸阳光不吸阴气

2.3 信因果

所说的信因果，即认识到天地间的一切事物没有无因之果，有因才有果，果是由于一定的因缘而发展变化来的。我们今天所看到的一切果，都是来自一定的因。只不过是有的因尚未被我们所发现。能明白这个道理，自然就不生怨气，而能生出信实的真土。

信因果的核心

信因果的本质：

    【什么是因果】
    ├─ 天地间一切事物没有无因之果
    ├─ 有因才有果
    ├─ 果是由于一定的因缘发展变化来的
    └─ 有的因尚未被我们发现

    【信因果的作用】
    ├─ 不生怨气 → 不再埋怨
    ├─ 生真土 → 培养信实的能量
    ├─ 明因果 → 理解事物发展的规律
    └─ 法自然 → 顺应自然规律

    【信因果的方法】
    ├─ 认识到一切都有因果
    ├─ 不对结果抱怨
    └─ 从因上努力

2.4 达天时

所说的达天时，即包含着老子说的"道法自然"的古代哲学思想。

达天时的核心

达天时的本质：

    【什么是天时】
    ├─ 道法自然
    ├─ 顺应天时
    ├─ 明了时代变化
    └─ 与道合一

    【达天时的作用】
    ├─ 明理 → 通达事理
    ├─ 达时 → 把握时机
    ├─ 生真火 → 培养光明的能量
    └─ 超三界 → 不被五行束缚

    【达天时的境界】
    ├─ 人本天地五行之气而生
    ├─ 有气斯有质
    ├─ 有质就有偏
    ├─ 偏则不中
    ├─ 不中就离道
    └─ 离了道则五行乖乱
