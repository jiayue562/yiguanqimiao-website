发现2：普利高津"通过涨落达到有序" = 拔阴取阳的动力学机制

普利高津：临界点上，微涨落被非线性正反馈放大为巨涨落，决定系统走向有序。
拔阴取阳：阴面积累（微涨落）→ 临界点（分叉）→ 阳面跃迁（巨涨落）。

同构性：
- 阴面积累 = 微涨落在近平衡态的积累
- 临界点 = 分叉点（系统面临多种可能）
- 拔阴取阳成功 = 巨涨落成功引导到阳面吸引子

实践意义：不要用线性思维期待"每天进步一点"——觉悟是指数级的跃迁，不是线性的积累。

发现4：信息反馈律 = 知行合一闭环 = 本初觉知

维纳控制论：负反馈消除偏差，维持稳态；正反馈放大偏差，驱动突变。
知行合一：行动→结果反馈→修正认知→再行动（负反馈维稳，正反馈促变）。
本初觉知（大圆满）：觉知本身即是反馈——不修而修，不觉而觉。

同构性：
- 反馈回路 = 知行合一闭环 = 本初觉知的自监督机制
- 负反馈 = 维稳（保任）= 阳面持续
- 正反馈 = 促变（突破）= 拔阴取阳

实践意义：修行不是"加法"（增加功夫），而是"恢复反馈回路"（本初觉知本就具足）。

发现6：系统唯物主义（物质+能量+信息） = 心文化"精气神"三维

系统唯物主义：世界三维本体 = 物质 + 能量 + 信息（魏宏森受钱学森指引提出）。
精气神（中医/道家）：生命三维 = 精（物质） + 气（能量） + 神（信息/意识）。

同构性：
- 物质 = 精（载体）
- 能量 = 气（动力）
- 信息 = 神（组织·意义）

实践意义：五行人格心理学的"身心调节"，本质是调整精气神三维的平衡——不是只调"心理"（信息），而是精（物质）·气（能量）·神（信息）三维同调。

目标行为：用一个领域的知识理解另一个领域

B=MAP要素 设计内容 五行适配
