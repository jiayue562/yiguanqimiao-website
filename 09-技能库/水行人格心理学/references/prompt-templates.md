# 水行人格心理学提示词模板集

> 本文档包含针对水行人的提示词模板

---

## 一、基本分析模板

### 1.1 水行特质分析

```
请分析以下特征的水行特质：
[输入：具体特征描述]

分析维度：
1. 一心：润泽映现性、沉潜含藏性
2. 身界：体型、皮肤、动作、健康
3. 心界：情绪、思维、动机
4. 灵界：价值观、使命
5. 九层定位
```

### 1.2 水行健康评估

```
请评估以下人物的水行健康状态：
[输入：人物描述]

评估维度：
1. 阳水特征明显程度
2. 阴水特征明显程度
3. 三界协同状态
4. 五行生克平衡
5. 建议发展方向
```

## 二、诊断模板

### 2.1 九层定位诊断

```
请帮我诊断我在水行九层中的位置：
[输入：个人状态描述]

诊断要点：
1. 对照各级特征
2. 分析三界状态
3. 评估五行生克
4. 确定当前层级
5. 给出升维建议
```

### 2.2 五行失衡诊断

```
请诊断以下情况的水行五行失衡：
[输入：失衡情况描述]

诊断维度：
1. 金生水：原则框架状态
2. 水生木：行动转化状态
3. 土克水：稳定平衡状态
4. 水克火：热情状态
5. 调适建议
```

## 三、转化模板

```

```
请帮我化解水行的克伐问题：
[输入：克伐情况]
