## 📌 一句话定义

**书面头脑风暴法**（书面脑暴法）是传统口头头脑风暴的变体，通过让参与者轮流填写创意表格，在短时间内提出大量创意的接力式创意生成工具。

---

## 🎯 核心定义

### 1.1 核心目的

> "将表格填满"——追求数量而非质量

### 1.2 解决的问题

传统口头头脑风暴的问题：
- 少数人主导
- 内向者不敢发言
- 思路容易跑偏
- 效率低

---

## 📊 详细解析

### 2.1 操作步骤

```
┌─────────────────────────────────────────────────────────────┐
│                书面头脑风暴法六步法                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Step 1️⃣ 准备阶段                                          │
│  ─────────────────────────────────────                      │
│  • 准备A3或A4大小的表格                                      │
│  • 上方写明明确的主题或问题                                   │
│  • 确保每位参与者有一支笔                                      │
│                                                             │
│  Step 2️⃣ 个人书写                                          │
│  ─────────────────────────────────────                      │
│  • 每位参与者在自己的表格上                                    │
│  • 在规定时间内（如3分钟）写下尽可能多的创意                  │
│  • 每格一个想法                                               │
│                                                             │
│  Step 3️⃣ 传递表格                                          │
│  ─────────────────────────────────────                      │
│  • 时间到后，将表格向左（或右）传递给下一位参与者               │
│                                                             │
│  Step 4️⃣ 接力创新                                          │
│  ─────────────────────────────────────                      │
│  • 阅读前人写下的创意                                         │
│  • 在其基础上延伸、补充或激发新的想法                          │
│  • 并写入空格中                                               │
│                                                             │
│  Step 5️⃣ 循环传递                                          │
│  ─────────────────────────────────────                      │
│  • 重复传递过程                                               │
│  • 直到表格被填满或进行了预定轮次                              │
│  • 例：6人参加 × 6轮 × 3分钟 = 108个创意                      │
│                                                             │
│  Step 6️⃣ 整理与评估                                         │
│  ─────────────────────────────────────                      │
│  • 收集所有表格                                               │
│  • 进行合并、分类和初步评估                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 核心规则

| 规则 | 说明 |
|------|------|
| **严禁批判** | 不批评任何想法 |
| **追求数量** | 表格填满为目标 |
| **受启发延伸** | 在他人想法基础上补充 |

### 2.3 应用案例

**案例：为面向Z世代的智能水杯命名**

```
【主题】为一款面向Z世代的智能水杯命名

【接力过程】
• 参与者A：HydroPal
• 参与者B（受A启发）：H₂O-Buddy
• 参与者C（受B启发）：AquaChat
• 参与者D（受C启发）：水聊
• 参与者E（全新思路）：DropPal

【结果】
创意数量滚雪球般增长，30分钟产生108个命名方案
```
