## 📌 一句话定义

**利弊均衡表**是一种最基本、最常用的决策辅助工具，用于对创意、方案或选择的正面因素（利）和负面因素（弊）进行全面整理和综合判断。

---

## 🎯 核心定义

### 1.1 核心价值

> "强迫自己进行系统化思考"——克服冲动决策，使决策过程更理性

---

## 📊 详细解析

### 2.1 操作步骤

```
┌─────────────────────────────────────────────────────────────┐
│                  利弊均衡表五步法                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Step 1️⃣ 列出选项                                          │
│  ─────────────────────────────────────                      │
│  明确要评估的一个或多个创意、方案                               │
│                                                             │
│  Step 2️⃣ 划分两栏                                          │
│  ─────────────────────────────────────                      │
│  在纸或表格上画出两栏，分别标为"利（优点）"和"弊（缺点）"     │
│                                                             │
│  Step 3️⃣ 头脑风暴填写                                      │
│  ─────────────────────────────────────                      │
│  • 尽可能多地列出该选项的所有优点和缺点                        │
│  • 不要遗漏，即使是次要的                                     │
│                                                             │
│  Step 4️⃣ 重要度赋权（可选）                                 │
│  ─────────────────────────────────────                      │
│  对每个优缺点的重要性进行赋权（如1-5分）                      │
│                                                             │
│  Step 5️⃣ 综合判断                                         │
│  ─────────────────────────────────────                      │
│  • 审视列表，权衡利弊的总体比重和重要性                       │
│  • 如需可将加权后的分数加总进行比较                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 应用案例

**案例：是否接受海外工作机会**

| 维度 | 内容 |
|------|------|
| **利（优点）** | ① 薪资提升50% ② 职业发展新平台 ③ 体验不同文化 ④ 拓宽视野 |
| **弊（缺点）** | ① 远离家人朋友 ② 生活成本更高 ③ 语言和文化障碍 ④ 家庭搬迁压力 |
| **综合判断** | 职业发展机会是长期关键，但家庭因素权重极大。最终：接受，但要求公司提供额外家庭安置支持和探亲假 |
