---

## 📌 一句话定义

**决策矩阵**是一种更精细的评估工具，通过设定多个评估指标，对创意或方案进行量化评分，从而帮助在多个选项中做出理性选择。

---

## 🎯 核心定义

### 1.1 与其他工具的区别

| 工具 | 维度 | 精度 |
|------|------|------|
| **利弊均衡表** | 定性 | 低 |
| **支付矩阵** | 2维度 | 中 |
| **决策矩阵** | 多维度（3-7个） | 高 |

---

## 📊 详细解析

### 2.1 操作步骤

```
┌─────────────────────────────────────────────────────────────┐
│                  决策矩阵七步法                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Step 1️⃣ 列出备选方案                                      │
│  ─────────────────────────────────────                      │
│  明确需要比较的所有创意或方案                                  │
│                                                             │
│  Step 2️⃣ 确定评估指标                                      │
│  ─────────────────────────────────────                      │
│  基于决策目标，设定3-7个关键评估维度                          │
│  如：成本、技术可行性、市场潜力、风险、战略契合度等           │
│                                                             │
│  Step 3️⃣ 设定权重                                          │
│  ─────────────────────────────────────                      │
│  为每个指标分配权重，表示其相对重要性                          │
│  所有权重之和通常为1或100%                                   │
│                                                             │
│  Step 4️⃣ 评分                                              │
│  ─────────────────────────────────────                      │
│  为每个备选方案在每个指标上的表现打分                          │
│  如1-5分，1分最差，5分最好                                   │
│                                                             │
│  Step 5️⃣ 计算加权分                                        │
│  ─────────────────────────────────────                      │
│  得分 × 权重 = 该指标上的加权分                               │
│                                                             │
│  Step 6️⃣ 汇总总分                                          │
│  ─────────────────────────────────────                      │
│  将每个方案所有指标的加权分相加                               │
│                                                             │
│  Step 7️⃣ 比较决策                                          │
│  ─────────────────────────────────────                      │
│  总分最高为最优选择，结合定性分析最终决策                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 决策矩阵模板

| 备选方案 | 指标1(权重) | 指标2(权重) | 指标3(权重) | ... | 总分 |
|---------|------------|------------|------------|-----|------|
| 选项A | 得分/加权 | 得分/加权 | 得分/加权 | ... | Σ |
| 选项B | 得分/加权 | 得分/加权 | 得分/加权 | ... | Σ |
| 选项C | 得分/加权 | 得分/加权 | 得分/加权 | ... | Σ |

### 2.3 应用案例

**案例：选择新办公地点**

| 指标 | 权重 | A地点 | B地点 | C地点 |
|------|------|-------|-------|-------|
| 租金成本 | 30% | 3/0.9 | 4/1.2 | 5/1.5 |
| 交通便利度 | 25% | 4/1.0 | 5/1.25 | 3/0.75 |
| 周边配套 | 20% | 3/0.6 | 4/0.8 | 4/0.8 |
| 办公空间质量 | 15% | 5/0.75 | 3/0.45 | 4/0.6 |
| 品牌形象 | 10% | 4/0.4 | 4/0.4 | 3/0.3 |
| **总分** | 100% | **3.65** | **4.10** | **3.95** |

**决策**：B地点总分最高（4.10），成为首选
