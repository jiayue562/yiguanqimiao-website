# AIDMA模型 · 消费者行为阶段分析

## 🎯 一、核心定义

**AIDMA模型**是描述消费者从注意到购买决策全过程的经典营销模型，由美国广告学家E.S.刘易斯提出：

| 字母 | 含义 | 中文 | 核心问题 |
|------|------|------|----------|
| **A** | Attention | 注意 | 如何让消费者看到？ |
| **I** | Interest | 兴趣 | 如何引发消费者兴趣？ |
| **D** | Desire | 欲望 | 如何激发购买欲望？ |
| **M** | Memory | 记忆 | 如何让消费者记住？ |
| **A** | Action | 行动 | 如何促使购买行动？ |

### 1.1 AIDMA曲线

```
消费者数量
    ▲
    │▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
    │
    │    ★★★★★★★★★★★★★★★★★★★  注意（A）
    │  ★★★★★★★★★★★★★★★★★★★  兴趣（I）
    │★★★★★★★★★★★★★★★  欲望（D）
    │★★★★★★★★★★  记忆（M）
    │★★★★★  行动（A）
    └──────────────────────────────────→ 时间/触点
```

### 1.2 AIDMA的演进

```
AIDMA（传统）→ AISCE（互联网）→ AIPL（电商）→ AFS（A面）
                                        ↓
                    社交时代：关注（Attention）→ 互动（Interaction）→ 分享（Share）→ 传播（传播）
```

## 👁️ 二、第一阶段：注意（Attention）

### 2.1 注意力获取策略

| 策略类型 | 做法 | 示例 |
|----------|------|------|
| **尺寸放大** | 增大视觉元素 | 大标题、大图片 |
| **色彩对比** | 使用高对比色 | 黑底黄字 |
| **动态吸引** | 动画、视频、动效 | 视频广告 |
| **悬念设计** | 制造好奇 | 悬念文案 |
| **名人效应** | KOL、明星代言 | 明星海报 |

### 2.2 注意力质量评估

| 评估维度 | 说明 |
|----------|------|
| **触达率** | 多少人看到了 |
| **关注率** | 多少人真正注意到 |
| **停留时间** | 消费者看了多久 |
| **首因效应** | 最初几秒的印象 |

### 2.3 Attention阶段关键指标

```
Attention KPI：
- 曝光量（Impressions）
- 点击率（CTR）
- 完播率（Video Completion Rate）
- 首次触达成本（CPM）
```

---

## 💡 三、第二阶段：兴趣（Interest）

### 3.1 兴趣激发策略

| 策略类型 | 做法 | 示例 |
|----------|------|------|
| **内容价值** | 提供有用信息 | 教程、攻略 |
| **故事叙事** | 讲述品牌故事 | 情感故事 |
| **互动参与** | 问答、游戏、测试 | 互动活动 |
| **场景关联** | 关联消费者场景 | 使用场景 |
| **问题解决** | 解决消费者痛点 | 问题解答 |

### 3.2 兴趣维持技巧

| 技巧 | 说明 |
|------|------|
| **分段信息** | 逐步释放信息 |
| **悬念递进** | 信息层层递进 |
| **互动反馈** | 即时回应用户 |
| **社交证明** | 展示其他用户兴趣 |

### 3.3 Interest阶段关键指标

```
Interest KPI：
- 页面停留时间
- 互动率（点赞、评论、收藏）
- 内容分享率
- 关注转化率
```

---

## 🔥 四、第三阶段：欲望（Desire）

### 4.1 欲望激发策略

| 策略类型 | 做法 | 示例 |
|----------|------|------|
| **痛点放大** | 强调问题严重性 | 痛点场景 |
| **解决方案展示** | 展示产品如何解决 | 产品功效 |
| **效果对比** | Before/After对比 | 效果图 |
| **社会证明** | 用户好评、案例 | 口碑展示 |
| **稀缺紧迫** | 限时限量 | 倒计时 |

### 4.2 欲望与需求的关系

```
需求 → 欲望 → 行动
  ↓       ↓       ↓
功能性  情感化   购买
需求    渴望
```

### 4.3 Desire阶段关键指标

```
Desire KPI：
- 加购率
- 收藏率
- 询盘转化率
- 试用申请率
```

---

## 🧠 五、第四阶段：记忆（Memory）

### 5.1 记忆强化策略

| 策略类型 | 做法 | 示例 |
|----------|------|------|
| **重复曝光** | 多触点持续触达 | 品牌广告 |
| **记忆锚点** | 建立品牌符号 | 视觉锤 |
| **情感连接** | 建立情感记忆 | 品牌故事 |
| **行动指令** | 明确下一步行动 | CTA按钮 |
| **会员体系** | 建立关系绑定 | 会员日 |

### 5.2 记忆类型与营销

| 记忆类型 | 营销应用 |
|----------|----------|
| **陈述性记忆** | 品牌故事、卖点 |
| **程序性记忆** | 使用习惯、操作步骤 |
| **情感记忆** | 品牌体验、情感连接 |

### 5.3 Memory阶段关键指标

```
Memory KPI：
- 品牌认知度
- 品牌提及率（BOT）
- 无提示recall
- 有提示recall
```

## 🎬 六、第五阶段：行动（Action）

### 6.1 行动转化策略

| 策略类型 | 做法 | 示例 |
|----------|------|------|
| **降低门槛** | 简化流程、减少步骤 | 一键购买 |
| **行动激励** | 优惠、赠品、积分 | 首单优惠 |
| **信任建立** | 保障、评价、认证 | 七天无理由 |
| **场景触发** | 场景化提醒 | 购物车提醒 |
| **紧迫营造** | 限时优惠 | 倒计时 |

### 6.2 行动障碍排查

| 障碍类型 | 表现 | 解决方案 |
|----------|------|----------|
| **信任障碍** | 不放心 | 背书、评价、保障 |
| **价格障碍** | 太贵 | 分期、优惠券 |
| **操作障碍** | 太复杂 | 简化流程 |
| **时机障碍** | 不是购买时机 | 场景触发 |
| **支付障碍** | 支付不便 | 多支付方式 |

### 6.3 Action阶段关键指标

```
Action KPI：
- 转化率（CR）
- 订单完成率
- 支付成功率
- ROI/ROAS
```

## 💡 九、核心金句

1. **"AIDMA不是线性的，消费者可能在任何阶段进出。"**
2. **"注意力是入口，行动是出口，中间环节决定转化效率。"**
3. **"最好的营销是让消费者'不知不觉'走完AIDMA全程。"**
4. **"每个阶段的触点设计都要对应该阶段的用户心理。"**
5. **"AIDMA模型需要随媒介和用户行为变化而迭代。"**
