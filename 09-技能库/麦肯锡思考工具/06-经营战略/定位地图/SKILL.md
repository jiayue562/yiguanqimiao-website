# 定位地图 · 竞争定位可视化分析

## 🎯 一、核心定义

**定位地图（Positioning Map）**是一种可视化分析工具，通过两个或多个维度展示市场中各竞争者的定位，帮助识别定位空间和差异化机会。

### 1.1 定位地图结构

```
定位地图示例（价格×品质）：
                  高品质
                    │
         品牌A ●    │    ● 品牌B
                    │
         品牌D ●    │    ● 品牌C
                    │
                    └────────────→ 高价格
              低品质        高价格
```

### 1.2 定位地图维度

| 类型 | 示例维度 |
|------|---------|
| **功能维度** | 价格/品质、功能/便利性 |
| **情感维度** | 年轻/成熟、高端/大众 |
| **使用维度** | 专业/入门、便携/桌面 |
| **价值维度** | 性价比/品牌溢价 |

### 1.3 定位地图的战略价值

```
定位地图价值：
┌─────────────────────────────────────┐
│  可视化   →  直观看到竞争格局        │
│  空间识别 →  发现定位空白           │
│  差异化   →  找到差异化方向         │
│  决策支持 →  支持定位战略选择       │
└─────────────────────────────────────┘
```

## 📊 二、定位地图制作

### 2.1 维度选择

| 选择原则 | 说明 |
|----------|------|
| **消费者关注** | 消费者真正关心的维度 |
| **区分度** | 维度能区分不同品牌 |
| **可测量** | 维度可以客观衡量 |
| **战略相关** | 维度与战略选择相关 |

### 2.2 维度组合推荐

| 定位类型 | 维度1 | 维度2 |
|----------|-------|-------|
| **功能定位** | 价格 | 品质/功能 |
| **情感定位** | 高端 | 大众 |
| **使用定位** | 专业度 | 易用性 |
| **价值定位** | 性价比 | 品牌 |

### 2.3 定位地图制作流程

```
定位地图制作流程：
步骤1：确定分析维度
       - 消费者调研
       - 竞品分析
       ↓
步骤2：收集定位数据
       - 消费者感知数据
       - 客观属性数据
       ↓
步骤3：绘制定位地图
       - 标注各品牌位置
       - 标注目标位置
       ↓
步骤4：分析定位空间
       - 识别空白区域
       - 分析拥挤区域
```

## 🎯 三、定位分析

### 3.1 定位空间分析

```
定位空间分析：
                  维度A
                    │
         拥挤区   │    机会区
         (红海)    │   (蓝海)
          ●●●      │
          ●●●      │
          ●●●      │
低维度B ────────────┼─────────── 高维度B
                    │
```

### 3.2 定位机会识别

| 机会类型 | 说明 |
|----------|------|
| **空白定位** | 没有品牌占据的位置 |
| **边缘定位** | 现有定位的边缘区域 |
| **重新定位** | 重新定义竞争维度 |
| **分众定位** | 针对特定细分群体 |

### 3.3 定位拥挤度分析

| 拥挤度 | 品牌数量 | 策略建议 |
|--------|---------|----------|
| **高度拥挤** | >5个品牌 | 避免正面竞争 |
| **中度拥挤** | 3-5个品牌 | 差异化定位 |
| **低度拥挤** | 1-2个品牌 | 可进入或加强 |
| **空白** | 无品牌 | 创新定位机会 |

## 🔄 五、定位地图应用

### 5.1 新品牌定位

| 应用 | 说明 |
|------|------|
| **空白进入** | 进入没有品牌的定位区域 |
| **边缘切入** | 从现有品牌的边缘进入 |
| **重新定义** | 创造新的维度打破现有格局 |
| **细分定位** | 针对特定小众人群 |

### 5.2 品牌重新定位

| 应用 | 说明 |
|------|------|
| **差异化** | 从拥挤区域移动到差异化区域 |
| **升级** | 向高端定位升级 |
| **扩展** | 向空白区域扩展 |
| **收缩** | 聚焦特定定位 |

### 5.3 定位地图监测

| 监测内容 | 频率 |
|----------|------|
| 竞争对手位置变化 | 季度 |
| 消费者感知变化 | 半年度 |
| 新进入者 | 实时 |
| 市场空白出现 | 持续 |
