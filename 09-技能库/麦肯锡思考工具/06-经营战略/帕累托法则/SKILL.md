# 帕累托法则 · 80/20效率法则

## 🎯 一、核心定义

**帕累托法则（Pareto Principle）**源于意大利经济学家维尔弗雷多·帕累托的观察：**80%的结果来自20%的原因**。这一法则广泛应用于资源优化、问题诊断和时间管理。

### 1.1 80/20法则的本质

```
80/20法则核心：
┌─────────────────────────────────────┐
│  80%的产出 ←── 20%的投入             │
│  80%的问题 ←── 20%的原因             │
│  80%的价值 ←── 20%的客户             │
│  80%的效果 ←── 20%的努力             │
└─────────────────────────────────────┘
```

### 1.2 帕累托分布

```
帕累托分布图：
结果
  │
  │    ████  ← 20%的投入
  │  ████
  │███           ← 80%的结果
  │███
  │███
  └──────────────────────→ 投入
```

### 1.3 帕累托法则的价值

```
帕累托法则价值：
┌─────────────────────────────────────┐
│  聚焦关键   →  识别最重要的20%        │
│  效率提升   →  用20%努力获得80%结果   │
│  资源优化   →  资源配置到关键少数     │
│  持续改进   →  找到杠杆解            │
└─────────────────────────────────────┘
```

## 📊 二、帕累托分析

### 2.1 帕累托分析步骤

```
帕累托分析流程：
步骤1：收集数据
       - 确定分析对象
       - 收集相关数据
       ↓
步骤2：分类统计
       - 按类别分组
       - 计算各类占比
       ↓
步骤3：排序分析
       - 按影响大小排序
       - 计算累计占比
       ↓
步骤4：识别关键
       - 找出累计80%的20%
```

## 📋 四、帕累托决策框架

### 4.1 识别关键少数

```
帕累托决策流程：
┌─────────────────────────────────────┐
│  Step 1：列出所有要素                │
│  Step 2：量化每个要素的贡献          │
│  Step 3：按贡献排序                 │
│  Step 4：找到80%累计点              │
│  Step 5：识别关键少数（20%）         │
│  Step 6：资源配置优先投入            │
└─────────────────────────────────────┘
```

### 4.2 资源配置矩阵

```
资源配置矩阵：
              高影响
                 │
     重点投入   │    持续投入
    （A类20%）  │   （识别出的关键）
                 │
低效率 ──────────┼─────────── 高效率
                 │
     减少投入    │    优化投入
    （C类）      │   （B类）
                 │
              低价值
```

### 4.3 行动优先级

| 优先级 | 类别 | 策略 | 资源配置 |
|--------|------|------|----------|
| P1 | A类（80%） | 重点投入 | 70-80% |
| P2 | B类 | 优化效率 | 15-20% |
| P3 | C类（20%） | 减少/外包 | 5-10% |
