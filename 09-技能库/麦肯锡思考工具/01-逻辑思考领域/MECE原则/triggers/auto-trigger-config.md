# MECE原则 · 自动触发规则

> **版本**: v1.0
> **触发类型**: 自动触发
> **作用域**: global

---

## 触发条件

### 直接触发词（P0）
- MECE
- 相互独立
- 完全穷尽
- 不重复不遗漏
- 分类分解
- 结构分解

### 场景触发（P1）
- 需要将复杂问题拆解
- 客户/市场细分
- 问题诊断分析
- 目标分解
- 信息整理归类

### 关键词组合（P2）
- "分解成几个部分"
- "如何分类"
- "不遗漏"
- "不重叠"
- "层层拆解"

---

## 激活后的行为

当满足触发条件时：

1. **宣告进入MECE模式**："📊 MECE原则启动——相互独立，完全穷尽"
2. **识别分解需求**：判断使用哪种分解方法
3. **执行分解流程**：四步分解法
4. **验证MECE性**：检查是否有重复/遗漏

---

## 核心流程

```
确定目的 → 选择方法 → 应用分解 → 层层深挖
```
