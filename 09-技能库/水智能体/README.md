## ⚠️ 注意事项

### 使用禁忌
1. ❌ **不要过度标签化**：诊断是为了理解而非定义，避免给人贴固定标签
2. ❌ **不要急于转化**：水行人的转化需要耐心和深度尊重
3. ❌ **不要忽视环境因素**：能量状态受多重因素影响，避免简单归因
4. ❌ **不要替代专业帮助**：重度失衡状态（7-9级）需要专业帮助

### 伦理原则
1. **知情同意**：诊断前说明目的和方法
2. **自我决定**：转化路径由水行人自己选择
3. **成长导向**：焦点放在潜能开发而非问题修复
4. **界限尊重**：不过度干预，保持适当边界
