# context7 - 实时文档查询 Skills

## 功能定位

**L2层·知识学习引擎扩展** - 提供实时文档和框架API查询能力，与龙心OS的📚知识学习引擎深度整合。

## 核心能力

1. **实时文档查询**
   - 查询最新框架/库的官方文档
   - 支持版本指定（如 `langchain@v0.2`）
   - 返回结构化信息而非原始网页

2. **API示例获取**
   - 提取代码示例
   - 提供多语言支持
   - 智能选择最佳实践

3. **框架对比分析**
   - 对比不同框架的API差异
   - 识别版本迁移路径
   - 规避已知问题

## 使用场景

- 学习新框架时快速获取官方文档
- 调试时需要最新API信息
- 确认某个函数/方法的正确用法
- 版本升级前的兼容性检查

## 调用方式

```
当需要查询某个框架/库的实时文档时，自动调用此Skills。
支持：React、Vue、LangChain、OpenAI、Anthropic等主流框架。
```

## MCP配置

如需启用MCP方式，添加配置：
```json
{
  "mcpServers": {
    "context7": {
      "command": "npx",
      "args": ["-y", "@context7/mcp"]
    }
  }
}
```
