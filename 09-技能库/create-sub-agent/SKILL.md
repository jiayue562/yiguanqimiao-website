# Create Sub-Agent Skill - 交互式子代理创建

## 技能概述
交互式创建专用子代理技能，根据用户需求动态创建专用子代理来处理特定任务。

## 核心功能

### 1. 子代理创建
- **需求分析**：分析需要什么类型的子代理
- **配置生成**：生成子代理的配置文件
- **初始化**：初始化子代理的上下文和记忆

### 2. 子代理管理
- **生命周期**：子代理的创建、运行、销毁
- **状态监控**：监控子代理的执行状态
- **资源管理**：控制子代理的资源使用

### 3. 交互式创建
- **引导式创建**：通过问答引导用户完成创建
- **模板选择**：提供预定义的子代理模板
- **自定义配置**：支持高级用户自定义配置

## 触发关键词
- 创建子代理、新建代理
- 子代理、专用代理
- 代理配置、代理模板

## 预定义模板

### 1. 研究代理
```
用途：信息收集、文献调研
技能：web-search, knowledge-learning
配置：深度搜索模式
```

### 2. 分析代理
```
用途：数据分析、模式识别
技能：data-analyzer, five-color-thinking
配置：详细分析模式
```

### 3. 创作代理
```
用途：内容创作、文案撰写
技能：creative-writing, five-color-thinking
配置：创新模式
```

## 创建流程

```
1. 需求输入
   ↓
2. 模板推荐
   ↓
3. 配置确认
   ↓
4. 创建代理
   ↓
5. 初始化完成
```

## 子代理通信

```yaml
通信协议:
  - 消息队列
  - 事件通知
  - 结果回调

消息格式:
  - task: 任务描述
  - context: 上下文信息
  - callback: 回调地址
```
