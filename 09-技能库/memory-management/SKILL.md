# Memory Management Skill - 分层记忆管理

## 技能概述
分层记忆管理技能，实现持久化和临时上下文管理，支持短期记忆、长期记忆和用户偏好的分层管理。

## 核心功能

### 1. 分层记忆架构
- **工作记忆**：当前会话的实时上下文（内存）
- **短期记忆**：最近N次会话的关键信息
- **长期记忆**：跨会话的持久化知识
- **用户记忆**：用户画像和偏好设置

### 2. 记忆操作
- **写入**：将信息写入对应层次的记忆
- **读取**：根据需求检索对应层次的记忆
- **遗忘**：自动清理过期或无关的记忆
- **强化**：重要信息的长期记忆巩固

### 3. 上下文管理
- **会话保持**：跨轮对话的上下文连续性
- **状态恢复**：新会话时恢复关键历史状态
- **上下文压缩**：长对话的上下文压缩和摘要

## 触发关键词
- 记忆、记住、回忆
- 上下文、上下文管理
- 长期记忆、短期记忆

## 记忆层次

| 层次 | 存储位置 | 容量 | 生命周期 |
|------|---------|------|---------|
| 工作记忆 | 内存 | 50条 | 会话内 |
| 短期记忆 | 文件 | 100条 | 7天 |
| 长期记忆 | 文件 | 无限 | 永久 |
| 用户记忆 | 文件 | 无限 | 永久 |

## 数据格式

```yaml
记忆条目:
  id: unique_id
  content: 内容
  type: working/short/long/user
  importance: 1-5
  tags: [标签列表]
  created_at: 创建时间
  accessed_at: 最后访问
  expires_at: 过期时间
```
