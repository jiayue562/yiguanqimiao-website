## 二、核心方法论

### 2.1 福格行为模型基础

**核心公式**：
```
B = M × A × P
行为 = 动机 × 能力 × 提示
```

**三要素详解**：

| 要素 | 定义 | 金行人特点 |
|------|------|-----------|
| **动机（M）** | 行为的驱动力引擎 | 对秩序、原则、卓越的追求是核心动机源 |
| **能力（A）** | 执行行为的容易程度 | 逻辑行为易，情感行为难 |
| **提示（P）** | 行为的触发开关 | 对规则性提示高度敏感 |

### 2.2 金行人五维度特质映射

| 维度 | 阳面（目标） | 阴面（问题） | 对应B=MAP |
|------|-------------|-------------|----------|
| 坚固 | 刚毅坚强 | 好胜计较 | M=证明自己强 |
| 收敛 | 含蓄持重 | 小气狭隘 | A=分享能力低 |
| 锋利 | 公正锋利 | 严苛无情 | A=共情能力低 |
| 光洁 | 智慧权威 | 虚荣自负 | M=外在认可 |
| 沉降 | 沉稳冷静 | 焦虑多疑 | A=决策能力低 |

## 三、诊断与设计方案库

### 3.1 维度一：坚固-好胜计较

**问题诊断**：

| B=MAP要素 | 问题分析 |
|-----------|---------|
| M动机 | "必须证明自己比他人更强"，对"输"感到恐惧 |
| A能力 | "寻找共赢点"、"欣赏对手优点"需要情感脑力 |
| P提示 | 内部"竞争警报"和外部"分歧场景"触发防卫 |

**微习惯方案库**：

| 方案名称 | 具体行为 | 适用场景 | 难度 |
|---------|---------|---------|------|
| 10秒黄金沉默 | 争论时默数10秒，找对方一个合理点 | 会议讨论 | ⭐ |
| 三赢表达法 | "我们都希望X目标..."框架 | 谈判沟通 | ⭐⭐ |
| 竞合转换 | 将竞争视角转为合作视角 | 团队协作 | ⭐⭐⭐ |

**锚点提示设计**：

```
锚点：在每次项目讨论会轮到我发言反驳前
行为：执行"10秒黄金沉默"并找一个合理点
庆祝：内心肯定"我保持了开放性"
```

### 3.2 维度二：收敛-小气狭隘

**问题诊断**：

| B=MAP要素 | 问题分析 |
|-----------|---------|
| M动机 | "分享即失去"，对"被超越"感到焦虑 |
| A能力 | "主动分享"触发损失厌恶心理 |
| P提示 | "求助请求"被解读为"索取" |

**微习惯方案库**：

| 方案名称 | 具体行为 | 适用场景 | 难度 |
|---------|---------|---------|------|
| 每日一微分享 | 每天分享一件极小的事 | 日常交流 | ⭐ |
| 播种式帮助 | 每周15分钟系统性分享 | 团队建设 | ⭐⭐ |
| 知识产品化 | 整理经验为Q&A条目 | 知识管理 | ⭐⭐⭐ |

**锚点提示设计**：

```
锚点：每天结束工作关闭电脑前
行为：执行今天的"微分享"
庆祝：记录"分享资产增加"
```

### 3.3 维度三：锋利-严苛无情

**问题诊断**：

| B=MAP要素 | 问题分析 |
|-----------|---------|
| M动机 | "必须立即严厉铲除错误"，对"混乱"感到愤怒 |
| A能力 | "批评前先肯定"需要极高共情能力 |
| P提示 | 观察到任何错误直接触发尖锐批评 |

**微习惯方案库**：

| 方案名称 | 具体行为 | 适用场景 | 难度 |
|---------|---------|---------|------|
| 缓冲-聚焦句式 | "我看到...可以更好...具体是..." | 反馈沟通 | ⭐ |
| SBIT反馈法 | Situation-Behavior-Impact-Target | 绩效面谈 | ⭐⭐ |
| 共情启动 | "我知道你..."开场 | 批评前 | ⭐⭐ |

**锚点提示设计**：

```
锚点：每次需要口头指出问题时
行为：先说一句共情启动语
庆祝：观察对方反应是否缓和
```

### 3.4 维度四：光洁-虚荣自负

**问题诊断**：

| B=MAP要素 | 问题分析 |
|-----------|---------|
| M动机 | "必须通过外在成就证明价值"，对"被忽视"恐惧 |
| A能力 | "赞美他人"、"承认不知"威胁自我价值感 |
| P提示 | 获得成功或遇到不如己者触发炫耀 |

**微习惯方案库**：

| 方案名称 | 具体行为 | 适用场景 | 难度 |
|---------|---------|---------|------|
| 成就谦述法 | 事实+归因模式 | 自我介绍 | ⭐ |
| 每日寻优记录 | 记录他人做得好的事 | 团队协作 | ⭐⭐ |
| 成长型思维 | "我还没学会"替换"我不会" | 学习成长 | ⭐⭐ |

**锚点提示设计**：

```
锚点：每次向他人介绍项目成果后
行为：主动询问"你有什么看法或建议？"
庆祝：记录"谦逊实践+1"
```

### 3.5 维度五：沉降-焦虑多疑

**问题诊断**：

| B=MAP要素 | 问题分析 |
|-----------|---------|
| M动机 | "所有不确定性都是威胁"，对"失控"持续焦虑 |
| A能力 | "信息不全时决策"带来失控感 |
| P提示 | 计划外变化触发焦虑反刍 |

**微习惯方案库**：

| 方案名称 | 具体行为 | 适用场景 | 难度 |
|---------|---------|---------|------|
| 1分钟焦虑快照 | 写下焦虑主题、强度、最小行动 | 决策前 | ⭐ |
| 同心圆工具 | 可控/可影响/无法控制 | 精力管理 | ⭐⭐ |
| 正念呼吸 | 5分钟身体扫描 | 焦虑时 | ⭐⭐ |

**锚点提示设计**：

```
锚点：每次感到心悸或呼吸急促时
行为：立即启动"三次深呼吸"
庆祝：记录"情绪调控有效"
```

### 4.1 步骤详解

| 步骤 | 名称 | 金行人适配要点 |

## 五、场景应用模板

### 场景模板一：决策犹豫（沉降维度）

```
【场景】克服"决策犹豫与过度准备"

愿望：提升决策效率，减少犹豫时间
黄金行为：设定"信息收集截止点"和"最小可行决策"标准

微习惯配方：
- 锚点：重复查找资料超过30分钟
- 行为：启动"15分钟决策计时器"
- 庆祝：记录"高效决策完成"

M动机：连接"时间成本也是成本"
A能力：15分钟限制+1个优势/风险简化框架
P提示：内部提示（意识到重复查找）+计时器提醒
```

### 场景模板二：团队沟通（锋利维度）

```
【场景】改善"团队沟通中的严厉感"

愿望：反馈精准但更容易被接受
黄金行为：增加"共情启动"步骤

微习惯配方：
- 锚点：需要口头指出问题时
- 行为：先说一句共情启动语
- 庆祝：观察对方反应

M动机：连接"先肯定让批评更有效"
A能力：简单开场白公式
P提示：内部提示（组织批评语言第一瞬间）
```

### 场景模板三：知识分享（收敛维度）

```
【场景】应对"不愿分享知识"

愿望：建立个人品牌同时促进团队成长
黄金行为：系统化分享非核心知识

微习惯配方：
- 锚点：每周五下午2点
- 行为：30分钟整理一个Q&A条目
- 庆祝：记录"资产增加"

M动机：连接"一次整理多次复用"
A能力：30分钟时间盒，目标明确
P提示：日历重复事件
```

## 六、诊断工具

### 6.1 B=MAP快速诊断问题

当期望行为未发生时，使用以下诊断框架：

```
【问题诊断清单】

问题描述：[期望行为是什么？]

□ 动机诊断（M）
  - 忘了价值？→ 连接阳面动机
  - 被情绪覆盖？→ 转化阴面情绪
  
□ 能力诊断（A）
  - 场景太复杂？→ 简化行为
  - 不知道怎么做？→ 提供技能培训
  
□ 提示诊断（P）
  - 锚点被错过？→ 优化锚点选择
  - 内部信号没识别？→ 训练自我觉察
```

### 6.2 五维度自测表

```
【金行人五维度行为自测】

□ 坚固维度
  - 我在讨论中是否经常想证明自己是对的？
  - 我能否在分歧中寻找共赢点？
  
□ 收敛维度
  - 我是否不愿意分享自己的核心知识？
  - 分享是否让我感到"被超越"的焦虑？
  
□ 锋利维度
  - 我的反馈是否经常让人感到"受伤"？
  - 我能否在批评前先肯定对方？
  
□ 光洁维度
  - 我是否过度追求外在认可？
  - 我能否坦然承认自己的不足？
  
□ 沉降维度
  - 我是否经常陷入过度担忧？
  - 我能否在信息不全时做出决策？
```
