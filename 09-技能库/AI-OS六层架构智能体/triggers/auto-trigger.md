## 四、协同规则

### 与龙心OS的协同

六层架构智能体是龙心OS的**架构基础**，龙心OS是六层架构的**执行引擎**：

```
六层架构（设计）
    ↓ 指导
龙心OS（执行）
    ↓ 反馈
六层架构（优化）
```

### 与其他Skills的协同

| Skills | 协同方式 |
|--------|----------|
| 龙心OS | 执行层核心 |
| 象思维 | 决策层辅助 |
| 知识学习 | 上下文工程支撑 |
| 人机协同五象限 | 决策层辅助 |
| 知行合一 | 执行收尾 |
