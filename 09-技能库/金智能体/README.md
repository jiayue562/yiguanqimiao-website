## 三、核心功能模块

### 3.1 拔阴取阳（转化技术一）

**核心口诀**：阴金恼转阳金行 + 阴金贪转阳金义

| 转化层级 | 阴面（问题） | 阳面（目标） | 实践问句 |
|:--------|:----------|:-----------|:---------|
| 心界 | 恼（烦躁挑剔） | 行（冷静分析） | "规则哪需完善？我能做些什么？" |
| 灵界 | 贪（狭隘自私） | 义（公正原则） | "见到利先问是否合道？" |

**四步法术**：
1. 觉察（看阴面）——问："我是'恼'还是'行'？"
2. 认不是（面对阴面）——承认："我又在用挑剔掩盖不安全感。"
3. 找好处（转阳面）——行动："我能为这件事找到什么价值？"
4. 明因果（升维）——智慧："这是'义'还是'贪'在主导？"

### 3.2 化克为生（转化技术二）

**金木失衡**：金克木过度 → 金生水（智慧沟通）→ 水生木（滋养创意）

**火金失衡**：火克金过度 → 水克火（冷静反思）→ 土生金（稳固抗压）

### 3.3 相生强化（转化技术三）

| 关系 | 强化实践 |
|:-----|:---------|
| 土生金 | 夯实基础、建立诚信、获取稳定支持 |
| 金生水 | 将原则转化为智慧沟通与灵活变通 |

### 3.4 福格行为模型 B=M×A×P（转化技术四）⭐【v1.7新增】

**公式**：B = M（动机）× A（能力）× P（提示）

**金行人五维度黄金微行为**：
