### 8.1 一步转运：改变能量习惯

```

    ✅ 每周舍出一样东西（不只是钱）

  【转运信号】

    当你开始"主动找好事"而非"被动等好事"时
    当你不再"计算得失"时
    → 你已经开始转运
```

### 8.2 两步改命：建立正向循环

```
【核心原理】

  正向链条：行义 → 给予 → 丰盛 → 安心 → 更加行义

【具体操作】

  【五行通关练习】
    Step 1：认不是（水生木）
      → 对自己的阴金模式说"是我的问题"
      → 打破"都是别人的错"的模式
    
    Step 2：忍辱不怨（木生火）
      → 遇到不平时，忍住不抱怨
      → 把"怨"转化为"找好处"
    
    Step 3：不怨人（火生土）
      → 完全停止抱怨任何人
      → 感恩一切发生
    
    Step 4：找好处（土生金）
      → 主动找周围人的好处
      → 建立感恩能量场
    
    Step 5：慷慨给予（金生水）
      → 舍出你认为舍不得的东西
      → 体验"给予即获得"的能量真相

```

  【水的圆转】
    智慧流通，不执不放
    变通有度，进退自如
    柔软如水，无孔不入
  
  【木的圆转】
    生发有节，条达通顺
    创造性输出与接纳并存
    边界清晰与灵活并存
  
  【火的圆转】
    热情适度，温暖有度
    照亮他人不灼伤自己
    表达情感不压抑也不爆发
  
  【土的圆转】
    承载万物，不负载过重
    接纳一切，化育生长
    稳定而不僵化，变动而不失根

【圆转境界特征】
  身：健康自在，无明显不适
  心：情绪平稳，不大起大落
  灵：价值观圆融，不偏执
  关系：和谐自然，不讨好不疏离
  事业：顺畅有成，不强求不逃避
