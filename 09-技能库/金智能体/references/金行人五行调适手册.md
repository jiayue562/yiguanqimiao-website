## 金行阴阳两面

| 类型 | 特质 | 核心表现 |
|:-----|:-----|:---------|
| **阳金（义德）** | 清明果决 | 坚守原则、仗义疏财、善于革新、精炼圆融 |
| **阴金（贪恼）** | 尖酸刻薄 | 固执计较、冷漠挑剔、好勇斗狠、贪婪占有 |

## 关键生克关系与调适

### 土生金（土 → 金）

| 维度 | 内容 |
|:-----|:-----|
| 本质 | 滋养之源——信实承载是金行原则的基础 |
| 健康状态 | 扎实的资源、稳定的支持、诚信的环境能滋养金行的原则与决断力 |
| 失衡状态 | 土弱则金虚：缺乏根基，原则模糊，权威不足 |
| 调适实践 | 夯实基础、建立诚信、获取稳定支持、重视传统与传承 |

### 金生水（金 → 水）

| 维度 | 内容 |
|:-----|:-----|
| 本质 | 产出之径——收敛决断催生智慧变通 |
| 健康状态 | 清晰的原则能催生圆融的智慧，果断的行动能带来灵活的方案 |
| 失衡状态 | 金不生水则燥：僵化教条，不懂得变通，缺乏智慧滋润 |
| 调适实践 | 将原则转化为智慧沟通、学习灵活变通、培养水之柔软 |

### 火克金（火 → 金）

| 维度 | 内容 |
|:-----|:-----|
| 本质 | 锤炼之机——热情冶炼增加韧性 |
| 健康状态 | 适度的热情、愿景与感染力可以软化金的冰冷，防止过度凝固 |
| 失衡状态 | 火过旺则灼金：烦躁不安，原则被动摇，内心持续焦虑 |
| 调适实践 | 引入热情与愿景、用利他之心软化冰冷、适度接触艺术与美 |

### 金克木（金 → 木）

| 维度 | 内容 |
|:-----|:-----|
| 本质 | 规范之力——规则约束散漫，引导创新有序 |
| 健康状态 | 必要的规则、标准与决断可以约束木行的散漫，引导创新有序 |
| 失衡状态 | 金过旺则伤木：压抑创意与活力，导致关系紧张，团队缺乏活力 |
| 调适实践 | 学会授权、尊重创意、用智慧沟通替代硬性控制、允许例外 |

## 失衡状态专项调适

### 金气过旺（阴金锋厉）

| 维度 | 内容 |
|:-----|:-----|
| 表现 | 过度批判、刻薄、控制、难以容忍不同意见 |
| 火调节 | 引入火（利他之心）：用热情和愿景软化冰冷的判断 |
| 水疏导 | 金生水：将硬性原则转化为智慧沟通，学习变通 |
| 实践 | 每天主动说一句温暖的话；练习"三问法"后再给反馈 |

### 金气不足（义理不彰）

| 维度 | 内容 |
|:-----|:-----|
| 表现 | 优柔寡断、原则模糊、缺乏权威、难以坚持立场 |
| 土生扶 | 引入土：夯实基础、建立诚信、获取稳定支持 |
| 火减少 | 减少火（热情消耗）：避免被琐事或他人情绪过度干扰 |
| 实践 | 每天明确一个底线并坚持；建立稳定的支持系统 |

### 金木交战（金克木过度）

| 维度 | 内容 |
|:-----|:-----|
| 表现 | 严厉压制团队创意、人际关系紧张、过度控制 |
| 金生水 | 先行：先将自身原则转化为圆融的沟通与理解 |
| 水生木 | 接续：通过理解和智慧去滋养对方的创意与成长 |
| 实践 | 每次想批评前，先问"对方这个创意背后的逻辑是什么？" |

### 火金相争（火克金过度）

| 维度 | 内容 |
|:-----|:-----|
| 表现 | 内心对完美的热情灼烧了冷静，导致持续烦躁、内耗 |
| 水克火 | 引入水：通过冷静反思、深度学习来降温 |
| 土生金 | 稳固：用稳定性和抗压能力增强自身原则的稳固性 |
| 实践 | 每天10分钟独处反思；建立稳定的日常工作节奏 |

## 日常五行调适实践

### 早晨（金旺时）

- 规划：制定今日最重要的3件事（强化金的决断与条理）
- 冥想：5分钟静心，感受"明镜映照"的清明

### 日间（木旺时）

- 觉察：观察是否过度控制或挑剔（识别金克木过度）
- 练习：主动倾听3次，尊重他人创意（补水生木）

### 傍晚（火旺时）

- 反思：今天是否被"恼"控制？（识别火克金失衡）
- 庆祝：肯定自己今天做得好的一件事（增强信心）

### 夜间（水旺时）

- 整理：清理桌面和思绪，删减不必要的任务（去芜存菁）
- 感恩：写下3件今天让你感到公正或被善待的事
