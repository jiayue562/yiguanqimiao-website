# MCP协议使用案例

## 📊 案例1：人机协同协议

### 场景
悟空发出高层决策指令："基于我的GEO思维模型分析AI行业趋势"

### 协议流程
1. 人通过CLI发出指令
2. CLI指令解析Skill解析个性化参数
3. Framework调度模块通过MCP人机协同协议接收指令
4. AI执行任务
5. AI通过MCP人机协同协议反馈结果给悟空

### 协议数据
见SKILL.md中的协议格式示例

---

## 📊 案例2：机内协同协议

### 场景
AI调用"个人思维模型分析Skill"，结果传递给"文案撰写Skill"

### 协议流程
1. Framework调度模块通过MCP机内协同协议调用"个人思维模型分析Skill"
2. "个人思维模型分析Skill"返回分析结果
3. Framework调度模块通过MCP机内协同协议将结果传递给"文案撰写Skill"
4. "文案撰写Skill"生成文案

### 协议数据
见SKILL.md中的协议格式示例

---

## 📊 案例3：系统扩展协议

### 场景
CodeBuddy CLI接入AI OS

### 协议流程
1. CodeBuddy CLI通过MCP系统扩展协议申请接入
2. AI OS验证CodeBuddy CLI的能力和认证信息
3. AI OS同意接入，分配调用接口
4. AI OS可以通过MCP协议调用CodeBuddy CLI的能力

### 协议数据
见SKILL.md中的协议格式示例
