## 第一层：问题类型分类

```
用户的问题属于哪种类型？
         ↓
┌─────────────────────────────────────────────────────────────┐
│ A. 理解类：为什么/本质是什么/底层逻辑                         │
│ B. 诊断类：这个人是什么状态/处于哪个层级                      │
│ C. 干预类：如何改变/转化/提升                                │
│ D. 关系类：两人/团队/家庭的互动格局                          │
│ E. 行为类：如何建立习惯/改变行为/从知道到做到                 │
│ F. 跨域类：西方科学如何看待/学术佐证                         │
│ G. 创新类：突破现有框架/0→1原创思考                          │
└─────────────────────────────────────────────────────────────┘
```

## 第二层：分类路由规则

### A. 理解类问题路由

```
用户问"为什么..."、"本质是..."、"底层逻辑..."
        ↓
是否涉及整体框架理解？
  是 → 调用 第5篇（总论·三链条）
  
是否涉及一心的本质？
  是 → 调用 第1篇（一心）

是否涉及某五行的本质？
  是 → 调用 第3篇（五行身心灵整合）+ 对应五行分智能体

是否涉及为什么某行为模式持续存在？
  是 → 调用 第2篇（三界）+ 第4篇（九层）→ 从灵界找根因
```

### B. 诊断类问题路由

```
用户描述某人/自己的状态，需要诊断
        ↓
第一步：三界诊断
  调用 第2篇（三界）
  → 判断失衡在灵界/心界/身界

第二步：五行类型确认
  调用 第3篇（五行整合）
  → 确认主五行类型

第三步：层级精准定位
  调用 第4篇（九层阶梯）
  → 1-9层精准定位

输出格式：
「三界诊断：[灵界/心界/身界 - 状态]
 五行类型：[X行人·阶段]
 层级定位：第X层·[层级名称]
 干预重点：[对应层级的干预入口]」
```

### C. 干预类问题路由

```
用户需要"如何改变/转化/提升"方案
        ↓
第一步：确定干预深度
  □ 轻度（4-6层的日常调适）→ 运象三法
  □ 中度（长期模式改变）→ 拔阴取阳+B=MAP整合
  □ 深度（7-9层的稳定化）→ 先调用第2篇灵界干预

第二步：选择干预工具
  阴面转化 → 调用 第8篇（拔阴取阳）
  冲突转化 → 调用 第9篇（化克为生）
  阳面巩固 → 调用 第10篇（相生强化）
  行为落地 → 调用 第12篇（B=MAP）
  
第三步：完整方案设计（深度干预）
  调用 第8篇 + 第9篇 + 第10篇 = 运象三法完整套餐
  + 第12篇（B=MAP落地）
```

### D. 关系类问题路由

```
用户问两人/团队/家庭的互动格局
        ↓
第一步：五行格局分析
  调用 第6篇（相生）+ 第7篇（相克）
  → 判断是相生还是相克格局

第二步：层级匹配分析
  调用 第4篇（九层）
  → 两人/各成员分别处于哪个层级？

第三步：关系调谐方案
  相生关系需要强化 → 调用 第6篇（相生强化策略）
  相克关系需要调谐 → 调用 第9篇（化克为生）
  
输出：[关系格局图] + [调谐路径] + [具体行动]
```

### E. 行为类问题路由

```
用户问"如何建立习惯"、"如何坚持"、"知道但做不到"
        ↓
第一步：B=MAP诊断
  调用 references/03-B=MAP行为设计工具箱.md
  → 行为失败分析矩阵（M/A/P哪一界失衡）
  
第二步：五行专属设计
  确认五行类型后：
  → 调用 B=MAP工具箱中"五行专属锚点推荐库"
  → 调用 B=MAP工具箱中"五行专属庆祝设计"
  
第三步：与五行干预整合
  如果是阴面行为反复 → 调用 第8篇（拔阴）整合
  如果是克制关系导致 → 调用 第9篇（化克）整合
  
输出标准格式：
「⚡ B=MAP行为设计方案
 M（动机·灵界）：[诊断+激活方案]
 A（能力·心界）：[诊断+降门槛方案]
 P（提示·身界）：[锚点设计]
 微习惯：[最小版本]
 庆祝仪式：[即时强化方案]」
```

### F. 跨域类问题路由

```
 西方侧：[概念/学科]

```
