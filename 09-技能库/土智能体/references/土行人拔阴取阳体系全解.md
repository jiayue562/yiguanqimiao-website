## 4. 核心心法

### 4.1 辞让

**定义**：不报委屈，多找自己的问题

**原理**：辞让是火生土的唯一桥梁

**操作**：
```
遇到不满 → 暂停 → 找自己问题 → 辞让
```

**实践**：
- 当感到委屈时，问："我哪里做得不好？"
- 当感到不公时，问："我能做什么改变？"
- 当感到怨恨时，问："我是否也伤害过他人？"

### 4.2 土生金

**定义**：将积累转化为有价值产出

**原理**：土不生金则壅，只囤积不知转化

**操作**：
```
积累（土）→ 提炼（金）→ 产出（价值）
```

**实践**：
- 将经验总结成方法论
- 将资源投入有意义的事业
- 将信任转化为行动

### 4.3 信实践行

**定义**：建立对他人、规律、生命的基本信任

**原理**：阴土疑转阳土信是根本转化

**操作**：
```
每日信任练习 → 积累信任 → 形成信实人格
```

**实践**：
- 每天说一句信任他人的话
- 每周兑现一个小承诺
- 每月尝试一次"先相信，后验证"

## 5. 三界操作技术

| 技术 | 操作 | 目的 |
|------|------|------|
| **运动疏通** | 太极、瑜伽、舞蹈等灵活运动 | 打破身体僵滞 |
| **经络调理** | 按摩足三里等脾胃经络 | 疏通土行能量 |
| **呼吸练习** | 腹式深呼吸，感受大地连接 | 稳定与放松 |

|------|------|------|

| **感恩练习** | 每天感谢三件事 | 培养阳土 |
| **提问训练** | "我能做什么"而非"为什么不" | 建设导向 |

| 技术 | 操作 | 目的 |
|------|------|------|
| **价值观反思** | 每日反思：我是信还是疑？ | 校准方向 |
