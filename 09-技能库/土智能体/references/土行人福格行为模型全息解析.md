## 1. 理论核心

### 1.1 福格行为模型

B = M × A × P

| 要素 | 含义 | 英文 |
|------|------|------|
| **B** | 行为 | Behavior |
| **M** | 动机 | Motivation |
| **A** | 能力 | Ability |
| **P** | 提示 | Prompt |

### 1.2 土行核心

"治本而非治标：通过滋养阳面，让阴面特质自然消减"

### 1.3 核心公式

```
B = M × A × P
├── B (Behavior)：具体、可观察的行动
├── M (Motivation)：PAC框架（人物/行动/情境）
└── A (Ability)：能力链五维度（时间/资金/体力/脑力/日程）
└── P (Prompt)：三种类型（人物/情境/行动提示）
```

## 3. 五维度行为设计

### 3.1 承载维度

| 项目 | 内容 |
|------|------|
| **微习惯** | "每天接纳一个不同意见" |
| **动机锚点** | 当遇到不同意见时 |
| **能力设计** | 从小事开始，如接纳同事的点子 |
| **提示设计** | 设置手机提醒 |
| **庆祝方式** | 告诉自己"我今天又包容了" |

### 3.2 稳定维度

| 项目 | 内容 |
|------|------|
| **微习惯** | "每天做一件小事的不确定性" |
| **动机锚点** | 当感到僵化时 |
| **能力设计** | 从小事开始，如换一条路回家 |
| **提示设计** | 每周三固定尝试新事物 |
| **庆祝方式** | 告诉自己"我今天打破了舒适区" |

### 3.3 思虑维度

| 项目 | 内容 |
|------|------|
| **微习惯** | "每天问一个'还有什么可能'" |
| **动机锚点** | 当钻牛角尖时 |
| **能力设计** | 准备一个问题清单 |
| **提示设计** | 贴在电脑旁 |
| **庆祝方式** | 告诉自己"我又拓展了思维" |

### 3.4 务实维度

| 项目 | 内容 |
|------|------|
| **微习惯** | "每周输出一个成果" |
| **动机锚点** | 当想要拖延时 |
| **能力设计** | 从5分钟开始 |
| **提示设计** | 每天早上第一个任务 |
| **庆祝方式** | 记录成果，感受踏实 |

### 3.5 信实维度

| 项目 | 内容 |
|------|------|
| **微习惯** | "每周兑现一个小承诺" |
| **动机锚点** | 当想要逃避承诺时 |
| **能力设计** | 承诺力所能及的事 |
| **提示设计** | 记录承诺清单 |
| **庆祝方式** | 划掉完成的承诺，感受诚信 |

## 4. 行为设计七步法

### 4.1 步骤概览

| 步骤 | 内容 | 连接 |
|------|------|------|
| 1 | 明确愿望 | 连接土之使命 |
| 2 | 探索行为选项 | 发散思维 |
| 3 | 匹配黄金行为 | 高影响×高动机×高能力 |
| 4 | 从微习惯开始 | 简单到不会失败 |
| 5 | 找到对的提示 | 利用高频率锚点 |
| 6 | 庆祝成功 | 点燃"踏实"感觉 |
| 7 | 排除障碍 | 实现"可持续承载" |

### 4.2 详细说明

**第一步：明确愿望**
```
连接到土之使命："厚德载物"
问自己："我想培养什么土行特质？"
```

**第二步：探索行为选项**
```
发散思维，列出10种可能的行为
不评判，不筛选
```

**第三步：匹配黄金行为**
```
高影响力：能显著提升土行能量
高动机：内心真正想要
高能力：能够做到
```

**第四步：从微习惯开始**
```
简单到不会失败
例如：每天5分钟、每天1次
```

**第五步：找到对的提示**
```
利用高频率锚点
时间锚点、情境锚点、行动锚点
```

**第六步：庆祝成功**
```
每次完成都庆祝
点燃"踏实"感觉
强化行为模式
```

**第七步：排除障碍**
```
预见可能的障碍
提前制定应对方案
实现"可持续承载"
```
