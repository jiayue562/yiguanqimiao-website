## 三、指令实现手册

### 指令一：写入记忆

**触发词**：写入记忆、帮我记住、把...写入记忆

**实现逻辑**：
```
1. 解析用户要记忆的内容
2. 写入MEMORY.md（长期记忆）
3. 写入当日日志（短期记忆）
4. 可选：同步到IMA笔记
```

**输出格式**：
```
✅ 已将以下内容写入长期记忆：
[内容摘要]
存储位置：MEMORY.md
```

### 指令二：调取记忆

**触发词**：调取记忆、读取记忆、告诉我你记住了什么

**实现逻辑**：
```
1. 读取MEMORY.md
2. 搜索相关主题
3. 输出结构化记忆
```

**输出格式**：
```
📚 关于[主题]的长期记忆：
1. [记忆点1]
2. [记忆点2]
3. [记忆点3]
来源：MEMORY.md
```

### 指令三：记忆蒸馏

**触发词**：记忆蒸馏、压缩记忆、整理记忆

**实现逻辑**：
```
1. 读取完整MEMORY.md
2. 识别重复、过时、冗余内容
3. 压缩为精炼版本
4. 保留核心信息
```

### 指令四：联网搜索

**触发词**：联网搜索、搜索最新、帮我搜索

**实现逻辑**：
```
1. 调用web_search获取最新信息
2. 提炼核心结论（结论优先）
3. 标注数据来源
4. 给出可行动洞察
```

**输出格式**：
```
📊 [话题]最新信息

【核心结论】
[一句话结论]

【关键数据】
- [数据点1]
- [数据点2]

【行动建议】
[可直接执行的建议]

来源：[搜索渠道]
```

### 指令五：深度研究

**触发词**：深度研究、深入分析、系统研究

**实现逻辑**：
```
1. 多角度搜索（web_search多轮）
2. 知识学习引擎深度分析
3. 四维输出：现状+数据+影响+行动
```

**输出格式**：
```
🔬 [话题]深度研究报告

一、现状与趋势
[当前格局+未来走向]

二、核心数据与案例
[关键数据+典型案例]

三、对我的直接影响
[与悟空业务的关联]

四、行动建议
[可执行的下一步]
```

### 指令六：读取本地文件

**触发词**：读取文件、提炼内容、分析文件

**实现逻辑**：
```
1. 识别文件格式
2. 调用对应技能读取
3. 提炼核心内容
4. 给出行动建议
```

**输出格式**：
```
📄 文件分析：[文件名]

【核心结论】
[一句话概括]

【关键信息】
- [要点1]
- [要点2]
- [要点3]

【行动建议】
[基于文件内容的建议]
```

### 指令七：批量读取知识库

**触发词**：知识库检索、读取知识库、批量读取

**实现逻辑**：
```
1. 调用obsidian技能搜索
2. 整合所有相关笔记
3. 输出结构化知识图谱
```

**输出格式**：
```
🗺️ [主题]知识图谱

【总体架构】
[主题的整体框架]

【核心结论】
1. [结论1]
2. [结论2]

【知识节点】
- [节点1] ← [关系] → [节点2]
- [节点3] ← [关系] → [节点4]

【行动建议】
[基于知识整合的建议]
```

### 指令八：每日知识沉淀

**触发词**：知识沉淀、每日总结、对话总结

**实现逻辑**（对话结束自动触发）：
```
1. 回溯今日对话
2. 提取新知识
3. 识别被验证/修正的偏好
4. 写入MEMORY.md
```

**输出格式**：
```
🔄 每日知识沉淀

【今日新知】
1. [新知识1]
2. [新知识2]

【长期保留结论】
- [结论1]
- [结论2]

【偏好确认/修正】
- [确认的偏好]
- [修正的方法]

已写入MEMORY.md ✓
```

### 指令九：定时学习任务

**触发词**：设置定时任务、每周总结、自动学习

**实现逻辑**：
```
1. 创建自动化任务
2. 设置执行频率
3. 定义任务内容
4. 启动任务
```

**当前可设置的任务**：
- 每周知识蒸馏（周一9:00）
- 每月记忆归档
- 每日23:30自动备份
