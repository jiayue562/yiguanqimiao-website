# 9条指令完整实现

## 指令一：写入记忆

### 触发词
- 写入记忆、帮我记住、把...写入记忆
- 记住这个、这个很重要请记住

### 实现代码
```
1. 解析用户输入，提取需要记忆的内容
2. 判断内容类型：
   - 用户偏好 → 写入"偏好"板块
   - 新知识 → 写入"知识"板块
   - 任务/约定 → 写入"待办"板块
3. 写入MEMORY.md（追加到相关板块）
4. 写入当日日志（时间线格式）
5. 可选：同步到IMA笔记

输出示例：
✅ 已将以下内容写入长期记忆：
【类型】用户偏好
【内容】悟空喜欢简洁直接的沟通方式，不喜欢冗长的铺垫
【时间】2026-03-28
存储位置：MEMORY.md
```

### 存储格式
```markdown
## 偏好
- 2026-03-28: 悟空喜欢简洁直接的沟通方式

## 知识
- 2026-03-28: [知识点]

## 待办
- 2026-03-28: [待办事项]
```

### 实现代码

| 任务名 | 频率 | 功能 |
