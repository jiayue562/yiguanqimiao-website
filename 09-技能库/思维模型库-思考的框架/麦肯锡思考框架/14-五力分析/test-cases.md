# 五力分析 · 测试用例

## 测试场景一：新能源汽车行业分析

### 输入
情境：某投资机构评估新能源汽车行业吸引力

### 期望输出
```
【五力分析 · 新能源汽车行业】

■ 供应商议价 ★★★☆☆
 - 电池供应商集中（宁德时代等）
 - 但多家供应商竞争
 - 结论：中等威胁

■ 买方议价 ★★☆☆☆
 - 个人消费者分散
 - 产品差异化明显，品牌溢价
 - 结论：低威胁

■ 新进入者 ★★★☆☆
 - 资本壁垒高但政策支持
 - 科技公司纷纷入局
 - 结论：中等威胁

■ 替代品 ★★☆☆☆
 - 燃油车仍是主流
 - 但禁售时间表逐步明确
 - 结论：低威胁（逐步上升）

■ 业内竞争 ★★★★★
 - 特斯拉、比亚迪、蔚小理
 - 竞争激烈，价格战
 - 结论：极高威胁

【行业吸引力】★★★☆☆（中等）
投资策略：选择有技术壁垒和品牌优势的龙头企业
```

## 测试场景二：在线教育行业分析

### 输入
情境：某创业公司评估K12在线教育赛道

### 期望输出
```
【五力分析 · K12在线教育】

■ 供应商议价 ★★☆☆☆
 - 教师资源分散
 - 在线教师供给充足
 - 结论：低威胁

■ 买方议价 ★★★★☆
 - 家长价格敏感
 - 信息透明容易比价
 - 结论：高威胁

■ 新进入者 ★★★★☆
 - 门槛低，技术成熟
 - 大量创业公司涌入
 - 结论：高威胁

■ 替代品 ★★★☆☆
 - 线下培训仍是主要替代
 - 公办学校延时服务
 - 结论：中等威胁

■ 业内竞争 ★★★★★
 - 双减后格局重塑
 - 新东方、好未来等巨头
 - 结论：极高威胁

【行业吸引力】★★☆☆☆（低吸引力）
战略建议：差异化定位 + 提升用户粘性
```
