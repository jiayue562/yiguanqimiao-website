# 🧪 安索夫矩阵 · 测试用例

## 测试场景一：咖啡品牌增长战略

### 输入
**情境**：某精品咖啡品牌在北上广深已有50家门店，年营收2亿，需要选择下一步增长战略

### 期望输出
```
【安索夫矩阵 · 增长战略选择】

■ 现状诊断
  产品：精品咖啡+轻食
  市场：一线城市核心商圈
  市场份额：约5%
  市场增速：15%/年

■ 四象限分析

  ┌────────────────┬────────────────┐
  │   市场开发      │    产品开发     │
  │   （新市场）    │   （新产品）    │
  │  → 下沉市场     │  → 茶饮/甜点   │
  │  → 企业团购     │  → 咖啡胶囊    │
  │  → 咖啡学院     │  → 周边产品    │
  ├────────────────┼────────────────┤
  │   多元化        │   市场渗透     │
  │   （新市场+）   │   （现有）     │
  │  → 海外市场     │  → 提升复购    │
  │  → 便利店渠道   │  → 会员运营    │
  └────────────────┴────────────────┘

■ 战略推荐

  优先级1：市场渗透（风险最低）
    - 重点：会员体系升级、数字化运营
    - 预期：提升客单价15%，复购率20%

  优先级2：产品开发（协同最强）
    - 新品：茶饮系列（已具备研发能力）
    - 逻辑：现有客群有茶饮需求

  优先级3：市场开发（中期重点）
    - 新市场：二线城市标杆店
    - 逻辑：品牌势能可复制

  优先级4：多元化（暂缓）
    - 原因：主业尚未建立绝对优势
```

### 验证标准
- [ ] 四象限都有策略输出
- [ ] 优先级有数据支撑
- [ ] 风险评估合理

## 测试场景二：软件公司产品线扩展

### 输入
**情境**：某SaaS公司核心产品"CRM系统"已有500家客户，年收入1亿，需制定产品延伸战略

### 期望输出
```
【安索夫矩阵 · 产品线扩展战略】

■ 现状诊断
  核心产品：CRM客户管理系统
  客户群体：中小企业（50-500人）
  地域市场：全国
  客户满意度：85%

■ 战略选择

  【市场渗透】★★★
    策略：
    - 提升续费率至90%+
    - 转介绍奖励机制
    - 增购触达（更多部门）

  【产品开发】★★★★★
    新产品矩阵：
    - 营销云（与CRM协同）
    - 客服系统（服务延伸）
    - 数据分析（能力延伸）
    逻辑：客户全生命周期需求

  【市场开发】★★★
    新市场：
    - 大型企业版（功能定制）
    - 行业版本（制造业/零售业）
    
  【多元化】★
    仅考虑相关多元化：
    - HR SaaS（人力资源管理）
    - 协同办公（企业服务生态）
```

### 验证标准
- [ ] 产品开发作为核心优先级
- [ ] 协同效应分析到位
- [ ] 风险分层合理

## 测试场景三：连锁餐饮扩张战略

### 输入
**情境**：某川菜连锁品牌在本地城市有30家门店，需要选择扩张路径

### 期望输出
```
【安索夫矩阵 · 餐饮扩张战略】

■ 现状诊断
  产品：中式正餐（川菜）
  市场：本地城市
  门店模型：150-200㎡，客单价60元
  日翻台率：3次

■ 战略分析

  【市场渗透】★★★★☆
    短期首选：
    - 优化菜单，提升点餐效率
    - 会员系统，提升复购
    - 外卖业务，拓展场景
    预期：营收增长20%

  【产品开发】★★★☆☆
    产品延伸：
    - 推出子品牌"川小面"（快餐）
    - 预制菜（零售化）
    - 定制礼盒（节庆消费）
    逻辑：川味生活方式全覆盖

  【市场开发】★★★★☆
    地域扩张：
    - 第一步：省会城市（3-5家）
    - 第二步：周边城市（加盟模式）
    - 第三步：一线城市（品牌升级）
    逻辑：区域品牌→全国品牌

  【多元化】★★☆☆☆
    谨慎探索：
    - 相关：火锅、烧烤（餐饮品类延伸）
    - 无关：暂不考虑
```

### 验证标准
- [ ] 餐饮行业特点体现（翻台率、外卖）
- [ ] 扩张节奏清晰
- [ ] 加盟模式考量
