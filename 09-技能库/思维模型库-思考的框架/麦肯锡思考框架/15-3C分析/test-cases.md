# 3C分析 · 测试用例

## 测试场景一：新能源汽车品牌战略

### 输入
情境：某新能源汽车品牌制定市场进入战略

### 期望输出
```
【3C分析 · 新能源汽车】

■ Customer 客户
目标客群：
 - 30-45岁中高收入家庭
 - 一二线城市用户
核心需求：
 - 续航里程无忧
 - 智能驾驶体验
 - 品牌身份认同
购买决策：
 - 安全性 > 智能化 > 价格
 - 试驾体验影响大

■ Company 公司
核心能力：
 - 纯电平台技术领先
 - 智能驾驶研发
 - 用户运营经验
资源优势：
 - 传统车企制造经验
 - 渠道网络覆盖
价值主张：
 "更安全的智能电动汽车"

■ Competitor 竞争
直接竞争：
 - 特斯拉（品牌强）
 - 比亚迪（性价比）
 - 蔚来（服务好）
差异化：
 - 安全技术创新
 - 家庭用户定位
 - 本土化服务

【战略机会】
1. 家庭安全出行市场
2. 二线城市消费升级
3. 智能驾驶差异化
```

## 测试场景二：在线教育平台定位

### 输入
情境：某在线教育平台进行差异化定位分析

### 期望输出
```
【3C分析 · 在线教育平台】

■ Customer 客户
目标客群：
 - 22-35岁职业人群
 - 三四五线城市用户
核心需求：
 - 实用技能提升
 - 时间灵活
 - 性价比
购买决策：
 - 课程口碑 > 品牌
 - 完课率保障

■ Company 公司
核心能力：
 - 技术驱动的课程开发
 - 名师运营
 - 数据化教学
资源优势：
 - 低成本获客
 - 用户下沉
价值主张：
 "普通人逆袭的实战学堂"

■ Competitor 竞争
直接竞争：
 - 得到（知识付费）
 - 网易云课堂（综合）
 - 腾讯课堂（流量）
差异化：
 - 实战派师资
 - 低价高质
 - 陪伴式学习

【战略机会】
1. 下沉市场增量
2. 实用技能赛道
3. 普通人的职业教育
```
