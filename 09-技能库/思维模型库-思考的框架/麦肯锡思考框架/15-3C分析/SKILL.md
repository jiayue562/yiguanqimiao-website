> 版本: v1.0 | 创建: 2026-04-15 | 维护者: 龙龟神将

---

## 一、核心定义

### 1.1 一句话定义
3C分析通过审视**客户(Customer)、公司(Company)、竞争对手(Competitor)** 三个维度，识别市场机会与竞争定位，是战略规划的基础框架。

### 1.2 核心公式

```
3C分析 = Customer(客户) + Company(公司) + Competitor(竞争对手)
```

---

## 二、三维度详解

### 2.1 客户 (Customer)

| 分析维度 | 具体内容 |
|---------|---------|
| 客户是谁 | 目标客群、细分市场 |
| 客户需求 | 痛点、需求、优先级 |
| 客户行为 | 购买决策、使用习惯 |
| 客户价值 | 客户生命周期价值 |

**核心问题**：
- 我们的目标客户是谁？
- 他们的核心需求是什么？
- 他们的购买决策如何做出？

### 2.2 公司 (Company)

| 分析维度 | 具体内容 |
|---------|---------|
| 核心能力 | 技术、产品、服务 |
| 资源优势 | 品牌、渠道、资金 |
| 组织能力 | 执行、运营、创新 |
| 价值主张 | 我们能提供什么独特价值 |

**核心问题**：
- 我们的核心优势是什么？
- 我们能做什么竞争对手做不到的？
- 我们如何为客户创造价值？

### 2.3 竞争对手 (Competitor)

| 分析维度 | 具体内容 |
|---------|---------|
| 直接竞争 | 同类产品的竞争者 |
| 间接竞争 | 替代品的提供者 |
| 潜在竞争 | 可能进入的新玩家 |
| 竞争策略 | 他们的定位和优势 |

**核心问题**：
- 谁是我们的主要竞争对手？
- 他们的优势和劣势是什么？
- 我们与他们的差异化在哪里？

## 四、实战模板

### 场景：智能音箱市场定位

```
【3C分析 · 智能音箱】

■ Customer 客户
目标客群：
 - 25-40岁科技爱好者
 - 追求便捷生活的家庭用户
核心需求：
 - 语音交互便捷性
 - 智能家居控制
 - 音乐等内容消费
购买决策：
 - 产品体验 > 价格
 - 生态协同影响大

■ Company 公司
核心能力：
 - AI语音技术领先
 - 品牌认知度高
 - 生态布局完整
资源优势：
 - 家庭用户基础
 - 渠道覆盖广
价值主张：
 "更懂你的智能助手"

■ Competitor 竞争
直接竞争：
 - 小度（百度）
 - 天猫精灵（阿里）
 - HomePod（苹果）
差异化：
 - AI技术领先
 - 音质体验优势
 - 隐私保护

【战略机会】
1. 高端音质市场空白
2. 老年人群便捷操作需求
3. 办公场景专业需求
```
