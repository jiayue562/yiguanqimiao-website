# 愿景与使命定义

> 版本: v1.0 | 创建: 2026-04-15 | 维护者: 龙龟神将

---

## 一、核心定义

### 1.1 一句话定义
愿景与使命是企业战略方向的锚点，使命回答"我们为什么存在"，愿景描绘"我们要去向何方"。

### 1.2 核心公式

```
愿景与使命 = 使命(Mission) + 愿景(Vision) + 价值观(Values)
```

---

## 二、三要素详解

### 2.1 使命 (Mission)

**定义**：组织存在的根本目的

| 问题 | 回答 |
|------|------|
| 我们做什么？ | 产品/服务 |
| 为谁做？ | 目标客户 |
| 创造什么价值？ | 客户价值 |
| 为什么存在？ | 社会价值 |

**使命陈述模板**：
> "通过[核心能力]，为[目标客户]创造[核心价值]，从而[社会意义]。"

### 2.2 愿景 (Vision)

**定义**：组织追求的理想状态

| 问题 | 回答 |
|------|------|
| 我们追求什么？ | 长期目标 |
| 成为什么？ | 未来地位 |
| 何时实现？ | 时间框架 |
| 如何衡量？ | 成功标准 |

**愿景陈述模板**：
> "到[时间]，成为[领域]的[地位]，实现[核心指标]。

### 2.3 价值观 (Values)

**定义**：组织遵循的核心原则

| 类型 | 示例 |
|------|------|
| **做人之本** | 诚信、正直、尊重 |
| **做事之道** | 创新、客户导向、追求卓越 |
| **协作之魂** | 团队合作、开放沟通 |

## 三、诊断框架

### 3.1 使命诊断

| 诊断问题 | 判断标准 |
|---------|---------|
| 是否清晰易懂？ | 员工能复述 |
| 是否有感召力？ | 能激发热情 |
| 是否独特差异化？ | 与竞品不同 |
| 是否可执行？ | 能落地到行动 |

### 3.2 愿景诊断

| 诊断问题 | 判断标准 |
|---------|---------|
| 是否足够远大？ | 10年+视角 |
| 是否可衡量？ | 有量化指标 |
| 是否可实现？ | 有实现路径 |
| 是否被共享？ | 全员认同 |

## 四、实战模板

### 4.1 创业公司愿景使命定义

```
【愿景使命 · 科技创业公司】

使命：
通过AI技术赋能中小企业，
让人人都能享受智能服务的便利，
推动商业民主化进程。

愿景：
到2030年，
成为中小企业AI服务的首选品牌，
服务100万家企业，
让AI普惠每一个梦想。

价值观：
 - 客户成功：客户成功我们才成功
 - 创新突破：技术驱动，突破边界
 - 开放协作：知识共享，生态共赢
 - 诚信担当：言行一致，使命必达
```

### 4.2 传统企业使命升级

```
【愿景使命 · 传统零售升级】

使命（原）：
为消费者提供优质的商品和服务

使命（新）：
以科技驱动体验升级，
为每个家庭创造智慧生活的无限可能，
让美好生活触手可及。

愿景（原）：
成为区域领先的零售企业

愿景（新）：
到2030年，
成为新零售的引领者，
重新定义人货场的边界，
开创智慧零售新时代。
```
