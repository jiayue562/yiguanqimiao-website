# 用户画像

> 版本: v1.0 | 创建: 2026-04-15 | 维护者: 龙龟神将

---

## 一、核心定义

### 1.1 一句话定义
用户画像是基于调研数据构建的**半虚拟理想用户原型**，通过人口统计、行为特征、目标动机、痛点需求等维度，刻画目标用户，帮助团队建立共识、指导决策。

### 1.2 核心公式

```
用户画像 = 基本属性 + 行为特征 + 目标动机 + 痛点需求 + 关键词
```

---

## 二、画像维度

### 2.1 基础属性

| 维度 | 内容 |
|------|------|
| 人口统计 | 年龄、性别、职业、收入 |
| 地域分布 | 城市、区域 |
| 教育背景 | 学历、专业 |
| 家庭状态 | 婚姻、子女 |

### 2.2 行为特征

| 维度 | 内容 |
|------|------|
| 消费行为 | 购买频次、客单价 |
| 媒体习惯 | 常用途径、内容偏好 |
| 决策模式 | 价格敏感度、品牌忠诚 |
| 使用习惯 | 场景、频率、方式 |

### 2.3 目标与动机

| 维度 | 内容 |
|------|------|
| 核心目标 | 用户想达成什么 |
| 驱动因素 | 什么激励用户 |
| 价值观 | 用户在乎什么 |

### 2.4 痛点与需求

| 维度 | 内容 |
|------|------|
| 核心痛点 | 未满足的需求 |
| 障碍与阻碍 | 什么阻止用户 |
| 潜在担忧 | 决策风险 |

## 四、实战模板

### 场景：知识付费用户画像

```
【用户画像 · 知识付费】

■ 画像一：职场晋升型

基本属性：
 - 25-35岁，中层管理者
 - 本科以上，一线城市
 - 月收入2-3万

行为特征：
 - 购买频次高，客单价中
 - 偏好职场管理类内容
 - 通勤时间听音频

目标动机：
 - 提升管理能力
 - 升职加薪
 - 系统化学习

痛点需求：
 - 时间碎片化
 - 难以坚持
 - 需要实践指导

关键词："高效学习"、"实用干货"

■ 画像二：副业变现型

基本属性：
 - 22-30岁，自由职业/斜杠
 - 大专以上，各线城市
 - 月收入波动大

行为特征：
 - 价格敏感度高
 - 偏好理财、副业类内容
 - 社交电商活跃

目标动机：
 - 增加收入来源
 - 探索第二职业
 - 财务自由

痛点需求：
 - 缺乏靠谱项目
 - 执行力不足
 - 需要社群陪伴

关键词："副业刚需"、"月入过万"
```
