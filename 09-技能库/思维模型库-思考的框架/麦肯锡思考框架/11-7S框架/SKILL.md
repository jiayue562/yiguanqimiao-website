## 一、核心定义

### 1.1 一句话定义
**7S框架**通过七大相互关联的要素（战略/结构/制度/风格/技能/人员/共同价值观），系统诊断组织能力与战略匹配度。

### 1.2 七要素结构

```
              共同价值观
                  ★
               /  |  \
        战略 /     |     \ 技能
           /      |      \
    ┌─────────────────────┐
    │      组织七要素       │
    │  ┌─┐  ┌─┐  ┌─┐     │
    │  │结│  │制│  │作│     │
    │  │构│  │度│  │风│     │
    │  └──┘  └──┘  └──┘     │
    └──────────┬──────────┘
                │
              人员
```

## 二、七要素详解

### 硬件要素（容易改变）

| 要素 | 定义 | 诊断问题 |
|------|------|---------|
| **战略** | 获取竞争优势的行动计划 | 战略是否清晰？ |
| **结构** | 权责关系与汇报线 | 结构是否适配？ |
| **制度** | 业务流程、考核机制 | 制度是否支持？ |

### 软件要素（难以改变）

| 要素 | 定义 | 诊断问题 |
|------|------|---------|
| **风格** | 领导风格与管理方式 | 风格是否匹配？ |
| **技能** | 核心能力与人才素质 | 技能是否支撑？ |
| **人员** | 员工结构、激励、发展 | 人员是否到位？ |
| **共同价值观** | 核心理念与文化 | 文化是否愿景驱动？ |

## 四、变革阻力分析

| 阻力类型 | 表现 | 应对策略 |
|---------|------|---------|
| 战略-结构脱节 | 架构不支持战略 | 调整架构 |
| 制度-人员矛盾 | 考核与激励不匹配 | 重塑制度 |
| 文化-变革冲突 | 价值观与变革相悖 | 文化引领 |

## 五、实战模板

### 战略转型组织诊断

```
现状：产品销售 → 解决方案提供商

战略：匹配度★★★★☆
结构：按产品线→按行业（★★☆☆☆ ⚠️）
制度：产品提成→解决方案考核（★★☆☆☆ ⚠️）
风格：老板驱动→赋能型（★★★☆☆）
技能：单一产品→综合方案（★★☆☆☆ ⚠️）
人员：产品型人才→复合型人才（★★☆☆☆ ⚠️）
价值观：销售导向→客户成功（★★★☆☆）

核心发现：人员能力与文化转型滞后

变革路径：
第一阶段（0-6月）：结构调整 + 制度重塑
第二阶段（6-12月）：技能培养 + 人才引进
第三阶段（12-24月）：文化深化 + 价值观整合
```
