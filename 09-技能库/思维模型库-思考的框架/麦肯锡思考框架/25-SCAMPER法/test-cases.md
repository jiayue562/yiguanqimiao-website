# SCAMPER法 · 测试用例

## 测试场景一：咖啡店服务创新

### 输入
情境：某精品咖啡店希望创新服务体验

### 期望输出
```
【SCAMPER · 咖啡店服务创新】

S替代：
 - 咖啡师→机器人手臂（标准化）
 - 到店消费→外卖配送

C组合：
 - 咖啡+阅读空间
 - 咖啡+办公工位
 - 咖啡+音乐体验

A适应：
 - 从咖啡店→办公室茶水间
 - 从到店→快取模式

M调整：
 - 推出mini杯（降低门槛）
 - 推出大杯装（家庭装）

P其他用途：
 - 成为商务会谈地点
 - 成为社交聚会场所
 - 承接团建活动

E消除：
 - 取消最低消费
 - 取消纸质菜单（电子化）
 - 取消排队（预约制）

R逆转：
 - 从"客人来"到"我们去"（移动咖啡车）
 - 从"慢体验"到"快节奏"（窗口即取）
```

## 测试场景二：会员体系创新

### 输入
情境：某电商平台希望创新会员体系

### 期望输出
```
【SCAMPER · 会员体系创新】

S替代：
 - 年费→月费（降低门槛）
 - 单一会员→积分会员制

C组合：
 - 会员+内容付费
 - 会员+线下权益
 - 会员+金融产品

A适应：
 - 从付费会员→免费会员+增值服务
 - 从单一平台→跨平台联盟

M调整：
 - 推出分级会员（基础/白银/黄金）
 - 调整权益组合比例

P其他用途：
 - 会员积分可兑换公益
 - 会员身份可共享（家庭卡）

E消除：
 - 消除复杂规则
 - 消除不常用权益
 - 消除会员降级机制

R逆转：
 - 从"付费享受"到"分享赚钱"（会员裂变）
 - 从"价格歧视"到"价值对等"
```
