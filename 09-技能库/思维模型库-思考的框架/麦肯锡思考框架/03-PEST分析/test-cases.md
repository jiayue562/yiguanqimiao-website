# 🧪 PEST分析 · 测试用例

## 测试场景一：科技公司战略规划

### 输入
**情境**：某AI创业公司计划未来3年拓展海外市场，需要进行宏观环境分析

### 期望输出
```
【PEST分析 · 海外市场拓展】

■ 政治维度（P）
  - 数据保护法规（GDPR等）影响产品合规
  - 政府AI产业政策支持力度
  - 国际关系对技术人才流动的影响
  影响程度：高 | 时间跨度：中期 | 判定：威胁

■ 经济维度（E）
  - 目标市场经济增长预期
  - 汇率波动对收入的影响
  - 当地融资环境与成本
  影响程度：中 | 时间跨度：短期 | 判定：机会

■ 社会维度（S）
  - AI接受度与市场需求
  - 当地工程师人才供给
  - 隐私意识与数据敏感度
  影响程度：高 | 时间跨度：长期 | 判定：机会

■ 技术维度（T）
  - 当地技术基础设施水平
  - 开源生态与技术社区活跃度
  - AI技术迭代速度
  影响程度：高 | 时间跨度：中期 | 判定：机会

【综合评估】
最大机会：AI市场需求旺盛 + 技术人才充裕
最大威胁：数据合规要求 + 国际关系不确定性
```

### 验证标准
- [ ] 四个维度完整覆盖
- [ ] 每维度识别2-3个关键趋势
- [ ] 影响程度评估有依据
- [ ] 机会/威胁判定明确

## 测试场景二：消费品企业市场评估

### 输入
**情境**：某新消费品牌评估进入东南亚市场的可行性

### 期望输出
```
【PEST分析 · 东南亚市场评估】

■ 政治维度（P）
  - 各国贸易协定与关税政策
  - 政局稳定性评估
  - 外资准入政策
  影响程度：高

■ 经济维度（E）
  - 东南亚经济增长趋势
  - 中产阶级规模与购买力
  - 电商渗透率与支付基础设施
  影响程度：高

■ 社会维度（S）
  - 年轻人口比例与消费习惯
  - 社交媒体使用偏好
  - 品牌认知与本地化需求
  影响程度：高

■ 技术维度（T）
  - 移动互联网普及率
  - 物流基础设施
  - 数字支付生态
  影响程度：中
```

### 验证标准
- [ ] 宏观趋势与行业特点结合
- [ ] 地域差异分析到位
- [ ] 战略建议具有可操作性

## 测试场景三：制造业转型评估

### 输入
**情境**：某传统制造企业评估智能制造转型机会

### 期望输出
```
【PEST分析 · 智能制造转型】

■ 政治维度
  - "中国制造2025"政策支持
  - 环保法规趋严
  - 产业升级补贴政策
  判定：机会（政策红利期）

■ 经济维度
  - 原材料价格波动
  - 人力成本上升趋势
  - 融资成本与信贷政策
  判定：威胁（成本压力）

■ 社会维度
  - 制造业人才流失
  - 工人技能结构变化
  - 年轻一代就业偏好
  判定：威胁（人才挑战）

■ 技术维度
  - 工业互联网技术成熟度
  - 机器人与自动化成本下降
  - 5G与AI技术赋能
  判定：机会（技术可行）
```

### 验证标准
- [ ] 转型相关因素识别完整
- [ ] 机会与威胁平衡分析
- [ ] 时间维度考量合理

## 测试场景四：教育培训行业分析

### 输入
**情境**：某在线教育平台进行K12业务战略规划

### 期望输出
```
【PEST分析 · K12在线教育】

■ 政治维度
  - "双减"政策影响
  - 教育监管方向
  - 民办教育法规
  影响程度：极高 | 判定：重大威胁

■ 经济维度
  - 家庭教育支出意愿
  - 居民收入增长预期
  - 教育消费升级趋势
  影响程度：高 | 判定：机会

■ 社会维度
  - 家长教育焦虑程度
  - 应试教育观念转变
  - 素质教育需求增长
  影响程度：高 | 判定：机会

■ 技术维度
  - 在线教育技术成熟度
  - AI辅导应用前景
  - 设备与网络普及率
  影响程度：中 | 判定：机会
```

### 验证标准
- [ ] 政策因素权重恰当
- [ ] 社会需求洞察深入
- [ ] 战略建议兼顾风险与机遇
