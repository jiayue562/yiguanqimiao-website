## 一、核心定义

### 1.1 一句话定义
品牌资产金字塔从低到高构建五层品牌资产：**品牌识别→品牌意义→品牌反应→品牌共鸣→品牌Bond**，层层递进形成忠诚的品牌关系。

### 1.2 金字塔结构

```
                    ┌─────────┐
```

## 二、五层详解

### 2.1 第一层：品牌识别

**定义**：消费者认出品牌的能力

| 要素 | 内容 |
|------|------|
| 品牌认知 | 知道这个品牌 |
| 品牌记忆 | 能在众多品牌中想起 |
| 品牌想起 | 购买时第一时间想起 |

### 2.2 第二层：品牌意义

**定义**：品牌在消费者心中的内涵

| 要素 | 内容 |
|------|------|
| 功能利益 | 产品/服务能做什么 |
| 情感利益 | 使用带来什么感受 |
| 自我表达 | 品牌代表什么身份 |

### 2.3 第三层：品牌反应

**定义**：消费者对品牌的评价

| 要素 | 内容 |
|------|------|
| 品质感知 | 看起来/用起来怎么样 |
| 品牌认可 | 这个品牌好不好 |
| 品牌差异化 | 和竞品有什么不同 |

### 2.4 第四层：品牌共鸣

**定义**：消费者与品牌的情感连接

| 要素 | 内容 |
|------|------|
| 品牌依恋 | 离不开这个品牌 |
| 品牌社区 | 参与品牌社群 |
| 主动推荐 | 愿意向他人推荐 |

### 2.5 第五层：品牌Bond（忠诚）

**定义**：品牌与消费者的终极关系

| 要素 | 内容 |
|------|------|
| 品牌忠诚 | 反复购买不换 |
| 品牌拥护 | 主动捍卫品牌 |
| 终身价值 | 最高客户价值 |

## 四、实战模板

### 场景：消费品牌升级

```
【品牌资产金字塔 · 消费品牌】

■ 现状诊断

品牌识别 ★★★★☆
 - 知名度高，但记忆度下降

品牌意义 ★★★☆☆
 - 功能利益清晰
 - 情感利益薄弱

品牌反应 ★★★★☆
 - 品质认可度高
 - 差异化不明显

品牌共鸣 ★★☆☆☆
 - 用户忠诚度低
 - 社群活跃度差

品牌Bond ★☆☆☆☆
 - 复购率低
 - 推荐意愿弱

■ 提升策略

第一阶段（0-6月）：
 - 强化品牌识别：社交媒体广告
 - 丰富品牌意义：讲好品牌故事

第二阶段（6-12月）：
 - 提升品牌反应：产品创新
 - 建立品牌共鸣：用户社群

第三阶段（12-24月）：
 - 构建品牌Bond：会员体系
 - 培养品牌拥护者：KOC运营
```
