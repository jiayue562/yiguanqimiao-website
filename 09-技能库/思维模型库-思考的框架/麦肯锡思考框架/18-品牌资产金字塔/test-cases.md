# 品牌资产金字塔 · 测试用例

## 测试场景一：新能源汽车品牌诊断

### 输入
情境：某新能源汽车品牌进行品牌资产诊断

### 期望输出
```
【品牌资产金字塔 · 新能源汽车】

■ 现状诊断

品牌识别 ★★★★☆
 - 新势力中知名度靠前
 - 但与特斯拉差距明显

品牌意义 ★★★☆☆
 - 功能：续航、智能
 - 情感：科技感、年轻
 - 自我表达：环保身份

品牌反应 ★★★☆☆
 - 品质感知两极分化
 - 智能化认可度高
 - 续航仍有争议

品牌共鸣 ★★☆☆☆
 - APP社区活跃度低
 - 用户活动参与少
 - 缺乏情感连接

品牌Bond ★☆☆☆☆
 - 复购率低
 - 推荐意愿弱

■ 提升路径

短期：强化识别+丰富意义
 - 代言人策略
 - 品牌故事讲述

中期：提升反应+建立共鸣
 - 产品力提升
 - 用户社群建设

长期：构建Bond
 - 会员体系
 - 品牌文化输出
```

## 测试场景二：消费品牌资产提升

### 输入
情境：某新消费品牌希望从网红品牌向经典品牌转型

### 期望输出
```
【品牌资产金字塔 · 消费品牌转型】

■ 当前状态

品牌识别 ★★★★★ - 网红知名度高
品牌意义 ★★★☆☆ - 功能为主，情感弱
品牌反应 ★★★★☆ - 口碑两极
品牌共鸣 ★★☆☆☆ - 缺乏忠诚
品牌Bond ★☆☆☆☆ - 复购低

■ 转型策略

第一阶段：夯实底层
 - 提升产品品质
 - 标准化服务体验

第二阶段：构建中层
 - 品牌故事升级
 - 情感价值塑造
 - 用户社群运营

第三阶段：打造顶层
 - 品牌文化输出
 - 会员体系构建
 - 品牌社区形成

【关键洞察】
从流量品牌到口碑品牌，需要3-5年持续建设
核心是构建情感共鸣和品牌忠诚
```
