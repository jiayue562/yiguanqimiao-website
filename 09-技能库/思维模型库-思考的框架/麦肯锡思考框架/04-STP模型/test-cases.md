# 🧪 STP模型 · 测试用例

## 测试场景一：新能源汽车品牌定位

### 输入
**情境**：某新能源车企推出新车型，需要进行STP分析确定目标市场与定位

### 期望输出
```
【STP分析 · 新能源汽车】

■ 市场细分（S）
  地理维度：一二线城市（充电设施完善）
  人口维度：30-45岁，家庭年收入50万+，本科及以上
  心理维度：环保意识强，追求科技感，愿意尝新
  行为维度：已有燃油车，考虑增购或换购

■ 目标市场选择（T）
  首选目标市场：35-42岁有孩家庭，二线省会城市
  选择理由：
    ✓ 购买力强：对价格敏感度低
    ✓ 需求匹配：空间大、安全性、智能化
    ✓ 竞争空白：高端家用市场竞品少
  目标市场特征：高学历、高收入、高环保意识的三高家庭

■ 市场定位（P）
  定位陈述：
  "专为中国精英家庭打造的安全智能新能源SUV，
   让每一次出行都成为科技与爱的体验。"
  核心差异化：
    - 安全：电池安全行业领先
    - 智能：城市NOA功能完整
    - 空间：六座设计，乘坐舒适
```

### 验证标准
- [ ] 细分维度覆盖4个方面
- [ ] 目标市场选择有理有据
- [ ] 定位清晰、可执行、可差异化

## 测试场景二：在线教育平台STP

### 输入
**情境**：某在线教育平台推出新品类"职业硬技能培训"，进行STP分析

### 期望输出
```
【STP分析 · 职业硬技能培训】

■ 市场细分（S）
  地理维度：全国覆盖（在线优势）
  人口维度：22-35岁年轻人，专科及以上
  心理维度：实用主义、焦虑驱动、终身学习意识
  行为维度：
    - 愿意为高质量内容付费
    - 偏好短时高频学习
    - 重视学习效果可量化

■ 目标市场选择（T）
  优先级A：IT/互联网从业者，技能提升需求强
  优先级B：传统行业转行者，快速入门需求
  优先级C：创业者，商业技能需求

  首选策略：选择性专业化
  理由：三类需求差异大，需差异化产品

■ 市场定位（P）
  定位陈述：
  "帮助职场人用3个月掌握硬技能，实现薪资跃迁的实战培训平台"
  
  核心价值主张：
    - 实战：真实项目驱动
    - 高效：3个月见效
    - 就业：对接企业需求
```

### 验证标准
- [ ] B端思维融入（对接就业）
- [ ] 目标市场分层清晰
- [ ] 定位体现差异化价值

## 测试场景三：智能家居产品STP

### 输入
**情境**：某科技公司推出智能音箱产品，进行STP分析

### 期望输出
```
【STP分析 · 智能音箱】

■ 市场细分（S）
  地理细分：
    - 一线城市：高接受度，早期采纳者
    - 二三线：增长期接受者，主流用户
  
  人口细分：
    - 年龄：25-40岁核心用户
    - 收入：家庭月收入2万+
  
  心理细分：
    - 科技爱好者：追求新鲜体验
    - 便捷追求者：注重实用价值
    - 品质生活者：关注生活品质提升
  
  行为细分：
    - 高频使用者：每日多次互动
    - 中频使用者：偶尔使用特定功能

■ 目标市场选择（T）
  密集型策略 → 选择一线城市25-35岁科技爱好者
  
  市场吸引力：★★★★★
  公司适配度：★★★★☆
  综合得分：20分（满分25）

■ 市场定位（P）
  定位陈述：
  "中国家庭的第一台智能音箱，让AI助手走进千家万户"
  
  差异化要素：
    - 音质：Hi-Fi级扬声器
    - 生态：米家IoT全面接入
    - 价格：性价比最高
```

### 验证标准
- [ ] 细分变量选择恰当
- [ ] 目标市场评估量化
- [ ] 定位体现产品核心优势
