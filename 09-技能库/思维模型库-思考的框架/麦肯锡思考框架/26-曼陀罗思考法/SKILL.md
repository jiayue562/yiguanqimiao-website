# 曼陀罗思考法

> 版本: v1.0 | 创建: 2026-04-15 | 维护者: 龙龟神将

---

## 一、核心定义

### 1.1 一句话定义
曼陀罗思考法是一种**视觉化的联想拓展工具**，用九宫格结构，从核心主题向四周辐射联想，打破线性思维，产生大量创意。

### 1.2 核心公式

```
曼陀罗 = 中心格(核心主题) + 8个周边格(一级联想) + 64个外延格(二级联想)
```

---

## 二，结构说明

### 2.1 九宫格结构

```
    ┌───────┬───────┬───────┐
    │  A2   │  A3   │  A4   │
    ├───────┼───────┼───────┤
    │  A1   │  核心  │  A5   │
    ├───────┼───────┼───────┤
    │  A8   │  A7   │  A6   │
    └───────┴───────┴───────┘
```

### 2.2 填写顺序

| 顺序 | 方法 | 特点 |
|------|------|------|
| 放射型 | 从中心向外扩散 | 发散思维 |
| 螺旋型 | 从外向中心聚焦 | 收敛思维 |

---

## 三，操作流程

### 3.1 放射型（发散）

```
第一步：写下核心主题
    ↓
第二步：从中心向四周发散
    ↓
第三步：每个格子继续发散
    ↓
第四步：记录所有联想
```

### 3.2 螺旋型（收敛）

```
第一步：写下核心问题
    ↓
第二步：从外向中心搜集
    ↓
第三步：层层聚焦要点
    ↓
第四步：形成核心洞察
```

---

## 四，联想路径

### 4.1 横向联想（8个方向）

| 方向 | 问题 |
|------|------|
| 上 | 更抽象/宏观 |
| 下 | 更具体/微观 |
| 左 | 时间维度/过去 |
| 右 | 时间维度/未来 |
| 斜上 | 情境/场景变化 |
| 斜下 | 细节/操作层面 |
| 其他 | 任意方向 |

### 4.2 纵向联想（多层级）

| 层级 | 深度 |
|------|------|
| 第一层 | 直接联想 |
| 第二层 | 联想的联想 |
| 第三层 | 深层次洞察 |
