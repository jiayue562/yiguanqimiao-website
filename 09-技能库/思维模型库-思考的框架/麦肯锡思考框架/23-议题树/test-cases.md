# 议题树 · 测试用例

## 测试场景一：企业增长战略选择

### 输入
情境：某企业选择增长战略，构建议题树

### 期望输出
```
【议题树 · 增长战略选择】

目标：实现30%年增长
            How？
        /    |      \
    内生增长  外延并购  生态扩张
       ↓       ↓        ↓
     How?     How?     How?

■ 内生增长
 - 如何提升现有业务？
   - 提客单价
   - 提复购率
   - 提用户数
 - 需要什么资源？

■ 外延并购
 - 并购什么标的？
   - 同业并购
   - 互补并购
 - 如何估值？
 - 如何整合？

■ 生态扩张
 - 哪些相关领域？
 - 如何选择入口？
 - 如何平衡主业与副业？
```

## 测试场景二：客户满意度问题

### 输入
情境：某SaaS产品NPS下降，分析原因

### 期望输出
```
【议题树 · 客户满意度问题】

核心议题：客户满意度为什么下降？
              Why？
          /    |      \
       产品    服务    价格
        ↓       ↓      ↓
       Why?    Why?   Why?

■ 产品问题
 - 功能缺失？
 - 性能问题？
 - 体验不佳？

■ 服务问题
 - 响应慢？
 - 支持不专业？
 - 主动关怀少？

■ 价格问题
 - 性价比下降？
 - 竞品更便宜？
 - 涨价感知？
```
