# PESTLE分析

> 版本: v1.0 | 创建: 2026-04-15 | 维护者: 龙龟神将

---

## 一、核心定义

### 1.1 一句话定义
PESTLE是PEST的扩展版，增加法律(Legal)和环境(Environmental)两个维度，提供更全面的宏观环境分析框架。

### 1.2 六维度结构

```
PESTLE = P(Political) + E(Economic) + S(Social) + T(Technological) + L(Legal) + E(Environmental)
```

---

## 二、六维度详解

### 2.1 政治维度 (P)

| 要素 | 内容 |
|------|------|
| 政策法规 | 产业政策、税收政策 |
| 政治稳定 | 政权稳定性 |
| 国际关系 | 外交关系、贸易壁垒 |

### 2.2 经济维度 (E)

| 要素 | 内容 |
|------|------|
| 经济增长 | GDP、PMI |
| 利率汇率 | 融资成本、汇率风险 |
| 通货膨胀 | 物价水平 |

### 2.3 社会维度 (S)

| 要素 | 内容 |
|------|------|
| 人口特征 | 规模、年龄结构 |
| 生活方式 | 消费习惯、价值观 |
| 环保意识 | 绿色消费 |

### 2.4 技术维度 (T)

| 要素 | 内容 |
|------|------|
| 技术创新 | 专利、研发投入 |
| 数字化 | 互联网、AI应用 |
| 自动化 | 机器人替代 |

### 2.5 法律维度 (L)

| 要素 | 内容 |
|------|------|
| 劳动法 | 员工权益保护 |
| 消费者保护 | 退换货政策 |
| 竞争法 | 反垄断合规 |

### 2.6 环境维度 (E)

| 要素 | 内容 |
|------|------|
| 气候变化 | 碳排放要求 |
| 环保法规 | 绿色生产要求 |
| 可持续发展 | ESG压力 |

## 四、实战应用

### 场景：制造业企业战略规划

```
【PESTLE分析 · 制造业战略规划】

■ 政治
产业政策：中国制造2025支持
一带一路：出口新机遇
监管趋严：合规成本上升
判定：机会（政策红利）

■ 经济
人力成本上升：自动化需求
消费升级：高端设备需求
判定：机会（结构升级）

■ 社会
技能人才短缺：培训需求
环保意识：绿色生产
判定：威胁+机会

■ 技术
工业4.0：智能制造机遇
5G应用：工业互联网
判定：机会（技术赋能）

■ 法律
环保法规：淘汰落后产能
知识产权：保护核心技术
判定：威胁（合规成本）

■ 环境
碳中和目标：能源转型
绿色金融：融资新渠道
判定：机会（绿色金融）
```
