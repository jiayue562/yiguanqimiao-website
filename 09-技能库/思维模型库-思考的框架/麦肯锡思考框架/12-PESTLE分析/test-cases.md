# PESTLE分析 · 测试用例

## 测试场景一：新能源车企海外拓展

### 输入
情境：某新能源车企进行海外市场拓展的宏观环境分析

### 期望输出
```
【PESTLE · 海外市场拓展】

■ 政治：贸易壁垒风险，判定威胁
■ 经济：消费升级，判定机会
■ 社会：接受度提升，判定机会
■ 技术：基础设施完备，判定机会
■ 法律：合规成本高，判定威胁
■ 环境：碳中和政策，判定机会

【综合】
最大机会：政策+消费+技术
最大威胁：地缘政治+合规
```

## 测试场景二：餐饮下沉市场扩张

### 输入
情境：某餐饮连锁品牌评估下沉市场扩张机会

### 期望输出
```
【PESTLE · 下沉市场扩张】

■ 政治：地方招商政策，机会+威胁
■ 经济：消费增长，判定机会
■ 社会：品牌认知低，判定威胁
■ 技术：外卖渗透率高，判定机会
■ 法律：食品许可合规，判定威胁
■ 环境：物流配送半径，判定机会

【结论】
下沉市场有机遇，但需差异化策略应对品牌认知和合规挑战
```
