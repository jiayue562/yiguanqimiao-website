# 🧠 思维模型库总调度 · v2.0

## 一、核心定位

**思维模型库**是龙脑OS知识库的核心组成部分，包含**59个思维模型**：
- **通用思维模型**：《思考的框架》系列31个跨学科模型
- **麦肯锡思考框架**：28个商业分析专业模型

**系统定位**：
- **所属系统**：龙脑OS知识库（被调用层）
- **调度中枢**：龙心OS（总智能体）
- **调用方式**：由龙心OS根据场景智能路由
- **最新升级**：2026-04-13 整合麦肯锡思考框架（28个模型）

## 三、龙心OS调用协议

### 调用方式

#### 1. 单模型调用
```
场景：需要特定思维模型分析
调用：直接调用对应模型Skill
输出：该模型的分析结果
```

#### 2. 多模型组合调用
```
场景：复杂问题需要多角度分析
调用：组合2-3个相关模型
输出：综合多模型的分析结果
```

#### 3. 场景套件调用
```
场景：特定场景的标准分析流程
调用：预设的模型组合套件
输出：完整的场景分析报告
```

## 八、中西合璧：通用模型 × 麦肯锡框架

### 组合使用场景

| 场景 | 通用模型 | 麦肯锡框架 | 整合价值 |
|------|---------|-----------|---------|
| **战略创新** | 第一性原理 + 二阶思维 | SWOT + BCG矩阵 | 本质洞察 + 商业分析 |
| **问题解决** | 逆向思维 + 概率思维 | MECE + 逻辑树 | 风险规避 + 结构化拆解 |

### 调用优先级建议

```
龙心OS路由决策：

IF 问题涉及商业分析、组织管理、战略规划：
    → 优先调用麦肯锡框架
    
IF 问题涉及认知提升、跨学科思考、个人成长：
    → 优先调用通用思维模型
    
IF 复杂综合问题：
    → 组合调用（通用模型 + 麦肯锡框架）
    → 象思维（0→1）→ 麦肯锡（1→N）→ 五色光（验证）
```
