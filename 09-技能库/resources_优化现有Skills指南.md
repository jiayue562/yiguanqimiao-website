# 优化现有Skills指南

## 🎯 优化目标

将CLI、Skills、MCP、Framework四大技术栈融入现有Skills，让所有Skills都支持：
- CLI指令前缀参数解析
- MCP协议通信
- 个性化参数传递
- Framework调度

---

## 📋 优化清单

### 1. 知识学习Skill优化

#### 需要增加的功能
- **CLI指令解析集成**：支持"按[思维模型]学习[文章]"指令
- **MCP协议集成**：Skills之间数据传递使用MCP格式
- **个性化参数**：学习结果携带个人思维模型、价值观参数

#### 具体优化点
1. 在SKILL.md中增加"CLI指令触发"章节
2. 输出格式中增加"个性化参数"字段
3. 与其他Skills的数据传递使用MCP协议格式

## 📖 优化模板

所有Skills的优化都遵循以下模板：

### CLI指令触发章节
```markdown
## 🎯 CLI指令触发

### 直接触发词
- "按[思维模型]学习[文章]"
- "用[人格特质]风格[动作]"

### CLI指令解析
- 从CLI指令中提取思维模型、价值观、人格特质、信仰文化参数
- 将参数传递给Skill的执行模块
- 确保执行结果贴合个人特质
```

### MCP协议集成章节
```markdown
## 🌐 MCP协议集成

### 数据输入
- 使用MCP协议格式接收输入数据
- 验证数据完整性和一致性
- 携带个性化参数传递

### 数据输出
- 使用MCP协议格式输出数据
- 包含个性化参数字段
- 支持Skills之间数据透明传递
```

### 个性化参数字段
```json
{
  "personal_params": {
    "思维模型": "...",
    "价值观": "...",
    "人格特质": "...",
    "信仰文化": "..."
  }
}
```
