# Task Router Skill - 智能任务路由

## 技能概述
智能任务路由技能，分析用户请求并生成确定性调度计划，将任务分发到合适的处理单元。

## 核心功能

### 1. 意图识别
- **用户意图分析**：理解用户真正想要什么
- **任务类型判断**：判断任务属于哪一类
- **复杂度评估**：评估任务的复杂程度

### 2. 路由决策
- **引擎选择**：选择最合适的处理引擎
- **技能匹配**：匹配合适的专业技能
- **优先级分配**：分配任务优先级

### 3. 调度生成
- **执行计划**：生成确定性调度计划
- **资源分配**：分配计算资源和时间
- **依赖管理**：处理任务间的依赖关系

## 触发关键词
- 路由、调度、分发
- 任务分配、执行计划
- 意图识别、任务分类

## 任务分类矩阵

| 任务类型 | 路由目标 | 复杂度 |
|---------|---------|--------|
| 信息查询 | 知识学习引擎 | 低 |
| 深度理解 | 知识学习+象思维 | 中 |
| 创新突破 | 象思维+五色光 | 高 |
| 分析决策 | 五色光思维 | 中 |
| 重大决策 | 全系统 | 高 |
| 任务执行 | 人机协同 | 低 |

## 路由决策树

```
用户输入
    ↓
意图识别
    ↓
├─ 简单问答 → 直接回答
├─ 信息查询 → 知识学习
├─ 深度理解 → 知识学习+象思维
├─ 创新突破 → 象思维+五色光
├─ 分析决策 → 五色光思维
├─ 重大决策 → 全系统
└─ 任务执行 → 人机协同
```

## 输出格式

```yaml
路由决策:
  意图: 用户意图
  类型: 任务类型
  复杂度: 低/中/高
  引擎组合: [引擎列表]
  执行计划: 
    - 步骤1
    - 步骤2
    - 步骤3
```
