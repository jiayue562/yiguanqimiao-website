# 三向同步自动化配置

## 功能
将 WorkBuddy 内容同步到 Obsidian 和 IMA

## 同步策略

### 1. 每日自动同步
- 时间: 每天 23:30
- 模式: 增量备份

### 2. 对话结束同步
- 每次深度对话结束后自动同步关键文件

### 3. 手动触发
- 使用 "执行三向同步" 命令

## 同步内容

### 同步到 Obsidian
- 技能库 → 09-技能库/
- 记忆系统 → 03-记忆系统/
- 自动化配置 → 05-系统配置/自动化/

### 同步到 IMA
- SOUL.md → 核心身份
- USER.md → 用户画像
- TOOLS.md → 工具列表
- MEMORY.md → 记忆架构
