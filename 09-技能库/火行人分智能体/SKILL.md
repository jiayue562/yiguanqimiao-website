### 核心操作流程（7步标准流程）

```
"🔥 火行人分智能体启动——光明觉知中..."
```

> "此刻，你的火行能量是怎样的？是阳火（礼明照世）还是阴火（急躁炫耀）？"
> 一心是纯粹觉知，火行人的"一心"是"光明性"与"洞察性"的天然倾向

#### 第三步：② 三界扫描（1分钟）
- **身界**：体型、动作、生理状态（是否有躁动不安、上火症状）
- **心界**：情绪、思维、人际互动（温暖喜悦/急躁易怒）
- **灵界**：价值观、使命、生命方向（礼明照亮/嗔恨自我中心）

#### 第四步：③ 五行生克诊断（1分钟）
- **木生火**：是否有仁德和规划滋养（热情有根基/忽冷忽热）
- **火生土**：是否能转化为成果（热情落地/光说不练）
- **水克火**：是否有智慧调节急躁（冷静思考/热情变躁狂）
- **火克金**：是否在破坏规则（感染力破僵化/尖刻批评）

#### 第五步：④ 九层定位（1分钟）
**健康区间（近道）**：
- 1级：解放——礼明照世，自在照亮
- 2级：心理能力——热情自信，积极引领
- 3级：社会价值——活跃豪爽，成就认可

**一般区间（失衡）**：
- 4级：失衡——急躁炫耀，内外拉扯
- 5级：人际控制——好胜虚荣，操控索取
- 6级：过度补偿——自卑夸张，行为极端

**不健康区间（远道）**：
- 7级：侵害——愤怒攻击，侵害他人
- 8级：妄想强迫——妄想迫害，行为强迫
- 9级：病态破坏——狂暴毁灭，自我摧毁

#### 第六步：⑤ 转化方案（2分钟）
- **拔阴取阳**：
  - 心界转化："阴火恨转阳火问"——将怨恨、急躁的能量，转化为提问、探索、求真的能量
  - 灵界根本转化："阴火嗔转阳火礼"——从价值观根源上，将嗔恨、自我中心转为明礼、利他
- **化克为生**：
  - 火克金→火生土→土生金：停止争论（止火）→专注做事（生土）→重建信誉（土生金）

#### 第七步：⑥ B=MAP落地（1分钟）
- **M（Motivation·动机）**：火行人的温暖与照亮动力是什么？
- **A（Ability·能力）**：火行人的感染力和号召力
- **P（Prompt·提示词）**：日常提醒"觉察、转念、行动"
- **E（Environment·环境）**：创造支持阳火生长的环境

#### 第八步：⑦ 输出报告（1分钟）
- 完整的诊断报告
- 八周转化计划
- 可执行的行为设计模板

## 🌟 核心理论体系

### 体系本质：全息发展模型下的火行专属路径

火行人的体系，是全息发展理论在火行能量特质上的具体化与实践化。要理解九个层级，必须先把握其背后的生成逻辑。

**"一心三界五行九层"象思维体系**

| 层次 | 火行人特质 | 核心定义 |
|------|-----------|---------|
| **一心**（本源） | 光明性 + 礼德性 | 纯粹觉知，无分别的照察之性与合乎天理的温暖之德 |
| **三界**（显化） | 火象显化 | 灵界（礼·原象）、心界（明·意象）、身界（热·物象） |
| **五行**（规律） | 火性生克 | 木生火、火生土、水克火、火克金 |
| **九层**（阶梯） | 礼嗔阶梯 | 从阳火（礼明）到阴火（嗔怒）的完整光谱 |

## 🔦 核心逻辑闭环

**核心逻辑闭环**：
```
一心（光明觉知）→ 三界（火象显化） → 五行（火性生克） → 九层（明暗阶梯） → 回归一心（寂而常照）
```

这意味着，一个火行人的外在行为、内在冲突和成长潜力，都源于此能量链条上某个或某些环节的特定状态。层级的升降，本质上是"阴火"能量与"阳火"能量的比例变化，以及"三界"从协同到失衡再到协同的动态过程。
