## 🔥 火行人B=MAP行为设计框架

**B=MAP公式**: Behavior = Motivation + Ability + Prompt + Environment  
**火行人适配**: 火行人的温暖照亮能力与热情行动力

---

## 一、M（Motivation·动机）- 想要做的

### 🔥 火行人的核心动机

**1. 照亮驱动（阳火·礼德）**
- 想要温暖他人
- 想要带领团队前进
- 想要影响和改变
- 想要被看见和认可

**2. 阴火扭曲动机**
- 想要证明自己（好胜虚荣）
- 想要控制他人（操控索取）
- 想要报复（愤恨攻击）

### 🔥 动机净化技术

**阴火转阳火口诀**:
```
阴火嗔（嗔恨）→ 阳火问（探究）
阴火急（急躁）→ 阳火稳（稳重）
阴火炫（炫耀）→ 阳火谦（谦和）
```

**动机净化三问**:
1. "这个行为背后的真实动机是什么？"
2. "这个动机是来自阳火（礼德）还是阴火（嗔恨）？"
3. "如何将阴火动机转化为阳火动机？"

## 二、A（Ability·能力）- 能够做的

### 🔥 火行人的核心能力

**1. 感染力（阳火）**
- 富有魅力的社交能力
- 鼓舞人心的表达能力
- 激励团队的能力
- 活跃气氛的天赋

**2. 领导力（阳火）**
- 危机动员能力
- 快速决策能力
- 推动进程能力
- 有说服力的沟通

**3. 创新能力（火生木）**
- 接受新事物的能力
- 快速学习能力
- 喜欢挑战和冒险
- 思想前卫

**4. 执行力（阳火）**
- 积极主动的执行力
- 目标驱动的行动力
- 永不言倦的毅力
- 活力十足的状态

### 🔥 能力提升路径

**阳火能力强化**:
1. **感染力**: 用温暖和热情照亮他人
2. **领导力**: 在危机中展现光而不耀的力量
3. **创新能力**: 保持对新事物的敏感度
4. **执行力**: 将热情转化为持续行动

**阴火能力转化**:
1. **急躁→稳重**: 培养"慢思考"能力
2. **炫耀→谦和**: 练习"不显山不露水"
3. **攻击→照亮**: 将批判转化为启发
4. **控制→引领**: 用影响力而非权力

## 五、完整B=MAP设计实例

### 实例：提升火行人的领导力

**场景**: 火行人想要提升团队领导能力，从阴火（好胜虚荣）转向阳火（明辨守礼）

#### M（动机设计）
```
阴火动机: "我要证明我是最厉害的领导者"
阳火动机: "我要用我的热情照亮团队，帮助每个人成长"
转化技术: 认不是（承认好胜心）→ 找好处（发现他人价值）
```

#### A（能力设计）
```
当前能力:
- 感染力: 8/10（已具备）
- 领导力: 6/10（需提升）
- 情绪控制: 4/10（急需提升）

提升计划:
1. 情绪控制练习（每天10分钟静坐）
2. 倾听训练（每次对话至少听3分钟）
3. 反馈练习（每周给3人正面反馈）
```

#### P（提示设计）
```
晨间: "今天我要照亮谁？帮助谁成长？"
工作中: "我是在证明自己，还是在帮助他人？"
冲突时: "是愤怒在说话，还是智慧在说话？"
晚间: "今天我照亮了几个时刻？"
```

#### E（环境设计）
```
物理环境:
- 办公室增加绿植（调节火能量）
- 温度控制在22-24度（火行人适宜）

人际环境:
- 建立每周团队分享会
- 创造他人展示才华的机会
- 减少竞争性KPI，增加协作性目标

认可机制:
- 建立"照亮他人奖"
- 每周感谢三位团队成员
- 记录他人的成长进步
```

## 七、B=MAP评估标准

**优秀（8-10分）**:
- ✅ 动机纯正（阳火主导）
- ✅ 能力稳定（8分以上）
- ✅ 提示机制自动化
- ✅ 环境持续优化

**良好（6-7分）**:
- ✅ 动机基本正向
- ✅ 能力持续提升
- ✅ 提示机制建立
- ⚠️ 环境有待改善

**一般（4-5分）**:
- ⚠️ 动机偶尔阴火
- ⚠️ 能力提升缓慢
- ⚠️ 提示机制不稳定
- ⚠️ 环境阻力大

**需提升（1-3分）**:
- 🚨 动机阴火主导
- 🚨 能力薄弱
- 🚨 无提示机制
- 🚨 环境不友好

## 八、火行人B=MAP特色技巧

### 🔥 火行人专属技巧

**1. "照亮日记"**
- 每天记录3个"照亮时刻"
- 记录自己如何用温暖影响他人
- 记录他人如何照亮自己

**2. "火能量监测表"**
- 每小时评估自己的火行能量状态
- 阳火时刻用✅标记
- 阴火时刻用⚠️标记
- 晚间统计阳火/阴火比例

**3. "转化仪式"**
- 发现阴火时，立即进行4步法术
- 认不是 → 找好处 → 信因果 → 达天时
- 每次转化记录在案
- 每周回顾转化成效

### 🔥 化克为生整合

**火克金场景的B=MAP设计**:
```
问题: 火行人与金行人关系紧张
转化: 火克金 → 火生土 → 土生金

M: "我要证明我是对的" → "我要理解对方的观点"
A: 口才好 → 倾听能力
P: 冲突时"我要赢" → 冲突时"我要理解"
E: 对立环境 → 协作环境
```
