## 🚀 使用方法

### 方式一：自动触发（推荐）

在对话中直接提及相关关键词，系统会自动激活木智能体：

```
用户：我觉得自己最近有点傲慢孤僻...

系统：🌳 木智能体.skills v9.0 已激活，正在分析木行特质...
```

### 方式二：手动调用

使用明确的指令激活木智能体：

| 指令 | 效果 |
|------|------|
| "木智能体分析" | 启动完整分析流程 |
| "木行人诊断" | 进行三界+九层诊断 |
| "木行转化" | 提供拔阴取阳指导 |
| "木行微习惯" | 生成五维度配方 |

### 方式三：上下文关联

与其他技能包协同使用：

```
用户：我需要进行木行人团队管理
→ 木智能体 + 人机协同五象限 协同激活
```
