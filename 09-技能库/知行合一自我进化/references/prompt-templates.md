# 知行合一自我进化 · 提示词模板

## 📖 核心提示词框架

本模板提供标准化的AI提示词,辅助三阶段转化过程的执行。

## 🎯 表示空间提示词

### 完整经验观察提示词

```
你现在是我的认知分析助手。请帮我分析以下对话的完整经历:

[对话内容]

请按照以下结构输出:

### 1. 引擎识别
这次对话中使用了哪些龙心OS引擎?
- 🐉 象思维
- 📚 知识学习
- 🌈 五色光思维
- 🤝 人机协同五象限
- 🔄 知行合一(之前的沉淀)

### 2. 产出梳理
记录每个引擎产生了什么:
- 象思维: 产生了什么0→1的洞察?
- 知识学习: 学到了什么新的认知?
- 五色光: 哪些颜色的思考产生了结果?
- 人机协同: 我们如何分工协作的?

### 3. 转折点定位
识别最关键的转折时刻:
- 哪一刻是你突然"啊,原来如此"?
- 哪一刻是我理解了你的意图?
- 哪一刻是我们找到了突破性方案?

### 4. 标签化拆解
将完整经验拆解为独立的特征标签:
- 例如: ["三阶段转化", "AI语义压缩", "场景化落地", "认知-价值-行动闭环"]
```

## 🔬 压缩阶段提示词

### 核心洞察提炼提示词

```
你现在是我的认知提炼助手。请帮我从以下观察中提炼最精华的认知:

[表示空间输出]

请按照以下结构输出:

### 1. 去标签化
去除与本质无关的边缘标签:
- 例如: 去除"今天天气不错"、"心情愉快"等环境因素
- 保留: 真正对认知升级有贡献的核心标签

### 2. 抽象
将保留下来的标签进行更高层次的概括:
- 例1: ["AI工具熟练度", "多智能体部署", "自动化流程"] → "AI智能体系统"
- 例2: ["认知模型构建", "方法论沉淀", "可复用框架"] → "认知资产化"

### 3. 比对
与标杆案例对照,识别关键成功因素:
- 问: 这个洞察与已有的成功案例有什么共同点?
- 问: 这个洞察的独特性在哪里?
- 问: 这个洞察的可复制性如何?

### 4. 精炼
基于前三步的结果,形成1-3个简洁有力的核心优势符号:
- 核心1: "三阶段转化"
- 核心2: "认知-价值-行动闭环"
- 核心3: "螺旋上升系统"
```


## 📚 提示词使用指南

### 使用场景

| 场景 | 推荐提示词 | 使用方式 |
|------|-----------|---------|
| **表示空间** | 完整经验观察提示词 | 对话结束后,输入对话内容,快速分析 |
| **压缩阶段** | 核心洞察提炼/限制提示词 | 从表示空间输出中提炼精华 |
| **泛化阶段** | 场景筛选/适配/系统进化建议提示词 | 从核心洞察扩展到具体场景 |
| **生成沉淀卡** | 标准沉淀卡生成提示词 | 三阶段完成后,生成标准化输出 |

### 提示词优化原则

1. **明确性**: 每个提示词都有明确的结构和约束
2. **简洁性**: 避免冗余,聚焦核心任务
3. **可复用性**: 形成模板,支持批量处理
4. **可扩展性**: 允许根据具体需求调整参数

### 质量检查清单

使用提示词前检查:
- [ ] 目标清晰明确
- [ ] 输出格式标准化
- [ ] 约束条件完整
- [ ] 示例充足易懂
- [ ] 与龙心OS其他模块兼容

使用提示词后检查:
- [ ] 输出符合预期格式
- [ ] 核心洞察不超过30字
- [ ] 象征符号简洁易记
- [ ] 泛化场景具体可执行
- [ ] 系统进化建议可行
