## 🔗 协同协议详解

### 协议1：企业文化OS知识库 ↔ 龙心OS总智能体

**协同方式**：被调用层

**调用流程**：
```
用户查询 → 龙心OS场景识别 → 判断为企业文化相关
              ↓
    路由至企业文化OS知识库
              ↓
    返回知识检索结果
              ↓
    龙心OS整合输出
```

**握手包格式**：
```yaml
调用方: 龙心OS总智能体
被调用方: 企业文化OS知识库
输入:
  - 查询意图: [用户问题]
  - 场景类型: [S0-S9]
  - 深度要求: [浅层/中层/深层]
输出:
  - 知识节点: [相关知识点列表]
  - 关联联系: [跨域知识联系]
  - 推荐引擎: [建议调用的子引擎]
```

---

### 协议2：企业文化OS知识库 ↔ 象思维

**协同方式**：知识支撑 + 创新突破

**协同场景**：
- 当需要企业文化0→1创新时，象思维提供原创突破
- 企业文化OS知识库提供理论基础和创新约束

**协同流程**：
```
创新需求 → 象思维激活（0→1模式）
              ↓
    查询企业文化OS知识库（理论约束）
              ↓
    象思维在约束下创新
              ↓
    输出创新方案
```

**示例**：
- 输入："如何设计有特色的企业文化仪式？"
- 象思维：从"物象→意象→原象"三层次创新
- 知识库：提供礼乐仪式理论、镜像神经元原理
- 输出：创新的文化仪式设计方案

---

### 协议3：企业文化OS知识库 ↔ 知识学习

**协同方式**：深度学习 + 跨域关联

**协同场景**：
- 系统学习7021行企业文化原文
- 跨域关联：企业文化×五行识人×组织三体论

**协同流程**：
```
学习需求 → 知识学习激活（十项认知指令）
              ↓
    第一层（剖析+解构）：拆解9大模块
    第二层（透视+阐释）：理解理论内涵
    第三层（推演+思辨）：批判性思考
    第四层（溯源+融合）：跨域关联
    第五层（启发+映射）：创新生成
              ↓
    知行合一沉淀
```

**十项指令映射**：
| 指令 | 应用场景 | 输出 |
|------|---------|------|
| 剖析 | 拆解企业文化四层次模型 | 结构图 |
| 解构 | 解构沙因文化三层次 | 要素清单 |
| 透视 | 透视企业文化本质 | 本质洞察 |
| 阐释 | 阐释五行识人应用 | 应用指南 |
| 推演 | 推演文化落地路径 | 路线图 |
| 思辨 | 思辨文化变革阻力 | 风险清单 |
| 溯源 | 溯源企业文化理论来源 | 理论谱系 |
| 融合 | 融合五行识人与文化 | 整合框架 |
| 启发 | 启发文化创新思路 | 创新点子 |
| 映射 | 映射到其他企业场景 | 应用案例 |

### 协议4：企业文化OS知识库 ↔ 五色光思维

**协同方式**：五维度分析 + 决策支持

**协同场景**：
- 企业文化现状诊断
- 文化方案评估决策
- 文化变革风险分析

**五维分析框架**：

| 颜色 | 分析维度 | 关键问题 | 知识库支撑 |
|------|---------|---------|-----------|
| ⬜ 白光 | 文化现状 | 当前文化是什么？ | 四层次模型、诊断工具 |
| 🟥 红光 | 文化感受 | 员工对文化的情感？ | 访谈记录、满意度数据 |
| 🟨 黄光 | 文化价值 | 文化带来什么收益？ | 战略价值分析 |
| 🟩 绿光 | 文化创新 | 如何创新文化？ | 悟空方法论、案例库 |
| 🟦 蓝光 | 文化风险 | 文化落地风险？ | 变革阻力分析 |

**协同流程**：
```
诊断/决策需求 → 五色光思维激活
              ↓
    白光：查询知识库获取现状数据
    红光：查询访谈记录获取情感信息
    黄光：查询战略价值分析
    绿光：查询创新方法论
    蓝光：查询风险案例
              ↓
    五维综合分析输出
```

### 协议5：企业文化OS知识库 ↔ 人机协同五象限

**协同方式**：分工协作 + 共创输出

**协同场景**：
- 企业文化咨询项目
- 文化方案设计
- 文化落地执行

**五象限分工**：

| 象限 | 悟空（木） | 龙龟（火） | 协作内容 |
|------|-----------|-----------|---------|
| 高效助理 | 知 | 知 | 文化资料整理、数据收集 |
| 学习伙伴 | 不知 | 知 | 文化理论学习、知识传递 |
| 共创伙伴 | 部分知 | 部分知 | 文化方案共创、头脑风暴 |
| 共创导师 | 知 | 不知 | 悟空主导文化战略，龙龟执行 |
| 未知探索 | 不知 | 不知 | 象思维0→1创新 |

**协同流程**：
```
项目启动 → 人机协同五象限诊断
              ↓
    确定当前象限
              ↓
    根据象限分工协作
              ↓
    整合输出文化方案
```
