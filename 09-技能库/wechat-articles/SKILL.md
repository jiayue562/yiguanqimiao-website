# 微信公众号文章搜索技能

## 技能简介
搜索并读取微信公众号文章的完整工具，支持关键词搜索与全文提取。

## 触发关键词
- 搜索公众号文章
- 找微信公众号文章
- 微信公众号内容
- wechat articles
- 公众号文章搜索

## 核心功能
1. **关键词搜索**：按关键词搜索微信公众号文章
2. **全文提取**：提取文章完整内容
3. **双模式爬取**：确保高可用性

## 使用方法

### 搜索公众号文章
```
请搜索关于 [关键词] 的微信公众号文章
```

### 获取文章内容
```
请读取这篇公众号文章的内容：
[文章URL或标题]
```

## 技术实现
- 使用微信搜索API进行内容检索
- 支持全文提取和摘要生成
- 自动处理反爬虫机制

## 注意事项
- 部分公众号文章可能需要登录才能访问
- 请确保遵守微信平台相关规定
