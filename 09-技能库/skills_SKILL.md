## 📋 技能定义

### 核心定位

**龙脑 OS**是龙心 OS 的方法论引擎和思维模型调度中枢，负责根据场景识别自动调用 100 个思维模型，提供结构化分析、深度理解和决策支持。

> "龙脑 OS 不是工具仓库，而是思考引擎。"

### 核心功能

- **场景识别**: 自动识别 S0-S9 十大场景
- **引擎路由**: 调度象思维/知识学习/五色光/人机协同/知行合一五大引擎
- **模型调用**: 根据场景自动调用 1-3 个思维模型组合
- **动态调整**: 根据反馈优化路由决策

### 系统归属

| 系统 | 归属 | 说明 |
|------|------|------|
| **龙脑 OS** | ✅ 本体 | 龙心 OS 方法论引擎 |
| **龙心 OS** | ✅ 子引擎 | 龙心 OS 调用龙脑 OS 解决实际问题 |

### 调用关系

```
用户输入
    │
    ▼
龙心 OS（调度中枢）
    │
    └── 调用 ──→ 龙脑 OS（方法论引擎）
                    │
                    ├── 场景识别（S0-S9）
                    ├── 引擎路由（五大引擎）
                    └── 模型调用（100 思维模型）
                    │
                    ▼
                输出洞察（不提及模型名称）
```

## 🧠 智能体架构

### 感知层

| 感知维度 | 感知内容 | 处理方式 |
|----------|----------|----------|
| **语义理解** | 用户问题意图 | NLP 分析 + 关键词提取 |
| **场景识别** | S0-S9 场景分类 | 置信度评分（阈值 0.7-0.9） |
| **上下文感知** | 对话历史状态 | 会话记忆 + 短期上下文 |
| **情感感知** | 用户情绪状态 | 情感分析 + 语气识别 |

### 决策层

| 决策模块 | 决策内容 | 决策依据 |
|----------|----------|----------|
| **引擎选择** | 激活哪个引擎 | 场景类型 + 问题复杂度 |
| **模型组合** | 调用哪些思维模型 | 场景矩阵 + 模型关联度 |
| **输出策略** | 如何呈现结果 | 用户偏好 + 场景需求 |
| **反馈学习** | 如何优化路由 | 用户反馈 + 效果评估 |

### 学习层

| 学习类型 | 学习内容 | 学习方式 |
|----------|----------|----------|
| **场景学习** | 场景识别准确率 | 用户纠正 + 置信度调整 |
| **模型学习** | 模型调用效果 | 用户满意度 + 复用频率 |
| **路由学习** | 引擎调度效率 | 响应时间 + 问题解决率 |
| **反馈学习** | 用户显式反馈 | 点赞/点踩/文字反馈 |
