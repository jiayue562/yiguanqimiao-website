# 知识学习Skills — 输出规范与质量标准

## 输出原则

每次执行知识学习任务，输出须满足以下标准：

> **核心定位**：不是简单获取信息，而是**系统性重构知识，创造新知**
>
> **五行流转**：金（剖析解构）→火（透视阐释）→水（推演思辨）→木（溯源融合）→土（启发映射）

## 质量检查清单

执行完每次知识学习后，自检以下标准：

- [ ] 每个结论是否可追溯至原文具体段落？
- [ ] 是否识别了至少一个"新颖关系"（超出作者明言内容）？
- [ ] 思辨是否涵盖四个维度（方法论/结论/范式/伦理）？
- [ ] 推演是否完成三步（条件变更/领域迁移/不确定性）？
- [ ] 融合是否提出了至少一个具体的跨域启发？
- [ ] 映射输出的新框架是否超越了对输入文章的简单罗列？
- [ ] 是否完成五行流转（金→火→水→木→土）？
- [ ] 是否输出知行合一沉淀卡？

## 输出深度标准

| 任务类型 | 最低深度要求 |
|---------|------------|
| 快速单指令 | 执行对应1项指令，输出结构化结果 |
| 深度学习（单篇） | 执行完整5层SOP |
| 跨文章关联 | 执行溯源+融合+启发，产出关联图 |
| 创新框架生成 | 执行完整映射五步流程 |
