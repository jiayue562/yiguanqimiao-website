## 第一章：一心——光明本源

### 1.1 核心定义

**"一心"是本体系的起点与归宿，指纯粹的、无分别的觉知（亦称觉性、道）。**

对于火行人而言，其"一心"觉知的天然倾向独具特色，即：

| 特质 | 核心诠释 | 阳面（健康） | 阴面（失衡） |
|:------|---------|------------|------------|
| **光明性** | 如太阳之光，普照万物而无分别，温暖而不灼人 | 照察万物，无分别心 | 虚荣的炫耀，追求表面光鲜 |
| **洞察性** | 敏锐穿透，瞬间照亮事物关联与本质 | 真知灼见，明辨是非 | 愤世嫉俗的评判，看人不顺 |

## 第六章：火行人五维度福格行为设计（v1.7核心）

### 6.1 福格行为模型核心原理

**公式**：B = M × A × P（行为 = 动机 × 能力 × 提示）

| 要素 | 定义 | 火行人特点 |
|:-----|:-----|:---------|
| **M动机** | 执行某个行为的欲望强度 | 波动大，追求即时反馈 |
| **A能力** | 执行该行为的容易程度 | 启动强，持续弱 |
| **P提示** | 触发行为的信号 | 易被情绪触发，缺稳定锚点 |

### 6.2 五维度行为诊断与设计方案

#### 【维度一：炎上】推动提升 ⇄ 急躁争斗

| 要素 | 失衡诊断 | 设计方案 |
|:-----|:--------|:---------|
| **M动机** | 扭曲为"必须快速取胜以证明自己" | 建立"耐心积分"系统，延迟反应积1分 |
| **A能力** | "耐心倾听"能力不足 | 情绪温度计（1-10分）实时自评 |
| **P提示** | 愤怒感、他人反对触发争斗 | 止怒石（触觉转换）；每小时节奏检查提醒 |
| **微习惯** | **"2分钟暂停法"**——决策前强制暂停120秒 | |

#### 【维度二：明亮】明辨守礼 ⇄ 炫耀虚荣

| 要素 | 失衡诊断 | 设计方案 |
|:-----|:--------|:---------|
| **M动机** | 扭曲为"必须通过外在表现获取赞美" | 转向"建立真诚连接"，社交后反思理解深度 |
| **A能力** | "倾听""共情"能力不足 | 角色互换练习，提升共情"脑力" |
| **P提示** | 社交场合、即将展示自我的时刻触发"表演模式" | 手机提醒："倾听时间 ≥ 发言时间" |
| **微习惯** | **"真实价值记录"**——每日睡前写3个不依赖外界认可的内在成就 | |

#### 【维度三：炽烈】热情自信 ⇄ 自卑夸张

| 要素 | 失衡诊断 | 设计方案 |
|:-----|:--------|:---------|
| **M动机** | 自我价值感完全绑定于外界反馈 | 建立"内在价值账户"，每天存入不依赖评价的自我肯定 |
| **A能力** | 情绪调节能力在剧烈波动时失效 | 90秒呼吸法；将"失败"重新定义为"反馈数据" |
| **P提示** | 成功/失败事件触发极端化 | 生理状态作为提示，前置性干预自我照顾 |
| **微习惯** | **"情绪波动预警"**——超7分时强制启动冷静程序 | |

#### 【维度四：发散】主动影响 ⇄ 情绪多变

| 要素 | 失衡诊断 | 设计方案 |
|:-----|:--------|:---------|
| **M动机** | 扭曲为"需要他人服从来确认影响力" | 重新定义成功影响：从"服从"到"激发潜能" |
| **A能力** | "管理期待""处理拒绝"能力弱 | 扩充情绪词汇库；渐进式暴露训练 |
| **P提示** | 他人不符合预期的反应触发情绪化回应 | 静心角；关系提示（约定非暴力提醒手势） |
| **微习惯** | **"影响意图检查表"**——行动前自查：我目标？双赢？尊重对方？ | |

#### 【维度五：迅疾】气势夺人 ⇄ 无序冲动

| 要素 | 失衡诊断 | 设计方案 |
|:-----|:--------|:---------|
| **M动机** | 沉迷于"启动的兴奋"，对枯燥执行缺乏动机 | 设置"可持续成就奖励"；连接使命感知 |
| **A能力** | "目标分解""持续专注"能力严重不足 | 番茄工作法（25+5）；冲动控制正念冥想 |
| **P提示** | 新奇灵感触发立即行动的冲动 | 可视化倒计时器；行为链提示（咖啡后→5分钟计划） |
| **微习惯** | **"冲动转化"**——灵感记入孵化清单，48小时后再评估 | |
