# 思维模型库.skills v1.0

## 核心定位

**一句话定义**：悟空原创的「思维模式-方法论-解决问题」模型体系，龙心OS的核心子智能体之一。

## 核心公式

```
问题 → 框架思维 → 方法论（实践+迭代） → 解决问题 + 思维进化
```

## 五大模块

### 【模块1】思维模式
- 内核：结构化思维
- 外层：创造性思维（突破）+ 批判性思维（验证）

### 【模块2】方法论
- 上部：方法论迭代升级
- 下部：方法论实践

### 【模块3】语言即结构
- 核心命题："语言即结构"
- 延伸："结构的界限，就是语言能抵达的深度"

### 【模块4】结构线哲学三论
- 存在论 → 结构
- 认识论 → 结构化
- 实践论 → 结构化分析

### 【模块5】螺旋闭环
- 六环节自我增强闭环
- 从"自在"到"自为"到"自在自为"

## 龙心OS调用

| 组合 | 场景 |
|------|------|
| 单独调用 | 结构化表达、复杂问题分析 |
| +象思维 | 原创突破 |
