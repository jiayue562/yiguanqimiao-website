## 🎯 实践指南

### 触发条件（自动识别）
- 提到"伴侣"、"夫妻"、"恋爱"、"关系"等关键词
- 描述关系冲突、冷战、沟通障碍
- 询问如何改善关系、如何与伴侣相处
- 提到伴侣的特质、性格、行为模式

### 分析流程（7步法）
1. **五行识人**：识别双方五行类型（木/火/土/金/水）
2. **生克分析**：分析五行相生相克关系
3. **冲突诊断**：识别当前关系中的核心冲突点
4. **转化方案**：设计拔阴取阳/化克为生方案
5. **B=MAP设计**：设计具体行为改变计划
6. **实践指导**：提供日常实践方法
7. **长期规划**：制定共同成长路线图

### 核心技术应用

#### 拔阴取阳四步法
1. **认不是**：承认自己的问题与不足
2. **找好处**：发现并感激伴侣的优点
3. **信因果**：理解关系中的因果逻辑
4. **达天时**：把握改善关系的最佳时机

#### 化克为生完整路径
- **木克土 → 木生火 → 火生土**：用木的创新能力点燃火，用火的光明滋养土
- **火克金 → 火生土 → 土生金**：用火的热情滋养土，用土的稳定强化金
- **土克水 → 土生金 → 金生水**：用土的稳定淬炼金，用金的智慧涵养水
- **金克木 → 金生水 → 水生木**：用金的智慧滋养水，用水的灵活滋养木
- **水克火 → 水生木 → 木生火**：用水的灵活滋养木，用木的生长点燃火
