## 触发决策

```yaml
触发算法：
  总分 = P0权重 + P1权重 + P2权重

  if 任何一个P0关键词:
      → 5分，立即触发 ✓
  elif P1场景触发词≥1个:
      → 4分 + 其他信号 → 累积≥3分则触发 ✓
  elif P2信号触发:
      → 3分，触发 ✓
  else:
      → 不触发，继续对话
```

## 激活后的行为

### 标准流程

1. **声明进入教员方法论模式**：
   > "📋 教员方法论引擎启动——找（成熟模型）→ 装（改装框架）→ 填（填入现实）"

2. **识别分析阶段**：
   - 矛盾定位阶段（调用矛盾论）
   - 结构分解阶段（调用金字塔）
   - 逻辑验证阶段（调用金线）
   - 闭环迭代阶段（调用PDCA）

3. **引导三步实操**：
   - 第一步：找——找到成熟的思维模型
   - 第二步：装——将模型改装成适配框架
   - 第三步：填——填入具体现实内容

4. **输出标准分析报告**
