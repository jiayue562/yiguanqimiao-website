## 🎯 核心概念速查

### 教员方法论四步流程
```
┌─────────────────────────────────────────────────────────┐
│  第一步：矛盾定位（矛盾论）                              │
│  ├── 根本矛盾识别（系统层面结构性失衡）                  │
│  ├── 六维矛盾分析（个人→部门→店面→公司→行业→时代）      │
│  └── 三维动力机制（物质体×能量体×信息体）                │
├─────────────────────────────────────────────────────────┤
│  第二步：结构分层（金字塔原理）                          │
│  ├── MECE原则（相互独立、完全穷尽）                      │
│  ├── 金字塔结构（结论→分论点→论据）                      │
│  └── SCQA模型（情境→冲突→问题→答案）                    │
├─────────────────────────────────────────────────────────┤
│  第三步：逻辑验证（金线原理）                            │
│  ├── 假设驱动（提出→验证→修正）                          │
│  ├── 数据支撑（相关系数>0.7）                            │
│  └── 逻辑自洽（纵向支撑、横向MECE）                      │
├─────────────────────────────────────────────────────────┤
│  第四步：实践闭环（实践论/PDCA）                         │
│  ├── Plan（计划）                                        │
│  ├── Do（执行）                                          │
│  ├── Check（检查）                                       │
│  └── Act（调整）                                         │
└─────────────────────────────────────────────────────────┘
```

### 核心公式
```
教员方法论 = 矛盾论（定位本质）
           × 金字塔原理（结构分层）
           × 金线原理（逻辑验证）
           × 实践论（闭环迭代）
```

### 关键阈值
- **矛盾运动阈值**: 任一维度变动 ±15%
- **数据验证阈值**: 相关系数 > 0.7
- **执行偏差阈值**: 偏差 > 15% 触发调整

## 🚀 使用流程

### 场景一：企业问题诊断
1. **读取模板**: `templates/problem-analysis-template.md`
2. **执行四步**:
   - 矛盾定位：识别根本矛盾（三维结构）
   - 结构分层：搭建金字塔（MECE）
   - 逻辑验证：假设驱动+数据支撑
   - 实践闭环：PDCA执行方案
3. **输出报告**: 完整问题诊断报告

### 场景二：培训课程设计
1. **读取模板**: `templates/teaching-design-template.md`
2. **执行四步**:
   - 矛盾定位：学员现状与期望差距
   - 结构分层：课程模块设计（MECE）
   - 逻辑验证：教学方法与效果验证
   - 实践闭环：课前→课中→课后PDCA
3. **输出方案**: 完整课程设计方案

### 场景三：战略规划
1. **读取模板**: `templates/strategic-planning-template.md`
2. **执行四步**:
   - 矛盾定位：战略环境六维分析
   - 结构分层：战略目标金字塔
   - 逻辑验证：假设与风险验证
   - 实践闭环：战略执行与监控
3. **输出规划**: 3-5年战略规划
