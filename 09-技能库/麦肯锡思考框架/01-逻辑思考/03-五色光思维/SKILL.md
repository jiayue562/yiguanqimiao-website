# 五色光思维 · 麦肯锡思考工具

> **框架编号**：03  
> **所属领域**：逻辑思考  
> **核心定位**：系统化视角拓展的平行思维工具

---

## 核心定义

五色光思维是由爱德华·德·波诺提出的平行思维工具，通过代表六种不同思考视角的"色光子"，帮助个人或团队系统性地拓展思维边界，避免陷入片面或混乱的讨论。

**核心理念**：同一时间，所有人都从同一角度看问题

---

## 六色视角详解

| 颜色 | 视角 | 核心关注 | 典型提问 |
|------|------|---------|---------|
| ⬜ **白光** | 客观 | 事实、数据、信息 | "我们知道什么？信息是否完整？" |
| 🟥 **红光** | 主观 | 情感、直觉、预感 | "我对此感觉如何？" |
| 🟨 **黄光** | 积极 | 优点、价值、利益 | "这个想法有什么好处？" |
| 🟦 **蓝光** | 谨慎 | 风险、困难、问题 | "这可能出错吗？问题在哪里？" |
| 🟩 **绿光** | 创新 | 新想法、替代方案 | "我们还有其他办法吗？" |
| 🎯 **主持人** | 控制 | 过程组织、总结 | "我们的目标是什么？下一步该思考什么？" |

---

## 操作步骤

```
Step 1: 设定议题
    ↓ 明确需要讨论或决策的问题
Step 2: 选择思维顺序
    ↓ 主持人（定调）→白光（摆事实）→黄光（看优势）
      →蓝光（找风险）→绿光（想创新）→红光（做决断）→主持人（总结）
Step 3: 角色扮演与讨论
    ↓ 所有参与者同时只戴一顶色光子，从该视角思考
Step 4: 主持人总结
    ↓ 形成综合结论或行动计划
```

---

## 经典序列参考

### 问题分析序列
**白 → 蓝 → 绿 → 黄**
- 先现状 → 再风险 → 再创新 → 最后价值

### 决策制定序列
**白 → 绿 → 黄 → 蓝 → 红**
- 事实 → 选项 → 收益 → 风险 → 直觉

### 创意激发序列
**红 → 白 → 绿 → 蓝 → 黄**
- 情绪 → 背景 → 发散 → 筛选 → 可行

## 实战案例

### 案例：讨论是否采用新技术方案

**白光阶段**：
- 技术参数：响应速度提升40%
- 成本：实施成本50万，年维护10万
- 行业采用率：头部企业60%已采用

**黄光阶段**：
- 效率提升带来的人力成本节约
- 可能形成的竞争优势
- 技术领先的品牌形象

**蓝光阶段**：
- 技术不成熟的风险
- 转换成本和学习成本
- 对现有系统的兼容性

**绿光阶段**：
- 分阶段实施降低风险
- 与技术供应商建立战略合作
- 内部培养技术专家团队

**红光阶段**：
- 团队成员凭直觉投票
- 考虑团队对新技术的接受度

**主持人总结**：
基于事实、优势、风险和创新想法，做出分阶段实施的决策

## 关键原则

| ✅ 正确做法 | ❌ 错误做法 |
|------------|------------|
| 所有人同时从同一角度看问题 | 各说各话，陷入对抗 |
| 严格遵守"严禁批判"原则 | 在绿光阶段就批评想法 |
| 主持人确保讨论有序 | 没有引导，讨论混乱 |

## 与五色光思维（悟空版）的整合

**注意**：本框架的"五色光"与悟空独创的"五色光思维"名称相同但体系不同。

| 德·波诺五色光 | 悟空五色光 | 对应关系 |
|--------------|-----------|---------|
| 白光（客观） | 白光（事实） | ✅ 一致 |
| 红光（情感） | 红光（直觉） | ✅ 一致 |
| 黄光（积极） | 黄光（价值） | ✅ 一致 |
| 绿光（创新） | 绿光（创新） | ✅ 一致 |
| 蓝光（谨慎） | 蓝光（风险） | ✅ 一致 |

**整合使用**：两个体系可以完美融合，本框架提供具体操作方法，悟空五色光提供更深层的哲学基础。
