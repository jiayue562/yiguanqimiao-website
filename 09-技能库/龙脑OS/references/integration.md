# 龙脑OS · 龙心OS调用协议

> **协议版本**：v1.0  
> **创建日期**：2026-04-09  
> **维护者**：龙龟神将  
> **核心关系**：龙心OS（调度中枢）→ 龙脑OS（知识库智能体）

---

## 📡 调用协议

### 请求格式

```yaml
调用请求:
  from: "龙心OS"
  to: "龙脑OS"
  
  必需参数:
    scene_type: "场景类型（S0-S9）"
    user_need: "用户需求描述"
    
  可选参数:
    target_knowledge_base: "目标知识库（思维模型库/五行人格/麦肯锡框架）"
    specific_model: "具体模型（如SWOT、波特五力等）"
    combination_needed: "是否需要组合（布尔值）"
    depth_level: "分析深度（快速/标准/深度）"
    
  上下文信息:
    conversation_history: "对话历史"
    user_preferences: "用户偏好"
    previous_analysis: "之前的分析结果"
```

### 响应格式

```yaml
调用响应:
  from: "龙脑OS"
  to: "龙心OS"
  
  核心结果:
    knowledge_base: "调用的知识库名称"
    model_used: "使用的具体模型"
    analysis_result: "分析结果（结构化数据）"
    
  建议输出:
    recommendations: "行动建议列表"
    next_steps: "后续步骤建议"
    related_models: "相关模型推荐"
    
  元数据:
    execution_time: "执行时间"
    confidence_score: "置信度评分"
    feedback_request: "是否需要反馈"
```

## 🔄 调用流程

### 标准调用流程

```
┌─────────────────────────────────────────────────────────────┐
│ 1. 龙心OS场景识别                                            │
│    - 识别S0-S9场景类型                                       │
│    - 判断是否需要知识库                                      │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│ 2. 构建调用请求                                              │
│    - 封装场景信息                                            │
│    - 传递用户需求                                            │
│    - 添加上下文                                              │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│ 3. 龙脑OS智能路由                                            │
│    - 解析请求                                                │
│    - 选择知识库                                              │
│    - 匹配模型                                                │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│ 4. 知识库执行分析                                            │
│    - 调用具体模型                                            │
│    - 执行分析流程                                            │
│    - 生成结果                                                │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│ 5. 结果返回与整合                                            │
│    - 格式化结果                                              │
│    - 返回龙心OS                                              │
│    - 整合输出                                                │
└─────────────────────────────────────────────────────────────┘
```

| 场景类型 | 场景特征 | 推荐知识库 | 推荐模型 |
|---------|---------|-----------|---------|

| **S5 重大决策** | 复杂决策 | 多知识库组合 | SWOT + 五行 + 五色光 |
| **S6 任务执行** | 执行落地 | 思维模型库 | IPO模型、SOP |
| **S7 系统规划** | 系统级规划 | 麦肯锡框架 + 思维模型库 | 金字塔 + 系统化 |

## 🎯 场景路由示例

### 示例1：商业分析场景

```yaml
场景:
  user_input: "用SWOT分析一下我的副业项目"
  scene_type: "S4分析决策"

龙心OS决策:
  need_knowledge_base: true
  target: "龙脑OS"

龙脑OS路由:
  knowledge_base: "思维模型库"
  model: "SWOT分析"
  
执行结果:
  analysis_result: "SWOT四象限分析"
  recommendations: "基于分析的行动建议"
```


## 🔧 错误处理

### 错误类型与处理

| 错误类型 | 错误代码 | 处理方式 | 返回信息 |
|---------|---------|---------|---------|
| 知识库未找到 | KB404 | 返回可用知识库列表 | "未找到指定知识库，可用知识库：..." |
| 模型未找到 | MD404 | 返回知识库内可用模型 | "未找到指定模型，可用模型：..." |
| 场景不匹配 | SC400 | 返回推荐场景 | "当前场景不匹配，推荐场景：..." |
| 参数缺失 | PM400 | 返回必需参数列表 | "缺少必需参数：..." |
| 执行超时 | EX504 | 返回部分结果 | "分析超时，返回部分结果：..." |
