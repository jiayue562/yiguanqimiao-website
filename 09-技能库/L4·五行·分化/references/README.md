### 核心调度算法

1. **感知引擎**：提取关键词、推断五行、分析情感、识别场景
2. **识别引擎**：识别任务类型、选择智能体、设计路由策略
3. **路由引擎**：设计最优调度路径（单/双/全系统）
4. **调度引擎**：并行调用智能体、激活五行通关点、实时监控
5. **学习引擎**：分析执行结果、收集用户反馈、优化算法

### 五行通关点能量循环

**核心公式**：
```
木生火（忍辱）→ 火生土（不怨）→ 土生金（感恩）→ 金生水（找好处）→ 水生木（认不是）
```

**通关点含义**：
- **木生火（忍辱）**：木行人通过忍辱，滋养火行人的热情
- **火生土（不怨）**：火行人不怨怼，滋养土行人的稳定
- **土生金（感恩）**：土行人感恩，孕育金行人的决断
- **金生水（找好处）**：金行人找好处，引导水行人的智慧
- **水生木（认不是）**：水行人认不是，滋养木行人的仁德
