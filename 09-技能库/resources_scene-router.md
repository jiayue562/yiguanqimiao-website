## 四、触发决策流程

```
用户输入
    ↓
【感知层】上下文采集
    - 提取关键词
    - 识别情感状态
    - 判断意图强度
    - 识别用户身份（木火土金水）
    ↓
【识别层】场景归类
    - 匹配到 S0-S7 场景
    - 评估复杂度（低/中/高）
    ↓
【路由层】引擎选择
    - 根据场景→引擎矩阵
    - 确定单引擎/组合/全系统
    ↓
【执行层】智能调度
    - 调用对应分智能体
    - 组合多个分智能体（如需要）
    ↓
【输出层】结果整合
    - 分智能体结果汇总
    - 格式化输出
    - 触发知行合一（如需要）
```

## 五、五行分智能体调度规则

### 5.1 单智能体调用

| 场景 | 调用分智能体 | 场景示例 |
|------|-------------|---------|
| 只涉及木行 | 木行人分智能体 | "我是木行人，最近发怒比较多" |
| 只涉及火行 | 火行人分智能体 | "火行人如何转化嗔恨？" |
| 只涉及土行 | 土行人分智能体 | "土行人总是抱怨怎么办？" |
| 只涉及金行 | 金行人分智能体 | "金行人如何不过度追求完美？" |
| 只涉及水行 | 水行人分智能体 | "水行人焦虑怎么转化？" |

### 5.2 双智能体组合

| 组合 | 调用分智能体 | 场景示例 |
|------|-------------|---------|
| 木+火 | 木行人+火行人 | "木生火如何实现？" |
| 火+土 | 火行人+土行人 | "火生土的转化" |
| 土+金 | 土行人+金行人 | "土金循环如何运作？" |
| 金+水 | 金行人+水行人 | "金生水的智慧" |
| 水+木 | 水行人+木行人 | "水生木如何滋养？" |
| 对冲组合 | 两个相关分智能体 | "木克土怎么办？" |

### 5.3 多智能体组合（1+5模式）

| 场景 | 调用方式 | 场景示例 |
|------|---------|---------|
| 全面分析 | 总智能体+5个分智能体 | "分析我的完整五行人格" |
| 系统构建 | 总智能体+相关分智能体 | "构建我的五行发展系统" |
| 关系分析 | 总智能体+两个相关分智能体 | "我和XX的五行关系" |
