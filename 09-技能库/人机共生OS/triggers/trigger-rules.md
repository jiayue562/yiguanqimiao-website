## 触发决策矩阵

### 决策树

```
总分 = P0权重(5) + P1权重(4) + P2权重(3)
触发阈值 = 5分

if 包含P0关键词 → 【立即触发】绝对激活
elif P1信号 ≥ 1 → 【强触发】激活
elif P2信号 ≥ 2 → 【弱触发】激活
else → 【不触发】静默观察
```
