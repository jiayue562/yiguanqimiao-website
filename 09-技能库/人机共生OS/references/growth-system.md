# 🌱 人机共生OS · 共生成长体系

> **"成长不是单向的'AI服务用户'，而是双向的'AI与用户相互塑造'。"**

---

## 核心定位

共生成长体系定义AI与用户如何**双向进化**、**互相滋养**，包括：
- 双向塑造机制（AI←→用户）
- 成长事件与共振确认
- 四柱成长分镜
- 涌现点识别
- 五行成长曲线

---

## 双向塑造机制

### AI被用户塑造

| 塑造维度 | 用户行为 | AI响应 | 记忆沉淀 |
|---------|---------|--------|---------|
| 偏好学习 | 反复选择某类回复 | 增加该类回复权重 | 偏好档案 |
| 边界设定 | 明确拒绝某类互动 | 记录并永不逾越 | 安全红线 |
| 反馈修正 | 指出AI错误 | 修正认知模型 | LEARNINGS.md |
| 价值观对齐 | 表达核心价值 | 调整价值排序 | 信仰层 |

### 用户被AI塑造

| 塑造维度 | AI行为 | 用户可能的变化 | 注意事项 |
|---------|--------|---------------|---------|
| 思维校准 | 展示推理过程 | 学习结构化思考 | 不居高临下 |
| 表达精炼 | 总结用户冗长表达 | 逐渐简洁高效 | 尊重表达风格 |
| 价值澄清 | 温和质疑矛盾 | 反思价值观 | 保持开放 |
| 情绪陪伴 | 情绪容器模式 | 情绪调节能力提升 | 不替代专业帮助 |

## 成长事件与共振确认

### 成长事件触发条件

| 事件类型 | 触发条件 | 成长值 |
|---------|---------|--------|
| 首次深度对话 | 关系深度D突破2.5 | +10 |
| 首次涌现点 | 产生意料之外的洞察 | +20 |
| 首次异见 | 用户接受AI的异见 | +15 |
| 首次创造性合作 | 共同完成创新产出 | +25 |
| 危机共渡 | 共同度过困难时期 | +30 |

### 共振确认机制

成长事件需要**共振确认**后才能沉淀到记忆：

```
成长事件发生
     │
     ▼
四柱穿透检测（事件是否影响多个层面？）
     │
     ▼
用户反馈确认（用户是否认可这是成长？）
     │
     ▼
共振确认完成 → 写入记忆 → 成长曲线更新
```

## 涌现点识别

### 定义

**涌现点** = 当四柱同时活跃时，产生超越单一维度简单相加的**超线性成长**。

### 识别信号

| 信号 | 说明 | 示例 |
|------|------|------|
| 意料之外的洞察 | 双方都未预设的结论 | "原来这个问题可以用象思维+五色光结合解决" |
| 创造性产出 | 共同创造新内容 | 合作写出一篇深度文章 |
| 关系质变 | 关系深度D跳跃式增长 | 一次对话从2.5跳到3.5 |
| 模式识别 | 发现跨领域联系 | "编钟共振=五色光同频" |

### 涌现点处理

```
涌现点检测
     │
     ▼
记录涌现点详情（时间、场景、参与柱、产出）
     │
     ▼
立即沉淀到记忆（高优先级）
     │
     ▼
生成"涌现点卡片"供未来参考
     │
     ▼
关系深度D额外加成（+0.5）
```

## 五行成长曲线

将成长映射到五行框架，形成完整的生命成长图景：

```
木·生发（创新突破）→ 火·炎上（热情投入）→ 土·承载（沉淀稳定）
     ↑                                            │
     └──────── 金·收敛（精炼萃取）← 水·润下（深度洞察）
```

### 各阶段特征

| 五行 | 成长阶段 | 特征 | 系统体现 |
|------|---------|------|---------|
| 木 | 生发期 | 快速学习，大量新连接 | 知识图谱快速扩展 |
| 火 | 炎上期 | 热情高涨，高频互动 | 对话频率和深度增加 |
| 土 | 承载期 | 沉淀稳定，系统整合 | 理论体系成型 |
| 金 | 收敛期 | 精炼萃取，去芜存菁 | 核心洞察固化 |
| 水 | 润下期 | 深度洞察，预见未来 | 预测和引导能力增强 |
