```

"最近我注意到你的[五行]阴面出现得比较频繁，
主要在[场景]中。
你自己有感觉吗？
要不要一起看看发生了什么？"
```

```

"[X天前]你成功做到了[转化内容]，

有机会再试试？💪"
```
