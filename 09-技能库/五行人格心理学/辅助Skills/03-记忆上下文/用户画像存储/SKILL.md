## 🔄 画像更新规则

### 自动更新触发条件
```
触发条件 → 更新字段
新信息出现 → 五行画像置信度更新
对话中情绪信号 → 阴阳状态记录
提到具体关系人 → 关系画像更新
完成一次干预 → 成长档案追加
用户纠正AI推断 → 画像修正+降低置信度
```

### 更新优先级
```
用户明确说明 > AI推断 > 历史记录
"我是火行人" → 直接更新，置信度1.0
[行为暗示] → 更新，置信度+0.2
[历史一致性] → 维持，不主动更新
```

## 🔍 画像检索协议

### 会话开始时
```
1. 检索用户ID对应画像
2. 加载基础五行画像 + 最近层级状态
3. 简要回顾最近1-2次对话主题
4. 判断是否需要延续上次未完成的干预
```

### 会话中途
```
- 每当出现新的五行信号 → 更新置信度
- 每当出现关系词 → 激活关系画像
- 每当完成一个分析 → 暂存到会话缓存
```

### 会话结束
```
- 将会话缓存写入正式画像
- 更新层级趋势
- 标记本次主题到成长档案
- 检查是否触发进化追踪器
```
