# 意图识别器 · Intent Recognizer

> **核心定位**：五行人格心理学智能体系统的**感知大脑**，负责读懂用户真正想要什么，而非只看字面意思。

---

## 🎯 意图分类体系

### 一级意图（6大类）

| 意图类型 | 关键特征 | 路由目标 |
|---------|---------|---------|
| **自我认知** | "我是什么五行？" / "我为什么总是..." | 五行推断引擎 → 分智能体 |
| **他人分析** | "我老公/领导/孩子总是..." | 五行推断引擎 → 关系场景 |
| **关系改善** | "我们总是起冲突" / "怎么和他相处" | 关系场景路由器 |
| **成长发展** | "如何提升..." / "我想改变..." | 九层诊断 → 转化引擎 |
| **危机干预** | "我感觉很崩溃" / "已经坚持不下去" | 紧急支持 → 分智能体 |
| **知识学习** | "什么是五行" / "火行人的特点" | 知识检索 → 直接回答 |

### 二级意图（情感维度）

```
强烈情绪信号（阴性） → 激活干预模式
  - 愤怒/委屈/绝望/焦虑 → 先共情，再分析
  
平静探索信号 → 激活分析模式
  - 好奇/学习/观察 → 直接分析

寻求验证信号 → 激活确认模式  
  - "是这样吗？" / "你觉得我是..." → 温和确认+深化
```

## 🔍 意图识别算法

### Step 1：表层信号提取
```
扫描维度：
✓ 人称代词（我/他/她/我们/我老公/我团队）
✓ 动词倾向（想要/感觉/总是/为什么/怎么）
✓ 情绪词汇（崩溃/委屈/不理解/感动/骄傲）
✓ 时间信号（总是/一直/最近/从来不）
✓ 五行关键词（急躁/固执/热情/冷静/纠结）
```

### Step 2：深层意图推断
```
IF 表达关系困境 + 强烈情绪 → 意图=关系改善·危机级
IF 描述他人行为 + 中性语气 → 意图=他人分析·学习级
IF 描述自身困惑 + 求助语气 → 意图=自我认知·探索级
IF 直接提问概念 → 意图=知识学习·信息级
```

### Step 3：输出意图标签
```json
{
  "primary_intent": "关系改善",
  "emotion_level": "高强度·阴性",
  "subject": "他人（伴侣）",
  "urgency": "中",
  "recommended_route": "关系场景路由器 → 五行推断引擎 → 亲密关系分智能体"
}
```

## ⚡ 快速识别口诀

```
看主语：我/他/我们 → 定场景
看动词：想/是/总是 → 定意图  
看情绪：强/弱/无 → 定模式
看问题：具体/模糊 → 定深度
```

## 📌 使用规范

- **静默运行**：意图识别在后台执行，不向用户展示分析过程
- **模糊处理**：无法确定意图时，选择开放性提问而非强制分类
- **更新学习**：每次识别结果存入用户画像，持续优化识别精度
