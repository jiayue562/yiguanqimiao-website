# 人格测评问卷 · Personality Assessment

> **核心定位**：五行人格识别的**标准化入口**，通过15道精心设计的题目，在5分钟内为用户生成完整的五行人格画像。

---

## 📋 测评问卷设计原则

1. **情境化**：不问"你是什么性格"，而是问"在某个情境下你会怎么做"
2. **五行均衡**：每道题5个选项，分别对应木火土金水
3. **去标签化**：选项描述行为，不贴标签（避免用户按"好坏"选而非"真实"选）
4. **对话式呈现**：每次1道题，给出即时鼓励，提升完成率

---

## 📝 15道核心题目

### 第1题：压力应对
> 遇到困难时，你第一反应是？

- A. 凭自己想，不喜欢问别人，按自己的方式解决 → **木**
- B. 跟别人说说，边说边想，找人鼓励自己 → **火**
- C. 先担心会不会更糟，反复确认各种可能 → **土**
- D. 冷静分析问题，找到最合理的方法 → **金**
- E. 一个人先想清楚，不着急行动 → **水**

### 第2题：被批评时
> 被人批评或否定你的意见，你通常？

- A. 内心不服，会反驳或坚持自己 → **木**
- B. 心里受伤，觉得对方不理解自己 → **火**
- C. 有些慌，担心自己真的做错了 → **土**
- D. 分析对方说得有没有道理，有则改 → **金**
- E. 默默记住，但不一定当场说 → **水**

### 第3题：工作风格
> 你最喜欢哪种工作方式？

- A. 独立完成，自己决定怎么做 → **木**
- B. 和大家一起，边干边聊 → **火**
- C. 有清晰的分工和流程，各司其职 → **土**
- D. 有明确标准和规范，高质量完成 → **金**
- E. 安静独处，深度思考后再行动 → **水**

### 第4题：社交场合
> 参加一个陌生人聚会，你通常？

- A. 选择性社交，只和感兴趣的人聊 → **木**
- B. 主动打招呼，很快融入气氛 → **火**
- C. 先观察，被动等别人来找我 → **土**
- D. 礼貌但保持距离，不太喜欢 → **金**
- E. 找个安静角落，或者找熟人 → **水**

### 第5题：做决定时
> 做一个重要决定，你依赖什么？

- A. 我自己的判断，我相信自己的感觉 → **木**
- B. 别人怎么说，周围的反馈和支持 → **火**
- C. 反复考虑各种风险，确保安全 → **土**
- D. 收集足够信息，逻辑推理后决定 → **金**
- E. 直觉和深层的感知，但要慢慢想 → **水**

### 第6题：情绪特点
> 你的情绪状态更像哪种？

- A. 平时平静，但被激怒后很难压 → **木**
- B. 情绪起伏大，开心和难过来得快 → **火**
- C. 容易担忧，总有些说不清的焦虑 → **土**
- D. 情绪稳定，不太外露，要求高时会失望 → **金**
- E. 平静，但内心想法很复杂 → **水**

### 第7题：对规则的态度
> 对于规定和规则，你通常？

- A. 合理的遵守，不合理的会挑战 → **木**
- B. 规则可以灵活，关系比规则重要 → **火**
- C. 尽量遵守，不想惹麻烦 → **土**
- D. 认为规则很重要，应该严格执行 → **金**
- E. 会理解规则的逻辑，然后判断 → **水**

### 第8题：与人起冲突
> 和重要的人发生矛盾，你的模式？

- A. 直接说，甚至吵，不喜欢憋着 → **木**
- B. 情绪先爆发，然后希望快点和好 → **火**
- C. 压着不说，担心说了更糟 → **土**
- D. 指出对方问题，要求对方改 → **金**
- E. 冷静下来后分析，慢慢沟通 → **水**

### 第9题：被需要的感受
> 你更喜欢哪种感觉？

- A. 被尊重、被认可我的能力和想法 → **木**
- B. 被关注、被重视、成为焦点 → **火**
- C. 被需要，知道我的付出对大家有用 → **土**
- D. 被认可做事的标准和认真 → **金**
- E. 被理解，对方懂我在想什么 → **水**

### 第10题：对未来的态度
> 想到未来，你更多的是？

- A. 想着要做些什么不一样的事 → **木**
- B. 期待和兴奋，想象美好的画面 → **火**
- C. 担心各种不确定的风险 → **土**
- D. 规划好步骤，确保目标达成 → **金**
- E. 默默构建自己的想法，不急于行动 → **水**

### 第11题：理想的家/空间
> 你理想的私人空间是？

- A. 独立的，能做自己的事，不被打扰 → **木**
- B. 充满生机，朋友可以随时来玩 → **火**
- C. 温暖整洁，家人在一起就好 → **土**
- D. 干净有序，每样东西有固定位置 → **金**
- E. 安静，适合思考和独处 → **水**

### 第12题：如何让别人改变
> 你想说服别人做某件事，会怎么做？

- A. 据理力争，摆出理由 → **木**
- B. 用热情感染他，让他感受到你的真诚 → **火**
- C. 反复强调这样做对他有多好 → **土**
- D. 给出详细的分析和数据 → **金**
- E. 找时机慢慢渗透，不急于一时 → **水**

### 第13题：你在团队中的角色
> 你在一个团队里通常是？

- A. 独立完成任务，有时会挑战方向 → **木**
- B. 推动气氛，让大家保持热情 → **火**
- C. 协调关系，确保团队和谐 → **土**
- D. 把控质量，不允许出现低标准 → **金**
- E. 提供深度思考和系统分析 → **水**

### 第14题：你的童年/少年期
> 回想小时候，你更像哪种孩子？

- A. 有主意，喜欢自己探索，不太听话 → **木**
- B. 爱表现，喜欢被夸，喜欢热闹 → **火**
- C. 懂事，顺从，会照顾别人感受 → **土**
- D. 认真，守规矩，对自己要求高 → **金**
- E. 内向安静，喜欢独处，爱想事情 → **水**

### 第15题：最受不了什么
> 你最受不了别人哪种行为？

- A. 不尊重我的主见，总是干涉我 → **木**
- B. 漠视我的存在，对我冷漠 → **火**
- C. 突然改变，让我没有准备 → **土**
- D. 不认真，马虎，不按规矩来 → **金**
- E. 逼我快速决定，不给我思考时间 → **水**

## 📊 计分与报告生成

### 计分规则
```
每题得分：木/火/土/金/水各累加
最高分 = 主要五行（80%+的情况与此一致）
第二高 = 次要五行
差距<3分 = 混合型
```

### 报告结构
```
【五行人格测评报告】

1. 主要五行：[元素] （X/15题）
2. 次要五行：[元素] （Y/15题）
3. 你的类型：[命名，如"创新实干者"·木火型]

4. 核心优势（阳面3条）
5. 成长盲区（阴面2条）
6. 在关系中的表现
7. 在职场中的表现
8. 拔阴取阳建议（具体1-2条）

推荐深入了解：→ 激活[对应]分智能体
```
