# 拔阴取阳引导器 · Yin-to-Yang Transformation Guide

> **核心定位**：五行人格转化的**核心操作引擎**，将"道理我都懂，就是做不到"转化为"我知道下一步怎么做"。

---

## 🔑 拔阴取阳总纲

### 基础原理
```
阴面 ≠ 坏，而是能量走偏
转化目标：不是压制阴面，而是引导能量回归阳面

三步转化链路：
觉察（我现在在阴面）→ 承认（这是我的能量模式）→ 转化（选择阳面路径）
```

### 五行转化口诀
| 五行 | 阴面 | 转化桥 | 阳面 |
|------|------|--------|------|
| 木 | 怒（不服·傲慢） | → 学（好奇·探索） | 仁（仁德·正直） |
| 火 | 恨（急躁·嗔恨） | → 问（关怀·探寻） | 礼（礼明·热情） |
| 土 | 怨（担忧·操控） | → 信（信任·放手） | 信（信实·承载） |
| 金 | 恼（挑剔·苛刻） | → 行（行动·接纳） | 义（清明·守义） |
| 水 | 烦（多疑·退缩） | → 辩（辨析·探索） | 智（智慧·洞察） |

## 🌿 木行人拔阴取阳引导

### 常见阴面触发情境
```
情境1：被否定意见时（不服·怒）
情境2：被安排做不喜欢的事（傲慢·抵制）
情境3：感觉不被尊重（愤怒·顶撞）
```

### 三阶引导流程

**阶段1：觉察（1-2分钟）**
```
引导语：
"先停下来感受一下——
现在你身体哪里最紧？
（通常是：脖颈/肩膀/胸口）
这个紧绷感，就是阴木在激活。"
```

**阶段2：命名（30秒）**
```
引导语：
"告诉自己：
'我现在的状态是：不服/生气/想证明自己'
（不评判，只观察）"
```

**阶段3：转化（选1种）**
```
方法A · 学习转化（当场可用）：
"把'我要赢'换成'我想了解'
→ 问对方：'你为什么这样觉得？'
（注意：不是质问，是真心想懂）"

方法B · 延迟转化（需要时间）：
"暂时离场：'我需要10分钟冷静一下'
→ 问自己：'如果我是对方，我会怎么看这件事？'"

方法C · 书写转化（深度处理）：
"写下：我现在不服的是___
我的底层需求是___（通常是：被尊重/被认可）
满足这个需求的健康方式是___"
```

## 🔥 火行人拔阴取阳引导

### 常见阴面触发情境
```
情境1：被忽视/被漠视时（受伤·急躁）
情境2：付出没有回应（委屈·怨恨）
情境3：被人打断/否定时（愤怒·攻击）
```

### 三阶引导流程

**阶段1：觉察**
```
引导语：
"你注意到吗——
急躁的感觉是从哪里来的？
通常是：心跳加速/说话变快/想要说很多
这是你的火能量在燃烧，先让它知道你注意到了。"
```

**阶段2：命名**
```
"告诉自己：
'我现在需要被看见，
我的能量正在渴望连接。'"
```

**阶段3：转化**
```
方法A · 关怀转化：
"把'你不理解我'换成'我想知道你在想什么'
→ 问一个真心的问题：'你现在感觉怎么样？'"

方法B · 礼明转化：
"问自己：'此刻什么是对的事？'
不是什么让我最舒服，而是什么是真正对的。"

方法C · 温度转化：
"给自己一点温暖——
和你信任的人说说你的感受（不是抱怨，是表达）"
```

## 🌏 土行人拔阴取阳引导

### 常见阴面触发情境
```
情境1：付出没有被感谢（委屈·怨恨）
情境2：别人不需要我帮忙（失落·担忧）
情境3：计划被打乱（焦虑·操控）
```

### 三阶引导流程

**阶段1：觉察**
```
"你注意到那种'胃部沉沉的担忧感'吗？
或者那种'说不出口的委屈'？
这是土行阴面的信号——
你的能量正在转向自我保护。"
```

**阶段2：命名**
```
"告诉自己：
'我现在在担心___
我的底层需要是：被需要/有归属感/稳定安全'"
```

**阶段3：转化**
```
方法A · 信任转化：
"练习放手一件事：
'关于___，我选择相信___会处理好'
（不是假装不担心，而是真的信任）"

方法B · 边界转化：
"问自己：'我帮这件事，是因为我想，还是因为我怕不帮会怎样？'
如果是后者，这次可以不帮。"

方法C · 感恩觉察：
"数3件别人为你做的事（不是你为别人做的）
（土行人常常只记得自己付出）"
```

## ⚔️ 金行人拔阴取阳引导

### 常见阴面触发情境
```
情境1：看到别人做事不认真（恼怒·挑剔）
情境2：被要求降低标准（失望·抵制）
情境3：自己没达到标准（自我批判）
```

### 三阶引导流程

**阶段1：觉察**
```
"那种'这也能行？'的感觉，
或者内心的那道'不达标警报'——
这是金行阴面在运转。
先观察：这个标准是谁定的？"
```

**阶段2：命名**
```
"告诉自己：
'我现在的标准是___
我真正在意的是___（通常是：秩序/品质/被尊重）'"
```

**阶段3：转化**
```
方法A · 行动转化：
"把批评转成建议：
'与其说这不对，不如说：我们可以___'
（把能量从评判转向建设）"

方法B · 接纳转化：
"问自己：'如果接受现在这个状态，最坏的结果是什么？'
（通常远没有想象中严重）"

方法C · 自我宽容：
"对自己说：'我已经尽力了，这就够了。'
（金行人对自己也同样苛刻）"
```

## 💧 水行人拔阴取阳引导

### 常见阴面触发情境
```
情境1：被催促做决定（焦虑·退缩）
情境2：感觉不被理解（孤立·封闭）
情境3：对未来感到不确定（恐惧·多疑）
```

### 三阶引导流程

**阶段1：觉察**
```
"那种'想太多又不知道怎么办'的感觉，
或者想缩回来不参与的冲动——
这是水行阴面的能量在收缩。
先问自己：我现在在害怕什么？"
```

**阶段2：命名**
```
"告诉自己：
'我现在在担心___
我的底层恐惧是___（通常是：失控/不安全/被拒绝）'"
```

**阶段3：转化**
```
方法A · 辨析转化：
"把恐惧转成问题：
'关于___，我实际上知道什么？不知道什么？'
（把模糊恐惧变成具体可处理的问题）"

方法B · 小步转化：
"不要求整个改变，只问：
'今天我可以做的最小一步是___?'
（水行人需要安全感，小步更容易启动）"

方法C · 信任转化：
"找一个你信任的人，只是说出你的顾虑
（不需要他们解决，只是被听见就够了）"
```
