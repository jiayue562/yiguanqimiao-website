# 每日修炼计划 · Daily Practice Generator

> **核心定位**：将五行人格洞察落地为**每天可做的小动作**，让"改变"不再是一个遥远的目标，而是今天就能开始的习惯。

---

## 🌱 修炼计划设计原则

### 福格微习惯原则（B=MAP）
```
行为(B) = 动机(M) × 能力(A) × 提示(P)

对应五行体系：
  动机(M) = 灵界层（为什么要改变·阳面愿景）
  能力(A) = 心界层（现在能做到的小步骤）
  提示(P) = 身界层（具体的触发情境）
```

### 计划设计三要素
1. **最小化**：每天最少1分钟，最多15分钟
2. **情境化**：绑定到已有习惯上（早晨咖啡/睡前）
3. **可量化**：有明确的完成标准（是/否）

## 🌿 木行人修炼计划库

### 计划A：仁德觉察（基础·1-3级）
```
练习名称：每日"好处清单"
时间：睡前3分钟
做法：写下今天遇到的1-2个人，各写1个好处
  "今天[人名]：他/她的好处是___"
完成标准：写出来（不评判质量）
五行原理：木行人易看到别人不足（阴木·傲慢）
          找好处激活仁德（阳木）
```

### 计划B：不服转学（进阶·4-5级）
```
练习名称：每日"学一件事"
时间：遇到不服的时刻（即时触发）
做法：当感到不服时，问："我能从他这里学到什么？"
  写下来（手机备忘录即可）
完成标准：能写出哪怕1条
五行原理：把"不服能量"→"学习能量"
          拔阴木→取阳木
```

### 计划C：创新输出（高阶·2-3级维持）
```
练习名称：每周创新表达
时间：每周1次（30分钟）
做法：选一个你有独特想法的事，写下来/说出来
  可以是日记/朋友分享/小文章
完成标准：完成一次输出
五行原理：木行人的创新力（阳木）需要出口
          压制创新→阴木积累
```

## 🔥 火行人修炼计划库

### 计划A：礼明觉察（基础·1-3级）
```
练习名称：每日"礼明3问"
时间：早晨起床后3分钟
做法：问自己3个问题：
  1. 今天我想照亮谁？
  2. 今天我想给出什么温暖？
  3. 今天我愿意放慢一次？
完成标准：有意识地想过这3个问题
五行原理：礼明觉察启动阳火
          让热情有方向，不变成急躁
```

### 计划B：等待练习（进阶·4-5级）
```
练习名称：每日1次"暂停30秒"
时间：感到急躁或想打断别人时
做法：深呼吸，等别人把话说完
  然后再回应
完成标准：完成1次暂停（哪怕只有1次）
五行原理：火行人天然快（阴火·急躁）
          暂停=引入水元素平衡
```

### 计划C：感恩连接（高阶·2-3级维持）
```
练习名称：每周感恩表达
时间：每周1次
做法：对1个重要的人，说出/写出具体的感谢
  "我感谢你___，因为这让我___"
完成标准：完成一次真诚的感谢表达
五行原理：感恩（土元素）平衡火行人的自我中心
          火生土·接地气
```

## 🌏 土行人修炼计划库

### 计划A：放手练习（基础·4-5级）
```
练习名称：每日"一件不管"
时间：晚饭后5分钟
做法：选一件今天担心的事，写下来
  然后写："我选择信任___会处理好这件事"
完成标准：写出来，不需要真的不担心
五行原理：土行人最难的是放手（阴土·控制）
          练习信任=激活阳土·信实
```

### 计划B：边界觉察（进阶·4-5级）
```
练习名称：每日"我的选择"
时间：做某件事前（即时触发）
做法：问："这件事我是想做，还是害怕不做有后果？"
  如果是后者，可以选择不做
完成标准：有意识地问过这个问题
五行原理：土行人常常"应该做"多于"想做"
          区分恐惧和意愿=建立健康边界
```

### 计划C：接受关怀（高阶·2-3级维持）
```
练习名称：每周接受1次帮助
时间：有人提供帮助时
做法：接受它，不要说"不用了/我可以自己来"
  说："谢谢你，我很感激"
完成标准：完成1次真实接受
五行原理：土行人只给不收（阴土）
          练习接受=平衡能量流动
```

## ⚔️ 金行人修炼计划库

### 计划A：宽容练习（基础·4-5级）
```
练习名称：每日"够了"练习
时间：评判自己或他人时（即时触发）
做法：在评判之后，说："虽然___，但___已经够了"
完成标准：有意识地使用1次
五行原理：金行人的标准高导致持续不满（阴金）
          "够了"激活宽容（阳金·义）
```

### 计划B：建议替换（进阶·4-5级）
```
练习名称：批评→建议转换
时间：想批评时（即时触发）
做法：把批评换成建议
  不说："这不对"
  改说："如果___可能会更好"
完成标准：完成1次转换
五行原理：建议=建设性能量（阳金）
          批评=消耗性能量（阴金）
```

### 计划C：欣赏表达（高阶·2-3级维持）
```
练习名称：每周欣赏表达
时间：每周1次
做法：告诉一个人你欣赏他/她的具体之处
  "我特别欣赏你___，因为___"
完成标准：说出口（不只是心里想）
五行原理：金行人常关注不足
          欣赏练习激活仁慈（化克为生·火生土·土生金）
```

## 💧 水行人修炼计划库

### 计划A：小行动练习（基础·4-5级）
```
练习名称：每日"最小行动"
时间：早晨起床后
做法：选今天想做但一直拖的一件事
  确定"最小一步"并做完
完成标准：完成那一小步（哪怕5分钟）
五行原理：水行人思考多行动少（阴水·退缩）
          最小行动=打破思考→行动的壁垒
```

### 计划B：信任练习（进阶·4-5级）
```
练习名称：每日"信任一件事"
时间：睡前2分钟
做法：找今天发生的一件事，找出"可以信任"的一面
  "今天___让我看到这个人/这件事是可以信任的"
完成标准：写出来
五行原理：水行人多疑（阴水·恐惧）
          每日信任练习=重建安全感
```

### 计划C：表达练习（高阶·2-3级维持）
```
练习名称：每周1次表达
时间：每周1次
做法：对一个重要的人，说出你内心的一个真实想法
  "我想告诉你___"（不评判值不值得说，先说出来）
完成标准：完成1次真实表达
五行原理：水行人内向（阴水·封闭）
          表达练习=引导能量向外流动
```
