# 五行人格心理学提示词模板

> 本文档提供五行人格心理学分析的标准提示词模板

---

## 一、基础诊断类

### 1.1 五行属性判定提示词

```
请根据以下观察信息，分析对方的五行属性：

【观察信息】
- 外貌特征（身形、面相、面色）：
- 动作节奏（快/慢/适中）：
- 语言模式（简洁/啰嗦/生动）：
- 情绪表现（稳定/波动）：
- 行为特点：

【分析要求】
1. 判断主导五行（木/火/土/金/水）
2. 判断次要五行
3. 说明判断依据
4. 给出置信度评分（1-10）
```

### 1.2 阴阳面辨识提示词

```
请分析以下信息，判断当前五行能量的阴阳属性：

【信息】
- 五行类型：[木/火/土/金/水]
- 行为表现：
- 情绪反应：
- 思维模式：

【分析要求】
1. 识别当前表现的阴阳属性
2. 列举具体的阳面特质表现
3. 列举具体的阴面特质表现
4. 评估阴阳平衡程度
```

### 1.3 九层定位提示词

```
请根据以下信息，评估对方的发展层级：

【信息】
- 五行类型：[木/火/土/金/水]
- 阴阳表现：
- 人际关系状况：
- 自我认知水平：
- 情绪管理能力：

【分析要求】
1. 给出1-9层的层级判断
2. 说明判断依据
3. 识别当前层级的核心特征
4. 指出向上一层发展的方向
```

## 二、深度分析类

### 2.1 三界失衡分析提示词

```
请对以下个案进行三界失衡分析：

【个案信息】
- 主诉问题：
- 背景情况：
- 行为表现：
- 情绪状态：
- 价值观倾向：

【分析要求】
1. 分析信息体（灵界）状态
2. 分析能量体（心界）状态
3. 分析物质体（身界）状态
4. 识别三界之间的相互影响
5. 找出主要失衡点
6. 提出协同调整建议
```

### 2.2 五行生克分析提示词

```
请分析以下人际互动的五行动力学：

【互动信息】
- 人物A（五行属性）：
- 人物B（五行属性）：
- 互动场景：
- 冲突表现：

【分析要求】
1. 分析两人之间的生克关系
2. 判断当前是相生还是相克状态
3. 分析相克/相生的具体表现
4. 提出"化克为生"的策略
```

## 三、干预方案类

### 3.1 拔阴取阳方案提示词

```
请为以下个案制定"拔阴取阳"干预方案：

【个案信息】
- 五行类型：[木/火/土/金/水]
- 阴面特质表现：
- 触发情境：
- 已有资源：

【方案要求】
1. 识别需要转化的阴面特质
2. 制定短期转化策略（7天）
3. 制定中期转化策略（30天）
4. 制定长期转化策略（90天）
5. 建议具体的实践方法
```

### 3.2 个人成长规划提示词

```
请基于五行人格理论，为以下个案制定个人成长规划：

【个案信息】
- 五行属性：
- 当前层级：
- 成长目标：
- 可用资源：

【规划要求】
1. 确定成长方向（提升哪个阳面特质）
2. 设计阶段性目标（1-3-6-12个月）
3. 建议具体的修炼方法
4. 提供自我评估标准
5. 制定应对挑战的策略
```

## 四、团队应用类

### 4.1 团队五行配伍提示词

```
请分析以下团队的五行配伍情况：

【团队信息】
- 团队成员五行属性：
- 团队目标：
- 当前协作模式：
- 存在的问题：

【分析要求】
1. 评估团队五行分布是否均衡
2. 识别潜在的相生/相克关系

4. 提出优化建议
5. 设计五行互补的协作机制
```

### 4.2 领导力发展提示词

```

【领导者信息】
- 五行属性：
- 现任职位：
- 管理风格：

【分析要求】
1. 分析其五行领导力优势
2. 识别其五行领导力短板

```

## 五、综合评估类

### 5.1 完整人格画像提示词

```
请对以下个案进行完整的五行人格画像：

【个案信息】
- 基本情况：
- 外貌特征：
- 行为模式：
- 情绪表现：
- 思维特点：
- 人际关系：
- 健康状况：
- 重大经历：

【画像要求】
1. 判定主导五行及次要五行
2. 分析阴阳面特质及平衡程度
3. 定位九层发展层级
4. 分析三界状态及失衡点
5. 评估五行生克关系
6. 制定综合干预方案
```

### 5.2 关系匹配分析提示词

```
请分析以下两人关系的五行动力学：

【关系信息】
- 人物A（五行/性别/角色）：
- 人物B（五行/性别/角色）：
- 关系类型（亲密/工作/其他）：
- 关系时长：
- 存在的主要问题：

【分析要求】
1. 分析双方的五行生克关系
2. 评估关系的相生/相克程度
3. 识别关系中的主要挑战
4. 提出优化关系互动的建议
5. 给出关系发展预测
```

## 使用指南

1. **选择模板**：根据分析目的选择对应的提示词模板
2. **填充信息**：将观察到的信息填入对应位置
3. **明确要求**：根据需要调整分析要求
4. **迭代深化**：初步分析后，可进一步追问细节
5. **综合判断**：多个维度综合分析，避免单一维度结论
