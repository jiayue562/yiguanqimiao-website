# 火行人分析提示词模板集

> **版本**：v2.0  
> **目标**：快速调用火行人分析能力

---

## 一、基础分析模板

### 模板1：快速诊断

```
请帮我分析[我/他/她]的火行特质：

1. 基本特征：
- 主导功能：[炎上/展现/照耀/联结]
- 能量方向：[向上/向外/扩散]

2. 阳面特质：
[描述热情、表达、照亮方面的表现]

3. 阴面特质：
[描述急躁、虚荣、情绪化方面的表现]

4. 层级定位：[猜测当前层级]

5. 发展建议：
[针对性建议]
```

### 模板2：深度诊断

```
请对[我/他/她]进行火行人深度诊断：

## 一心觉察
- 内在光源：[照亮/燃烧/混合]
- 觉知状态：[清明/浑浊/其他]

## 三界定位
### 灵界
- 价值观：[礼/嗔]
- 使命：[照亮他人/证明自己]

### 心界
- 主导情绪：[喜/恨/急躁/其他]
- 思维特征：[串联/跳跃/其他]
- 内在动机：[追求认同/控制他人/其他]

### 身界
- 体型特征：[...]
- 动作特点：[...]
- 健康状况：[...]

## 五行平衡
- 木生火（理想燃料）：[状态]
- 火生土（成果转化）：[状态]
- 水克火（平衡调节）：[状态]
- 火克金（破立关系）：[状态]

## 九层定位
- 当前层级：[X级]
- 特征描述：[...]
- 向上跃迁路径：[...]

## 发展方案
### 短期（1-4周）
[具体行动]

### 中期（1-3个月）
[具体行动]

### 长期（3-12个月）
[具体行动]
```

```
1. 关系中的火行优势：
[热情/感染力/照亮能力]

2. 关系中的火行挑战：

3. 具体场景分析：
```

```
[...]

[...]
```

### 模板5：团队分析

```
请分析团队中的火行能量：

1. 火行成员特征：
[列出团队中的火行人或火行特质明显者]
```

## 三、转化发展模板

### 模板6：层级跃迁

```
我现在处于火行人发展的[第X]级，
特征是[描述当前状态]。

请帮我：
1. 确认当前层级是否准确
2. 分析向上一级跃迁的关键点
3. 提供具体的转化练习
4. 设定3个月的跃迁目标
```

### 模板7：即时转化

```
刚才我出现了以下火行失衡表现：
[描述具体场景和情绪]

请帮我：
1. 识别这是什么层级的失衡
2. 提供即时转化口诀
3. 分析触发这种失衡的深层原因
4. 提供长期预防方案
```

### 模板8：日常修炼

```
请为火行人设计一个[一周/一个月]的修炼计划：

1. 每日功课：
[具体练习]

2. 每周主题：
[重点转化方向]

3. 每月评估：
[如何检查进展]

4. 注意事项：
[常见陷阱和应对]
```

## 六、特殊场景模板

### 模板13：情绪急救

```
我现在感到[急躁/愤怒/焦虑/其他]，
原因是[具体事件]。

请立即给我：
1. 情绪名称识别
2. 层级判断
3. 3个即时缓解步骤
4. 1句转化口诀
```

### 模板14：关系冲突

```
我在与[具体人/角色]的关系中遇到冲突：
- 我的反应：[...]
- 对方的反应：[...]
- 冲突点：[...]

请帮我：
1. 分析我的火行特质在其中的作用
2. 识别"火克金"是否过度
3. 提供"火生土→土生金"的转化方案
4. 设计修复关系的具体话术
```

### 模板15：决策参考

```
我面临[具体决策/选择]：
选项A：[...]
选项B：[...]

请从火行特质角度帮我分析：
1. 我的火行特质会如何影响这个决策？
2. 我的热情是"照亮"还是"燃烧"？
3. 是否有"虚荣/冲动"在起作用？
4. 建议的决策原则是什么？
```
