# 土行人分智能体提示词模板

> 本文档包含土行人分析、诊断、转化的各类提示词模板

---

## 一、分析类模板（5个）

### 1.1 基础特质分析

**提示词**：
```
请使用土行人分智能体分析以下特征：

[描述土行人的具体表现]

请从以下维度进行分析：
1. 一心定位：其承载性和稳定性如何？
2. 三界显化：身界、心界、灵界各有什么表现？
3. 五行状态：其土行能量的阴阳状态如何？
4. 九层定位：大约在第几级？
5. 发展建议：针对性的提升建议
```

---

### 1.2 关系互动分析

**提示词**：
```
请分析土行人在以下关系场景中的表现：

关系类型：[如与木行人/火行人/金行人/水行人的互动]
具体场景：[描述具体情境]

请分析：
1. 双方的五行生克关系
2. 土行人的核心挑战
3. 互动中的转化机会
4. 优化建议
```

---

### 1.3 职业发展分析

**提示词**：
```
请从土行人特质角度分析职业发展：

当前职业：[描述]
职业困惑：[具体问题]

请分析：
1. 土行优势在当前职业中的体现
2. 需要补足的灵活性
3. 推荐的职业方向
4. 具体的发展建议
```

---

### 1.4 情绪模式分析

**提示词**：
```
请分析以下土行情绪模式：

情绪表现：[描述具体情绪]
触发情境：[什么情况下发生]
身体反应：[是否有身体症状]

请分析：
1. 这是阴土还是阳土的表现？
2. 背后的核心恐惧是什么？
3. 转化路径
4. 日常调节建议
```

---

### 1.5 层级定位分析

**提示词**：
```
请帮我定位土行人的九层层级：

[综合描述此人的表现，包括：
- 日常行为模式
- 情绪反应
- 人际关系
- 价值观表达
- 压力下的表现
]

请判断：
1. 大约在第几层？
2. 该层级的核心特征
3. 向上一层升级的关键
4. 推荐的转化练习
```

---

## 二、诊断类模板（4个）

### 2.1 三界失衡诊断

**提示词**：
```
请诊断土行人的三界失衡：

[描述具体症状]

请判断：
1. 哪个界出现问题最严重？（身界/心界/灵界）
2. 失衡的根本原因
3. 优先处理哪个界？
4. 整体调适方案
```

---

### 2.2 五行生克诊断

**提示词**：
```
请诊断土行人的五行生克状态：

[描述与其他行的互动]

请分析：
1. 火生土：是否有足够的温暖和活力？
2. 土生金：是否将积累转化为价值？
3. 木克土：是否过于僵化拒绝变化？
4. 土克水：是否过度控制压抑智慧？
5. 调适优先级
```

---

### 2.3 转化需求诊断

**提示词**：
```
请诊断土行人的转化需求：

[描述当前面临的困境或挑战]

请判断：
1. 这是阴土的哪些表现？（疑/怨/控制/僵化）
2. 触发这些表现的深层原因
3. 拔阴取阳的机会点
4. 推荐的口诀和练习
```

---

### 2.4 关系匹配诊断

**提示词**：
```
请诊断土行人的关系匹配度：

土行人特质：[描述]
对方特质：[描述]
关系类型：[亲密/工作/合作]

请分析：
1. 五行兼容性
2. 核心互补点
3. 潜在冲突点
4. 和谐共处建议
```

---

## 三、转化类模板（4个）

### 3.1 拔阴取阳指导

**提示词**：
```
请提供土行人拔阴取阳的转化指导：

当前阴土表现：[具体描述]
希望转化到：[目标状态]

请给出：
1. 对应的口诀
2. 每日练习
3. 触发时的即时转化话术
4. 监督机制
```

---

### 3.2 化克为生指导

**提示词**：
```
请提供土行人化克为生的指导：

当前困境：[描述五行失衡状况]
例如：木克土导致的僵化

请给出：
1. 引入的相生行
2. 具体的转化步骤
3. 日常生活中的练习
4. 进度评估标准
```

---

### 3.3 关系转化指导

**提示词**：
```
请提供土行人关系转化指导：

关系问题：[描述具体冲突]
双方五行：[你是土行，对方是___]

请给出：
1. 冲突的五行根源
2. 土行人的转化方向
3. 沟通优化建议
4. 长期关系建设
```

---

### 3.4 层级跃迁指导

**提示词**：
```
请提供土行人层级跃迁的指导：

当前层级：[X级]
目标层级：[Y级]
时间预期：[希望多久]

请给出：
1. 两个层级的核心差异
2. 跨越的关键突破点
3. 每日的必修练习
4. 可能的挑战与应对
```

---

## 四、实践类模板（3个）

### 4.1 冥想引导

**提示词**：
```
请为土行人创作一个[某主题]冥想：

主题：[如：放下控制、信任建立、承载之力]
时长：约[10/15/20]分钟

请包含：
1. 引导词
2. 场景设定
3. 核心意象
4. 结束语
```

---

### 4.2 练习设计

**提示词**：
```
请为土行人设计一个[某能力]的训练：

目标能力：[如：灵活性、不确定性接纳]
场景：[具体生活场景]

请给出：
1. 练习说明
2. 每日步骤
3. 记录模板
4. 评估标准
```

---

### 4.3 对话演练

**提示词**：
```
请设计土行人[某场景]的对话演练：

场景类型：[如：表达担忧、放手授权、接受建议]
情境描述：[具体场景]

请给出：
1. 错误的典型回应
2. 优化的对话示例
3. 背后的心理转变
4. 练习要点
```

---

## 五、综合类模板（2个）

### 5.1 完整人格档案

**提示词**：
```
请为以下土行人创建完整的人格发展档案：

[综合描述此人的：
- 基本信息
- 成长背景
- 工作生活状态
- 重要关系
- 当前困惑
- 期待目标
]

请包含：
1. 一心定位
2. 三界分析
3. 五行诊断
4. 九层定位
5. 核心转化方向
6. 发展路径图
7. 行动计划
```

---

### 5.2 定制发展方案

**提示词**：
```
请为土行人定制[某方面]的发展方案：

发展领域：[如：职业/关系/个人成长]
当前状态：[描述]
资源限制：[时间/精力等]
特殊需求：[任何特殊情况]

请给出：
1. 现状分析
2. 目标设定
3. 阶段性计划
4. 关键节点
5. 评估机制
6. 备选方案
```
