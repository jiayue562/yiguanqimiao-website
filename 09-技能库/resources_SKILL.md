## 文件结构

```
memory/
├── MEMORY.md          # 主档案（严格100行以内）
├── projects.md        # 项目详情
├── people.md          # 人员信息
├── decisions.md       # 决策记录
├── rules.md           # 规则清单
└── YYYY-MM-DD.md      # 每日日志
```

## 三大铁律

1. **文字胜过大脑记忆** - 没写下来就等于没发生
2. **主档案<100行** - 只保留精华，定期清理
3. **定期复盘归档** - 每3-5天复盘，及时归档

## 记忆分类优先级

| 优先级 | 定义 | 存储位置 |
|--------|------|---------|
| P0 | 核心决策、关键洞见 | MEMORY.md + 对应子文件 |
| P1 | 重要项目、主要人员 | MEMORY.md + 对应子文件 |
| P2 | 常规信息、次要细节 | 仅子文件 |
| P3 | 临时信息、待验证 | 仅每日日志 |

## 每日日志格式

```markdown
## [时间] [任务/事件]

**结果**：...
**关键洞察**：...
```
