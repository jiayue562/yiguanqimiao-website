## 🎯 体系定位

100个实战思维模型体系覆盖从基础认知到高级战略的完整思维工具库，**核心价值：**
> 用AI杠杆撬动效率，用一人公司跑通闭环，用个人IP放大价值

---

## 📚 六大阶段体系

### 阶段一：通用底层思维（15个）
基础认知框架，适用于所有场景

**代表模型：**
- 马斯克第一性原理 → 从根本出发，打破假设
- 麦肯锡MECE原则 → 相互独立、完全穷尽
- 金字塔原理 → 结构化表达
- SCQA模型 → 情境-冲突-问题-答案

**AI结合：** 用ChatGPT做MECE分析，结构化拆解问题

---

### 阶段二：自我认知与定位
**代表模型：**
- 王阳明心学 → 知行合一，良知本体
- 五行人格心理学 → 金木水火土人格识别
- 象思维 → 整体直观，悟性洞察
- 稻盛和夫哲学 → 利他精神，经营十二条

**AI结合：** 用AI做性格分析报告，识别五行人格

---

### 阶段三：能力与工具匹配打磨
**代表模型：**
- 五色光思维 → 分色思考，同频协作
- 人机协作四象限 → 确定AI与人的边界
- 华为灰度管理 → 非黑即白的中间地带

**AI结合：** 用Midjourney做视觉表达，用ChatGPT做文案

---

### 阶段四：最小闭环验证
**代表模型：**
- 精益创业MVP → 最小可行产品验证
- 教员方法论 → 矛盾定位到闭环验证
- 知行合一三阶段 → 表示-压缩-泛化

**AI结合：** 用AI快速生成MVP，测试市场反应

---

### 阶段五：规模化复制与效率放大
**代表模型：**
- 字节跳动方法论 → 数据驱动，快速迭代
- 阿里管理方法 → 组织赋能，体系复制
- OKR目标管理 → 目标对齐，关键结果

**AI结合：** 用AI自动化重复工作，人专注高价值决策

---

### 阶段六：价值跃迁与生态构建（13个）
**代表模型：**
- 超级个体赋能公式 → AI × 一人公司 × IP
- 生态系统思维 → 共生共赢
- 观其妙书院体系 → 古今智慧融合平台

**AI结合：** 用AI管理知识生态，形成个人影响力网络

## 🔧 人机协作四象限

```
                高复杂度
                    │
    人类主导区        │    人机协作区
    （创意/判断）     │  （策略+执行）
─────────────────────┼──────────────────
    AI辅助区         │    AI主导区
    （分析/建议）     │  （重复/标准化）
                    │
                低复杂度
       AI难完成              AI易完成
```

---

## 🚀 与AI工具的结合原则

1. **确定性工作** → AI自动化（数据整理、格式转换）
2. **半确定性工作** → 人机协作（文案撰写、分析报告）
3. **创造性工作** → 人主导+AI辅助（战略决策、创新设计）
4. **关系性工作** → 人主导（情感连接、信任建立）
