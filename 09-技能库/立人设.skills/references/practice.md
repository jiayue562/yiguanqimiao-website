## 一、叙事主权分析实操

### 1.1 六问法：快速判断叙事主权归属

**使用场景**：分析任何故事、新闻、叙事文本

| 问题 | 分析要点 |
|------|---------|
| **Q1: 谁在讲述？** | 讲述者的身份、立场、利益 |
| **Q2: 谁获益？** | 叙事为谁的利益服务 |
| **Q3: 谁受损？** | 谁被污名化/边缘化 |
| **Q4: 谁定义善恶？** | 善恶标准的制定者是谁 |
| **Q5: 二元对立如何构建？** | 正面vs负面形象如何设定 |
| **Q6: 情感投射在哪里？** | 读者/观众被引导向谁投射情感 |

### 1.2 叙事主权审计表

```
□ 讲述者身份是否中立？
□ 是否存在利益偏向？
□ 负面形象是否有辩解机会？
□ 善恶标准是否单一？
□ 是否存在可替代的解读？
□ 情感导向是否被操纵？
```

## 二、情感物理学设计实操

### 2.1 五步情感转化设计

**步骤1：事件选择**
- 选择具有冲击力的核心事件
- 事件必须能触发代入感
- 避免过于复杂或具体的设定

**步骤2：恐惧植入**
- 让读者/观众感到"这可能发生在我身上"
- 通过具体细节增强真实感
- 控制恐惧的强度和频率

**步骤3：怜悯唤醒**
- 提供可共情的人物处境
- 展现人物的脆弱面
- 建立情感连接

**步骤4：净化设计**
- 提供情绪宣泄的出口
- 设计主角战胜困境的路径
- 让读者感到"情绪被释放"

**步骤5：认知更新**
- 在故事结尾提供新的认知框架
- 帮助读者重新理解相关主题
- 完成价值观的潜移默化

### 2.2 情感物理学检测表

```
□ 是否有冲击力足够的事件？
□ 恐惧植入是否自然而非刻意？
□ 怜悯是否可共情而非煽情？
□ 净化是否完成而非强制说教？
□ 认知更新是否润物无声？
```

## 三、人物塑造实操

### 3.1 矛盾光谱构建法

**工具：人物矛盾矩阵**

| 维度 | 正面特质 | 矛盾阴影 | 触发场景 |
|------|---------|---------|---------|
| 能力 | 勇敢 | 鲁莽 | 危急时刻 |
| 情感 | 深情 | 薄情 | 面对失去 |
| 性格 | 细心 | 粗心 | 紧急决策 |
| 价值观 | 正直 | 妥协 | 利益诱惑 |

### 3.2 人物弧光设计表

**基础弧光（适用于主角）**：
```
初始状态 → 压力挑战 → 内心裂缝 → 挣扎/选择 → 蜕变/堕落 → 最终状态
```

**反派弧光（适用于后妈类角色）**：
```
表面形象 → 深层动机 → 合理性构建 → 矛盾暴露 → 救赎/毁灭 → 永恒印记
```

### 3.3 后妈角色重构清单

□ 表面形象：恶毒/嫉妒/控制
□ 深层动机：对衰老的恐惧？对生存的焦虑？对男权的依赖？
□ 合理性构建：她的经历是否解释了她的行为？
□ 矛盾暴露：她的矛盾面是什么？
□ 人性光辉：在哪个瞬间她是一个真实的人？

## 四、商业叙事实操

### 4.1 短剧创作检查表

**叙事层面**：
□ 是否有强冲击力的开场事件？
□ 是否在第30秒内触发情感？
□ 是否有明确的情绪开关设计？
□ 是否有2-3个转折点？
□ 结尾是否有认知升华？

**商业层面**：
□ 目标受众是否清晰？
□ 情感共鸣点是否对准受众痛点？
□ 是否有社交传播设计？
□ 是否有系列化空间？
□ 是否符合平台调性？

### 4.2 叙事变现路径

**基础路径**：
```
优质叙事 → 情感共鸣 → 口碑传播 → 商业变现
```

**短剧特有路径**：
```
碎片化叙事 → 情绪钩子 → 付费解锁 → 持续追更 → 生态变现
```

## 五、立人设应用场景

### 5.1 个人IP叙事

**核心问题**：
- 你想传递什么价值观？
- 你的故事如何让他人产生共鸣？
- 你的"后妈"是谁（竞争对手/旧观念/自己）？

**操作要点**：
1. 明确你的叙事主权
2. 设计你的情感物理学路径
3. 构建你的矛盾光谱
4. 完成认知更新闭环

### 5.2 企业文化叙事

**核心问题**：
- 企业的故事谁在讲？
- 企业的价值观如何通过故事传递？
- 企业的"英雄"和"反派"是谁？

**操作要点**：
1. 建立企业叙事主权
2. 设计情感物理学的内部传播
3. 构建企业人物矛盾光谱
4. 完成组织文化的认知更新

### 5.3 产品叙事

**核心问题**：
- 产品解决了什么"恐惧"？
- 用户使用产品后产生什么"怜悯"共情？
- 产品如何完成用户认知更新？

**操作要点**：
1. 定位用户的情感痛点
2. 设计产品使用的情感转化
3. 构建用户的自我叙事
4. 完成产品价值的认知升级

## 六、立人设质量评估

### 6.1 自检七问

| # | 问题 | 及格线 |
|---|------|-------|
| 1 | 叙事主权是否清晰？ | 能回答"谁在讲这个故事" |
| 2 | 情感物理学是否完整？ | 有事件→恐惧→怜悯→净化→升华 |
| 3 | 人物是否有矛盾光谱？ | 至少有一个性格裂缝 |
| 4 | 善恶标准是否多维？ | 不是简单的二元对立 |
| 5 | 是否有代入感设计？ | 读者能找到"可能是我" |
| 6 | 是否有认知更新？ | 故事后读者认知有变化 |
| 7 | 是否符合伦理底线？ | 不恶意污名化他人 |

### 6.2 优化方向

**叙事主权强化**：
- 增加讲述视角的多元性
- 给"反派"更多辩解空间
- 避免单一价值观输出

**情感物理学优化**：
- 增强事件的冲击力
- 自然化恐惧和怜悯的植入
- 提升净化的深度

**人物塑造深化**：
- 挖掘更多矛盾维度
- 增强裂缝的真实感
- 让人物更"人性化"
