## 🔍 隐秘知识联系

### 联系1：五行识人 × 团队建设

```
五行识人理论 ──→ 团队五行构成分析 ──→ 相生相克优化
     │                                    ↓
     └────────→ 拔阴取阳转化 ←──────── 冲突解决
```

**核心洞察**：团队冲突往往源于五行相克，通过化克为生可实现团队和谐。

### 联系2：组织三体论 × 企业文化

```
信息体（战略/文化）
      ↓
能量体（执行/流程）──→ 文化落地效果
      ↓
物质体（资源/设备）
```

**核心洞察**：企业文化建设必须三体协同，信息体指引、能量体驱动、物质体支撑。

### 联系3：沙因模型 × 文化诊断

```
人工制品层（可见）──→ 表象观察
      ↓
价值层（可感知）──→ 深度访谈
      ↓
基本假设层（不可见）──→ 根因挖掘
```

**核心洞察**：文化诊断必须深入三层，才能找到真正的文化基因。

### 联系4：行为经济学 × 文化落地

```
损失厌恶 ──→ 福利阶梯设计 ──→ 员工保留
锚定效应 ──→ 价值观锚定 ──→ 文化认同
```

**核心洞察**：利用心理机制设计文化干预，可实现"润物细无声"的落地效果。

### 联系5：24节气 × 营销节奏

```
24节气（自然规律）──→ 营销节奏设计 ──→ 业绩波动控制
      ↓
传统文化 ──→ 品牌差异化 ──→ 竞争壁垒
```

**核心洞察**：将营销节奏与自然规律对齐，可实现"天人合一"的可持续增长。

### 联系6：复盘方法论 × 持续改进

```
复盘画布 ──→ 经验提炼 ──→ 知识沉淀
   ↓
PDCA ──→ 循环优化 ──→ 持续改进
```

**核心洞察**：复盘是知识生产的核心环节，将经验转化为可复用的组织能力。

### 联系7：AI赋能 × 知识管理

```
RAG技术 ──→ 知识检索增强 ──→ 智能问答
五大引擎 ──→ 全流程赋能 ──→ 效率提升
```

**核心洞察**：AI不是替代人，而是放大人的能力，实现"人机协同"。

### 联系8：教员方法论 × 问题解决

## 🎯 使用指南

### 如何浏览知识图谱

1. **按模块浏览**：从总入口进入各模块，了解体系结构
2. **按标签检索**：使用标签快速定位相关知识
3. **按联系探索**：通过关联联系发现知识间的隐秘连接
4. **按场景应用**：根据实际问题找到解决方案路径
5. **按智能体架构**：了解智能体的感知-决策-学习机制

### 如何贡献知识联系

发现新的知识联系时：
1. 在knowledge-graph.md中添加新的联系节点
2. 说明联系类型（支撑/关联/转化/协同/智能体协同）
3. 提炼核心洞察（一句话）
4. 更新关联联系数统计
