# rag-skill - 知识库RAG增强 Skills

## 功能定位

**L2层·知识学习引擎扩展** - 增强知识检索与生成能力，与龙心OS的📚知识学习引擎深度整合。

## 核心能力

1. **向量检索增强**
   - 语义相似度匹配
   - 多文档跨域关联
   - 上下文感知回答

2. **知识库管理**
   - 文档分块与索引
   - 元数据标注
   - 定期更新机制

3. **RAG流程**
   - 检索（Retrieval）：从知识库提取相关文档
   - 增强（Augmentation）：将检索结果融入提示
   - 生成（Generation）：基于增强上下文生成回答

## 当前知识库架构

```
Obsidian知识库 ──┬── 核心独创Skills
                ├── 五行人格心理学
                ├── 龙心OS
                └── 观其妙书院

WorkBuddy Brain ──── 工作记忆
                └── 对话上下文

IMA笔记 ──────── 移动端快速记录
```

## RAG增强方案

### 短期（轻量化）
- 利用现有Obsidian + filesystem MCP
- 手动维护知识索引
- 通过对话触发知识调用

### 中期（自动化）
- 集成向量数据库（Chroma/Pinecone）
- 自动文档分块与嵌入
- 语义检索增强

### 长期（智能化）
- 多模态RAG（文本+图片+音频）
- 个性化知识图谱
- 主动知识推送

## 龙心OS整合

- 与📚知识学习协同：学习→索引→检索→应用
- 与🔄知行合一协同：知识沉淀→RAG增强→再次应用
- 与obsidian Skills协同：知识库↔RAG闭环

## 使用场景

- 检索历史案例参考
- 跨文档知识关联
- 专业领域问答增强
- 个性化知识助手
