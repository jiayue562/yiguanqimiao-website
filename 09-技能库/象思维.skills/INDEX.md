## 🎯 核心概念速查

### 三层次结构

```
原象（道象） ←── 灵界·信息体 ──→ 大圆满·法身
    ↑
意象（象征） ←── 心界·能量体 ──→ 大圆满·报身
    ↑
物象（形象） ←── 身界·物质体 ──→ 大圆满·化身
```

### 四大核心方法

1. **观物取象** → 认知：悬置→感知→提取→符号化
2. **立象尽意** → 表达：意会→寻象→立象→尽意→共鸣
3. **取象比类** → 推理：识别→寻找→建立→推导→验证
4. **观象制器** → 应用：观象→悟理→设计→制器→验证

### 六非两核心

**六非**：
- 非实体性、非对象性、非现成性
- 前语言性、前逻辑性、非确定性

**两核心**：
- 整体直观性、悟性

## 🎓 学习路径建议

### 初学者路径（1-2周）

1. 阅读 [[SKILL.md|主文档]] 了解核心概念
2. 阅读 [[references/theory.md|理论体系]] 第一部分（总论）
3. 阅读 [[references/practice-guide.md|实践指南]] 第一部分（日常修炼四步路径）
4. 开始日常观象练习

### 进阶者路径（1-2月）

1. 完整阅读 [[references/theory.md|理论体系]]
2. 深入学习 [[references/practice-guide.md|实践指南]] 的应用场景
3. 阅读 [[references/integration.md|整合详解]] 了解与其他体系的协同
4. 在实际问题中应用象思维

### 高阶者路径（3月+）

1. 研究 [[references/knowledge-graph.md|知识图谱]] 建立系统认知
2. 深入 [[references/wuxing-knowledge-network.md|五行图谱双向链接理论]] 构建五行知识网络
3. 探索象思维与五行人格的深度整合
4. 实践大圆满视角下的象思维修炼
5. 创造自己的"象"的应用案例
