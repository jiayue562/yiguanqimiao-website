# 岗位智能体 · 龙心OS整合说明

## 调用关系

```
龙心OS总智能体
  └─ 场景识别 → 路由到岗位智能体
                      │
                      ├─ 提示词模式 → 输出人用提示词
                      │                  └─ 知识学习技能（深度学习）
                      │                  └─ 知行合一技能（沉淀经验）
                      │
                      └─ 龙爪模式 → 输出 SKILL.md + Rule
                                     └─ 直接保存到 skills/ 目录
                                     └─ 直接保存到 rules/ 目录
```

## 与其他Skills的调用

| Skill | 调用时机 |
|-------|---------|
| 五行人格总智能体 | 识别性格类型时 |
| 知学习 | 生成方案后深度分析 |
| 知行合一 | 方案使用后沉淀经验 |
| 象思维 | 创新任务时 |
| 五色光思维 | 复杂分析时 |
