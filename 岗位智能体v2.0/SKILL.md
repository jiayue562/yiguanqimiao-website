# 🤖 岗位智能体 · SKILL.md

> **核心定位**：基于五行人格心理学OS的岗位AI工作流设计系统  
> **一句话功能**：通过"行业×企业×岗位×性格×任务×模型"六维框架，生成精准适配的AI提示词与智能体使用方案  
> **版本**：v1.0 | **创建日期**：2026-04-20 | **维护者**：龙龟神将

---

## 📖 核心定义

### What（是什么）

**岗位智能体**是龙龟神将独创的**业务场景×人格特质×思维模型**深度融合的AI提示词生成与岗位工作流设计工具。

它解决了传统提示词"一刀切"的问题，通过六维框架精准识别岗位特性，结合五行人格心理学，为每个岗位生成专属的AI协作方案。

**核心组成**：
- **六维输入框架**：行业 + 企业简介 + 岗位 + 性格（五行人格）+ 核心任务 + 适用模型
- **关键动作设计**：目标量化（结果指标）+ 资源分配 + 数据验证（过程指标）+ 风险控制
- **AI增强关键词**：洞察、剖析、透视、阐释、推演、解构、思辨、溯源、融合、启发、映射
- **提示词工程体系**：黄金公式 + 基础结构 + 高级技巧 + 专家级资产

### Why（为什么）

**现状问题**：
- 🔴 通用提示词无法满足特定岗位的深度需求
- 🔴 忽视人格特质导致AI协作效果不佳
- 🔴 缺乏系统化工作流设计方法
- 🔴 结果指标与过程指标混淆
- 🔴 风险控制缺乏预警机制

**解决方案**：
- ✅ **五行人格×业务场景深度融合**（木生火共生视角）
- ✅ **工作流化思维**（信息交汇点精准识别）
- ✅ **六维框架全面覆盖**（行业→企业→岗位→性格→任务→模型）
- ✅ **结果指标与过程指标双轨制**
- ✅ **风险指标前置预警**

### How（怎么做）

**核心工作流**：
```
六维输入（行业/企业/岗位/性格/任务/模型）
    ↓
五行人格识别与适配（金木水火土）
    ↓
关键动作拆解（目标量化/资源分配/数据验证/风险控制）
    ↓
AI增强关键词选择（十项认知指令）
    ↓
提示词框架生成（黄金公式/基础结构/高级技巧）
    ↓
完整智能体方案输出
```

## 🎮 双模式选择

> **核心升级**：岗位智能体现在支持两种模式。根据你的需求选择。

### 模式 A：👤 提示词模式（默认）
- **输出**：人用 AI 提示词
- **适用**：给企业员工设计 AI 协作助手
- **示例**：味藏总经理的 AI 助手提示词

### 模式 B：🐉 龙爪模式
- **输出**：WorkBuddy SKILL.md + 可选 Rule .mdc
- **适用**：给 WorkBuddy 造龙爪（AI 智能体）
- **示例**：五行人格诊断咨询师龙爪

### 模式切换指令

| 你想做什么 | 输入 |
|-----------|------|
| 给人设计提示词 | 默认模式，直接按六维框架输入 |
| 造 WorkBuddy 龙爪 | 先说"切换到龙爪模式"，再输入设计 |

### 两种模式的流程对比

```
提示词模式：
六维输入 → 五行识别 → 关键动作 → AI关键词 → 提示词框架 → 输出人用提示词

龙爪模式：
六维输入 → 五行识别 → 关键动作 → AI关键词 → 选择输出类型
  └→ 选项B：输出 SKILL.md + 资产引用 + 可选 Rule
```

### Step 1: 六维信息采集

**输入模板**：

```yaml
六维框架:
  行业: [餐饮/科技/教育/金融等]
  企业简介: [规模/阶段/文化特色]
  岗位: [市场总监/总经理/运营经理等]
  性格: [水行人-灵活/火行人-热情/木行人-创新/金行人-目标/土行人-稳定]
  核心任务: [渠道投放/用户增长/品牌策划等]
  适用模型: [GROW/OKR/第一性原理/五色光方法论等]
```

**示例**：
```yaml
六维框架示例:
  行业: 餐饮行业，日料，金枪鱼专门店
  企业简介: 味藏餐饮，专注东方健康烹饪，蓝鳍金枪鱼核心产品
  岗位: 总经理
  性格: 金行人-目标计划
  核心任务: 执行高盈利产品战略目标
  适用模型: 企业经营管理模型
```

### Step 2: 五行人格识别与适配

**五行-岗位适配表**：

| 五行 | 核心特质 | 适配岗位类型 | 适配任务类型 |
|------|----------|--------------|--------------|
| **木行人** | 创新、生发、仁德 | 战略规划、产品研发、创意设计 | 0→1突破、长期规划、创新项目 |
| **火行人** | 光明、热情、礼明 | 市场推广、团队管理、公关传播 | 品牌传播、团队激励、危机公关 |
| **土行人** | 承载、稳定、信实 | 运营管理、客户服务、供应链管理 | 流程优化、服务提升、稳定执行 |
| **金行人** | 决断、精准、义理 | 财务管理、数据分析、质量管理 | 决策分析、风险控制、标准制定 |
| **水行人** | 智慧、流动、润泽 | 战略规划、人力资源、商务拓展 | 策略制定、人才发展、资源整合 |

**五行适配策略**：

```yaml

  提示词风格: 强调稳定、执行、落地性
  AI关键词: 剖析、解构、溯源

金行人适配:
  提示词风格: 强调精准、数据、决策性
  AI关键词: 思辨、推演、解构
  协作模式: 共创导师（第三象限）

  提示词风格: 强调灵活、策略、应变性
```

### Step 4: AI增强关键词选择

**十项认知指令**：

| 关键词 | 功能 | 适用场景 | 五行适配 |
|--------|------|----------|----------|
| **洞察** | 深入理解本质 | 需求分析、问题诊断 | 木行人、水行人 |
| **剖析** | 拆解概念要素 | 复杂问题分解 | 金行人、土行人 |
| **透视** | 透过现象看本质 | 根因分析、趋势判断 | 火行人、水行人 |
| **阐释** | 全面解释展开 | 概念传达、方案说明 | 火行人、土行人 |
| **推演** | 逻辑推理预测 | 方案评估、风险预测 | 金行人、水行人 |
| **解构** | 拆分为基本元素 | 流程优化、系统重构 | 金行人、木行人 |
| **思辨** | 批判性多角度分析 | 决策分析、方案对比 | 金行人、水行人 |
| **溯源** | 追根溯源 | 历史分析、根因挖掘 | 水行人、木行人 |
| **融合** | 整合不同领域 | 跨界创新、综合方案 | 木行人、火行人 |
| **启发** | 开拓新思路 | 创意激发、突破瓶颈 | 木行人、火行人 |
| **映射** | 框架结构转化 | 知识迁移、方案套用 | 水行人、土行人 |

**关键词选择策略**：

```yaml
初级复杂度（1-2个关键词）:
  适用: 简单任务、明确需求
  示例: [洞察] + [阐释]
  
中级复杂度（3-4个关键词）:
  适用: 中等复杂度、需要分析
  示例: [洞察] + [剖析] + [推演] + [思辨]
  
高级复杂度（5个以上关键词）:
  适用: 复杂任务、系统方案
  示例: [洞察] + [剖析] + [透视] + [推演] + [融合] + [启发]
```

### Step 5: 提示词框架生成

#### 黄金公式

```
提示词 = 角色 + 任务 + 上下文 + 要求
```

**示例**：
```
角色: 你是一位经验丰富的餐饮行业运营总经理，拥有15年高端日料管理经验
任务: 设计一份高盈利产品的战略方案
上下文: 味藏餐饮，专注东方健康烹饪，蓝鳍金枪鱼核心产品，客单价80元
要求: 
  1. 符合SMART原则
  2. 具有可操作性、易复制性
  3. 包含具体行动项和负责人
  4. 输出格式为Markdown表格
```

#### 基础结构技巧

**1. 角色扮演**：
```
你是一位[具体角色]，拥有[具体背景]。请以[语气/风格]完成[任务]。
```

**2. 任务分解**：
```
请分步骤完成[任务]：
第一步: [具体步骤]
第二步: [具体步骤]
每完成一步后等我确认再继续。
```

**3. 格式化约束**：
```
请用[格式]输出，包含[具体要求]。
限制条件: [字数/格式/风格限制]
```

**4. 上下文与约束**：
```
背景信息: [详细背景]
约束条件: [字数/语气/避免内容]
目标受众: [受众描述]
```

#### 高级技巧

**5. 少样本提示**：
```
请参考以下示例风格：
示例1: [示例内容]
示例2: [示例内容]
请按照以上风格完成: [新任务]
```

**6. 思维链推理**：
```
请一步一步思考：
1. 首先分析[问题]
2. 然后考虑[因素]
3. 最后得出[结论]
请展示完整推理过程后再给出建议。
```

**7. 多视角分析**：
```
请从以下视角分析[问题]：
- 用户视角: [关注点]
- 公司视角: [关注点]
- 竞争对手视角: [关注点]
```
