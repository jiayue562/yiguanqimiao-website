1.3 五维加权计分算法

五行综合得分 = 心理得分×0.20 + 身体得分×0.30 + 灵性得分×0.15 + 阴阳子维度得分×0.30 + 社会掩饰扣分

其中：
- 心理得分 = 心理30题中木/火/土/金/水各得分数
- 身体得分 = 身体23题中木/火/土/金/水各得分数
- 灵性得分 = 灵性10题中阳面/阴面各得分数
- 阴阳子维度得分 = 140题中阳面总得分 - 阴面总得分
   - 阳面>阴面 = 健康区（4-6层）
   - 阴面>阳面 = 不健康区（1-3层）
- 社会掩饰扣分 = 社会掩饰10题得分
   - 0-2分：不扣分
   - 3-5分：总分乘以0.95
   - 6-8分：总分乘以0.80
   - 9-10分：结果无效

1.4 阴阳判定规则

条件 判定结果 健康层级
阳面得分 > 阴面得分 + 2.0 阳面主导·健康区 4-6层
阴面得分 > 阳面得分 + 2.0 阴面主导·不健康区 1-3层
|阳面得分 - 阴面得分| ≤ 2.0 阴阳混杂·转化期 3-4层边界
社会掩饰得分 ≥ 6 结果存疑·需要重测 待定

1.5 四重判定规则

  1. 单行确认：最高分 - 次高分 ≥ 2.0 → 单行确认（激活对应分智能体）
  2. 主副行并行：最高分 - 次高分 1.0~2.0 → 主行+副行并行分析
  3. 多行混合：最高分 - 次高分 < 1.0 → 多行混合·深度追问;
  4. 阴阳转化：阳面得分 - 阴面得分 符号变化 → 近期有转化发生·追问生活事件;
