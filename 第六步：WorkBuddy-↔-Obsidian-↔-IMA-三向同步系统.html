⚠️ 常见问题

1. IMA API 连接失败

错误信息：认证失败，请检查凭证

解决方案：
1. 检查 Client ID 和 API Key 是否正确
2. 确认网络连接正常
3. 访问 https://ima.qq.com/agent-interface 重新获取凭证

2. Obsidian 文件权限错误

错误信息：Permission denied

解决方案：
1. 确认 Obsidian 未运行（文件可能被锁定）
2. 检查目录写入权限
3. 使用管理员权限执行脚本

3. 内容过长错误

错误信息：错误码 100009

解决方案：
1. 将内容拆分为多个部分
2. 分批次写入 IMA
3. 压缩不重要的历史内容
