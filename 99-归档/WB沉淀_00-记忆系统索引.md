## 四、核心原则

### 记忆加载顺序
```
每次会话开始：
1. 读取 MEMORY.md（长期记忆）
2. 读取 USER.md（用户偏好）
3. 读取 SESSION.md（会话状态）
4. 读取 SOUL.md（身份设定）
```

### 记忆沉淀规则
```
每次有价值的信息：
1. 评估重要性
2. 分类到对应层
3. 写入对应文件
4. 更新总索引
```
