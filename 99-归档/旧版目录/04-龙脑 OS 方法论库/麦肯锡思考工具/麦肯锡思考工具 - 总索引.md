## 📋 核心定义

> **麦肯锡思考工具**是全球顶级咨询公司麦肯锡在商业分析、战略规划和问题解决中广泛应用的 46 种高效思维框架，覆盖逻辑思考、创意思考、问题解决、市场营销、组织管理与经营战略六大领域。

**核心理念**: 利用"框架"这一思维架构，将复杂、模糊的商业问题结构化，实现高效决策和精准解决问题。

**三大价值**:
1. 提升分析和验证的准确度
2. 具备快速决策能力
3. 具备逻辑表达能力

## 📚 麦肯锡七步问题解决法

```
步骤 1: 陈述问题 → 5W1H 确保问题边界清晰
    │
    ▼
步骤 2: 分解问题 → 逻辑树+MECE 原则拆解
    │
    ▼
步骤 3: 消除非关键议题 → 帕累托法则+优先级矩阵
    │
    ▼
步骤 4: 制定详细工作计划 → 议题树明确验证方法
    │
    ▼
步骤 5: 进行关键分析 → 假设思考 + 快速验证
    │
    ▼
步骤 6: 综合结果并建立有结构的结论 → 金字塔结构
    │
    ▼
步骤 7: 整理一套有力度的文件 → PREP 法 + 空 - 雨 - 伞
```
