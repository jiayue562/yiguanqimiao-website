## 📋 核心定义

> **思维模型独立 Skills**是将 100 个思维模型每个都封装为独立的 Skill，采用 Skill Builder 5 阶 20 步标准化流程创建，每个 Skill 都可被龙心 OS 独立调用或组合调用。

**系统归属**: 龙脑 OS（被龙心 OS 按需调用）

**调用关系**: 
```
用户问题 → 龙心 OS（场景识别）→ 调用龙脑 OS（单个或多个思维模型 Skills）→ 返回分析结果 → 龙心 OS（调度输出）
```

**核心原则**:
1. 每个思维模型独立封装（7 个文件/Skill）
2. 龙心 OS 统一调度
3. 按需调用，灵活组合
4. 中西合璧：象思维（0→1）→ 思维模型（1→N）→ 五色光（验证）
