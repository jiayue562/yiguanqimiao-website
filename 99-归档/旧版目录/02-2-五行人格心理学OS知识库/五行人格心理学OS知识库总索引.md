## 🧠 核心理论框架

### 一心三界五行九层

**一心**：纯粹觉知
- 木：生长性 + 觉照性
- 火：光明性 + 洞察性
- 土：承载性 + 稳定性
- 金：清明性 + 决断性
- 水：润泽映现性 + 沉潜含藏性

**三界**：身界（物象） / 心界（意象） / 灵界（原象）

**五行**：木 / 火 / 土 / 金 / 水

**九层**：解放（1-3级）→ 失衡（4-6级）→ 远道（7-9级）

### 转化技术体系

1. **拔阴取阳**：认不是 → 找好处 → 信因果 → 达天时
2. **化克为生**：木克土 → 木生火 → 火生土
3. **相生强化**：土生金 → 金生水
