## 🎯 使用示例

### 示例 1: 编译单篇文章
```
用户："编译《知识管理.pdf》"
↓
LLM Wiki Skill 自动触发
↓
1. 读取 raw/articles/知识管理.pdf
2. 提炼核心概念 (3-5 个)
3. 生成 wiki/concepts/知识管理.md
4. 建立双向链接
5. 执行健康检查
6. 更新索引
7. 询问是否反哺三库
```

### 示例 2: 建立领域知识库
```
用户："建立五行人格知识库"
↓
1. 收集 raw/中的五行人格资料
2. 编译核心概念
3. 创建 wiki/theories/五行人格/目录
4. 建立双向链接
5. 执行健康检查
6. 更新分类索引
```

### 示例 3: 定期健康检查
```
用户："运行健康检查"
或 Cron 自动触发 (每周一 9:00)
↓
python scripts/llm-wiki-health-check.py
↓
生成健康检查报告
提出修复建议
```
