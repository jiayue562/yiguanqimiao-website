## 🌟 核心功能

### 功能 1: 会话上下文管理

**组件**:
- `optimizer.py`: 压缩冗余，保留关键信息
- `summarizer.py`: 自动生成会话摘要
- `prioritizer.py`: 智能排序上下文重要性

**使用示例**:
```python
from session-context.optimizer import optimize_context
from session-context.summarizer import summarize
from session-context.prioritizer import prioritize

# 优化上下文
optimized = optimize_context(context, target_length=1500)

# 生成摘要
summary = summarize(conversation, max_length=500)

# 优先级排序
ranked = prioritize(contexts, criteria=['relevance', 'recency'])
```
