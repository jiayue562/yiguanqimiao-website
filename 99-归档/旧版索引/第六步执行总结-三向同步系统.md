## 🔄 三向同步架构

```
                    ┌─────────────────┐
                    │  WorkBuddy      │
                    │  (工作记忆)     │
                    └────────┬────────┘
                             │
              ┌──────────────┼──────────────┐
              │              │              │
              ▼              ▼              ▼
    ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
    │  Obsidian   │ │     IMA     │ │  自动化任务  │
    │ (主知识库)  │ │ (移动笔记)  │ │  (定时同步)  │
    │             │ │             │ │  23:30/日   │
    └─────────────┘ └─────────────┘ └─────────────┘
```

### 同步流程

1. **扫描阶段**
   - 扫描 WorkBuddy Skills 目录
   - 扫描 WorkBuddy Brain 目录
   - 扫描 WorkBuddy Automations 目录

2. **Obsidian 同步**
   - 按分类规则映射到对应目录
   - 保持文件结构和元数据
   - 创建备份目录结构

3. **IMA 同步**
   - 测试 API 连接
   - 获取笔记本 ID
   - 搜索现有笔记
   - 创建或更新笔记
   - 处理内容长度限制

4. **报告生成**
   - 统计同步文件数量
   - 记录同步时间
   - 生成 Markdown 报告
   - 保存到 Obsidian

## 🔧 故障排除

### IMA API 连接失败

**错误信息**：`认证失败，请检查凭证`

**解决方案**：
1. 检查 Client ID 和 API Key 是否正确
2. 访问 https://ima.qq.com/agent-interface 重新获取凭证
3. 更新 `config.json` 中的凭证

### Obsidian 文件权限错误

**错误信息**：`Permission denied`

**解决方案**：
1. 确认 Obsidian 未运行（文件可能被锁定）
2. 检查目录写入权限
3. 使用管理员权限执行脚本

### 同步报告未生成

**可能原因**：
1. 同步过程中出现异常
2. 目录权限问题
3. 磁盘空间不足

**解决方案**：
1. 检查脚本输出日志
2. 验证目录权限
3. 清理磁盘空间
