## 🔧 使用指南

### 快速查找
1. 按类型浏览：Areas/Projects/Resources/Journal
2. 按标签筛选：#标签名
3. 搜索关键词：`obsidian-agent search "关键词"`

### 添加新笔记
```powershell
# 创建 Area
obsidian-agent note "新领域" area

# 创建 Project
obsidian-agent note "新项目" project

# 创建 Resource
obsidian-agent note "新资源" resource

# 创建 Journal
obsidian-agent journal
```

### 建立链接
- 使用 `[[笔记名]]` 创建双向链接
- 在相关笔记中互相引用
- 定期运行 `obsidian-agent orphans` 检查孤立笔记
