## 一、木行人职业性格特征

### 1. 阳性特质

| 特质 | 表现 |
|------|------|
| **+ 逻辑性强的** | 善于逻辑分析，思维清晰，善于解决复杂问题 |
| **+ 放松、安静的** | 情绪稳定，不易焦虑，能在压力下保持冷静 |
| **+ 独特、有思想的** | 有独特见解，不随波逐流，善于创新思考 |
| **+ 理想主义** | 有理想有追求，不满足于现状，追求更高目标 |
| **+ 在习惯性领域** | 在熟悉领域表现出色，能深入理解专业知识 |

### 2. 阴性特征

| 特征 | 表现 |
|------|------|
| **- 关注细节的** | 过于关注细节，忽略大局，容易陷入琐碎 |
| **- 冷淡、抗上的** | 对上级冷淡，不善于处理上下级关系 |
| **- 忧愁、不服的** | 容易忧愁，不服输，容易与人发生冲突 |
| **- 不喜风险的** | 不喜欢冒险，宁愿选择安全稳定的道路 |
| **- 节奏缓慢的** | 工作节奏慢，不着急，容易拖延 |

### 3. 职业行为特征

| 行为 | 阳性表现 | 阴性表现 |
|------|---------|---------|
| **自认天生与众不同** | 衣着言行均有个人风格,拒绝平庸,行为独特丰富的想象力可以成为任何角色,最担心和别人毫无区别,若证明品位与他人一样就无趣 | 服饰装扮玩酷,重视视觉刺激、生活在自己的思维世界里,不遗余力地制造遥不可及的感受 |
| **善于并乐于通过解决工作中的难题展示自己的才华出众** | 在擅长的领域孤芳自赏，非常介意别人在专业上超越自己,工作中遇到问题不愿向他人请教，宁可自己研究 | 追求专业的深度，知识广博,见解独到,喜欢成为专家和顾问。关注发展个人品牌及魅力。善于收集知识及咨讯,需要知道宏观及全面的信息 |
| **愿意代表专业的权威发表言论** | 不屑接受他人的意见。工作中喜欢自行其是，不愿进行团队的配合,不愿与人分享成果 | 经常能在工作中产生有创意的办法和思路追求超越常规工作方法或流程的改进 |
| **敢于挑战权威** | 认为领导不够专业、能力不足，没有自己来担当更具有能力。 | 喜欢从事帮助他人的工作、工作尽职尽责；行事温和儒雅。 |
| **总认为公司或社会没有给自己足够空间和平台施展自己的才华** |  |  |

## 二、火行人职业性格特征

### 1. 阳性特质

| 特质 | 表现 |
|------|------|
| **+ 积极主动** | 工作积极主动，不需要别人督促，主动承担责任 |
| **+ 鼓舞、热情的** | 能鼓舞团队士气，充满热情，感染他人 |
| **+ 健谈好客的** | 善于交际，喜欢与人交谈，好客，容易建立人际关系 |
| **+ 精力充沛的** | 精力充沛，能承受高强度工作，不知疲倦 |
| **+ 以人驱动目标任务** | 善于用人际关系驱动目标达成，重视人的因素 |

### 2. 阴性特征

| 特征 | 表现 |
|------|------|
| **- 情绪多变的** | 情绪不稳定，容易波动，时而热情时而冷漠 |
| **- 急躁、苛求的** | 性格急躁，没有耐心，对别人要求过高 |
| **- 夸大、炫耀的** | 喜欢夸大自己的成就，爱炫耀，给人不踏实的感觉 |
| **- 虚荣、好争的** | 虚荣心强，好争理，喜欢与人争论，不肯认输 |
| **- 超级明星** | 自认为是超级明星，不容易与团队合作 |

### 3. 职业行为特征

| 行为 | 阳性表现 | 阴性表现 |
|------|---------|---------|
| **讲话底气足,能量大,声音洪亮** | 追求完美,穿着得体,礼仪举止合度,落落大方,越在人多的地方表达越自信,不触场面 | 对外表特别注重,色彩鲜艳,时尚名牌,讲究排场,引人注目 |
| **善于制定准则,并要求大家认同,跟随自己原则做事** | 不容易降低标准或妥协。不喜欢做自己没有影响能力的工作 | 好争理,缺乏耐心接受其他的意见和批评，总觉得自己是真理的化身 |
| **对社会道德正义有是非感** | 自己严格遵守，主动监督他人。喜欢引申或上升为社会的价值和标准规范 | 好面子,情感世界薄弱,害怕被否定,经常为自己的观点进行辩解和争论，并多谈自己的长处。 |
| **激励的、乐观的、参与式的良好的协调者** | 友善地及引起他人兴趣的,易促进事件发展 | 工作中总希望别人服从自己的决定，而不迁就他人的感受或意见。 |
| **敢担风险，挑战的空间越大越敢于接受** | 富于冒险精神，有自信心、进取心，雄心壮志 | 言语苛求,对人指责,态度严肃,极力督察。 |

## 三、土行人职业性格特征

### 1. 阳性特质

| 特质 | 表现 |
|------|------|
| **+ 以团队为中心的** | 重视团队利益，愿意为团队做出贡献，有团队合作精神 |
| **+ 具有合作精神的** | 善于与人合作，不独断专行，能融入团队 |
| **+ 诚信、助人的** | 诚实守信，愿意帮助他人，有服务意识 |
| **+ 有责任感** | 有责任感，敢于承担责任，不推诿 |
| **+ 以资源整合为优势** | 善于整合资源，能调动各方力量完成任务 |

### 2. 阴性特征

| 特征 | 表现 |
|------|------|
| **- 目标不突出的** | 目标感不强，容易迷失方向，不知道自己要什么 |
| **- 保守、固执的** | 思想保守，不接受新事物，固执己见 |
| **- 沉闷、迟钝的** | 性格沉闷，反应迟钝，不善于表达 |
| **- 疑心重的** | 疑心重，不容易相信别人，防卫心理强 |
| **- 不自信的** | 不自信，容易否定自己，不敢承担责任 |

### 3. 职业行为特征

| 行为 | 阳性表现 | 阴性表现 |
|------|---------|---------|
| **衣着简朴,不铺张浪费,不搞奢华** | 举止稳健,言语诚实,不说谎话,答应了就一定办到。帮助别人而不夸耀,不和其他同事攀比,只是作好自己的工作,以实干证明自己的信誉 | 不重外表,不加修饰，说话声音单调,不生动。神采不足,招惹不服 |
| **在工作中总能为其他流程的工作着想，为他人提供方便** | 勇于承担责任,讲诚信,关键时刻,带头加班完成任务 | 缺乏计划与监控,同时做许多件事,重点不突出。愿意做重复平庸的事,缺乏创新。工作中基本上属于没有什么过失,也没有什么成就。做事顾虑重重，思来想去,难下决断, |
| **不喜结交权贵,不巴结上级** | 与伙伴齐心同力,群众基础好,以身作则,带头吃苦 | 对权威的领导相当不信任，表现出顺从的姿态的同时又带有怀疑的眼光 |
| **勤恳实干，稳定持久** | 恶劣环境依然从容。生理耐受力强,胜任高强度的体力和精力压力如加班出差的工作 | 不自信,放大困难,怀疑一切,好怨人。认为事情做不好都是比人的错。 |
| **包容量大,同事觉得您是可以信赖的人** | 待人友善,为人忠诚。不恶意对待同事,生气或不满时也能冷静回避；被冤枉也不叫屈。有忍让精神,是很好的倾谈对象,不轻易打断对方的谈话 | 接受新鲜事物的能力慢,循序渐进,按部就班,容易落伍。 |

## 四、金行人职业性格特征

### 1. 阳性特质

| 特质 | 表现 |
|------|------|
| **+ 刚毅果断的** | 性格刚毅，果断决策，不拖泥带水 |
| **+ 有远见的** | 有远见，能看到未来的趋势，善于规划 |
| **+ 注意力集中的** | 注意力集中，能长时间专注一件事 |
| **+ 有管理组织能力的** | 善于管理组织，能带领团队完成任务 |
| **+ 以目标为导向的** | 以目标为导向，重视结果，不重过程 |

### 2. 阴性特征

| 特征 | 表现 |
|------|------|
| **- 冷酷、集权的** | 性格冷酷，集权，不容易相信别人 |
| **- 忽略他人的** | 忽略他人的感受，只关注任务和目标 |
| **- 计较的、比较的** | 斤斤计较，喜欢比较，不容易满足 |
| **- 自负的** | 自负，自以为是，听不进别人的意见 |
| **- 小气的** | 小气，吝啬，不愿意分享 |

### 3. 职业行为特征

| 行为 | 阳性表现 | 阴性表现 |
|------|---------|---------|
| **着装正式,符合成功人士的形象** | 标准的衣着打扮,给人留下深刻的印象 | 有竞争意识,支配他人的愿望强,比他能力强不服气,比他能力差也不成嫌人懒恨人笨 |
| **能力出色,鞠躬尽瘁地工作** | 恐怕功高镇主,常怀感恩心态 | 不断追逐下一个目标,拜金主义,讲究物质享受,认为是能力与身份的象征 |
| **严格遵守职业操守，从不破坏制度并以身作则** | 是典型的工作狂,容易紧张,回避感情,看重自己的表现和成就 | 独立,高效,在工作中奖罚分明,行动有计划,善于制定目标及执行，做事充满动力，全力以赴 |
| **善良中透着精明与挑剔** | 自尊建立在对结果的认可上,而非别人对他的喜爱上,工作风格:对事不对人 | 不喜欢和能力差的同事交流,认为浪费时间.喜欢以自我为中心，认识刻板、固执，缺乏灵活 |
| **努力工作的实干家** | 认为人的价值在于所从事的事业,天生具有直觉,多才多艺永远符合职业要求 |  |

## 五、水行人职业性格特征

### 1. 阳性特质

| 特质 | 表现 |
|------|------|
| **+ 智慧的** | 有智慧，善于思考，能洞察事物本质 |
| **+ 善解人意的** | 善解人意，能理解他人的感受，有同理心 |
| **+ 沉稳的、安静的** | 性格沉稳，安静，不浮躁，能冷静思考 |
| **+ 和谐团建的** | 善于和谐团队，能化解矛盾，促进团队和谐 |
| **+ 以情感为导向的** | 以情感为导向，重视人际关系，不重视任务 |

### 2. 阴性特征

| 特征 | 表现 |
|------|------|
| **- 松懈怠工的** | 松懈怠工，不努力工作，容易偷懒 |
| **- 懒散、混乱的** | 懒散，生活工作混乱，没有条理 |
| **- 消沉、抑郁的** | 消沉，容易抑郁，情绪低落 |
| **- 缺乏安全感的** | 缺乏安全感，容易焦虑，需要别人保护 |
| **- 意志薄弱的** | 意志薄弱，容易放弃，不能坚持 |

### 3. 职业行为特征

| 行为 | 阳性表现 | 阴性表现 |
|------|---------|---------|
| **追求自然平和，不注重衣服的款式** | 但注重衣服的质地，更注重精神修养 | 随遇而安,不喜冲突,工作不求上进,得过且过,能将就就将就 |
| **为人真实,待人真诚,办事真切** | 从不当面顶撞,经常调解矛盾 | 言语富有感情色彩,喜欢发展私人情感。喜欢送礼请客,工作以拉关系为主 |
| **办事灵活,适应性强** | 城府较深，高度内向,愿意与人交往。愿意跟团队的成员共同学习共同进步,不挑事,只灭火。内心世界解码:和谐 | 关注关系情感胜过目标任务,承担压力的能力低 |
| **认为工作要快乐** | 只有快乐的工作才是真正的工作.看中工作场所的环境与布置，注重情调 | 为了维护关系自己承担了太多的付出,一旦超出自己的包容,就会选择逃避, |
| **助人付出,不求回报** | 受气包,爱烦人,爱哭,懒散混乱,生闷气,想的多做的少，消沉抑郁。文饰过失,胜人一筹 |  |
