## 使用方法

| 步骤 | 内容 |
|------|------|
| **第一步** | 选择对比组：根据自己当前的困惑，选择对应的五行对比组（如：木火对比/木土对比/木金对比/木水对比） |
| **第二步** | 逐条对照：逐一阅读该对比组的各个小指标，标记符合自己的条目 |
| **第三步** | 统计得分：统计符合的条目数量，判断自己更偏向哪一行的阴面或阳面 |
| **第四步** | 分析根源：分析这些表现的根本原因（五行过盛/不及/相克） |
| **第五步** | 制定方案：根据五行信任模型，制定"拔阴取阳"转化方案 |
| **第六步** | 培养阳面：针对性培养对应的阳面品质 |
| **第七步** | 团队能量成像：通过三次角色扮演，可视化团队五行能量分布与流动 |
| **第八步** | 职业匹配：根据自己的五行类型，选择适合的行业和职业 |
| **第九步** | 外观识别：通过五行人金口诀，快速识别他人的五行类型 |
